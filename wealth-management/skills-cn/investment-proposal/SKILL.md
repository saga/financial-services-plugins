# 投资提案

为潜在客户创建专业的投资提案。

## 工作流

### 步骤1：潜在客户背景

收集：
- **潜在客户名称**及家庭详情
- **当前情况**：现有顾问？自主管理？什么促使会议？
- **资产**：预估AUM、账户类型、当前持仓（如分享）
- **目标**：退休、财富保值、增长、教育、遗产
- **风险承受能力**：保守、适度、激进（或问卷分数）
- **限制**：ESG偏好、控股股票、流动性需求
- **费用敏感度**：他们现在付多少？
- **竞争**：他们还在考虑谁？

### 步骤2：提案结构

**一、关于我们**（1页）
- 公司概览、历史、AUM
- 投资理念（通俗易懂）
- 团队简介（与此客户相关）
- 客户服务模式（多久见一次，他们联系谁）

**二、了解您的需求**（1页）
- 重述他们的目标和担忧——表明你倾听了
- 发现中确定的关键规划考虑
- 对他们来说成功是什么样的

**三、建议的投资策略**（2-3页）
- 建议的资产配置及理由
- 配置如何映射到他们的目标和风险承受能力
- 投资工具（ETF、个股、另类）
- 税务意识策略（资产位置、税损收割）

建议配置：

| 资产类别 | 配置 | 工具 | 理由 |
|----------|------|------|------|
| | | | |

**四、预期结果**（1-2页）
- 预测增长情景（保守、适度、乐观）
- 蒙特卡洛实现目标的概率
- 收入预测（如退休或收入导向）
- 风险指标（最大回撤、波动率）
- 与当前投资组合比较（如已知）

**五、费用结构**（1页）
- 顾问费表（如适用）
- 底层基金费用
- 总费用估算
- 费用与行业平均比较
- 价值主张——他们付的费用得到什么

**六、开始**（1页）
- 账户开设流程
- 资产转移时间表
- 过渡计划（如从其他顾问处转移）
- 前90天——预期什么
- 所需文件和后续步骤

### 步骤3：定制

- 根据潜在客户调整语气（企业高管vs小企业主vs退休人员）
- 如果他们有控股股票，直接处理
- 如果他们在比较机器人顾问，强调规划和关系价值
- 如果他们价格敏感，以总价值和结果领先，而不仅仅是费用

### 步骤4：输出

- 带公司品牌的PowerPoint演示（12-15页）
- PDF留存版本
- 后续邮件的一页摘要

## 重要说明

- 提案应感觉是定制的，而不是模板化的——引用他们的具体情况
- 不要过度销售业绩——设定切合实际的期望并强调过程
- 始终包含免责声明（预测是假设性的、过往业绩等）
- 过渡计划很重要——客户担心更换顾问的干扰
- 48小时内跟进提案和明确的后续步骤
- 合规必须在呈现给潜在客户前审查
