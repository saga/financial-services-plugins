# 客户审查准备

为客户审查会议做准备。

## 工作流

### 步骤1：客户背景

收集或查询：
- **客户名称**及家庭成员
- **账户类型**：应税、IRA、Roth、401(k)、信托等
- **管理资产总额（AUM）**
- **投资政策声明（IPS）**：目标配置、风险承受能力、限制
- **人生阶段**：积累、退休前、退休、传承
- **上次会议日期**及任何未完成的行动项目

### 步骤2：投资组合业绩

对于每个账户和家庭汇总：

| 指标 | QTD | YTD | 1年 | 3年 | 成立以来 |
|------|-----|-----|-----|-----|----------|
| 投资组合回报 | | | | | |
| 基准回报 | | | | | |
| Alpha | | | | | |

**业绩归因：**
- 哪些资产类别/持仓推动了回报？
- 前3大贡献者和前3大拖累
- 是否有任何单一持仓的过大影响？

### 步骤3：配置审查

当前vs目标配置：

| 资产类别 | 目标 | 当前 | 漂移 | 行动 |
|----------|------|------|------|------|
| 美国大盘股 | | | | |
| 美国中小盘 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 固定收益 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记超过IPS再平衡阈值（通常3-5%）的任何漂移。

### 步骤4：讨论要点

生成会议议程：

1. **市场概览**（2-3分钟）：简要宏观背景和展望
2. **投资组合业绩**（5分钟）：我们表现如何？为什么？
3. **配置审查**（5分钟）：需要再平衡吗？
4. **规划更新**（5-10分钟）：
   - 生活变化？（工作、健康、家庭、房屋、教育）
   - 收入需求变化？
   - 税务情况更新
   - 遗产规划更新
5. **行动项目**（5分钟）：下次会议前我们要做什么？

### 步骤5：主动建议

根据审查提出：
- 再平衡交易（如果漂移超过阈值）
- 税损收割机会
- 现金部署或提取规划
- Roth转换机会（如适用）
- 受益人更新或遗产规划需求
- 保险审查（人寿、残疾、长期护理）

### 步骤6：输出

- 一页客户审查摘要（Word或PDF）
- 带基准的业绩表
- 配置饼图（当前vs目标）
- 建议的行动项目
- 会议议程

## 重要说明

- 会议前了解你的客户——回顾上次会议的笔记
- 从客户关心的开始，而不是你想谈的
- 如果业绩不好，直接处理——不要隐藏或辩解
- 始终以明确的行动项目和带日期的后续步骤结束
- 记录会议笔记和IPS的任何变更
- 合规：确保所有材料符合公司政策和监管要求
