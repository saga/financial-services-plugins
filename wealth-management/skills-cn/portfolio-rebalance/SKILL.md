# 投资组合再平衡

分析投资组合配置漂移并生成再平衡交易建议。

## 工作流

### 步骤1：当前状态

对于每个账户，记录：
- 账户类型（应税、IRA、Roth、401k）
- 持仓及当前市值
- 成本基础（对于应税账户）
- 每个持仓的未实现盈亏

### 步骤2：漂移分析

将当前配置与IPS目标比较：

| 资产类别 | 目标% | 当前% | 漂移 | $超/不足 |
|----------|-------|-------|------|---------|
| 美国大盘股 | | | | |
| 美国中小盘 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 投资级债券 | | | | |
| 高收益/信用 | | | | |
| TIPS/通胀保护 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记超过再平衡区间（通常±3-5%）的持仓。

### 步骤3：交易建议

生成使配置回归目标的交易：

**税务意识再平衡规则：**
- 优先在税收优惠账户（IRA、Roth）中再平衡——无税务后果
- 在应税账户中，避免出售有大方期短期收益的持仓
- 在再平衡时尽可能收割损失
- 留意跨账户的洗售规则（30天窗口）
- 考虑将新投入 направлять到低配资产类别而不是交易

**交易清单：**

| 账户 | 操作 | 证券 | 股数/$ | 理由 | 税务影响 |
|------|------|------|--------|------|----------|
| | 买入/卖出 | | | 再平衡/TLH | ST收益/LT收益/损失 |

### 步骤4：资产位置审查

优化资产在哪种账户类型中持有：
- **税收递延（IRA/401k）**：债券、REITs、高周转基金（最高税务拖累）
- **Roth**：预期增长最高的资产（免税增长）
- **应税**：税收高效的股权（指数基金、ETF、市政债）、税损收割候选

### 步骤5：实施

- 各账户总交易
- 预估交易成本
- 预估税务影响（已实现盈亏）
- 漂移净影响

### 步骤6：输出

- 漂移分析表
- 建议交易清单（Excel）
- 税务影响摘要
- 前后配置比较

## 重要说明

- 不要为再平衡而再平衡——区间内的小漂移是可以的
- 税务成本可能超过应税账户再平衡的好处——计算盈亏平衡点
- 交易前考虑待发生的现金流（投入、提取、RMD）
- 检查任何客户特定限制（ESG、控股股票、锁定期）
- 为合规记录记录每笔交易的理由
- 洗售规则适用于跨账户——协调家庭账户间的交易
