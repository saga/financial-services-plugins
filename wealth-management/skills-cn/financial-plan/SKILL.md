# 财务计划

构建或更新全面的财务计划。

## 工作流

### 步骤1：客户档案

收集或确认：
- **人口统计**：年龄、配偶年龄、抚养人、预期寿命假设
- **就业**：当前收入、预期加薪、目标退休年龄
- **账户**：所有投资账户的余额和资产配置
- **收入来源**：工资、奖金、租金收入、社保估算、养老金
- **支出**：当前年度支出、预期变化（抵押贷款还清、孩子独立）
- **负债**：抵押贷款、学生贷款、其他债务
- **保险**：人寿、残疾、长期护理、健康
- **遗产**：遗嘱、信托、受益人指定、赠与策略

### 步骤2：现金流分析

构建年度现金流预测：

| 年份 | 年龄 | 税前收入 | 税款 | 生活支出 | 储蓄 | 净现金流 |
|------|-----|---------|------|---------|------|----------|
| | | | | | | |

关键输入：
- 通胀率假设（通常2.5-3%）
- 税率（边际和实际）
- 储蓄率及储蓄去向（税前、Roth、可税）

### 步骤3：退休预测

**积累阶段：**
- 当前投资组合价值
- 年度贡献（401k、IRA、可税）
- 各资产类别预期回报
- 蒙特卡洛模拟：不同支出水平的成功概率

**分配阶段：**
- 退休所需年度支出（今天美元→经通胀调整）
- 社保开始年龄和福利
- 养老金收入（如有）
- 投资组合提取率和顺序
- 最低 Required Minimum Distributions（RMDs）

**关键输出：**
- 退休时预测投资组合价值
- 可持续提取率
- 不会耗尽资金的概率（目标>85%）
- "如果"情景：提前退休、市场下跌、更高的支出

### 步骤4：目标特定分析

#### 教育资金
- 孩子年龄和目标大学入学时间
- 当前529余额
- 目标资金水平（公立vs私立、4年vs研究生）
- 达到目标所需的月度储蓄
- 财务援助考虑

#### 遗产规划
- 当前遗产价值和预期增长
- 遗产税敞口（联邦和州）
- 现有信托结构
- 赠与策略（年度排除、终身豁免使用）
- 慈善捐赠计划
- 受益人审查

#### 风险管理
- 人寿保险需求分析（收入替代、债务偿还、教育资金）
- 残疾保险充足性
- 长期护理规划
- 超额责任险覆盖

### 步骤6：建议

优先行动项目：
1. 储蓄率变更
2. 资产配置调整
3. 税务优化（Roth转换、税损收割、资产位置）
4. 需要填补的保险缺口
5. 遗产文件更新
6. 受益人指定审查

### 步骤7：输出

- 财务计划文档（Word/PDF，15-25页）
- 现金流预测电子表格（Excel）
- 退休预测图表
- 目标资金分析
- 情景比较表
- 行动项目清单

## 重要说明

- 财务计划是动态文档——每年或重大生活事件后审查和更新
- 回报假设要保守——高估回报会给出虚假信心
- 税务规划与投资回报一样重要——模拟每项建议的税务影响
- 社保时间是主要杠杆——模拟62、67和70岁的开始年龄
- 始终压力测试计划——只在基准案例下可行的计划不是好计划
- 合规：确保建议符合适合性/受托人标准
