# 公司简介构建器

## 工作流程

### 1. 明确需求
- **询问用户**：单张幻灯片还是多张幻灯片（3-4张）？
- **询问用户**：是否有特定重点领域或主题需要强调？
- **仅在用户确认后**，进行研究

### 2. 研究与规划
**数据来源：**
- **主要**：公司文件（BamSEC、SEC EDGAR）、投资者演示、公司网站
- **市场数据**：Bloomberg、FactSet、CapIQ
- **估计**：FactSet/CapIQ共识

**必需指标：**
- **财务数据**：收入、EBITDA、利润率(%)、EPS、FCF（±3年）
- **估值**：市值、EV、EV/收入、EV/EBITDA、P/E倍数
- **增长**：同比增长率(%)
- **所有权**：前5大股东及持股比例
- **分部**：产品组合和/或地理组合(%明细)

**构建前：**
- 打印大纲到聊天，每项4-5个要点（实际数字，无占位符）
- 打印样式选择：字体、颜色、图表类型
- 获取用户确认

### 3. 逐页创建
**关键：必须一次创建一张幻灯片，在进入下一张前获得用户批准。**

**对每张幻灯片：**
1. 仅用PptxGenJS创建这一张幻灯片
2. **必须：转换为图像审阅**
3. **必须视觉审阅**：
   - 文本重叠检查
   - 文本截断检查
   - 图表边界检查
   - 象限完整性
4. **如检测到任何问题**：立即修复
5. 向用户展示幻灯片图像及下载链接
6. **停止并等待用户明确批准**

## 幻灯片格式要求

### 信息密度至关重要

**#1目标是最大化信息密度。** 忙碌的高管应该在30秒内理解整个公司故事。

**每个象限目标：**
- **公司概述**：最少6-8个要点
- **业务与定位**：6-8个要点
- **关键财务数据**：8-10行的表格或图表+4-5个关键指标
- **第四象限**：5-7个要点

**信息打包技巧：**
- 组合相关事实："总部：德州奥斯汀；成立于2003年；14万员工"
- 始终包含数字："$50B收入"而非"大额收入"
- 添加背景："EBITDA利润率：25%（vs 行业平均18%）"
- 包含同比变化："收入：$125M（+28% YoY）"

## 第一页布局

### 幻灯片设置（关键）
**使用4:3宽高比**（标准投行推介格式）：
```javascript
const pptx = new pptxgen();
pptx.layout = 'LAYOUT_4x3';  // 10"宽 × 7.5"高
```

### 第一页定位（英寸）
```
┌─────────────────────────────────────────────────────────────────┐
│ y=0.2  标题：公司名称（股票代码）                                │
├────────────────────────────┬────────────────────────────────────┤
│ y=0.6  公司概述            │ y=0.6  业务与定位                  │
│ x=0.3, w=4.7               │ x=5.0, w=4.7                       │
├────────────────────────────┼────────────────────────────────────┤
│ y=3.7  关键财务数据        │ y=3.7  股票/近期发展               │
│ x=0.3, w=4.7               │ x=5.0, w=4.7                       │
└────────────────────────────┴────────────────────────────────────┘
```

### 四象限布局

| 象限 | 位置 | 内容 |
|------|------|------|
| **1** | 左上 | **公司概述**：总部、成立、关键数据、业务摘要 |
| **2** | 右上 | **业务与定位**：收入驱动因素、产品/服务、竞争地位 |
| **3** | 左下 | **关键财务数据**：收入、EBITDA、利润率 + 估值 |
| **4** | 右下 | **上市公司**：股价图 + 前十大股东 |

### 字体大小
| 元素 | 大小 | 备注 |
|------|------|------|
| 幻灯片标题 | 24pt | 粗体，公司品牌颜色 |
| 象限标题 | 14pt | 粗体，带强调条 |
| 正文/要点文本 | 11pt | 常规字重 |
| 表格文本 | 10pt | 密集表格使用9pt |

### 视觉强调（必需）
每个象限标题左侧必须有彩色强调条：
```javascript
slide.addShape(pptx.shapes.RECTANGLE, {
  x: 0.3, y: 0.6, w: 0.08, h: 0.25,
  fill: { color: 'E31937' }  // 使用公司品牌颜色
});
```

## 图表

| 数据类型 | 图表类型 |
|----------|----------|
| 收入趋势 | 折线图或柱状图 |
| 地理明细 | 水平条形图 |
| 产品组合 | 带百分比的饼图 |
| 财务比较 | 柱状图 |
| 股价 | 折线图 |

## 质量清单

### 第一页
- [ ] 带公司名称、股票代码、行业的标题部分
- [ ] 标题下方恰好4个等大象限
- [ ] 全部要点，无段落，每项最多1行
- [ ] 财务数据在表格或图表中（非两者）

### 所有幻灯片
- [ ] 无文本溢出或截断
- [ ] 全文一致的字体和颜色
- [ ] 图表正确渲染
- [ ] 无占位符文本 - 全部实际数据
- [ ] 一致的规模（$mm或$bn，不混用）
- [ ] 引用来源
- [ ] 投行质量（GS/MS/JPM标准）
