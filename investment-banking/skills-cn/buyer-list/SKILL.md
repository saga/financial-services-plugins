# 买家名单构建

## 工作流程

### 第一步：了解标的公司

- 公司描述、行业和商业模式
- 收入、EBITDA 和增长概况
- 核心资产和能力（知识产权、客户关系、地理布局、团队）
- 预期估值范围
- 卖方偏好（战略买家 vs 财务投资者、管理层延续性、时间表）

### 第二步：战略买家

识别各类战略收购方：

**直接竞争对手**
- 同行业公司，可获取市场份额
- 理由：收入协同效应、消除竞争对手、规模效应

**相邻市场参与者**
- 相邻市场公司，可扩展至标的公司所在领域
- 理由：产品延伸、交叉销售、新市场进入

**垂直整合者**
- 可进行垂直整合的客户或供应商
- 理由：供应链控制、利润获取、战略锁定

**平台构建者**
- 通过并购在行业内构建平台的大型公司
- 理由：补强收购、填补能力缺口

对每个战略买家评估：

| 买家 | 行业 | 收入 | 战略匹配度 | 财务能力 | 并购记录 | 可能性 | 优先级 |
|------|------|------|-----------|----------|----------|--------|--------|
| | | | 高/中/低 | | 活跃/一般/无 | | A/B/C |

### 第三步：财务投资者

识别 PE/财务买家：

**平台投资者**
- 在该行业寻找新平台的基金
- 筛选标准：基金规模、行业聚焦、交易规模范围

**补强收购者**
- 拥有现有投资组合公司可将标的公司作为补强收购的基金
- 识别具体的投资组合公司和协同效应理由

**成长型股权**
- 针对早期或高增长标的公司
- 少数股权 vs 多数股权偏好

对每个基金：

| 基金 | 基金规模 | 行业聚焦 | 投资组合重叠 | 近期活动 | 优先级 |
|------|----------|----------|-------------|----------|--------|
| | | | | | A/B/C |

### 第四步：优先级排序

对买家名单分层：

- **第一梯队 (5-10家)**：最高战略匹配度、经验证的收购方、清晰的理由 — 优先联系
- **第二梯队 (10-15家)**：匹配度良好但不那么明显 — 第二轮联系
- **第三梯队 (10-20家)**：可能但概率较低 — 如需扩大范围再联系

### 第五步：联系人映射

对每个第一梯队买家：
- 关键决策者（CEO、企业发展负责人、合伙人）
- 关系状态（现有关系、冷启动、需要介绍）
- 已知偏好或限制（规模、地理、交易结构）
- 最佳接触渠道

### 第六步：输出

- Excel 工作簿包含：
  - 战略买家标签页（按梯队排序）
  - 财务投资者标签页（按梯队排序）
  - 第一梯队联系人映射
  - 汇总统计（按梯队和类型的买家总数）
- 一页买家名单摘要，用于委托函或推介材料

## 重要提示

- 质量优于数量 — 30-40 家经过充分研究的买家名单优于 200 家名单
- 研究近期并购活动 — 刚在该行业完成交易的买家要么渴望更多，要么已无力
- 检查直接竞争对手的反垄断问题 — 标记可能面临监管问题的买家
- 财务投资者：检查基金存续期和投资进度 — 接近投资期末的基金可能更有动力
- 始终询问卖方是否有希望纳入或排除的买家
- 随着交易进程更新名单 — 根据反馈在梯队间调整买家
