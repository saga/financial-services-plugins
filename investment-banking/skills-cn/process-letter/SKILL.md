# 流程函

## 工作流程

### 第一步：确定函件类型

- **初始流程函**：随 Teaser/CIM 发送，概述流程和 IOI 要求
- **IOI 说明**：首轮意向函的具体要求
- **第二轮/最终投标函**：尽调后提交约束性报价的说明
- **管理层会议邀请**：现场管理层演示的安排

### 第二步：初始流程函/IOI 说明

**抬头：**
- 日期、交易代号
- "保密"
- 致潜在买家

**内容：**

1. **引言**：机会和卖方目标的简要概述
2. **流程概述**：时间表、关键日期、预期轮次
3. **IOI 要求**：初步意向函应包含的内容：
   - 拟议估值范围（企业价值）
   - 对价形式（现金、股票、或有对价、股权滚存）
   - 融资来源和确定性
   - 主要尽职调查要求
   - 交割指示性时间表
   - 任何条件或前提
   - 买家简介和战略理由
4. **提交详情**：发送地点、截止日期（日期和时间）、格式
5. **保密提醒**：引用 NDA、数据室访问
6. **联系信息**：银行家联系方式

### 第三步：最终投标/第二轮函

IOI 之外的额外要求：

1. **购买协议修订**：提供 SPA/APA 草案并要求修订
2. **详细融资承诺**：需要承诺融资函
3. **剩余尽调事项**：说明预期的确认性尽调内容
4. **独家条款**：任何独家期的期限和条件
5. **监管分析**：反垄断申报要求和时间表
6. **关键人员条款**：雇佣协议、薪酬、股权滚存
7. **约束性 vs 非约束性**：明确此阶段的约束程度
8. **评估标准**：如何评估报价（价格、确定性、速度、匹配度）

### 第四步：管理层会议邀请

1. **安排**：日期、时间、地点（或视频链接）、时长
2. **参会者**：公司方演示人员、买方应参会人员
3. **议程**：典型管理层演示议程（概述、财务、运营、增长、问答）
4. **基本规则**：禁止录音、保密、问答格式
5. **材料**：将分发的材料（演示文稿、数据室访问）
6. **后续**：会后提交额外问题的流程

### 第五步：输出

- 专业函件格式的 Word 文档（.docx）
- 公司抬头纸占位符
- 供客户审核的修订模式版本

## 重要提示

- 流程函为整个交易定调 — 清晰、专业、有条理
- 截止日期应坚定但合理 — IOI 通常2-3周，最终报价3-4周
- 始终包含评估标准 — 买家想知道如何被评判
- 与法律顾问协调函件中的任何陈述或承诺
- 客户应在发送前审核批准 — 他们可能希望调整语调或条款
- 记录谁在何时收到每封函件 — 这成为流程跟踪器
