# Teaser 起草

## 工作流程

### 第一步：收集输入

- 公司描述（做什么、如何赚钱）
- 行业/领域
- 关键财务指标：收入、EBITDA、增长率、利润率
- 地理布局
- 核心卖点（3-5个亮点）
- 需要匿名 vs 可披露的内容
- 目标买家受众（战略、财务或两者）

### 第二步：Teaser 结构

一页，专业格式：

**抬头**
- 交易代号（如"项目 [名称]"）
- 行业描述（如"领先的专业工业服务平台"）
- "保密 — 仅供讨论之用"

**公司描述**（2-3句）
- 公司做什么，不点名
- 市场地位（如"领先供应商..."、"前三名玩家..."）
- 地理（区域级别，非具体城市）

**投资亮点**（4-6个要点）
- 市场领导地位/定位
- 收入质量（经常性%、留存、多元化）
- 增长概况和轨迹
- 利润率特征和扩张机会
- 管理团队实力
- 战略价值/协同潜力

**财务摘要**（表格或关键指标）

| 指标 | 价值 |
|------|------|
| 收入 | $XXM |
| 收入增长 | XX% CAGR |
| EBITDA | $XXM |
| EBITDA 利润率 | XX% |
| 员工 | XXX |

**交易概述**（2-3句）
- 出售内容（100%出售、多数股权、成长型股权）
- 指示性时间表
- 表达意向的联系信息

### 第三步：匿名检查

确保 Teaser 不会无意中识别公司：
- 无公司名称、品牌名称或产品名称
- 无具体城市（使用区域："美国东南部"、"中西部"）
- 无具名客户或合作伙伴
- 如果员工数量太独特则不要提及
- 如果行业较小，使用收入范围而非精确数字
- 无 Logo、截图或可识别的图像

### 第四步：输出

- Word 文档（.docx）— 一页，整洁格式
- 用于分发的 PDF 版本
- 可选 PowerPoint 版本（单页幻灯片）

## 重要提示

- Teaser 的作用是产生兴趣，而不是达成交易 — 保持简洁有力
- 少即是多 — 好的 Teaser 让买家想签 NDA 了解更多
- 使用有抱负但准确的语言 — "领先"、"差异化"、"高增长"如果属实可以使用
- 包含足够的财务细节以筛选认真买家，但不要太多让试探者浪费时间
- 分发前始终让客户和法律审核
- 跟踪谁收到 Teaser — 它成为流程外联日志
