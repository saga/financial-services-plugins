## 关键验证公式

### CAGR预测

**公式：**
```
终值 = 现值 × (1 + CAGR)^n
```

**变量：**
- 现值：当前/基准年市场规模
- CAGR：复合年增长率（小数形式，如16.4% = 0.164）
- n：基准年到目标年的年数

**验证示例：**
```
源数据声称：$22.1bn (2024) 按16.4% CAGR = $55.0bn (2030)

验证：22.1 × (1.164)^6 = 22.1 × 2.488 = 55.0 ✓
```

**计算n（年数）：** 计算基准年到目标年之间的年数。示例：2024→2030 = 6年，2025→2030 = 5年。

### 估值倍数

**EV/收入：**
```
EV/收入倍数 = 企业价值 ÷ 收入
隐含EV = 收入 × 倍数
```

**EV/EBITDA：**
```
EV/EBITDA倍数 = 企业价值 ÷ EBITDA
隐含EV = EBITDA × 倍数
```

**验证示例：**
```
源数据声称：$436m交易，收入$45m，倍数9.7x

验证：436 ÷ 45 = 9.69 ≈ 9.7x ✓
```

### 市场份额

**公式：**
```
市场份额 = (细分市场规模 ÷ 总市场规模) × 100
```

**验证示例：**
```
源数据声称：在线细分市场($18bn)占总市场($65bn)的28%

验证：18 ÷ 65 = 0.277 = 27.7% ≈ 28% ✓
```

### 增长率

**同比：**
```
同比增长 = (当年 - 上年) ÷ 上年 × 100
```

**从端点计算CAGR：**
```
CAGR = (终值 ÷ 起始值)^(1/n) - 1
```

## 共识方法论

当源数据包含多个估计值时，验证共识计算：

### 规模共识（范围）

**方法：** 所有来源的完整最小-最大范围

**示例：**
```
来源：$14.9bn, $18.3bn, $21.1bn, $21.2bn, $22.1bn
共识：$15-22bn（舍入到最近的$1bn）
```

### CAGR共识（中心集群）

**方法：** 排除异常值（最高和最低），使用中心集群范围

**示例：**
```
来源：10.6%, 16.4%, 17.2%, 19.0%, 22.7%
排除异常值：10.6%（低）, 22.7%（高）
中心集群：16.4%, 17.2%, 19.0%
共识：16-19% 或 16-17%（保守）
```

### 预测共识

**方法：** 将共识CAGR应用于规模范围中点

**示例：**
```
规模范围：$15-22bn → 中点：$18.5bn
CAGR共识：16-17%
按16%：18.5 × (1.16)^6 = $45.1bn
按17%：18.5 × (1.17)^6 = $47.5bn
共识预测：$45-48bn
```

## 舍入指南

这些是**典型惯例**——根据数值大小和模板风格调整：

| 数值类型 | 典型舍入 | 示例 |
|----------|----------|------|
| 大型市场规模（$10bn+） | 最近的$1bn | 18.47 → $18bn |
| 较小市场规模（<$10bn） | 最近的$0.5bn或$0.1bn | 2.3 → $2.5bn |
| 规模范围 | 匹配来源精度 | 14.9-22.1 → $15-22bn |
| CAGR | 整数%或0.5% | 16.4% → 16%或16.5% |
| 市场份额 | 最近的5%或匹配来源 | 27.7% → 25%或30% |
| 收入（$m） | 1位小数 | 18.47 → $18.5m |
| 倍数 | 1位小数 | 9.688 → 9.7x |

**舍入原则：**
- 舍入不应实质性改变数值——对于较小数值，使用更精细的精度
- 一致性比精度更重要——对相似数值使用相同舍入
- 创建范围时，低端向下舍入，高端向上舍入
- 对于汇总统计（均值、中位数），匹配输入数据的精度

## 验证检查清单

使用源数据中的任何计算值之前：

### 公式验证
- [ ] 预测使用正确的CAGR公式：`PV × (1 + r)^n`
- [ ] 倍数计算为EV ÷ 指标（而非反向）
- [ ] 增长率在分母中使用正确的基准年
- [ ] 百分比份额在适用处总和约为100%

### 输入验证
- [ ] 基准年数值匹配源文档
- [ ] CAGR/增长率匹配声明的来源方法论
- [ ] 时间段（n）计算正确
- [ ] 货币和单位一致（$bn vs $m）

### 输出验证
- [ ] 计算结果匹配来源声明的数值
- [ ] 如有差异，调查方法论差异
- [ ] 舍入应用一致
- [ ] 结果合理（无数量级错误）

### 共识验证
- [ ] 所有来源包含在范围计算中
- [ ] 异常值排除方法论已记录
- [ ] 中点计算使用正确的平均方法
- [ ] 范围边界代表实际最小/最大值或记录的子集

## 需调查的警示信号

**预测不匹配：**
- 计算预测与来源差异>5%
- 可能原因：不同基准年、不同CAGR或舍入

**倍数不匹配：**
- 计算倍数与来源不同
- 可能原因：不同指标定义（LTM vs NTM，收入 vs 净收入）

**共识不匹配：**
- 您的共识与来源共识不同
- 可能原因：来源排除了某些数据点，不同的异常值处理

**如有疑问：** 在脚注中注明差异并展示您的计算方法论。
