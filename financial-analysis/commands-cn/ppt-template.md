# PPT 模板创建命令

从用户提供的 PowerPoint 模板创建一个自包含的 PPT 模板技能。

## 操作指令

1. **获取模板文件**（若未提供）：提示用户提供路径。
2. **加载 ppt-template-creator-cn 技能**。
3. **收集信息**：模板名称、主要用途（如：路演、董事会汇报）。
4. **执行工作流**：分析版式和占位符，生成包含 `assets/` 和 `SKILL.md` 的技能目录。
5. **交付打包后的技能**给用户。
