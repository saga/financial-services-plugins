# 可比公司分析命令

构建专业级的可比公司分析，包含运营指标、估值倍数和统计基准分析。

## 工作流程

### 第一步：收集公司信息
如果提供了名称或代码，则直接使用。否则询问：
- "您想分析哪家公司？"

### 第二步：加载 Comps 分析技能
使用 `skill: "comps-analysis-cn"` 进行分析：
1. **明确分析目的**：受众是谁？核心关注估值、效率还是增长？
2. **确定同行组**：筛选商业模式、规模和行业相似的 4-6 家上市公司。
3. **收集数据**：优先使用 MCP 数据源。
4. **构建分析**：生成运营统计和估值倍数两个核心部分。

### 第三步：生成 Excel 输出
生成的 Excel 应包含标题块、运营指标、估值倍数、统计摘要及方法论注释。
