# DCF 估值命令

构建专业级的 DCF 模型，并使用可比公司分析来校准估值范围。

## 工作流程

1. **收集公司信息**：确定要估值的对象。
2. **运行可比公司分析**：加载 `comps-analysis-cn`，获取行业中位倍数（EV/EBITDA 等），用于指导 DCF 的终值假设。
3. **构建 DCF 模型**：加载 `dcf-model-cn` 构建现金流预测和 WACC 计算。
4. **交叉验证**：将 DCF 得出的隐含倍数与同行中位倍数进行对比校验。
5. **交付产出**：提供包含情景分析和敏感性表的 Excel 文件及摘要说明。
