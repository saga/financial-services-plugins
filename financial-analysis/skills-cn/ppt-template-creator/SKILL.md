# PPT 模板技能创建器

**本技能的作用是创建“技能”，而不是演示文稿。** 当用户希望将其 PPT 模板（.pptx 或 .potx）转换为一个以后可以自动生成演示文稿的“可重复使用技能”时，请使用本工具。

## 工作流

1. **接收用户模板** (.pptx 或 .potx)。
2. **分析模板**：提取版式 (Layouts)、占位符 (Placeholders) 和幻灯片尺寸。
3. **初始化技能**：使用 `skill-creator` 技能设置目录结构。
4. **添加资产**：将模板文件拷贝至 `assets/template.pptx`。
5. **编写 SKILL.md**：按照下方的模板编写详细的 PPT 操作指令。
6. **创建示例**：生成一个样例 PPT 以验证技能可用性。
7. **打包**：使用 `skill-creator` 技能打包为 `.skill` 文件。

## 关键分析点：确定真实内容起始位置

**重要：** 内容区域并不总是紧跟在副标题占位符之后。许多模板在副标题和正文之间留有视觉边框或装饰线空间。

**最佳实践：** 查找版式 2（或类似的“内容”版式）中的 **OBJECT** 类型占位符（type=7）。该占位符的 `y` 坐标才是内容应该开始的真实位置。

## 自动生成的 SKILL.md 模板结构

生成的技能应具有以下结构：
```
[公司名]-ppt-template/
├── SKILL.md
└── assets/
    └── template.pptx
```

### SKILL.md 内容要点：
1. **版式映射表**：列出索引、名称及其用途（如：[0] 封面，[1] 目录，[2] 纯文字内容）。
2. **占位符坐标**：记录每个版式中 TITLE 和 BODY 的精确 `idx` 以及 `x, y, w, h`（单位：英寸）。
3. **内容边界**：明确定义安全的内容区域（左边距、顶部起始 y、可用宽度和高度），以防止自定义形状/表格/图表遮挡页眉页脚。
4. **Python 代码示例**：
    - 如何删除模板中自带的幻灯片。
    - 如何通过 `idx` 填充标题和正文。
    - 如何利用 `paragraph.level` 设置多级缩进（不要手动添加项目符号）。

## 生成技能的 PPT 特定规则

1. **资产打包**：必须将 .pptx 文件放入 `assets/` 目录。
2. **自包含**：生成的 SKILL.md 必须包含所有操作指令，不应依赖本创建器。
3. **分级填充**：使用 `paragraph.level = 0` 表示标题，`level = 1` 表示子要点。
4. **先清后增**：在添加新幻灯片前，务必先清除模板文件中的现有幻灯片。
