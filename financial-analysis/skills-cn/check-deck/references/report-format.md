# 演示文稿检查报告格式

## 报告模板

```markdown
# 演示文稿检查报告：[文稿名称]

## 摘要
- 问题总数：X
- 严重 (Critical)：X (数据不一致、事实错误)
- 重要 (Important)：X (叙事对齐、语言规范)
- 次要 (Minor)：X (排版格式)

## 严重问题 (Critical)

### 数据一致性
1. **[问题名称]** (幻灯片 X, Y)
   - 幻灯片 X: [数值]
   - 幻灯片 Y: [数值]
   - 处理建议：[具体操作]

### 数据与叙事对齐
1. **[问题名称]** (幻灯片 X, Y)
   - 描述："[引述文字]"
   - 实际数据：[矛盾之处]
   - 处理建议：[具体操作]

## 重要问题 (Important)

### 语言规范
1. **[问题类型]** (幻灯片 X)
   - 当前内容："[引述文字]"
   - 建议替换："[建议内容]"

## 次要问题 (Minor)

### 排版格式
1. **[问题类型]** (幻灯片 X)
   - [描述与修复方法]

## 最终检查清单
- [ ] 数据已核对一致
- [ ] 叙事与图表数据匹配
- [ ] 语言符合投行标准
- [ ] 图表来源已标注
- [ ] 全文格式统一
```

## 问题严重程度分类

**严重 (Critical)**（交付客户前必须修复）：
- 幻灯片之间的数据冲突。
- 计算错误。
- 事实性不准确（名称、标题、日期错误）。
- 数据与文字结论相反。

**重要 (Important)**（应当修复）：
- 使用了非正式/口语化语言。
- 缺乏数据支撑的模糊断言。
- 术语使用不统一。
- 图表缺失数据来源。

**次要 (Minor)**（细节完善）：
- 字体/颜色不统一。
- 日期格式不一致。
- 间距或对齐微调。
- 孤行文本。
