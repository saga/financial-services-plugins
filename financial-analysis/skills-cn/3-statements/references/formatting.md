# 格式标准参考

| 元素 | 格式规范 |
|------|----------|
| 硬编码输入项 | 蓝色字体 |
| 公式计算项 | 黑色字体 |
| 跨表链接项 | 绿色字体 |
| 检查项单元格 | 报错为红色，平衡为绿色 |
| 负值表示 | 使用括号 `( )`，不使用负号 `-` |
| 货币单位 | 大额数字不留小数，每股金额保留2位小数 |
| 百分比 | 保留1位小数 |
| 标题 | 粗体，带下边框 |
| 单位行 | 在标题下方标注单位（如：百万元、%、x 等） |

## 视觉分隔准则

- 历史年度与预测年度列之间使用细垂直边框。
- 每一节合计（如：资产总计）下方使用粗底边框。
- 小计项使用单底边框。
- 总计项使用双底边框。

## 合计与小计行格式

所有合计和小计行的数值必须使用**粗体字**，以便清晰区分汇总数据与明细行项目。

### 利润表 (P&L)
| 行项目 | 格式 |
|--------|------|
| 营业总收入 | 粗体 |
| 总营业成本 | 粗体 |
| 毛利润 | 粗体 |
| 总销售及管理费用 | 粗体 |
| EBITDA / EBIT / 税前利润 | 粗体 |
| 税后净利润 | 粗体 |

### 资产负债表
| 行项目 | 格式 |
|--------|------|
| 流动资产/非流动资产总计 | 粗体 |
| 资产总计 | 粗体 |
| 流动负债/非流动负债总计 | 粗体 |
| 股东权益合计 | 粗体 |
| 负债及所有者权益总计 | 粗体 |

## 资产负债表平衡检查行

检查行（位于负债及所有者权益下方）使用条件格式：
- **= 0（平衡）**：标准黑色字体。
- **≠ 0（报错）**：显示为红色。

**实现方式**：使用自定义数字格式 `[Red][<>0]0.00;[Red][<>0](0.00);0.00` 或 Excel 条件格式。

## 信用指标阈值颜色

| 指标 | 绿色 (安全) | 黄色 (预警) | 红色 (风险) |
|------|-------------|-------------|-------------|
| 总债务 / EBITDA | < 2.5x | 2.5x-4.0x | > 4.0x |
| 净债务 / EBITDA | < 2.0x | 2.0x-3.5x | > 3.5x |
| 利息覆盖率 | > 4.0x | 2.5x-4.0x | < 2.5x |
| 债务 / 总资本 | < 40% | 40%-60% | > 60% |
| 流动比率 | > 1.5x | 1.0x-1.5x | < 1.0x |
