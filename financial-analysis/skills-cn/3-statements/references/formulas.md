# 公式参考指南

**重要提示：** 除非用户另有说明，否则请始终使用本参考文档中定义的公式。

---

## 核心勾稽关系

```
资产负债表：        资产 = 负债 + 股东权益
净利润流向：        利润表净利润 → 现金流量表经营活动（起始点）
现金流变动：        现金净增加额 = 经营活动 + 投资活动 + 融资活动
现金余额勾稽：      现金流量表期末现金 = 资产负债表货币资金
留存收益滚动：      期初留存 + 净利润 - 股利 = 期末留存
股权融资勾稽：      资产负债表股本/公积变化 = 现金流量表股权发行
```

## 盈利指标公式

**重要：** 毛利润必须基于“营业净收入”计算，而非总收入。

| 术语 | 定义 |
|------|------|
| 营业总收入 | 未扣除退货和折扣前的总额 |
| 营业净收入 | 总收入 - 退货 - 折让 - 折扣 |
| 营业成本 (COGS) | 与生产已售商品/服务直接相关的成本 |
| 毛利润 | 营业净收入 - 营业成本 |

```
毛利率 %          = 毛利润 / 营业净收入
EBITDA            = EBIT + 折旧与摊销 (D&A)
EBITDA 利润率 %   = EBITDA / 营业净收入
净利率 %          = 净利润 / 营业净收入
```

## 信用与杠杆指标

```
总债务            = 短期借款 + 长期借款（含一年内到期）
净债务            = 总债务 - 现金及现金等价物
总债务 / EBITDA   = 总债务 / LTM EBITDA
利息覆盖率        = EBITDA / 利息支出
资产负债率        = 总负债 / 总资产
净资产负债率      = 总债务 / 总净资产
流动比率          = 流动资产 / 流动负债
速动比率          = (流动资产 - 存货) / 流动负债
```

## 营运资本 (Working Capital)

```
应收账款 (AR)
  期初 AR
  + 营业收入
  - 现金回收 (Plug)
  = 期末 AR
  应收账款周转天数 (DSO) = (应收账款 / 营收) × 365

存货 (Inventory)
  期初存货
  + 本期采购 (Plug)
  - 营业成本 (COGS)
  = 期末存货
  存货周转天数 (DIO) = (存货 / 营业成本) × 365

净营运资本 (NWC) = 应收账款 + 存货 - 应付账款
营运资本变动 = 本期 NWC - 上期 NWC
```

## 净经营亏损 (NOL) 滚动

```
期初 NOL 余额 (新企业通常为 0)
+ 本期产生的 NOL (若 EBT < 0，则为绝对值，否则为 0)
- 本期利用的 NOL (受限于应税所得额及 80% 的抵扣上限)
= 期末 NOL 余额

所得税计算 (含 NOL)：
应交所得税 = MAX(0, (税前利润 - 已利用 NOL) × 税率)
注：亏损不会产生负税额，而是形成递延所得税资产 (DTA)。
```
