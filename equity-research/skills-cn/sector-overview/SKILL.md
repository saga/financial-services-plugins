# 板块概览

创建全面的行业和板块概览报告。

## 工作流程

### 步骤1：定义范围

- **板块/子板块**：什么行业，定义多窄？
- **目的**：客户报告、内部研究、推销材料、思路生成
- **深度**：高层概览（5-10页）或深度研究（20-30页）
- **角度**：中立格局vs.主题论点（例如，"AI基础设施构建"）
- **范围**：仅上市公司，或包括私有公司？

### 步骤2：市场概览

**市场规模与增长**
- 总可寻址市场（TAM）及来源
- 历史增长率（5年CAGR）
- 预测增长率和关键假设
- 市场细分（按产品、地区、终端市场、客户类型）

**行业结构**
- 分散vs.集中——前5名市场份额
- 价值链地图——价值积累在哪里？
- 商业模式类型（订阅、交易、许可、服务）
- 进入壁垒（资本、监管、技术、网络效应）

**关键趋势与驱动因素**
- 长期顺风（3-5个主要趋势）
- 逆风与风险
- 技术颠覆向量
- 监管发展
- 并购活动和整合趋势

### 步骤3：竞争格局

**公司概况**（前5-10名参与者）：

| 公司 | 收入 | 增长 | EBITDA利润率 | 市场份额 | 关键差异化因素 |
|------|------|------|-------------|---------|--------------|
| | | | | | |

每家公司简要概况：
- 业务描述（2-3句话）
- 战略定位和护城河
- 最新发展（业绩、并购、产品发布）
- 估值快照（P/E、EV/EBITDA、EV/收入）

**竞争动态**
- 公司如何竞争？（价格、产品、服务、分销）
- 谁在增加/失去份额，为什么？
- 新进入者或相邻参与者的颠覆风险

### 步骤4：估值背景

- 板块交易倍数（当前和历史区间）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 板块与更广泛市场比较如何？

### 步骤5：投资含义

- 最好的风险/回报机会在哪里？
- 什么主题赌注可以通过这个板块表达？
- 板块关键辩论（牛市vs.熊市论点）
- 可能改变板块叙事的催化剂

### 步骤6：输出

- Word文档或PowerPoint包含：
  - 市场概览和规模
  - 竞争格局图
  - 公司比较表
  - 估值摘要
  - 关键图表：市场增长、份额趋势、估值历史
- 详细公司数据的Excel附录

## 重要说明

- 来源所有市场规模数据——引用研究机构或方法论
- 区分TAM炒作和现实可寻址市场
- 板块概览老化很快——注明日期并标记可能过时的数据
- 图表必不可少——市场规模瀑布图、竞争定位矩阵、估值散点图
- 如为客户定制，根据他们的具体情况定制"所以呢"（并购目标识别、竞争定位、市场进入）
