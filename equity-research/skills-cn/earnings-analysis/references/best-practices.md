# 最佳实践、示例和质量指南

本文档提供示例、成功技巧、常见错误避免和全面的质量检查清单。

## 示例标题

### 好的业绩更新标题：
- "耐克 Q2 FY24：DTC优势抵消批发疲软 - 维持增持，目标价$95"
- "特斯拉 Q3'24：Cybertruck爬坡超预期 - 上调预期，目标价至$285"
- "LVMH Q4'24：时尚皮具稳健，酒类疲软 - 符合预期，重申买入"
- "苹果 Q1 FY24：服务超预期，iPhone不及预期 - 季度表现参差，下调目标价至$185"

### 不好的标题（避免）：
- "耐克季度更新"（太笼统，无结论）
- "公司发布业绩"（陈述显而易见，无分析）
- "Q3结果分析"（无公司名称，无观点）

## 成功技巧

1. **速度很重要**：业绩发布后24-48小时内发布，不要拖到几天后

2. **结论先行**：超预期还是不及预期？上调还是下调预期？

3. **量化一切**："强劲"毫无意义，"$1.2B收入超预期$150M"才清晰

4. **关注驱动因素**：不要只说"收入超预期"，解释为什么

5. **展示工作**：旧预期 → 新预期及原因

6. **重大变化时更新目标价**：如预期变化>5%，通常目标价也应变化

7. **承认电话会**：引用管理层评论，不要只分析新闻稿

8. **与同行比较**：如类似公司已发布业绩，注明相对表现

9. **保持简洁**：这不是全面报告，专注于季度结果

10. **绘制趋势图**：季度进展图表最有价值

## 常见错误避免

❌ **太全面**：不要为季度结果撰写首次覆盖长度的报告

❌ **缺失超预期/不及预期**：开头必须说明结果是否超预期或不及预期

❌ **不更新预期**：必须提供更新的前瞻预期

❌ **模糊语言**："表现强劲"没有量化

❌ **忽视指引**：如公司提供指引，需彻底分析

❌ **太慢**：业绩发布5天后再发布失去相关性

❌ **重复基础内容**：不要花3页解释公司做什么

❌ **缺失目标价更新**：如预期重大变化，目标价也应变化

❌ **无投资影响**：必须将结果与论点和评级联系起来

❌ **缺失引用**：每个数字都需要带可点击超链接的来源

❌ **纯文本URL**：所有URL必须格式化为可点击超链接

## 全面质量控制检查清单

发布业绩更新前，验证以下所有项目：

### 内容与分析检查清单

**超预期/不及预期分析：**
- [ ] 超预期/不及预期分析置于报告开头
- [ ] 具体差异已量化（如"超预期$120M或3%"）
- [ ] 解释结果为何与预期不同
- [ ] 分析每个关键指标（收入、EPS、利润率等）

**指标与表现：**
- [ ] 所有关键指标均带同比比较讨论
- [ ] 在相关处包含环比比较
- [ ] 提供分部/地区/产品细分
- [ ] 分析运营指标（客户数、ARPU、单位等）

**指引与预期：**
- [ ] 指引变化已分析和量化（如提供）
- [ ] 如无指引，明确注明
- [ ] 提供当年更新预期
- [ ] 提供明年更新预期
- [ ] 清晰展示旧预期与新预期
- [ ] 解释变化内容及原因

**估值与评级：**
- [ ] 目标价已更新（如结果需要）
- [ ] 如目标价不变，明确维持
- [ ] 估值方法已解释
- [ ] 评级已确认或更改并附明确理由
- [ ] 投资论点已评估并在需要时更新

### 格式与长度检查清单

**整体结构：**
- [ ] 报告为8-12页（不短也不长）
- [ ] 第1页有业绩摘要格式
- [ ] 第1页标题含"业绩更新"（非"首次覆盖"）
- [ ] 事件驱动标题（如"Q3业绩强劲..."）

**表格：**
- [ ] 包含1-3个摘要表（非全面表格）
- [ ] 所有表格有清晰列标题
- [ ] 所有表格有标题行底纹
- [ ] 所有表格底部有来源行
- [ ] 预期表显示旧预期与新预期及变化列

**图表：**
- [ ] 文档中嵌入8-12个图表
- [ ] 所有图表上方有"图X - [标题]"说明
- [ ] 所有图表下方有"来源：[来源]"行
- [ ] 图表专注于季度趋势
- [ ] 图表突出变化（超预期/不及预期、修正）
- [ ] 图表使用专业样式

### 引用与来源检查清单 ⭐⭐⭐ 必需

**图表与表格引用：**
- [ ] 每个图表有具体来源及文档名称和日期
- [ ] 每个表格有具体来源及文档参考
- [ ] 来源引用包含页码或幻灯片编号（如适用）

**超预期/不及预期引用：**
- [ ] 超预期/不及预期分析引用共识来源（Bloomberg、FactSet等）
- [ ] 共识来源包含"截至"日期（业绩发布前收盘）
- [ ] 公司报告结果引用业绩新闻稿或10-Q

**指引引用：**
- [ ] 当前指引引用业绩电话会记录或新闻稿
- [ ] 之前指引引用上一季度材料
- [ ] 当前和之前指引来源均已超链接

**统计与指标：**
- [ ] 关键统计有带来源的脚注
- [ ] 脚注引用具体文档和页码/幻灯片编号
- [ ] 管理层引语引用发言人姓名和来源文档

**超链接：** ⭐⭐⺀⭐ 关键
- [ ] 所有URL为可点击超链接（非纯文本）
- [ ] 超链接格式化为有意义的显示文本
- [ ] Word文档中超链接为蓝色下划线格式
- [ ] 超链接已测试可用（Ctrl+点击打开正确页面）
- [ ] 所有SEC文件超链接至EDGAR查看器
- [ ] 所有业绩材料已超链接（新闻稿、记录、演示）
- [ ] 上一季度材料已超链接供比较
- [ ] 文档中无任何原始URL显示

**来源部分：**
- [ ] 报告末尾包含"来源与参考"部分
- [ ] 部分列出所有业绩材料及日期
- [ ] 所有材料有可点击超链接
- [ ] 列出共识数据来源（即使订阅数据无链接）
- [ ] 包含前期参考

### 准确性检查清单

**数值准确性：**
- [ ] 数字与公司报告结果完全匹配
- [ ] 所有计算数学正确
- [ ] 预期变化计算正确
- [ ] 估值数学准确
- [ ] 图表与文字描述匹配

**事实准确性：**
- [ ] 股票代码无拼写错误
- [ ] 公司名称无拼写错误
- [ ] 日期为当前且准确
- [ ] 季度/年度引用正确
- [ ] 年份标注正确（A表示实际，E表示预期）

### 时效性检查清单

**发布时机：**
- [ ] 报告在业绩发布后24-48小时内发布
- [ ] 如超过48小时，注明为"延迟反应"
- [ ] ✅ **已验证所有数据来自最新季度（通过搜索近期业绩）**
- [ ] ✅ **未依赖知识截止日期 - 主动搜索当前数据**
- [ ] 共识预期为业绩发布前（非发布后）
- [ ] 无过时信息
- [ ] 业绩发布日期在过去1-3个月内（非6个月以上）

### 写作风格检查清单

**清晰与直接：**
- [ ] 数字先行（"收入增长15%至$1.2B"而非"收入强劲"）
- [ ] 使用"vs."而非"versus"
- [ ] 全文直接简洁
- [ ] 专注于新内容（非重复公司基础信息）
- [ ] 避免模糊语言（"表现强劲"无量化）

**专业标准：**
- [ ] 保持机构语气
- [ ] 全文术语一致
- [ ] 无非正式语言
- [ ] 正确的财务标注

## 发布前最终检查

发送报告给用户前进行此快速最终检查：

### 5分钟最终审核：
1. **第1页**：评级清晰？目标价已更新？关键要点有说服力？
2. **数字**：报告结果是否与公司新闻稿完全匹配？
3. **引用**：抽查3-4个图表/表格 - 都有带可点击超链接的来源？
4. **预期**：旧预期与新预期清晰展示？变化已解释？
5. **图表**：所有8-12个已嵌入？都已编号和说明？
6. **长度**：是否为8-12页（非6页，非15页）？
7. **超链接**：测试3-4个超链接 - Ctrl+点击是否可用？
8. **时效性**：是否在业绩发布后48小时内发布？

如所有项目通过检查，报告即可发布。

## 摘要发布格式

向用户发布完成报告时，提供此摘要：

```
[公司] Q[X] [年] 业绩更新完成

结果：[超预期 / 符合预期 / 不及预期]
- 收入：$X.XB（超预期/不及预期$XXM或X%）
- EPS：$X.XX（超预期/不及预期$X.XX）

关键要点：
■ [要点1]
■ [要点2]
■ [要点3]

更新预期：
- FY[年]E收入：$XX.XB（之前：$XX.XB，[+/-]X%）
- FY[年]E EPS：$X.XX（之前：$X.XX，[+/-]X%）

评级：[维持/上调/下调] [评级]
目标价：$XXX（之前：$XXX）- [+/-]XX%上涨空间

交付物：
✓ 8-12页业绩更新报告（DOCX）
✓ 8-12个嵌入图表
✓ 更新预期及旧/新比较
✓ 完整来源部分及可点击超链接
✓ [可选：更新XLS财务模型]

文件：[公司]_Q[X]_[年]_业绩更新.docx
```
