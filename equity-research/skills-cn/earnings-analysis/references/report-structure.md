# 报告结构与模板

本文档提供业绩更新DOCX报告的完整逐页模板和格式要求。

## 完整报告结构

**报告结构：**

---

## 第1页：业绩摘要

**顶部区域 - 标题：**
```
[公司名称]（[股票代码]）
[季度] [年] 业绩更新

[当前日期]

评级：[维持/上调/下调] [评级]
股价（截至[日期]）：$XX.XX
目标价：[旧 → 新如变化，或维持$XXX]
```

**顶部区域 - 快速摘要框：**
```
业绩摘要
─────────────────────────────────────────────────
Q[X] [年]结果：[超预期 / 符合预期 / 不及预期]

                报告值    预期值    差异
收入            $X,XXX    $X,XXX   +$XXX (+X%)
EPS（调整后）   $X.XX     $X.XX    +$X.XX (+X%)

关键要点：
■ [要点1 - 一句话]
■ [要点2 - 一句话]
■ [要点3 - 一句话]
```

**主要内容 - 投资影响（3-4个要点）：**

使用■字符配合**粗体标题**和段落长度解释：

```
■ **结果因[分部/地区/产品]强劲而超预期，维持积极势头**

Q[X]收入$X.XB超出我们$X.XB预期X%，超出共识X%，主要受[具体驱动因素]推动。
[分部]收入同比增长X%（vs.我们X%预期），而[分部]增长X%（vs. X%预期）。
管理层强调[具体产品/举措]为关键增长动力，并对前景保持信心。
超预期展示了[论点要点]，强化了我们的正面观点。

■ **利润率同比扩张XX个基点，尽管存在[不利因素]，展示运营杠杆**

[详细利润率分析段落...]

■ **指引上调/维持/下调 - 意味着[解读]**

[详细指引分析段落...]

■ **维持[评级]，目标价[上调/不变]$XXX**

[投资结论段落...]
```

**底部区域 - 更新预期表：**

```
更新财务预期
─────────────────────────────────────────────────────────────────
                     FY2024E（旧）  FY2024E（新）  变化  FY2025E（新）
收入（$M）           XX,XXX         XX,XXX         +X%   XX,XXX
收入增长（%）        X.X%           X.X%           +XXbp X.X%
毛利率（%）         XX.X%          XX.X%          +XXbp XX.X%
EBITDA（$M）        X,XXX          X,XXX          +X%   X,XXX
EBITDA利润率（%）   XX.X%          XX.X%          +XXbp XX.X%
EPS（调整后）（$）  X.XX           X.XX           +X%   X.XX
P/E（倍）           XX.Xx          XX.Xx          -X%   XX.Xx

注："E" = 预期。旧预期来自[之前报告日期]。
来源：公司数据，[机构名称]预期。
```

## 第2-3页：详细结果分析

按以下方式分解结果：

### 收入分析（1页）
- 总收入超预期/不及预期解释
- 分部/地区/产品细分
- 同比和环比趋势
- 与指引比较（如提供）

**表格：季度收入进展**
```
                        Q[X-3]  Q[X-2]  Q[X-1]  Q[X]    同比变化  环比变化
总收入（$M）            X,XXX   X,XXX   X,XXX   X,XXX   +X%      +X%
  [分部A]（$M）         XXX     XXX     XXX     XXX     +X%      +X%
  [分部B]（$M）         XXX     XXX     XXX     XXX     +X%      +X%
  [分部C]（$M）         XXX     XXX     XXX     XXX     +X%      +X%

注：Q[X] = [季度] [年]
来源：公司报告，[机构名称]分析
```

### 盈利能力分析（1页）
- 毛利率分析（驱动因素、趋势）
- 营业利润率分析
- 线下项目（利息、税等）
- EPS调节（调整后vs. GAAP）

```

- [负面驱动因素1]
- [负面驱动因素2]

```

## 格式要求

### 1. 第1页要求
- 清晰评级（维持跑赢大盘、上调至买入等）
- 更新目标价显著展示
- 摘要表带旧/新预期
- 3-4个段落长度要点带■字符

### 2. 所有表格要求
- 底部有来源行
- 清晰列标题
- 标题行底纹

### 3. 所有图表要求
- 上方有"图X - [标题]"说明
- 下方有"来源：[来源]"行
- 专业样式

### 4. 年份标注
- 使用A表示实际（Q3'24A）
- 使用E表示预期（Q4'24E）

### 5. 写作风格
- 数字先行（"收入增长15%至$1.2B"而非"收入强劲增长"）
- 使用"vs."而非"versus"
- 直接简洁
- 专注于新内容

### 6. 超链接要求 ⭐⭐⭐
- 所有URL必须是Word中可点击超链接
- 蓝色下划线文本，Ctrl+点击可打开
- 显示文本有意义（非原始URL）
- 每个来源引用应有可点击链接（如适用）
- 无纯文本URL - 始终格式化为超链接
