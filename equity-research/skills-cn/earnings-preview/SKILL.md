# 业绩预览

在季度业绩发布前构建分析框架。

## 工作流程

### 步骤1：收集背景信息

- 识别公司和报告季度
- 通过网络搜索获取共识预期（收入、EPS、关键分部指标）
- 查找业绩日期和时间（盘前vs盘后）
- 审阅公司上季度业绩电话会以了解任何指引或评论

### 步骤2：关键指标框架

构建公司特定的"关注要点"框架：

**财务指标：**
- 收入vs.共识（总计和按分部）
- EPS vs.共识
- 利润率（毛利润、营业利润、净利润）——扩张还是收缩？
- 自由现金流
- 前瞻指引vs.共识

**运营指标**（按行业）：
- 科技/SaaS：ARR、净留存率、RPO、客户数量
- 零售：同店销售、流量、客单价
- 工业：订单积压、book-to-bill、价格vs.量
- 金融：NIM、信贷质量、贷款增长、费用收入
- 医疗：处方量、患者人数、管线更新

### 步骤3：情景分析

构建3个情景及股价影响：

| 情景 | 收入 | EPS | 关键驱动因素 | 股价反应 |
|------|------|-----|------------|---------|
| 牛市 | | | | |
| 基准 | | | | |
| 熊市 | | | | |

每个情景：
- 需要在运营上发生什么
- 管理层评论将发出什么信号
- 历史背景——类似业绩发布时股价如何表现？

### 步骤4：催化剂清单

识别将决定股价反应的3-5个事项：

1. [指标] vs. [共识/耳语数字]——为什么重要
2. [指引项目]——买方期望听到什么
3. [叙事转变]——任何战略变化、并购、重组

### 步骤5：输出

单页业绩预览包含：
- 公司、季度、业绩日期
- 共识预期表
- 关键观察指标（按重要性排序）
- 牛/基准/熊情景表
- 催化剂清单
- 交易设置：近期股价表现、期权隐含波动

## 重要说明

- 共识预期会变动——始终注明预期来源和日期
- 买方调查的"耳语数字"通常比发布的共识更相关
- 历史业绩反应有助于校准预期（搜索"[公司]业绩反应历史"）
- 期权隐含波动告诉市场预期什么——与你的情景比较
