# 思路生成

系统化的股票筛选和投资思路来源。

## 工作流程

### 步骤1：定义筛选标准

询问用户参数：
- **方向**：多头思路、空头思路或两者
- **市值**：大盘、中盘、小盘、微盘
- **行业**：特定行业或跨行业
- **风格**：价值、成长、质量、特殊情形、事件驱动
- **地区**：美国、国际、全球
- **主题**：任何特定主题角度（AI、回流、人口老龄化等）

### 步骤2：定量筛选

按风格运行筛选：

**价值筛选**
- P/E低于行业中位数
- EV/EBITDA低于历史平均水平
- 自由现金流收益率>5%
- 市净率<1.5x
- 过去90天内部人士买入
- 股息收益率高于市场平均

**成长筛选**
- 收入增长>15%同比
- 盈利增长>20%同比
- 收入加速（增长率上升）
- 利润率扩张
- 投资资本回报率>15%（高）
- 净留存率>110%（SaaS）

**质量筛选**
- 收入增长稳定（5+年）
- 利润率稳定或扩张
- ROE>15%
- 负债/权益低
- 自由现金流转换率高
- 内部人士持股>5%

**空头筛选**
- 收入下降或增长减速
- 利润率压缩
- 应收账款/库存vs.销售上升
- 内部人士卖出
- 估值相对同行溢价无合理解释
- 高做空兴趣但基本面恶化
- 会计红旗（审计师变更、重述）

**特殊情形筛选**
- 最近IPO/SPAC有禁售期到期
- 过去12个月的分拆
- 正在走出重组的公司
- 激进投资者参与
- 表现不佳公司的管理层变动

### 步骤3：主题扫描

对于主题思路，研究主题并识别受益者：

1. 定义论点（例如，"AI基础设施支出加速至2026年"）
2. 映射价值链——谁直接受益，谁间接受益？
3. 识别纯业务vs.多元化敞口
4. 评估哪些名字已经"计入价格"vs.未被充分认识
5. 寻找市场尚未连接到主题的二级受益者

### 步骤4：思路呈现

对于通过筛选的每个思路，呈现：

**[公司名称]** — **[多头/空头]** — **[一句话论点]**

| 指标 | 值 | vs.同行 |
|------|-----|---------|
| 市值 | | |
| EV/EBITDA (NTM) | | |
| P/E (NTM) | | |
| 收入增长 | | |
| EBITDA利润率 | | |
| FCF收益率 | | |

**论点（3-5点）：**
- 为什么定价错误
- 市场遗漏了什么
- 实现价值的催化剂

**关键风险：**
- 什么会证明这是错的

**建议后续步骤：**
- 构建完整模型？深度尽职调查？专家电话？

### 步骤5：输出

- 5-10个思路的短清单及各一页摘要
- 筛选标准和方法论文档
- 所有思路的比较表
- 优先列表：哪些思路优先研究

## 重要说明

- 筛选产生候选，非结论——每个筛选输出都需要基本面工作
- 最好的思路往往来自交叉点（例如，质量公司在暂时性不利因素下以价值价格交易）
- 避免拥挤交易——检查所有权数据、做空兴趣和多少分析师覆盖
- 逆向思路需要催化剂——没有催化剂的早期与错误相同
- 随时间跟踪思路命中率——哪些筛选和方法论产生最好的思路？
- 空头思路需要更高信念——时机更难且风险不对称
