# 模型更新

使用新数据更新财务模型。

## 工作流程

### 步骤1：识别变化内容

确定更新触发因素：
- **业绩发布**：需要填入的新季度实际数据
- **指引变化**：公司更新了前瞻展望
- **预期修订**：分析师基于新数据更改假设
- **宏观更新**：利率、外汇、大宗商品价格变化
- **事件驱动**：并购、重组、新产品、管理层变动

### 步骤2：填入新数据

#### 业绩发布后
用报告的实际数据更新模型：

| 行项目 | 原预期 | 实际 | 差异 | 备注 |
|--------|--------|------|------|------|
| 收入 | | | | |
| 毛利率 | | | | |
| 营业费用 | | | | |
| EBITDA | | | | |
| EPS | | | | |
| [关键指标1] | | | | |
| [关键指标2] | | | | |

**分部详情**（如适用）：
- 更新每个分部的收入和利润率
- 标记任何分部组合变化

**资产负债表/现金流更新：**
- 现金和债务余额
- 流通股数（回购、稀释）
- 资本支出实际vs.预期
- 营运资本变化

### 步骤3：修订前瞻预期

基于新数据，调整前瞻预期：

| | 原FY预期 | 新FY预期 | 变化 | 原下一FY | 新下一FY | 变化 |
|---|---------|---------|------|---------|---------|------|
| 收入 | | | | | | |
| EBITDA | | | | | | |
| EPS | | | | | | |

**关键假设变化：**
- 你在改变什么假设，为什么？
- 收入增长率：旧→新（原因）
- 利润率假设：旧→新（原因）
- 任何新项目（重组费用、一次性收益等）

### 步骤4：估值影响

用更新后的预期重新计算估值：

| 估值方法 | 原值 | 更新后 | 变化 |
|----------|------|--------|------|
| DCF公允价值 | | | |
| P/E (NTM EPS ×目标倍数) | | | |
| EV/EBITDA (NTM EBITDA ×目标倍数) | | | |
| **目标价** | | | |

### 步骤5：摘要与行动

**预期变化摘要：**
- 一段话：什么变了，为什么，对股票意味着什么？
- 这是改变论点的事件还是噪音？

**评级/目标价：**
- 维持或改变评级？
- 新目标价（如变化）及方法论
- 相对当前价格的上涨/下跌空间

### 步骤6：输出

- 更新的Excel模型（如用户提供现有模型）
- 预期变化摘要（markdown或Word）
- 更新后的目标价推导

## 重要说明

- 在向前预测之前，始终将你的预期与公司报告的数字对账
- 标记任何非经常性项目及你的预期是GAAP还是调整后
- 跟踪你的预期修订历史——显示你的分析进展
- 如果季度嘈杂，在预期变化中分离信号和噪音
- 更新后检查共识——你修订的预期与市场预期相比如何？
- 流通股数很重要——股票薪酬、可转换债券或回购的稀释可能严重影响EPS
