# 晨会笔记

撰写简洁的晨会笔记。

## 工作流程

### 步骤1：隔夜动态

扫描覆盖股票池的相关事件：

**业绩与指引**
- 有覆盖公司隔夜或盘前发布业绩？
- 业绩超预期/不及预期（收入、EPS、关键指标）
- 指引变化（上调、下调、维持）

**新闻与事件**
- 并购公告或谣言
- 管理层变动
- 产品发布或监管决策
- 竞争对手分析师上调/下调评级
- 影响行业的宏观数据或政策变化

**市场背景**
- 隔夜期货/盘前走势
- 行业ETF表现
- 相关大宗商品或货币变动
- 今天发布的关键经济数据

### 步骤2：晨会笔记格式

保持紧凑——晨会笔记应在2分钟内可阅读：

---

**[日期]晨会笔记 — [分析师姓名]**
**[行业覆盖]**

**首要通话：[标题——基金经理需要听的一件事]**
- 关键发展及为什么重要的2-3句话
- 股票影响：目标价、评级重申/变更

**隔夜/盘前动态**
- [公司A]：一句话总结业绩/新闻+我们的观点
- [公司B]：一句话总结+我们的观点
- [行业/宏观]：相关行业范围发展

**今天关键事件**
- [时间]：[公司]业绩电话会
- [时间]：[经济数据发布]（预期vs.我们的观点）
- [时间]：会议或投资者日

**交易思路**（如有）
- [多头/空头][公司]：1-2句话论点+催化剂
- 风险：什么会证明这是错的

---

### 步骤3：业绩速评

如果覆盖公司发布了业绩，提供快速反应：

| 指标 | 共识 | 实际 | 超预期/不及预期 |
|------|------|------|---------------|
| 收入 | | | |
| EPS | | | |
| [关键指标] | | | |
| 指引 | | | |

**我们的观点**：2-3句话——这对该股票是好还是坏？是否改变我们的论点？

**行动**：维持/上调/下调评级？调整目标价？

### 步骤4：输出

- 用于邮件/Slack分发的Markdown文本
- 如需正式分发则用Word文档
- 最多1页——基金经理和交易员不会阅读更多

## 重要说明

- 要有观点——只总结新闻而没有观点的晨会笔记毫无价值
- 以最重要的事开头——不要埋没标题
- "没有新闻"是有效的晨会笔记——说"隔夜没有实质性内容，维持头寸"
- 区分可操作事件（业绩、并购）和噪音（小分析师笔记、非事件）
- 为你的观点加上时间戳——如果你在早上6点写，注明盘前可能在开盘前变化
- 如果你错了，在下一封晨会笔记中承认——信誉比每次都正确更重要
