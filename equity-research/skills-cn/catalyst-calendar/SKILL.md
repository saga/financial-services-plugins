# 催化剂日历

构建和维护覆盖股票池的即将到来的催化剂日历。

## 工作流程

### 步骤1：定义覆盖范围

- 需要跟踪的公司列表（股票代码或名称）
- 行业/产业关注点
- 是否包括宏观事件？（美联储会议、经济数据、监管截止日期）
- 时间范围（未来2周、一个月、一个季度）

### 步骤2：收集催化剂

为每家公司识别即将到来的事件：

**业绩与财务事件**
- 季度业绩日期和时间（盘前/盘后）
- 年度股东大会
- 投资者日/分析师日
- 资本市场日
- 债务到期/再融资日期

**公司事件**
- 产品发布或公告
- FDA批准/监管决策
- 合同续签或到期
- 并购里程碑（完成日期、监管批准）
- 管理层变动
- 内部人士交易窗口期（禁售期到期）

**行业事件**
- 主要会议（日期、哪些公司发言）
- 贸易展览和展会
- 监管评议期或裁决
- 行业数据发布（月度销售、流量等）

**宏观事件**
- 美联储会议（FOMC日期）
- 就业报告、CPI、GDP发布
- 央行决策（欧洲央行、日本央行等）
- 具有市场影响的地缘政治事件

### 步骤3：日历视图

| 日期 | 事件 | 公司/行业 | 类型 | 影响程度（H/M/L） | 我们的头寸 | 备注 |
|------|------|----------|------|-------------------|-----------|------|
| | | | 业绩/公司/行业/宏观 | | 多头/空头/中性 | |

### 步骤4：每周预览

每周生成前瞻性摘要：

**本周关键事件：**
1. [日期]：[公司] Q[X]业绩——共识[$X EPS]，我们的预期[$X]，关键关注点：[指标]
2. [日期]：[事件]——为什么这对[股票]重要
3. [日期]：[宏观发布]——预期和头寸

**下周预览：**
- 提前通知即将到来的重要事件

**头寸影响：**
- 可能影响特定头寸的事件
- 建议的任何预先头寸安排
- 重大事件前的风险管理

### 步骤5：输出

- 带有日历视图和可排序列的Excel工作簿
- 每周预览邮件/笔记（markdown格式）
- 可选：与Google日历集成

## 重要说明

- 业绩日期会变动——临近日期时验证公司IR页面和Bloomberg/FactSet
- 预公告风险：跟踪有预公告历史的公司（正面或负面）
- 会议出席名单很有价值——哪些公司在发言，哪些明显缺席？
- 一些催化剂是重复的（月度行业数据）——建立模板并自动填充
- 按影响程度着色：红色=高影响，黄色=中等，绿色=常规
- 存档过去的催化剂及实际结果——建立模式识别
