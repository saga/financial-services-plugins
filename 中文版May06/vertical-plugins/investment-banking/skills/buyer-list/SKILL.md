# 买家名单 (Buyer List)

## 工作流

### 步骤 1：了解目标公司 (Target)

- 公司描述、业务部门和商业模式
- 收入、EBITDA 和增长情况
- 核心资产和能力（知识产权、客户关系、地理布局、团队）
- 预期估值范围
- 卖方偏好（战略买家 vs. 财务买家、管理层留任要求、交易时间表）

### 步骤 2：战略买家 (Strategic Buyers)

按类别识别战略收购方：

**直接竞争对手**
- 同行业的公司，收购后可获得市场份额
- 逻辑：收入协同效应、消除竞争、规模效应

**相邻领域玩家**
- 相邻市场的公司，可通过收购目标公司扩张版图
- 逻辑：产品线延伸、交叉销售、进入新市场

**垂直整合者**
- 可能进行垂直整合的客户或供应商
- 逻辑：供应链控制、利润留存、战略锁定

**平台构建者**
- 通过并购在行业内构建大型平台的公司
- 逻辑：补强型收购 (Tuck-in)、填补能力缺口

针对每个战略买家，进行如下评估：

| 买家名称 | 行业部门 | 收入规模 | 战略匹配度 | 财务实力 | 并购历史记录 | 成交可能性 | 优先级 |
|-------|--------|---------|--------------|-------------------|------------------|------------|----------|
| | | | 高/中/低 | | 活跃/一般/无 | | A/B/C |

### 步骤 3：财务赞助人 (Financial Sponsors)

识别私募股权 (PE) / 财务买家：

**平台投资者**
- 正在该领域寻找新平台型项目的赞助人
- 标准：基金规模、行业关注点、交易规模区间

**附加收购方 (Add-on Buyers)**
- 旗下已有投资组合公司可将目标公司作为“补强”收购的赞助人
- 识别具体的组合公司及其协同效应逻辑

**成长股权 (Growth Equity)**
- 针对早期或高增长的目标公司
- 少数股权 vs. 多数股权偏好

针对每个赞助人：

| 赞助人名称 | 基金规模 | 行业专注点 | 投资组合重叠度 | 近期活跃度 | 优先级 |
|---------|-----------|-------------|-------------------|-----------------|----------|
| | | | | | A/B/C |

### 步骤 4：优先级排序 (Prioritization)

将买家名单分层：

- **第一梯队 Tier 1 (5-10家)**：战略匹配度最高、有记录的活跃收购方、逻辑清晰——首批接触
- **第二梯队 Tier 2 (10-15家)**：匹配度良好但非显而易见——第二波接触
- **第三梯队 Tier 3 (10-20家)**：有可能但概率较低——若需扩大流程范围时接触

### 步骤 5：联系人映射

针对每个第一梯队买家：
- 核心决策者（CEO、企业发展部负责人、合伙人）
- 关系状态（现有关系、冷接触、需要引荐）
- 已知的偏好或限制（规模、地理位置、交易结构）
- 最佳接触渠道

### 步骤 6：产出物

- Excel 工作簿，包含：
  - 战略买家标签页（按梯队排序）
  - 财务赞助人标签页（按梯队排序）
  - 第一梯队联系人映射
  - 汇总统计（按梯队、按类型划分的买家总数）
- 卖方委托书或 Pitch 用的单页买家宇宙汇总

---

## 重要注意事项

- **质大于量**：一份包含 30-40 个经过深入研究的买家名单，远胜于一份 200 个名字的罗列。
- **研究近期并购动态**：刚在同行业完成交易的买家要么“饿”求更多，要么已经“吃饱”短期内不再出手。
- **反垄断检查**：针对直接竞争对手，应标记任何可能面临监管问题的买家。
- **财务赞助人**：检查基金的年份和投资节奏——接近投资期末尾的基金可能更有动力。
- **咨询卖方**：始终询问卖方是否有他们希望包含或排除的买家。
- **动态更新**：随着流程推进，根据反馈在不同梯队间移动买家。

### 【金融知识小贴士】—— 给小白的 Appendix

*   **Strategic Buyer (战略买家)**：通常是同行业的公司。他们买你是因为你的业务能跟他的业务合在一起，产生“1+1>2”的效应（协同效应）。
*   **Financial Sponsor/PE (财务赞助人/私募股权)**：这些是由专业投资人管理的基金。他们买你通常是为了通过杠杆和经营改善，几年后以更高价格卖掉赚取差价。
*   **Sell-side Mandate (卖方委托)**：投行接受公司老板的委托，帮他把公司卖个好价钱。
*   **Tuck-in Acquisition (补强式收购)**：就像大乐高模型缺了一块，大公司买下一个小公司来填补某个特定技术或市场的空白。
*   **Fund Vintage (基金年份)**：PE 基金有寿命。如果一个基金快到期了还没把钱投出去，他们会非常急于成交。
