# 行业概览 (Sector Overview)

**描述**：撰写涵盖市场动态、竞争定位、关键参与者和主题趋势的综合性行业与板块版图报告。适用于客户请求、行业首覆盖（Initiation）、主题研究或内部知识积累。触发词包括：“行业概览”、“行业报告”、“市场版图”、“行业分析”、“行业深度调研”或“主题研究”。

## 工作流

### 第 1 步：定义范围

- **板块 / 细分板块**：研究什么行业？定义出的范围有多窄？
- **目的**：客户报告、内部研究、推介资料（Pitch）、投资逻辑生成。
- **深度**：高层级概览（5-10 页）或深度钻研（20-30 页）。
- **视角**：中立的版图梳理 vs. 主题性投资逻辑（例如：“AI 基础设施建设”）。
- **研究宇宙 (Universe)**：仅限上市公司，还是包含私有公司？

### 第 2 步：市场概览

**市场规模与增长**
- 获取总可寻址市场 (TAM) 及其数据来源。
- 历史增长率（5 年年复合增长率 CAGR）。
- 预测增长率及关键假设。
- 市场细分（按产品、地域、终端市场、客户类型）。

**行业结构**
- 分散型 vs. 集中型 —— 前 5 名的市场份额。
- 价值链图谱 —— 利润在哪个环节累积？
- 业务模式类型（订阅、交易、许可、服务）。
- 准入门槛（资本、监管、技术、网络效应）。

**关键趋势与驱动因素**
- 长期增长动力（Secular tailwinds，3-5 个主要趋势）。
- 阻力与风险。
- 技术颠覆矢量。
- 监管动态。
- 并购 (M&A) 活动与行业整合趋势。

### 第 3 步：竞争格局

**公司简介**（针对前 5-10 名参与者）：

| 公司名称 | 收入 | 增速 | EBITDA 利润率 | 市场份额 | 核心差异化优势 |
|---------|---------|--------|--------------|-------------|-------------------|
| | | | | | |

对每家公司进行简要说明：
- 业务描述（2-3 句话）。
- 战略定位与护城河。
- 近期动态（财报、并购、产品发布）。
- 估值快照（P/E, EV/EBITDA, EV/Revenue）。

**竞争动态**
- 公司如何竞争？（价格、产品、服务、分发途径）。
- 谁在获得/失去份额，为什么？
- 来自新进入者或相邻领域参与者的颠覆风险。

### 第 4 步：估值背景

- 行业交易倍数（当前及历史区间）。
- 溢价/折价驱动因素（增速、利润率、市场地位）。
- 近期并购交易倍数。
- 行业与大盘的对比情况。

### 第 5 步：投资启示

- 哪里是风险收益比（Risk/Reward）最佳的机会？
- 该行业可以体现哪些主题性投注逻辑？
- 行业内的核心争议（多空辩论 Bull vs. Bear）。
- 可能改变行业叙事逻辑的催化剂。

### 第 6 步：产出

- Word 文档或 PowerPoint，包含：
  - 市场概览与规模测算。
  - 竞争格局图谱。
  - 公司对比表。
  - 估值摘要。
  - 关键图表：市场增长、份额趋势、估值历史。
- Excel 附录，包含详细的公司数据。

## 注意事项

- 标记所有市场规模数据的来源 —— 引用研究机构或测算方法。
- 区分高大上的 TAM 概念与现实可触达的有效市场（Addressable Market）。
- 行业概览时效性极强 —— 标明日期并标记可能过时的数据。
- 图表必不可少 —— 市场规模瀑布图、竞争定位矩阵、估值散点图。
- 如果面向客户，需根据其具体情况定制“结论说明”（例如：并购目标识别、竞争定位、市场进入）。

### 🏮 金融小白附录：重点词汇详解

1. **Value Chain (价值链)**：钱在谁手里过了一道。比如手机行业，从挖矿的、做芯片的、组装的到卖手机的。看价值链就是为了看谁拿走了大头的利润，谁在干最苦最累的活。
2. **TAM (Total Addressable Market, 总可寻址市场)**：金融圈的“画大饼”。假设全世界的人都买你的产品，这买卖能做多大。
3. **CAGR (Compound Annual Growth Rate, 年复合增长率)**：衡量增长的“平均分”。不是看今年涨了多少，而是看过去几年平均每年稳步涨了多少。
4. **Moat (护城河)**：你的“必杀技”。别人抄不走、打不垮的竞争优势（比如苹果的生态系统或者可口可乐的配方）。
5. **Tailwinds / Headwinds (顺风/逆风)**：顺风就是行业的大趋势在推着你走（比如新能源车）；逆风就是环境在阻碍你（比如利率上涨导致买房的人少了）。
