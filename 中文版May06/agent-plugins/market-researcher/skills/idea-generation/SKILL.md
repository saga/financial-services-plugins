# 投资逻辑生成 (Idea Generation)

**描述**：系统化的股票筛选与投资逻辑挖掘。结合定量筛选、主题研究和模式识别，挖掘新的多头（Long）和空头（Short）投资机会。适用于寻找新项目、运行筛选器或进行行业扫街场景。触发词包括：“投资逻辑生成”、“股票筛选”、“找项目”、“有什么有趣的标的”、“新点子”或“给我推介个项目”。

## 工作流

### 第 1 步：定义搜索标准

询问用户相关参数：
- **方向**：多头（看涨）、空头（看跌）或两者兼有。
- **市值**：大盘、中盘、小盘、微盘。
- **板块**：特定板块或跨板块。
- **风格**：价值型、增长型、质量型、特殊情况（Special Situation）、事件驱动。
- **地域**：美国、国际、全球。
- **主题**：任何特定的主题视角（AI、制造业回流、人口老龄化等）。

### 第 2 步：定量筛选 (Quantitative Screens)

根据投资风格运行筛选器：

**价值策略筛选**
- 市盈率 (P/E) 低于行业中位数。
- EV/EBITDA 低于历史平均水平。
- 自由现金流收益率 (FCF Yield) > 5%。
- 市净率 (P/B) 低于 1.5 倍。
- 过去 90 天内有内部人士买入。

**增长策略筛选**
- 收入同比增速 > 15%。
- 利润同比增速 > 20%。
- 收入加速增长（增速在提高）。
- 利润率扩张。
- 高投入资本回报率 (ROIC > 15%)。

**质量策略筛选**
- 持续的收入增长（5 年以上）。
- 稳定或扩张的利润率。
- 净资产收益率 (ROE) > 15%。
- 低债务权益比（资产负债率低）。

**空头（看跌）策略筛选**
- 收入下滑或增长失速。
- 利润率受压。
- 应收账款/存货相对于销售额异常上升。
- 内部人士减持。
- 基本面恶化但估值较同行有异常溢价。

### 第 3 步：主题扫街 (Thematic Sweep)

对于主题性机会，研究该主题并识别受益者：

1. **定义逻辑**（例如：“AI 基础设施支出到 2026 年将加速”）。
2. **绘制价值链图谱** —— 谁是直接受益者 vs. 间接收益者？
3. **识别纯正标的 (Pure-play) vs. 业务多样化标的**。
4. **评估溢价情况** —— 哪些标的已“反映预期 (Priced in)”，哪些被低估。
5. **寻找二阶受益者** —— 市场尚未将其与该主题联系起来的对象。

### 第 4 步：项目呈现

对于通过筛选的每个项目，呈现如下内容：

**[公司名称] —— [多头/空头] —— [一句话核心逻辑]**

| 指标 | 数值 | 与同行对比 |
|--------|-------|-----------|
| 市值 | | |
| EV/EBITDA (未来 12 个月) | | |
| 收入增速 | | |
| EBITDA 利润率 | | |

**投资逻辑（3-5 个重点）：**
- 为什么定价错了。
- 市场忽略了什么。
- 价值兑现的催化剂是什么。

### 第 5 步：产出

- 5-10 个推荐项目的短名单（附带一页摘要）。
- 记录筛选标准和方法论。
- 所有推荐项目的对比表。

### 🏮 金融小白附录：重点词汇详解

1. **Long / Short (多头/空头)**：看涨和看跌。Long 就是觉得会涨，买入等它升值；Short（做空）就是觉得会跌，想办法从它下跌中赚钱。
2. **Priced in (反映预期)**：意思是“好消息大家都知道了，也买得差不多了，股价已经涨上去了”。这种时候哪怕公司真的很好，也没什么获利空间了。
3. **Catalyst (催化剂)**：能让股价动起来的大事。比如出一份超预期的财报、发布一款牛逼的产品，或者是被大公司收购。没有催化剂的股票就像“死水”，虽然值钱但没人买。
4. **Pure-play (纯正标的)**：业务很单纯的公司。比如你想投资“挖矿”，找一家 100% 收入都靠卖金矿的公司就是 Pure-play；如果你找一家有金矿业务同时还卖保险和开餐馆的公司，那它的逻辑就“不纯”了。
5. **Mispriced (定价错误)**：捡漏的机会。市场因为某些原因（比如大家太恐慌，或者没人注意到这家小公司）给出的价格比它的真实价值低得多。
