# 客户回顾准备 (Client Review Prep)

**描述**：为客户回顾会议做准备，包括投资组合业绩摘要、资产配置分析、谈话要点和待办事项。将账户数据汇总成简明且可直接用于会议的格式。适用于季度评估、年度检查或临时客户会议。触发词包括：“客户回顾”、“为 [客户名] 准备会议资料”、“季度评估”、“为 [客户名] 做准备”或“客户会议”。

## 工作流

### 第 1 步：了解客户背景

收集或查阅以下信息：
- **客户姓名**及家庭成员情况。
- **账户类型**：应税账户、个人退休账户 (IRA)、Roth IRA、401(k)、信托等。
- **总资产规模 (AUM)**：跨账户的总资产。
- **投资策略声明 (IPS)**：目标资产配置、风险承受能力、投资限制。
- **生命周期阶段**：财富积累期、准退休期、退休期、财富传承期。
- **上次会议日期**及任何尚未完成的待办事项。

### 第 2 步：投资组合业绩

针对每个账户及家庭资产总额：

| 指标 | 本季度至今 (QTD) | 本年至今 (YTD) | 1 年期 | 3 年期 | 自开户以来 |
|--------|-----|-----|--------|--------|----------------|
| 投资组合收益率 | | | | | |
| 基准收益率 | | | | | |
| 超额收益 (Alpha) | | | | | |

**业绩归因分析：**
- 哪些资产类别 / 持仓驱动了收益？
- 前 3 大贡献项和前 3 大亏损项。
- 是否存在受单一持仓影响过大的情况？

### 第 3 步：配置回顾

当前配置 vs. 目标配置：

| 资产类别 | 目标 | 当前 | 偏离度 | 建议操作 |
|------------|--------|---------|-------|--------|
| 美国大盘股 | | | | |
| 美国中/小盘股 | | | | |
| 发达国家股票 | | | | |
| 新兴市场股票 | | | | |
| 固定收益 (债券) | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记任何超过 IPS 再平衡阈值（通常为 3-5%）的偏离。

### 第 4 步：谈话要点

生成会议议程：

1. **市场概览** (2-3 分钟)：简要的市场宏观背景及展望。
2. **投资组合业绩** (5 分钟)：我们的表现如何？原因是什么？
3. **配置回顾** (5 分钟)：是否需要进行资产再平衡？
4. **规划更新** (5-10 分钟)：
   - 生活变化？（工作、健康、家庭、房产、教育）。
   - 收入需求是否有变化？
   - 税务情况更新。
   - 遗产规划/财富传承更新。
5. **待办事项** (5 分钟)：下次会议前我们需要做什么？

### 第 5 步：主动建议

基于本次回顾，建议：
- 再平衡交易（如果偏离度超过阈值）。
- 税收损益收割 (Tax-loss harvesting) 机会。
- 现金部署或提款规划。
- Roth 转换机会（如适用）。
- 受益人更新或遗产规划需求。
- 保险回顾（人寿、伤残、长期护理险）。

### 第 6 步：产出

- 一页纸的客户回顾摘要 (Word 或 PDF)。
- 带基准对比的业绩表。
- 资产配置饼图（当前 vs. 目标）。
- 建议的待办事项清单。
- 会议议程。

## 注意事项

- **会前背景调查**：查看上次会议记录，充分了解客户情况。
- **客户导向**：先聊客户关心的事，而不是你想聊的事。
- **直面问题**：如果业绩不佳，请直接说明，不要隐瞒或修饰。
- **明确结论**：会议结束时必须有清晰的待办事项，并附上最后期限。
- **合规性**：确保所有材料符合公司政策和监管要求。

### 🏮 金融小白附录：重点词汇详解

1. **AUM (Assets Under Management, 管理资产规模)**：通俗讲就是客户把多少钱放在咱们这儿管。
2. **IPS (Investment Policy Statement, 投资策略声明)**：一份“投前协议”。规定了客户想赚多少钱、能亏多少钱、哪些东西不能买。
3. **Alpha (超额收益)**：你比大盘多赚的那部分。如果大盘涨 10%，你涨了 12%，那 2% 就是你的本事（Alpha）。
4. **Drift / Rebalancing (偏离/再平衡)**：原本计划股票和债券各占一半，结果因为股票涨得快变成了七三开，这就叫“偏离”。把它调回五五开的手法叫“再平衡”。
5. **Tax-loss Harvesting (税收损益收割)**：金融圈的“止损抵税”。故意卖掉一些亏钱的股票来抵掉赚钱股票产生的税，变相省钱。
