# 审查固定收益组合

> 本命令使用 LSEG 债券定价、YieldBook 分析和收益率曲线工具。可用工具参见 [CONNECTORS.md](../CONNECTORS.md)。

通过对所有持仓定价、丰富参考数据、预测现金流以及在利率情景下进行压力测试，生成综合固定收益组合风险和回报报告。

领域知识参见 **fixed-income-portfolio** 技能。

## 工作流程

### 1. 收集组合持仓

询问用户：
- 债券标识符（必填）— 逗号分隔的 ISIN、CUSIP 或 RIC
- 持仓规模/权重（可选 — 如未提供，假设等权重）
- 特定测试情景（可选 — 如 "+100bp"，默认为标准网格）
- 估值日期（可选，默认今日）

### 2. 对所有债券定价

使用所有标识符调用 `bond_price`。

每只债券提取：净价/全价、收益率、久期、凸性、DV01、货币。

汇总组合级别：加权收益率、加权久期、总 DV01、总市值。

### 3. 丰富参考数据

为每只债券调用 `yieldbook_bond_reference`。

提取：证券类型、行业、评级、票息类型、赎回条款、发行人、国家。

构建组成分解：按行业、评级、到期区间、货币。

### 4. 预测现金流

为每只债券调用 `yieldbook_cashflow`。

汇总为季度现金流瀑布图。标记集中到期的时段。

### 5. 运行情景分析

调用 `yieldbook_scenario`，设置利率变动：-200bp、-100bp、-50bp、0bp、+50bp、+100bp、+200bp。

识别哪些债券对上行和下行的风险贡献最大。

### 6. 曲线背景

为组合主要货币调用 `interest_rate_curve`。

计算每只债券相对曲线的利差。评估曲线环境。

### 7. 整合报告

呈现：组合汇总指标、组成分解、现金流瀑布、含风险贡献的情景损益表、曲线敞口。

## 输出格式

以组合汇总指标开头，然后分部分详述组成、现金流和风险分析。

## 📖 本篇金融知识小贴士

**固定收益组合（Fixed Income Portfolio）**：主要由债券组成的投资组合。债券持有者定期收到利息（票息），到期收回本金。

**久期（Duration）**：衡量债券价格对利率变动敏感度的指标，单位为年。久期 5 意味着利率升 1% 价格跌约 5%。组合久期 = 各债券久期的市值加权平均。

**凸性（Convexity）**：久期本身的"纠偏器"。利率变动大时，久期的线性估计不准，凸性帮你修正。凸性为正 = 利率跌时债券涨得比久期预测的多，利率涨时跌得少。

**DV01**：利率每变动 1 个基点，债券价格变动多少元。组合的总 DV01 告诉你"利率动一下，组合亏赚多少钱"。

**现金流瀑布图（Cashflow Waterfall）**：按时间顺序排列的未来现金流（票息 + 本金到期），看起来像瀑布一样逐层跌落。帮助识别哪些时间段有大量资金回笼。

**情景分析（Scenario Analysis）**：假设不同的利率变动情景（如利率涨100bp、跌100bp），计算组合在每个情景下的损益。压力测试的核心工具。

**收益率至最差（YTW, Yield to Worst）**：在所有可能的提前赎回情景中，取最低的那个收益率。最保守的收益衡量方式。

**基准（Benchmark）**：用来比较组合表现的参考标准，常见的是某只国债指数。组合跑赢基准 = 超额收益。
