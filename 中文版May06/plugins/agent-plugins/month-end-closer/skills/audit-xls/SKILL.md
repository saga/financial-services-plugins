# 审计电子表格

审计公式和数据以确保准确性和无错误。范围决定深度 — 从对选择的快速公式检查到完整的财务模型完整性审计。

## 步骤1：确定范围

如果用户已经给出了范围，使用它。否则**询问他们**：

> 你想让我审计什么范围？
> - **选择** — 仅当前选定的范围
> - **工作表** — 仅当前活动工作表
> - **模型** — 整个工作簿，包括财务模型完整性检查（BS平衡、现金核对、滚动转发、逻辑合理性）

**模型**范围是最深入的 — 在发送给客户或IC之前，用于DCF、LBO、三表、合并、可比公司或任何综合财务模型。

---

## 步骤2：公式级别检查（所有范围）

无论范围如何都运行这些检查：

| 检查 | 要查找的内容 |
|---|---|
| 公式错误 | `#REF!`、`#VALUE!`、`#N/A`、`#DIV/0!`、`#NAME?` |
| 公式内的硬编码 | `=A1*1.05` — `1.05`应该是一个单元格引用 |
| 不一致的公式 | 在行/列中与其邻居模式不同的公式 |
| 范围偏移一 | `SUM`/`AVERAGE`遗漏了第一行或最后一行 |
| 粘贴覆盖公式 | 看起来像公式但实际上是硬编码值的单元格 |
| 循环引用 | 有意的或意外的 |
| 断开的跨表链接 | 引用已移动或已删除的单元格 |
| 单位/规模不匹配 | 千与百万混合，%存储为整数 |
| 隐藏行/标签 | 可能包含覆盖或过时计算 |

## 步骤3：模型完整性检查（仅模型范围）

如果范围是**模型**，识别模型类型（DCF / LBO / 三表 / 合并 / 可比公司 / 自定义）并运行以下适当的完整性检查。

### 3a. 结构审查

| 检查 | 要查找的内容 |
|---|---|
| 输入/公式分离 | 输入是否与计算清楚分离？ |
| 颜色约定 | 蓝色=输入、黑色=公式、绿色=链接 — 或模型使用的任何约定，是否一致应用？ |
| 标签流程 | 逻辑顺序（假设 → IS → BS → CF → 估值）？ |
| 日期标题 | 所有标签一致？ |
| 单位 | 一致（千 vs 百万 vs 实际值）？ |

### 3b. 资产负债表

| 检查 | 测试 |
|---|---|
| BS平衡 | 总资产 = 总负债 + 权益（每个期间） |
| RE滚动转发 | 期初RE + 净利润 - 股息 = 期末RE |
| 商誉/无形资产 | 来自收购假设的流动（如果是M&A） |

如果BS不平衡，**量化每个期间的差距并追踪哪里出错** — 在修复之前其他都不重要。

### 3c. 现金流量表

| 检查 | 测试 |
|---|---|
| 现金核对 | CF期末现金 = BS现金（每个期间） |
| CF总和 | CFO + CFI + CFF = Δ现金 |
| D&A匹配 | CF上的D&A = IS上的D&A |
| CapEx匹配 | CF上的CapEx与BS上的PP&E滚动转发匹配 |
| WC变化 | 符号与BS变动匹配（ΔAR、ΔAP、Δ存货） |

### 3d. 利润表

| 检查 | 测试 |
|---|---|
| 收入构建 | 与细分市场/产品明细相关联 |
| 税收 | 税费 = 税前收入 × 税率（允许递延税调整） |
| 股份数 | 与稀释计划相关联（期权、可转债、回购） |

### 3e. 循环引用

- 利息 → 债务余额 → 现金 → 利息是LBO/三表模型中常见的有意识循环
- 如果有意识：验证迭代开关存在并工作
- 如果无意识：追踪循环并标记如何打破它

### 3f. 逻辑和合理性

| 检查 | 标记条件 |
|---|---|
| 增长率 | >100%收入增长无解释 |
| 利润率 | 超出行业规范 |
| 终值主导 | TV > DCF EV的~75%（黄旗） |
| 曲棍球棒 | 预测在远年不现实地加速 |
| 复利 | EBITDA复利到第10年达到荒谬的$ |
| 边缘情况 | 模型在0%或负增长、负EBITDA、杠杆变负时出错 |

### 3g. 模型类型特定错误

**DCF：**
- 折现率应用于错误期间（年中 vs 年末）
- 终值未折现回现值
- WACC使用账面价值而非市场价值
- FCF包含利息费用（应该是无杠杆的）
- 税盾重复计算

**LBO：**
- 债务偿还与现金清扫机制不匹配
- PIK利息未计入本金
- 管理层展期未反映在回报中
- 退出倍数应用于错误的EBITDA（LTM vs NTM）
- 费用/支出未从第1天权益中扣除

**合并：**
- 增厚/稀释使用错误的股份数（交易前 vs 交易后）
- 协同效应未分阶段实施
- 收购价格分配不平衡
- 现金损失的利息未包括
- 交易费用未在来源和用途中

**三表：**
- 营运资本变化符号错误
- 折旧与PP&E计划不匹配
- 债务到期计划与本金还款不匹配
- 股息超过净利润无解释

## 步骤4：报告

输出发现表：

| # | 工作表 | 单元格/范围 | 严重性 | 类别 | 问题 | 建议修复 |
|---|---|---|---|---|---|---|

**严重性：**
- **关键** — 输出错误（BS不平衡、公式损坏、现金不匹配）
- **警告** — 有风险（硬编码、不一致公式、边缘情况失败）
- **信息** — 风格/最佳实践（颜色编码、布局、命名）

对于**模型**范围，在前面加上摘要行：

> 模型类型：[DCF/LBO/三表/...] — 总体：[清洁 / 次要问题 / 主要问题] — [N]关键，[N]警告，[N]信息

**不要在未询问的情况下更改任何内容** — 先报告，根据请求修复。

## 注意事项

- **BS平衡优先** — 如果不平衡，下游所有内容都可疑
- **硬编码覆盖是无声错误的第一来源** — 积极搜索
- **符号约定错误**（现金流出的正 vs 负）极其常见
- 如果模型使用VBA宏，注意任何无法仅从公式审计的宏驱动计算

## 金融术语和知识详细解释

### 1. 月末结账审计（Month-End Close Audit）

**定义**：在月末结账过程中对财务数据和报表进行的审计检查。

**审计重点**：
- 账户余额准确性
- 期间截止正确性
- 调整分录完整性
- 内部控制有效性

**为什么重要**：
- 确保财务报告准确
- 支持管理决策
- 满足合规要求
- 发现潜在问题

---

### 2. 公式错误类型

**常见错误**：
| 错误 | 含义 | 常见原因 |
|------|------|---------|
| #REF! | 无效引用 | 单元格已删除 |
| #VALUE! | 值类型错误 | 文本与数字运算 |
| #N/A | 值不可用 | VLOOKUP未找到 |
| #DIV/0! | 除以零 | 分母为零 |
| #NAME? | 名称错误 | 函数名拼写错 |

**为什么重要**：
- 影响计算准确性
- 可能导致错误决策
- 需要立即修复
- 专业标准要求

---

### 3. 硬编码风险

**定义**：在公式中直接输入数字而非引用单元格。

**风险**：
- 难以追踪假设
- 更新容易遗漏
- 审计困难
- 错误传播

**最佳实践**：
- 所有输入放在输入标签
- 使用命名范围
- 颜色编码区分
- 文档记录假设

---

### 4. 循环引用处理

**定义**：公式直接或间接引用自身。

**类型**：
- 有意识循环：如利息费用依赖债务余额
- 无意识循环：公式设计错误

**处理方法**：
- 有意识：启用迭代计算
- 无意识：重新设计公式
- 文档记录循环原因

---

### 5. BS平衡检查

**核心公式**：
```
总资产 = 总负债 + 股东权益
```

**不平衡原因**：
- 公式错误
- 数据输入错误
- 调整遗漏
- 期间错误

**为什么重要**：
- 会计恒等式
- 财务报表准确性基础
- 审计必查项目
- 专业标准要求

---

### 6. 现金核对

**定义**：验证现金流量表期末现金与资产负债表现金一致。

**核对要点**：
- 期间一致性
- 货币单位一致
- 调整项完整
- 计算正确

**为什么重要**：
- 现金是关键资产
- 流动性管理
- 财务报表一致性
- 审计重点

---

### 7. 营运资本变化

**定义**：流动资产和流动负债的期间变化。

**计算公式**：
```
营运资本 = 流动资产 - 流动负债
Δ营运资本 = 期末营运资本 - 期初营运资本
```

**符号约定**：
- 流动资产增加：现金流出（负）
- 流动负债增加：现金流入（正）

**为什么重要**：
- 现金流影响
- 运营效率指标
- 流动性分析
- 工作资本管理

---

### 8. 递延税调整

**定义**：会计利润与应税利润差异导致的税务调整。

**常见项目**：
- 折旧方法差异
- 收入确认时点
- 费用预提
- 损失结转

**为什么重要**：
- 税务规划
- 现金流影响
- 财务报表准确性
- 合规要求

---

### 9. 稀释计划

**定义**：计算完全稀释股份数的计划。

**包含项目**：
- 股票期权
- 限制性股票
- 可转换债券
- 认股权证

**为什么重要**：
- EPS计算
- 估值准确性
- 股东权益
- 并购分析

---

### 10. 月末结账流程

**标准流程**：
1. 交易截止确认
2. 调整分录编制
3. 账户余额核对
4. 财务报表编制
5. 差异分析
6. 管理层审阅

**为什么重要**：
- 及时准确报告
- 内部控制
- 合规要求
- 管理决策支持
