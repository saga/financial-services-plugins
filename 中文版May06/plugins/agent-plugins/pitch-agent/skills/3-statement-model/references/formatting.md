# 格式标准参考

| 元素 | 格式 |
|---------|--------|
| 硬编码输入 | 蓝色字体 |
| 公式 | 黑色字体 |
| 链接到其他工作表 | 绿色字体 |
| 检查单元格 | 错误时红色，平衡时绿色 |
| 负值 | 括号，而非减号 |
| 货币 | 大数字无小数点，每股数据2位小数 |
| 百分比 | 1位小数 |
| 标题 | 粗体，底部边框 |
| 单位行 | 在标题下方包含单位行（百万美元、%等） |

## 视觉分隔指南

- 历史列和预测列之间使用细垂直边框
- 部分合计后使用粗底部边框（例如，总资产）
- 小计使用单底部边框
- 合计使用双底部边框

## 合计和小计行格式

所有合计和小计行必须对其数值使用**粗体字体格式**，以清楚区分汇总数据与单个行项目。

## 资产负债表检查行格式

资产负债表检查行（在负债和权益合计下方）使用条件数字格式，当非零值时显示红色。当资产负债表正确平衡时（检查=0），值显示为黑色或标准格式。

| 检查值 | 字体颜色 |
|-------------|---------|
| = 0（平衡） | 黑色（标准） |
| ≠ 0（错误） | 红色 |

**实现：** 应用自定义数字格式`[Red][<>0]0.00;[Red][<>0](0.00);0.00`或使用Excel条件格式，规则为"单元格值≠0"→红色字体。

## 利润率行格式

| 元素 | 格式 |
|---------|--------|
| 利润率%行 | 缩进，斜体，1位小数 |
| 正向趋势 | 无特殊格式（或微妙绿色） |
| 负向趋势 | 标记供审查（微妙黄色） |
| 低于同行平均 | 考虑高亮以便讨论 |

## 信用指标格式

| 元素 | 格式 |
|---------|--------|
| 杠杆倍数 | 1位小数，带"x"后缀（例如，2.5x） |
| 百分比 | 1位小数，带"%"后缀 |
| 净债务为负 | 括号，表示净现金状况 |
| 部分标题 | 粗体，"信用指标" |
| 分隔线 | 信用指标部分上方细边框 |

## 信用指标阈值颜色

| 指标 | 绿色 | 黄色 | 红色 |
|--------|-------|--------|------|
| 总债务/EBITDA | < 2.5x | 2.5x-4.0x | > 4.0x |
| 净债务/EBITDA | < 2.0x | 2.0x-3.5x | > 3.5x |
| 利息保障倍数 | > 4.0x | 2.5x-4.0x | < 2.5x |
| 债务/总资本 | < 40% | 40%-60% | > 60% |
| 流动比率 | > 1.5x | 1.0x-1.5x | < 1.0x |
| 速动比率 | > 1.0x | 0.75x-1.0x | < 0.75x |

## 检查选项卡的条件格式

- 单元格包含通过指示 → 绿色填充
- 单元格包含失败指示 → 红色填充
- 单元格包含警告 → 黄色填充
- 差异单元格 = 0 → 浅绿色填充
- 差异单元格 ≠ 0 → 浅红色填充

## 利润率合理性标志

- 毛利率 < 0% → 错误：审查COGS
- 毛利率 > 80% → 警告：验证收入/COGS
- EBITDA利润率 < 0% → 标记：运营亏损
- EBITDA利润率 > 50% → 警告：异常高
- 净利润率 < 0% → 标记：净亏损（可能在增长阶段可接受）
- 净利润率 > 毛利率 → 错误：公式问题

## 金融术语和知识详细解释

### 1. 财务建模中的颜色编码系统

**行业标准颜色约定**：

| 颜色 | 含义 | 示例 |
|------|------|------|
| 蓝色/黑色 | 硬编码输入值 | 历史实际数据、假设驱动因素 |
| 黑色 | 公式计算值 | 汇总、小计、派生值 |
| 绿色 | 跨工作表链接 | 链接到其他标签页的值 |
| 红色 | 错误或检查失败 | 公式错误、核对失败 |
| 紫色 | 场景切换 | 选择的情景输入 |

**为什么重要**：
- 一目了然地识别单元格类型
- 防止意外覆盖公式
- 加速模型审查
- 行业最佳实践

---

### 2. 条件格式（Conditional Formatting）

**定义**：条件格式是Excel根据单元格值自动应用格式的功能。

**应用场景**：
- 当检查值≠0时显示红色
- 当指标超出阈值时高亮
- 当利润率低于预期时标记

**在财务模型中的使用**：
- 资产负债表平衡检查
- 信用指标阈值警告
- 利润率异常标记
- 场景层次验证

---

### 3. 阈值颜色系统（交通灯系统）

**绿色、黄色、红色阈值**：

| 指标类型 | 绿色（安全） | 黄色（警告） | 红色（危险） |
|----------|-------------|-------------|-------------|
| 杠杆比率 | 低 | 中 | 高 |
| 利息保障 | 高 | 中 | 低 |
| 利润率 | 高于平均 | 接近平均 | 低于平均 |

**为什么重要**：
- 快速识别需要关注的项目
- 支持决策过程
- 行业标准做法
- 投资委员会报告常见格式
