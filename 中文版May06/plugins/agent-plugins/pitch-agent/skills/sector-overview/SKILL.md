# 行业概览

description: 创建全面的行业和板块格局报告，涵盖市场动态、竞争定位、主要参与者和主题趋势。用于客户请求、行业启动、主题研究或内部知识建设。触发条件包括"行业概览"、"行业报告"、"市场格局"、"行业分析"、"行业深度研究"或"主题研究"。

## 工作流程

### 第1步：定义范围

- **行业/子行业**：什么行业以及定义有多窄？
- **目的**：客户报告、内部研究、推介材料、创意生成
- **深度**：高层概览（5-10页）或深度研究（20-30页）
- **角度**：中性格局 vs 主题论点（例如，"AI基础设施建设"）
- **范围**：仅上市公司，还是包括私营公司？

### 第2步：市场概览

**市场规模与增长**
- 总可寻址市场（TAM）及来源
- 历史增长率（5年CAGR）
- 预测增长率和关键假设
- 市场细分（按产品、地区、终端市场、客户类型）

**行业结构**
- 分散 vs 集中 — 前5名市场份额
- 价值链图 — 价值在哪里积累？
- 商业模式类型（订阅、交易、许可、服务）
- 进入壁垒（资本、监管、技术、网络效应）

**关键趋势与驱动因素**
- 长期顺风（3-5个主要趋势）
- 逆风和风险
- 技术颠覆向量
- 监管发展
- 并购活动和整合趋势

### 第3步：竞争格局

**公司简介**（前5-10名参与者）：

| 公司 | 收入 | 增长 | EBITDA利润率 | 市场份额 | 关键差异化因素 |
|---------|---------|--------|--------------|-------------|-------------------|
| | | | | | |

对于每家公司，简要简介：
- 业务描述（2-3句话）
- 战略定位和护城河
- 近期发展（财报、并购、产品发布）
- 估值快照（P/E、EV/EBITDA、EV/收入）

**竞争动态**
- 公司如何竞争？（价格、产品、服务、分销）
- 谁在获得/失去份额，为什么？
- 新进入者或相邻参与者的颠覆风险

### 第4步：估值背景

- 行业交易倍数（当前和历史范围）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 行业与更广泛市场相比如何？

### 第5步：投资启示

- 最佳风险/回报机会在哪里？
- 可以通过这个行业表达什么主题押注？
- 行业中的关键争论（多头 vs 空头论点）
- 可能改变行业叙事的催化剂

### 第6步：输出

- Word文档或PowerPoint，包括：
  - 市场概览和规模
  - 竞争格局图
  - 公司比较表
  - 估值摘要
  - 关键图表：市场增长、份额趋势、估值历史
- Excel附录，包含详细公司数据

## 重要说明

- 为所有市场规模数据提供来源 — 引用研究公司或方法论
- 区分TAM炒作和现实的可寻址市场
- 行业概览老化快 — 注明日期并标记可能过时的数据
- 图表至关重要 — 市场规模瀑布图、竞争定位矩阵、估值散点图
- 如果为客户，将"那又怎样"量身定制到他们的具体情况（并购目标识别、竞争定位、市场进入）

## 金融术语和知识详细解释

### 1. 总可寻址市场（TAM - Total Addressable Market）

**定义**：产品或服务在理论上可以覆盖的全部市场需求。

**计算方法**：
```
TAM = 目标客户数量 × 客户平均支出
```

**市场层次**：
| 层次 | 定义 | 示例 |
|------|------|------|
| TAM | 总可寻址市场 | 全球智能手机市场 |
| SAM | 可服务可寻址市场 | 北美智能手机市场 |
| SOM | 可服务可获取市场 | 公司实际能获得的市场份额 |

**为什么重要**：
- 评估市场机会
- 支持投资决策
- 业务规划基础
- 投资者关注指标

---

### 2. CAGR（复合年增长率）

**定义**：一段时间内的平均年增长率，考虑了复利效应。

**计算公式**：
```
CAGR = (终值/初值)^(1/年数) - 1
```

**示例**：
- 2019年收入：$100M
- 2024年收入：$200M
- CAGR = (200/100)^(1/5) - 1 = 14.9%

**为什么重要**：
- 平滑波动
- 便于比较
- 长期趋势分析
- 行业基准

---

### 3. 价值链分析（Value Chain Analysis）

**定义**：分析产品从原材料到最终消费者的价值创造过程。

**价值链环节**：
- 研发
- 采购
- 生产
- 分销
- 营销
- 售后服务

**为什么重要**：
- 识别价值积累点
- 发现竞争优势
- 理解行业结构
- 投资决策支持

---

### 4. 进入壁垒（Barriers to Entry）

**定义**：阻止新竞争者进入行业的障碍。

**常见壁垒类型**：

| 类型 | 示例 | 行业 |
|------|------|------|
| 资本壁垒 | 高额初始投资 | 半导体制造 |
| 监管壁垒 | 许可证要求 | 金融、医疗 |
| 技术壁垒 | 专利保护 | 制药 |
| 网络效应 | 用户基础 | 社交媒体 |
| 规模经济 | 成本优势 | 零售 |

**为什么重要**：
- 评估竞争强度
- 预测行业盈利能力
- 投资决策因素
- 战略规划

---

### 5. 护城河（Moat）

**定义**：公司维持竞争优势并保护长期利润的能力。

**护城河类型**：
- 无形资产（品牌、专利）
- 转换成本
- 网络效应
- 成本优势
- 规模优势

**为什么重要**：
- 可持续竞争优势
- 长期盈利能力
- 估值溢价
- 投资决策

---

### 6. EBITDA利润率

**定义**：EBITDA占收入的百分比，衡量运营盈利能力。

**计算公式**：
```
EBITDA利润率 = EBITDA / 收入 × 100%
```

**行业比较**：
| 行业 | 典型EBITDA利润率 |
|------|-----------------|
| 软件 | 20-40% |
| 零售 | 5-10% |
| 制造 | 10-15% |
| 金融 | 30-50% |

**为什么重要**：
- 运营效率指标
- 行业比较基准
- 估值倍数基础
- 盈利能力分析

---

### 7. 市场集中度

**定义**：行业被少数大公司控制的程度。

**衡量指标**：
- CR4：前4名公司市场份额
- CR5：前5名公司市场份额
- HHI：赫芬达尔指数

**集中度分类**：
| 类型 | CR5 | 特点 |
|------|-----|------|
| 高度集中 | >60% | 寡头垄断 |
| 中度集中 | 40-60% | 寡头竞争 |
| 低度集中 | <40% | 竞争激烈 |

**为什么重要**：
- 竞争强度
- 定价能力
- 行业盈利能力
- 监管关注

---

### 8. 估值倍数（Valuation Multiples）

**定义**：公司价值相对于财务指标的比率。

**常见倍数**：
| 倍数 | 公式 | 用途 |
|------|------|------|
| P/E | 股价/每股收益 | 盈利公司 |
| EV/EBITDA | 企业价值/EBITDA | 运营比较 |
| EV/收入 | 企业价值/收入 | 增长公司 |
| P/B | 股价/账面价值 | 金融公司 |

**为什么重要**：
- 相对估值
- 行业比较
- 交易参考
- 投资决策

---

### 9. 颠覆风险（Disruption Risk）

**定义**：新技术或商业模式改变行业格局的可能性。

**颠覆类型**：
- 技术颠覆
- 商业模式颠覆
- 监管颠覆
- 消费者行为变化

**评估因素**：
- 技术成熟度
- 采用速度
- 资本投入
- 监管环境

**为什么重要**：
- 投资风险评估
- 战略规划
- 行业分析
- 风险管理

---

### 10. 催化剂（Catalyst）

**定义**：可能显著影响公司或行业价值的事件。

**催化剂类型**：
| 类型 | 示例 |
|------|------|
| 公司特定 | 新产品发布、管理层变动 |
| 行业层面 | 监管变化、技术突破 |
| 宏观经济 | 利率变化、经济周期 |

**为什么重要**：
- 投资时机
- 风险管理
- 投资论点验证
- 投资组合管理
