# 行业概览

description: 创建涵盖市场动态、竞争定位、主要参与者和主题趋势的综合行业报告。用于客户请求、行业启动、主题研究或内部知识建设。触发词包括"行业概览"、"行业报告"、"市场格局"、"行业分析"、"行业深度分析"或"主题研究"。

## 工作流程

### 步骤1：定义范围

- **行业/子行业**：什么行业，定义多窄？
- **目的**：客户报告、内部研究、推介材料、创意生成
- **深度**：高层概述（5-10页）或深度分析（20-30页）
- **角度**：中立格局 vs. 主题论点（例如"AI基础设施建设"）
- **范围**：仅上市公司，还是包括私营公司？

### 步骤2：市场概述

**市场规模与增长**
- 总体可寻址市场（TAM）及来源
- 历史增长率（5年CAGR）
- 预测增长率和关键假设
- 市场细分（按产品、地理区域、终端市场、客户类型）

**行业结构**
- 分散 vs. 整合 — 前5大市场份额
- 价值链图 — 价值在哪里积累？
- 商业模式类型（订阅、交易、许可、服务）
- 进入壁垒（资本、监管、技术、网络效应）

**关键趋势与驱动因素**
- 长期顺风（3-5个主要趋势）
- 逆风与风险
- 技术颠覆方向
- 监管发展
- 并购活动和整合趋势

### 步骤3：竞争格局

**公司概况**（前5-10名参与者）：

| 公司 | 收入 | 增长率 | EBITDA利润率 | 市场份额 | 关键差异化因素 |
|------|------|--------|--------------|----------|----------------|
| | | | | | |

每个公司的简要概况：
- 业务描述（2-3句话）
- 战略定位和护城河
- 近期发展（收益、并购、产品发布）
- 估值快照（P/E、EV/EBITDA、EV/收入）

**竞争动态**
- 公司如何竞争？（价格、产品、服务、分销）
- 谁在获得/失去份额，为什么？
- 来自新进入者或相邻玩家的颠覆风险

### 步骤4：估值背景

- 行业交易倍数（当前和历史范围）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 行业与大盘相比如何？

### 步骤5：投资影响

- 最佳风险/回报机会在哪里？
- 通过这个行业可以表达哪些主题性投资？
- 行业内的关键争论（看多 vs. 看空论点）
- 可能改变行业叙事的催化剂

### 步骤6：输出

- Word文档或PowerPoint演示文稿，包含：
  - 市场概述和规模
  - 竞争格局图
  - 公司比较表
  - 估值摘要
  - 关键图表：市场增长、份额趋势、估值历史
- 包含详细公司数据的Excel附录

## 重要注意事项

- 所有市场规模数据都要注明来源 — 引用研究公司或方法
- 区分TAM炒作与实际可寻址市场
- 行业概览过时快 — 注明日期并标记可能过时的数据
- 图表至关重要 — 市场规模瀑布图、竞争定位矩阵、估值散点图
- 如果是为客户准备的，根据他们的具体情况定制"那么又怎样"（并购目标识别、竞争定位、市场进入）

## 金融术语解释

### 1. TAM（总体可寻址市场）

**定义**：TAM（Total Addressable Market）是指公司产品或服务理论上可以服务的最大市场规模。

**计算方法**：
- **自上而下法**：从宏观经济数据推导
- **自下而上法**：基于客户数量和平均消费
- **市场因子法**：基于关键市场指标

**注意事项**：
- TAM只是理论潜力，实际可获取的市场要小得多
- 常用术语：
  - **TAM**：总体可寻址市场
  - **SAM**：可服务市场（Serviceable Addressable Market）
  - **SOM**：可获取市场（Serviceable Obtainable Market）

### 2. 价值链（Value Chain）

**定义**：价值链是指产品或服务从原材料到最终消费者手中的一系列增值活动。

**组成部分**：
- **上游**：原材料、零部件供应
- **中游**：生产、制造、加工
- **下游**：分销、零售、售后服务

**价值分配**：
- 不同环节的利润水平差异很大
- 技术密集型环节通常获得更高利润
- 品牌和渠道控制能力决定议价权

### 3. 进入壁垒（Barriers to Entry）

**定义**：进入壁垒是指阻止新公司进入特定市场的障碍。

**主要类型**：
- **资本壁垒**：需要大量初始投资
- **技术壁垒**：专利、专有技术、研发能力
- **监管壁垒**：许可证、合规要求
- **规模经济**：现有企业的成本优势
- **网络效应**：用户越多，产品价值越高

**重要性**：
- 高壁垒保护现有企业利润
- 低壁垒意味着竞争激烈
- 壁垒变化可能颠覆行业格局

### 4. 护城河（Moat）

**定义**：护城河是指企业拥有的可持续竞争优势，使其能够在长期内保护市场份额和利润。

**来源**：
- **网络效应**：用户增长增加产品价值
- **成本优势**：规模经济或独特成本结构
- **无形资产**：品牌、专利、客户关系
- **转换成本**：客户转换到竞争对手的成本
- **有效规模**：市场主导地位带来的优势

**评估方法**：
- 分析企业的定价能力
- 观察市场份额稳定性
- 比较竞争对手的盈利能力

### 5. 估值倍数（Valuation Multiples）

**定义**：估值倍数是用于比较和评估公司价值的财务比率。

**常用倍数**：
- **P/E（市盈率）**：股价/每股收益，适用于盈利稳定的公司
- **EV/EBITDA**：企业价值/息税折旧摊销前利润，适用于资本密集型行业
- **EV/Revenue**：企业价值/收入，适用于成长型或亏损公司

**使用注意事项**：
- 确保可比性（相同会计处理、业务模式）
- 考虑增长阶段和资本结构差异
- 结合绝对估值方法使用

### 6. 催化剂（Catalyst）

**定义**：催化剂是指可能显著影响公司股价或行业表现的事件或因素。

**类型**：
- **公司特定催化剂**：财报发布、产品发布、并购公告
- **行业催化剂**：监管变化、技术突破、需求转变
- **宏观催化剂**：利率变化、经济数据、地缘政治事件

**投资意义**：
- 识别催化剂可以帮助预测股价走势
- 催化剂时间不确定，需要风险管理
- 预期差是投资机会的来源

### 7. 竞争格局（Competitive Landscape）

**定义**：竞争格局描述了行业内竞争对手的分布、相对地位和竞争关系。

**分析维度**：
- **市场份额分布**：集中度分析
- **竞争定位**：差异化策略
- **战略群组**：具有相似战略的公司群体
- **进入/退出动态**：新进入者和退出者

**工具**：
- **波特五力模型**：分析竞争强度
- **SWOT分析**：评估优势、劣势、机会、威胁
- **竞争矩阵**：多维度比较竞争对手

### 8. 行业生命周期（Industry Life Cycle）

**定义**：行业生命周期描述了行业从诞生到衰退的发展阶段。

**阶段**：
- **导入期**：新产品、高不确定性、低渗透率
- **成长期**：快速增长、竞争加剧、规模扩张
- **成熟期**：增长放缓、整合、效率竞争
- **衰退期**：需求下降、产能过剩、退出

**投资启示**：
- 成长期寻找高增长机会
- 成熟期关注现金流和分红
- 衰退期警惕价值陷阱

### 9. 主题投资（Thematic Investing）

**定义**：主题投资是基于宏观趋势或结构性变化的投资策略。

**特点**：
- 跨越传统行业边界
- 关注长期结构性趋势
- 需要深入的研究和判断

**常见主题**：
- 技术创新（AI、新能源、生物科技）
- 人口结构变化（老龄化、城市化）
- 可持续发展（ESG、碳中和）

### 10. 风险/回报比（Risk/Reward Ratio）

**定义**：风险/回报比是评估投资机会的指标，衡量潜在收益与潜在损失的比例。

**计算方法**：
- 风险/回报比 = 潜在回报 / 潜在风险

**应用**：
- 通常寻求至少1:2或1:3的风险回报比
- 结合概率分析使用
- 用于投资组合优化和资本配置
