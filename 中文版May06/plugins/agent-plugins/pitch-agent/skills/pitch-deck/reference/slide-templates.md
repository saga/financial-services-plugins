## 模板分析流程

在填充任何模板之前，分析其结构：

### 步骤1：识别所有内容区域

扫描每张幻灯片以查找：
- **标题/页眉占位符** — 幻灯片标题放置位置
- **副标题/定义区域** — 二级标题或定义
- **内容框** — 主要内容区域（可能有标签侧边栏）
- **表格占位符** — 指定用于表格数据的区域
- **图表/视觉区域** — 图表、图表或图像的空间
- **指标标注框** — 突出显示的关键数字
- **脚注/来源栏** — 底部区域用于引用和注释
- **标志占位符** — 通常在右上角

### 步骤2：记录模板约定

每个模板都有自己的风格。观察：
- **配色方案** — 标题、背景、强调使用什么颜色？
- **字体选择** — 已设置了什么字体和大小？
- **框样式** — 内容框是否有侧边栏、边框或阴影？
- **项目符号样式** — 模板使用什么项目符号符号？
- **对齐模式** — 平行部分如何对齐？

### 步骤3：识别说明与输出区域

模板通常包含指导信息：
- **说明框** — 带指导文本的彩色框（通常是黄色背景、白色文本）
- **占位符文本** — 方括号 [] 中的文本，表示要替换的内容
- **示例内容** — 显示预期格式的示例内容

**关键区别**：说明框告诉你要做什么；在最终输出中应该重新格式化或删除。输出区域是你的内容放置的地方。

## 内容映射工作流程

### 步骤1：清点源数据

创建所有可用数据的列表：
- 市场规模数字和范围
- 增长率（CAGR、同比）
- 公司名称和描述
- 细分市场定义
- 财务指标
- 来源引用和日期
- 脚注内容

### 步骤2：将数据匹配到模板部分

对于每个模板部分，识别：

| 模板部分 | 需要的数据 | 来源位置 |
|----------|------------|----------|
| [部分名称] | [需要的数据] | [在哪里找到] |

### 步骤3：识别差距

映射后，记录：
- **缺失数据** — 模板需要但来源中没有的数据
- **额外数据** — 来源包含但模板中没有位置的数据
- **格式不匹配** — 数据存在但格式错误

### 步骤4：填充前解决差距

- 缺失数据：向用户标记或搜索其他来源
- 额外数据：确认是否应该排除或是否需要调整模板
- 格式不匹配：将数据转换为所需格式

## 常见幻灯片类型和数据要求

这些是常见幻灯片类型的典型数据要求。你的特定模板可能有所不同——始终遵循模板的实际结构。

### 市场定义幻灯片

**典型内容区域：**
- 范围内的细分市场（带示例/主要参与者）
- 范围外的细分市场（带示例）
- 市场定义文本
- 范围理由/合理性

**数据映射注意事项：**
- 源数据应明确区分包含和排除的细分市场
- 主要参与者应映射到各自的细分市场
- 定义文本应与来源定义市场的方式一致

**通常需要的数据：**
- 要包含的市场细分列表（带主要参与者示例）
- 要排除的市场细分列表（带示例）
- 市场定义文本
- 范围理由或合理性

**格式原则：** 平行部分（包含与排除）应使用匹配的格式。

**验证问题：**
- 每个细分市场是否有适当的符号（✓表示包含，×表示排除）？
- 主要参与者是否正确分配到细分市场？
- 定义是否与来源方法匹配？

### 市场规模 / TAM 幻灯片

**典型内容区域：**
- 当前市场规模（带年份）
- 增长率（带期间的CAGR）
- 未来预测（带目标年份）
- 按来源细分的表格
- 共识/汇总数字
- 关键要点或见解

**通常需要的数据：**
- 带基准年的市场规模数字
- 增长率（带时间段的CAGR）
- 带目标年份的预测数字
- 每个数据点的来源引用

**示例列标题：** 来源 | [基准年]规模 | CAGR | [目标年]预测

**格式原则：** 如果显示多个来源，包括共识/汇总行。

**数据映射注意事项：**
- 多个来源可能有不同的估计值——将每个映射到表格行
- 共识数字需要从各个来源计算
- 预测应可使用CAGR公式验证

**验证问题：**
- 所有来源数字是否与原始文档匹配？
- 共识是否计算正确（不仅仅是从一个来源复制）？
- 预测年份在所有数字中是否一致？
- 基于CAGR的预测手动验证时是否匹配？

### 竞争格局幻灯片

**典型内容区域：**
- 竞争对手作为列的比较表
- 功能/能力行
- 财务指标行（收入、增长、市场份额）
- 关键观察或定位注释

**通常需要的数据：**
- 要比较的竞争对手列表
- 每个的功能或能力
- 财务指标（收入、增长、市场份额）（如果可用）
- 财务数据的时间段

**格式原则：** 主题公司应与竞争对手视觉区分（例如，粗体文本、不同背景颜色、边框或位于最右列）。

**数据映射注意事项：**
- 确保包含源数据中的所有竞争对手
- 功能比较应使用一致的标准
- 财务数字应来自可比时期

**验证问题：**
- 源数据中的所有竞争对手是否都有代表？
- 主题公司是否在视觉上有所区分？
- 财务数字是否来自同一时期？
- ✓/×用法是否一致准确？

### 财务摘要幻灯片

**典型内容区域：**
- 关键指标标注（标题数字）
- 历史财务表（实际数据）
- 预测财务表（估计值）
- 增长率和利润率
- 可选趋势图表

**通常需要的数据：**
- 最近几年的历史财务数据（实际数据）
- 未来几年的预测财务数据（估计值）
- 关键指标：收入、增长率%、利润率、EBITDA

**示例列标题：** 指标 | FY[年份-2] | FY[年份-1] | FY[年份]A | FY[年份+1]E | FY[年份+2]E

**格式原则：** 清楚地区分历史（A）和预测（E）数据。

**数据映射注意事项：**
- 清楚地区分历史（A）和预测（E）数据
- 确保指标定义与来源匹配（收入与净收入，EBITDA与调整后EBITDA）
- 增长率应一致计算

**验证问题：**
- 历史与预测期间是否清晰标记？
- 计算的增长率是否与来源或手动计算匹配？
- 指标定义是否与源文档一致？

### 交易可比性幻灯片

**典型内容区域：**
- 交易表（日期、目标、收购方、交易价值）
- 估值倍数（EV/收入、EV/EBITDA）
- 汇总统计（均值、中位数、高、低）
- 主题公司的隐含估值

**通常需要的数据：**
- 交易详情：日期、目标、收购方、交易价值
- 估值倍数：EV/收入、EV/EBITDA
- 主题公司指标用于隐含估值

**格式原则：** 包括倍数的汇总统计（均值、中位数、高、低）。

**数据映射注意事项：**
- 倍数应从交易数据计算，而不仅仅是复制
- 汇总统计需要对所有交易进行计算
- 隐含估值将倍数应用于主题公司指标

**验证问题：**
- 是否包含了来源中的所有相关交易？
- 倍数计算是否正确（EV ÷ 指标）？
- 汇总统计是否涵盖表格中的所有交易？
- 隐含估值是否明确标记为说明性？

## 映射验证清单

在进行格式化之前，验证映射完整性：

### 数据完整性
- [ ] 每个模板占位符都有映射的源数据
- [ ] 所有来源引用都记录在脚注中
- [ ] 没有未映射的占位符 [方括号]

### 数据准确性
- [ ] 数字与原始源文档完全匹配
- [ ] 年份和时间段正确记录
- [ ] 公司名称拼写正确
- [ ] 计算值（共识、预测、倍数）已验证

### 逻辑一致性
- [ ] 包含与排除的细分市场在逻辑上连贯
- [ ] 历史数据按时间顺序先于预测数据
- [ ] 比较数据使用一致的时间段
- [ ] 总计和小计求和正确

### 来源归属
- [ ] 每个数据点都可以追溯到来源
- [ ] 来源名称和出版年份已记录
- [ ] 特殊注释分配了脚注编号

## 处理数据-模板不匹配

### 模板需要的数据比可用的多

**选项：**
1. 明确向用户标记差距
2. 将部分标记为"数据不可用"并说明
3. 适当时搜索其他来源
4. 如果数据不存在，建议调整模板

**不要：** 捏造数据或做出无根据的估计。

### 来源的数据比模板容纳的多

**选项：**
1. 包含最相关/最新的数据点
2. 在适当的地方汇总或聚合
3. 添加引用其他可用数据的脚注
4. 如果数据至关重要，建议扩展模板

### 数据格式与模板格式不匹配

**常见转换：**
- 单个数字 → 范围（使用来源的最小-最大值）
- 详细细分 → 汇总类别
- 年度数字 → CAGR（从端点计算）
- 绝对值 → 百分比（计算份额）
- 多个来源 → 共识（应用方法）

### 模板使用不同的术语

**解决流程：**
1. 识别模板术语和来源术语
2. 确认它们指的是同一个概念
3. 在输出中使用模板术语
4. 如果需要澄清，添加脚注

## 模板特定适配

记住：本指南描述常见模式，而非要求。始终：

1. **遵循模板** — 如果模板使用不同的部分名称，使用这些名称
2. **匹配模板样式** — 使用模板现有的字体、颜色、项目符号样式
3. **保留模板结构** — 除非必要，否则不要重新排列部分
4. **遵守模板间距** — 内容应适合指定区域而不溢出

目标是按设计填充模板，而不是重新设计它。

## 金融术语解释

### 1. TAM（总体可寻址市场）

**定义**：TAM（Total Addressable Market）是指公司产品或服务可以服务的最大市场规模。

**计算方法**：
- 基于市场规模和目标客户数量
- 考虑产品定价和市场渗透率

**重要性**：
- 评估市场机会大小
- 支持商业计划和融资
- 帮助战略规划

### 2. CAGR（复合年增长率）

**定义**：CAGR（Compound Annual Growth Rate）是衡量投资或指标在特定时期内平均年增长率的指标。

**计算公式**：
- CAGR = (终值 ÷ 初值)^(1/年数) - 1

**特点**：
- 假设增长率是恒定的
- 平滑了期间的波动
- 便于比较不同时期的增长率

### 3. 市场细分（Market Segmentation）

**定义**：市场细分是将整体市场划分为具有相似需求和特征的子群体的过程。

**细分维度**：
- **地理细分**：按地理位置划分
- **人口细分**：按年龄、性别、收入等划分
- **心理细分**：按生活方式、价值观划分
- **行为细分**：按购买行为、使用频率划分

**用途**：
- 精准定位目标客户
- 制定差异化营销策略
- 优化产品设计

### 4. 竞争格局（Competitive Landscape）

**定义**：竞争格局是指市场中竞争对手的分布和相对地位。

**分析维度**：
- 市场份额分布
- 竞争对手优势和劣势
- 市场进入壁垒
- 竞争策略

**工具**：
- SWOT分析
- 波特五力模型
- 竞争矩阵

### 5. 估值倍数（Valuation Multiples）

**定义**：估值倍数是用于评估公司价值的比率，通常基于财务指标。

**常用倍数**：
- **EV/Revenue**：企业价值与收入比
- **EV/EBITDA**：企业价值与EBITDA比
- **P/E**：市盈率

**应用场景**：
- 相对估值
- 投资决策
- 并购定价

### 6. 企业价值（Enterprise Value, EV）

**定义**：企业价值是衡量公司总价值的指标，考虑了股权和债务。

**计算公式**：
- EV = 市值 + 债务 - 现金及等价物

**组成部分**：
- **市值**：普通股的市场价值
- **债务**：短期和长期债务
- **现金**：现金及现金等价物

### 7. EBITDA

**定义**：EBITDA（Earnings Before Interest, Taxes, Depreciation, and Amortization）是指扣除利息、税项、折旧和摊销前的收益。

**计算公式**：
- EBITDA = 净利润 + 利息 + 税费 + 折旧 + 摊销

**用途**：
- 评估公司核心业务盈利能力
- 跨公司比较
- 财务建模

### 8. 同比增长率（YoY Growth）

**定义**：同比增长率是指与上一年相比的增长百分比。

**计算公式**：
- YoY增长率 = (本年值 - 上年值) / 上年值 × 100%

**优点**：
- 直观易懂
- 便于年度比较
- 反映季节性影响

### 9. 交易可比性分析（Transaction Comparables）

**定义**：交易可比性分析是通过比较类似交易来评估目标公司价值的方法。

**步骤**：
1. 识别类似交易
2. 收集交易数据（交易价值、估值倍数）
3. 计算汇总统计（均值、中位数）
4. 将倍数应用于目标公司

**应用场景**：
- 并购估值
- 投资决策
- 公平性意见

### 10. 财务预测（Financial Projections）

**定义**：财务预测是对未来财务表现的估计，通常包括收入、利润和现金流。

**组成部分**：
- 收入预测
- 成本预测
- 利润预测
- 现金流预测

**方法**：
- 自上而下方法
- 自下而上方法
- 基于市场的方法
