# 投资建议书

description: 为潜在客户创建专业的投资建议书。涵盖公司方法、建议配置、预期结果和费用结构。用于推介新客户或展示新投资策略。触发词包括"投资建议书"、"潜在客户演示"、"推介新客户"、"[客户]建议书"或"新客户演示"。

## 工作流程

### 步骤1：潜在客户背景

收集：
- **潜在客户姓名**和家庭详情
- **现状**：现有顾问？自主投资？什么促使这次会议？
- **资产**：估计管理资产（AUM）、账户类型、当前持仓（如分享）
- **目标**：退休、财富保值、增长、收入、教育、遗产
- **风险承受能力**：保守、温和、激进（或问卷得分）
- **约束条件**：ESG偏好、集中持股、流动性需求
- **费用敏感度**：他们现在支付什么？
- **竞争对手**：他们还在考虑谁？

### 步骤2：建议书结构

**I. 关于我们公司（1页）**
- 公司概览、历史、管理资产
- 投资哲学（用通俗英语）
- 团队简介（与此客户相关）
- 客户服务模式（我们多久会面一次，他们联系谁）

**II. 了解您的需求（1页）**
- 重述他们的目标和担忧 — 表明你倾听了
- 发现阶段识别的关键规划考虑
- 成功对他们来说是什么样子

**III. 建议投资策略（2-3页）**
- 带理由的建议资产配置
- 配置如何映射到他们的目标和风险承受能力
- 投资工具（ETF、共同基金、个股、另类投资）
- 税务 aware 策略（资产定位、税损收获）

建议配置：

| 资产类别 | 配置 | 工具 | 理由 |
|------------|-----------|---------|-----------|
| | | | |

**IV. 预期结果（1-2页）**
- 预测增长情境（保守、温和、乐观）
- 实现目标的蒙特卡洛概率
- 收入预测（如退休或收入焦点）
- 风险指标（最大回撤、波动率）
- 与当前投资组合比较（如已知）

**V. 费用结构（1页）**
- 咨询费时间表（如适用，分级）
- 底层基金费用
- 总全部成本估算
- 费用与行业平均比较
- 价值主张 — 他们为费用得到什么

**VI. 开始使用（1页）**
- 开户流程
- 资产转移时间表
- 过渡计划（如果从另一位顾问转移）
- 前90天 — 期待什么
- 必需文件和后续步骤

### 步骤3：定制

- 根据潜在客户调整语气（公司高管 vs. 小企业主 vs. 退休人员）
- 如果他们有集中持股头寸，直接处理
- 如果他们在将你与机器人顾问比较，强调规划和关系价值
- 如果他们价格敏感，从总价值和结果入手，不仅仅是费用

### 步骤4：输出

- 带公司品牌的PowerPoint演示文稿（12-15张幻灯片）
- PDF留后版本
- 用于跟进邮件的一页摘要

## 重要说明

- 建议书应感觉个性化，不是模板化 — 引用他们的具体情况
- 不要过度推销表现 — 设定现实预期并强调流程
- 始终包含免责声明（预测是假设的、过往表现等）
- 过渡计划很重要 — 客户害怕转换顾问的中断
- 48小时内跟进建议书和清晰的后续步骤
- 在向潜在客户展示前，合规必须审查

## Appendix: 金融背景知识

### 什么是投资建议书（Investment Proposal）？

投资建议书是财富管理公司向潜在客户展示的正式文件，目的是赢得新业务。它包含：
- 公司介绍和资质
- 对客户需求和目标的理解
- 建议的投资策略和配置
- 预期回报和风险
- 费用结构
- 下一步行动

### 关键术语解释

**潜在客户（Prospect）**：尚未成为客户但有可能转化为客户的个人或机构。

**管理资产（AUM，Assets Under Management）**：顾问为公司管理的资产总额。通常按管理资产的百分比收取费用（如1%/年）。

**投资哲学（Investment Philosophy）**：公司投资方法的核心信念，如"长期价值投资"、"被动指数化"、"战术资产配置"等。

**风险承受能力（Risk Tolerance）**：
- **保守**：优先资本保值，接受低回报
- **温和**：平衡增长和保值
- **激进**：优先增长，接受高波动

**资产配置（Asset Allocation）**：将投资资金分配到不同资产类别的策略。研究表明，资产配置决定了投资组合回报的90%以上。

**ETF（Exchange-Traded Fund，交易所交易基金）**：像股票一样在交易所交易的投资基金，通常费用低、流动性高。

**共同基金（Mutual Fund）**：由专业基金经理主动管理的投资基金，通常费用较高。

**另类投资（Alternatives）**：传统股票债券之外的投资，如房地产、大宗商品、对冲基金、私募股权。

**税务 aware 策略（Tax-Aware Strategy）**：
- **资产定位（Asset Location）**：将税务低效资产（如债券）放在 tax-advantaged 账户（如IRA），将税务高效资产（如ETF）放在应税账户
- **税损收获（Tax-Loss Harvesting）**：卖出亏损持仓实现损失，用于抵减资本利得税

**蒙特卡洛模拟（Monte Carlo Simulation）**：通过数千次随机模拟预测投资结果的概率方法。例如，"有85%的概率在退休时拥有足够资产"。

**最大回撤（Max Drawdown）**：从峰值到谷底的最大损失百分比。例如，投资组合从100万跌到70万，最大回撤30%。

**分级费用（Tiered Fee）**：根据管理资产规模分段收费，通常资产越多费率越低。例如：
- 首100万：1.0%
- 100-500万：0.8%
- 超过500万：0.6%

**机器人顾问（Ro-bo-advisor）**：自动化投资平台，用算法构建和管理的投资组合，通常费用较低（如0.25%/年），但缺乏人工顾问的个性化规划。

**尽职调查（Due Diligence）**：客户在选择顾问时进行的调查，包括：
- 审查ADV表格（披露费用、冲突、监管历史）
- 核实资质（CFP、CFA等）
- 了解投资哲学和策略
- 比较费用和服务

**ADV表格（Form ADV）**：美国投资顾问必须向SEC提交的披露文件，包含：
- 公司业务描述
- 费用结构
- 利益冲突披露
- 监管历史（如有任何处罚）

**CFP（Certified Financial Planner，注册财务规划师）**：财务规划领域的专业认证，要求考试、经验、伦理标准。

**CFA（Chartered Financial Analyst，特许金融分析师）**：投资和证券分析领域的专业认证，被称为"投资界MBA"。
