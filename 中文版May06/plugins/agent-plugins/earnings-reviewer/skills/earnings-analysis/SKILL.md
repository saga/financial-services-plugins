# 股权研究财报更新

创建专业的**财报更新报告**，按照机构标准（摩根大通、高盛、摩根士丹利格式）分析已覆盖公司的季度业绩。

**核心特征：**
- **篇幅**：8-12页
- **字数**：3,000-5,000字
- **表格**：1-3个汇总表（非全面详表）
- **图表**：8-12张
- **交付周期**：1-2天（财报发布后24-48小时内）
- **读者**：已熟悉该公司的客户
- **焦点**：新变化 — 超预期/不及预期、更新预测、对投资论点的影响
- **字体**：全文使用Times New Roman（除非用户另有指定）

## 使用场景

在以下情况下使用：
- "为[公司]创建2024年Q3财报更新"
- "分析[公司]的季度业绩"
- "[公司]财报后报告"
- "[公司]Q1/Q2/Q3/Q4更新"

**不使用场景：**
- 用户要求"首次覆盖报告" → 使用不同技能
- 用户要求"快讯"或"快速点评" → 不同格式
- 公司未被覆盖 → 需要先做首次覆盖

## 关键要求

### 1. 速度与及时性
- 在财报发布后24-48小时内发布
- 仅关注新信息
- 不要大量复述公司背景

### 2. 超预期/不及预期分析
- 首先说明公司是超预期还是不及预期
- 量化差异（如"收入超预期1.2亿美元，即3%"）
- 解释结果与预期不同的原因

### 3. 汇总格式
- 表格保持1-3个（仅汇总，非全面）
- 不需要完整的损益表/现金流量表/资产负债表（仅关键指标）
- 假设读者已阅读首次覆盖报告

### 4. 引用与来源标注 ⭐⭐⭐ 强制要求

**关键**：用特定来源和可点击超链接正确引用所有数据。

**在每个图表和表格中包含带可点击链接的具体引用：**

```
来源：2024年Q3 10-Q文件，2024年11月8日提交；公司财报新闻稿
        [将"10-Q"超链接至：https://www.sec.gov/cgi-bin/viewer?accession=...]
        [将"财报新闻稿"超链接至：https://investor.company.com/news/q3-2024]
```

**Word文档中超链接应如何显示：**
- 文档名称显示为蓝色、带下划线的可点击链接
- 读者可以按Ctrl+点击直接打开来源
- 不是纯文本URL — 是带显示文本的格式化工链接

**要求的来源列表：**

在每个财报更新中引用：
- ✅ 财报新闻稿（含日期和URL）
- ✅ 10-Q申报文件（含申报日期和EDGAR链接）
- ✅ 财报电话会议记录（含日期）
- ✅ 投资者演示/补充材料（如有）
- ✅ 一致预期来源（Bloomberg/FactSet等，含日期）
- ✅ 前期指引（来自上季度材料）

**带可点击超链接的参考部分：**

在报告末尾包含"来源"部分：

```
来源与参考

财报材料（2024年Q3）：
• 财报新闻稿（2024年11月7日）
  [将整行超链接至：https://investor.company.com/news/q3-2024-earnings]

• 10-Q表格（2024年11月8日提交）
  [超链接至：https://www.sec.gov/cgi-bin/viewer?accession=...]

• 财报电话会议记录（2024年11月7日）
  [超链接至：https://seekingalpha.com/article/...]

• 投资者演示（2024年11月7日）
  [超链接至：https://investor.company.com/presentations/q3-2024.pdf]
```

**验证清单：**
- [ ] 每个数字都有含具体文件和日期的来源
- [ ] 每个表格都有文件引用的来源
- [ ] 超预期/不及预期分析引用含日期的一致预期来源
- [ ] 指引变更引用当前和前期的指引来源
- [ ] 关键统计有脚注
- [ ] 来源部分列出所有带URL的材料
- [ ] 所有URL都是可点击的超链接（非纯文本）
- [ ] 所有SEC申报文件都超链接至EDGAR查看器

### 5. 更新后的预测
- 根据结果更新未来预测
- 清晰展示旧预测 vs. 新预测
- 解释变更内容及原因

## 高层工作流

财报更新流程分为5个阶段：

### 阶段1：数据收集（30-60分钟）

**🚨🚨🚨 关键：训练数据已过时 🚨🚨🚨**

**开始前 — 按顺序完成这4步：**
1. **检查今天日期** — 写下当前日期
2. **搜索最新** — 使用网页搜索："[公司] 最新财报结果"
3. **验证日期** — 确认财报发布在最近3个月内
4. **检查记录日期** — 验证电话会议记录日期与发布日期匹配

**常见错误**：使用训练数据中的过时财报电话会议，而非搜索最新信息。

**要求：**
- ✅ 搜索最新财报 — 不要依赖训练数据
- ✅ 写下今天日期和找到的发布日期
- ✅ 验证发布日期在今天3个月内
- ✅ 验证电话会议记录日期与发布日期匹配
- ✅ 如果日期不匹配或超过3个月，重新搜索

**详见 [references/workflow.md](references/workflow.md)** 获取详细搜索程序和验证步骤。

## Appendix: 金融背景知识

### 什么是财报更新（Earnings Update）？

财报更新是股权研究分析师在上市公司发布季度财报后撰写的简短研究报告。与全面的首次覆盖报告不同，财报更新聚焦于：
- 实际业绩与市场预期的对比（超预期/不及预期）
- 关键指标的变化
- 对投资论点的影响
- 预测和估值目标的调整

### 关键术语解释

**超预期/不及预期（Beat/Miss）**：公司实际业绩与市场一致预期（Consensus）的比较。例如，预期EPS为1.00美元，实际为1.05美元，则为"超预期5%"。

**一致预期（Consensus）**：华尔街分析师对公司业绩预测的平均值，通常来自Bloomberg、FactSet、Refinitiv等数据提供商。

**10-Q**：美国上市公司向SEC提交的季度报告，包含未经审计的财务报表和管理层讨论。

**EDGAR**：SEC的电子数据收集、分析和检索系统，所有公开申报文件都可在https://www.sec.gov/edgar 查询。

**电话会议记录（Call Transcript）**：公司财报发布后与分析师和投资者的问答会议记录，包含管理层对业绩的讨论和未来指引。

**指引（Guidance）**：管理层提供的未来业绩预期，通常包括收入增长、利润率等关键指标的预测范围。

**投资论点（Investment Thesis）**：分析师支持买入/持有/卖出建议的核心逻辑，包括公司竞争优势、增长驱动因素、估值合理性等。

**目价（Price Target）**：分析师设定的未来12个月股价目标，通常基于估值模型（如DCF、倍数法）。

**评级（Rating）**：分析师对股票的建议，通常为：
- **买入/增持（Buy/Overweight）**：预期跑赢大盘
- **持有/中性（Hold/Neutral）**：预期与大盘持平
- **卖出/减持（Sell/Underweight）**：预期跑输大盘

**NTM（Next Twelve Months，未来12个月）**：前瞻性的财务指标，如NTM EPS、NTM EBITDA。

**分部（Segment）**：大公司按业务线或地理区域划分的运营部门，如苹果公司的iPhone分部、服务分部等。

**调整后（Adjusted）vs. GAAP**：
- **GAAP（公认会计原则）**：按照会计准则编制的财务数据
- **调整后（Non-GAAP/Adjusted）**：剔除一次性项目（如重组费用、诉讼费用）后的数据，更能反映持续经营表现
