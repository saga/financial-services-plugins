# 股票研究财报更新

按照机构标准（摩根大通、高盛、摩根士丹利格式），为已覆盖公司创建专业的**财报更新报告**，分析季度业绩。

**主要特点：**
- **篇幅**：8-12页
- **字数**：3,000-5,000字
- **表格**：1-3个汇总表（非全面性）
- **图表**：8-12个图表
- **周转时间**：1-2天（财报发布后24-48小时内）
- **受众**：已经熟悉该公司的客户
- **重点**：新内容 — 业绩超预期/不及预期、更新预测、投资主题影响
- **字体**：全文使用Times New Roman（除非用户另有指定）

## 使用场景

当用户请求时使用：
- "为[公司]2024年Q3创建财报更新"
- "分析[公司]的季度业绩"
- "[公司]的财报后报告"
- "[公司]的Q1/Q2/Q3/Q4更新"

**不适用场景：**
- 用户请求"首次覆盖报告" → 使用不同技能
- 用户请求"快讯"或"速评" → 使用不同格式
- 公司尚未被覆盖 → 需要先进行首次覆盖

## 关键要求

### 1. 速度与时效性
- 在财报发布后24-48小时内发布
- 仅关注新信息
- 不要过多重复公司背景

### 2. 业绩超预期/不及预期分析
- 首先说明公司业绩是超出还是低于预期
- 量化差异（例如："收入超出预期1.2亿美元或3%"）
- 解释业绩与预期不同的原因

### 3. 摘要格式
- 表格限制在1-3个（仅摘要，非全面性）
- 不包含完整的损益表/现金流量表/资产负债表（仅关键指标）
- 假设读者已阅读首次覆盖报告

### 4. 引用与来源归属 ⭐⭐⭐ 强制性

**至关重要**：正确引用所有数据，注明具体来源并提供可点击的超链接。

**在每个图表和表格中包含具体引用（带可点击链接）：**

```
来源：2024年Q3 10-Q表格（2024年11月8日提交）；公司财报发布
        [将"10-Q"超链接到：https://www.sec.gov/cgi-bin/viewer?accession=...]
        [将"财报发布"超链接到：https://investor.company.com/news/q3-2024]
```

**Word中超链接的显示方式：**
- 文档名称显示为蓝色带下划线的可点击链接
- 读者可以Ctrl+点击直接打开来源
- 不是纯文本URL - 是带有显示文本的格式化超链接

**必需来源列表：**

每份财报更新中必须引用：
- ✅ 财报发布（带日期和URL）
- ✅ 10-Q表格（带提交日期和EDGAR链接）
- ✅ 财报电话会议记录（带日期）
- ✅ 投资者演示/补充材料（如可用）
- ✅ 一致预期来源（Bloomberg/FactSet等，带日期）
- ✅ 先前指引（来自上一季度的材料）

**带可点击超链接的参考文献部分：**

在报告末尾包含"来源"部分：

```
来源与参考文献

财报材料（2024年Q3）：
• 财报发布（2024年11月7日）
  [将整行超链接到：https://investor.company.com/news/q3-2024-earnings]

• 10-Q表格（2024年11月8日提交）
  [超链接到：https://www.sec.gov/cgi-bin/viewer?accession=...]

• 财报电话会议记录（2024年11月7日）
  [超链接到：https://seekingalpha.com/article/...]

• 投资者演示（2024年11月7日）
  [超链接到：https://investor.company.com/presentations/q3-2024.pdf]
```

**验证清单：**
- [ ] 每个图表都有来源，包含具体文档和日期
- [ ] 每个表格都有来源，包含文档引用
- [ ] 业绩超预期/不及预期分析引用一致预期来源和日期
- [ ] 指引变更引用当前和先前指引来源
- [ ] 关键统计数据有脚注
- [ ] 来源部分列出所有材料和URL
- [ ] 所有URL都是可点击的超链接（非纯文本）
- [ ] 所有SEC文件都超链接到EDGAR查看器

### 5. 更新预测
- 根据业绩更新前瞻性预测
- 清晰显示新旧预测对比
- 解释变化内容和原因

## 高级工作流程

财报更新流程分为5个阶段：

### 阶段1：数据收集（30-60分钟）

**🚨🚨🚨 重要提示：训练数据已过时 🚨🚨🚨**

**开始前 - 按顺序完成以下4个步骤：**
1. **检查今天的日期** - 写下当前日期
2. **搜索最新信息** - 使用网络搜索："[公司]最新财报业绩"
3. **验证日期** - 确认财报发布在过去3个月内
4. **检查会议记录日期** - 验证会议记录日期与发布日期匹配

**常见错误**：使用训练数据中的过时财报电话会议，而不是搜索最新信息。

**要求：**
- ✅ 搜索最新财报 - 不要依赖训练数据
- ✅ 写下今天的日期和找到的发布日期
- ✅ 验证发布日期在今天之前的3个月内
- ✅ 验证会议记录日期与发布日期匹配
- ✅ 如果日期不匹配或已过期（>3个月），重新搜索

**详见 [references/workflow.md](references/workflow.md)** 获取详细搜索程序和验证步骤。

### 阶段2：分析（2-3小时）
- 每个关键指标的业绩超预期/不及预期分析
- 部门/地区/产品细分分析
- 利润率和指引分析
- 更新财务模型和预测

**详见 [references/workflow.md](references/workflow.md)** 获取详细分析框架。

### 阶段3：图表生成（1-2小时）
创建8-12个图表，重点关注季度趋势和新内容：
- 季度收入进展
- 季度EPS进展
- 季度利润率趋势
- 按部门/地区划分的收入
- 关键运营指标
- 业绩超预期/不及预期汇总
- 预测修正
- 估值图表

**详见 [references/workflow.md](references/workflow.md)** 获取图表规格。

### 阶段4：报告创建（2-3小时）
创建8-12页的DOCX报告，具有特定结构。

**详见 [references/report-structure.md](references/report-structure.md)** 获取完整的逐页模板和格式要求。

**高级结构：**
- 第1页：财报摘要，包含评级和目标价格
- 第2-3页：详细业绩分析
- 第4-5页：关键指标与指引
- 第6-7页：更新后的投资主题
- 第8-10页：估值与预测

### 阶段5：质量检查与交付（30分钟）
在交付前验证内容、格式、准确性和时效性。

## 金融术语和知识解释

### 1. 股票研究报告 (Equity Research Report)

股票研究报告是分析师对特定上市公司的分析和投资建议。主要类型包括：
- **首次覆盖报告 (Initiation Report)**：首次对某只股票进行全面分析
- **财报更新报告 (Earnings Update)**：季度财报发布后的快速分析
- **深度报告 (Deep Dive)**：对特定主题的深入分析

### 2. 业绩超预期/不及预期 (Beat/Miss)

- **Beat（超预期）**：公司实际业绩好于市场一致预期
- **Miss（不及预期）**：公司实际业绩差于市场一致预期
- **In-line（符合预期）**：公司实际业绩与市场一致预期相符

### 3. EDGAR系统

EDGAR（Electronic Data Gathering, Analysis, and Retrieval）是美国SEC的电子数据收集、分析和检索系统。所有美国上市公司必须通过EDGAR提交法定文件。

### 4. 投资评级 (Investment Rating)

研究报告通常包含对股票的投资评级，常见评级包括：
- **买入 (Buy)**：建议买入该股票
- **持有 (Hold)**：建议保持现有仓位
- **卖出 (Sell)**：建议卖出该股票
- **跑赢大盘 (Outperform)**：预期表现优于市场
- **中性 (Neutral)**：预期表现与市场持平
- **跑输大盘 (Underperform)**：预期表现差于市场

### 5. 目标价格 (Price Target)

目标价格是分析师对股票在未来12-18个月内合理价值的估计。它是基于估值模型计算得出的。

### 6. 关键指标 (Key Metrics)

关键指标是评估公司财务健康状况的重要数据点，包括：
- **收入 (Revenue)**：公司销售额
- **每股收益 (EPS)**：每股普通股收益
- **毛利率 (Gross Margin)**：毛利润占收入百分比
- **EBITDA**：息税折旧摊销前利润
- **净利润 (Net Income)**：税后利润

### 7. 投资者演示 (Investor Presentation)

投资者演示是公司管理层在财报发布后向投资者展示的幻灯片演示，通常包含：
- 财务亮点
- 业务更新
- 未来展望
- 关键指标可视化

### 8. 估值方法 (Valuation Methods)

常用的股票估值方法包括：
- **折现现金流模型 (DCF)**：通过折现未来现金流计算公司价值
- **可比公司分析 (Comps)**：与同行业公司进行比较
- **交易倍数法 (Transaction Multiples)**：基于类似并购交易的估值

### 9. 前瞻性预测 (Forward Estimates)

前瞻性预测是分析师对公司未来财务业绩的预测，通常包括：
- 未来1-2年的收入和盈利预测
- 关键财务指标预测
- 基于当前信息的合理假设

### 10. 投资主题 (Investment Thesis)

投资主题是分析师对某只股票的核心投资观点，解释为什么认为该股票会表现良好或不佳，通常包括：
- 核心投资论点
- 关键催化剂
- 风险因素
- 目标价格推导
