# 晨会笔记

description: 起草简洁的晨会笔记，总结隔夜动态、交易构想和覆盖股票的关键事件。专为早7点晨会设计 — 紧凑、有观点、可操作。触发词包括"晨会笔记"、"晨会"、"隔夜发生了什么"、"交易构想"、"晨会准备"或"每日笔记"。

## 工作流程

### 步骤1：隔夜动态

扫描覆盖范围内的相关事件：

**财报与指引**
- 是否有覆盖公司在隔夜或盘前发布财报？
- 财报意外（收入、EPS、关键指标超预期/不及预期）
- 指引变更（上调、下调、维持）

**新闻与事件**
- 并购公告或传闻
- 管理层变更
- 产品发布或监管决定
- 竞争对手的分析师上调/下调评级
- 影响行业的宏观数据或政策变化

**市场背景**
- 隔夜期货 / 盘前走势
- 行业ETF表现
- 相关大宗商品或货币走势
- 今日关键经济数据发布

### 步骤2：晨会笔记格式

保持紧凑 — 晨会笔记应在2分钟内可读完：

---

**[日期] 晨会笔记 — [分析师姓名]**
**[行业覆盖]**

**头条呼吁：[PM最需要听到的一件事]**
- 2-3句话说明关键动态及其重要性
- 股票影响：目价、评级重申/变更

**隔夜/盘前动态**
- [公司A]：财报/新闻的一句话摘要 + 我们的观点
- [公司B]：一句话摘要 + 我们的观点
- [行业/宏观]：全行业相关动态

**今日关键事件**
- [时间]：[公司]财报电话会议
- [时间]：经济数据发布（预期 vs. 我们的观点）
- [时间]：会议或投资者日

**交易构想**（如有）
- [做多/做空] [公司]：1-2句话投资论点 + 催化剂
- 风险：什么情况会使这个构想错误

---

### 步骤3：财报快速解读

如果覆盖公司发布了财报，提供快速反应：

| 指标 | 一致预期 | 实际值 | 超预期/不及预期 |
|------|----------|--------|------------------|
| 收入 | | | |
| EPS | | | |
| [关键指标] | | | |
| 指引 | | | |

**我们的观点**：2-3句话 — 这对股票是利好还是利空？是否改变我们的投资论点？

**行动**：维持 / 上调 / 下调评级？调整目价？

### 步骤4：输出

- 用于邮件/Slack分发的Markdown文本
- 如需正式分发，可提供Word文档
- 控制在1页以内 — PM和交易员不会读更多

## 重要说明

- 要有观点 — 仅总结新闻而无观点的晨会笔记毫无用处
- 最重要的事情放在开头 — 不要埋没头条
- "无新闻"也是有效的晨会笔记 — 写明"隔夜无重大事件，维持仓位"
- 区分可操作事件（财报、并购）和噪音（次要分析师笔记、非事件）
- 为你的观点加上时间戳 — 如果你在早上6点撰写，注明盘前情况可能在开盘前变化
- 如果你错了，在次日晨会笔记中承认 — 可信度比每次都对更重要

## Appendix: 金融背景知识

### 什么是晨会（Morning Meeting）？

投资银行、对冲基金、资产管理公司的分析师和交易员每天早晨（通常7:00-7:30）举行的简短会议，讨论：
- 隔夜市场和新闻动态
- 当日重要事件和经济数据
- 交易机会和投资建议

### 关键术语解释

**PM（Portfolio Manager，投资组合经理）**：负责基金投资决策和组合管理的专业人士。晨会笔记主要服务于PM。

**盘前（Pre-market）**：美国股市正式开盘（9:30 AM ET）前的交易时段，通常从4:00 AM开始。盘前交易量和波动性通常较大。

**隔夜期货（Overnight Futures）**：在亚洲和欧洲交易时段交易的美国股指期货（如标普500期货），预示美股开盘方向。

**行业ETF（Sector ETF）**：跟踪特定行业表现的交易型开放式指数基金，如XLK（科技）、XLF（金融）、XLE（能源）等。

**上调/下调评级（Upgrade/Downgrade）**：分析师将股票评级从持有上调至买入，或从买入下调至持有/卖出。

**做多/做空（Long/Short）**：
- **做多**：买入股票，期待价格上涨获利
- **做空**：借入股票卖出，期待价格下跌后低价买回归还，赚取差价

**催化剂（Catalyst）**：推动股价变动的事件或因素，如财报发布、新产品发布、并购传闻等。

**一致预期（Consensus）**：华尔街分析师对公司业绩预测的平均值。

** whisper number（耳语数字）**：市场传闻的、非正式的业绩预期，通常比官方一致预期更激进。

**期权隐含波动（Options-implied Move）**：从期权价格反推的市场预期股价变动幅度。

**可信度（Credibility）**：分析师在PM和交易员中的声誉，建立在预测准确性和坦率承认错误的态度上。
