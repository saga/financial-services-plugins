# 晨会纪要

description: 起草简洁的晨会纪要，总结隔夜动态、交易思路和覆盖股票的关键事件。专为早上7点的晨会格式设计 — 简洁、有观点、可操作。触发词包括"morning note"、"morning meeting"、"what happened overnight"、"trade idea"、"morning call prep"或"daily note"。

## 工作流程

### 步骤1：隔夜动态

扫描覆盖范围内的相关事件：

**财报与指引**
- 是否有覆盖公司在隔夜或盘前发布财报？
- 财报惊喜（收入、EPS、关键指标超预期/不及预期）
- 指引变化（上调、下调、维持）

**新闻与事件**
- 并购公告或传闻
- 管理层变动
- 产品发布或监管决定
- 竞争对手分析师的评级上调/下调
- 影响行业的宏观数据或政策变化

**市场背景**
- 隔夜期货 / 盘前走势
- 行业ETF表现
- 相关大宗商品或货币走势
- 今日关键经济数据发布

### 步骤2：晨会纪要格式

保持简洁 — 晨会纪要应在2分钟内可读：

---

**[日期] 晨会纪要 — [分析师姓名]**
**[行业覆盖]**

**头条推荐：[标题 — 投资组合经理需要知道的一件事]**
- 2-3句话说明关键动态及其重要性
- 股票影响：目标价格、评级重申/变更

**隔夜/盘前动态**
- [公司A]：财报/新闻摘要 + 我们的观点
- [公司B]：摘要 + 我们的观点
- [行业/宏观]：相关行业动态

**今日关键事件**
- [时间]：[公司]财报电话会议
- [时间]：经济数据发布（预期 vs. 我们的观点）
- [时间]：会议或投资者日

**交易思路**（如有）
- [多头/空头] [公司]：1-2句投资主题 + 催化剂
- 风险：什么情况会证明这是错误的

---

### 步骤3：财报快速点评

如果覆盖公司发布了财报，提供快速反应：

| 指标 | 一致预期 | 实际值 | 超预期/不及预期 |
|------|----------|--------|-----------------|
| 收入 | | | |
| EPS | | | |
| [关键指标] | | | |
| 指引 | | | |

**我们的观点**：2-3句话 — 这对股票是好是坏？是否改变我们的投资主题？

**行动**：维持 / 上调 / 下调评级？调整目标价格？

### 步骤4：输出

- Markdown文本用于邮件/Slack分发
- Word文档（如需要正式分发）
- 最多1页 — 投资组合经理和交易员不会阅读更多内容

## 重要注意事项

- 要有观点 — 只总结新闻而没有观点的晨会纪要毫无用处
- 以最重要的事情开头 — 不要隐藏标题
- "无新闻"是有效的晨会纪要 — 可以说"隔夜无重大消息，维持仓位"
- 区分可操作事件（财报、并购）和噪音（次要分析师报告、非事件）
- 为你的观点添加时间戳 — 如果在早上6点撰写，注意盘前可能在开盘前变化
- 如果错了，在下一份晨会纪要中承认 — 可信度比每次都正确更重要

## 金融术语和知识解释

### 1. 晨会纪要 (Morning Note)

晨会纪要是金融机构每天早上发布的简报，总结：
- 隔夜市场动态
- 重要新闻和事件
- 交易思路和投资建议

通常在早上7-8点发布，供投资组合经理和交易员快速了解市场情况。

### 2. 盘前交易 (Pre-market Trading)

盘前交易是在正常交易时段之前进行的股票交易，通常在早上4:00-9:30（美国东部时间）进行。盘前价格变动可以反映市场对隔夜新闻的反应。

### 3. 投资组合经理 (Portfolio Manager, PM)

投资组合经理负责管理投资组合，做出买入和卖出决策，以实现投资目标。

### 4. 交易思路 (Trade Idea)

交易思路是基于分析的具体投资建议，通常包括：
- 方向（多头/空头）
- 目标公司
- 投资主题
- 催化剂
- 风险因素

### 5. 催化剂 (Catalyst)

催化剂是指可能导致股票价格变动的事件或因素，如：
- 财报发布
- 产品发布
- 并购公告
- 监管决定

### 6. 评级上调/下调 (Upgrade/Downgrade)

分析师对股票投资建议的变更：
- **上调(Upgrade)**：从持有上调到买入，或从卖出上调到持有
- **下调(Downgrade)**：从买入下调到持有，或从持有下调到卖出

### 7. 行业ETF (Sector ETF)

行业ETF是跟踪特定行业指数的交易所交易基金，如科技ETF、金融ETF等。

### 8. 宏观数据 (Macro Data)

宏观数据是指影响整体经济的统计数据，如：
- GDP增长
- 失业率
- 通货膨胀率
- 利率

### 9. 一致预期 (Consensus Estimates)

一致预期是所有分析师对某一指标的平均预测，是衡量公司业绩的基准。

### 10. 投资主题 (Investment Thesis)

投资主题是分析师对公司的核心投资观点，解释为什么认为公司股票值得投资或避免。
