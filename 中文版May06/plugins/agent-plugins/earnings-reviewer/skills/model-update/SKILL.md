# 模型更新

description: 使用新数据更新财务模型 — 季度财报、管理层指引、宏观变化或修订假设。调整预测、重新计算估值并标记重大变化。在财报发布后、指引更新后或需要刷新假设时使用。触发词包括"update model"、"plug earnings"、"refresh estimates"、"update numbers for [company]"、"new guidance"或"revise estimates"。

## 工作流程

### 步骤1：确定变更内容

确定更新触发因素：
- **财报发布**：插入新的季度实际数据
- **指引变更**：公司更新了未来展望
- **预测修订**：分析师根据新数据更改假设
- **宏观更新**：利率、汇率、大宗商品价格变化
- **事件驱动**：并购、重组、新产品、管理层变动

### 步骤2：插入新数据

使用报告的实际数据更新模型：

|------|----------|--------|------|------|
| 收入 | | | | |
| 毛利率 | | | | |
| 运营费用 | | | | |
| EBITDA | | | | |
| EPS | | | | |

**部门明细**（如适用）：
- 更新每个部门的收入和利润率
- 注意任何部门组合变化

**资产负债表/现金流更新**：
- 现金和债务余额
- 股份数量（回购、稀释）
- 实际资本支出 vs. 预测
- 营运资金变化

### 步骤3：修订未来预测

根据新数据调整未来预测：

| | 旧财年预测 | 新财年预测 | 变化 | 旧下一财年 | 新下一财年 | 变化 |
|---|------------|------------|------|------------|------------|------|
| 收入 | | | | | | |

**关键假设变化：**
- 您正在更改哪些假设以及原因？
- 收入增长率：旧 → 新（原因）
- 利润率假设：旧 → 新（原因）
- 任何新项目（重组费用、一次性收益等）

### 步骤4：估值影响

使用更新的预测重新计算估值：

| 估值方法 | 先前值 | 更新值 | 变化 |
|----------|--------|--------|------|

### 步骤5：总结与行动

**预测变化总结：**
- 一段说明：什么变了、为什么变以及对股票意味着什么
- 这是改变投资主题的事件还是噪音？

**评级/目标价格：**
- 维持还是改变评级？
- 新目标价格（如变更）及方法论
- 当前价格的上涨/下跌空间

### 步骤6：输出

- 更新后的Excel模型（如果用户提供现有模型）
- 预测变化摘要（markdown或Word）
- 更新后的目标价格推导

## 重要注意事项

- 在向前预测之前，始终将您的预测与公司报告的数字进行核对
- 注意任何非经常性项目以及您的预测是GAAP还是调整后
- 跟踪您的预测修订历史 — 这显示了您的分析进展
- 如果本季度数据噪音较大，在预测变化中分离信号与噪音
- 更新后检查一致预期 — 您修订后的预测与市场相比如何？
- 股份数量很重要 — 股权激励、可转债或回购造成的稀释会对EPS产生重大影响

## 金融术语和知识解释

### 1. 财务模型更新 (Financial Model Update)

财务模型更新是指根据新信息调整预测和假设的过程，常见触发因素包括：
- 季度财报发布
- 管理层指引变更
- 宏观经济变化
- 公司特定事件（并购、重组等）

### 2. 实际值 vs 预测值 (Actuals vs Estimates)

- **实际值(Actuals)**：公司报告的真实财务业绩
- **预测值(Estimates)**：分析师对未来业绩的预期

两者之间的差异称为"惊喜"（surprise），可以是正面的（超预期）或负面的（不及预期）。

### 3. NTM (Next Twelve Months)

NTM指未来12个月，是计算估值倍数时常用的时间框架，例如：
- NTM EPS：未来12个月预期每股收益
- NTM EBITDA：未来12个月预期EBITDA

### 4. 非经常性项目 (Non-recurring Items)

非经常性项目是不寻常或不频繁发生的收入或费用，例如：
- 重组费用
- 资产减值
- 一次性收益或损失

分析师通常会剔除这些项目来计算调整后收益。

### 5. GAAP vs 调整后 (GAAP vs Adjusted)

- **GAAP收益**：按照公认会计原则计算的收益
- **调整后收益**：剔除一次性或非核心项目后的收益

调整后收益更能反映公司核心业务的真实表现。

### 6. 一致预期 (Consensus Estimates)

一致预期是华尔街所有分析师预测的平均值，是衡量公司业绩表现的基准。

### 7. 稀释 (Dilution)

稀释是指由于发行新股导致每股收益(EPS)下降的现象，常见原因包括：
- 股权激励计划
- 可转债转股
- 新股发行

### 8. 股份回购 (Share Buybacks)

股份回购是公司回购自己股票的行为，会减少流通股数量，通常会提高EPS。

### 9. 营运资金 (Working Capital)

营运资金 = 流动资产 - 流动负债，反映公司日常运营所需的资金。

### 10. DCF估值 (DCF Valuation)

DCF（折现现金流）是一种估值方法，通过将未来现金流折现到当前价值来计算公司的内在价值。
