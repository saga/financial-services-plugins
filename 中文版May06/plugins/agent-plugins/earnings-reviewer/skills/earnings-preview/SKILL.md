# 财报预览

description: 构建财报前分析，包含预测模型、情境框架和需关注的关键指标。在公司发布季度财报前使用，用于准备仓位笔记、设定牛市/熊市情境，并识别将推动股价变动的因素。触发词包括"财报预览"、"[公司]财报需关注什么"、"财报前"、"财报准备"或"预览[公司]Q[X]"。

## 工作流程

### 步骤1：收集背景

- 确定公司和报告季度
- 通过网页搜索提取一致预期（收入、EPS、关键分部指标）
- 查找财报发布日期和时间（盘前 vs. 盘后）
- 回顾公司上季度财报电话会议中的任何指引或评论

### 步骤2：关键指标框架

构建针对该公司的"需关注内容"框架：

**财务指标：**
- 收入 vs. 一致预期（总计和按分部）
- EPS vs. 一致预期
- 利润率（毛利率、运营利润率、净利率）— 扩张还是收缩？
- 自由现金流
- 前瞻指引 vs. 一致预期

**运营指标**（行业特定）：
- 科技/SaaS：ARR、净留存率、RPO、客户数量
- 零售：同店销售额、客流量、客单价
- 工业：积压订单、订单出货比、价格 vs. 销量
- 金融：净息差、信贷质量、贷款增长、手续费收入
- 医疗：处方量、患者数量、研发管线更新

### 步骤3：情境分析

构建3种情境及其对股价的影响：

| 情境 | 收入 | EPS | 关键驱动因素 | 股价反应 |
|------|------|-----|------------|----------------|
| 牛市 | | | | |
| 基准 | | | | |
| 熊市 | | | | |

对每种情境：
- 运营上需要发生什么？
- 管理层评论会释放什么信号？
- 历史背景 — 类似财报后股价如何变动？

### 步骤4：催化剂清单

识别将决定股价反应的3-5个因素：

1. [指标] vs. [一致预期/耳语数字] — 为何重要
2. [指引项目] — 买方期待听到什么
3. [叙事转变] — 任何战略变化、并购、重组

### 步骤5：输出

一页财报预览，包含：
- 公司、季度、财报日期
- 一致预期表格
- 需关注的关键指标（按重要性排序）
- 牛市/基准/熊市情境表
- 催化剂清单
- 交易设置：近期股价表现、期权隐含波动

## 重要说明

- 一致预期会变化 — 始终注明预期的来源和日期
- 买方调查的"耳语数字"通常比发布的一致预期更相关
- 历史财报反应有助于校准预期（搜索"[公司]财报反应历史"）
- 期权隐含波动告诉你市场预期 — 与你的情境对比

## Appendix: 金融背景知识

### 什么是财报预览（Earnings Preview）？

在上市公司发布季度财报前，分析师撰写的研究报告，帮助投资者：
- 了解市场预期（一致预期）
- 准备不同情境（牛市/基准/熊市）
- 识别关键催化剂
- 设定交易策略

### 关键术语解释

**盘前 vs. 盘后（Pre-market vs. After-hours）**：
- **盘前**：美股正式开盘（9:30 AM ET）前的交易时段
- **盘后**：美股收盘（4:00 PM ET）后的交易时段
- 大多数公司选择在盘后发布财报，以便第二天市场充分消化信息

**ARR（Annual Recurring Revenue，年度经常性收入）**：SaaS（软件即服务）公司每年从订阅客户获得的稳定收入，是衡量业务健康度的关键指标。

**净留存率（Net Retention Rate）**：衡量现有客户收入留存和扩张的百分比。>100%表示客户不仅在续约，还在增加消费。

**RPO（Remaining Performance Obligations，剩余履约义务）**：公司已签约但尚未确认收入的合同金额，是未来收入的先行指标。

**同店销售额（Same-store Sales / Comparable Sales）**：开业至少一年的门店销售额变化，剔除新店开业的扭曲影响，反映零售业务真实增长。

**积压订单（Backlog）**：已签约但尚未交付或确认收入的订单金额，是未来收入的另一个指标。

**订单出货比（Book-to-bill Ratio）**：新订单金额除以同期收入。>1表示订单增长快于交付，未来收入将增长。

**净息差（Net Interest Margin, NIM）**：银行净利息收入除以盈利资产，衡量银行核心盈利能力。

**信贷质量（Credit Quality）**：银行贷款组合的风险状况，通常用不良贷款率、拨备覆盖率等指标衡量。

**耳语数字（Whisper Number）**：市场传闻的、非正式的业绩预期，通常比官方一致预期更激进，更能反映买方真实预期。

**期权隐含波动（Options-implied Move）**：从期权价格反推的市场预期股价变动幅度。例如，期权隐含波动为±5%，意味着市场预计财报后股价将在±5%范围内波动。

**情境分析（Scenario Analysis）**：评估不同假设情况下（牛市/基准/熊市）的结果，帮助投资者为各种可能性做好准备。

**买方（Buy-side）vs. 卖方（Sell-side）**：
- **买方**：共同基金、对冲基金、养老金等实际进行投资的机构
- **卖方**：投资银行、券商等提供研究、交易服务的中间商

**一致预期（Consensus）**：华尔街分析师对公司业绩预测的平均值，通常来自Bloomberg、FactSet、Refinitiv等。

**EPS（Earnings Per Share，每股收益）**：公司净利润除以流通股数，是衡量盈利能力的核心指标。

**指引（Guidance）**：管理层提供的未来业绩预期范围，如"Q4收入增长预计5-7%"。

**催化剂（Catalyst）**：推动股价变动的事件或因素，如财报发布、新产品发布、并购传闻等。
