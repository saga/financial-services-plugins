# 财报预览

description: 构建财报前分析，包含预测模型、情景框架和需要关注的关键指标。在公司发布季度财报前使用，准备定位笔记、设置多头/空头情景，并确定哪些因素会影响股价。触发词包括"earnings preview"、"what to watch for [company] earnings"、"pre-earnings"、"earnings setup"或"preview Q[X] for [company]"。

## 工作流程

### 步骤1：收集背景信息

- 确定公司和报告季度
- 通过网络搜索获取一致预期（收入、EPS、关键部门指标）
- 找到财报日期和时间（盘前 vs. 盘后）
- 回顾公司上一季度财报电话会议的任何指引或评论

### 步骤2：关键指标框架

构建针对特定公司的"需要关注"框架：

**财务指标：**
- 收入 vs. 一致预期（总计和按部门）
- EPS vs. 一致预期
- 利润率（毛利率、营业利润率、净利率）—— 扩大还是收缩？
- 自由现金流
- 未来指引 vs. 一致预期

**运营指标**（行业特定）：
- 科技/SaaS：ARR、净留存率、RPO、客户数量
- 零售：同店销售、客流量、客单价
- 工业：订单积压、接单出货比、价格 vs. 销量
- 金融：净息差、信贷质量、贷款增长、手续费收入
- 医疗：处方量、患者数量、管线更新

### 步骤3：情景分析

构建3种情景及股价影响：

| 情景 | 收入 | EPS | 关键驱动因素 | 股价反应 |
|------|------|-----|--------------|----------|
| 多头 | | | | |
| 基准 | | | | |
| 空头 | | | | |

对于每个情景：
- 运营层面需要发生什么
- 管理层评论会发出什么信号
- 历史背景 — 股票在类似业绩下如何波动？

### 步骤4：催化剂清单

确定决定股价反应的3-5件事：

1. [指标] vs. [一致预期/耳语数字] — 为什么重要
2. [指引项目] — 买方期望听到什么
3. [叙事转变] — 任何战略变化、并购、重组

### 步骤5：输出

单页财报预览，包含：
- 公司、季度、财报日期
- 一致预期表
- 需要关注的关键指标（按重要性排序）
- 多头/基准/空头情景表
- 催化剂清单
- 交易设置：近期股票表现、期权隐含波动

## 重要注意事项

- 一致预期会变化 — 始终注明预期的来源和日期
- 买方调查的"耳语数字"通常比公开的一致预期更相关
- 历史财报反应有助于校准预期（搜索"[company] earnings reaction history"）
- 期权隐含波动告诉你市场的预期 — 与你的情景进行比较

## 金融术语和知识解释

### 1. 财报预览 (Earnings Preview)

财报预览是在公司发布财报前发布的分析报告，帮助投资者：
- 了解市场预期
- 识别关键催化剂
- 准备投资策略

### 2. 一致预期 (Consensus Estimates)

一致预期是华尔街分析师对公司财务指标的平均预测，包括收入、EPS等。

### 3. 耳语数字 (Whisper Numbers)

耳语数字是未公开的、更精准的预期，通常来自买方机构的调查，比公开的一致预期更能反映市场真实预期。

### 4. 情景分析 (Scenario Analysis)

情景分析是评估不同假设情况下投资结果的方法：
- **多头情景**：最好的情况
- **基准情景**：最可能的情况
- **空头情景**：最坏的情况

### 5. 催化剂 (Catalyst)

催化剂是可能导致股价变动的关键事件或因素。

### 6. ARR (Annual Recurring Revenue)

ARR是年度经常性收入，是SaaS公司的关键指标，衡量订阅业务的规模和增长。

### 7. 净留存率 (Net Retention Rate)

净留存率衡量现有客户收入的增长情况，计算公式：(当前客户收入 + 扩展收入 - 流失收入) / 先前客户收入。

### 8. 期权隐含波动 (Options-Implied Move)

期权隐含波动是根据期权价格计算的市场预期股价波动幅度，反映市场对财报的不确定性。

### 9. 同店销售 (Same-Store Sales)

同店销售是指开业至少一年的门店的销售额变化，衡量零售业务的有机增长。

### 10. 净息差 (Net Interest Margin, NIM)

净息差是银行利息收入与利息支出之间的差额占生息资产的比例，是银行盈利能力的关键指标。
