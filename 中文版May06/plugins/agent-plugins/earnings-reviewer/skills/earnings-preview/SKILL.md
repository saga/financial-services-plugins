# 财报预览

description: 构建财报前分析，包含预测模型、情景框架和需要关注的关键指标。在公司发布季度财报前使用，准备定位笔记、设置多头/空头情景，并确定哪些因素会影响股价。触发词包括"earnings preview"、"what to watch for [company] earnings"、"pre-earnings"、"earnings setup"或"preview Q[X] for [company]"。

## 工作流程

### 步骤1：收集背景信息

- 确定公司和报告季度
- 通过网络搜索获取一致预期（收入、EPS、关键部门指标）
- 找到财报日期和时间（盘前 vs. 盘后）
- 回顾公司上一季度财报电话会议的任何指引或评论

### 步骤2：关键指标框架

构建针对特定公司的"需要关注"框架：

**财务指标：**
- 收入 vs. 一致预期（总计和按部门）
- EPS vs. 一致预期
- 利润率（毛利率、营业利润率、净利率）—— 扩大还是收缩？
- 自由现金流
- 未来指引 vs. 一致预期

**运营指标**（行业特定）：
- 科技/SaaS：ARR、净留存率、RPO、客户数量
- 零售：同店销售、客流量、客单价
- 工业：订单积压、接单出货比、价格 vs. 销量
- 金融：净息差、信贷质量、贷款增长、手续费收入
- 医疗：处方量、患者数量、管线更新

### 步骤3：情景分析

构建3种情景及股价影响：

| 情景 | 收入 | EPS | 关键驱动因素 | 股价反应 |
|------|------|-----|--------------|----------|
| 多头 | | | | |
| 基准 | | | | |
| 空头 | | | | |

对于每个情景：
- 运营层面需要发生什么
- 管理层评论会发出什么信号
- 历史背景 — 股票在类似业绩下如何波动？

### 步骤4：催化剂清单

确定决定股价反应的3-5件事：

1. [指标] vs. [一致预期/耳语数字] — 为什么重要
2. [指引项目] — 买方期望听到什么
3. [叙事转变] — 任何战略变化、并购、重组

### 步骤5：输出

单页财报预览，包含：
- 公司、季度、财报日期
- 一致预期表
- 需要关注的关键指标（按重要性排序）
- 多头/基准/空头情景表
- 催化剂清单
- 交易设置：近期股票表现、期权隐含波动

## 重要注意事项

- 一致预期会变化 — 始终注明预期的来源和日期
- 买方调查的"耳语数字"通常比公开的一致预期更相关
- 历史财报反应有助于校准预期（搜索"[company] earnings reaction history"）
- 期权隐含波动告诉你市场的预期 — 与你的情景进行比较

## 金融术语和知识解释

### 1. 财报预览详解

**定义**：财报预览是在公司发布财报前发布的分析报告，帮助投资者提前准备。

**目的**：
- 了解市场预期
- 识别关键催化剂
- 准备投资策略
- 设定交易计划

**内容结构**：
| 部分 | 内容 |
|------|------|
| 一致预期 | 收入、EPS、关键指标 |
| 关键指标 | 需要关注的重点 |
| 情景分析 | 多头/基准/空头 |
| 催化剂清单 | 决定股价的因素 |
| 交易设置 | 近期表现、隐含波动 |

**发布时间**：财报前1-3天

**使用场景**：
- 机构投资者准备仓位
- 交易员制定策略
- 分析师更新观点

---

### 2. 一致预期机制

**定义**：一致预期是所有跟踪分析师对公司财务指标预测的平均值或中位数。

**数据来源**：
- **FactSet**：综合金融数据平台
- **Bloomberg**：实时终端服务
- **Thomson Reuters**：详细分析师数据

**计算方法**：
```
一致预期 = (预测1 + 预测2 + ... + 预测n) / n
```

**常用指标**：
- 收入
- EPS（每股收益）
- EBITDA
- 毛利率
- 净利润

**时间维度**：
- **LTM（过去12个月）**：历史数据
- **NTM（未来12个月）**：前瞻预测
- **FY（财年）**：财年预测

**质量评估**：
| 指标 | 评估内容 |
|------|----------|
| 分散度 | 分析师预测差异 |
| 准确性 | 过往预测偏差 |
| 更新频率 | 预测调整频繁程度 |

---

### 3. 耳语数字详解

**定义**：耳语数字是未公开的、更精准的市场预期。

**来源**：
- 买方机构调查
- 行业内部消息
- 高频交易数据
- 非公开渠道

**特点**：
- **保密性**：不对外公开
- **精准性**：通常比公开预期更准确
- **时效性**：临近财报时更新

**获取方式**：
- 机构内部调查
- 卖方分析师沟通
- 专业数据服务

**市场影响**：
- 耳语数字与实际业绩的差异是股价波动的主要驱动因素
- 超出耳语数字通常导致大幅上涨
- 低于耳语数字可能引发抛售

---

### 4. 情景分析深度解析

**定义**：情景分析是评估不同假设情况下投资结果的方法。

**情景构建**：
| 情景 | 概率 | 假设条件 |
|------|------|----------|
| **多头情景** | 20-30% | 最好的情况 |
| **基准情景** | 50-60% | 最可能的情况 |
| **空头情景** | 10-20% | 最坏的情况 |

**分析步骤**：
1. 定义情景边界
2. 设定关键假设
3. 预测财务影响
4. 估算股价反应
5. 制定应对策略

**示例**：
```
公司：Apple (AAPL)
财报日期：10月26日

多头情景（30%概率）：
- iPhone销量超预期10%
- 服务收入增长20%+
- 股价反应：+8-10%

基准情景（50%概率）：
- 符合一致预期
- 维持当前指引
- 股价反应：±2%

空头情景（20%概率）：
- iPhone销量不及预期5%
- 毛利率收缩
- 股价反应：-5-7%
```

**应用价值**：
- 帮助投资者应对不确定性
- 制定风险控制策略
- 评估风险/回报比

---

### 5. 催化剂识别与分析

**定义**：催化剂是可能导致股价显著变动的关键事件或因素。

**催化剂类型**：
| 类型 | 示例 | 时效性 |
|------|------|--------|
| **财报催化剂** | 财报发布、指引更新 | 短期 |
| **公司催化剂** | 并购、产品发布 | 中短期 |
| **行业催化剂** | 监管政策、技术突破 | 中长期 |
| **宏观催化剂** | 利率变化、经济数据 | 系统性 |

**催化剂评估**：
| 标准 | 评估内容 |
|------|----------|
| 时间明确 | 知道何时发生 |
| 影响可量化 | 能估计股价影响 |
| 市场预期 | 了解市场定价 |
| 验证机制 | 知道如何验证 |

**催化效力矩阵**：
| 催化效力 | 市场预期 | 投资策略 |
|----------|----------|----------|
| 高 | 低 | 买入 |
| 高 | 高 | 等待 |
| 低 | 低 | 忽略 |
| 低 | 高 | 卖出 |

---

### 6. ARR详解

**定义**：ARR（Annual Recurring Revenue）即年度经常性收入。

**计算公式**：
```
ARR = 当前订阅收入 × 12
```

**重要性**：
- 衡量SaaS公司核心业务规模
- 反映订阅模式的稳定性
- 评估增长质量

**增长驱动**：
| 驱动因素 | 说明 |
|----------|------|
| 新客户获取 | 新增订阅收入 |
| 客户扩展 | 现有客户增购 |
| 价格提升 | 涨价或增值服务 |
| 客户留存 | 减少流失 |

**行业基准**：
| 公司规模 | ARR范围 |
|----------|----------|
| 初创公司 | <$100M |
| 中型公司 | $100M-$1B |
| 大型公司 | >$1B |

---

### 7. 净留存率详解

**定义**：净留存率衡量现有客户收入的增长情况。

**计算公式**：
```
净留存率 = (期初ARR + 扩展收入 - 流失收入) / 期初ARR × 100%
```

**指标解读**：
- **>100%**：现有客户收入增长（理想）
- **=100%**：现有客户收入持平
- **<100%**：现有客户收入流失

**行业标准**：
| 行业 | 优秀水平 | 良好水平 |
|------|----------|----------|
| SaaS | >120% | 100-120% |
| 订阅电商 | >110% | 90-110% |
| 媒体订阅 | >95% | 85-95% |

**提升策略**：
1. 产品扩展
2. 交叉销售
3. 定价优化
4. 客户成功管理

---

### 8. 期权隐含波动

**定义**：期权隐含波动是根据期权价格计算的市场预期股价波动幅度。

**计算方法**：
- 通过Black-Scholes模型反推
- 反映市场对未来不确定性的预期

**财报前应用**：
```
隐含波动 = 预期财报后股价变动幅度
示例：隐含波动20% → 预期股价变动±20%
```

**解读要点**：
| 情况 | 含义 |
|------|------|
| 隐含波动高 | 市场预期波动大 |
| 隐含波动低 | 市场预期波动小 |
| 隐含波动上升 | 不确定性增加 |
| 隐含波动下降 | 不确定性减少 |

**交易应用**：
- 隐含波动高时卖出期权（收取高溢价）
- 隐含波动低时买入期权（低成本投机）
- 比较隐含波动与历史波动

---

### 9. 同店销售详解

**定义**：同店销售是开业至少一年的门店销售额变化。

**计算公式**：
```
同店销售增长率 = (当期同店销售额 - 上期同店销售额) / 上期同店销售额 × 100%
```

**重要性**：
- 衡量零售业务有机增长
- 排除新店开业影响
- 反映核心业务健康状况

**影响因素**：
| 因素 | 影响 |
|------|------|
| 客流量 | 到店顾客数量 |
| 客单价 | 平均消费金额 |
| 产品组合 | 高/低毛利产品占比 |
| 定价策略 | 折扣促销影响 |

**行业差异**：
| 零售类型 | 同店销售基准 |
|----------|--------------|
| 奢侈品 | 高端增长稳定 |
| 大众零售 | 受经济周期影响 |
| 电商 | 无同店概念 |

---

### 10. 净息差详解

**定义**：NIM（Net Interest Margin）即净息差。

**计算公式**：
```
净息差 = (利息收入 - 利息支出) / 平均生息资产 × 100%
```

**构成要素**：
| 要素 | 说明 |
|------|------|
| 利息收入 | 贷款、债券等利息收入 |
| 利息支出 | 存款、借款等利息支出 |
| 生息资产 | 产生利息的资产 |

**影响因素**：
- **市场利率**：利率上升通常扩大NIM
- **资产配置**：高收益资产占比
- **资金成本**：存款利率水平
- **信贷质量**：不良贷款影响

**行业对比**：
| 银行类型 | NIM范围 |
|----------|----------|
| 大型银行 | 2.5-3.5% |
| 区域性银行 | 3.0-4.5% |
| 零售银行 | 3.5-5.0% |

**重要性**：
- 银行盈利能力的核心指标
- 反映定价能力和风险管理
- 受货币政策直接影响
