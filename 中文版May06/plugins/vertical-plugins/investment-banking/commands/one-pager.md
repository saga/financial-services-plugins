# 单页概览创建指令 (One-Pager Strip Profile Command)

为商业计划书 (Pitch books) 和交易材料创建一份专业的单页公司概况简介。

## 工作流 (Workflow)

### 第 1 步：收集公司信息 (Gather Company Information)

如果提供了公司名称或股票代码，请使用它。否则询问：
- "您希望为您想为哪家公司制作简介？"

### 第 2 步：检查可用的 PPT 模板技能 (Check for Available PPT Template Skills)

**首先，在技能目录中检查是否有现成的 ppt-template 技能**：

```bash
ls skills/ | grep -E "ppt-template|brand-guidelines"
```

如果存在模板技能 (例如 `techcorp-ppt-template`, `gs-brand-guidelines`)：
1. 向用户列出当前可用的模板
2. 询问他们希望使用哪一个，或者是否希望使用干净的无品牌通用专业格式
3. 使用 `skill: "[模板名字]"` 加载用户选定的模板技能

如果没有找到模板技能，则询问：
- "请问您有必须使用的品牌专属 PowerPoint 模板文件吗？如果有，请提供它的绝对路径。否则我将使用通用型的专业排版。"

如果用户提供了模板文件：
1. 分析模板的结构以理解页面排版规则
2. 为这段“单页简介”挑选最合适的版式 (Layout)

### 第 3 步：加载单页简介技能 (Load Strip Profile Skill)

使用 `skill: "strip-profile"` 技能执行页面创建操作：

1. **理清需求 (Clarify requirements)**:
   - 确认输出格式为单页 PPT (one-pager)
   - 询问重点想在简介中强调什么领域的亮点

2. **调研公司核心数据 (Research company data)**:
   - 公司概览 (总部位置、成立年份、员工总数、高管团队)
   - 业务描述及市场定位
   - 核心财务数据 (营收 Revenue、息税折旧摊销前利润 EBITDA、利润率 margins、业务增长率)
   - 估值指标 (市值 Market Cap、企业价值 EV、估值倍数 multiples)
   - 近期动态及新闻
   - 前几大主要股东 (仅限上市公司收集这部分数据)

3. **制作内容排版 (Create the strip profile)**:
   - 默认使用 4:3 的比例尺寸 (10" x 7.5")
   - 采用四象限经典排版布局 (4-quadrant layout):
     - 左上：公司概况 (项目符号)
     - 右上：业务及定位 (项目符号)
     - 左下：核心财务数据 (数据表格)
     - 右下：1年期股价走势图 + 股权结构 或 近期重大新闻
   - 匹配并应用公司的品牌主题色
   - 为各个象限的小节标题加上强调装饰条 (accent bars)

### 第 4 步：视觉审核 (Visual Review)

在生成幻灯片后:
1. 在系统内将其转化为图片以供复核检查
2. 严肃检查是否有文本重叠或遮挡超出边框的情况 (overlap/cutoff)
3. 验证所有数据空隙是否被真实填满 (不要粗心留下 Placeholder 占位符)
4. 向用户展示粗略预览以便获得最终审批

### 第 5 步：交付输出 (Deliver Output)

提供：
1. **PowerPoint 演示文稿文件** (.pptx) - 该单页简介文件
2. **图片预览图** - 供用户通过平台界面快速审核排版
3. **关键数据摘要** - 总结页面内部包含的最具代表性的财务数字

## 单页排版布局参考指南 (Layout Reference)

```
┌─────────────────────────────────────────────────────────────────┐
│ 公司名称 (股票代码)                                      [公司Logo] │
├────────────────────────────┬────────────────────────────────────┤
│ 公司概观 (COMPANY OVERVIEW) │ 业务及定位 (BUSINESS & POSITIONING) │
│ • 总部, 成立年份, 员工数      │ • 核心业务的商业模式描述               │
│ • CEO, CFO 名字             │ • 关键产品 / 服务类别                │
│ • 总市值, 所处行业板块       │ • 市场竞争分析                      │
│ • 其它关键运营指标           │ • 业绩增长的主要驱动力               │
├────────────────────────────┼────────────────────────────────────┤
│ 核心财务 (KEY FINANCIALS)    │ 股价表现与股权结构                    │
│ ┌──────────────────────┐   │ [(此处放置1年期股价走势图表)]          │
│ │ 指标 │ 2024 │ 2025E│   │                                    │
│ │ 营收 │ $XXB │ $XXB │   │ 前三大主要股东 (Top Shareholders):   │
│ │ EBITDA │ $XXB │ $XXB │   │ • 先锋领航 (Vanguard): X.X%        │
│ │ 利润率 │ XX%  │ XX%  │   │ • 贝莱德 (BlackRock):  X.X%        │
│ │ EV/EBITDA │ XXx │ XXx │   │ • 道富银行 (State Street): X.X%    │
│ └──────────────────────┘   │                                    │
└────────────────────────────┴────────────────────────────────────┘
数据来源: 标的公司公开披露文件, FactSet 等金融数据库
```

## 质量核对清单 (Quality Checklist)

在最终交付给用户前自行在后台核对:
- [ ] 幻灯片的所有 4 个象限都已填写真实考究过的数据
- [ ] 没有任何遗留的 [XX] 占位符或空白未修改的文本
- [ ] 已经应用到符合客户定位的品牌调色板
- [ ] 所有区块的标题都带有规范的装饰强调条
- [ ] 财务数据表格排版对齐专业规范
- [ ] 已在幻灯片最底部边缘标注数据来源 (Source)
- [ ] 没有任何文本溢出框线、或者被其他图片裁切遮挡
- [ ] 整体达到了顶级投行（如高盛 MS 等九大行标准）的苛刻排版审美品控标准

---

> **💡 Appendix: 领域知识小贴士**
>
> 1. **Strip Profile / One-Pager (单页简介)**：投行 Pitch Book 里最核心的一种页面格式，通常只用一页极度密集的PPT就把一家大公司拔光看个通透。对于老练的顶尖银行家，看这一页就够决定要不要花精力去买或者接触这家公司了。
> 2. **Pitch Book (推介书/路演材料)**：投行初级员工辛辛苦苦熬夜做的精美、长篇大论的 PPT 大礼包，用来在会议桌上试图说服客户大老板“请雇佣我们代表你们去卖公司/买公司吧”。
> 3. **EBITDA / EV**：EBITDA 叫“息税折旧及摊销前利润”，EV 叫“企业价值”(Enterprise Value)。在并购圈内，比起散户最爱讨论的 P/E（市盈率），大家最爱看的估值标尺是 EV/EBITDA 倍数，因为它排除了由于不同国家税收政策差异、以及公司借钱多少带来的账面会计干扰。
