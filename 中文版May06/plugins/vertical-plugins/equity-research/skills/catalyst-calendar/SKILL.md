# 催化剂日历

description: 为覆盖的投资标的池构建并追踪即将到来的催化剂日历——涵盖财报日、行业会议、产品发布、监管决议及宏观经济事件。旨在协助优化投研关注重点，并在重大事件落地前完成持仓布局。在输入“catalyst calendar”（催化剂日历）、“upcoming events”（近期事件）、“what's coming up”（未来动态）、“earnings calendar”（财报日历）、“event calendar”（事件日历）或“catalyst tracker”（催化剂追踪器）时触发。

## 工作流程

### 第一步：界定核心覆盖范围 (Define Coverage Universe)

- 核心追踪对象名单（股票代码或公司名称）
- 聚焦的板块及其子行业
- 是否纳入宏观事件考量？（如美联储议息会议、重磅经济数据、监管截止节点）
- 前瞻的时间跨度（未来两周、本自然月、或者是当个季度）

### 第二步：汇总关键催化剂事件

针对覆盖池中的每一家公司，梳理近期即将到来的各类事件：

**财报与财务事件**
- 季度财报发布具体日期及时间点（盘前或盘后）
- 年度股东大会 (AGM)
- 投资者日 (Investor Day) / 分析师日 (Analyst Day)
- 资本市场日 (Capital Markets Day)
- 核心债务到期日 / 重大再融资窗口节点

**公司基本面事件**
- 核心产品发布会或重磅公告
- FDA（美国食品药品监督管理局）审批 / 关键性监管决策
- 重要合同续签或到期节点
- 并购重组 (M&A) 分步里程碑（交易交割日、反垄断监管审批进程）
- 核心管理层交接换届
- 内部人士交易窗口（禁售期锁股解禁日）

**行业动态事件**
- 顶级行业峰会及会议（起始日期，哪些重量级公司出席并演讲）
- 重要的贸易展览及博览会
- 行业性监管意见征询期临近或最终裁决出台
- 高频行业数据出炉（例如月度零售数据、月度互联网流量报告等）

**宏观经济事件**
- 美联储议息会议（FOMC 决议日程）
- 非农就业报告、CPI（消费者物价指数）、GDP等高维数据发布
- 全球主流央行货币政策决议（欧洲央行ECB、日本央行BOJ等）
- 具备全局市场冲击力的地缘政治事件

### 第三步：日历视图构建

| 日期 | 事件 | 公司/板块 | 类别 | 影响程度 (高/中/低) | 持仓策略 | 备忘录 |
|------|-------|---------------|------|-----------------|----------------|-------|
| | | | 财报/公司/行业/宏观 | | 做多/做空/中性 | |

### 第四步：周度前瞻输出

于每周初生成一份极具实操价值的前瞻性简报：

**本周核心焦点事件：**
1. [具体周几]：[公司名称] Ｑ[X] 财报披露 —— 华尔街共识预期 [EPS $X]，本机构内部测算 [$X]，核心追踪指标：[具体财务或营运指标]
2. [具体周几]：[待发生事件] —— 该事件缘何至关重要及对 [相关特定股票] 之影响逻辑
3. [具体周几]：[宏观经济数据发布] —— 市场主流预期差捕捉与我们的持仓应对

**下周雷达前瞻：**
- 针对即将杀至的重要波动事件做好早期预警准备

**持仓影响与操作映射：**
- 预判可能引发特定标的仓位剧烈波动的诱因
- 基于预期推演而推荐的任何前置布局策略
- 在二元事件（如非此即彼的审批决策）引爆前所需进行的严格风险管理

### 第五步：交付产出形式

- 包含直观日历视图及各项维度可精细排序的 Excel 工作簿
- 周度前瞻内参邮件或备忘录 (以 Markdown 格式呈现)
- 附加选项（按需配置）：在 Google Calendar 中实现无缝事件同步整合

## 投研实操注意事项

- **极度注意财报窗口位移：** 公司财报发布日时常突发更改——临窗前务必通过公司官方 IR（投资者关系）页面、彭博 (Bloomberg) 或 辉盛 (FactSet) 终端完成精准二次核对。
- **警惕前瞻性业绩指引风险：** 对于那些在历史上存在“前瞻盈利预警（无论是盈喜还是盈警）”惯犯记录的上市公司，必须要建立专项追踪档案。
- **挖掘参会名单的隐晦阿尔法：** 重大行业会议的出席座次名单极其珍贵——需要密切关注“哪些公司高调现身布道”，又有哪些本该在座的公司却“不合常理地异常缺席”？
- **实现周期性催化剂的自动化处理：** 诸如月度行业产销数据等部分催化剂具备强烈的规律周期性，应当为此构建底层模板并跑通自动抓取填充流。
- **针对事件冲击量级进行色彩梯度标记：** 强制规范——红色代表高烈度影响，黄色代表中性影响，绿色代表寻常的例行常规事件。
- **事件复盘库建设：** 务必将历史发生过的催化剂事件结合其实际落地的市场反馈进行归档封存——这有助于投研体系在岁月沉淀中建立起对极值异动模式的“肌肉记忆与模式识别”。

> **💡 Appendix: 领域知识小贴士**
> 
> 嗨！如果你刚接触这篇“硬核”的金融研报规范文档，别担心，我们来把这些高大上的华尔街黑话“翻译”成通俗易懂的大白话：
> 
> - **🚀 催化剂 (Catalyst)：** 股票世界的“引爆点”。就像化学实验里加速反应的物质一样，在股市中，催化剂就是那些能让股票价格突然暴涨或暴跌的重大事件（比如发财报、发重磅新药、甚至马斯克发一条推文）。
> - **🎯 覆盖标的池 (Coverage Universe)：** 投资大佬们的“长期狙击清单”。股市里上市公司成千上万，精力根本看不过来。所以机构研究员会精挑细选几十家自己最懂的公司重点盯防，这个小圈子就叫“覆盖池”。
> - **📊 EPS (每股收益：Earnings Per Share)：** 简单说就是公司今年总利润分摊到每一股上赚了多少钱。它是衡量公司最核心的“钞能力”指标。大家每次都在猜下次公布的数字是多少（市场共识预期），要是官方拿出来的数字比大家猜的还要凶悍，股价大概率带量起飞！
> - **🎲 二元事件 (Binary Events)：** 一翻两瞪眼的“盲盒时刻”。这里通常只有两种极端结果，没有中间地带。最经典的就是创新药企向FDA（药监局）申请新药上市：要么“盖章通过”（股价瞬间翻倍），要么“残忍拒绝”（闪崩腰斩），是心跳玩家最关注的节点。
> - **⚖️ 头寸/持仓 (Positioning) - 做多(Long)与做空(Short)：** “做多”就是觉得未来会涨，于是按现价买入，等涨了卖了赚钱；“做空”就是觉得未来会跌，于是先找券商借点股票卖掉，等股价暴跌后再以便宜价格买回来还回去，赚中间的差价。“头寸”意思就是指你在赌局里手里现在捏着多少做多或做空的筹码。
> - **🔓 禁售解禁 (Lockup Expirations)：** 公司刚上市（或者核心员工刚拿到股权激励）时，大股东和高管们的股票会被上锁“冻结”一段时间，不允许立马变现。一旦熬到了“解禁日”这天，封印解除，市场上往往会突然涌出一大波急于套现买大别墅的股票抛盘，这有时会导致股价短时间内面临巨大的承受压力。
> - **🌍 宏观事件 (Macro Events)：** 指的是那些能直接冲击全体老百姓钱包和震荡全球大盘的“国家级大事件”。比如美联储每个月开会讨论要不要多印点钞票，也就是大家常听到的“加息”或者“降息”决定。
