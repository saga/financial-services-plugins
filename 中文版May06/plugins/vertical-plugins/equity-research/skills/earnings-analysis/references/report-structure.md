## 格式排版要求

### 1. 首页规范
- 明确无误的投资评级（如：维持跑赢大盘、上调至买入等）
- 在显眼位置标注最新目标价
- 包含包含新旧预测对比的财务摘要表
- 3-4 个使用 ■ 符号起头的段落化核心观点总结

### 2. 表格规范
- 底部必须配有来源（Source）标注
- 表头说明必须清晰明确
- 标题行应有底纹背景加以区分

### 3. 图表规范
- 图表上方需加上“图表 X - [标题]”（"Figure X - [Title]"）说明
- 图表下方必须标注“来源：[出处]”（"Source: [Source]"）
- 维持专业商务的配色与排版风格

### 4. 年份与预测标记
- 实际已发生年份/季度使用 'A'（Actual）（例：Q3'24A）
- 预测年份/季度使用 'E'（Estimate）（例：Q4'24E）

### 5. 撰写风格
- 数据先行（例如写“营收增长15%至12亿美元”，千万不要虚写“营收增长强劲”）
- 比较时使用 "vs." 而不是 "versus"
- 语言风格应直接且精炼
- 聚焦于“边际增量”（NEW），即最新的变化与信息

### 6. 超链接规范 ⭐⭐⭐
- 所有 URL 必须在 Word 中制作为可点击的**超链接**
- 必须是蓝字、带下划线且支持 Ctrl+Click 即可打开的文本
- 超链接必须附在有阅读意义的文本上（严禁直接展示生硬的 URL 代码）
- 能够引用的每一条数据/消息出处均需添加超链接
- 不允许存在纯文本形式的 URL —— 必须永远格式化为文本超链接

> **💡 Appendix: 领域知识小贴士**
> 
> 欢迎来到华尔街的术语丛林！如果你是投资新手，这篇报告里可能有一些让你一头雾水的词汇。别担心，我们为你准备了一份“大白话”指南：
> 
> 🎯 **Beat / Inline / Miss (业绩超预期 / 符合预期 / 不及预期)**
> 华尔街的分析师们会在财报发布前给出一个“平均预测及期望”（这叫**市场共识/Consensus**）。如果公司赚的钱比大家猜的还多，就是“Beat”（在金融圈意味着考了高分，股价通常会涨！）；正好和大家猜的一样就是“Inline”；没达标就是跌破眼镜的“Miss”。
> 
> 📊 **EPS / Adjusted EPS (每股收益 / 调整后每股收益)**
> EPS 就是“公司属于普通股东的净利润 ÷ 总股本”，也就是你手里这一股股票为你赚了多少钱。为什么要有“调整后（Adjusted）”呢？因为公司有时候会有一笔极其罕见的花销或收入（比如卖了一栋祖传的楼），把这些“一次性”的账目剔除，能更真实地看出它日常主营业务到底有多会赚钱。
> 
> 📅 **YoY 与 QoQ (同比与环比)**
> *   **YoY (Year-over-Year / 同比)**：跟**去年同一个季度**比。比如今年冬天买羽绒服的钱和去年冬天比（这很科学，因为排除了季节性的影响）。
> *   **QoQ (Quarter-over-Quarter / 环比)**：跟**上一个季度**直接比。比如第三季度和第二季度比，适合考察一家公司最近的业绩爆发速度或衰退趋势。
> 
> 📏 **bps (基点 / Basis Points)**
> 金融精英们觉得“百分之零点几”听起来不够专业，于是发明了“基点”。**1 个基点（1 bps）= 0.01%**。所以 100 意味着 1%。如果报告说利润率扩张了 50bps，它的意思就是大白话的“利润率涨了 0.5%”。
> 
> 🔮 **Guidance (业绩指引)**
> 这就是公司的“官方剧透”。管理层提前预测下一个季度或未来一整年自家能赚多少钱。很多时候，就算眼下的财报“Beat”了，如果公司对未来的“Guidance”给出悲观下调，股价照样会闪崩。因为股市永远是对**未来**下注的地方。
> 
> ⚖️ **估值工具：P/E 与 EV/EBITDA**
> *   **P/E (市盈率)**：买一家公司需要多少年回本？P/E为20x（20倍），相当于假设它的净利润永远不变，你拿钱买下这家公司需要 20 年才能回本。
> *   **EV/EBITDA (企业价值倍数)**：升级版的“回本年限计算器”。它仔细扣除了公司账上的现金、加上了还欠着的债务引起的价值变动情况，非常适合用来横向比较同一行业里不同段位的公司。这个数字通常也是越低代表这支股票越“便宜”。
> 
> ⏱️ **DCF (现金流折现模型 / Discounted Cash Flow)**
> 华尔街的终极算命术！假设这家公司未来十年每个月都能赚一把现金回来，但考虑到通胀和利息（也就是今天的100元肯定比十年后的100元更值钱），把它未来所有的钱按照一定折扣率“折现”算到今天，就能加总推算出这家公司现在理论上到底值多少钱，这就是传说中的 Fair Value（合理价值）。
