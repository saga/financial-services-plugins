# 股票研究：财报更新报告

遵循机构级标准（摩根大通、高盛、摩根士丹利等投行格式），为已纳入研究覆盖范围的公司撰写专业的**财报更新报告 (EARNINGS UPDATE REPORTS)**，分析其季度业绩。

**核心特征：**
- **篇幅**：8-12 页
- **字数**：3,000-5,000 字
- **表格**：1-3 个汇总表格（非全面资产负债/利润表）
- **图表**：8-12 张图表
- **交付周期**：1-2 天（财报发布后 24-48 小时内）
- **受众**：对该公司已有一定了解的客户
- **重点**：最新动向 —— 业绩超预期/低于预期情况、财务预测更新、对投资逻辑的影响
- **字体**：全文使用 Times New Roman（除非用户另有指定）

## 适用场景

当用户提出以下请求时使用：
- "制作一份[某公司] 2024 年第三季度的财报更新报告"
- "分析[某公司]的季度业绩"
- "[某公司]的财报发布后报告"
- "[某公司]的第一/二/三/四季度更新"

**在以下情况中请勿使用：**
- 用户要求撰写“首次覆盖报告 (initiation report)” → 使用其他特定技能
- 用户要求撰写“简评 (flash note)”或“快报 (quick take)” → 格式不符
- 该公司尚未被纳入研究覆盖范围 → 需先撰写首次覆盖报告

## 核心要求

### 1. 速度与时效性
- 必须在财报发布后的 24-48 小时内发布
- 仅聚焦于**最新信息**
- 切忌大篇幅重复介绍公司背景

### 2. 超预期/低于预期分析 (Beat/Miss Analysis)
- 首要说明公司业绩是超出还是低于市场预期
- 量化差异（例如：“营收超出预期 1.2 亿美元或 3%”）
- 深度解释实际业绩与预期产生差异的**原因**

### 3. 精简汇总格式
- 表格数量严格控制在 1-3 个（仅做核心汇总，非完整报表）
- 不展示完整的利润表/现金流量表/资产负债表（仅展示关键指标）
- 假设读者已经阅读过该公司的首次覆盖报告

### 4. 引用与数据来源归属 ⭐⭐⭐ 强制要求

**核心准则**：所有数据必须有明确的来源标注，并提供**可点击的超链接 (CLICKABLE HYPERLINKS)**。

**在每一张图表和表格中必须提供具体来源及可点击的链接：**

```
来源：2024年11月8日提交的2024年第三季度10-Q报表；公司业绩新闻稿
        [将“10-Q报表”超链接至：https://www.sec.gov/cgi-bin/viewer?accession=...]
        [将“业绩新闻稿”超链接至：https://investor.company.com/news/q3-2024]
```

**WORD 中超链接的格式规范：**
- 文件名称必须显示为蓝色、带下划线的可点击链接
- 读者可以按下 Ctrl+点击 直接在浏览器中打开来源页面
- 请勿直接粘贴纯文本 URL —— 必须将其格式化为带有显示文本的超链接

**必填来源引用清单：**

在每份财报更新报告中必须引用以下内容：
- ✅ 财报新闻稿（含日期和 URL）
- ✅ 10-Q 备案文件（含提交日期和美国证监会 EDGAR 数据库链接）
- ✅ 财报电话会议实录（含日期）
- ✅ 投资者推介文件/补充说明材料（如有）
- ✅ 市场一致预期数据源（彭博/FactSet 等，含日期）
- ✅ 前期业绩指引（来自上一季度的资料）

**包含可点击超链接的参考文献清单部分：**

在报告末尾必须包含“来源 (Sources)”部分：

```
来源与参考文献 (SOURCES & REFERENCES)

财报相关材料 (2024年第三季度)：
• 财报新闻稿 (2024年11月7日)
  [将整行内容超链接至：https://investor.company.com/news/q3-2024-earnings]

• 10-Q 报表 (提交于2024年11月8日)
  [超链接至：https://www.sec.gov/cgi-bin/viewer?accession=...]

• 财报电话会议实录 (2024年11月7日)
  [超链接至：https://seekingalpha.com/article/...]

• 投资者推介文件 (2024年11月7日)
  [超链接至：https://investor.company.com/presentations/q3-2024.pdf]
```

**合规审核清单 (VERIFICATION CHECKLIST)：**
- [ ] 每张图表均配有来源说明（包含具体文件名称及日期）
- [ ] 每个表格均配有来源说明及文件引用
- [ ] 业绩超预期/低于预期分析引用了一致预期数据源及日期
- [ ] 业绩指引变动引用了当前与前期的指引材料来源
- [ ] 核心财务数据配有脚注说明
- [ ] 报告末尾的来源部分详列了所有材料及相应 URL
- [ ] 所有 URL 均为**可点击的超链接**（严禁使用纯文本 URL）
- [ ] 所有美国证监会 (SEC) 备案文件均已准确链接至 EDGAR 查阅系统

### 5. 财务预测更新 (Updated Estimates)
- 需根据最新业绩表现更新远期预测
- 清晰对比并展示旧预期与新预测间的变化
- 详细说明预期调整的具体内容及背后的核心驱动因素

## 整体工作流程设计 (High-Level Workflow)

撰写财报更新报告的过程主要分为 5 个阶段：

### 阶段 1：数据收集（30-60 分钟）

**🚨🚨🚨 极其重要警告：训练数据中包含的财报可能已过时 🚨🚨🚨**

**在启动之前 - 必须严格按顺序完成以下 4 个步骤：**
1. **确认今天的日期** - 明确记录当前系统日期
2. **搜索最新数据** - 使用网络搜索：“[公司名称] latest earnings results”
3. **核实发布日期** - 确认检索到的财报发布日期在过去 3 个月内
4. **核查电话会议记录日期** - 确认会议记录的日期与财报发布日期相匹配

**常见致命错误**：依赖模型训练数据中过时的财报电话会议内容，而没有去搜寻最新的业绩数据。

**硬性执行要求：**
- ✅ 必须搜索最新财报 - 绝不依赖历史训练数据
- ✅ 写下今天的当前日期以及检索到的财报发布日期
- ✅ 核实验证财报发布日期距离当前时间未超过 3 个月
- ✅ 验证电话会议记录时间与业绩发布时间一致
- ✅ 如若日期不符或财报已过时（>3个月），必须重新进行网络搜索

**详见 [references/workflow.md](references/workflow.md)** 获取深度的搜索指引与核实步骤。

### 阶段 2：分析研判（2-3 小时）
- 针对各项关键财务指标开展超预期/低于预期深度分析
- 进行业务分部/地理区域/产品线的精细化数据拆解
- 深入剖析利润率表现及管理层给出的业绩指引
- 全面更新财务估值模型及前瞻盈利预测

**详见 [references/workflow.md](references/workflow.md)** 获取详细分析框架指引。

### 阶段 3：图表构图与生成（1-2 小时）
制作 8-12 张高质量图表，精准反映季度财报趋势及最新业务动态：
- 季度营业收入变化趋势对比
- 季度每股收益 (EPS) 变化趋势对比
- 季度利润率表现追踪
- 营收结构分解（按业务分部/地域范围）
- 核心运营指标走势
- 超预期/低于预期情况多维汇总
- 盈利预测修正及调整轨迹
- 历史估值相关图表

**详见 [references/workflow.md](references/workflow.md)** 参考详细图表绘制规范。

### 阶段 4：报告编撰排版（2-3 小时）
创建符合特定规范结构的 8-12 页 DOCX 格式投研报告。

**详见 [references/report-structure.md](references/report-structure.md)** 获取完整的逐页编排模板及严苛的格式要求。

**报告宏观框架结构：**
- 第 1 页：包含独立评级与目标价的开篇财报摘要
- 第 2-3 页：深度业绩数据剖析
- 第 4-5 页：关键运营指标及管理层业绩指引解读
- 第 6-7 页：基于当前表现的最新投资逻辑复盘
- 第 8-10 页：公司估值模型与财务预期推展
- 第 11-12 页：相关附录（可选参数）

### 阶段 5：质量审验与最终交付（30 分钟）
在最终交付前，严密审核业务内容、排版格式、数据精准度及交付时效性。

**详见 [references/best-practices.md](references/best-practices.md)** 研读标准质控检查清单及常见的分析师低级错误防范策略。

**核心交付物**：DOCX 文档报告（8-12 页）
**文档命名规范**：`[公司名称]_Q[季度]_[年份]_Earnings_Update.docx`
**案例呈现**：`Nike_Q2_FY24_Earnings_Update.docx`

**涵盖要件清单：**
- 第 1 页：包含评级、目标价、核心投资要点的提纲摘要
- 第 2-3 页：深度业绩数据剖析
- 第 4-5 页：关键运营指标及管理层业绩指引解读
- 第 6-7 页：最新投资逻辑有效性复盘
- 第 8-10 页：公司估值模型与财务预期推展
- 第 11-12 页：相关附录（非必须项）
- 8-12 张专业级嵌入式图表
- 1-3 个精简化的财务汇总表格
- 完整的资料来源梳理板块，确保内置可点击超链接

**附加交付项**：XLS 财务估值模型更新（财报更新场景下非强制执行）

|--------|----------------|-------------------|
| **文章篇幅** | 8-12 页 | 30-50 页 |
| **文案字数** | 3,000-5,000 字 | 10,000-15,000 字 |
| **涵盖表格** | 1-3 个精简汇总表 | 12-20 个完备度极高的详尽表格 |
| **附带图表** | 8-12 张 | 25-35 张 |
| **制作周期** | 1-2 天 | 3-6 周 |
| **研究范围** | 本季度财务表现 | 企业全貌生态深度起底 |
| **核心聚焦** | **最新**动向及短期催化 | 涵盖万端的所有基本面 |
| **公司背景简述** | 仅寥寥几笔带过 | 占据 6-10 页宏大篇幅 |
| **XLS 底表模型** | 可选项 | 必须执行的强制要求 |

> **💡 Appendix: 领域知识小贴士**
>
> 欢迎来到小白金融科普时间！这份文件里提到了很多华尔街的“接头暗号”，为了让你迅速上手，我们用大白话来翻译一下这里的核心金融概念：
> 
> - **股票研究 (Equity Research)**：在投行里，有一群智商爆表的分析师，他们的日常就是“研究公司”。他们通过扒数据、看市场，来告诉客户某支股票是该“买”还是该“卖”。如果一家公司被分析师重点盯上了，这就叫“被纳入覆盖范围 (Under Coverage)”。
> - **超预期/不及预期 (Beat/Miss)**：这就像是资本市场的“期末考试”。分析师们会在财报发布前给公司打个“期望分”（也就是**市场一致预期 Consensus Estimates**）。如果公司实际赚的钱比预期多，就是“超预期 (Beat)”，股价往往会受到利好大涨；如果考试没及格，就是“不及预期 (Miss)”，股价可能就会挨大板子。
> - **每股收益 (EPS)**：全称 Earnings Per Share。简单说，就是把公司赚的净利润平分到每一股上。这是衡量公司赚钱能力最硬核的指标。EPS 越高，意味着每一股为你赚的钱越多！
> - **10-Q 季报 (Form 10-Q)**：这可不是普通的报告，这是美国上市公司每个季度必须向美国证监会（SEC）提交的“官方成绩单”。里面的数据经过了最为严格的审核，是分析师看财报时最客观、最权威的财务依据。
> - **业绩指引 (Guidance)**：这是公司管理层对未来的“提前剧透”。他们会在电话会议上告诉大家：“下个季度我们大概能拼出个什么成绩来”。这个对未来的预判由于反映了管理层的信心，有时候比当下的成绩更能影响股价的涨跌！
> - **评级与目标价 (Rating & Price Target)**：分析师给出的“终极行动建议”。评级一般直接了当：“买入 (Buy)”、“持有 (Hold)” 或“卖出 (Sell)”；而目标价则是分析师拍的一个数字，代表他们认为这支股票在未来一年内合理应该涨（或跌）到的标尺价位。
