# 早间纪要

description: 撰写简明扼要的早间会议纪要，总结隔夜的市场进展、交易建议以及所覆盖股票的关键事件。专为早晨 7 点的早间会议格式设计——文字精炼、观点鲜明、具备极强的可操作性。触发词包括：“早间纪要”、“早晨会议”、“隔夜动态”、“交易建议”、“早晨电话准备”或“每日纪要”。

## 工作流程

### 第一步：隔夜动态分析

审视覆盖研究范围内所有相关的事件：

**业绩发布与指引**
- 在隔夜或盘前是否有覆盖的公司发布财报？
- 业绩惊喜（营收、每股收益 EPS 及关键指标是否超出或逊于预期）
- 指引变更（上调、下调或维持业绩指引）

**新闻与事件**
- 并购（M&A）公告或传闻
- 管理层变动
- 产品发布或监管决策
- 竞争对手分析师的评级上调/下调
- 影响该行业的宏观数据或政策变动

**市场背景**
- 隔夜期货 / 盘前走势
- 行业 ETF 表现
- 相关的商品或货币波动
- 今日待发布的关键经济数据

### 第二步：早间纪要格式

保持精炼——一份早间纪要应能在 2 分钟内读完：

---

**[日期] 早间纪要 — [分析师姓名]**
**[覆盖行业]**

**核心观点 (Top Call)：[标题 — 投资组合经理最需要获知的核心信息]**
- 用 2-3 句话阐述关键进展及其重要性
- 对股价的影响：目标价调整、重申或变更评级

**隔夜/盘前动态**
- [公司 A]：业绩/新闻的一句话总结 + 我们的评价
- [公司 B]：一句话总结 + 我们的评价
- [行业/宏观]：相关的全行业发展动态

**今日关键事件**
- [时间]：[公司] 业绩电话会议
- [时间]：经济数据发布（市场预期 vs. 我们的观点）
- [时间]：行业会议或投资者日

**交易策略建议**（如有）
- [做多/做空] [公司]：1-2 句投资逻辑 + 催化因素
- 风险点：什么情况下该逻辑会失效

---

### 第三步：业绩快评

如果覆盖的公司发布了财报，请提供快速反应：

| 指标 | 市场共识预期 | 实际数值 | 超预期/逊预期 |
|------|------------|---------|-------------|
| 营收 |            |         |             |
| EPS |            |         |             |
| [关键指标] |       |         |             |
| 业绩指引 |          |         |             |

**我们的看法**：用 2-3 句话说明——这对股价是利好还是利空？是否改变了我们的投资逻辑？

**操作建议**：维持 / 上调 / 下调评级？是否调整目标价？

### 第四步：输出

- 用于电子邮件/Slack 分发的 Markdown 文本
- 如果需要正式分发，则提供 Word 文档
- 篇幅严格控制在 1 页以内 — 投资组合经理和交易员没时间看更多内容

## 注意事项

- **观点鲜明**：那些只摘要新闻而没有观点的纪要是毫无价值的。
- **开门见山**：把最重要的信息放在最前面，不要埋没重点标题。
- **“平安无事”也是一条纪要**：可以说“隔夜无实质性变动，维持现有仓位配置”。
- **区分信号与噪音**：分清可操作的事件（如业绩、并购）和市场噪音（如次要的分析师笔记、无关痛痒的事件）。
- **标注时间戳**：如果你是在早上 6 点撰写的纪要，请注明盘前走势在开盘时可能会发生变化。
- **敢于认错**：如果你看错了，请在下一份早间纪要中诚实交代——信誉比每次都猜对更重要。

> **💡 Appendix: 领域知识小贴士**
>
> 欢迎来到充满活力的股票研究世界！如果你是理财菜鸟，这里有几个锦囊帮你秒懂这份文档：
>
> 1. **早间纪要 (Morning Note) 是干嘛的？**
>    想象一下，股市就像一场 24 小时不停歇的马拉松。在每天早上交易所开门前，分析师要把昨晚（特别是美股隔夜）发生的所有大事浓缩成一份“通关攻略”，告诉那些手握亿万资金的投资组合经理（PM）：「嘿，昨晚这几家公司出事了，今天咱得这么买！」
>
> 2. **业绩惊喜：超预期 vs. 逊预期 (Beat/Miss)**
>    市场就像个“爱操心的大合唱”，大家对一家公司下季度的表现会有一个平均的猜测，这叫“共识预期”。如果公司公布的成绩比大家想的还好，就是 Beat (超预期)；如果考砸了，就是 Miss (逊预期)。这通常是股价暴涨或暴跌的直接导火索。
>
> 3. **业绩指引 (Guidance)**
>    财报是“过去式”，而业绩指引是公司对“未来”的自述。就像考试成绩发下来了，顺便再说一句：“下学期我打算考全校第一。”即使这次考得一般，如果指引给力，大家也会觉得有盼头，股价照样能涨。
>
> 4. **催化因素 (Catalyst)**
>    也就是能让股价“动起来”的火药。光有好的投资逻辑是不够的，你还得需要一个事件（比如新手机发布会、财报日）来引爆市场情绪，让逻辑变现。
>
> 5. **做多/做空 (Long/Short)**
>    简单说：做多就是“看好，买入等涨”；做空就是“看衰，借来先卖，等跌了再买回来还”，目的都是为了赚差价。
