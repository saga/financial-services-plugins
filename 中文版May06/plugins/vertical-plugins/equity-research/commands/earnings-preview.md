> **💡 Appendix: 领域知识小贴士 (对于金融初学者的背景补充)**
>
> 1. **Earnings / Pre-earnings Preview (财报 / 财报前瞻)**:
>    * **概念**：上市公司每季度必须要向大众公布自己的“成绩单”（财务报表）。在成绩单正式公布前，金融分析师们会写一份“预测报告”，也就是“财报前瞻”。
>    * **作用**：告诉投资者这份即将到来的财报大概率是“惊吓”还是“惊喜”，帮助基金经理决定在公布前是进场还是离场。
>
> 2. **Ticker (股票代码)**:
>    * **概念**：每家上市公司的唯一字母缩写代号。比如苹果公司是 `AAPL`，微软是 `MSFT`。
>
> 3. **Consensus Estimates (市场一致预期)**:
>    * **概念**：相当于“市场上所有专业人士对某只股票业绩的平均预测值”。在华尔街，通常会有几十个分析师分别预测，把这些预测值取一个平均数（或中位数），就构成了“一致预期”。
>    * **为什么重要**：在资本市场里，“符合预期”往往引不起波澜。如果公司赚了100亿，但一致预期是120亿，股票依然会大跌（这叫 **Miss / 不及预期**）；如果只赚了80亿，但预期是退市破产，股票反而可能大涨（这叫 **Beat / 超出预期**）。
>
> 4. **Bull / Base / Bear Scenarios (乐观/基准/悲观情景，即 牛市/盘整/熊市 假设)**:
>    * **Base (基准)**：大家一般认为最有可能发生的情况。
>    * **Bull (乐观/牛)**：如果公司表现极好（例如新产品大卖），最好的情况会怎样。
>    * **Bear (悲观/熊)**：如果公司表现极差（例如供应链断裂），最差的情况会跌到哪里。
>    * 金融分析师很少只给出一个死板的数字，而是给出这三个维度的测算，以评估风险和收益的边界。
