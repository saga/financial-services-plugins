# 客户审查准备 (Client Review Prep)

description: 为客户审查会议准备投资组合绩效摘要、配置分析、讨论要点和行动项目。将账户数据整合到简洁的会议就绪格式中。用于季度审查、年度体检或临时客户会议之前。触发词："客户审查"、"为[客户]准备会议"、"季度审查"、"为[客户姓名]准备"或"客户会议"。

## 工作流程

### 步骤1：客户背景

收集或查找：
- **客户姓名**和家庭成员
- **账户类型**：应税账户、IRA、Roth、401(k)、信托等
- **总资产管理规模(AUM)**
- **投资政策声明(IPS)**：目标配置、风险承受能力、约束条件
- **人生阶段**：积累期、退休前、退休期、遗产规划
- **上次会议日期**和任何未完成的行动项目

### 步骤2：投资组合绩效

每个账户和家庭汇总：

| 指标 | 本季度 | 年初至今 | 1年 | 3年 | 成立至今 |
|------|--------|----------|-----|-----|----------|
| 投资组合回报 | | | | | |
| 基准回报 | | | | | |
| Alpha | | | | | |

**绩效归因：**
- 哪些资产类别/持仓驱动了回报？
- 前3大贡献者和前3大拖累者
- 任何单一持仓的过大影响？

### 步骤3：配置审查

当前vs目标配置：

| 资产类别 | 目标 | 当前 | 偏离 | 行动 |
|----------|------|------|------|------|
| 美国大盘股 | | | | |
| 美国中小盘 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 固定收益 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记任何超出IPS再平衡阈值（通常3-5%）的偏离。

### 步骤4：讨论要点

生成会议议程：

1. **市场概述**（2-3分钟）：简要宏观背景和展望
2. **投资组合绩效**（5分钟）：表现如何？为什么？
3. **配置审查**（5分钟）：需要再平衡吗？
4. **规划更新**（5-10分钟）：
   - 生活变化？（工作、健康、家庭、住房、教育）
   - 收入需求变化？
   - 税务状况更新
   - 遗产规划更新
5. **行动项目**（5分钟）：下次会议前我们做什么？

### 步骤5：主动建议

根据审查，建议：
- 再平衡交易（如果偏离超过阈值）
- 税务亏损收割机会
- 现金部署或取款规划
- Roth转换机会（如适用）
- 受益人更新或遗产规划需求
- 保险审查（人寿、残疾、长期护理）

### 步骤6：输出

- 一页客户审查摘要（Word或PDF）
- 带基准的绩效表
- 配置饼图（当前vs目标）
- 建议的行动项目
- 会议议程

## 重要提示

- 会前了解客户——回顾上次会议记录
- 以客户关心的内容开头，而不是你想谈论的内容
- 如果表现不佳，直接解决——不要隐藏或粉饰
- 始终以明确的行动项目和带日期的下一步结束
- 记录会议记录和IPS的任何变更
- 合规：确保所有材料符合公司政策和监管要求

## 金融术语和知识解释

### 1. 客户审查会议 (Client Review Meeting)

**定义**：财富顾问与客户定期举行的会议，审查投资组合表现和财务规划。

**目的**：
- 审查投资绩效
- 更新财务目标
- 调整投资策略
- 加强客户关系

**频率**：
- 季度审查（最常见）
- 年度全面审查
- 重大生活事件时临时会议

### 2. 资产管理规模 (Assets Under Management, AUM)

**定义**：财富管理公司管理的客户资产总价值。

**计算范围**：
- 投资账户
- 退休账户
- 信托账户
- 其他托管资产

**重要性**：
- 衡量公司规模
- 通常与费用收入相关

### 3. 风险承受能力 (Risk Tolerance)

**定义**：客户愿意承受的投资波动程度。

**评估维度**：
| 维度 | 描述 |
|------|------|
| 财务能力 | 承受损失的财务能力 |
| 心理承受 | 面对波动的心理舒适程度 |
| 投资期限 | 投资时间范围 |

**分类**：
- 保守型
- 稳健型
- 积极型
- 激进型

### 4. 人生阶段规划 (Life Stage Planning)

**定义**：根据客户的人生阶段制定财务规划。

**主要阶段**：

| 阶段 | 年龄范围 | 重点 |
|------|----------|------|
| 积累期 | 20-40岁 | 储蓄、投资增长 |
| 高峰期 | 40-60岁 | 最大化储蓄、风险管理 |
| 退休前 | 55-65岁 | 资产配置调整、退休收入规划 |
| 退休期 | 65岁+ | 收入提取、遗产规划 |

### 5. 配置偏离 (Allocation Drift)

**定义**：投资组合实际配置与目标配置的偏差。

**原因**：
- 不同资产类别的表现差异
- 现金流入/流出
- 市场波动

**管理**：
- 设置偏离阈值（通常3-5%）
- 定期再平衡
- 监控触发条件

### 6. 税务亏损收割 (Tax-Loss Harvesting)

**定义**：通过出售亏损资产来抵消资本利得税的策略。

**机制**：
1. 识别亏损资产
2. 出售实现亏损
3. 用亏损抵消资本利得
4. 购买类似但不相同的资产维持配置

**规则**：
- 避免洗售规则（30天内不得回购相同资产）
- 适用于应税账户
- 可结转未使用亏损

### 7. Roth转换 (Roth Conversion)

**定义**：将传统IRA或401(k)转换为Roth IRA的过程。

**特点**：
- 转换时缴纳所得税
- 未来收益免税
- 无强制提款要求

**优势**：
- 税务多元化
- 退休后免税收入
- 遗产规划优势

**考虑因素**：
- 当前税率vs预期未来税率
- 转换成本的资金来源
- 时间范围

### 8. 财务目标 (Financial Goals)

**定义**：客户希望通过投资实现的具体目标。

**常见目标**：
| 目标类型 | 示例 |
|----------|------|
| 退休 | 积累退休资产 |
| 教育 | 子女大学费用 |
| 购房 | 首付储蓄 |
| 遗产 | 财富传承 |

**SMART原则**：
- Specific（具体）
- Measurable（可衡量）
- Achievable（可实现）
- Relevant（相关）
- Time-bound（有时限）

### 9. 流动性需求 (Liquidity Needs)

**定义**：客户在特定时间范围内需要现金的金额。

**分类**：
- 紧急备用金（3-6个月支出）
- 短期目标（1-3年）
- 中期目标（3-10年）
- 长期目标（10年+）

**影响**：
- 决定现金储备水平
- 影响资产配置（流动性vs收益性）

### 10. 受益人指定 (Beneficiary Designation)

**定义**：指定在账户持有人去世后接收账户资产的个人或实体。

**重要性**：
- 避免遗嘱认证
- 确保资产按意愿分配
- 可能有税务优势

**类型**：
- 主要受益人
- 次要受益人
- 可撤销vs不可撤销
