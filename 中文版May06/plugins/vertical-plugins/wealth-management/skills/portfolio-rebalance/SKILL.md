# 投资组合再平衡 (Portfolio Rebalance)

description: 分析投资组合配置偏离并生成跨账户的再平衡交易建议。考虑税务影响、交易成本和洗售规则。触发词："再平衡"、"投资组合偏离"、"配置检查"、"再平衡交易"或"我的投资组合失衡"。

## 工作流程

### 步骤1：当前状态

对于每个账户，捕获：
- 账户类型（应税、IRA、Roth、401k）
- 当前市值持仓
- 成本基础（应税账户）
- 每个持仓的未实现收益/损失

### 步骤2：偏离分析

将当前配置与IPS目标进行比较：

| 资产类别 | 目标% | 当前% | 偏离 | 超额/不足金额 |
|----------|-------|-------|------|--------------|
| 美国大盘股 | | | | |
| 美国中小盘 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 投资级债券 | | | | |
| 高收益/信贷 | | | | |
| TIPS/通胀保护 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记超过再平衡区间（通常±3-5%）的持仓。

### 步骤3：交易建议

生成交易以使配置回到目标：

**税务意识再平衡规则：**
- 优先在税收优惠账户（IRA、Roth）中再平衡——无税务后果
- 在应税账户中，避免卖出具有大幅短期收益的持仓
- 在再平衡时尽可能收割亏损
- 注意洗售规则（30天窗口期）跨所有账户
- 考虑将新缴款引导至低配资产类别而非交易

**交易清单：**

| 账户 | 行动 | 证券 | 股数/金额 | 原因 | 税务影响 |
|------|------|------|----------|------|----------|
| | 买入/卖出 | | | 再平衡/税务亏损收割 | 短期收益/长期收益/亏损 |

### 步骤4：资产位置审查

优化哪些资产持有在哪些账户类型中：
- **递延税(IRA/401k)**：债券、REITs、高换手率基金（最高税务拖累）
- **Roth**：最高预期增长资产（免税增长）
- **应税**：税收效率高的权益（指数基金、ETF、市政债券）、税务亏损收割候选

### 步骤5：实施

- 按账户划分的交易总数
- 预计交易成本
- 预计税务影响（已实现收益/损失）
- 对配置偏离的净影响

### 步骤6：输出

- 偏离分析表
- 建议交易清单（Excel）
- 税务影响摘要
- 再平衡前后配置比较

## 重要提示

- 不要为了再平衡而再平衡——区间内的小幅偏离是可以的
- 在应税账户中，税务成本可能超过再平衡收益——计算盈亏平衡点
- 在交易前考虑待处理现金流（缴款、取款、RMDs）
- 检查任何客户特定限制（ESG、集中持股、锁定期）
- 为合规记录记录每笔交易的理由
- 洗售规则适用于跨账户——协调家庭所有账户的交易

## 金融术语和知识解释

### 1. 投资组合再平衡 (Portfolio Rebalancing)

**定义**：调整投资组合持仓以维持目标资产配置的过程。

**目的**：
- 控制风险水平
- 维持目标风险敞口
- 实现低买高卖的纪律

**方法**：
| 方法 | 描述 |
|------|------|
| 日历再平衡 | 定期（季度/年度） |
| 阈值再平衡 | 偏离达到一定程度时 |
| 现金流再平衡 | 通过新资金调整 |

### 2. 配置偏离 (Allocation Drift)

**定义**：投资组合实际配置与目标配置的偏差。

**原因**：
- 不同资产类别的表现差异
- 市场波动
- 现金流入/流出

**示例**：
- 目标：60%股票/40%债券
- 股票上涨后：65%股票/35%债券
- 偏离：+5%股票，-5%债券

### 3. 洗售规则 (Wash Sale Rule)

**定义**：美国税法禁止在卖出亏损资产后30天内回购相同或"实质相同"资产以获得税收优惠。

**规则细节**：
- 30天前/后窗口期
- 适用于所有账户（包括IRA）
- 实质相同的证券包括同一公司的不同份额类别

**后果**：
- 亏损不能立即扣除
- 亏损添加到新购买资产的成本基础
- 持有期顺延

### 4. 税务亏损收割 (Tax-Loss Harvesting)

**定义**：通过出售亏损资产来抵消资本利得税的策略。

**流程**：
1. 识别亏损资产
2. 出售实现亏损
3. 购买类似但不相同的资产维持配置
4. 用亏损抵消资本利得

**优势**：
- 减少当前税负
- 提高税后回报
- 亏损可结转至未来年度

### 5. 资产位置 (Asset Location)

**定义**：根据账户的税收待遇战略性地放置资产。

**优化策略**：

| 资产类型 | 最佳账户 | 原因 |
|----------|----------|------|
| 高收益债券 | IRA/401k | 利息收入税率高 |
| REITs | IRA/401k | 分配通常按普通收入征税 |
| 高换手率基金 | IRA/401k | 频繁交易产生资本利得 |
| 指数基金/ETF | 应税账户 | 税收效率高 |
| 市政债券 | 应税账户 | 利息免税 |
| 成长型股票 | Roth | 长期增长免税 |

### 6. 成本基础 (Cost Basis)

**定义**：资产的原始购买价格，用于计算资本利得或损失。

**计算方法**：
```
资本利得/损失 = 卖出价格 - 成本基础
```

**调整项**：
- 佣金和费用
- 股票分割
- 股息再投资
- 洗售亏损结转

**重要性**：
- 确定税务负债
- 影响投资决策

### 7. 未实现收益/损失 (Unrealized Gain/Loss)

**定义**：资产当前市值与成本基础之间的差额（未卖出时）。

**特点**：
- 未实现：未产生税务后果
- 浮动：随市场价格变化
- 纸面收益/损失

**示例**：
- 成本基础：$100
- 当前价格：$150 → 未实现收益$50
- 当前价格：$80 → 未实现损失$20

### 8. 已实现收益/损失 (Realized Gain/Loss)

**定义**：卖出资产后确认的收益或损失。

**税务影响**：
- **短期收益**（持有<1年）：按普通收入税率征税
- **长期收益**（持有≥1年）：按优惠税率征税（通常15-20%）

**税务筹划**：
- 利用亏损抵消收益
- 考虑持有期
- 年度税务损失限额

### 9. 交易成本 (Transaction Costs)

**定义**：执行投资交易产生的费用。

**组成部分**：
- 佣金
- 买卖差价
- 市场影响
- 机会成本

**最小化策略**：
- 使用低成本经纪商
- 批量交易
- 减少交易频率
- 使用ETF（通常成本低于共同基金）

### 10. 税收优惠账户 (Tax-Advantaged Accounts)

**定义**：享受特殊税收待遇的投资账户。

**主要类型**：

| 账户类型 | 税收待遇 | 适用人群 |
|----------|----------|----------|
| 传统IRA | 税前缴款，递延纳税 | 有收入的个人 |
| Roth IRA | 税后缴款，免税增长 | 收入低于限额的个人 |
| 401(k) | 税前缴款，递延纳税 | 雇主计划参与者 |
| 529计划 | 税后缴款，教育支出免税 | 教育储蓄 |

**优势**：
- 税收递延或免税增长
- 更高的税后回报
- 长期财富积累加速
