# 投资建议 (Investment Proposal)

description: 为潜在客户创建专业的投资建议。涵盖公司方法、拟议配置、预期结果和费用结构。用于向新客户推介或展示新投资策略。触发词："投资建议"、"潜在客户演示"、"向新客户推介"、"为[客户]准备建议"或"新客户演示"。

## 工作流程

### 步骤1：潜在客户背景

收集：
- **潜在客户姓名**和家庭详情
- **当前状况**：现有顾问？自主管理？什么促使会面？
- **资产**：预计资产管理规模、账户类型、当前持仓（如共享）
- **目标**：退休、财富保全、增长、收入、教育、遗产
- **风险承受能力**：保守、稳健、积极（或问卷分数）
- **约束条件**：ESG偏好、集中持股、流动性需求
- **费用敏感性**：他们现在支付多少？
- **竞争对手**：他们还在考虑谁？

### 步骤2：建议结构

**I. 关于我们公司**（1页）
- 公司概述、历史、资产管理规模
- 投资理念（用简单英语）
- 团队介绍（与该客户相关）
- 客户服务模式（多久会面一次，联系谁）

**II. 理解您的需求**（1页）
- 重申他们的目标和关注点——展示你在倾听
- 在发现阶段确定的关键规划考虑因素
- 对他们来说成功是什么样子

**III. 拟议投资策略**（2-3页）
- 推荐的资产配置及理由
- 配置如何映射到他们的目标和风险承受能力
- 投资工具（ETF、共同基金、个人证券、另类投资）
- 税务意识策略（资产位置、税务亏损收割）

拟议配置：

| 资产类别 | 配置 | 工具 | 理由 |
|----------|------|------|------|
| | | | |

**IV. 预期结果**（1-2页）
- 预计增长情景（保守、稳健、乐观）
- 达成目标的蒙特卡洛概率
- 收入预测（如退休或收入导向）
- 风险指标（最大回撤、波动率）
- 与当前投资组合比较（如已知）

**V. 费用结构**（1页）
- 咨询费用表（如有适用的分级）
- 底层基金费用
- 总费用估算
- 与行业平均水平的比较
- 价值主张——他们付费获得什么

**VI. 开始行动**（1页）
- 开户流程
- 资产转移时间表
- 过渡计划（如从其他顾问转移）
- 前90天——预期事项
- 所需文件和下一步

### 步骤3：定制化

- 根据潜在客户调整语气（企业高管vs小企业主vs退休人员）
- 如果他们有集中持股，直接解决
- 如果他们将你与机器人顾问比较，强调规划和关系价值
- 如果他们对价格敏感，以总价值和结果为导向，而不仅仅是费用

### 步骤4：输出

- PowerPoint演示（12-15张幻灯片），带公司品牌
- PDF留档版本
- 用于后续电子邮件的一页摘要

## 重要提示

- 建议应感觉个性化，而不是模板化——引用他们的具体情况
- 不要过度推销业绩——设定现实期望并强调流程
- 始终包含免责声明（预测是假设的，过往业绩等）
- 过渡计划很重要——客户害怕更换顾问的干扰
- 48小时内跟进建议和明确的下一步
- 合规必须在向潜在客户展示前审查

## 金融术语和知识解释

### 1. 投资建议 (Investment Proposal)

**定义**：向潜在客户提出的正式投资方案，概述服务、策略和预期结果。

**目的**：
- 展示专业能力
- 建立信任
- 赢得新业务
- 传达价值主张

**关键要素**：
- 个性化内容
- 清晰的价值主张
- 现实的预期
- 透明的费用

### 2. 资产管理规模 (Assets Under Management, AUM)

**定义**：公司管理的客户资产总价值。

**计算**：
- 所有投资账户余额之和
- 包括经纪账户、退休账户、信托等

**重要性**：
- 衡量公司规模和信誉
- 通常与费用收入直接相关
- 影响服务能力和资源

### 3. 投资理念 (Investment Philosophy)

**定义**：指导投资决策的核心原则和信念。

**常见理念**：
| 理念 | 描述 |
|------|------|
| 价值投资 | 寻找被低估的资产 |
| 成长投资 | 投资高增长公司 |
| 指数投资 | 跟踪市场指数 |
| 因子投资 | 基于特定因子构建组合 |

**重要性**：
- 一致性决策框架
- 客户预期管理
- 差异化定位

### 4. ESG投资 (ESG Investing)

**定义**：将环境、社会和治理因素纳入投资决策。

**ESG维度**：
| 维度 | 内容 |
|------|------|
| 环境 | 碳排放、资源使用、环保政策 |
| 社会 | 员工待遇、社区关系、产品安全 |
| 治理 | 董事会结构、高管薪酬、透明度 |

**策略**：
- 筛选排除（负面筛选）
- 最佳-in-class选择
- 积极参与/股东倡导

### 5. 集中持股 (Concentrated Stock Position)

**定义**：单一股票占投资组合很大比例。

**风险**：
- 非系统性风险集中
- 单一公司风险
- 流动性问题

**管理策略**：
| 策略 | 描述 |
|------|------|
| 逐步出售 | 定期卖出降低集中度 |
| 对冲策略 | 使用期权保护 |
| 10b5-1计划 | 预设卖出计划 |
| 慈善捐赠 | 捐赠股票获得税收优惠 |

### 6. 机器人顾问 (Robo-Advisor)

**定义**：基于算法提供自动化投资建议的在线平台。

**特点**：
- 低成本
- 自动化资产配置
- 有限的人工咨询
- 适合被动投资

**与传统顾问对比**：
| 方面 | 机器人顾问 | 传统顾问 |
|------|-----------|----------|
| 费用 | 低 | 高 |
| 个性化 | 有限 | 高 |
| 规划服务 | 基础 | 全面 |
| 人际互动 | 少 | 多 |

### 7. 蒙特卡洛概率 (Monte Carlo Probability)

**定义**：通过模拟多种情景计算达成目标的可能性。

**应用**：
- 评估退休资金可持续性
- 计算投资目标达成概率
- 量化风险

**解读**：
- 80-90%：高信心
- 60-80%：中等信心
- <60%：需要调整

### 8. 最大回撤 (Maximum Drawdown)

**定义**：投资组合从峰值到谷底的最大跌幅。

**计算公式**：
```
最大回撤 = (谷底价值 - 峰值价值) / 峰值价值
```

**重要性**：
- 衡量下行风险
- 评估投资组合韧性
- 管理客户预期

**示例**：
- 从100万跌至70万 → 回撤30%

### 9. 咨询费用 (Advisory Fee)

**定义**：财富顾问为提供投资建议收取的费用。

**常见结构**：
- 基于资产管理规模的百分比（最常见）
- 固定费用
- 小时费率

**分级费率**：
```
AUM < $1M: 1.00-1.50%
$1M - $5M: 0.75-1.00%
$5M - $10M: 0.50-0.75%
> $10M: 0.25-0.50%
```

**费用透明度**：
- 披露所有费用
- 区分咨询费和底层基金费用
- 说明服务价值

### 10. 资产转移 (Asset Transfer)

**定义**：将投资账户从一个机构转移到另一个机构。

**流程**：
1. 开立新账户
2. 发起转移请求
3. 资产清算或转移
4. 确认完成

**类型**：
- **ACATS转移**：自动化客户账户转移服务
- 清算转移：卖出资产后转移现金

**考虑因素**：
- 转移时间（通常3-7天）
- 税务影响
- 账户类型兼容性
