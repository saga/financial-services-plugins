# 可比公司分析指令 (Comps Command)

用于构建具备对冲基金级水准的“可比公司分析”模型，应包含运营指标对比横切、核心估值乘数拆解以及跨分位数的统计分布基准线。

## 工作流 (Workflow)

### 第 1 步：探查并收集标的公司 (Gather Company Information)
若未提供特定参数，首先询问核心意图：“您确切希望给哪一家公司开刀构建可比池？”

### 第 2 步：加载 Comps 分析功能树 (Load Comps Analysis Skill)
调用底层 `skill: "comps-analysis"` 进行逻辑排查计算：
1. **确认分析目的靶点**: 此举是为了敲定估值上限、盘点渠道效率，还是寻找被市场错杀的低光点？读者受众是谁？
2. **筛选核心同行标的池 (Peer group)**: 精心挑选出 4-6 家真正在业务盈利模式上同频、市值体量相似的大众标的。
3. **搜集核心交易口径指标**:
   - 运营截面：过往/远期营收表现、超预期的同比增长率 (Growth)、毛利率瓶颈、EBITDA 获利表现。
   - 价格截面：总市值天花板、账上的企业实际支付价值 (EV)、EV/Revenue 流水倍数、 EV/EBITDA 乘数、P/E (最后扣非市盈率倍数)。
4. **构建跨越式横评矩阵表**: 从最高溢价封包到低位兜底：展示所有同行的绝对最大值、优越段75分位 (75th Percentile)、中位数 (Median)、及底层25分位。

### 第 3 步：生成并落盘 Excel (Create Excel Output)
自动在内网推流生成结构规范、具有可追溯引用来源注释、无野战强改硬编的专业表册 (.xlsx)。

### 第 4 步：交付总结
输出完整数字模型，并同时向用户生成一份执行高赞的结论速览摘要（点出哪些同行被严重高估，行业中位数的黄金分界何在）。

---
> **💡 Appendix: 领域知识小贴士**
> 1. **Comps (可比公司分析 / 估值乘数法)**: 最普及使用的主流相对估值手段。简单来说就是在二手市场买房时，只要看看附近同档次的另外三个小区最近别人用什么单价成交买走，就能反向推估你眼前的这套房到底合不合算。
> 2. **LTM (Last Twelve Months / 最近过去十二个月)**: 评估一家具有季节周期的公司时，单独测去年底的老旧报表或单一旺季表都是可悲的骗局。合并取用它离今天最近的前连续 4 个季度（足足十二月），就是消除盲点的 LTM 手段。
