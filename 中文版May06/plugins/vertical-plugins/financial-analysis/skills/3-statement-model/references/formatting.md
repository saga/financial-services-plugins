# 财务建模格式规范指南

| 元素 | 格式规范 |
|---------|--------|
| 硬编码输入值 (Hard-coded inputs) | 蓝色字体 |
| 公式 (Formulas) | 黑色字体 |
| 跨表链接 (Links to other sheets) | 绿色字体 |
| 检查项单元格 (Check cells) | 报错显示红色，平稳显示绿色 |
| 负值 (Negative values) | 使用括号表示，不使用负号 |
| 货币 (Currency) | 大额数值不保留小数，每股收益保留 2 位小数 |
| 百分比 (Percentages) | 保留 1 位小数 |
| 表头 (Headers) | 加粗，带下框线 |
| 单位行 (Units row) | 在表头下方标注单位（百万元、$ millions、% 等） |

## 视觉分隔准则

- 历史年度与预测年度列之间使用细垂直边框分隔
- 各部分总计后使用粗下框线（例如：总资产）
- 小计使用单下框线
- 总计使用双下框线

## 总计与小计行格式

所有总计和小计行对应的数值必须使用**加粗字体**，以便清晰区分汇总数据与具体明细项。

### 利润表 (P&L) 选项卡
| 行项目 | 格式规范 |
|-----|------------|
| 营业总收入 (Gross Revenue) | 加粗 |
| 营收总成本 (Total Cost of Revenue) | 加粗 |
| 毛利 (Gross Profit) | 加粗 |
| 销售及行管费用总计 (Total SG&A) | 加粗 |
| EBITDA (息税折旧摊销前利润) | 加粗 |
| EBIT (息税前利润) | 加粗 |
| EBT (税前利润) | 加粗 |
| 税后净利润 (Net Profit After Tax) | 加粗 |

### 资产负债表 选项卡
| 行项目 | 格式规范 |
|-----|------------|
| 流动资产总计 (Total Current Assets) | 加粗 |
| 非流动资产总计 (Total Non-Current Assets) | 加粗 |
| 其他资产总计 (Total Other Assets) | 加粗 |
| 资产总计 (Total Assets) | 加粗 |
| 流动负债总计 (Total Current Liabilities) | 加粗 |
| 非流动负债总计 (Total Non-Current Liabilities) | 加粗 |
| 权益总额 (Total Equity) | 加粗 |
| 负债及权益总计 (Total Liabilities and Equity) | 加粗 |

### 现金流量表 选项卡
| 行项目 | 格式规范 |
|-----|------------|
| 营运资本变动前经营活动现金流 | 加粗 |
| 营运资本变动总计 | 加粗 |
| 经营活动产生的现金流量净额 | 加粗 |
| 投资活动产生的现金流量净额 | 加粗 |
| 筹资活动产生的现金流量净额 | 加粗 |
| 期末现金余额 | 加粗 |

**注：** 此列表未尽所有项目。凡是代表总计、小计或模型中汇总计算的行，均应应用加粗格式。

## 资产负债表平衡检查行格式

资产负债表检查行（位于负债及权益总计下方）需应用条件格式，非零值显示为红色。当资产负债表正确平稳（检查值 = 0）时，数值显示为黑色或标准格式。

| 检查值状态 | 字体颜色 |
|-------------|------------|
| = 0 (平衡/配平) | 黑色 (标准) |
| ≠ 0 (报错) | 红色 |

**实现方式：** 应用自定义数字格式 `[Red][<>0]0.00;[Red][<>0](0.00);0.00` 或使用 Excel 条件格式规则：“单元格数值 ≠ 0” → 红色字体。

## 利润率行格式

| 元素 | 格式规范 |
|---------|--------|
| 利润率 % 行 | 缩进，斜体，保留 1 位小数 |
| 正向趋势 | 无需特殊格式（或浅绿色） |
| 负向趋势 | 标记需审核（浅黄色） |
| 低于行业平均水平 | 考虑高亮以备讨论 |

## 信用指标格式

| 元素 | 格式规范 |
|---------|--------|
| 杠杆倍数 | 保留 1 位小数并带“x”后缀（如 2.5x） |
| 百分比 | 保留 1 位小数并带“%”后缀 |
| 净债务为负值 | 用括号表示，代表处于净现金状态 |
| 部分表头 | 加粗，“CREDIT METRICS” |
| 分隔线 | 信用指标部分上方留细边框 |

## 信用指标阈值颜色

| 指标 | 绿色 (安全) | 黄色 (预警) | 红色 (风险) |
|--------|-------|--------|-----|
| 总债务 / EBITDA | < 2.5x | 2.5x-4.0x | > 4.0x |
| 净债务 / EBITDA | < 2.0x | 2.0x-3.5x | > 3.5x |
| 利息保障倍数 | > 4.0x | 2.5x-4.0x | < 2.5x |
| 负债 / 总资本 | < 40% | 40%-60% | > 60% |
| 流动比率 | > 1.5x | 1.0x-1.5x | < 1.0x |
| 速动比率 | > 1.0x | 0.75x-1.0x | < 0.75x |

## 检查 (Checks) 选项卡的条件格式

- 单元格包含“通过 (pass)”指示 → 绿色填充
- 单元格包含“失败 (fail)”指示 → 红色填充
- 单元格包含“警告 (warning)”指示 → 黄色填充
- 差额单元格 = 0 → 浅绿色填充
- 差额单元格 ≠ 0 → 浅红色填充

## 利润率合理性标记

- 毛利率 < 0% → 错误：检查营收成本 (COGS)
- 毛利率 > 80% → 警告：核实收入/营收成本
- EBITDA 利润率 < 0% → 标记：存在经营亏损
- EBITDA 利润率 > 50% → 警告：数值异常偏高
- 净利率 < 0% → 标记：净亏损（在增长阶段可能视为可接收）
- 净利率 > 毛利率 → 错误：公式计算逻辑错误
