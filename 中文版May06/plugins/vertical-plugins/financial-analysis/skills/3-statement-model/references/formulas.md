# 公式参考手册

**重要提示：** 除非用户另有说明，请务必使用本参考文档中列出的公式。

---

## 核心勾稽关系

```
资产负债表：        资产 = 负债 + 股东权益
净利润：           利润表净利润 → 经营活动现金流 (起点)
现金流量：         现金变动 = 经营现金流 + 投资现金流 + 筹资现金流
现金平账：         期末现金 (现金流量表) = 现金 (资产负债表资产项)
现金月度/年度：    期末现金 (月度) = 期末现金 (年度)
留存收益：         期初留存收益 + 净利润 - 股利 = 期末留存收益
股权融资：         普通股/资本公积变动 (资产负债表) = 股票发行 (筹资现金流)
第0年股权：        第0年募集的股权 = 第1年期初股权
```

## 毛利计算

**重要提示：** 毛利必须基于净收入（Net Revenue）计算，而非总收入（Gross Revenue）。

```
净收入 - 收入成本 = 毛利
```

| 术语 | 定义 |
|------|------------|
| 总收入 (Gross Revenue) | 扣除任何项目前的收入总额 |
| 净收入 (Net Revenue) | 总收入 - 退货 - 折让 - 折扣 |
| 收入成本 (Cost of Revenue) | 可直接归属于所销售产品/服务生产的直接成本 |
| 毛利 (Gross Profit) | 净收入 - 收入成本 |

**注意：** 在进行盈利能力计算时，始终以净收入（在大多数报表中也称为“净销售额”或简称为“收入”）作为起点。总收入会夸大真实的顶线表现。

## 利润率公式

```
毛利率 %            = 毛利 / 净收入
EBITDA              = EBIT + 折旧与摊销 (或 = 毛利 - 运营支出)
EBITDA 利润率 %     = EBITDA / 净收入
EBIT 利润率 %       = EBIT / 净收入
净利润率 %          = 净利润 / 净收入
```

## 信用指标公式

```
总债务              = 短期借款 + 长期借款
净债务              = 总债务 - 现金
总债务 / EBITDA      = 总债务 / EBITDA (取自利润表)
净债务 / EBITDA      = 净债务 / EBITDA (取自利润表)
利息覆盖率          = EBITDA / 利息支出 (取自利润表)
净利息支出 % 债务   = 净利息支出 / 长期借款
总杠杆率            = 总债务 / (总债务 + 总权益)
产权比率            = 总债务 / 总权益
流动比率            = 流动资产合计 / 流动负债合计
速动比率            = (流动资产合计 - 存货) / 流动负债合计
```

## 预测公式（净收入百分比法）

```
收入成本 (预测)      = 净收入 × 收入成本 % 假设
销售项 (预测)        = 净收入 × 销售项 % 假设
管理费 (预测)        = 净收入 × 管理费 % 假设
研发费 (预测)        = 净收入 × 研发费 % 假设
股权激励 (预测)      = 净收入 × 股权激励 % 假设
```

## 营运资金公式

```
应收账款 (AR)
  期初应收
  + 收入 (来自利润表)
  - 现金回收 (调和项)
  = 期末应收
  DSO (应收账款周转天数) = (应收 / 收入) × 365

存货 (Inventory)
  期初存货
  + 采购额 (调和项)
  - 销货成本 (来自利润表)
  = 期末存货
  DIO (存货周转天数) = (存货 / 销货成本) × 365

应付账款 (AP)
  期初应付
  + 采购额 (来自存货计算)
  - 现金支付 (调和项)
  = 期末应付
  DPO (应付账款周转天数) = (应付 / 销货成本) × 365

营运资金净额 = 应收账款 + 存货 - 应付账款
营运资金变动 (ΔWC) = 当前营运资金净额 - 上期营运资金净额
```

## 折旧与摊销表公式

```
期初 PP&E (原值)
+ 资本开支 (CapEx)
= 期末 PP&E (原值)

期初累计折旧
+ 折旧费用
= 期末累计折旧

PP&E (净值) = PP&E 原值 - 累计折旧
```

## 债务表公式

```
期初债务余额
+ 新增借款
- 偿还额
= 期末债务余额

利息支出 = 平均债务余额 × 利率
  (使用期初余额以避免循环引用，或在启用循环引用时进行迭代计算)
```

## 留存收益公式

```
期初留存收益
+ 净利润 (来自利润表)
+ 股权激励支付 (SBC) (来自利润表)
- 股利
= 期末留存收益
```

## 净经营亏损 (NOL) 表公式

```
NOL 结转表

期初 NOL 余额 (第1年 / 成立时 = 0)
+ 新产生的 NOL (如果 EBT < 0，则取 ABS(EBT)，否则为 0)
- 已抵用的 NOL (受限于应纳税所得额及抵用上限)
= 期末 NOL 余额

初始余额规则

对于新业务或第一个建模期间：
  期初 NOL 余额 = 0
  NOL 只能通过实现的亏损 (EBT < 0) 增加
  NOL 不能凭空产生或假设

NOL 抵用计算

税前利润 (EBT)
  如果 EBT > 0：
    可用 NOL = 期初 NOL 余额
    抵用上限 = EBT × 80% (2017年后联邦限制)
    已抵用 NOL = MIN(可用 NOL, 抵用上限)
    应纳税所得额 = EBT - 已抵用 NOL
  如果 EBT ≤ 0：
    已抵用 NOL = 0
    应纳税所得额 = 0
    新产生的 NOL = ABS(EBT)

带 NOL 的税费计算

应交所得税 = MAX(0, 应纳税所得额 × 税率)
  (税额不能为负；亏损会产生 NOL 资产)

NOL 对应的递延所得税资产 (DTA)

DTA - NOL 结转 = 期末 NOL 余额 × 税率
ΔDTA = 本期 DTA - 上期 DTA
  (DTA 增加 = 现金流量表中的非现金收益)
  (DTA 减少 = 现金流量表中的非现金支出)
```

## 资产负债表结构

```
资产
  现金 (来自现金流量表期末现金)
  应收账款 (来自营运资金)
  存货 (来自营运资金)
  流动资产合计
  
  PP&E 净值 (来自折旧摊销表)
  递延所得税资产 - NOL (来自 NOL 表)
  非流动资产合计
  资产总计

负债
  应付账款 (来自营运资金)
  一年内到期的长期借款 (来自债务表)
  流动负债合计
  
  长期借款 (来自债务表)
  负债合计

股东权益
  普通股
  留存收益 (来自留存收益表)
  股东权益合计

平衡校验：资产 - 负债 - 股东权益 = 0
```

## 现金流量表结构

```
经营活动现金流 (CFO)
  净利润 (关联：利润表)
  + 折旧与摊销 (关联：折旧摊销表)
  + 股权激励支付 (SBC) (关联：利润表或假设)
  - ΔDTA (递延所得税资产) (关联：NOL 表；DTA 增加代表现金流出)
  - ΔAR (关联：营运资金)
  - Δ存货 (关联：营运资金)
  + ΔAP (关联：营运资金)
  = CFO

投资活动现金流 (CFI)
  - 资本开支 (CapEx) (关联：折旧摊销表)
  = CFI

筹资活动现金流 (CFF)
  + 债务发行 (关联：债务表)
  - 债务偿还 (关联：债务表)
  + 股票发行 (关联：资产负债表普通股/资本公积)
