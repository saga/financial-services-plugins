# DCF 模型排障指南

**何时阅读此文件：** 若 `recalc.py` 报错、估值结果不合理或情景选择器（Case Selector）运行异常。

## 模型返回错误值

### #REF! 错误
- 通常由于插入标题行后，公式引用的行号发生偏移所致
- 解决方案：重新建立正确的行引用，或遵循布局规划从头开始
- 预防措施：在编写公式**之前**先定义所有行位置

### #DIV/0! 错误
- 除数为零或引用了空单元格
- 解决方案：添加 IF 语句以处理零值状况：`=IF([除数]=0, 0, [被除数]/[除数])`

### #VALUE! 错误
- 计算中涉及错误的数据类型（如文本而非数字）
- 解决方案：验证所有输入项均已格式化为数字格式

## 估值结果看似不合理

### 隐含股价远高于预期
- 检查终值（Terminal Value）占比是否超过企业价值（EV）的 80%
- 验证永续增长率是否小于加权平均资本成本（WACC）
- 审查增长假设是否符合实际情况
- 评估利润率假设是否过于乐观

### 隐含股价远低于预期
- 验证净负债（Net Debt）与净现金（Net Cash）的核算是否准确
- 检查 WACC 是否设置过高
- 审查预测数据（Projections）是否过于保守
- 考虑永续增长率是否设定得过低

## 情景选择器失效

### 切换情景时，汇总列数据未更新
- 验证情景选择器单元格的值是否为 1、2 或 3
- 检查 INDEX/OFFSET 公式所引用的行范围及选择器单元格是否正确
- 确保在选择器引用中使用了绝对引用（如 $B$6）
- 通过手动更改选择器单元格的值，动态验证预测数据是否同步更新

> **💡 Appendix: 领域知识小贴士**
>
> **1. 什么是 DCF？**
> 想象你有一台能下金蛋的鹅，DCF（现金流折现）就是算算这只鹅未来一生能下多少蛋。考虑到“明天的钱不如今天的钱值钱”，我们需要把未来的蛋都按一定的折扣换算成现在的价值，加在一起就是这只鹅现在的身价。
>
> **2. WACC 是干嘛的？**
> 公司运作需要钱，要么找银行借，要么由股东投，这些钱都不是白用的。WACC（加权平均资本成本）就是公司使用这些资金的“平均利息”。在模型里，它就像一个“折扣率”，WACC 越高，未来的钱折算到现在就越不值钱。
>
> **3. 终值 (Terminal Value) 意味着什么？**
> 我们没法预测公司 100 年后的表现。所以我们通常精准预测前 5-10 年，剩下的日子就假设公司进入一个稳定收租的养老阶段。这“剩下的所有未来价值”打包在一起，就是终值。
>
> **4. 企业价值 (EV) 与股价**
> 企业价值（EV）是公司这门生意本身的总价值。如果你想买下整家公司，你不仅要付钱给股东（股权价值），还得帮公司把欠银行的债也给还掉。所以，我们要从总价值里扣掉欠款，剩下属于股东的那部分除以总股数，才是我们心心念念的“股价”。
