# 回报率分析

description: 为私募股权（PE）交易评估构建快速的 IRR/MOIC 敏感性分析表。在不同的入场乘数、杠杆水平、退出乘数、业绩增长率和持有期情境下进行回报建模。适用于初步评估交易规模、对假设条件进行压力测试或准备投资委员会（IC）的回报展示材料。触发指令包括（Triggers on）："returns analysis"、"IRR sensitivity"、"MOIC table"、"what's the return at"、"model the returns" 或 "back of the envelope"。

## 工作流程

### 步骤 1：收集交易输入数据

询问（或从先前的分析中提取）：

**入场假设（Entry）：**
- 入场 EBITDA（过去十二个月 LTM 或 未来十二个月 NTM）
- 入场乘数（企业价值 EV / EBITDA）
- 企业价值（EV）
- 交割时净债务（Net debt）
- 股权出资规模（Equity check size）
- 交易费用与支出

**融资安排（Financing）：**
- 优先债务（EBITDA 倍数、利率、摊销）
- 次级债务 / 夹层债务（如有）
- 入场总杠杆（EBITDA 倍数）
- 股权占比 / 股权贡献

**运营假设（Operating Assumptions）：**
- 营业收入增长率（年度）
- EBITDA 利润率轨迹
- 资本支出（Capex）占比（% 营业收入）
- 营运资金的变动
- 债务偿还计划

**退出假设（Exit）：**
- 持有期（年）
- 退出乘数（企业价值 EV / EBITDA）
- 退出 EBITDA（根据增长假设计算得出）

## 核心公式

- **MOIC（投资回报倍数）** = 退出股权价值 / 股权投资额
- **IRR（内部收益率）** = 求解 r：股权投资额 × (1 + r)^n = 退出股权价值（需根据期间现金流进行调整）
- **收益归因（Returns attribution）**：
  - 增长贡献：(退出 EBITDA - 入场 EBITDA) × 退出乘数 / 股权投资额
  - 乘数贡献：(退出乘数 - 入场乘数) × 入场 EBITDA / 股权投资额
  - 杠杆贡献：持有期间偿付的债务总额 / 股权投资额

## 重要注意事项

- 在适用情况下，务必同时展示扣除管理费与附带权益（Carry）前后的毛回报和净回报（Gross and net returns）。
- 管理层结转投资（Management rollover）和跟投（Co-invest）会改变实际的股权出资规模——需询问是否存在这种情况。
- 股息资本重组（Dividend recaps）或期间分配会显著影响 IRR——若有计划请务必纳入计算。
- 不要遗漏交易成本（通常为 EV 的 2-4%）——这会直接扣减第一天的股权价值。
- 税务考量（资产收购 vs 股权收购、338(h)(10) 税务选择）可能会对税后回报产生实质性影响。

> **💡 Appendix: 领域知识小贴士**
> 
> 嗨！如果你是金融模型新手，不要被满篇的英文缩写吓到。这份文件其实在描述一个经典的**杠杆收购（LBO, Leveraged Buyout）**是如何算账赚钱的。简单来说，就是用“首付+贷款”买下一家公司，通过经营让它更值钱，同时用公司赚的钱还债，最后高价把公司卖掉获得丰厚利润。下面是几个核心概念的“通俗版”解释：
> 
> - **EBITDA（息税折旧摊销前利润）**：可以理解为公司“最纯粹的印钞能力”。它剥离了借款利息、交税以及非现金流出（折旧）的影响，让买家能一针见血地看出公司的主营业务究竟能捏出多少真金白银。
> - **企业价值（EV）与股权价值（Equity）**：把买公司想象成买房子。**企业价值 (EV)** 就是这套房子的“全款总价”。而**股权价值 (Equity)** 是你实际掏的“首付”（剩下的钱则是借来的“净债务 Net Debt”）。
> - **乘数（Multiple）**：买菜看“多少钱一斤”，买公司看“多少倍利润”。大家通常用 EV 是 EBITDA 的几倍来给公司定价。如果你花 $1000 万买了一家 EBITDA 为 $100 万的公司，那你的**入场乘数（Entry Multiple）**就是 10x。后来你把公司做大，以 12x 的倍数卖掉，这赚出来的2倍差价就叫**乘数扩张（Multiple Expansion）**。
> - **MOIC 与 IRR**：这是衡量你赚钱水平的两个黄金指标。
>   - **MOIC（投资回报倍数）**：简单粗暴地看绝对金额。投了1块钱，最后拿回3块钱，MOIC就是“3倍”。它不在乎你熬了多少年。
>   - **IRR（内部收益率）**：它是带有“时间魔法”的指标。它考虑了资金的时间成本。同样的3倍回报，如果你花3年做到，IRR会非常惊艳（大约44%）；但如果你花了10年才做到，IRR就显得平平无奇（大约11%）了。资本更喜欢跑得快的钱！
> - **收益瀑布图（Returns Waterfall）**：就像把赚到的钱“大卸八块”，清清楚楚地告诉你，这笔交易赚的每一分钱，到底是因为你把公司业绩做好了（增长），还是得益于市场估值水涨船高（乘数），又或者是仅仅因为用公司赚的钱还清了银行贷款，导致你自己的股权变厚了（杠杆）。
