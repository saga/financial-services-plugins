# 交易筛选 (Deal Screening)

description: 根据基金的投资标准快速筛选 inbound 交易流——CIM、teaser和经纪人材料。提取关键交易指标，运行通过/失败框架，并输出一页筛选备忘录。用于审查新交易流、分类 inbound 材料或决定是否进行首次电话。触发词："筛选此交易"、"审查此CIM"、"我们应该看这个吗"、"分类此teaser"或"交易筛选"。

## 工作流程

### 步骤1：提取交易事实

从提供的CIM、teaser或描述中提取：

- **公司**：名称、地点、行业/细分行业
- **描述**：他们做什么（1-2句话）
- **财务数据**：收入、EBITDA、利润率、增长率
- **交易类型**：平台、附加收购、资本重组、少数股权、分拆
- **要价/估值**：倍数、如已说明的企业价值
- **卖方动机**：为什么现在出售
- **管理层**：留任还是退出
- **主要客户**：集中度风险
- **主要风险**：明显的红旗

### 步骤2：根据标准筛选

应用基金的投资标准（如未知则询问用户）：

| 标准 | 目标 | 实际 | 通过/失败 |
|------|------|------|-----------|
| 收入范围 | | | |
| EBITDA范围 | | | |
| EBITDA利润率 | | | |
| 增长概况 | | | |
| 行业匹配 | | | |
| 地理位置 | | | |
| 交易规模/EV | | | |
| 估值(x EBITDA) | | | |
| 客户集中度 | | | |
| 管理层连续性 | | | |

### 步骤3：快速评估

提供三部分评估：

1. **结论**：通过 / 进一步尽职调查 / 拒绝
2. **看涨理由**（2-3点）：为什么这可能是一笔好交易
3. **看跌理由**（2-3点）：主要风险和关注点
4. **关键问题**：首次电话需要回答的问题

### 步骤4：输出

一页筛选备忘录，适合与合伙人分享或投资委员会快速筛选。

## 重要提示

- 速度很重要——筛选应该用分钟而不是小时
- 直接说明红旗。不要掩盖担忧
- 如果财务数据看起来不一致或不完整，明确标记
- 如果这是第一次筛选，提前询问基金的标准
- 确认后将筛选标准保存在记忆中以供未来交易使用

## 金融术语和知识解释

### 1. 交易筛选 (Deal Screening)

**定义**：在投入大量资源进行全面尽职调查之前，对潜在投资机会进行的初步评估。

**目的**：
- 快速识别不符合投资标准的交易
- 优先处理最有潜力的机会
- 优化资源配置

**筛选标准**：
- 行业聚焦
- 规模要求
- 财务指标门槛
- 地理位置偏好
- 交易结构兼容性

### 2. CIM (Confidential Information Memorandum)

**定义**：由卖方准备的详细介绍目标公司的机密文档。

**主要内容**：
- 执行摘要
- 公司概况
- 市场分析
- 财务报表（历史和预测）
- 管理团队
- 客户和产品
- 风险因素
- 交易条款建议

**用途**：
- 向潜在买家介绍业务
- 作为尽职调查的起点

### 3. Teaser

**定义**：在CIM之前提供的简短、非机密的公司介绍。

**特点**：
- 1-2页
- 匿名性（不披露公司名称）
- 突出关键亮点
- 激发兴趣

**内容要点**：
- 行业和地理位置
- 关键财务指标
- 业务亮点
- 交易机会类型

### 4. Inbound Deal Flow

**定义**：主动流向投资者的潜在投资机会。

**来源**：
- 投资银行/经纪人
- 直接联系
- 行业人脉
- 现有投资组合公司
- 中介机构

**管理要点**：
- 建立标准化筛选流程
- 及时响应
- 维护关系网络

### 5. 投资标准 (Investment Criteria)

**定义**：基金用于评估潜在投资的具体要求和偏好。

**常见标准**：

| 类别 | 示例 |
|------|------|
| 规模 | 收入$10M-$100M |
| 盈利能力 | EBITDA > $2M |
| 增长 | 年增长率 > 15% |
| 行业 | B2B SaaS、医疗保健 |
| 地理位置 | 北美、欧洲 |
| 交易类型 | 控股权、平台收购 |

**重要性**：
- 确保投资策略一致性
- 提高决策效率
- 管理LP期望

### 6. 交易类型 (Deal Type)

**定义**：投资的结构和目的。

**主要类型**：

| 类型 | 描述 | 特点 |
|------|------|------|
| 平台收购 | 作为行业整合基础的公司 | 规模较大，有整合潜力 |
| 附加收购 | 被平台公司收购的公司 | 规模较小，协同效应导向 |
| 成长型股权 | 少数股权投资 | 不控股，支持增长 |
| 资本重组 | 调整资本结构 | 通常涉及债务重组 |
| 分拆 | 从母公司分离 | 独立运营 |

### 7. 卖方动机 (Seller Motivation)

**定义**：卖方决定出售业务的原因。

**常见动机**：
- 退休
- 多元化投资组合
- 战略调整
- 财务压力
- 增长瓶颈

**重要性**：
- 影响谈判地位
- 揭示潜在风险
- 影响交易结构

### 8. 企业价值 (Enterprise Value, EV)

**定义**：公司的总价值，包括股权和债务。

**计算公式**：
```
EV = 股权价值 + 债务 - 现金及等价物
```

**用途**：
- 计算估值倍数（如EV/EBITDA）
- 比较不同资本结构的公司
- 确定收购价格

### 9. 估值倍数 (Valuation Multiple)

**定义**：用于衡量公司价值的比率。

**常见倍数**：

| 倍数 | 计算公式 | 适用场景 |
|------|----------|----------|
| EV/EBITDA | 企业价值/EBITDA | 大多数行业 |
| EV/Revenue | 企业价值/收入 | 高增长、未盈利公司 |
| P/E | 股价/每股收益 | 成熟盈利公司 |
| P/B | 股价/每股净资产 | 金融公司 |

**解读**：
- 高于行业平均：市场预期高增长或竞争优势
- 低于行业平均：增长预期低或风险较高

### 10. 首次电话 (First Call)

**定义**：投资者与目标公司管理层的首次沟通。

**目的**：
- 建立初步关系
- 验证基本信息
- 评估管理团队
- 确定下一步

**准备要点**：
- 阅读CIM/teaser
- 准备关键问题
- 了解行业背景
- 设定明确目标

**后续步骤**：
- 发送会议纪要
- 请求额外信息
- 决定是否继续尽职调查
