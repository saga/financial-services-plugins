# 投资组合监控 (Portfolio Monitoring)

description: 跟踪和分析投资组合公司的业绩与计划对比。接收月度/季度财务包（Excel、PDF），提取关键绩效指标，标记与预算的偏差，并生成汇总仪表板。用于审查投资组合公司财务状况、准备董事会材料或监控契约合规性。触发词："审查投资组合公司"、"月度财务"、"[公司]表现如何"、"契约检查"或"投资组合更新"。

## 工作流程

### 步骤1：接收财务包

- 接受用户的投资组合公司财务包（Excel工作簿、PDF或CSV）
- 提取关键财务数据：收入、EBITDA、现金余额、未偿还债务、资本支出、营运资本
- 确定报告期并与前期和预算/计划进行比较

### 步骤2：关键绩效指标提取与差异分析

需要跟踪的关键指标（根据公司行业调整）：

**财务关键绩效指标：**
- 收入vs预算（金额和百分比）
- EBITDA和EBITDA利润率vs预算
- 现金余额和净债务
- 杠杆比率（净债务/LTM EBITDA）
- 利息覆盖比率
- 资本支出vs预算
- 自由现金流

**运营关键绩效指标**（询问用户或从数据推断）：
- 客户数量/每位客户收入
- 员工人数/每位员工收入
- 未完成订单/销售渠道
- 流失率/留存率

### 步骤3：标记与汇总

- **绿色**：在计划的5%以内
- **黄色**：低于计划5-15%——标记讨论
- **红色**：低于计划>15%或存在契约违约风险——立即关注

输出简洁摘要：
1. 一段执行摘要（"X公司[超前/落后/符合]计划..."）
2. 关键绩效指标表，包含实际vs预算vs前期
3. 红色/黄色标记及背景
4. 契约合规状态（如适用）
5. 向管理层提出的问题

### 步骤4：趋势分析

如果提供多个期间：
- 绘制关键指标随时间变化的图表（收入、EBITDA、现金）
- 识别趋势——加速、减速或稳定
- 与承保案例进行比较

## 重要提示

- 如果未提供预算/计划，务必要求
- 不要假设行业特定关键绩效指标——询问对该公司重要的指标
- 如果契约水平未知，询问用户信贷协议条款
- 输出应为董事会就绪——简洁、事实、无废话

## 金融术语和知识解释

### 1. 投资组合监控 (Portfolio Monitoring)

**定义**：对投资组合公司进行持续跟踪和评估的过程。

**目的**：
- 确保投资按计划执行
- 及早识别问题
- 支持决策制定
- 向投资者报告

**关键活动**：
- 定期财务审查
- 关键绩效指标跟踪
- 风险监控
- 董事会汇报

### 2. 关键绩效指标 (Key Performance Indicator, KPI)

**定义**：衡量业务绩效的量化指标。

**类型**：
| 类别 | 示例 |
|------|------|
| 财务 | 收入、EBITDA、现金流 |
| 运营 | 客户数量、员工效率 |
| 客户 | 留存率、满意度 |
| 市场 | 市场份额、增长率 |

**选择标准**：
- 与战略目标对齐
- 可量化
- 可操作
- 定期跟踪

### 3. 杠杆比率 (Leverage Ratio)

**定义**：衡量公司债务水平的指标。

**常见比率**：

| 比率 | 计算公式 |
|------|----------|
| 净债务/EBITDA | 净债务 / LTM EBITDA |
| 债务/股权 | 总债务 / 股东权益 |
| 债务/资产 | 总债务 / 总资产 |

**解读**：
- 高杠杆：高风险，利息负担重
- 低杠杆：低风险，但可能错失增长机会

**行业基准**：
- 科技：3-4x
- 工业：4-5x
- 金融：更高

### 4. 利息覆盖比率 (Interest Coverage Ratio)

**定义**：衡量公司支付利息能力的指标。

**计算公式**：
```
利息覆盖比率 = EBIT / 利息费用
```

**解读**：
- **>3x**：安全
- **1.5-3x**：关注
- **<1.5x**：危险

**重要性**：
- 债权人评估信用风险的关键指标
- 低比率可能触发违约条款

### 5. 自由现金流 (Free Cash Flow, FCF)

**定义**：公司运营产生的现金减去资本支出后的剩余现金。

**计算公式**：
```
FCF = 经营活动现金流 - 资本支出
```

**用途**：
- 支付股息
- 回购股票
- 偿还债务
- 进行收购

**重要性**：
- 真正可支配的现金
- 衡量财务健康的关键指标

### 6. 契约合规 (Covenant Compliance)

**定义**：遵守贷款协议中约定的条款和条件。

**常见契约**：

| 类型 | 描述 |
|------|------|
| 财务契约 | 维持特定财务比率 |
| 运营契约 | 限制某些运营决策 |
| 报告契约 | 按时提交财务报告 |

**违反后果**：
- 技术违约
- 加速还款要求
- 更高的利率
- 贷款终止

### 7. LTM EBITDA (Last Twelve Months EBITDA)

**定义**：过去12个月的税息折旧摊销前利润。

**计算方法**：
```
LTM EBITDA = 最近四个季度EBITDA之和
```

**用途**：
- 计算估值倍数
- 评估杠杆水平
- 契约计算基准

**优势**：
- 平滑季节性影响
- 提供最新的业绩视图

### 8. 预算偏差分析 (Budget Variance Analysis)

**定义**：比较实际业绩与预算的差异。

**分析方法**：

| 方法 | 描述 |
|------|------|
| 差异百分比 | (实际-预算)/预算 × 100% |
| 因素分析 | 分解差异的原因 |
| 趋势分析 | 观察差异随时间变化 |

**差异分类**：
- **有利差异**：实际>预算（收入）或实际<预算（成本）
- **不利差异**：实际<预算（收入）或实际>预算（成本）

### 9. 承保案例 (Underwriting Case)

**定义**：投资决策时使用的财务预测和假设。

**组成部分**：
- 收入预测
- 成本预测
- 现金流预测
- 退出假设

**监控目的**：
- 跟踪实际vs预测
- 识别假设偏差
- 调整投资策略

### 10. 董事会就绪材料 (Board-Ready Materials)

**定义**：准备提交给董事会的专业报告和演示材料。

**特点**：
- 简洁明了
- 数据驱动
- 重点突出
- 专业格式

**典型内容**：
- 执行摘要
- 关键指标仪表板
- 问题和风险
- 建议行动
