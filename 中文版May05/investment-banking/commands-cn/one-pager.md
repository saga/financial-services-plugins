# 公司一页纸简讯概览指令 (One-Pager Strip Profile Command)

为投行推介材料 (Pitch Books) 和交易材料创建专业的一页纸公司简讯概览。

## 工作流程

### 第一步：收集公司信息

如果提供了公司名称或股票代码，请直接使用。否则请询问：
- "您想为哪家公司创建概览？"

### 第二步：检查可用的 PPT 模板技能

**首先，在技能目录中检查现有的 ppt-template 技能：**

```bash
ls skills/ | grep -E "ppt-template|brand-guidelines"
```

如果存在模板技能（例如：`techcorp-ppt-template`, `gs-brand-guidelines`）：
1. 向用户列出可用模板
2. 询问使用哪个模板，或者是否需要简洁的专业格式
3. 使用 `skill: "[template-name]"` 加载选定的模板技能

如果不存在模板技能，请询问：
- "您是否有品牌化的 PowerPoint 模板文件可以使用？如果有，请提供路径。否则我将使用简洁的专业格式。"

如果提供了模板文件：
1. 分析模板结构以了解页面布局
2. 为一页纸内容选择合适的布局

### 第三步：加载简讯概览技能

使用 `skill: "strip-profile"` 执行概览创建：

1. **明确需求**：
   - 确认单页幻灯片格式（一页纸）
   - 询问是否有特定的重点关注领域

2. **调研公司数据**：
   - 公司概况（总部、成立时间、员工人数、管理层）
   - 业务描述与定位
   - 核心财务数据（营收、EBITDA、利润率、增长率）
   - 估值指标（市值、企业价值 EV、倍数）
   - 近期动态与新闻
   - 主要股东（针对上市公司）

3. **创建简讯概览**：
   - 使用 4:3 宽高比（10" x 7.5"）
   - 四象限布局：
     - 左上：公司概况 (要点)
     - 右上：业务与地位 (要点)
     - 左下：核心财务数据 (表格)
     - 右下：股价走势图 + 股东构成 或 近期新闻
   - 应用公司品牌色
   - 在各板块标题处添加强调栏

### 第四步：视觉审核

创建幻灯片后：
1. 转换为图片进行审核
2. 检查文字重叠或截断问题
3. 验证所有数据均已填充（无占位符）
4. 向用户展示预览以供批准

### 第五步：交付产出

提供：
1. **PowerPoint 文件** (.pptx) - 一页纸概览
2. **图片预览** - 用于快速查看
3. **关键数据摘要** - 包含在文件中的核心要点

## 质量核查表

交付前确认：
- [ ] 所有四个象限均已填充真实数据
- [ ] 无剩余占位符文本
- [ ] 已应用公司品牌色
- [ ] 所有板块标题均有强调栏
- [ ] 财务表格格式规范
- [ ] 底部注明数据来源
- [ ] 无文字溢出或截断
- [ ] 达到投资银行级品质 (高盛/大摩/小摩标准)

> **💡 Appendix: 领域知识小贴士**
>
> 1. **一页纸简讯概览 (One-Pager/Strip Profile)**：投行里的“相亲简历”。它把一间庞杂的公司浓缩进一张纸里，让大佬们在几秒钟内就能看清这家公司是干什么的、赚不赚钱。
> 2. **演示文稿 (Pitch Book)**：投行用来“忽悠”（专业地说，是推介）客户的 PPT 合集。一份高质量的 Pitch Book 是投行专业度的敲门砖。
> 3. **EBITDA**：中文叫“息税折旧摊销前利润”。你可以把它理解为公司**纯靠经营业务**赚回来的真金白银，还没来得及交给政府（税）、还没付给银行（息），也没算上设备折旧（非现金支出）。它是衡量公司赚钱能力的核心指标。
> 4. **企业价值 (EV)**：这可不是市值！EV = 市值 + 欠银行的钱 - 手里的现金。它代表的是如果要完整买下这家公司，你实际上需要掏多少钱。
> 5. **EV/EBITDA 倍数**：一种常用的“估值尺子”。简单说，就是看看买下这家公司花费的代价，需要几年能靠它目前的经营利润赚回来。倍数越高，通常意味着大家觉得它未来越有前途，或者现在太贵了。
