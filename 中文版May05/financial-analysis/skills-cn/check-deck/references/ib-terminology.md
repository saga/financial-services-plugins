# 投行专业术语参考

## 常用替换：从口语到专业术语

| 口语/非正式 | 投行标准表达 |
|-------------|--------------|
| "增长很快" (a lot of growth) | "显著增长" (significant growth) 或具体的 "X% 增速" |
| "利润率挺好" (pretty good margins) | "具有吸引力的利润率" (attractive margins) |
| "他们买了这家公司" | "该公司被收购" (the company was acquired) |
| "大交易" (big deal) | "具有转型意义的交易" (transformative transaction) |
| "估值便宜" (cheap) | "具有吸引力的估值" (attractive valuation) |
| "贵" (expensive) | "溢价估值" (premium valuation) |
| "多赚钱" (make more money) | "提升盈利能力" (enhance profitability) |
| "变大" (getting bigger) | "追求规模扩张" (pursuing growth) |
| "削减成本" (cut costs) | "实施成本优化" (implement cost optimization) |
| "很合适" (good fit) | "具有协同效应的战略契合" (strategic fit) |

## 应避免的语言模式

- **缩写**：避免使用 Don't, Won't 等，应写完整。
- **感叹号**：投行材料中严禁使用感叹号，应保持中立专业。
- **第一人称**：避免“我们认为”，改用“管理层相信”或使用被动语态。
- **缺乏证据的最高级**：除非有数据支撑，否则不要轻易使用“行业最佳”(best-in-class)。
- **模糊量词**：避免使用“一些”、“很多”，应提供具体数字。

## 推荐句式

**增长叙事：**
- "在[期间]内实现了 X% 的营收年复合增长率 (CAGR)"
- "展现了持续的利润率扩张能力"

**市场地位：**
- "在[特定细分市场]中排名前 X"
- "作为[特定产品/服务]的领先供应商"

**战略逻辑：**
- "由……驱动的引人注目的战略契合"
- "通过……实现的估值创造机会"
