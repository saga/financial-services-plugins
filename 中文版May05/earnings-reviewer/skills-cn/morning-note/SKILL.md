# 晨间简报

description: 起草简明扼要的晨会纪要，汇总隔夜市场动态、交易策略及所覆盖个股的关键事件。专为早 7 点晨会格式设计——内容精炼、观点鲜明、具备可操作性。触发词包括：“morning note”、“晨会纪要”、“隔夜动态”、“策略建议”、“早会筹备”或“每日摘要”。

## 工作流程

### 第一步：隔夜动态

扫描覆盖的研究范围内所有相关事件：

**业绩表现与指引**
- 覆盖名单中是否有公司在隔夜或盘前发布财报？
- 业绩是否存在意外（营收、每股收益 (EPS) 或关键指标超预期/逊于预期）？
- 业绩指引是否变动（上调、下调或维持不变）？

**新闻与事件**
- 并购 (M&A) 公告或传闻。
- 管理层变动。
- 产品发布或监管决策。
- 竞争对手分析师的评级上调/下调。
- 影响该行业的宏观数据或政策变动。

**市场背景**
- 隔夜指数期货 / 盘前走势。
- 行业 ETF 表现。
- 相关大宗商品或汇率变动。
- 今日即将发布的关键经济数据。

### 第二步：晨间简报格式

保持精炼——一份晨间简报应能在 2 分钟内读完：

---

**[日期] 晨间简报 —— [分析师姓名]**
**[覆盖行业]**

**核心观点：[标题 —— 投资经理最需要听到的那件事]**
- 用 2-3 句话阐述核心进展及其重要性。
- 个股影响：目标价变动、维持或调整评级。

**隔夜/盘前动态**
- [公司 A]：业绩/新闻一句话总结 + 我们的评论。
- [公司 B]：一句话总结 + 我们的评论。
- [板块/宏观]：相关的全行业动态。

**今日关键事件**
- [时间]：[公司] 业绩电话会议。
- [时间]：经济数据发布（市场预期 vs. 我们的看法）。
- [时间]：行业会议或投资者日。

**交易策略**（如有）
- [做多/做空] [公司]：1-2 句核心逻辑 + 催化剂。
- 风险点：什么情况下该逻辑会失效。

---

### 第三步：财报快评

如果覆盖的公司发布了财报，请提供快速反馈：

| 指标 | 市场共识 | 实际值 | 偏离度 (超/逊预期) |
|--------|-----------|--------|-----------|
| 营业收入 | | | |
| 每股收益 (EPS) | | | |
| [关键指标] | | | |
| 业绩指引 | | | |

**我们的看法**：2-3 句话——这对比股价是利好还是利空？是否改变了我们的投资逻辑？

**建议操作**：维持 / 上调 / 下调评级？是否调整目标价？

### 第四步：输出方式

- 用于邮件/Slack 发布的 Markdown 文本。
- 如果需要正式分发，则提供 Word 文档。
- 篇幅严格控制在 1 页以内——投资经理和交易员没时间看更多内容。

## 注意事项

- 观点要鲜明——没有立场、只汇总新闻的简报毫无价值。
- 结论先行——不要把最重要的标题藏在正文里。
- “无事发生”也是有效的简报——可以说“隔夜无实质性变动，维持现有头寸”。
- 区分“可操作事件”（如财报、并购）和“市场噪音”（如微小的分析师评论、非关键事件）。
- 标注评论的时间戳——如果你是在凌晨 6 点写的，请注明盘前走势到开盘时可能会有变动。
- 如果判断错误，在下一份简报中大方承认——公信力比“永远正确”更重要。

> **💡 Appendix: 领域知识小贴士**
>
> 欢迎来到金融世界的“早起鸟”时间！为了让你丝滑理解这些术语，我们准备了这一份小白科普：
>
> 1. **覆盖范围 (Coverage)**：就像购物清单一样，分析师会盯着一筐特定的公司股票。这些被“盯着”并定期写报告的公司就叫“覆盖公司”。
> 2. **业绩指引 (Guidance)**：这是公司管理层给出的“预告片”。他们会预测未来几个月能赚多少钱。如果预告片比大家想象的精彩，股票通常会涨。
> 3. **EPS (每股收益)**：简单来说，就是公司赚的总钱数除以发行的股票数量。这是衡量公司赚钱成色的重要尺子。
> 4. **市场共识 (Consensus)**：在考试（发财报）前，全班同学（分析师们）会猜测你考多少分。这个平均分就是“共识”。如果你的实际分高于共识，那就是“超预期 (Beat)”。
> 5. **催化剂 (Catalyst)**：促使股价变动的“火药桶”。比如新产品发布或重大利好政策，它会让原本平淡的股价突然活跃起来。
> 6. **投资经理 (PM, Portfolio Manager)**：他们是掌管巨额资金的“总舵手”。分析师的工作就是给他们递“情报”，帮他们决定买什么、卖什么。
> 7. **做多与做空 (Long/Short)**：看好它会涨，先买入等涨了卖，这叫“做多”；如果你觉得它要跌，通过借钱买跌的方式赚钱，就叫“做空”。
