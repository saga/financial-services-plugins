# 投资建议书

description: 为潜在客户创建专业的投资建议书。涵盖公司投资方法、拟定资产配置、预期成果及费用结构。适用于争取新客户或展示新投资策略。触发词包括：“投资建议书”、“潜在客户演示”、“争取新客户”、“[客户名称]建议书”或“新客户演示”。

## 工作流程

### 第一步：潜在客户背景调研

收集：
- **潜在客户姓名**及家庭成员详情
- **现状**：已有投顾？自主交易？促成此次会谈的契机？
- **资产**：预估管理资产规模 (AUM)、账户类型、当前持仓（如有分享）
- **目标**：退休规划、财富保全、资产增长、现金流收入、教育基金、遗产继承
- **风险承受能力**：保守型、稳健型、进取型（或基于风险测评问卷分值）
- **限制条件**：ESG（环境、社会、治理）偏好、集中持股、流动性需求
- **费用敏感度**：目前支付的费用水平如何？
- **竞争对手**：他们同时还在考虑哪些机构？

### 第二步：建议书结构

**I. 关于本公司** (1页)
- 公司概况、成长历史及管理资产规模 (AUM)
- 投资理念（使用通俗易懂的语言）
- 团队简介（针对该客户背景匹配的相关成员）
- 客户服务模式（会面频率、核心联系人）

**II. 理解您的需求** (1页)
- 再次陈述其投资目标与担忧——展示您已充分聆听
- 在初步调研中确定的关键规划考量因素
- 对其而言，“财务成功”的定义是什么

**III. 拟定投资策略** (2-3页)
- 建议的资产配置及逻辑支撑
- 该配置如何匹配其目标和风险承受能力
- 投资工具选择（ETF、公募基金、个股、另类投资）
- 税务高效策略（资产课税区划优化、税收损失采集）

拟定资产配置：

| 资产类别 | 配置比例 | 投资工具 | 推荐理由 |
|------------|-----------|---------|-----------|
| | | | |

**IV. 预期成果** (1-2页)
- 预估增长情景（保守、适中、乐观）
- 达成目标的蒙特卡罗模拟概率
- 收益预测（针对退休规划或以现金流为中心的目标）
- 风险指标（最大回撤、波动率）
- 与当前投资组合的对比（如已知）

**V. 费用结构** (1页)
- 投资咨询费率表（注明阶梯费率，如适用）
- 底层基金管理费用
- 综合总成本估算
- 费用水平与行业平均水平的对比
- 价值主张——客户支付费用所获得的专业服务价值

**VI. 开始执行** (1页)
- 开户流程说明
- 资产转账时间表
- 转换计划（如果从其他投顾处迁移资产）
- 前 90 天的工作安排与预期
- 所需文件清单及后续步骤

### 第三步：定制化调整

- 语调适配：针对企业高管、小企业主或退休人员使用不同的专业语气
- 特殊情况：如果客户存在集中持股（如公司股票期权），需直接给出针对性方案
- 优势对比：如果客户正在考虑智能投顾，应强调专业财务规划和长期关系价值
- 价值导向：对于价格敏感型客户，应重点突出整体价值和最终成果，而非单纯强调费用

### 第四步：产出物

- 带有公司品牌标识的 PowerPoint 演示文稿（12-15页）
- PDF 打印留存版本
- 一页纸汇报大纲（用于后续跟进邮件）

## 重要说明

- 建议书应具备高度的个性化色彩，避免模板化内容——务必引用其具体情况
- 切勿夸大收益表现——设定现实的预期并强调严谨的投资流程
- 必须包含合规免责声明（预测具有假设性、过往业绩不代表未来表现等）
- 资产转换计划至关重要——客户往往担心更换投顾会带来管理上的混乱
- 在 48 小时内完成跟进，发送建议书并明确下一步安排
- 所有建议书在向潜在客户展示之前，必须经过合规部门审核

> **💡 Appendix: 领域知识小贴士**
>
> 欢迎来到财富管理的殿堂！这里有几个文中提到的核心概念，帮你快速变身投资达人：
>
> 1. **AUM (管理资产规模)**：这就像是一个管家的“资历证”。它代表有多少客户把钱交给我们打理。数字越大，说明这家机构在业内越受欢迎，积累的资源也越多。
> 2. **资产配置 (Asset Allocation)**：投资界的“超级配方”。简单说，就是不要把所有鸡蛋放在一个篮子里。我们会根据你的心脏承受能力（风险偏好），把钱分到股票、债券、黄金等地方，让波动小一点，收益稳一点。
> 3. **蒙特卡罗模拟 (Monte Carlo Simulation)**：投资界的“平行时空演算”。通过计算机运行成千上万次模拟，看看在各种极端的市场环境下（比如大牛市或金融危机），你的财富目标达成的概率是多少。
> 4. **税收损失采集 (Tax-loss Harvesting)**：投资界的“变废为宝”。当某些投资亏损时，我们策略性地卖出它们来抵扣你需要缴税的盈利部分。这样在法律允许范围内，你就能少交税，多留点钱在兜里。
> 5. **ESG**：投资界的“三好学生”指标。不仅看公司赚不赚钱，还要看它对环境好不好（E）、社会责任感强不强（S）以及管理是否规范（G）。这是一种更具情怀且能规避长线风险的投资方式。
