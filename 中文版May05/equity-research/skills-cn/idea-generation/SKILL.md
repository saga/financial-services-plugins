# 投资想法生成

description: 系统化的股票筛选与投资建议获取。结合定量筛选、主题研究和模式识别，挖掘新的做多与做空机会。适用于寻找新创意、运行筛选器或进行专题扫描。触发词包括：“投资想法生成”、“股票筛选”、“寻找创意”、“看好什么”、“筛选”、“新点子”或“给我推介个标的”。

## 工作流程

### 第一步：定义搜索标准

向用户询问参数：
- **方向**：做多创意、做空创意或两者兼有
- **市值**：大盘、中盘、小盘、微盘
- **行业**：特定行业或跨行业
- **风格**：价值、成长、质量、特种情况、事件驱动
- **地域**：美国、国际、全球
- **主题**：任何特定的主题视角（人工智能、制造业回流、人口老龄化等）

### 第二步：定量筛选

根据投资风格运行筛选器：

**价值筛选**
- 市盈率 (P/E) 低于行业中位数
- 企业价值倍数 (EV/EBITDA) 低于历史平均水平
- 自由现金流收益率 > 5%
- 市净率 (P/B) 低于 1.5 倍
- 过去 90 天内有内部人增持
- 股息率高于市场平均水平

**成长筛选**
- 营收同比 (YoY) 增速 > 15%
- 盈利同比增速 > 20%
- 营收提速（增速正在加快）
- 利润率扩张
- 高投入资本回报率 (ROIC > 15%)
- 强劲的净留存率 (SaaS 类公司 > 110%)

**质量筛选**
- 持续的营收增长（5 年以上）
- 稳定或扩张的利润率
- 净资产收益率 (ROE) > 15%
- 低资产负债率 (Debt/Equity)
- 高自由现金流转化率
- 内部人持股比例 > 5%

**做空筛选**
- 营收下降或增长放缓
- 利润率受压
- 应收账款/存货相对于销售额上升
- 内部人减持
- 估值溢价远高于同行且缺乏合理理由
- 空头头寸高企且基本面恶化
- 会计红旗信号（审计师变更、财务重报）

**特种情况筛选**
- 近期禁售期届满的 IPO / SPAC 标的
- 过去 12 个月内的分拆上市 (Spin-offs)
- 从债务重组中恢复的公司
- 激进投资者 (Activist) 介入
- 业绩不佳公司的管理层更替

### 第三步：主题扫描

针对主题型创意，研究主题并识别受益者：

1. 定义投资逻辑（例如：“AI 基础设施支出在 2026 年前将持续加速”）
2. 梳理价值链 —— 谁是直接受益者，谁是间接受益者？
3. 区分纯粹标的 (Pure-play) 与多元化敞口标的
4. 评估哪些名字已“被市场计价” (Priced in)，哪些还未被充分挖掘
5. 寻找市场尚未将其与该主题联系起来的二阶受益者

### 第四步：创意展示

对于通过筛选的每个创意，展示以下内容：

**[公司名称] — [做多/做空] — [一句话投资逻辑]**

| 指标 | 数值 | 与同行对比 |
|--------|-------|-----------|
| 市值 | | |
| EV/EBITDA (远期 NTM) | | |
| P/E (远期 NTM) | | |
| 营收增长 | | |
| EBITDA 利润率 | | |
| 自由现金流收益率 | | |

**投资逻辑（3-5 个要点）：**
- 为什么价格被错估
- 市场忽略了什么
- 价值实现的催化剂

**关键风险：**
- 哪些因素会导致判断错误

**建议下一步行动：**
- 建立完整模型？深度尽职调查？专家访谈？

### 第五步：输出

- 包含 5-10 个创意的候选名单及一页纸总结
- 记录筛选标准和方法论
- 所有创意的对比表
- 优先级列表：哪些创意应首先进行研究

## 重要注意事项

- 筛选器选出的是“候选人”而非“结论” —— 每一个筛选输出都需要基础研究支撑
- 最好的创意往往来自交集（例如：因短期不利因素导致价格处于价值区间的优质公司）
- 避开拥挤交易 —— 检查持仓数据、空头余额以及覆盖该标的的分析师人数
