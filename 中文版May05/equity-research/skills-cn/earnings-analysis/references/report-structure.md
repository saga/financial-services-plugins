# 报告结构与模板

本文档提供了业绩更新 DOCX 报告的逐页完整模板及格式要求。

## 完整报告结构

**报告结构：**

---

## 第 1 页：业绩摘要

**顶部区域 - 页眉：**
```
[公司名称] ([股票代码])
[季度] [年份] 业绩更新

[当前日期]

评级：[维持/上调/下调] [评级]
股价（截至 [日期]）：$XX.XX
目标价：[若有变动则显示 旧 → 新，或 维持 $XXX]
```

**顶部区域 - 快速摘要栏：**
```
业绩摘要
─────────────────────────────────────────────────
Q[X] [年份] 业绩：[超预期 / 符合预期 / 低于预期]

                实际值      预期      差异
营业收入         $X,XXX      $X,XXX   +$XXX (+X%)
每股收益 (经调整) $X.XX       $X.XX    +$X.XX (+X%)

核心要点：
■ [要点 1 - 一句话概括]
■ [要点 2 - 一句话概括]
■ [要点 3 - 一句话概括]
```

**正文内容 - 投资影响（3-4 个要点）：**

使用 ■ 符号配合 **加粗标题** 以及段落长度的解释：

```
■ **受强劲的 [业务板块/地区/产品] 驱动，业绩超预期并保持增长势头**

Q[X] 营收录得 $X.XB，超过我们预测的 $X.XB（高出 X%）以及市场共识预期（高出 X%），
主要受到 [特定驱动因素] 的推动。[分部] 营收同比增长 X%（对比我们预测的
X%），而 [分部] 增长了 X%（对比预测的 X%）。管理层强调 [特定产品/举措] 
是核心增长指标，并对前景保持乐观态度。此次超预期表现论证了 [投资逻辑要点]，
强化了我们的积极看法。

■ **尽管面临 [不利因素]，利润率仍同比扩张 XX 个基点，彰显经营杠杆效应**

[详细的利润率分析段落...]

■ **业绩指引 上调 / 维持 / 下调 - 意味着 [解读]**

[详细的指引分析段落...]

■ **维持 [评级]，目标价 [调整至/维持] $XXX**

[投资结论段落...]
```

                     FY2024E (原预测) FY2024E (现预测) 变动    FY2025E (现预测)

EBITDA ($M)          X,XXX           X,XXX           +X%     X,XXX
EBITDA 利润率 (%)     XX.X%           XX.X%           +XXbps  XX.X%
每股收益 (经调整) ($)  X.XX            X.XX            +X%     X.XX
市盈率 P/E (x)        XX.Xx           XX.Xx           -X%     XX.Xx

## 第 2-3 页：业绩详细指标分析

按以下维度拆解业绩：

### 营收分析（1 页）
- 总营收超预期/低于预期的解释
- 业务板块/地理区域/产品拆解
- 同比（YoY）与环比（sequential）趋势
- 与业绩指引（若有）的对比

**表格：季度营收趋势**
```
                        Q[X-3]  Q[X-2]  Q[X-1]  Q[X]    同比变动  环比变动
总营收 ($M)              X,XXX   X,XXX   X,XXX   X,XXX   +X%      +X%
  [业务 A] ($M)          XXX     XXX     XXX     XXX     +X%      +X%
  [业务 B] ($M)          XXX     XXX     XXX     XXX     +X%      +X%
  [业务 C] ($M)          XXX     XXX     XXX     XXX     +X%      +X%

注：Q[X] = [季度] [年份]
资料来源：公司财报，[机构名称] 分析
```

### 盈利能力分析（1 页）
- 毛利率分析（驱动因素、趋势）
- 营业利润率分析
- 线下项目（利息、税项等）
- 每股收益调节（经调整对比 GAAP 准则）

- [利空因素 1]
- [利空因素 2]

- 趋势与前景展望
