## 现金流折现 (DCF) 分析

### 概述

DCF 分析基于公司未来预期现金流的现值来衡量其价值。从理论上讲，这是最严谨的估值方法，因为它植根于基本的价值创造逻辑。

### DCF 逐步操作流程

#### 1. 历史财务分析
- 收集过去 3-5 年的历史财务数据。
- 计算历史自由现金流 (FCF) = EBIT(1-税率) + 折旧与摊销 (D&A) - 资本支出 (CapEx) - 营运资金净额 (NWC) 变动。
- 分析历史增长率和利润率。
- 识别业务趋势和周期性。

#### 2. 构建营收预测 (5-10 年)
**预测方法：**
- **自上而下 (Top-down)**：从市场总量 (TAM) 开始 → 市场份额 → 营业收入。
- **自下而上 (Bottom-up)**：销量 × 单价。
- **混合法**：结合多个驱动因素。

**关键考量因素：**
- 管理层指引与历史增长记录。
- 行业增长率及市场趋势。
- 竞争格局与市场份额演变。
- 产品管线与新市场机遇。
- 宏观经济因素。

#### 3. 预测营业费用
- **营业成本 (COGS)**：占营收的百分比（分析历史毛利率）。
- **销售、一般及行政费用 (SG&A)**：通常具有半固定性质；结合规模效应按营收百分比建模。
- **研发费用 (R&D)**：对科技/医药行业至关重要；按营收百分比建模。
- **折旧与摊销 (D&A)**：基于资本支出假设进行预测。

**计算息税前利润 (EBIT)** = 营收 - 营业成本 - 营业费用

#### 4. 计算无杠杆自由现金流 (UFCF)
```
EBIT (息税前利润)
× (1 - 税率)
= NOPAT (税后净营业利润)
+ 折旧与摊销 (D&A)
- 资本支出 (CapEx)
- 营运资金净额 (NWC) 的增加
= 无杠杆自由现金流 (UFCF)
```

**资本支出假设：**
- 维持性资本支出：维持现有运营所需（通常占营收的 2-4%）。
- 增长性资本支出：业务扩张所需。
- 参考行业基准及公司指引。

**营运资金净额：**
- NWC = (应收账款 + 存货) - 应付账款。
- 按营收百分比或周转天数 (DSO, DIO, DPO) 建模。
- NWC 的增加代表现金的流出。

#### 5. 确定终值 (Terminal Value)

**方法 A：永续增长法 (Perpetuity Growth Method)**
```
终值 = 最终预测年度 FCF × (1 + g) / (WACC - g)
```
- g = 永续增长率（通常为 2-3%，不应超过 GDP 增长率）。
- 适用于已进入稳定、成熟增长阶段的公司。

**方法 B：退出倍数法 (Exit Multiple Method)**
```
终值 = 最终预测年度 EBITDA × 退出倍数
```
- 基于当前可比公司的交易倍数。
- 对周期性业务更为适用。

#### 6. 计算加权平均资本成本 (WACC)

```
WACC = (E/V × 权益成本) + (D/V × 债务成本 × (1 - 税率))
```

**权益成本（使用 CAPM 模型）：**
```
权益成本 = 无风险利率 + Beta × 权益风险溢价
```
- 无风险利率：通常参考 10 年期国债收益率。
- Beta：股票回报率相对于市场的回归系数（或参考可比公司 Beta）。
- 权益风险溢价：历史平均水平约为 5-6%。

**债务成本：**
```
债务成本 = 无风险利率 + 信用利差
```
- 使用公司当前的借贷利率或从债券价格中隐含得出。
- 若无存续债券，则根据信用评级进行调整。

**资本结构：**
- E/V = 权益市值 / 总价值。
- D/V = 债务市值 / 总价值。
- 若当前结构差异巨大，应使用目标资本结构。

#### 7. 将现金流折现为现值

```
PV (现值) = Σ [FCFt / (1 + WACC)^t] + [终值 / (1 + WACC)^n]
```

#### 8. 计算企业价值与权益价值

```
企业价值 (Enterprise Value) = 预测期 FCF 现值 + 终值现值
减：净债务 (总债务 - 现金)
加：非营利性资产
减：少数股东权益
减：优先股
= 权益价值 (Equity Value)

每股价格 = 权益价值 / 稀释后发行在外的股份数
```

### DCF 敏感性分析

务必对关键变量进行敏感性分析：

1. **双向敏感性表**：WACC vs. 永续增长率。
2. **营收增长情景**：基准 / 乐观 / 悲观情景。
3. **利润率假设**：经营杠杆情景。
4. **终值倍数敏感性**：如果使用的是退出倍数法。

**敏感性表示例：**
```
           永续增长率
WACC      2.0%    2.5%    3.0%
8.0%      $45     $48     $52
9.0%      $40     $43     $46
10.0%     $36     $39     $41
```

### 应避免的 DCF 常见误区

1. **增长的双重计算**：不要在未配置相应投资（资本支出、营运资金）的情况下预测高增长。
2. **不切实际的永续增长**：不应超过长期 GDP 增长率。
3. **忽略周期性**：对于周期性企业，应使用平准化收益。
4. **现金流定义错误**：使用无杠杆自由现金流 (UFCF)，而非净利润。
5. **假设不一致**：折现率必须与现金流匹配（UFCF 对应 WACC）。

## 可比公司分析 (Trading Comparables Analysis)

### 概述

可比公司分析（简称“Comps”）通过参考公开市场上同类公司的估值水平来为目标公司定价。这反映了当前的市场情绪和相对估值。

### 逐步操作流程

#### 1. 选择可比公司

**筛选标准：**
- 同一行业/板块（首要标准）。
- 相似的业务模式和收入来源。
- 规模相当（市值、营收）。
- 相似的增长特征和利润率水平。
- 相似的终端市场和地理区域。

**典型样本库：**
- 初选 8-15 家公司。
- 剔除存在特殊情况的公司。
- 最终保留 5-10 家核心可比公司。

#### 2. 收集财务信息

**所需数据：**
- 当前股价及发行在外股份数。
- 最近一个财政年度的财务报表。
- 市场一致预期的下一年度 (NTM) 预测。
- 历史增长率。

**计算市场指标：**
- 市值 = 股价 × 发行在外股份数。
- 企业价值 (EV) = 市值 + 债务 + 少数股东权益 + 优先股 - 现金。
- 净债务 = 总债务 - 现金及现金等价物。

#### 3. 计算估值倍数

**企业价值倍数：**
- **EV/Revenue (市销率)**：适用于早期阶段或高增长公司。
- **EV/EBITDA**：最常用；适用于资本密集型行业。
- **EV/EBIT**：当各公司间折旧与摊销差异较大时非常有用。

**权益价值倍数：**
- **P/E (市盈率)**：应用最广泛的指标。
- **P/B (市净率)**：适用于金融机构。
- **P/S (每股市销率)**：适用于尚未盈利的公司。

**计算周期：**
- 过去十二个月 (LTM) - 历史数据。
- 未来十二个月 (NTM) - 前瞻性数据（通常首选）。

#### 4. 分析并筛选倍数

**创建可比公司对照表：**

| 公司 | 市值 | EV/Revenue | EV/EBITDA | EV/EBIT | P/E (NTM) | 营收增长 | EBITDA 利润率 |
|---------|-----------|------------|-----------|---------|-----------|----------------|---------------|
| 公司 A  | $100亿    | 3.5x       | 12.0x     | 18.0x   | 22.0x     | 15%            | 28%           |
| 公司 B  | $80亿     | 3.0x       | 10.5x     | 16.0x   | 19.0x     | 12%            | 27%           |
| ...     | ...       | ...        | ...       | ...     | ...       | ...            | ...           |
| 中位数  | -         | **3.2x**   | **11.0x** | **17.0x** | **20.5x** | 13%            | 27.5%         |

**调整：**
- 剔除异常值（通常指偏离 2 个标准差以上的数据）。
- 优先考虑中位数而非平均值（受异常值影响较小）。
- 如果某些公司更具对比性，可进行加权处理。
- 针对增长、利润率及风险差异进行调整。

#### 5. 将倍数应用于目标公司

**计算示例：**
```
目标公司 NTM EBITDA = $5亿
选定的 EV/EBITDA 倍数 = 11.0x
隐含企业价值 = $5亿 × 11.0x = $55亿

减：净债务 = $10亿
权益价值 = $45亿

发行在外股份数 = 1亿股
隐含每股价格 = $45.00
```

#### 6. 选择合适的倍数

**选择依据：**
- **EV/Revenue**：高增长、未盈利公司（科技、盈利前的生物医药）。
- **EV/EBITDA**：最通用；资本密集型行业（制造业、电信）。
- **P/E**：盈利稳定且资本结构稳定的公司（消费、零售）。
- **行业特定指标**：银行用 P/B，油气用 EV/产量，媒体用 EV/订阅用户。

### 溢价/折价分析

基于以下因素给予溢价或折价：
- **增长溢价**：增长越高 → 倍数越高。
- **盈利能力**：利润率越高 → 倍数越高。
- **规模**：大公司通常享有流动性溢价。
- **市场地位**：市场领导者享有溢价。
- **地理因素**：发达市场 vs. 新兴市场。

## 先例交易分析 (Precedent Transactions Analysis)

### 概述

先例交易分析通过参考类似公司在并购 (M&A) 交易中的成交价格进行估值。这通常反映了控制权溢价和战略价值。

### 逐步操作流程

#### 1. 识别相关交易

**筛选标准：**
- 同一或相似行业。
- 规模相似（通常在目标公司规模的 0.5 倍至 2 倍之间）。
- 相似的业务特征。
- 近期发生的交易（通常为近 3-5 年）。
- 已宣布并完成的交易（避免撤回的交易）。

**典型样本库：**
- 至少 5-10 笔交易。
- 重点关注近期交易（赋予更高权重）。

#### 2. 收集交易细节

**所需信息：**
- 交易日期（宣布日及完成日）。
- 收购价格及结构（现金、股票、混合）。
- 交易时目标公司的财务状况。
- 战略逻辑及协同效应。
- 支付的控制权溢价。

**数据来源：**
- 证监会文件（S-4, 8-K, 代理声明）。
- 新闻稿及投资者演示材料。
- 并购数据库（CapIQ, FactSet, Bloomberg）。

#### 3. 计算交易倍数

**指标与可比公司分析一致，但基于交易价值计算：**

```
交易价值 = 权益收购价格 + 承继债务 - 收购现金

EV/Revenue (LTM) = 交易价值 / 目标公司 LTM 营收
EV/EBITDA (LTM) = 交易价值 / 目标公司 LTM EBITDA
EV/EBIT (LTM) = 交易价值 / 目标公司 LTM EBIT
```

**计算控制权溢价：**
```
控制权溢价 = (要约价格 - 未受影响价格) / 未受影响价格
```
- 未受影响价格指交易宣布前 1-2 天的目标公司股价。
- 典型范围：20-40%。

#### 4. 分析先例交易

**创建先例交易对照表：**

| 日期 | 目标公司 | 收购方 | 交易金额 | EV/Revenue | EV/EBITDA | 溢价 | 交易逻辑 |
|------|--------|----------|------------|------------|-----------|---------|-----------|
| 2024Q1 | CompX | BuyerA | $50亿 | 4.0x | 14.0x | 35% | 市场整合 |
| 2023Q3 | CompY | BuyerB | $35亿 | 3.5x | 12.5x | 28% | 战略契合 |
| 中位数 | - | - | - | **3.8x** | **13.0x** | **31%** | - |

#### 5. 应用于目标公司

**重要注意事项：**
- 先例交易倍数通常高于可比公司分析（包含控制权溢价）。
- 针对交易背景和逻辑差异进行调整。
- 考虑当时的市场环境与现状的差异。
- 对近期的交易给予更高权重。

**计算示例：**
```
目标公司 LTM EBITDA = $4.5亿
选定的 EV/EBITDA 倍数 = 13.0x (先例交易)
对比：可比公司分析倍数 = 11.0x

隐含企业价值 (先例) = $4.5亿 × 13.0x = $58.5亿
隐含企业价值 (可比) = $4.5亿 × 11.0x = $49.5亿

隐含控制权溢价 = $58.5亿 / $49.5亿 - 1 = 18%
```

### 交易倍数的调整因素

**考量因素：**
- **市场环境**：牛市 vs. 熊市（并购活跃程度）。
- **交易结构**：战略买家 vs. 财务买家。
- **协同效应**：具有高协同效应的交易往往伴随更高溢价。
- **竞争态势**：单方竞标 vs. 多方竞标。
- **时间价值**：较早的交易参考价值较低。

## 估值汇总与对账

### 构建估值桥梁

在一个框架下呈现三种方法的结果。

**估值汇总示例：**

| 方法 | 企业价值 | 权益价值 | 每股价格 | 权重 | 隐含价值贡献 |
|--------|------------------|--------------|-------------|--------|---------------|
| DCF 分析 | $52.00亿 | $42.00亿 | $42.00 | 50% | $21.00 |
| 可比公司分析 | $55.00亿 | $45.00亿 | $45.00 | 30% | $13.50 |
| 先例交易分析 | $58.50亿 | $48.50亿 | $48.50 | 20% | $9.70 |
| **加权平均值** | - | - | **$44.20** | - | **$44.20** |

### 确定各方法权重

**典型权重分配：**
- **DCF**：40-60%（反映内在价值，但高度依赖预测准确性）。
- **可比公司**：25-40%（反映当前市场情绪）。
- **先例交易**：15-25%（除非并购意图明显，否则优先级较低）。

**调整依据：**
- **对财务预测的信心**：信心越高 → DCF 权重越大。
- **市场成熟度**：牛/熊市会影响可比公司倍数的可靠性。
- **并购可能性**：若行业正在整合，先例交易权重增加。
- **公司成熟度**：成熟公司看重可比公司；增长型公司看重 DCF。

### 估值区间

始终提供一个估值区间，而非单一的点估计值：

**方法：**
- **基准情景 (Base Case)**：最可能发生的场景。
- **乐观情景 (Bull Case)**：进取的增长和利润率假设。
- **悲观情景 (Bear Case)**：保守的驱动因素假设。

**示例：**
```
悲观区间：$38 - $40
基准区间：$42 - $46
乐观区间：$48 - $52

投资建议：买入，目标价 $45（基准情景的中值）
```

### 合理性检查 (Sanity Checks)

**跨维度验证估值：**
1. **历史倍数**：当前估值是否与公司历史水平一致？
2. **同业对比**：相对于同行的溢价/折价是否合理？
3. **隐含增长率**：市场目前的定价中隐含了怎样的增长预期？
4. **隐含回报率**：从当前股价到目标价的内含报酬率 (IRR)。
5. **市值分析**：计算出的总市值是否符合常识？

## 结论

综合使用这三种方法可以为确定公允价值提供稳健的框架：

- **DCF** 基于基本面提供内在价值参考。
- **可比公司分析** 反映了当前市场的估值基准。
- **先例交易分析** 揭示了并购价值及控制权成本。

关键在于深入理解各方法背后的驱动假设，并呈现一个兼顾多种情景与方法论、逻辑严密的估值区间。

> **💡 Appendix: 领域知识小贴士**
>
> 嘿！如果你是第一次接触这些“金融黑话”，别被它们吓到。其实给公司估值和我们在生活中买东西是一个道理：
>
> 1. **DCF（现金流折现）—— “会下蛋的鹅”**：
>    想象你买了一只会下蛋的鹅。这只鹅值多少钱？你会算算它未来每年能下多少蛋（现金流），但是未来的蛋不如今天的蛋值钱（因为有通货膨胀和风险），所以你要折算一下。把这辈子它能下的所有蛋都换算成今天的价格加起来，就是这只鹅的“内在价值”。
>
> 2. **Trading Comps（可比分析）—— “看邻居房价”**：
>    这就像你要卖房子。你怎么定价？你会看看隔壁老王家和你户型差不多的房子最近在市场上挂多少钱。如果邻居们的房子平均值 5 万一平，那你的房子大概也值这个价。这就是观察市场大环境给出的“参考价”。
>
> 3. **Precedent Transactions（先例交易）—— “整栋楼打包卖”**：
>    还是说房子。如果某个大老板最近把整栋单元楼都买下来了，他出的单价肯定比零散买卖要高（因为他想要绝对的控制权）。这种“大手笔交易”的价格更能反映出公司在并购时的战略意义和“打包价”。
>
> 4. **WACC（加权平均资本成本）—— “借钱的成本”**：
>    公司做生意需要钱，这些钱可能来自老板自己（权益），也可能来自银行（债务）。你需要付利息或回报，这就是成本。WACC 就像是一个“门槛”，只有当生意赚得比这个成本多时，公司才算真的在创造价值！
