2. **无杠杆自由现金流**

   - 营运资本变动   ($XX)   ($XX)   ($XX)   ($XX)   ($XX)

3. **资产负债表数据（当前）**

### 第2步：构建DCF分析

#### A. 计算WACC

**1. 确定无风险利率**
   - 使用10年期国债收益率（检查当前利率）
   - 示例：截至2024年底为4.0-4.5%

**2. 确定股权成本（CAPM）**
   ```
   股权成本 = 无风险利率 + Beta × 股权风险溢价

   输入：
   - 无风险利率：[当前10年期国债，例如4.2%]
   - Beta：[来自Bloomberg/FactSet的公司Beta或可比公司平均值]
   - 股权风险溢价：5-6%（历史平均值）

   示例：
   股权成本 = 4.2% + 1.3 × 5.5% = 11.35%
   ```

**3. 确定债务成本**
   ```
   债务成本 = 当前借款利率或债券隐含收益率

   对于私营公司：
   债务成本 = 无风险利率 + 信用利差（基于评级）

   示例：
   税前债务成本 = 6.5%
   税后债务成本 = 6.5% × (1 - 25%税率) = 4.875%
   ```

**4. 确定资本结构**
   ```
   使用市场价值（而非账面价值）：

   股权市场价值(E) = 股价 × 流通股数
   债务市场价值(D) = 总债务（如债券未交易则使用账面价值）
   总价值(V) = E + D

   股权权重 = E / V
   债务权重 = D / V

   示例：
   E = $5,000M (90.9%)
   D = $500M (9.1%)
   V = $5,500M (100%)
   ```

**5. 计算WACC**
   ```
   WACC = (E/V × 股权成本) + (D/V × 债务成本 × (1 - 税率))

   示例：
   WACC = (90.9% × 11.35%) + (9.1% × 6.5% × (1 - 25%))
   WACC = 10.32% + 0.44% = 10.76%

   四舍五入至：基准情况10.8%
   ```

#### B. 计算终值

**方法1：永续增长法（首选）**
```
终值 = FCF(2029) × (1 + g) / (WACC - g)

其中：
- FCF(2029) = 模型中最后一年的无杠杆自由现金流
- g = 永续增长率（通常为2.0-3.0%）
  - 不应超过长期GDP增长
  - 基准情况使用2.5%

示例：
FCF(2029) = $500M
g = 2.5%
WACC = 10.8%

终值 = $500M × (1.025) / (0.108 - 0.025)
终值 = $512.5M / 0.083 = $6,175M
```

**方法2：退出倍数法（替代）**
```
终值 = EBITDA(2029) × 退出倍数

其中：
- 退出倍数 = 当前可比公司交易中位数（例如12-15x EBITDA）

示例：
EBITDA(2029) = $800M
退出倍数 = 13x

终值 = $800M × 13x = $10,400M
```

**选择一种方法或取两者平均值。**
