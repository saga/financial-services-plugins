# 股权研究初始报告模板

本模板提供了创建综合性股权研究初始报告的结构。在构建最终报告文档时以此为指导。

**注意：** 实际报告必须使用DOCX技能创建。不要生成markdown内容。

**关键要求：**
1. 在创建Word文档之前，使用Python（matplotlib/plotly）生成20-30+张图表图片
2. **使用DOCX技能**：使用正确的样式、页眉/页脚和格式创建专业报告
3. **嵌入实际图表图片**：将生成的PNG/JPG图表文件插入到Word文档的适当位置
4. **不要生成markdown**：使用DOCX技能创建.docx文件。

**关键格式指导：**
- **最大密度**：每页都应充满信息。在全文中穿插文字、图表和表格。
- **不要有孤立部分**：不要让章节标题单独一页或让单个图表独占一页。
- **20-30+张实际图表图片**：将图表生成为图片文件，然后使用DOCX技能将其嵌入文本部分。

## 第1页：投资更新（最重要的页面）

**关键**：第1页不是传统的执行摘要。它是一页**投资更新**，使用专业股权研究公司的特定机构格式。

**重要结构说明：**
- 这是"投资更新"或"公司更新"页面，不是"执行摘要"
- 左上角使用评级框
- 突出显示股票价格表现图表（图1）
- 包含3-4个详细要点，使用■字符
- 每个要点有**粗体主题标题**，后跟3-5句解释
- 底部是财务和估值指标表
- 所有图表都有图表编号（图1、图2等）和来源行

## 图表编号和格式标准

**关键**：所有图表、图形和表格必须遵循专业股权研究中使用的严格图表编号规范。

### 图表编号格式

**每个图表/表格必须有：**
1. **顺序编号**：图1、图2、图3等（贯穿全文顺序编号）
2. **描述性标题**："图X - [公司] [具体指标] [图表/分析类型]"
3. **来源行**（始终在底部）："来源：公司数据，[公司名称]估计。"

**示例：**
- 图1 - [公司] 股票价格表现
- 图2 - [公司] 历史和预测收入组合（按产品）
- 图3 - [公司] 按地理区域划分的收入
- 图4 - [产品名称] 每位患者每年的收入和价格
- 图5 - [公司] 毛利率演变
- 图6 - DCF敏感性分析（美元/股）
- 图7 - 估值足球场

### 标题格式

```
图X - [描述性标题]
[图表/表格/图形内容]
来源：公司数据，[公司名称]估计。
```

对于有多个数据源的表格：
```
图X - [描述性标题]
[表格内容]
来源：公司文件，FactSet，[公司名称]估计。
```

### 放置指南

- 图表应按在报告中出现的顺序编号
- 第一张图表（图1）通常是第1页上的股价图表或收入增长轨迹
- 每个图表的标题必须直接放在视觉元素下方
- 来源行应使用较小字体、斜体，放在图表底部
