# 投资构思生成

description: 系统化的股票筛选与投资构思溯源。结合定量筛选、主题研究和模式识别，挖掘新的做多与做空构思。适用于寻找新思路、运行筛选器或进行主题性梳理。当用户提及“投资构思生成”、“股票筛选”、“寻找思路”、“有什么有趣的标的”、“筛选”、“新点子”或“给我推票”时触发。

## 工作流

### 第 1 步：定义搜索标准

询问用户相关参数：
- **方向**：做多构思、做空构思或两者兼有
- **市值**：大盘、中盘、小盘、微盘
- **行业**：特定行业或跨行业
- **风格**：价值、成长、质量、特殊情况、事件驱动
- **地域**：美国、国际、全球
- **主题**：特定的主题视角（AI、制造业回流、人口老龄化等）

### 第 2 步：定量筛选

根据投资风格运行筛选器：

**价值筛选 (Value Screen)**
- 市盈率 (P/E) 低于行业中位数
- 企业价值/息税折旧摊销前利润 (EV/EBITDA) 低于历史均值
- 自由现金流收益率 (FCF Yield) >5%
- 市净率 (P/B) 低于 1.5 倍
- 过去 90 天内有内部人士增持
- 股息率高于市场平均水平

**成长筛选 (Growth Screen)**
- 营收同比增长 (YoY) >15%
- 盈利同比增长 (YoY) >20%
- 营收加速增长（增长率持续提升）
- 利润率扩张
- 高投入资本回报率 (ROIC >15%)
- 强净收入留存率（SaaS 行业 >110%）

**质量筛选 (Quality Screen)**
- 持续的营收增长（5 年以上）
- 利润率稳定或持续扩张
- 净资产收益率 (ROE) >15%
- 低权益负债比 (Debt/Equity)
- 高自由现金流转化率
- 内部人士持股比例 >5%

**做空筛选 (Short Screen)**
- 营收下降或增长放缓
- 利润率受压
- 应收账款/存货相对于销售额比例上升
- 内部人士减持
- 估值显著高于同行且缺乏合理性
- 基本面恶化且空头仓位 (Short Interest) 高企
- 会计警示信号（审计师变更、财报重述）

**特殊情况筛选 (Special Situation Screen)**
- 近期 IPO / SPAC 锁定期届满
- 过去 12 个月内的分拆上市 (Spin-offs)
- 破产重组后重新上市的公司
- 积极投资者 (Activist) 介入
- 业绩不佳公司的管理层变动

### 第 3 步：主题性梳理

针对主题性构思，研究相关主题并识别受益者：

1. 定义投资逻辑（例如：“AI 基础设施支出将在 2026 年前持续加速”）
2. 绘制价值链——谁是直接受益者，谁是间接受益者？
3. 识别纯业务标的 (Pure-play) 与多元化敞口
4. 评估哪些标的已“定价充分” (Priced in)，哪些尚未被市场发掘
5. 寻找市场尚未与该主题建立关联的二阶受益者

### 第 4 步：构思陈述

针对通过筛选的每个构思，按以下格式呈现：

**[公司名称] — [做多/做空] — [一句话核心逻辑]**

| 指标 | 数值 | 行业对比 |
|--------|-------|-----------|
| 市值 | | |
| EV/EBITDA (远期) | | |
| P/E (远期) | | |
| 营收增长率 | | |
| EBITDA 利润率 | | |
| 自由现金流收益率 | | |

**核心逻辑 (3-5 个要点)：**
- 为什么当前定价错误
- 市场忽略了什么
- 实现价值修复的催化剂

**关键风险：**
- 哪些因素会导致该逻辑失效

**建议后续步骤：**
- 构建完整财务模型？开展深度尽职调查？专家访谈？

### 第 5 步：输出

- 包含 5-10 个构思的入围名单及单页总结
- 记录筛选标准和方法论
- 所有构思的对比表
- 优先级清单：建议优先研究的标的

## 重要说明

- 筛选器只能筛选出候选标的，而非结论——每个筛选结果都需要进一步的基本面研究
- 最好的构思往往产生于交集处（例如：因短期利空导致价格处于价值区间的优质公司）
- 避开拥挤交易——检查持仓数据、空头仓位以及分析师覆盖人数
- 逆向投资构思需要催化剂——如果选对了标的但买入太早且缺乏催化剂，效果等同于看错
- 持续跟踪构思的胜率——哪些筛选器和方法能够产出最高质量的构思？
- 做空构思需要更强的信心——时机更难把握，且风险是不对称的

> **💡 Appendix: 领域知识小贴士**
>
> 欢迎来到投资的“实验室”！这里有几个核心概念，能帮你快速看懂这些金融行话：
>
> *   **做多 (Long) vs 做空 (Short)**：简单来说，做多就是“看好”，期待股价上涨；做空则是“看跌”，通过股价下跌获利。
> *   **估值倍数 (Multiples, 如 P/E, EV/EBITDA)**：这就像买菜时看的单价。市盈率 (P/E) 衡量你为每 1 块钱盈利支付的价格。倍数越低，通常意味着越便宜。
> *   **自由现金流收益率 (FCF Yield)**：这是衡量一家公司“含金量”的指标。它告诉我们在扣除所有开支和投资后，公司手里剩下的“活钱”相对于股价的比例。
> *   **投入资本回报率 (ROIC)**：衡量公司用钱生钱的能力。如果你给公司 100 块钱，它能产生 15 块钱利润，那它的 ROIC 就是 15%。这是衡量公司“质量”的金标准。
> *   **催化剂 (Catalyst)**：这是点燃股价的火药桶。一个公司就算再便宜，如果没有新产品发布、管理层更替或行业政策等“催化剂”出现，股价可能还是会原地踏步。
