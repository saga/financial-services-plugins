# 交易来源

PE交易来源工作流。

## 工作流

此技能遵循3步来源流程：

### 步骤1：发现公司

根据用户标准研究和识别潜在目标公司：

- **行业/重点领域**：询问用户想要关注的领域（例如，"医疗保健领域的B2B SaaS"、"东南部的工业服务"）
- **交易参数**：收入范围、EBITDA范围、增长概况、地区、所有权类型（创始人所有、PE支持、企业剥离）
- **来源**：使用网络搜索找到符合标准的公司。查看行业报告、会议参与者名单、行业出版物和竞争格局
- **输出**：公司短名单，包含：名称、描述、预估收入/规模、地点、创始人/CEO姓名、网站，以及为什么符合投资论点

### 步骤2：CRM检查

在 Outreach 之前，检查公司或创始人是否已存在于公司CRM中：

- 搜索用户的电子邮件（Gmail）中是否有与该公司或创始人的先前往来
- 搜索Slack中是否有内部提及或关于目标的先前讨论
- 询问用户："您或您的团队与[公司]有过先前接触吗？"
- 标记任何现有关系、先前放弃或已知背景
- **输出**：对于每个公司，注明："新建"（无先前接触）、"现有"（发现先前往来——总结）或"先前放弃"（如有先前放弃的证据）

### 步骤3：起草创始人 Outreach

起草给创始人/CEO的个性化冷邮件：

- **语气**：专业但温暖。不要过于正式——创始人更容易回复真诚、简洁的 Outreach
- **结构**：
  1. 简短介绍——你是谁，你的公司（如果不知道询问用户）
  2. 为什么这家公司引起你的注意——引用具体内容（产品、市场地位、增长）
  3. 你在寻找什么——合作伙伴，而不仅仅是交易
  4. 软性请求："你愿意简短聊一聊吗？"
- **个性化**：引用公司的具体产品、近期新闻或市场地位。永远不要使用通用模板
- **长度**：最多4-6句话。创始人很忙
- **语气匹配**：如果用户发送过先前 Outreach 邮件，研究他们以匹配语气和风格。搜索Gmail中包含"reaching out"、"introduction"、"partnership"等关键词的"已发送"邮件以找到示例

### 邮件起草指南

- 主题行：简短且具体。引用公司或行业，而不是"投资机会"
- 第一次联系不附加任何东西
- 包含清晰但低压力的行动号召
- 如果有Gmail则起草在Gmail中，否则输出文本供用户复制

## 示例互动

**用户**："帮我找德州做1000万-5000万收入的创始人所有的工业服务公司"

**助手**：
1. 在网上搜索符合标准的德州工业服务公司
2. 呈现5-8家公司的短名单及关键详情
3. 对于每个，检查Gmail/Slack是否有先前接触
4. 为标记为"新建"的公司起草个性化 Outreach 邮件
5. 在发送前呈现草稿供用户审查

## 重要说明

- 起草邮件前总是先呈现短名单供用户审查
- 永远不要在没有用户明确批准的情况下发送邮件
- 如果用户的公司介绍或投资标准不清楚，起草前先询问
- 质量优先于数量——5个精心研究的目标胜过20个通用的
