# 单位经济效益分析

分析PE目标的单位经济效益。

## 工作流

### 步骤1：识别商业模式

确定收入模型以定制分析：
- **SaaS/订阅**：ARR、净留存、队列
- **经常性服务**：合同价值、续费率、扩展销售
- **交易/使用量**：每笔交易收入、成交量趋势、费率
- **混合**：按收入流细分

### 步骤2：核心指标

#### ARR/收入质量
- **ARR桥**：期初ARR → 新增 → 扩展 → 收缩 → 流失 → 期末ARR
- **按队列ARR**：年份分析——每个年度队列如何留存和增长？
- **收入集中度**：前10/20/50客户占总收入的百分比
- **按类型收入**：经常性vs非经常性vs专业服务
- **合同结构**：ACV分布、多年百分比、自动续约百分比

#### 客户经济效益
- **CAC（客户获取成本）**：总销售营销支出/获取的新客户数
- **LTV（终身价值）**：（ARPU × 毛利率）/ 流失率
- **LTV:CAC比率**：健康企业目标>3x
- **CAC回本周期**：收回获取成本的月数
- **混合vs细分**：按客户细分（企业vs SMBvs中端市场）

#### 留存与扩展
- **毛留存**：期初ARR留存百分比（不含扩展）
- **净留存（NDR）**：期初ARR留存百分比含扩展
- **Logo流失**：流失客户百分比
- **美元流失**：流失收入百分比（通常与Logo流失不同）
- **扩展率**：扩展销售+交叉销售占期初ARR百分比

#### 队列分析
构建队列矩阵：

| 队列 | 第0年 | 第1年 | 第2年 | 第3年 | 第4年 |
|------|-------|-------|-------|-------|-------|

| 2021 | $1.5M | $1.7M | $1.8M | | |
| 2022 | $2.0M | $2.3M | | | |
| 2023 | $3.0M | | | | |

显示绝对美元和指数化（第0年=100%）两种视图。

#### 利润率瀑布
- 收入 → 毛利润 → 贡献边际 → EBITDA
- 全加载单位经济效益：获取、服务和留存一个客户的成本是多少？
- 按收入流的毛利率（订阅vs服务vs其他）

### 步骤3：基准对比

将单位经济效益与相关基准对比：
- **SaaS 40法则**：增长率+EBITDA利润率>40%
- **SaaS魔数**：净新增ARR/上期销售营销支出>0.75x
- **净留存基准**：最佳>120%，良好>110%，担忧<100%
- **LTV:CAC**：最佳>5x，良好>3x，担忧<2x
- **毛留存**：最佳>95%，良好>90%，担忧<85%
- **CAC回本**：最佳<12个月，良好<18个月，担忧>24个月

## 重要说明

- 如果有原始客户级数据总是询问——汇总指标可能隐藏问题
- 净留存超过100%可能掩盖高毛流失如果扩展足够强——始终显示两者
- 队列分析是收入质量最重要的单一视图——争取这个数据
- 区分合同ARR和实际确认收入
- 对于使用量模型，关注消费趋势和扩展模式而不是传统ARR指标
- 专业服务收入应单独评估——它不是经常性的且利润率通常较低
