# 交易筛选

快速筛选入站交易流。

## 工作流

### 步骤1：提取交易事实

从提供的CIM、推介或描述中提取：

- **公司**：名称、地点、行业/子行业
- **描述**：他们做什么（1-2句话）
- **财务**：收入、EBITDA、利润率、增长率
- **交易类型**：平台、增补型、重组、少数股权、剥离
- **要价/估值**：倍数、企业价值（如注明）
- **卖家动机**：为什么现在出售
- **管理层**：留任或退出
- **关键客户**：集中度风险
- **关键风险**：明显的红旗

### 步骤2：按标准筛选

应用基金的投资标准（如不了解则询问用户）：

| 标准 | 目标 | 实际 | 通过/不通过 |
|------|------|------|-----------|
| 收入范围 | | | |
| EBITDA范围 | | | |
| EBITDA利润率 | | | |
| 增长概况 | | | |
| 行业契合度 | | | |
| 地区 | | | |
| 交易规模/企业价值 | | | |
| 估值（EBITDA倍数） | | | |
| 客户集中度 | | | |
| 管理层连续性 | | | |

### 步骤3：快速评估

提供三部分评估：

1. **结论**：通过 / 进一步尽调 / 硬通过
2. **牛市案例**（2-3点）：为什么这可能是一笔好交易
3. **熊市案例**（2-3点）：关键风险和担忧
4. **关键问题**：你需要在第一个电话上回答什么

### 步骤4：输出

适合与合伙人分享或IC快速筛选的一页筛选备忘录。

## 重要说明

- 速度很重要——筛选应该用分钟而不是小时
- 直接指出红旗。不要隐藏担忧
- 如果财务看起来不一致或不完整，明确标记
- 如果这是第一次筛选，先询问基金的标准
- 一旦确认，在记忆中保存筛选标准以供未来交易使用
