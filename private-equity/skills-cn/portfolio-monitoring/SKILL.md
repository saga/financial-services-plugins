# 投资组合监控

根据计划跟踪和分析投资组合公司表现。

## 工作流

### 步骤1：提取财务包

- 接受用户的投资组合公司财务包（Excel工作簿、PDF或CSV）
- 提取关键财务：收入、EBITDA、现金余额、债务余额、资本支出、营运资本
- 识别报告期间并与前期和预算/计划比较

### 步骤2：KPI提取与差异分析

跟踪的关键指标（根据公司行业调整）：

**财务KPI：**
- 收入vs预算（美元和百分比）
- EBITDA和EBITDA利润率vs预算
- 现金余额和净债务
- 杠杆率（净债务/最近12个月EBITDA）
- 利息保障倍数
- 资本支出vs预算
- 自由现金流

**运营KPI**（询问用户或从数据推断）：
- 客户数/每客户收入
- 员工人数/每员工收入
- 积压/管道
- 流失/留存率

### 步骤3：标记与摘要

- **绿色**：在计划5%以内
- **黄色**：低于计划5-15%——标记供讨论
- **红色**：低于计划超过15%或契约违约风险——需要立即关注

输出简洁摘要：
1. 一段执行摘要（"公司X表现[超计划/低于计划/符合计划]..."）
2. KPI表（实际vs预算vs前期）
3. 带背景的红色/黄色标记
4. 契约合规状态（如适用）
5. 给管理层的问题

### 步骤4：趋势分析

如果提供多个期间：
- 绘制关键指标随时间的变化（收入、EBITDA、现金）
- 识别趋势——加速、减速或稳定
- 与承销案例比较

## 重要说明

- 如果未提供预算/计划进行比较，总是先询问
- 不要假设行业特定的KPI——询问什么对这个公司重要
- 如果不知道契约水平，询问用户信贷协议条款
- 输出应就绪供董事会使用——简洁、事实、无空话
