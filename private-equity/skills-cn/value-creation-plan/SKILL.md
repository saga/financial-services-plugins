# 价值创造计划

构建收购后的价值创造计划。

## 工作流

### 步骤1：基线评估

了解起点：
- 当前收入、EBITDA和利润率
- 组织结构和能力
- 各职能的关键运营指标
- 管理层优势和改进空间
- 尽调期间已识别的快速见效项目

### 步骤2：价值创造杠杆

在持有期内将所有杠杆映射到EBITDA桥：

#### 收入增长杠杆
- **有机增长**：提价、销量增长、市场扩张
- **交叉销售/扩展销售**：向现有客户销售新产品
- **新市场进入**：地理扩张、新垂直领域、新渠道
- **销售团队效率**：招聘销售、提升转化、缩短周期
- **并购/增补**：增加收入和能力的补充收购

每个杠杆：
- 当前状态 → 目标状态
- 收入影响（$）
- 产生影响的时间
- 所需投资
- 信心水平（高/中/低）

#### 利润率扩张杠杆
- **定价优化**：提价、组合转变、捆绑销售
- **COGS降低**：采购节约、供应商整合、自动化
- **OpEx优化**： overhead削减、共享服务、外包
- **技术投资**：自动化、系统集成、数据分析
- **规模杠杆**：随着收入增长固定成本杠杆

#### 战略/倍数扩张
- **平台建设**：增补收购、Tuck-in
- **经常性收入转变**：从项目转向经常性/订阅
- **市场定位**：品类领导力、品牌建设
- **管理层升级**：关键招聘以使业务专业化
- **ESG/治理**：董事会组建、报告改进

### 步骤4：100天计划

优先处理交割后前100天：

**第1-30天：稳定与评估**
- 管理层对齐和留存（签署雇佣协议、设定薪酬）
- 快速见效——定价、明显成本削减、易摘的果实
- 各职能详细运营评估
- 客户沟通计划
- 建立报告和KPI仪表板

**第31-60天：计划与启动**
- 最终确定战略计划并向组织传达
- 启动前3-5个价值创造举措
- 开始增补并购管道开发
- 填补关键空缺的招聘
- 实施新报告节奏（周报、月度回顾、季度董事会）

**第61-100天：执行与衡量**
- 快速见效举措的首批结果
- 首次董事会会议及运营指标
- 各价值创造杠杆进展报告
- 根据早期学习调整计划

### 步骤6：输出

- Word文档或PowerPoint，包含：
  - 执行摘要（1页）
  - EBITDA桥图表
  - 价值创造杠杆详情（每杠杆1页）
  - 100天计划时间表
  - KPI仪表板
  - 问责矩阵（谁负责什么）
- 支持EBITDA桥的Excel模型

## 重要说明

- 对时间表要现实——大多数PE价值创造需要12-24个月才能在财务中显示
- 快速见效对势气和信誉很重要，但不要为了增长而过度强调成本削减
- 管理层认同至关重要——共同制定计划，而不是强加
- 跟踪举措级P&L影响，而不仅仅是表面EBITDA——你需要知道什么在起作用
- 增补并购通常是最大的价值创造杠杆——第1天开始管道
- 始终用运营伙伴或行业专家压力测试假设
