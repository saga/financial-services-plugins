# 回报分析

为PE交易评估构建快速的IRR/MOIC敏感性表。

## 工作流

### 步骤1：收集交易输入

询问（或从先前分析提取）：

**进入：**
- 进入EBITDA（最近12个月或未来12个月）
- 进入倍数（企业价值/EBITDA）
- 企业价值
- 交割时净债务
- 股权投资额
- 交易费用和支出

**融资：**
- 优先债务（EBITDA倍数、利率、摊销）
- 次级债务/夹层（如有）
- 进入时总杠杆（EBITDA倍数）
- 股权投资

**运营假设：**
- 收入增长率（年度）
- EBITDA利润率轨迹
- 资本支出占收入百分比
- 营运资本变化
- 债务偿还计划

**退出：**
- 持有期（年）
- 退出倍数（企业价值/EBITDA）
- 退出EBITDA（从增长假设计算）

### 步骤5：输出

## 关键公式

- **MOIC** = 退出股权价值 / 股权投资
- **IRR**：求解r使得股权投资 × (1 + r)^n = 退出股权价值（根据期间现金流调整）
- **回报归因**：
  - 增长：（退出EBITDA - 进入EBITDA）× 退出倍数 / 股权
  - 倍数：（退出倍数 - 进入倍数）× 进入EBITDA / 股权
  - 杠杆：持有期债务偿还 / 股权

## 重要说明

- 始终显示含费用/Carry和不含费用的回报
- 管理层滚动投资和共同投资改变股权投资——如果相关则询问
- 股息再融资或期间分配显著影响IRR——如果计划包括则纳入
- 不要忘记交易成本（通常为企业价值的2-4%）——它们降低第一天股权价值
- 税务考虑（资产vs股票交易、338(h)(10)选择）可能显著影响税后回报
