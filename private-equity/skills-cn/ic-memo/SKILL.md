# 投资委员会备忘录

为PE交易审批起草结构化的投资委员会备忘录。

## 工作流

### 步骤1：收集输入

从用户收集（或从会话中先前分析）：

- 公司概览和业务描述
- 行业/市场背景
- 历史财务（3-5年）
- 管理层评估
- 交易条款（价格、结构、融资）
- 尽调发现（商业、财务、法律、运营）
- 价值创造计划/100天计划
- 回报分析（基准、上涨、下行）

### 步骤2：起草备忘录结构

标准IC备忘录格式：

**一、执行摘要**（1页）
- 公司描述、交易理由、关键条款
- 推荐和标题回报
- 前3大风险和缓释方案

**二、公司概览**（1-2页）
- 业务描述、产品/服务
- 客户群和商业化
- 竞争定位
- 管理团队

**三、行业与市场**（1页）
- 市场规模和增长
- 竞争格局
- 长期趋势/顺风
- 监管环境

**四、财务分析**（2-3页）
- 历史表现（收入、EBITDA、利润率、现金流）
- 盈利质量调整
- 营运资本分析
- 资本支出需求

**五、投资论点**（1页）
- 为什么这是有吸引力的投资（3-5个支柱）
- 价值创造杠杆（有机增长、利润率扩张、并购、倍数扩张）
- 100天优先事项

**六、交易条款与结构**（1页）
- 企业价值和隐含倍数
- 资金来源与用途
- 资本结构/杠杆
- 关键法律条款

**七、回报分析**（1页）
- 基准、上涨和下行情景
- 各情景的IRR和MOIC
- 推动回报的关键假设
- 敏感性分析

**八、风险因素**（1页）
- 按严重程度和可能性排序的关键风险
- 每个风险的缓释方案
- 交易破坏者风险（如有）

**九、推荐**
- 明确推荐：推进 / 放弃 / 有条件推进
- 关键条件或后续步骤

### 步骤3：输出格式

- 默认：具有专业格式的Word文档（.docx）
- 替代：Markdown用于快速审查
- 包含财务和回报表格，而不仅仅是文字

## 重要说明

- IC备忘录应事实且平衡——诚实呈现牛市和熊市案例
- 不要最小化风险。IC成员无论如何都会发现；信誉很重要
- 如果用户提供，使用公司的标准备忘录模板
- 财务表格应衔接——检查EBITDA桥、资金来源与用途平衡，回报计算一致
- 对于缺失的输入询问用户，而不是对交易条款或回报做出假设
