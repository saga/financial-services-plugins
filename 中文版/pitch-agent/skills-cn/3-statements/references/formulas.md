# 公式参考手册

**重要提示：** 除非用户另有说明，请务必使用本参考文档中列出的公式。

---

## 核心勾稽关系

```
资产负债表：           资产 = 负债 + 股东权益
净利润：              利润表净利润 → 现金流量表经营活动现金流（起点）
现金流量：            现金净变动 = 经营活动现金流 (CFO) + 投资活动现金流 (CFI) + 筹资活动现金流 (CFF)
现金余额核对：         期末现金（现金流量表）= 现金（资产负债表资产项）
月度/年度现金一致性：   期末现金（月度）= 期末现金（年度）
留存收益：            期初留存收益 + 净利润 - 股利 = 期末留存收益
股权融资：            普通股/资本公积变动（资产负债表）= 股票发行（筹资活动现金流）
第 0 年权益：         已筹集权益（第 0 年）= 期初权益（第 1 年）
```

## 毛利计算

**重要提示：** 毛利必须基于“净收入”而非“总收入”计算。

```
净收入 - 收入成本 = 毛利
```

| 术语 | 定义 |
|------|------------|
| 总收入 (Gross Revenue) | 扣除任何项目前的总营业收入 |
| 净收入 (Net Revenue) | 总收入 - 退货 - 折让 - 折扣 |
| 收入成本 (Cost of Revenue) | 与销售货物或提供服务直接相关的直接成本 |
| 毛利 (Gross Profit) | 净收入 - 收入成本 |

**注意：** 在进行盈利能力计算时，始终以净收入（在多数财务报表中也称为“净销售额”或简称“收入”）作为起点。总收入会夸大真实的顶线业绩。

## 利润率公式

```
毛利率 %          = 毛利 / 净收入
EBITDA            = 息税前利润 (EBIT) + 折旧与摊销 (D&A)（或 = 毛利 - 运营支出）
EBITDA 利润率 %    = EBITDA / 净收入
EBIT 利润率 %      = EBIT / 净收入
净利润率 %         = 净利润 / 净收入
```

## 信用指标公式

```
总负债                = 短期借款及一年内到期的长期负债 + 长期负债
净负债                = 总负债 - 现金
总负债 / EBITDA       = 总负债 / EBITDA（取自利润表）
净负债 / EBITDA       = 净负债 / EBITDA（取自利润表）
利息保障倍数          = EBITDA / 利息支出（取自利润表）
净利息支出占负债比 %  = 净利息支出 / 长期负债
资产负债率            = 总负债 / (总负债 + 总权益)
产权比率              = 总负债 / 总权益
流动比率              = 总流动资产 / 总流动负债
速动比率              = (总流动资产 - 存货) / 总流动负债
```

## 预测公式（净收入百分比法）

```
收入成本（预测）= 净收入 × 收入成本 % 假设
销售及市场推广费用（预测）= 净收入 × S&M % 假设
一般及行政费用（预测）= 净收入 × G&A % 假设
研发费用（预测）= 净收入 × R&D % 假设
股份支付费用（预测）= 净收入 × SBC % 假设
```

## 营运资金公式

```
应收账款 (AR)
  上期 AR
  + 营业收入（取自利润表）
  - 现金回收（轧差项）
  = 期末 AR
  应收账款周转天数 (DSO) = (AR / 营业收入) × 365

存货 (Inventory)
  上期存货
  + 采购额（轧差项）
  - 营业成本 (COGS)（取自利润表）
  = 期末存货
  存货周转天数 (DIO) = (存货 / COGS) × 365

应付账款 (AP)
  上期 AP
  + 采购额（取自存货计算）
  - 现金支付（轧差项）
  = 期末 AP
  应付账款周转天数 (DPO) = (AP / COGS) × 365

营运资金净额 (NWC) = 应收账款 + 存货 - 应付账款
营运资金变动 (ΔWC) = 本期 NWC - 上期 NWC
```

## 折旧与摊销 (D&A) 计划公式

```
期初固定资产原值 (PP&E Gross)
+ 资本开支 (CapEx)
= 期末固定资产原值 (PP&E Gross)

期初累计折旧
+ 折旧费用
= 期末累计折旧

固定资产净值 (PP&E Net) = 固定资产原值 - 累计折旧
```

## 债务计划公式

```
期初债务余额
+ 新增借款
- 偿还本金
= 期末债务余额

利息支出 = 平均债务余额 × 利率
  （使用期初余额以避免循环引用；若开启循环引用，则进行迭代计算）
```

## 留存收益公式

```
期初留存收益
+ 净利润（取自利润表）
+ 股份支付支出 (SBC)（取自利润表）
- 股利
= 期末留存收益
```

## 净营运亏损 (NOL) 结转计划公式

```
NOL 弥补计划表

期初 NOL 余额（第 1 年 / 成立时 = 0）
+ 新增 NOL（若 EBT < 0，则取 ABS(EBT)，否则为 0）
- 已使用 NOL（受应纳税所得额及抵减比例上限限制）
= 期末 NOL 余额

初始余额规则

对于新业务或首个建模周期：
  期初 NOL 余额 = 0
  NOL 仅能通过实现的亏损（EBT < 0）增加
  NOL 不得凭空产生或假设

NOL 使用额计算

税前利润 (EBT)
  若 EBT > 0：
    可用 NOL = 期初 NOL 余额
    抵减上限 = EBT × 80%（2017 年后联邦标准）
    实际使用 NOL = MIN(可用 NOL, 抵减上限)
    应纳税所得额 = EBT - 实际使用 NOL
  若 EBT ≤ 0：
    实际使用 NOL = 0
    应纳税所得额 = 0
    新增 NOL = ABS(EBT)

带 NOL 的税额计算

应交所得税 = MAX(0, 应纳税所得额 × 税率)
  （税额不能为负数；损失将转化为 NOL 资产）

NOL 对应的递延所得税资产 (DTA)

DTA - NOL 结转 = 期末 NOL 余额 × 税率
ΔDTA = 本期 DTA - 上期 DTA
  （DTA 增加 = 现金流量表中的非现金收益）
  （DTA 减少 = 现金流量表中的非现金支出）
```

## 资产负债表结构

```
资产
  现金（取自现金流量表期末现金）
  应收账款（取自营运资金）
  存货（取自营运资金）
  流动资产合计
  
  固定资产净值（取自 D&A 计划）
  递延所得税资产 - NOL（取自 NOL 计划）
  非流动资产合计
  资产总计

负债
  应付账款（取自营运资金）
  一年内到期的长期负债（取自债务计划）
  流动负债合计
  
  长期负债（取自债务计划）
  负债合计

股东权益
  普通股
  留存收益（取自留存收益计划）
  股东权益合计

核对项目：资产 - 负债 - 股东权益 = 0
```

## 现金流量表结构

```
经营活动现金流 (CFO)
  净利润（关联项：利润表）
  + 折旧与摊销 (D&A)（关联项：D&A 计划）
  + 股份支付支出 (SBC)（关联项：利润表或假设）
  - Δ递延所得税资产 (DTA)（关联项：NOL 计划；DTA 增加代表现金支出）
  - Δ应收账款 (AR)（关联项：营运资金）
  - Δ存货（关联项：营运资金）
  + Δ应付账款 (AP)（关联项：营运资金）
  = CFO

投资活动现金流 (CFI)
  - 资本开支 (CapEx)（关联项：D&A 计划）
  = CFI

筹资活动现金流 (CFF)
  + 债务发行（关联项：债务计划）
  - 债务偿还（关联项：债务计划）
  + 股权融资（关联项：资产负债表普通股/资本公积）
  - 股利（关联项：留存收益计划）
  = CFF

现金净变动 = CFO + CFI + CFF
期初现金
+ 现金净变动
= 期末现金（关联至：资产负债表现金项）
```

## 校验公式

```
资产负债表平衡校验：    = 资产 - 负债 - 权益（必须 = 0）
现金一致性核对：        = 资产负债表现金 - 现金流量表期末现金（必须 = 0）
留存收益滚算：          = 上期留存收益 + 净利润 + SBC - 股息 - 期末留存收益（必须 = 0）
递延所得税资产核对：    = NOL 计划表 DTA - 资产负债表 DTA（必须 = 0）
股权融资核对：          = 资产负债表股本变动 - 现金流量表股权发行（必须 = 0）
第 0 年权益核对：       = 已筹集权益（第 0 年）- 期初权益（第 1 年）（必须 = 0）
月度 vs 年度现金核对：  = 期末现金（月度）- 期末现金（年度）（必须 = 0）
NOL 抵减上限规则：      = NOL 使用额 ≤ EBT × 80%（2017 年后必须为真）
NOL 非负校验：          = 期末 NOL 余额 ≥ 0（必须为真）
NOL 初始余额校验：      = 第 1 年期初 NOL = 0（对于新业务必须为真）
NOL 增长逻辑：          = NOL 仅在 EBT < 0 时增加（亏损产生 NOL）
```

> **💡 Appendix: 领域知识小贴士**
>
> 欢迎来到金融建模的世界！这里的公式看起来虽然多，但其实它们像一块块乐高积木，共同拼出了一家公司的“健康报告”。
>
> 1.  **三表勾稽关系（Financial Linkages）：** 这是会计的灵魂。记住，利润表告诉我们公司赚了多少钱，资产负债表告诉我们公司有多少家底，而现金流量表则核实这些钱是不是真的流进了口袋。它们之间是环环相扣的，比如利润表的净利润会存入资产负债表的“留存收益”里。
> 2.  **毛利 vs 净收入：** 在金融圈，我们非常在意“净”。总收入是账面数字，但减去退货、折扣等水分后的“净收入”才是公司真实的能力起点。毛利则是公司产品本身赚不赚钱的第一道门槛。
> 3.  **营运资金（Working Capital）：** 想象你在开一家店，还没卖掉的货（存货）和客欠的钱（应收账款）都是你暂时被占用的钱。营运资金管理得好，意味着公司不需要借太多外债也能转得开。
> 4.  **NOL（净营运亏损）：** 这是一个很有趣的概念，它像是一张“税务抵用券”。如果公司今年亏了钱，政府允许你把亏损攒起来，等到以后赚钱的时候抵扣税款，这样公司在翻身初期就能多留点钱加速发展。
> 5.  **校验公式（Checks）：** 专业的建模师从不完全相信自己的手速。每一列计算最后都必须有个“Check”，如果资产减去负债和权益不等于零，那就说明你的“金融乐高”拼歪了，必须回去找原因！
