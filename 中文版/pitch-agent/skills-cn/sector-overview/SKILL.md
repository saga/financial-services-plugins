# 行业概览 (Sector Overview)

description: 创建涵盖市场动态、竞争格局、主要核心参与者及主题趋势的综合性行业与板块图谱报告。适用于响应客户需求、行业覆盖启动、主题研究报告或内部知识储备。触发词包括：“行业概览”、“行业报告”、“市场格局”、“行业分析”、“行业深度调研”或“主题研究”。

## 工作流程

### 步骤 1：界定研究范围

- **行业 / 细分板块**：涉及哪个行业，界定的范畴有多细？
- **研究目的**：客户报告、内部研究、推介材料、投资策略生成
- **深度深度**：高层级概览（5-10 页）还是深度调研（20-30 页）
- **切入角度**：中立的格局描述 vs. 主题性逻辑（例如“AI 基础设施建设”）
- **样本范围**：仅限上市公司，还是包含私有化企业？

### 步骤 2：市场概览

**市场规模与增长**
- 潜在市场总量 (TAM) 及其数据来源
- 历史增长率（5 年复合年增长率 CAGR）
- 预测增长率及核心假设
- 市场细分（按产品、地域、终端市场、客户类型）

**行业结构**
- 分散型 vs. 集中型 —— 前五大市场占有率
- 价值链图谱 —— 利润在何处累积？
- 商业模式类型（订阅制、交易型、授权许可、咨询服务）
- 准入门槛（资本密集度、监管要求、技术壁垒、网络效应）

**核心趋势与驱动因素**
- 长期增长利好（3-5 个主要趋势）
- 阻碍因素与潜在风险
- 技术颠覆矢量
- 监管动向发展
- 并购 (M&A) 活动与行业整合趋势

### 步骤 3：竞争格局

**公司概况**（针对前 5-10 名领先者）：

| 公司 | 营收 | 增长率 | EBITDA 利润率 | 市场份额 | 核心差异化优势 |
|---------|---------|--------|--------------|-------------|-------------------|
| | | | | | |

针对每家公司提供简要分析：
- 业务描述（2-3 句话）
- 战略定位与护城河
- 近期动态（财报表示、并购交易、产品发布）
- 估值快照（P/E、EV/EBITDA、EV/Revenue）

**竞争动态**
- 企业如何竞争？（价格、产品、服务、分销渠道）
- 谁在获得/失去市场份额，原因何在？
- 来自新进入者或相邻领域玩家的颠覆风险

### 步骤 4：估值背景

- 行业交易倍数（当前及历史波动范围）
- 溢价/折价驱动因素（增长预期、利润率、市场地位）
- 近期并购交易倍数 (M&A transaction multiples)
- 行业与大盘的对比情况

### 步骤 5：投资策略与启示

- 最佳的风险收益机会点在哪里？
- 可以通过该行业表达哪些主题性博弈？
- 行业核心争端（多空观点对垒）
- 可能改变行业叙事的催化剂

### 步骤 6：成果产出

- Word 文档或 PowerPoint 演示文稿，包含：
  - 市场概览与规模测算
  - 竞争格局图
  - 公司对比表
  - 估值汇总
  - 核心图表：市场增长、份额趋势、估值历史曲线
- Excel 附录，包含详细的公司底层数据

## 重要提示

- 引用所有市场规模数据 —— 注明研究机构或测算方法
- 严格区分 TAM（潜在天花板）的噱头与真实的触达市场
- 行业报告时效性极强 —— 务必标注日期并对可能过时的数据进行预警
- 图表至关重要 —— 市场规模瀑布图、竞争定位矩阵、估值散点图
- 若为客户编写，需根据其具体情况定制“核心启示”（如并购标的识别、竞争定位、市场准入决策）

> **💡 Appendix: 领域知识小贴士**
>
> 1. **TAM (Total Addressable Market) - 潜在市场总量**：你可以把它想象成“蛋糕最大的可能”。如果你卖冰淇淋，TAM 就是全世界想吃冰淇淋的人加起来能贡献多少钱。
> 2. **CAGR (Compound Annual Growth Rate) - 复合年增长率**：金融界的“平均分”。它排除了一年暴涨一年暴跌的噪音，告诉你这一段时间内平均每年增长了多少。
> 3. **EBITDA - 息税折旧摊销前利润**：这是衡量公司“纯赚钱能力”的一个重要指标。它把税收、利息、折旧这类会计上的干扰项踢开，看公司业务本身到底攒不攒钱。
> 4. **Moat - 护城河**：巴菲特最爱的词。指一家公司让对手无法模仿或者无法抢走客户的特殊能力，比如不可逾越的专利或者强大的品牌口碑。
> 5. **Multiples - 估值倍数**：金融界的“性价比标签”。比如 P/E（市盈率）是 10 倍，简单理解就是你需要 10 年才能靠现在的利润回本。倍数越高，说明大家对未来的期望越高。
