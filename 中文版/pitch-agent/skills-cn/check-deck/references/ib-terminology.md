# 投行业务术语参考

## 从非正式用语到专业术语的替换

| 非正式/口语化表达 | 投行标准专业表达 |
|-----------------|-------------|
| "a lot of growth" (大量增长) | "significant growth" (显著增长) 或 "X% growth" (X% 的增长) |
| "pretty good margins" (不错的利润率) | "attractive margins" (具备吸引力的利润率) 或 "margins of X%" (X% 的利润率) |
| "they bought the company" (他们买了这家公司) | "the company was acquired" (该公司被收购) |
| "big deal" (大交易) | "transformative transaction" (具有转型意义的交易) |
| "cheap valuation" (估值便宜) | "attractive valuation" (具备吸引力的估值) 或 "valuation discount" (估值折价) |
| "expensive" (贵) | "premium valuation" (溢价估值) |
| "make more money" (赚更多钱) | "enhance profitability" (提升盈利能力) 或 "drive margin expansion" (推动利润率扩张) |
| "getting bigger" (变得更大) | "pursuing growth" (追求增长) 或 "expanding operations" (拓展业务规模) |
| "cut costs" (削减成本) | "implement cost optimization" (实施成本优化) 或 "drive operational efficiencies" (提升运营效率) |
| "good fit" (很合适) | "strategic fit" (战略契合) 或 "compelling strategic rationale" (具有说服力的战略逻辑) |
| "help with" (帮帮忙) | "support" (提供支持) 或 "facilitate" (促成/协助) |
| "a bunch of" (一堆/一些) | "multiple" (多项) 或 "numerous" (诸类) |
| "kind of" / "sort of" (有点/大概) | [建议移除或具体化] |
| "really" / "very" (真地/非常) | [建议移除或通过数据量化] |
| "tons of" (大量的) | "substantial" (实质性的/大量的) 或通过数据量化 |
| "huge" (巨大的) | "significant" (重大的/显著的) 或通过数据量化 |
| "pretty much" (几乎/差不多) | [建议移除或保持精确] |
| "basically" (基本上) | [建议移除或进一步阐明] |

## 应避免的语言模式

- **缩写使用**: 将 Don't 改为 Do not，won't 改为 will not（在中文正式文档中，避免使用过于口语化的省略表达）。
- **感叹号**: 在投行材料中通常被视为不专业，应避免使用。
- **第一人称**: 将 "We think..."（我们认为...）改为 "Management believes..."（管理层认为...）或使用中性的客观表述。
- **缺乏证据的最高级表达**: 使用 "best-in-class"（同类最佳）等词汇时必须附带支持性数据。
- **模糊的量化词**: 将 "some" (一些), "many" (许多), "several" (若干) 替换为具体的数字。

## 首选表达范式

**增长叙述 (Growth narratives)**:
- "营收实现了 X% 的复合年增长率 (CAGR)"
- "在 [特定期间] 内实现了持续的利润率扩张"
- "证明了产生有机增长的能力"

**市场地位 (Market position)**:
- "在 [特定细分市场] 排名第 X"
- "作为 [特定产品/服务] 的领先供应商"
- "通过 [特定属性] 实现差异化定位"

**战略逻辑 (Strategic rationale)**:
- "由...驱动的极具吸引力的战略契合度"
- "通过...创造价值的极具吸引力的机会"
- "$X 百万的协同效应潜力，来源于 [特定来源]"

> **💡 Appendix: 领域知识小贴士**
>
> 想要看懂这些“高大上”的词汇吗？其实它们背后的逻辑很简单：
>
> 1.  **IB (Investment Banking, 投行)**：你可以把它想象成企业的“超级理财顾问”和“高级媒婆”。当一家大公司想上市、借大钱，或者想买下另一家公司时，就会找投行来帮忙出主意、算细账。
> 2.  **CAGR (复合年增长率)**：这是衡量公司成长速度的“标尺”。它不只看一年的增长，而是假设每年的增长都在“利滚利”，算出一段长时间内的平均增速。
> 3.  **Margins (利润率)**：就是公司每卖出 100 块钱的东西，扣掉本钱后手里能剩下的钱。剩下的比例越高，说明这生意越赚钱。
> 4.  **Valuation (估值)**：就是给公司定身价。就像买菜要看一斤多少钱，买公司也要通过各种计算，看看这家公司到底值多少钱。
> 5.  **Synergies (协同效应)**：也就是所谓的“1+1 > 2”。比如两家公司合并后，不再需要两个仓库，合用一个就能省钱，这多省出来的钱（或者多赚到的钱）就叫协同效应。
> 6.  **Organic Growth (有机增长)**：指公司靠着自己的产品卖得好、服务更吸引人来实现的自然增长，而不是靠砸钱买别的公司来凑规模。
