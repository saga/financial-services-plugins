# 业绩前瞻

description: 构建包含预测模型、情景框架和核心关注指标的财报前分析。在公司发布季度业绩前使用，用于准备头寸说明、设定牛市/熊市情景，并识别驱动股价波动的关键因素。在提及“业绩前瞻”、“[公司]财报关注点”、“财报前准备”、“财报设定”或“预览[公司]第[X]季度财报”时触发。

## 工作流程

### 第一步：收集背景信息

- 确定公司名称及目标财报季度
- 通过网页搜索提取一致预期（营收、每股收益、核心业务部门指标）
- 获取财报发布日期和具体时间（盘前发布 vs. 盘后发布）
- 回顾公司上季度业绩电话会议，关注业绩指引或管理层评论

### 第二步：核心指标框架

根据公司特性构建“关注清单”框架：

**财务指标：**
- 营收对比一致预期（总额及各业务部门）
- 每股收益（EPS）对比一致预期
- 利润率（毛利率、经营利润率、净利率）—— 处于扩张还是收缩状态？
- 自由现金流
- 前瞻性指引对比一致预期

**运营指标**（行业特定）：
- 科技/软件服务（SaaS）：年度经常性收入（ARR）、净存续率（NRR）、剩余履约义务（RPO）、客户数
- 零售业：同店销售额、客流量、客单价
- 工业：订单积压、订单出货比（Book-to-bill）、价格对比销量
- 金融业：净利息收益率（NIM）、信贷质量、贷款增长、手续费及佣金收入
- 医疗健康：处方量、接诊量、研发管线更新

### 第三步：情景分析

构建包含股价影响预测的三种情景：

| 情景 | 营收 | 每股收益（EPS） | 核心驱动因素 | 股价反应 |
|----------|---------|-----|------------|----------------|
| 牛市（Bull） | | | | |
| 基准（Base） | | | | |
| 熊市（Bear） | | | | |

针对每种情景：
- 需要在运营层面达成哪些条件
- 管理层评论中会出现哪些信号
- 历史背景 —— 过去类似业绩发布后股价如何波动？

### 第四步：催化剂检查清单

识别决定股价反应的 3-5 个核心因素：

1. [指标] 对比 [一致预期/密报数] —— 为什么这个指标至关重要
2. [业绩指引项] —— 买方市场（Buy-side）期待听到的内容
3. [叙事转变] —— 任何战略调整、并购（M&A）或重组动向

### 第五步：输出结果

单页业绩前瞻，包含以下内容：
- 公司名称、财报季度、发布日期
- 一致预期汇总表
- 核心关注指标（按重要性排序）
- 牛市/基准/熊市情景表
- 催化剂检查清单
- 交易设定：近期股价表现、期权隐含波动幅度

## 重要说明

- 一致预期会随时间变动 —— 务必记录数据来源及预期日期
- 来自买方调查的“密报数（Whisper numbers）”通常比公开的一致预期更具参考价值
- 历史业绩反应有助于校准预期（搜索“[公司]历史业绩股价波动记录”）
- 期权隐含波动幅度可以告诉你市场的预期水平 —— 请将其与你的情景分析进行对比

> **💡 Appendix: 领域知识小贴士**
>
> 1. **一致预期 (Consensus Estimates)**：就像是全班同学一起猜考试平均分，华尔街的分析师们会对公司的收入、利润给出一个平均预测。它是市场的“及格线”。
> 2. **EPS (每股收益)**：公司赚的总钱数除以发出的总股票数。简单说，就是你手里的每一股股票在这个季度帮你赚了多少钱。
> 3. **业绩指引 (Guidance)**：公司管理层对未来的“天气预报”。市场不仅看公司过去表现如何，更在乎管理层觉得自己明年能赚多少。
> 4. **买方 (Buy-side)**：指那些真正拿着巨额资金去买股票的机构，比如养老金、公募基金或对冲基金。他们才是市场的“金主爸爸”。
> 5. **密报数 (Whisper numbers)**：一种非官方的预期。大家都说及格线是 60 分，但大佬们私下里可能觉得这公司考 80 分才算好。如果公司只考了 70 分，虽然过了及格线，股价还是可能因为没达到“密报数”而下跌。
> 6. **隐含波动 (Implied Move)**：期权市场预示的“大震指数”。它能直接告诉你，投资者预期财报发布那天股价可能上下“蹦跶”的剧烈程度。
