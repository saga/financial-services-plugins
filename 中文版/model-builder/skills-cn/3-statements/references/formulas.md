# 公式参考指南

**重要提示：** 除非用户另有说明，请务必使用本参考文档中列出的公式。

---

## 核心勾稽关系

```
资产负债表：        资产 = 负债 + 权益
净利润：           损益表净利润 → 经营活动现金流（起点）
现金流量：         现金变动额 = 经营活动现金流 (CFO) + 投资活动现金流 (CFI) + 筹资活动现金流 (CFF)
现金勾稽：         期末现金（现金流量表）= 现金（资产负债表资产项）
月度/年度现金一致性：期末现金（月度）= 期末现金（年度）
留存收益：         期初留存收益 + 净利润 - 股利 = 期末留存收益
股权融资：         普通股/资本公积变动（资产负债表）= 股票发行（筹资活动现金流）
第 0 年权益：      融资总额（第 0 年）= 期初权益（第 1 年）
```

## 毛利润计算

**重要提示：** 毛利润必须基于“营业净收入”计算，而非“营业总收入”。

```
营业净收入 - 营业成本 = 毛利润
```

| 术语 | 定义 |
|------|------------|
| 营业总收入 (Gross Revenue) | 扣除任何项目前的总收入 |
| 营业净收入 (Net Revenue) | 营业总收入 - 退货 - 折让 - 折扣 |
| 营业成本 (Cost of Revenue) | 与销售产品/提供服务直接相关的成本 |
| 毛利润 (Gross Profit) | 营业净收入 - 营业成本 |

**注意：** 在进行盈利能力计算时，始终以营业净收入（在多数财务报表中也称为“净销售额”或直接简称为“收入”）作为起点。营业总收入会夸大实际的顶线业绩。

## 利润率公式

```
毛利率 %            = 毛利润 / 营业净收入
EBITDA              = EBIT + 折旧与摊销 (或 = 毛利润 - 营业费用)
EBITDA 利润率 %     = EBITDA / 营业净收入
EBIT 利润率 %       = EBIT / 营业净收入
净利润率 %          = 净利润 / 营业净收入
```

## 信用指标公式

```
总债务              = 短期借款及长期借款的一年内到期部分 + 长期借款
净债务              = 总债务 - 现金
总债务 / EBITDA     = 总债务 / EBITDA（源自损益表）
净债务 / EBITDA     = 净债务 / EBITDA（源自损益表）
利息保障倍数        = EBITDA / 利息支出（源自损益表）
净利息支出 % 债务   = 净利息支出 / 长期借款
资产负债率          = 总债务 / (总债务 + 总权益)
产权比率            = 总债务 / 总权益
流动比率            = 总流动资产 / 总流动负债
速动比率            = (总流动资产 - 存货) / 总流动负债
```

## 预测公式（百分比法 - 基于营业净收入）

```
营业成本（预测）    = 营业净收入 × 营业成本占比假设
销售与市场费用（预测）= 营业净收入 × 销售与市场费用占比假设
管理费用（预测）    = 营业净收入 × 管理费用占比假设
研发费用（预测）    = 营业净收入 × 研发费用占比假设
股份支付费（预测）  = 营业净收入 × 股份支付费占比假设
```

## 营运资本公式

```
应收账款
  期初应收账款
  + 营业收入（源自损益表）
  - 现金回收（轧差）
  = 期末应收账款
  应收账款周转天数 (DSO) = (应收账款 / 营业收入) × 365

存货
  期初存货
  + 采购额（轧差）
  - 营业成本（源自损益表）
  = 期末存货
  存货周转天数 (DIO) = (存货 / 营业成本) × 365

应付账款
  期初应付账款
  + 采购额（源自存货计算）
  - 现金支付（轧差）
  = 期末应付账款
  应付账款周转天数 (DPO) = (应付账款 / 营业成本) × 365

营运资本净额 = 应收账款 + 存货 - 应付账款
营运资本变动额 = 本期营运资本净额 - 上期营运资本净额
```

## 折旧与摊销 (D&A) 计划公式

```
期初固定资产 (PP&E) 原值
+ 资本支出 (CapEx)
= 期末固定资产 (PP&E) 原值

期初累计折旧
+ 折旧费用
= 期末累计折旧

固定资产净值 = 固定资产原值 - 累计折旧
```

## 债务计划公式

```
期初债务余额
+ 新增借款
- 偿还额
= 期末债务余额

利息支出 = 平均债务余额 × 利率
  (可使用期初余额以避免循环引用，若启用循环引用则使用平均值)
```

## 留存收益公式

```
期初留存收益
+ 净利润（源自损益表）
+ 股份支付费 (SBC)（源自损益表）
- 股利
= 期末留存收益
```

## 净经营亏损 (NOL) 计划公式

```
NOL 结转计划

期初 NOL 余额（第 1 年 / 成立时 = 0）
+ 新增 NOL（若 EBT < 0，则取 EBT 的绝对值，否则为 0）
- 已利用 NOL（受限于应纳税所得额及利用率上限）
= 期末 NOL 余额

初始余额规则

对于新业务或首个建模周期：
  期初 NOL 余额 = 0
  NOL 仅能通过实现的亏损（EBT < 0）增加
  NOL 不能凭空产生或假设

NOL 利用额计算

税前利润 (EBT)
  若 EBT > 0：
    可用 NOL = 期初 NOL 余额
    利用上限 = EBT × 80% (2017 年后美国联邦税法限制)
    已利用 NOL = MIN(可用 NOL, 利用上限)
    应纳税所得额 = EBT - 已利用 NOL
  若 EBT ≤ 0：
    已利用 NOL = 0
    应纳税所得额 = 0
    新增 NOL = ABS(EBT)

带 NOL 的所得税计算

应交所得税 = MAX(0, 应纳税所得额 × 税率)
  (税额不能为负；亏损将转化为 NOL 资产)

NOL 递延所得税资产 (DTA)

递延所得税资产 - NOL 结转 = 期末 NOL 余额 × 税率
DTA 变动额 = 本期 DTA - 上期 DTA
  (DTA 增加 = 现金流量表中的非现金收益)
  (DTA 减少 = 现金流量表中的非现金支出)
```

## 资产负债表结构

```
资产
  现金（源自现金流量表期末现金）
  应收账款（源自营运资本）
  存货（源自营运资本）
  流动资产合计
  
  固定资产净值（源自折旧与摊销计划）
  递延所得税资产 - NOL（源自 NOL 计划）
  非流动资产合计
  资产总计

负债
  应付账款（源自营运资本）
  一年内到期的长期债务（源自债务计划）
  流动负债合计
  
  长期债务（源自债务计划）
  负债合计

权益
  普通股
  留存收益（源自留存收益计划）
  权益合计

平衡校验：资产 - 负债 - 权益 = 0
```

## 现金流量表结构

```
经营活动现金流 (CFO)
  净利润（勾稽：损益表）
  + 折旧与摊销（勾稽：折旧与摊销计划）
  + 股份支付费 (SBC)（勾稽：损益表或假设）
  - 递延所得税资产 (ΔDTA) 变动（勾稽：NOL 计划；DTA 增加即为现金流出）
  - 应收账款变动 (ΔAR)（勾稽：营运资本）
  - 存货变动 (ΔInventory)（勾稽：营运资本）
  + 应付账款变动 (ΔAP)（勾稽：营运资本）
  = 经营活动现金流

投资活动现金流 (CFI)
  - 资本支出 (CapEx)（勾稽：折旧与摊销计划）
  = 投资活动现金流

筹资活动现金流 (CFF)
  + 债务发行（勾稽：债务计划）
  - 债务偿还（勾稽：债务计划）
  + 股票发行（勾稽：资产负债表普通股/资本公积）
  - 股利支付（勾稽：留存收益计划）
  = 筹资活动现金流

现金净变动额 = CFO + CFI + CFF
期初现金
+ 现金净变动额
= 期末现金（勾稽至：资产负债表现金项）
```

```
营业净收入
  增长率 %
(-) 营业成本
  % 营业净收入占比
────────────────
毛利润 (= 营业净收入 - 营业成本)
  毛利率 %

(-) 销售与市场费用
  % 营业净收入占比
(-) 管理费用
  % 营业净收入占比
(-) 研发费用
  % 营业净收入占比
(-) 折旧与摊销
(-) 股份支付费
  % 营业净收入占比
────────────────
EBIT（息税前利润）
  EBIT 利润率 %

EBITDA（息税折旧摊销前利润）
  EBITDA 利润率 %

(-) 利息支出
────────────────

(-) 所得税（应纳税所得额 × 税率）
────────────────

  净利润率 %
```

## 校验公式

```
资产负债表平衡校验：    = 资产 - 负债 - 权益（必须 = 0）
现金勾稽校验：          = 资产负债表现金 - 现金流量表期末现金（必须 = 0）
留存收益滚动校验：      = 期初留存收益 + 净利润 + 股份支付费 - 股利 - 期末留存收益（必须 = 0）
DTA 勾稽校验：          = NOL 计划 DTA - 资产负债表 DTA（必须 = 0）
股权融资勾稽校验：      = 普通股/资本公积变动 - 股票发行（必须 = 0）
第 0 年权益勾稽校验：   = 融资总额（第 0 年）- 期初权益（第 1 年）（必须 = 0）
月度 vs 年度现金校验：  = 期末现金（月度）- 期末现金（年度）（必须 = 0）
NOL 利用率上限：        = 已利用 NOL ≤ EBT × 80%（2017 年后必须为真）
NOL 非负校验：          = 期末 NOL 余额 ≥ 0（必须为真）
NOL 初始余额：          = 第 1 年期初 NOL = 0（新业务必须为真）
NOL 累积逻辑：          = 仅当 EBT < 0 时 NOL 增加（亏损产生 NOL）
```

> **💡 Appendix: 领域知识小贴士**
>
> 欢迎来到财务建模的世界！这里有三个核心概念，能帮你快速看懂这些“天书”般的公式：
>
> 1.  **三大报表的“连连看”**：财务报表不是孤立的。你可以把**损益表**看作你的“工资条”（赚了多少），**资产负债表**是你的“家底清单”（你有多少钱和欠多少债），而**现金流量表**则是你的“记账本”（钱到底是怎么进出的）。公式里的“勾稽”就是确保这三个表的数据能对得上，就像拼图一样。
> 2.  **什么是 EBITDA？**：这是金融圈最爱用的词。简单来说，它就像是抛开了所有“会计魔法”（折旧）和“政府抽成”（税收）之后，一家公司最原始的赚钱能力。之所以要把它算出来，是因为这样可以更公平地比较不同公司（比如一家公司是租设备，另一家是买设备）的生意好坏。
> 3.  **NOL 是什么“神仙”？**：NOL 的全称是“净经营亏损结转”。想象一下，如果你今年做生意亏了 100 万，政府会允许你在明年赚钱时，先扣掉这 100 万再交税。这就像是一个“亏损抵扣券”，能在未来帮你省下不少现金。不过要注意，现在有限制，你每赚 1 块钱，最多只能用这个“抵扣券”抵掉 8 毛钱。
