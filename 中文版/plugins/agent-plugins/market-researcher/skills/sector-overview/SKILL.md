# 行业概览

description: 创建全面的行业和领域格局报告，涵盖市场动态、竞争定位、关键参与者和主题趋势。用于客户请求、行业首次覆盖、主题研究或内部知识构建。触发词包括"行业概览"、"行业报告"、"市场格局"、"行业分析"、"行业深度研究"或"主题研究"。

## 工作流程

### 步骤1：定义范围

- **行业/子行业**：什么行业，如何狭义定义？
- **目的**：客户报告、内部研究、推介材料、投资构想生成
- **深度**：高层概览（5-10页）还是深度研究（20-30页）
- **角度**：中性格局 vs. 主题论点（如"AI基础设施扩建"）
- **范围**：仅上市公司，还是包括私营公司？

### 步骤2：市场概览

**市场规模与增长**
- 总可寻址市场（TAM）及来源
- 历史增长率（5年CAGR）
- 预测增长率和关键假设
- 市场细分（按产品、地理、终端市场、客户类型）

**行业结构**
- 分散 vs. 整合 — 前5大市场份额
- 价值链地图 — 价值在哪里累积？
- 商业模式类型（订阅、交易、许可、服务）
- 进入壁垒（资本、监管、技术、网络效应）

**关键趋势与驱动因素**
- 长期顺风（3-5个主要趋势）
- 逆风和风险提示
- 技术颠覆向量
- 监管发展
- 并购活动和整合趋势

### 步骤3：竞争格局

**公司概况**（前5-10名参与者）：

| 公司 | 收入 | 增长率 | EBITDA利润率 | 市场份额 | 关键差异化因素 |
|------|------|--------|--------------|----------|----------------|
| | | | | | |

对每个公司，简要概况：
- 业务描述（2-3句话）
- 战略定位和护城河
- 近期动态（财报、并购、产品发布）
- 估值快照（P/E、EV/EBITDA、EV/收入）

**竞争动态**
- 公司如何竞争？（价格、产品、服务、分销）
- 谁在赢得/失去份额？为什么？
- 新进入者或相邻玩家的颠覆风险

### 步骤4：估值背景

- 行业交易倍数（当前和历史范围）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 该行业与大盘相比如何？

### 步骤5：投资启示

- 最佳风险/回报机会在哪里？
- 可以通过该行业表达哪些主题投资？
- 行业内的关键辩论（多头 vs. 空头论点）
- 可能改变行业叙事的催化剂

### 步骤6：输出

- Word文档或PowerPoint，包含：
  - 市场概览和规模
  - 竞争格局地图
  - 公司对比表
  - 估值汇总
  - 关键图表：市场增长、份额趋势、估值历史
- 包含详细公司数据的Excel附录

## 重要说明

- 引用所有市场规模数据 — 注明研究公司或方法论
- 区分TAM炒作和现实可寻址市场
- 行业概览更新很快 — 注明日期并标记可能过时的数据
- 图表至关重要 — 市场规模瀑布图、竞争定位矩阵、估值散点图
- 如果用于客户，将"所以呢"定制到他们的具体情况（并购目标识别、竞争定位、市场进入）

## Appendix: 金融背景知识

这份文件是"行业概览（Sector Overview）"技能的详细说明。在投资分析中，当你面对一个完全陌生的行业时，Sector Overview 是你读的第一份材料——它帮你快速建立对整个行业的宏观认知。

---

### 1. 什么是 Sector Overview？

**类比**：你去一个陌生城市旅游。你不会第一站就去某条小巷找某个网红咖啡——你会先看城市地图：哪个区域是商业区、哪条是主干道、机场在哪、酒店应该订在哪。

Sector Overview 就是投资的"城市地图"——让你先看清整个行业的全貌，再决定深入哪个角落。

---

### 2. Sector Overview 的"五大必答问题"

| 问题 | 具体内容 |
|------|---------|
| 这个行业多大？ | TAM（总市场规模）——$100 亿还是 $10000 亿？ |
| 在增长还是萎缩？ | CAGR 正向增长还是负增长？增速在加速还是放缓？ |
| 怎么赚钱？ | 行业利润最高的环节在哪？（上游/中游/下游） |
| 谁是老大？ | 前 5 名市占率加起来多少？是集中还是分散？ |
| 什么在变？ | 有什么新技术/新规则/新玩家在颠覆行业？ |

---

### 3. TAM vs SAM vs SOM

| 缩写 | 含义 | 类比 |
|------|------|------|
| TAM | 总可寻址市场 | 全中国所有人每年在"喝东西"上花的总钱 |
| SAM | 可服务市场 | 其中"咖啡"的市场 |
| SOM | 可获取市场 | 其中"在你们城市能买到的咖啡"的市场 |

**投资原则**：TAM 越大越好——但 SAM/SOM 也必须足够大。

---

### 给小白的一句话

> Sector Overview 就是投资的"行业地图"——当你面对一个全新行业（比如"基因编辑"或"核聚变"），第一时间不是研究某家公司，而是搞清楚：这个行业到底多大（TAM）、在长在缩、谁在赚钱、什么力量在改变格局。就像你去一个新城市先看地图，Sector Overview 让你从"一无所知"到"能跟上讨论"——这是研究的第 0 步。
