# 行业概览

description: 创建全面的行业和领域格局报告，涵盖市场动态、竞争定位、关键参与者和主题趋势。用于客户请求、行业首次覆盖、主题研究或内部知识构建。触发词包括"行业概览"、"行业报告"、"市场格局"、"行业分析"、"行业深度研究"或"主题研究"。

## 工作流程

### 步骤1：定义范围

- **行业/子行业**：什么行业，如何狭义定义？
- **目的**：客户报告、内部研究、推介材料、投资构想生成
- **深度**：高层概览（5-10页）还是深度研究（20-30页）
- **角度**：中性格局 vs. 主题论点（如"AI基础设施扩建"）
- **范围**：仅上市公司，还是包括私营公司？

### 步骤2：市场概览

**市场规模与增长**
- 总可寻址市场（TAM）及来源
- 历史增长率（5年CAGR）
- 预测增长率和关键假设
- 市场细分（按产品、地理、终端市场、客户类型）

**行业结构**
- 分散 vs. 整合 — 前5大市场份额
- 价值链地图 — 价值在哪里累积？
- 商业模式类型（订阅、交易、许可、服务）
- 进入壁垒（资本、监管、技术、网络效应）

**关键趋势与驱动因素**
- 长期顺风（3-5个主要趋势）
- 逆风和风险提示
- 技术颠覆向量
- 监管发展
- 并购活动和整合趋势

### 步骤3：竞争格局

**公司概况**（前5-10名参与者）：

| 公司 | 收入 | 增长率 | EBITDA利润率 | 市场份额 | 关键差异化因素 |
|------|------|--------|--------------|----------|----------------|
| | | | | | |

对每个公司，简要概况：
- 业务描述（2-3句话）
- 战略定位和护城河
- 近期动态（财报、并购、产品发布）
- 估值快照（P/E、EV/EBITDA、EV/收入）

**竞争动态**
- 公司如何竞争？（价格、产品、服务、分销）
- 谁在赢得/失去份额？为什么？
- 新进入者或相邻玩家的颠覆风险

### 步骤4：估值背景

- 行业交易倍数（当前和历史范围）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 该行业与大盘相比如何？

### 步骤5：投资启示

- 最佳风险/回报机会在哪里？
- 可以通过该行业表达哪些主题投资？
- 行业内的关键辩论（多头 vs. 空头论点）
- 可能改变行业叙事的催化剂

### 步骤6：输出

- Word文档或PowerPoint，包含：
  - 市场概览和规模
  - 竞争格局地图
  - 公司对比表
  - 估值汇总
  - 关键图表：市场增长、份额趋势、估值历史
- 包含详细公司数据的Excel附录

## 重要说明

- 引用所有市场规模数据 — 注明研究公司或方法论
- 区分TAM炒作和现实可寻址市场
- 行业概览更新很快 — 注明日期并标记可能过时的数据
- 图表至关重要 — 市场规模瀑布图、竞争定位矩阵、估值散点图
- 如果用于客户，将"所以呢"定制到他们的具体情况（并购目标识别、竞争定位、市场进入）

## Appendix: 金融背景知识

### 什么是行业研究（Sector Research）？

行业研究是对特定行业（如科技、医疗、金融）的全面分析，帮助投资者了解：
- 行业规模和增长前景
- 竞争格局和关键参与者
- 驱动行业发展的趋势和风险
- 投资机会和估值背景

### 关键术语解释

**TAM（Total Addressable Market，总可寻址市场）**：公司产品或服务理论上可以达到的最大市场规模。例如，全球云计算市场规模。

**CAGR（Compound Annual Growth Rate，复合年增长率）**：一项投资或市场在特定时期内的年均增长率。例如，5年CAGR为15%意味着每年平均增长15%。

**价值链（Value Chain）**：从原材料到最终消费者的完整流程和参与者。例如，锂电池价值链包括：锂矿开采 → 锂精炼 → 电池制造 → 电动车制造 → 消费者。

**护城河（Moat）**：公司的可持续竞争优势，使其能长期保持超额利润。常见护城河：
- **品牌**：如苹果、可口可乐
- **网络效应**：如微信、Facebook
- **监管壁垒**：如银行牌照
- **转换成本**：如企业软件、医疗器械
- **成本优势**：如沃尔玛、特斯拉

**进入壁垒（Barriers to Entry）**：新公司进入行业面临的障碍，包括：
- **资本要求**：需要大量初始投资
- **监管**：需要牌照、审批
- **技术**：需要专利、专有技术
- **网络效应**：用户基数越大越有价值

**顺风（Tailwinds）**：推动行业增长的有利趋势，如人口老龄化推动医疗需求。

**逆风（Headwinds）**：阻碍行业增长的不利因素，如利率上升打压房地产。

**并购活动（M&A Activity）**：行业内的并购交易频率和规模，反映行业整合程度。

**交易倍数（Trading Multiples）**：基于市场交易价格的估值指标，如P/E、EV/EBITDA。

**溢价/折价（Premium/Discount）**：相比行业平均估值，某公司/行业被给予更高（溢价）或更低（折价）的估值。

**私营公司（Private Companies）**：未上市公司的信息通常较少，但可能代表行业重要部分（如独角兽公司）。
