# 投资构想生成

description: 系统化股票筛选和投资构想挖掘。结合量化筛选、主题研究和模式识别来发现新的多空构想。用于寻找新构想、运行筛选器或进行主题扫描。触发词包括"构想生成"、"股票筛选"、"寻找构想"、"什么看起来有趣"、"筛选"、"新构想"或"给我推介点什么"。

## 工作流程

### 步骤1：定义搜索标准

向用户询问参数：
- **方向**：做多构想、做空构想，或两者
- **市值**：大盘、中盘、小盘、微盘
- **行业**：特定行业或跨行业
- **风格**：价值、成长、质量、特殊情况、事件驱动
- **地理**：美国、国际、全球
- **主题**：任何特定主题角度（AI、回流、人口老龄化等）

### 步骤2：量化筛选

根据风格运行筛选：

**价值筛选**
- P/E低于行业中位数
- EV/EBITDA低于历史平均
- 自由现金流收益率 >5%
- 市净率低于1.5倍
- 近90天内部人买入
- 股息收益率高于市场平均

**成长筛选**
- 收入增长 >15% 同比
- 盈利增长 >20% 同比
- 收入加速（增长率递增）
- 利润率扩张
- 高投入资本回报率（>15%）
- 强净留存率（SaaS >110%）

**质量筛选**
- 持续收入增长（5年以上）
- 稳定或扩张的利润率
- ROE >15%
- 低债务/股权比
- 高自由现金流转换率
- 内部人持股 >5%

**做空筛选**
- 收入下降或增长减速
- 利润率压缩
- 应收账款/库存 vs. 销售上升
- 内部人抛售
- 无理由的估值溢价 vs. 同行
- 高卖空利率伴随基本面恶化
- 会计红旗（审计师变更、财务重述）

**特殊情况筛选**
- 近期IPO/SPAC锁定期满
- 过去12个月的分拆
- 重组中的公司
- 激进投资者介入
- 表现不佳公司的管理层变更

### 步骤3：主题扫描

对于主题构想，研究主题并识别受益者：

1. 定义论点（如"AI基础设施支出加速至2026"）
2. 映射价值链 — 谁直接受益 vs. 间接受益？
3. 识别纯业务 exposure vs. 多元化 exposure
4. 评估哪些名字已"计入价格" vs. 被低估
5. 寻找市场尚未联系到该主题的二阶受益者

### 步骤4：构想展示

对每个通过筛选的构想，展示：

**[公司名称] — [做多/做空] — [一句话论点]**

| 指标 | 值 | vs. 同行 |
|------|-----|-----------|
| 市值 | | |
| EV/EBITDA (NTM) | | |
| P/E (NTM) | | |
| 收入增长 | | |
| EBITDA利润率 | | |
| FCF收益率 | | |

**论点（3-5个要点）：**
- 为何被错误定价
- 市场遗漏了什么
- 实现价值的催化剂

**关键风险：**
- 什么情况会使这个构想错误

**建议后续步骤：**
- 构建完整模型？深度尽调？专家电话？

### 步骤5：输出

- 5-10个构想的短名单，附一页摘要
- 筛选标准和方法论文档化
- 所有构想的对比表
- 优先列表：哪些构想应首先研究

## 重要说明

- 筛选器提供候选，不是结论 — 每个筛选输出都需要基本面工作
- 最佳构想往往来自交叉点（如由于临时逆风而以价值价格交易的质量公司）
- 避免拥挤交易 — 检查持股数据、卖空利率和多少分析师覆盖该名字
- 逆向构想需要催化剂 — 没有催化剂的过早等同于错误
- 跟踪构想命中率随时间 — 哪些筛选器和方法产生最佳构想？
- 做空构想需要更高确信度 — 时机更难且风险不对称

- **特殊情况（Special Situation）**：由企业行动（并购、分拆、重组）创造的机会。

- **卖空利率（Short Interest）**：被卖空股份占总流通股比例。高值可能预示悲观，但也可能带来空头回补反弹。

## Appendix: 金融背景知识

这份文件是"投资主题生成（Idea Generation）"技能的详细说明。在投资管理中，Idea Generation 是最重要的"创意环节"——不是听别人说"买哪只"，而是自己从数据和趋势中发现投资机会。

---

### 1. 什么是 Idea Generation？

**类比**：你是乐队经纪人，在寻找下一支会火的乐队。你不能只盯着"已经在排行榜上的"（因为大家都知道，太贵了）——你要去地下音乐节、去 B 站翻新作品、去找那种"有几个铁粉、但还没火的"——这就是 Idea Generation：在别人还没发现之前找到宝藏。

---

### 2. Idea Generation 的"三大来源"

| 来源 | 方法 | 例子 |
|------|------|------|
| 自上而下 | 判断宏观趋势 → 找受益公司 | 碳减排趋势 → 找新能源股票 |
| 自下而上 | 翻财报发现异常信号 | 某公司毛利率突然大幅提升 → 深挖原因 |
| 数据驱动 | 用 screen 筛出符合标准的 | PE<15 + 收入增速>20% + 机构低配 |

---

### 3. 一个好 Idea 的"三要素"

| 要素 | 含义 |
|------|------|
| 非共识 | 不是"所有人都说好"——如果所有分析师都推荐，那你已经晚了 |
| 可量化 | 不是说"这公司不错"——而是"PE 仅 12x，同行平均 18x" |
| 有催化剂 | 不是"总有一天会涨"——而是"下个月有新药审批结果" |

---

### 给小白的一句话

> Idea Generation 就是基金经理的"寻宝图制作"——不是等别人告诉你"该买哪只股票"，而是你从数据、趋势和财报异常中自己发现机会。好的 Idea 有三个特点：1）非共识（大家都说好的已经涨上去了），2）可量化（不能只说"公司很棒"，要说"PE 只有 12 倍"），3）有催化剂（不能是"总有一天会涨"，而是"下个月要出新药审批结果"）。
