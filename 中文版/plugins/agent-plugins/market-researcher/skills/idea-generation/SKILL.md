# 投资构想生成

description: 系统化股票筛选和投资构想挖掘。结合量化筛选、主题研究和模式识别来发现新的多空构想。用于寻找新构想、运行筛选器或进行主题扫描。触发词包括"构想生成"、"股票筛选"、"寻找构想"、"什么看起来有趣"、"筛选"、"新构想"或"给我推介点什么"。

## 工作流程

### 步骤1：定义搜索标准

向用户询问参数：
- **方向**：做多构想、做空构想，或两者
- **市值**：大盘、中盘、小盘、微盘
- **行业**：特定行业或跨行业
- **风格**：价值、成长、质量、特殊情况、事件驱动
- **地理**：美国、国际、全球
- **主题**：任何特定主题角度（AI、回流、人口老龄化等）

### 步骤2：量化筛选

根据风格运行筛选：

**价值筛选**
- P/E低于行业中位数
- EV/EBITDA低于历史平均
- 自由现金流收益率 >5%
- 市净率低于1.5倍
- 近90天内部人买入
- 股息收益率高于市场平均

**成长筛选**
- 收入增长 >15% 同比
- 盈利增长 >20% 同比
- 收入加速（增长率递增）
- 利润率扩张
- 高投入资本回报率（>15%）
- 强净留存率（SaaS >110%）

**质量筛选**
- 持续收入增长（5年以上）
- 稳定或扩张的利润率
- ROE >15%
- 低债务/股权比
- 高自由现金流转换率
- 内部人持股 >5%

**做空筛选**
- 收入下降或增长减速
- 利润率压缩
- 应收账款/库存 vs. 销售上升
- 内部人抛售
- 无理由的估值溢价 vs. 同行
- 高卖空利率伴随基本面恶化
- 会计红旗（审计师变更、财务重述）

**特殊情况筛选**
- 近期IPO/SPAC锁定期满
- 过去12个月的分拆
- 重组中的公司
- 激进投资者介入
- 表现不佳公司的管理层变更

### 步骤3：主题扫描

对于主题构想，研究主题并识别受益者：

1. 定义论点（如"AI基础设施支出加速至2026"）
2. 映射价值链 — 谁直接受益 vs. 间接受益？
3. 识别纯业务 exposure vs. 多元化 exposure
4. 评估哪些名字已"计入价格" vs. 被低估
5. 寻找市场尚未联系到该主题的二阶受益者

### 步骤4：构想展示

对每个通过筛选的构想，展示：

**[公司名称] — [做多/做空] — [一句话论点]**

| 指标 | 值 | vs. 同行 |
|------|-----|-----------|
| 市值 | | |
| EV/EBITDA (NTM) | | |
| P/E (NTM) | | |
| 收入增长 | | |
| EBITDA利润率 | | |
| FCF收益率 | | |

**论点（3-5个要点）：**
- 为何被错误定价
- 市场遗漏了什么
- 实现价值的催化剂

**关键风险：**
- 什么情况会使这个构想错误

**建议后续步骤：**
- 构建完整模型？深度尽调？专家电话？

### 步骤5：输出

- 5-10个构想的短名单，附一页摘要
- 筛选标准和方法论文档化
- 所有构想的对比表
- 优先列表：哪些构想应首先研究

## 重要说明

- 筛选器提供候选，不是结论 — 每个筛选输出都需要基本面工作
- 最佳构想往往来自交叉点（如由于临时逆风而以价值价格交易的质量公司）
- 避免拥挤交易 — 检查持股数据、卖空利率和多少分析师覆盖该名字
- 逆向构想需要催化剂 — 没有催化剂的过早等同于错误
- 跟踪构想命中率随时间 — 哪些筛选器和方法产生最佳构想？
- 做空构想需要更高确信度 — 时机更难且风险不对称

## Appendix: 金融背景知识

### 什么是投资构想生成（Idea Generation）？

投资构想生成是系统寻找有吸引力的投资机会的过程，结合：
- **量化筛选**：用财务指标过滤股票
- **主题研究**：识别长期趋势受益者
- **模式识别**：发现市场定价的错误

### 关键术语解释

**做多（Long）**：买入股票，期待价格上涨获利。
**做空（Short）**：借入股票卖出，期待价格下跌后低价买回归还，赚取差价。

**市值分类**：
- **大盘（Large Cap）**：市值 > 100亿美元
- **中盘（Mid Cap）**：市值 20-100亿美元
- **小盘（Small Cap）**：市值 3-20亿美元
- **微盘（Micro Cap）**：市值 < 3亿美元

**价值投资（Value）**：寻找交易价格低于内在价值的股票，通常特征：
- 低P/E、低EV/EBITDA
- 高股息收益率
- 资产折价交易

**成长投资（Growth）**：寻找盈利快速增长的公司，通常特征：
- 高收入/盈利增长率
- 高P/E（市场愿意为增长付费）
- 再投资而非分红

**质量投资（Quality）**：寻找财务健康、商业模式优秀的公司，通常特征：
- 高ROE/ROIC
- 稳定利润率
- 低债务
- 强劲现金流

**特殊情况（Special Situation）**：由企业行动（并购、分拆、重组）创造的机会。

**事件驱动（Event-driven）**：由特定事件（财报、管理层变更、激进投资者）驱动的投资策略。

**自由现金流收益率（FCF Yield）**：自由现金流/市值，衡量公司产生现金回报的能力。

**内部人交易（Insider Trading）**：
- **买入**：通常看涨信号
- **抛售**：可能看跌信号（但需注意定期抛售计划）

**卖空利率（Short Interest）**：被卖空的股份占总流通股的比例。高卖空利率可能预示悲观情绪，但也可能带来空头回补反弹。

**护城河评估**：
- **网络效应**：用户/供应商飞轮
- **转换成本**：技术集成深度、合同锁定
- **规模经济**：产量的单位成本优势
- **无形资产**：品牌、专利、监管牌照

**催化剂（Catalyst）**：推动股价变动的事件，如财报发布、并购公告、新产品发布、管理层变更。

**二阶效应（Second-order Effects）**：主题的直接受益者之外的间接受益者。例如，AI主题不仅利好英伟达（直接），还利好冷却系统公司（间接）。

**拥挤交易（Crowded Trade）**：太多投资者持有相同头寸，一旦情绪转变，可能导致踩踏。

**误定价（Mispricing）**：市场价格与公司内在价值之间的差异，创造投资机会。

**NTM（Next Twelve Months，未来12个月）**：前瞻性指标，如NTM P/E指未来12个月的预期市盈率。

**EV（Enterprise Value，企业价值）**：市值 + 债务 - 现金，反映收购整个公司的总成本。

**ROIC（Return on Invested Capital，投入资本回报率）**：衡量公司用投入资本产生回报的效率。

**ROE（Return on Equity，净资产收益率）**：净利润/股东权益，衡量股东投资的回报率。
