# 投资建议书

description: 为潜在客户创建专业的投资建议书。涵盖公司方法、建议配置、预期结果和费用结构。用于推介新客户或展示新投资策略。触发词包括"投资建议书"、"潜在客户演示"、"推介新客户"、"[客户]建议书"或"新客户演示"。

## 工作流程

### 步骤1：潜在客户背景

收集：
- **潜在客户姓名**和家庭详情
- **现状**：现有顾问？自主投资？什么促使这次会议？
- **资产**：估计管理资产（AUM）、账户类型、当前持仓（如分享）
- **目标**：退休、财富保值、增长、收入、教育、遗产
- **风险承受能力**：保守、温和、激进（或问卷得分）
- **约束条件**：ESG偏好、集中持股、流动性需求
- **费用敏感度**：他们现在支付什么？
- **竞争对手**：他们还在考虑谁？

### 步骤2：建议书结构

**I. 关于我们公司（1页）**
- 公司概览、历史、管理资产
- 投资哲学（用通俗英语）
- 团队简介（与此客户相关）
- 客户服务模式（我们多久会面一次，他们联系谁）

**II. 了解您的需求（1页）**
- 重述他们的目标和担忧 — 表明你倾听了
- 发现阶段识别的关键规划考虑
- 成功对他们来说是什么样子

**III. 建议投资策略（2-3页）**
- 带理由的建议资产配置
- 配置如何映射到他们的目标和风险承受能力
- 投资工具（ETF、共同基金、个股、另类投资）
- 税务 aware 策略（资产定位、税损收获）

建议配置：

|------------|-----------|---------|-----------|

**IV. 预期结果（1-2页）**
- 预测增长情境（保守、温和、乐观）
- 实现目标的蒙特卡洛概率
- 收入预测（如退休或收入焦点）
- 风险指标（最大回撤、波动率）
- 与当前投资组合比较（如已知）

**V. 费用结构（1页）**
- 咨询费时间表（如适用，分级）
- 底层基金费用
- 总全部成本估算
- 费用与行业平均比较
- 价值主张 — 他们为费用得到什么

**VI. 开始使用（1页）**
- 开户流程
- 资产转移时间表
- 过渡计划（如果从另一位顾问转移）
- 前90天 — 期待什么
- 必需文件和后续步骤

### 步骤3：定制

- 根据潜在客户调整语气（公司高管 vs. 小企业主 vs. 退休人员）
- 如果他们有集中持股头寸，直接处理
- 如果他们在将你与机器人顾问比较，强调规划和关系价值
- 如果他们价格敏感，从总价值和结果入手，不仅仅是费用

### 步骤4：输出

- 带公司品牌的PowerPoint演示文稿（12-15张幻灯片）
- PDF留后版本
- 用于跟进邮件的一页摘要

## 重要说明

- 建议书应感觉个性化，不是模板化 — 引用他们的具体情况
- 不要过度推销表现 — 设定现实预期并强调流程
- 始终包含免责声明（预测是假设的、过往表现等）
- 过渡计划很重要 — 客户害怕转换顾问的中断
- 48小时内跟进建议书和清晰的后续步骤
- 在向潜在客户展示前，合规必须审查

## Appendix: 金融背景知识

本文件涉及的通用金融概念已整理至知识库，请按需查阅：

- **财富管理**：AUM / IPS / 资产配置 / 退休账户 / 税务优化 → [财富管理.md](../../../../金融知识学习/财富管理.md)

**本文件专属**：
- **投资建议书（Investment Proposal）**：财富管理公司向潜在客户展示的正式文件，包含公司资质、投资策略、预期回报、费用结构。
- **分级费用（Tiered Fee）**：根据 AUM 规模分段收费，资产越多费率越低。
- **机器人顾问（Robo-advisor）**：自动化投资平台，费用低（如 0.25%/年），但缺乏个性化规划。
- **ADV表格（Form ADV）**：美国投资顾问必须向 SEC 提交的披露文件。
- **CFP / CFA**：注册财务规划师 / 特许金融分析师，专业认证。
