# 客户审查准备

description: 准备客户审查会议，包含投资组合表现摘要、配置分析、谈话要点和行动项。将账户数据汇总为简洁的会议就绪格式。用于季度审查、年度检查或临时客户会议前。触发词包括"客户审查"、"[客户]会议准备"、"季度审查"、"[客户姓名]准备"或"客户会议"。

## 工作流程

### 步骤1：客户背景

收集或查找：
- **客户姓名**和家庭成员
- **账户类型**：应税账户、IRA、Roth、401(k)、信托等
- **总管理资产（AUM）** 跨账户
- **投资政策声明（IPS）**：目标配置、风险承受能力、约束条件
- **人生阶段**：积累期、退休前、退休、遗产规划
- **上次会面日期**和任何未完成的行动项

### 步骤2：投资组合表现

对每个账户和家庭总计：

| 指标 | 本季度 | 年初至今 | 1年 | 3年 | 成立以来 |
|------|--------|--------|------|-----|----------|
| 投资组合回报 | | | | | |
| 基准回报 | | | | | |
| 阿尔法（超额收益） | | | | | |

**表现归因：**
- 哪些资产类别/持仓推动了回报？
- 前3大贡献者和前3大拖累者
- 任何过大的单一持仓影响？

### 步骤3：配置审查

当前 vs. 目标配置：

| 资产类别 | 目标 | 当前 | 偏离 | 行动 |
|------------|--------|-------|------|--------|
| 美国大盘 | | | | |
| 美国中小盘 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 固定收益 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记任何超过IPS再平衡阈值（通常3-5%）的偏离。

### 步骤4：谈话要点

生成会议议程：

1. **市场概览**（2-3分钟）：简要宏观背景和展望
2. **投资组合表现**（5分钟）：我们做得如何？为什么？
3. **配置审查**（5分钟）：需要再平衡吗？
4. **规划更新**（5-10分钟）：
   - 生活变化？（工作、健康、家庭、住房、教育）
   - 收入需求变化？
   - 税务状况更新
   - 遗产规划更新
5. **行动项**（5分钟）：下次会面前我们要做什么？

### 步骤5：主动建议

基于审查，建议：
- 再平衡交易（如果偏离超过阈值）
- 税损收获机会
- 现金部署或提取规划
- Roth转换机会（如适用）
- 受益人更新或遗产规划需求
- 保险审查（人寿、残疾、长期护理）

### 步骤6：输出

- 一页客户审查摘要（Word或PDF）
- 带基准的表现表格
- 配置饼图（当前 vs. 目标）
- 建议行动项
- 会议议程

## 重要说明

- 会议前了解你的客户 — 回顾上次会面的笔记
- 从客户关心的事情入手，不是你想谈论的事情
- 如果表现不好，直接面对 — 不要隐藏或粉饰
- 始终以清晰的行动项和带日期的后续步骤结束
- 记录会议笔记和对IPS的任何更改
- 合规：确保所有材料符合公司政策和监管要求

## Appendix: 金融背景知识

### 什么是客户审查（Client Review）？

客户审查是财富管理中的定期会议，通常每季度或每年举行，审查：
- 投资组合表现 vs. 基准
- 资产配置是否符合目标
- 客户生活变化对财务规划的影响
- 税务优化机会
- 退休/教育/遗产规划进展

### 关键术语解释

**AUM（Assets Under Management，管理资产）**：顾问为客户管理的资产总额。

**IPS（Investment Policy Statement，投资政策声明）**：客户投资指南文件，规定：
- 目标资产配置（如60%股票/40%债券）
- 风险承受能力（保守/温和/激进）
- 投资约束（如"不投资烟草股"）
- 再平衡阈值（如偏离目标5%时再平衡）

**阿尔法（Alpha）**：投资组合回报超过基准的部分。正阿尔法表示跑赢市场。

**本季度（QTD，Quarter-to-Date）**：本季度开始至今的表现。
**年初至今（YTD，Year-to-Date）**：本年度开始至今的表现。

**偏离（Drift）**：当前配置与目标配置的差异。由于市场波动，股票涨得多可能导致股票配置超过目标，需要卖出股票买入债券来再平衡。

**税损收获（Tax-Loss Harvesting）**：卖出亏损持仓实现损失，用于抵减资本利得税，然后买入类似（但不完全相同）的证券维持市场敞口。

**Roth转换（Roth Conversion）**：将传统IRA资金转换为Roth IRA，现在缴税但未来免税增长和提取。

**Roth IRA vs. 传统IRA**：
- **传统IRA**：投入时抵税，提取时缴税
- **Roth IRA**：投入时缴税，提取时免税

**401(k)**：美国雇主赞助的退休储蓄计划，通常有雇主匹配。

**529计划**：美国教育储蓄计划，用于支付教育费用，通常州税有优惠。

**再平衡（Rebalancing）**：将投资组合调整回目标配置的过程。例如，目标60/40但现在是70/30，卖出10%股票买入10%债券。

**另类投资（Alternatives）**：传统股票债券之外的投资，如房地产、大宗商品、对冲基金、私募股权。

**ESG（Environmental, Social, Governance）**：环境、社会、治理因素投资，不考虑财务回报外还考虑可持续性。
