# 客户审查准备

description: 准备客户审查会议，包含投资组合表现摘要、配置分析、谈话要点和行动项。将账户数据汇总为简洁的会议就绪格式。用于季度审查、年度检查或临时客户会议前。触发词包括"客户审查"、"[客户]会议准备"、"季度审查"、"[客户姓名]准备"或"客户会议"。

## 工作流程

### 步骤1：客户背景

收集或查找：
- **客户姓名**和家庭成员
- **账户类型**：应税账户、IRA、Roth、401(k)、信托等
- **总管理资产（AUM）** 跨账户
- **投资政策声明（IPS）**：目标配置、风险承受能力、约束条件
- **人生阶段**：积累期、退休前、退休、遗产规划
- **上次会面日期**和任何未完成的行动项

### 步骤2：投资组合表现

对每个账户和家庭总计：

| 指标 | 本季度 | 年初至今 | 1年 | 3年 | 成立以来 |
|------|--------|--------|------|-----|----------|
| 投资组合回报 | | | | | |
| 基准回报 | | | | | |
| 阿尔法（超额收益） | | | | | |

**表现归因：**
- 哪些资产类别/持仓推动了回报？
- 前3大贡献者和前3大拖累者
- 任何过大的单一持仓影响？

### 步骤3：配置审查

当前 vs. 目标配置：

| 资产类别 | 目标 | 当前 | 偏离 | 行动 |
|------------|--------|-------|------|--------|
| 美国大盘 | | | | |
| 美国中小盘 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 固定收益 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记任何超过IPS再平衡阈值（通常3-5%）的偏离。

### 步骤4：谈话要点

生成会议议程：

1. **市场概览**（2-3分钟）：简要宏观背景和展望
2. **投资组合表现**（5分钟）：我们做得如何？为什么？
3. **配置审查**（5分钟）：需要再平衡吗？
4. **规划更新**（5-10分钟）：
   - 生活变化？（工作、健康、家庭、住房、教育）
   - 收入需求变化？
   - 税务状况更新
   - 遗产规划更新
5. **行动项**（5分钟）：下次会面前我们要做什么？

### 步骤5：主动建议

基于审查，建议：
- 再平衡交易（如果偏离超过阈值）
- 税损收获机会
- 现金部署或提取规划
- Roth转换机会（如适用）
- 受益人更新或遗产规划需求
- 保险审查（人寿、残疾、长期护理）

### 步骤6：输出

- 一页客户审查摘要（Word或PDF）
- 带基准的表现表格
- 配置饼图（当前 vs. 目标）
- 建议行动项
- 会议议程

## 重要说明

- 会议前了解你的客户 — 回顾上次会面的笔记
- 从客户关心的事情入手，不是你想谈论的事情
- 如果表现不好，直接面对 — 不要隐藏或粉饰
- 始终以清晰的行动项和带日期的后续步骤结束
- 记录会议笔记和对IPS的任何更改
- 合规：确保所有材料符合公司政策和监管要求

## Appendix: 金融背景知识

这份文件是"客户审阅（Client Review）"技能的详细说明。Client Review 是比 Client Report 更深入的"面对面沟通"——不是发一份 PDF 完事，而是你和客户坐下来深度讨论他的财富状况。

---

### 1. Client Review 的"三阶段"

| 阶段 | 内容 |
|------|------|
| 会前准备 | 更新组合数据、准备 3-5 个谈话要点、预估客户可能问的问题 |
| 会中执行 | 先报业绩、逐层解释、回答疑问、给出建议 |
| 会后跟进 | 48 小时内发会议记录、落实答应的事情 |

---

### 2. 客户常见的"压力问题"

| 客户问 | 潜台词 |
|--------|--------|
| "为什么别人都赚钱我亏了？" | 我需要一个让我不焦虑的解释 |
| "费用是不是太高了？" | 你要证明你物有所值 |
| "我想加仓比特币" | 你需要帮我理性分析风险 |

---

### 给小白的一句话

> Client Review 就是财富管家的"季度述职"——你面前坐着一个大客户，他的钱就是你的饭碗。你不能对着 PPT 念稿——要准备好"如果客户问那个刁钻问题我该怎么回答"。最厉害的财富管家不是"永远不会让客户亏钱"的人，而是"能让亏钱的客户依然信任你"的人。
