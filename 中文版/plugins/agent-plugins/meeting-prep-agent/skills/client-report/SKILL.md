# 客户报告

description: 生成专业的面向客户的表现报告，包含投资组合回报、配置细分和市场评论。适用于季度或年度分发。触发词包括"客户报告"、"表现报告"、"[客户]季度报告"、"生成报告"或"客户对账单"。

## 工作流程

### 步骤1：报告参数

- **客户姓名**和家庭
- **报告期间**：季度、年初至今、年度、自定义范围
- **账户**：所有账户或特定账户
- **基准**：标普500、60/40组合、匹配IPS的自定义基准
- **公司品牌**：标志、颜色、免责声明

### 步骤2：表现摘要

**家庭摘要：**

| 证券 | 资产类别 | 股数 | 价格 | 价值 | 占投资组合% | 本季度回报 |

| | | | | | | |

### 步骤5：市场评论

根据客户的成熟程度定制的简要市场摘要：
- 本季度市场发生了什么（2-3句话）
- 如何影响投资组合
- 展望和定位逻辑（2-3句话）
- 零售客户不用行话；对成熟投资者可以更技术化

### 步骤6：活动摘要

- 期间执行的交易
- 供款和提取
- 收到的股息和利息
- 收取的费用
- 再平衡活动

### 步骤7：规划笔记

- 迈向财务目标的进展（退休、教育等）
- 任何计划变更或建议
- 即将到来的行动项
- 下次审查日期

### 步骤8：输出

- 带公司品牌的PDF报告（8-12页）
- Word文档用于定制
- Excel数据附录（可选）

**报告结构：**
1. 封面（客户姓名、期间、公司标志）
2. 执行摘要（1页）
3. 表现摘要（1-2页）
4. 配置概览带图表（1页）
5. 持仓明细（1-2页）
6. 市场评论（1页）
7. 活动摘要（1页）
8. 规划笔记（1页）
9. 披露和免责声明（1页）

## 重要说明

- 除非客户/合规要求毛收益，否则表现必须按费后计算
- 始终包含适当的免责声明和披露（过往表现、风险因素）
- 报告应在客户间保持一致 — 使用标准模板
- 根据客户匹配细节水平 — 有些人想要每个持仓，其他人想要一页摘要
- 基准选择很重要 — 使用IPS中的基准，不是看起来最好的那个
- 新模板首次分发前需合规审查

## Appendix: 金融背景知识

### 什么是客户报告（Client Report）？

客户报告是财富管理公司向客户定期发送的正式文件，总结：
- 投资组合表现（vs. 基准）
- 资产配置和持仓
- 市场评论和展望
- 账户活动（交易、供款、费用）
- 财务规划进展

### 关键术语解释

**基准（Benchmark）**：用于衡量投资组合表现的市场指数或组合。常见的有：
- **标普500（S&P 500）**：美国大盘股基准
- **60/40组合**：60%股票/40%债券的传统平衡组合
- **定制基准**：匹配客户IPS目标配置的组合

**费后回报（Net of Fees）**：扣除所有管理费、托管费后的净回报。通常比毛回报低1-2%。

**毛回报（Gross Return）**：未扣除费用的投资回报。法规要求通常展示费后回报以保护投资者。

**持仓明细（Holdings Detail）**：投资组合中每个证券的详细信息，包括股数、成本基础、未实现损益等。

**资产配置（Asset Allocation）**：将投资资金分配到不同资产类别（股票、债券、现金、另类等）的策略。研究表明，资产配置决定了投资组合回报的90%以上。

**执行摘要（Executive Summary）**：报告开篇的一页精华，包含关键数字和要点，让忙碌的客户快速了解状况。

**免责声明（Disclaimers）**：法律要求的披露，包括：
- "过往表现不代表未来结果"
- "未注册证券风险"
- "费用披露"
- "税务声明"

**合规审查（Compliance Review）**：在发送给客户前，合规部门必须审查报告，确保：
- 没有误导性声明
- 适当的风险披露
- 符合监管要求（如SEC、FINRA）
- 品牌和措辞符合公司政策

**托管（Custody）**：持有客户资产并提供账户服务的银行或机构（如Schwab、Fidelity）。

**经纪账户（Brokerage Account）**：可自由买卖证券的应税投资账户。

**传统IRA（Traditional IRA）**：个人退休账户，供款时可抵税，提取时缴税。

**Roth IRA**：个人退休账户，供款时缴税，合格提取时免税。

**529计划**：州政府赞助的教育储蓄计划，用于支付高等教育费用，通常提供州税优惠。

**教育储蓄账户（Education Savings Account）**：专门用于教育费用的储蓄账户。
