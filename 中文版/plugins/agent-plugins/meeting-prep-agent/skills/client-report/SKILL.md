# 客户报告

description: 生成专业的面向客户的表现报告，包含投资组合回报、配置细分和市场评论。适用于季度或年度分发。触发词包括"客户报告"、"表现报告"、"[客户]季度报告"、"生成报告"或"客户对账单"。

## 工作流程

### 步骤1：报告参数

- **客户姓名**和家庭
- **报告期间**：季度、年初至今、年度、自定义范围
- **账户**：所有账户或特定账户
- **基准**：标普500、60/40组合、匹配IPS的自定义基准
- **公司品牌**：标志、颜色、免责声明

### 步骤2：表现摘要

**家庭摘要：**

|----------|-----------|--------|-------|-------|---------------|-----------|
| | | | | | | |

### 步骤5：市场评论

根据客户的成熟程度定制的简要市场摘要：
- 本季度市场发生了什么（2-3句话）
- 如何影响投资组合
- 展望和定位逻辑（2-3句话）
- 零售客户不用行话；对成熟投资者可以更技术化

### 步骤6：活动摘要

- 期间执行的交易
- 供款和提取
- 收到的股息和利息
- 收取的费用
- 再平衡活动

### 步骤7：规划笔记

- 迈向财务目标的进展（退休、教育等）
- 任何计划变更或建议
- 即将到来的行动项
- 下次审查日期

### 步骤8：输出

- 带公司品牌的PDF报告（8-12页）
- Word文档用于定制
- Excel数据附录（可选）

**报告结构：**
1. 封面（客户姓名、期间、公司标志）
2. 执行摘要（1页）
3. 表现摘要（1-2页）
4. 配置概览带图表（1页）
5. 持仓明细（1-2页）
6. 市场评论（1页）
7. 活动摘要（1页）
8. 规划笔记（1页）
9. 披露和免责声明（1页）

## 重要说明

- 除非客户/合规要求毛收益，否则表现必须按费后计算
- 始终包含适当的免责声明和披露（过往表现、风险因素）
- 报告应在客户间保持一致 — 使用标准模板
- 根据客户匹配细节水平 — 有些人想要每个持仓，其他人想要一页摘要
- 基准选择很重要 — 使用IPS中的基准，不是看起来最好的那个
- 新模板首次分发前需合规审查

## Appendix: 金融背景知识

这份文件是"客户报告（Client Report）"技能的详细说明。在财富管理中，Client Report 是每季度/半年度发给客户的"成绩单"——告诉客户他的投资组合表现如何、为什么、接下来怎么做。

---

### 1. 报告的三段结构

| 段 | 内容 |
|----|------|
| 总结 | 一句话：赚/亏了多少、为什么 |
| 归因 | 贡献最大/拖后腿最多的品种 |
| 展望 | 接下来关注什么、建议做什么调整 |

---

### 2. "说人话"原则

| 不要说 | 要说 |
|--------|------|
| "因子暴露偏差导致跟踪误差扩大" | "你的组合里某类股票比例偏高，导致波动超过了你的目标" |
| "alpha 显著为正" | "你的经理选股能力帮你多赚了 3%" |

---

### 给小白的一句话

> Client Report 就是财富管家的"成绩单"——客户不关心你的模型多复杂，只关心"赚了没、亏了为什么、接下来怎么办"。最好的报告是"让客户看完三分钟就知道该怎么办"的报告。记住：你是写给一个可能连 P/E 是什么都不知道的人看的——用"讲故事"代替"堆术语"。
