# 模型更新

description: 使用新数据更新财务模型 — 季度财报、管理层指引、宏观变化或修订假设。调整预测、重新计算估值并标记重大变化。在财报发布后、指引更新后或需要刷新假设时使用。触发词包括"update model"、"plug earnings"、"refresh estimates"、"update numbers for [company]"、"new guidance"或"revise estimates"。

## 工作流程

### 步骤1：确定变更内容

确定更新触发因素：
- **财报发布**：插入新的季度实际数据
- **指引变更**：公司更新了未来展望
- **预测修订**：分析师根据新数据更改假设
- **宏观更新**：利率、汇率、大宗商品价格变化
- **事件驱动**：并购、重组、新产品、管理层变动

### 步骤2：插入新数据

#### 财报发布后
使用报告的实际数据更新模型：

| 项目 | 先前预测 | 实际值 | 差异 | 备注 |
|------|----------|--------|------|------|
| 收入 | | | | |
| 毛利率 | | | | |
| 运营费用 | | | | |
| EBITDA | | | | |
| EPS | | | | |
| [关键指标1] | | | | |
| [关键指标2] | | | | |

**部门明细**（如适用）：
- 更新每个部门的收入和利润率
- 注意任何部门组合变化

**资产负债表/现金流更新**：
- 现金和债务余额
- 股份数量（回购、稀释）
- 实际资本支出 vs. 预测
- 营运资金变化

### 步骤3：修订未来预测

根据新数据调整未来预测：

| | 旧财年预测 | 新财年预测 | 变化 | 旧下一财年 | 新下一财年 | 变化 |
|---|------------|------------|------|------------|------------|------|
| 收入 | | | | | | |
| EBITDA | | | | | | |
| EPS | | | | | | |

**关键假设变化：**
- 您正在更改哪些假设以及原因？
- 收入增长率：旧 → 新（原因）
- 利润率假设：旧 → 新（原因）
- 任何新项目（重组费用、一次性收益等）

### 步骤4：估值影响

使用更新的预测重新计算估值：

| 估值方法 | 先前值 | 更新值 | 变化 |
|----------|--------|--------|------|
| DCF公允价值 | | | |
| 市盈率（NTM EPS × 目标倍数） | | | |
| EV/EBITDA（NTM EBITDA × 目标倍数） | | | |
| **目标价格** | | | |

### 步骤5：总结与行动

**预测变化总结：**
- 一段说明：什么变了、为什么变以及对股票意味着什么
- 这是改变投资主题的事件还是噪音？

**评级/目标价格：**
- 维持还是改变评级？
- 新目标价格（如变更）及方法论
- 当前价格的上涨/下跌空间

### 步骤6：输出

- 更新后的Excel模型（如果用户提供现有模型）
- 预测变化摘要（markdown或Word）
- 更新后的目标价格推导

## 重要注意事项

- 在向前预测之前，始终将您的预测与公司报告的数字进行核对
- 注意任何非经常性项目以及您的预测是GAAP还是调整后
- 跟踪您的预测修订历史 — 这显示了您的分析进展
- 如果本季度数据噪音较大，在预测变化中分离信号与噪音
- 更新后检查一致预期 — 您修订后的预测与市场相比如何？
- 股份数量很重要 — 股权激励、可转债或回购造成的稀释会对EPS产生重大影响

## 金融术语和知识解释

### 1. 财务模型更新详解

**定义**：财务模型更新是根据新信息调整预测和假设的过程，确保模型反映最新的业务状况和市场环境。

**触发因素**：

| 触发事件 | 更新内容 | 频率 |
|----------|----------|------|
| 季度财报发布 | 录入实际数据，调整预测 | 每季度 |
| 管理层指引变更 | 更新收入、盈利预期 | 不定期 |
| 宏观经济变化 | 调整利率、汇率假设 | 定期/不定期 |
| 公司事件（并购、重组） | 更新资本结构、业务范围 | 事件驱动 |
| 竞争格局变化 | 调整市场份额假设 | 定期评估 |

**更新流程**：
1. **数据录入**：将财报实际数据输入模型
2. **差异分析**：比较实际值与预测值，分析原因
3. **假设调整**：根据新信息更新关键假设
4. **预测更新**：重新计算未来财务预测
5. **估值重算**：更新目标价格
6. **敏感性测试**：测试关键变量变化的影响

**最佳实践**：
- 保持审计追踪，记录每个变更的原因
- 使用版本控制，便于回溯历史版本
- 定期验证模型逻辑和计算准确性

---

### 2. 实际值与预测值分析

**定义**：实际值是公司报告的真实财务业绩，预测值是分析师对未来业绩的预期。

**差异分析框架**：

| 差异类型 | 定义 | 计算公式 |
|----------|------|----------|
| **绝对差异** | 实际值与预测值的差额 | 实际值 - 预测值 |
| **百分比差异** | 差异占预测值的比例 | (实际值 - 预测值) / 预测值 × 100% |
| **惊喜幅度** | 超出/低于预期的程度 | 同上 |

**惊喜类型**：
- **正面惊喜 (Positive Surprise)**：实际值 > 预测值 → 通常推高股价
- **负面惊喜 (Negative Surprise)**：实际值 < 预测值 → 通常打压股价
- **符合预期 (In-line)**：实际值 ≈ 预测值 → 影响中性

**示例分析**：
```
Q3 2024 EPS：
- 实际值：$2.15
- 一致预期：$2.00
- 绝对差异：+$0.15
- 惊喜幅度：+7.5%（正面惊喜）
```

### 3. NTM（下12个月）详解

**定义**：NTM（Next Twelve Months）指未来12个月，是计算估值倍数时常用的时间框架。

**计算方法**：
```
NTM EPS = 当前季度EPS + 未来3个季度预期EPS
或
NTM EPS = 过去12个月EPS（滚动12个月）
```

**常用NTM指标**：
- **NTM EPS**：未来12个月预期每股收益
- **NTM EBITDA**：未来12个月预期EBITDA
- **NTM Revenue**：未来12个月预期收入
- **NTM P/E**：基于NTM EPS的市盈率

**与LTM的对比**：
| 指标 | LTM（过去12个月） | NTM（未来12个月） |
|------|-------------------|-------------------|
| 数据性质 | 历史数据 | 预测数据 |
| 反映内容 | 过去表现 | 未来预期 |
| 估值应用 | 参考 | 主要依据 |

**应用场景**：
- 估值倍数计算（如NTM P/E、NTM EV/EBITDA）
- 业绩目标设定
- 同比比较基准

---

### 4. 非经常性项目深度解析

**定义**：非经常性项目是不寻常或不频繁发生的收入或费用，不属于公司核心业务。

**常见类型**：

| 类别 | 具体项目 | 影响 |
|------|----------|------|
| **费用类** | 重组费用、资产减值、关停成本 | 减少当期利润 |
| **收益类** | 资产出售收益、一次性补贴 | 增加当期利润 |
| **特殊项目** | 自然灾害损失、法律诉讼赔偿 | 可正可负 |

**处理方法**：
- **GAAP口径**：包含所有项目
- **调整后口径**：剔除非经常性项目
- **分析师常用**：关注调整后收益，更能反映核心业务

**示例**：
```
GAAP净利润：$5亿（包含$1亿重组费用）
调整后净利润：$6亿（剔除重组费用）
调整后EPS：$2.40 vs GAAP EPS：$2.00
```

**注意事项**：
- 确认项目是否真的"非经常性"
- 关注管理层是否滥用调整后指标
- 比较时使用相同口径

---

### 5. GAAP与调整后收益

**定义**：
- **GAAP（公认会计原则）**：标准会计准则，确保财务报告的一致性和可比性
- **调整后收益**：公司管理层或分析师剔除特殊项目后的收益指标

**对比分析**：

| 方面 | GAAP收益 | 调整后收益 |
|------|----------|------------|
| **计算依据** | 严格遵循会计准则 | 管理层自主调整 |
| **可比性** | 跨公司可比 | 可比性较差 |
| **透明度** | 高 | 需详细披露调整项 |
| **用途** | 合规报告 | 经营业绩分析 |

**常见调整项**：
- 股票期权费用
- 重组和整合费用
- 资产减值
- 一次性法律费用
- 并购相关费用

**监管要求**：
- 公司必须同时披露GAAP和调整后指标
- 调整后指标需详细说明调整项目
- 禁止选择性披露

---

### 6. 一致预期机制详解

**定义**：一致预期是市场上所有跟踪某只股票的分析师对特定财务指标预测的平均值或中位数。

**数据来源**：
- **FactSet**：综合金融数据平台
- **Bloomberg**：实时终端服务
- **Thomson Reuters**：详细分析师数据

**计算方法**：
- **算术平均**：所有预测值简单平均
- **中位数**：排序后取中间值，更稳健
- **加权平均**：按机构影响力加权

**时间节点**：
- **财报前一致预期**：财报发布前的最后预期
- **财报后一致预期**：财报发布后更新的预期

**质量评估**：
- **预测分散度**：分析师预测的差异程度
- **更新频率**：预期调整的频繁程度
- **历史准确性**：过往预测与实际的偏差

---

### 7. 稀释效应分析

**定义**：稀释是指由于发行新股或其他证券导致每股收益(EPS)下降的现象。

**稀释原因**：

| 原因 | 机制 | 影响 |
|------|------|------|
| **股权激励** | 员工行使期权或限制性股票 | 增加流通股数 |
| **可转债转股** | 债券持有人转换为股票 | 增加流通股数和债务减少 |
| **新股发行** | 公司增发股票融资 | 显著增加流通股数 |
| **认股权证** | 持有人行使认股权 | 增加流通股数 |

**稀释计算**：
```
基本EPS = 净利润 / 基本流通股数
稀释EPS = 净利润 / (基本流通股数 + 潜在稀释股数)
```

**示例**：
```
净利润：$10亿
基本流通股：4亿股 → 基本EPS：$2.50
潜在稀释股：0.5亿股 → 稀释EPS：$2.22（稀释11.2%）
```

**反稀释措施**：
- 股份回购：减少流通股数
- 限制性股票单位(RSU)替代期权
- 回购库藏股用于股权激励

---

### 8. 股份回购详解

**定义**：股份回购是公司用现金回购自己流通股票的行为。

**回购方式**：
- **公开市场回购**：在二级市场购买
- **要约回购**：向股东发出固定价格收购
- **大宗交易回购**：私下协商购买

**回购目的**：
1. **提升EPS**：减少流通股数，提高每股收益
2. **回报股东**：替代现金分红
3. **股价支撑**：在股价低迷时提供支撑
4. **资本结构优化**：提高财务杠杆

**会计处理**：
- 回购股票计入"库藏股"（股东权益减少）
- 库藏股可用于股权激励或注销

**示例**：
```
回购前：10亿股 × $50 = $500亿市值
回购：用$50亿回购1亿股
回购后：9亿股 × $50 = $450亿市值
EPS提升：净利润不变，股数减少10% → EPS增加约11%
```

**监管限制**：
- 回购窗口期限制（财报前）
- 回购比例限制
- 内幕交易规则

---

### 9. 营运资金管理

**定义**：营运资金 = 流动资产 - 流动负债，反映公司日常运营所需的资金。

**组成部分**：

| 流动资产 | 流动负债 |
|----------|----------|
| 现金及等价物 | 短期借款 |
| 应收账款 | 应付账款 |
| 存货 | 应付职工薪酬 |
| 预付费用 | 应交税费 |

**营运资金周期**：
```
现金 → 存货 → 应收账款 → 现金
周期天数 = 存货周转天数 + 应收账款周转天数 - 应付账款周转天数
```

**分析要点**：
- **营运资金为正**：公司有足够资金维持运营
- **营运资金为负**：公司依赖供应商融资（可能是高效运营的标志）
- **变化趋势**：增加可能表明资金占用增加

**行业差异**：
- **零售行业**：存货和应收账款周期较长
- **科技行业**：应收账款周期较短
- **公用事业**：稳定的营运资金需求

### 10. DCF估值深度解析

**定义**：DCF（Discounted Cash Flow）即折现现金流模型，通过将未来现金流折现到当前价值来计算公司内在价值。

**核心公式**：
```
企业价值 = Σ(FCF_t / (1+WACC)^t) + TV / (1+WACC)^n

其中：
- FCF = 自由现金流 = 营业现金流 - 资本支出
- WACC = 加权平均资本成本
- TV = 终值（永续增长法或退出倍数法）
- n = 预测期年数
```

**计算步骤**：
1. **预测自由现金流**：通常预测5-10年
2. **计算WACC**：考虑股权成本和债务成本
3. **计算终值**：
   - 永续增长法：TV = FCF_n × (1+g) / (WACC - g)
   - 退出倍数法：TV = EBITDA_n × 退出倍数
4. **折现求和**：将所有现金流和终值折现到当前

**关键假设**：
- **收入增长率**：未来增长预期
- **利润率**：运营效率假设
- **资本支出**：投资需求
- **WACC**：折现率
- **终值增长率**：长期增长假设

**敏感性分析**：
| 变量 | 变动幅度 | 对估值的影响 |
|------|----------|--------------|
| WACC | +1% | 估值下降约10% |
| 长期增长率 | +0.5% | 估值上升约5-10% |
| 利润率 | +1个百分点 | 估值上升约5% |

**优缺点**：
| 优点 | 缺点 |
|------|------|
| 理论严谨 | 对假设高度敏感 |
| 考虑时间价值 | 预测不确定性大 |
| 适合长期投资 | 短期波动可能被忽略 |

## Appendix: 金融背景知识

这份文件是"模型更新（Model Update）"技能的详细说明。在股权研究中，每一次公司发布财报后，分析师必须更新 Excel 财务模型——录入实际数据、滚动预测、重新计算估值。

---

### 1. 模型更新的"五类触发事件"

| 触发事件 | 操作 |
|---------|------|
| 财报发布 | 把实际季度数据填入模型 |
| 指引变更 | 公司上调/下调预期——调模型参数 |
| 预测修订 | 分析师改了对未来的看法——更新假设 |
| 宏观更新 | 利率/汇率变了——调 DCF 折现率 |
| 事件驱动 | 并购/重组——修改模型结构 |

---

### 2. 财报后的"三步更新法"

```
第一步：录入实际值 → 把 Q3 的每个数字（收入、成本、EPS）写进去
第二步：标记差异 → 实际 vs 你的预测差了多少？标注百分比
第三步：滚动预测 → Q3 过了，新建 Q4 及以后年份的预测
```

---

### 给小白的一句话

> Model Update 就是给财务模型"换季"——每次公司发布新财报，你不能让模型停留在上个季度。你要把新数字填进去（录入实际值），搞清楚你之前预测的什么错了（标记差异），然后把预测窗口往后推一个季度（滚动预测）。最忌讳的是：只更新了数字，没搞清楚"为什么我的预测错了"——错误没有被理解，下次还会犯。
