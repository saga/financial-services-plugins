```

[当前日期]

目标价格：[旧→新（如变更），或维持XXX美元]

```

─────────────────────────────────────────────────

                实际值      预期值      差异

而[部门]增长X%（vs.预期的X%）。管理层强调[具体产品/举措]是关键增长驱动因素，

## 格式要求

### 1. 第1页要求
- 清晰的评级（维持跑赢大盘、上调至买入等）
- 突出显示更新后的目标价格
- 包含新旧预测的摘要表
- 3-4个段落长度的项目符号，使用■字符

### 2. 所有表格要求
- 底部有来源行
- 清晰的列标题
- 标题行有阴影

### 3. 所有图表要求
- 上方有"图X - [标题]"标题
- 下方有"来源：[来源]"行
- 专业样式

### 4. 年份符号
- 使用A表示实际值（Q3'24A）
- 使用E表示预测值（Q4'24E）

### 5. 写作风格
- 以数字开头（"收入增长15%至12亿美元"而非"强劲的收入增长"）
- 使用"vs."而非"versus"
- 直接简洁
- 专注于新内容

### 6. 超链接要求 ⭐⭐⭐
- 所有URL必须在Word中是可点击的超链接
- 蓝色带下划线的文本，Ctrl+点击可打开
- 显示文本有意义（不是原始URL）
- 每个来源引用应在适用处有可点击链接
- 没有纯文本URL - 始终格式化为超链接

## 金融术语和知识解释

### 1. 财报报告结构 (Earnings Report Structure)

标准的股票研究财报更新报告通常包含以下部分：
- **摘要页**：快速概览业绩、评级和目标价格
- **详细分析**：收入、盈利能力和关键指标的深入分析
- **投资主题评估**：业绩对投资论点的影响
- **估值与预测**：更新后的财务预测和目标价格计算
- **附录**：补充信息和支持材料

### 2. DCF模型 (Discounted Cash Flow Model)

DCF是一种估值方法，通过折现未来现金流来计算公司的内在价值。关键输入包括：
- 收入增长率
- 利润率
- 终值增长率
- 加权平均资本成本(WACC)

### 3. NTM市盈率 (Next Twelve Months P/E)

NTM市盈率是基于未来12个月预期盈利的市盈率，计算公式：当前股价 / 未来12个月预期EPS。

### 4. EV/EBITDA倍数 (Enterprise Value/EBITDA Multiple)

EV/EBITDA是企业价值与EBITDA的比率，用于比较不同资本结构的公司。

### 5. 基点 (Basis Points)

基点是金融中常用的单位，1个基点等于0.01%。例如，利率上升25个基点意味着上升0.25%。

### 6. 净收入留存率 (Net Revenue Retention)

净收入留存率衡量现有客户收入的增长情况，是SaaS公司的关键指标。计算公式：(当前客户收入 + 客户扩展收入 - 客户流失收入) / 先前客户收入。

### 7. WACC (Weighted Average Cost of Capital)

WACC是公司的加权平均资本成本，用于DCF模型中的折现率。它考虑了股权成本和债务成本。

### 8. 可比公司分析 (Comparable Companies Analysis)

可比公司分析是通过与同行业其他公司比较来评估目标公司价值的方法，常用指标包括市盈率、EV/EBITDA等。

### 9. GAAP vs 调整后收益 (GAAP vs Adjusted Earnings)

- **GAAP收益**：按照公认会计原则计算的收益
- **调整后收益**：剔除一次性项目后的收益，更能反映核心业务表现

### 10. 隐含上涨空间 (Implied Upside)

隐含上涨空间是当前股价与目标价格之间的百分比差异，计算公式：(目标价格 - 当前价格) / 当前价格。
