# 公式参考

**重要提示：** 除非用户另有指定，否则使用本参考文件中概述的公式。

---

## 核心链接

```
资产负债表：        资产 = 负债 + 权益
净利润：           IS净利润 → CF运营（起点）
现金流量：         Δ现金 = CFO + CFI + CFF
现金核对：         期末现金（CF）= 现金（BS资产）
现金月度/年度：     期末现金（月度）= 期末现金（年度）
留存收益：          期初留存收益 + 净利润 - 股息 = 期末留存收益
权益募集：           Δ普通股/APIC（BS）= 权益发行（CFF）
第0年权益：         募集的权益（第0年）= 期初权益（第1年）
```

## 毛利润计算

**重要提示：** 毛利润必须使用净收入计算，而非总收入。

```
净收入 - 销货成本 = 毛利润
```

| 术语 | 定义 |
|------|------|
| 总收入 | 任何扣除前的总收入 |
| 净收入 | 总收入 - 退货 - 让步 - 折扣 |
| 销货成本 | 可归因于已售商品/服务生产的直接成本 |
| 毛利润 | 净收入 - 销货成本 |

**注意：** 始终使用净收入（也称为"净销售额"或简单称为"收入"）作为盈利能力计算的起点。总收入会高估真实的顶线表现。

## 利润率公式

```
毛利率%      = 毛利润 / 净收入
EBITDA              = EBIT + D&A （或 = 毛利润 - 运营费用）
EBITDA利润率%     = EBITDA / 净收入
EBIT利润率%       = EBIT / 净收入
净利润率% = 净利润 / 净收入
```

## 信用指标公式

```
总债务            = 流动债务部分 + 长期债务
净债务              = 总债务 - 现金
总债务/EBITDA   = 总债务 / EBITDA（来自IS）
净债务/EBITDA     = 净债务 / EBITDA（来自IS）
利息保障倍数     = EBITDA / 利息费用（来自IS）
净利息费用%债务    = 净利息费用 / 长期债务
债务/总资本      = 总债务 / （总债务 + 总权益）
债务/权益         = 总债务 / 总权益
流动比率         = 总流动资产 / 总流动负债
速动比率           = （总流动资产 - 存货）/ 总流动负债
```

## 预测公式（净收入百分比法）

```
销货成本（预测） = 净收入 × 销货成本%假设
S&M（预测）             = 净收入 × S&M%假设
G&A（预测）             = 净收入 × G&A%假设
R&D（预测）             = 净收入 × R&D%假设
SBC（预测）             = 净收入 × SBC%假设
```

## 营运资本公式

```
应收账款
  期初应收账款
  + 收入（来自IS）
  - 现金收款（插入值）
  = 期末应收账款
  DSO = (AR / 收入) × 365

存货
  期初存货
  + 采购（插入值）
  - COGS（来自IS）
  = 期末存货
  DIO = (存货 / COGS) × 365

应付账款
  期初应付账款
  + 采购（来自存货计算）
  - 现金付款（插入值）
  = 期末应付账款
  DPO = (AP / COGS) × 365

营运资本净额 = AR + 存货 - AP
ΔWC = 当前NWC - 期初NWC
```

## D&A明细表公式

```
期初PP&E（总额）
+ CapEx
= 期末PP&E（总额）

期初累计折旧
+ 折旧费用
= 期末累计折旧

PP&E净额 = 总额PP&E - 累计折旧
```

## 债务明细表公式

```
期初债务余额
+ 新借款
- 偿还
= 期末债务余额

利息费用 = 平均债务余额 × 利率
  （为避免循环使用期初余额，或在启用循环引用时迭代）
```

## 留存收益公式

```
期初留存收益
+ 净利润（来自IS）
+ 股权激励（SBC）（来自IS）
- 股息
= 期末留存收益
```

## NOL（经营净亏损）明细表公式

```
NOL结转明细表

期初NOL余额（第1年/成立期 = 0）
+ 产生的NOL（如果EBT < 0，则为ABS(EBT)，否则为0）
- 使用的NOL（受限于应税收入和使用上限）
= 期末NOL余额

期初余额规则

对于新业务或第一个建模期间：
  期初NOL余额 = 0
  NOL只能通过已实现亏损（EBT < 0）增加
  NOL不能无中生有或假设

NOL使用计算

税前收入（EBT）
  如果EBT > 0：
    可用NOL = 期初NOL余额
    使用上限 = EBT × 80% （2017年后联邦限制）
    已用NOL = MIN(可用NOL, 使用上限)
    应税收入 = EBT - 已用NOL
  如果EBT ≤ 0：
    已用NOL = 0
    应税收入 = 0
    产生的NOL = ABS(EBT)

含NOL的税务计算

应付税款 = MAX(0, 应税收入 × 税率)
  （税款不能为负；亏损产生NOL资产）

NOL的递延税项资产（DTA）

DTA - NOL结转 = 期末NOL余额 × 税率
ΔDTA = 当前DTA - 期初DTA
  （DTA增加 = 现金流量上的非现金收益）
  （DTA减少 = 现金流量上的非现金费用）
```

## 资产负债表结构

```
资产
  现金（来自CF期末现金）
  应收账款（来自WC）
  存货（来自WC）
  流动资产合计
  
  PP&E净额（来自DA）
  递延税项资产 - NOL（来自NOL明细表）
  非流动资产合计
  总资产

负债
  应付账款（来自WC）
  流动债务部分（来自债务）
  流动负债合计
  
  长期债务（来自债务）
  负债合计

权益
  普通股
  留存收益（来自RE明细表）
  权益合计

检查：资产 - 负债 - 权益 = 0
```

## 现金流量表结构

```
经营现金（CFO）
  净利润（链接：IS）
  + D&A（链接：DA明细表）
  + 股权激励（SBC）（链接：IS或假设）
  - ΔDTA（递延税项资产）（链接：NOL明细表；DTA增加=现金使用）
  - ΔAR（链接：WC）
  - Δ存货（链接：WC）
  + ΔAP（链接：WC）
  = CFO

投资现金（CFI）
  - CapEx（链接：DA明细表）
  = CFI

融资现金（CFF）
  + 债务发行（链接：债务）
  - 债务偿还（链接：债务）
  + 权益发行（链接：BS普通股/APIC）
  - 股息（链接：RE明细表）
  = CFF

现金净变化 = CFO + CFI + CFF
期初现金
+ 现金净变化
= 期末现金（链接至：BS现金）
```

## 检查公式

```
BS平衡检查：       资产 - 负债 - 权益  （必须 = 0）
现金核对：           BS现金 - CF期末现金       （必须 = 0）
RE结转：            期初RE + NI + SBC - 股息 - BS RE  （必须 = 0）
DTA核对：            NOL明细表DTA - BS DTA      （必须 = 0）
权益募集核对：     Δ普通股/APIC（BS）- 权益发行（CFF）  （必须 = 0）
第0年权益核对：    募集的权益（第0年）- 期初权益（第1年）  （必须 = 0）
现金月度vs年度：    期末现金（月度）- 期末现金（年度）  （必须 = 0）
NOL使用上限：       已用NOL ≤ EBT × 80%       （2017年后必须为真）
NOL非负：          期末NOL余额 ≥ 0         （必须为真）
NOL期初余额：       第1年期初NOL = 0     （新企业必须为真）
NOL积累：          NOL仅在EBT < 0时增加（亏损产生NOL）
```

## 金融术语和知识详细解释

### 1. 会计等式（Accounting Equation）

**定义**：会计等式是财务会计的基础：
```
资产 = 负债 + 权益
```

**含义**：
- 企业拥有的一切（资产）等于欠一切（负债）+ 所有者权益
- 每笔交易都必须保持等式平衡
- 资产负债表必须永远平衡

**为什么重要**：
- 所有财务模型的基础约束
- 检测建模错误的第一个检查
- 审计师验证报表的核心原则

---

### 2. 间接法现金流量表（Indirect Method Cash Flow）

**定义**：间接法从净利润开始，通过调整非现金项目和营运资本变化来计算经营现金流。

**调整流程**：
```
净利润
+ 非现金费用（D&A, SBC等）
- 非现金收益
+/- 营运资本变化
= 经营现金流
```

**为什么重要**：
- 展示净利润与实际现金产生的差异
- 识别应计制与现金制的差异
- 揭示收益质量

---

### 3. 营运资本周转天数（Working Capital Days）

**DSO（应收账款周转天数）**：
```
DSO = (应收账款 / 收入) × 365
```
含义：收回客户欠款平均需要多少天

**DIO（存货周转天数）**：
```
DIO = (存货 / COGS) × 365
```
含义：存货在销售前平均持有多少天

**DPO（应付账款周转天数）**：
```
DPO = (应付账款 / COGS) × 365
```
含义：支付供应商账单平均需要多少天

**现金转换周期（CCC）**：
```
CCC = DSO + DIO - DPO
```
含义：从支付原材料到收到货款的现金周期

---

### 4. 递延税项资产/负债（DTA/DTL）

**递延税项资产（DTA）**：
- 由于 временные разницы 导致未来可抵扣的税款
- 例如：NOL结转、坏账准备
- 未来产生税盾

**递延税项负债（DTL）**：
- 由于 временные разницы 导致未来应付的税款
- 例如：折旧差异、预收收入
- 未来需要支付

**在现金流量表上**：
- DTA增加 = 现金使用（非现金收益）
- DTA减少 = 现金来源（非现金费用）

---

### 5. 循环引用（ Circular Reference）

**定义**：当公式引用自身直接或间接时创建循环引用。

**财务模型中的典型循环**：
```
利息费用 → 净利润 → 现金 → 债务余额 → 利息费用
```

**解决方案**：
1. 使用平均债务余额计算利息（避免循环）
2. 启用迭代计算（Excel选项）
3. 添加断路器切换（关闭迭代直到稳态）

**为什么棘手**：
- 可能导致计算不收敛
- 影响计算速度
- 需要仔细模型构建

## Appendix: 金融背景知识

这份文件是"三表模型公式参考（Formulas Reference）"文档的详细说明。在投行和股权投资中，三表模型（损益表、资产负债表、现金流量表）是财务分析的"地基"——所有估值模型都建立在三表模型之上。

---

### 1. 三表模型的核心公式

**资产负债平衡公式：**
```
资产 = 负债 + 股东权益
```
这是会计的"万有引力定律"——永远成立，永远不能打破。如果你算出来不相等，一定是某个地方算错了。

**现金流转公式：**
```
期末现金 = 期初现金 + 经营性现金流 + 投资性现金流 + 融资性现金流
```

---

### 2. 三张表之间的关键"勾稽关系"

| 从 | 到 | 链接内容 |
|----|-----|---------|
| 损益表 → 资产负债表 | 净利润 → 留存收益（权益增加） |
| 损益表 → 资产负债表 | 折旧 → 固定资产减少 |
| 资产负债表 → 现金流 | 应收账款变动 → 经营性现金流调整 |
| 资产负债表 → 现金流 | 存货变动 → 经营性现金流调整 |
| 资产负债表 → 损益表 | 债务余额 → 利息费用 |

---

### 3. 为什么公式如此重要？

**类比**：三表模型就像一个"水力系统"——损益表是入水口（收入），资产负债表是水库（水位），现金流量表是水管（实际流动）。三条管道必须无缝连接——任何一处堵塞（公式错误），水就流不动了。

---

### 给小白的一句话

> Formulas Reference 就是三表模型的"公式小抄"——它列出了损益表、资产负债表、现金流量表之间所有的数学链接。最核心的三条：1）资产 = 负债 + 权益（永远成立）；2）净利润从损益表流入现金流量表的"经营性现金流"起始行；3）期末现金 = 期初现金 + 三个现金流（经营性+投资性+融资性）。记住：三表模型不是三个独立的 Excel 表——它们是"一个水系统的三个阀门"。
