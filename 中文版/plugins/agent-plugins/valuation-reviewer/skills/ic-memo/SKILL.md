# 投资委员会备忘录

description: 为私募股权交易审批起草结构化的投资委员会备忘录。将尽职调查结果、财务分析和交易条款综合成专业的IC就绪文档。在准备投资委员会材料、撰写交易报告或创建正式建议时使用。触发词："写IC备忘录"、"投资委员会备忘录"、"交易报告"、"准备IC材料"或"建议备忘录"。

## 工作流程

### 步骤1：收集输入

从用户收集（或从会话中的先前分析获取）：

- 公司概况和业务描述
- 行业/市场背景
- 历史财务数据（3-5年）
- 管理层评估
- 交易条款（价格、结构、融资）
- 尽职调查结果（商业、财务、法律、运营）
- 价值创造计划 / 100天计划
- 回报分析（基准、上行、下行情景）

### 步骤2：起草备忘录结构

标准IC备忘录格式：

**I. 执行摘要**（1页）
- 公司描述、交易理由、关键条款
- 建议和主要回报指标
- 前三大风险及缓解措施

**II. 公司概述**（1-2页）
- 业务描述、产品/服务
- 客户群和市场拓展策略
- 竞争定位
- 管理团队

**III. 行业与市场**（1页）
- 市场规模和增长
- 竞争格局
- 长期趋势/有利因素
- 监管环境

**IV. 财务分析**（2-3页）
- 历史表现（收入、EBITDA、利润率、现金流）
- 盈利质量调整
- 营运资金分析
- 资本支出需求

**V. 投资主题**（1页）
- 为什么这是一个有吸引力的投资（3-5个支柱）
- 价值创造杠杆（有机增长、利润率扩张、并购、倍数扩张）
- 100天优先事项

**VI. 交易条款与结构**（1页）
- 企业价值和隐含倍数
- 资金来源与用途
- 资本结构/杠杆率
- 关键法律条款

**VII. 回报分析**（1页）
- 基准、上行和下行情景
- 各情景下的IRR和MOIC
- 驱动回报的关键假设
- 敏感性分析

**VIII. 风险因素**（1页）
- 按严重程度和可能性排序的关键风险
- 每项风险的缓解措施
- 交易终止风险（如有）

**IX. 建议**
- 明确建议：继续/放弃/有条件继续
- 关键条件或下一步行动

### 步骤3：输出格式

- 默认：具有专业格式的Word文档（.docx）
- 备选：用于快速审查的Markdown格式
- 包含财务和回报表格，而非纯文本

## 重要提示

- IC备忘录应实事求是、平衡客观——诚实地呈现牛市和熊市情景
- 不要低估风险。IC成员无论如何都会发现它们；可信度很重要
- 如果用户提供公司标准备忘录模板，请使用该模板
- 财务表格应相互关联——检查EBITDA桥、资金来源与用途平衡、回报计算是否一致
- 对于缺失的输入，应询问用户而非对交易条款或回报做出假设

## 📚 金融术语和知识解释

### 1. 投资委员会（Investment Committee, IC）

**定义**：由私募股权基金或投资机构的高级成员组成的决策机构，负责审批重大投资决策。

**职责**：
- 审查投资提案
- 评估风险回报
- 批准或拒绝投资
- 监控投资组合表现

**组成**：
- 基金合伙人/董事总经理
- 投资总监
- 行业专家
- 独立顾问

### 2. IRR（内部收益率）

**定义**：Internal Rate of Return，使投资净现值为零的折现率。

**计算公式**：
```
NPV = Σ [CFt / (1 + IRR)^t] = 0
```

**解释**：
- IRR是衡量投资回报率的核心指标
- 表示投资的年化收益率
- 用于比较不同投资机会

**示例**：
- 投资100万元，5年后收回200万元
- IRR ≈ 14.87%

### 3. MOIC（投资倍数）

**定义**：Multiple on Invested Capital，衡量投资回报倍数。

**计算公式**：
```
MOIC = 退出时股权价值 / 投入股权金额
```

**示例**：
- 投入5000万元
- 退出时获得2亿元
- MOIC = 4.0x

**与IRR的区别**：
- MOIC不考虑时间价值
- IRR考虑资金的时间成本

### 4. 交易结构（Deal Structure）

**定义**：投资交易的法律和财务框架。

**组成部分**：
- **估值**：企业价值、股权价值
- **融资**：股权、债务比例
- **条款**：保护性条款、治理结构
- **对价**：现金、股票、混合支付

**常见结构**：
- **杠杆收购（LBO）**：使用债务融资
- **少数股权投资**：获得部分股权
- **成长资本**：为扩张提供资金

### 5. 尽职调查（Due Diligence）

**定义**：对目标公司进行全面调查和分析的过程。

**类型**：
| 类型 | 内容 |
|------|------|
| **财务DD** | 财务报表、现金流、会计政策 |
| **法律DD** | 合同、合规、诉讼风险 |
| **商业DD** | 市场定位、竞争优势 |
| **运营DD** | 业务流程、供应链 |
| **管理DD** | 团队能力、企业文化 |

**目的**：识别风险、验证假设、支持估值

### 6. 价值创造计划（Value Creation Plan）

**定义**：私募股权投资者制定的提升投资组合公司价值的战略计划。

**常见杠杆**：
- **有机增长**：扩大市场份额、新产品开发
- **利润率提升**：成本优化、运营效率
- **并购整合**：战略收购、行业整合
- **倍数扩张**：改善治理、提升估值

**100天计划**：投资完成后前100天的关键行动清单

### 7. 敏感性分析（Sensitivity Analysis）

**定义**：分析关键假设变化对投资回报的影响。

**常见变量**：
- 进入/退出倍数
- EBITDA增长率
- 杠杆水平
- 持有期限

**呈现方式**：
- 双向敏感性矩阵
- 龙卷风图
- 情景分析（牛市/基准/熊市）

### 8. 企业价值（Enterprise Value, EV）

**定义**：公司的总价值，包括股权和债务。

**计算公式**：
```
EV = 股权价值 + 债务 - 现金及等价物
```

**用途**：
- 计算估值倍数（EV/EBITDA、EV/Revenue）
- 评估并购交易价值
- 比较不同资本结构的公司

### 9. 资金来源与用途（Sources & Uses）

**定义**：描述交易资金来源和使用方式的表格。

**资金来源**：
- 股权出资
- 高级债务
- 夹层债务
- 卖方融资

**资金用途**：
- 收购对价
- 交易费用
- 再投资
- 债务偿还

### 10. 风险缓解（Risk Mitigation）

**定义**：识别和降低投资风险的策略。

**常见风险类型**：
- **市场风险**：行业变化、竞争加剧
- **运营风险**：执行失败、管理问题
- **财务风险**：杠杆过高、现金流不足
- **监管风险**：政策变化、合规问题

**缓解策略**：
- 结构化保护条款
- 管理层激励机制
- 定期监控和报告
- 应急计划
