# 投资组合监控

description: 跟踪和分析投资组合公司的业绩与计划对比。接收月度/季度财务包（Excel、PDF），提取关键绩效指标，标记与预算的偏差，并生成汇总仪表板。在审查投资组合公司财务状况、准备董事会材料或监控契约合规性时使用。触发词："审查投资组合公司"、"月度财务"、"[公司]表现如何"、"契约检查"或"投资组合更新"。

## 工作流程

### 步骤1：接收财务包

- 接受用户的投资组合公司财务包（Excel工作簿、PDF或CSV）
- 提取关键财务数据：收入、EBITDA、现金余额、未偿债务、资本支出、营运资金
- 确定报告期并与上期和预算/计划进行比较

### 步骤2：KPI提取与差异分析

需要跟踪的关键指标（根据公司行业调整）：

**财务KPI：**
- 收入 vs. 预算（金额和百分比）
- EBITDA和EBITDA利润率 vs. 预算
- 现金余额和净债务
- 杠杆比率（净债务 / LTM EBITDA）
- 利息覆盖率
- 资本支出 vs. 预算
- 自由现金流

**运营KPI**（询问用户或从数据推断）：
- 客户数量 / 每位客户收入
- 员工人数 / 每位员工收入
- 待处理订单 / 销售管道
- 客户流失率 / 留存率

### 步骤3：标记与汇总

- **绿色**：在计划的5%范围内
- **黄色**：低于计划5-15% — 标记讨论
- **红色**：低于计划>15%或存在契约违约风险 — 立即关注

输出简明摘要：
1. 一段执行摘要（"公司X的业绩[领先/落后/符合]计划..."）
2. 包含实际值vs预算vs上期的KPI表格
3. 红色/黄色警告及背景说明
4. 契约合规状态（如适用）
5. 向管理层提出的问题

### 步骤4：趋势分析

如果提供多个期间：
- 绘制关键指标随时间变化的图表（收入、EBITDA、现金）
- 识别趋势 — 加速、减速或稳定
- 与承保案例进行比较

## 重要提示

- 如果未提供预算/计划，务必询问用户
- 不要假设行业特定的KPI — 询问对该公司重要的指标
- 如果不知道契约水平，请向用户询问信贷协议条款
- 输出应为董事会就绪 — 简洁、真实、无冗余内容

## 📚 金融术语和知识解释

### 1. 投资组合管理（Portfolio Management）

**定义**：对投资组合进行监控、评估和优化的过程。

**目标**：
- 最大化风险调整后回报
- 实现投资目标
- 监控投资表现
- 管理风险敞口

**关键活动**：
- 定期业绩审查
- 风险评估
- 再平衡决策
- 退出策略规划

### 2. KPI（关键绩效指标）

**定义**：Key Performance Indicator，用于衡量业务目标实现程度的量化指标。

**选择原则**：
- 与战略目标对齐
- 可衡量、可跟踪
- 具有时效性
- 责任明确

**财务KPI示例**：
- 收入增长率
- EBITDA利润率
- 自由现金流
- ROIC（投资回报率）

**运营KPI示例**：
- 客户获取成本（CAC）
- 客户生命周期价值（LTV）
- 员工生产率
- 库存周转率

### 3. 预算差异分析（Variance Analysis）

**定义**：比较实际业绩与预算目标差异的过程。

**计算方法**：
```
差异 = 实际值 - 预算值
差异百分比 = (实际值 - 预算值) / 预算值 × 100%
```

**差异类型**：
- **有利差异**：实际优于预算
- **不利差异**：实际劣于预算

**分析步骤**：
1. 识别重大差异
2. 分析原因
3. 确定责任方
4. 提出改进措施

### 4. 杠杆比率（Leverage Ratio）

**定义**：衡量公司债务负担的指标。

**常见比率**：
| 比率 | 计算公式 | 含义 |
|------|----------|------|
| **净债务/EBITDA** | 净债务 / LTM EBITDA | 债务覆盖能力 |
| **利息覆盖率** | EBITDA / 利息费用 | 利息支付能力 |
| **债务/股权** | 总债务 / 股东权益 | 资本结构 |

**解读**：
- 高杠杆意味着高财务风险
- 行业标准差异较大
- 影响融资成本和条款

### 5. 契约合规（Covenant Compliance）

**定义**：遵守贷款协议中规定的财务和运营限制条件。

**常见契约类型**：
- **财务契约**：维持特定财务比率
- **运营契约**：限制某些经营活动
- **报告契约**：定期提供财务报告

**违反后果**：
- 技术性违约
- 利息增加
- 提前还款要求
- 贷款加速到期

### 6. 自由现金流（Free Cash Flow）

**定义**：公司在支付运营费用和资本支出后剩余的现金。

**计算公式**：
```
FCF = 经营现金流 - 资本支出
```

**用途**：
- 评估公司产生现金的能力
- 支付股息和回购股票
- 偿还债务
- 投资增长机会

**重要性**：
- 真正的"可自由支配"现金
- 驱动企业价值的核心指标

### 7. LTM（过去12个月）

**定义**：Last Twelve Months，指最近连续12个月的财务数据。

**计算方法**：
- 对于季度数据：最近4个季度之和
- 对于月度数据：最近12个月之和

**用途**：
- 提供最新的财务状况视图
- 平滑季节性波动
- 用于估值倍数计算

### 8. 承保案例（Underwriting Case）

**定义**：投资决策时基于的预期财务模型和假设。

**组成部分**：
- 收入预测
- EBITDA预测
- 现金流预测
- 退出假设
- 回报预测

**监控目的**：
- 跟踪实际vs预期表现
- 识别偏差原因
- 更新投资 thesis

### 9. 董事会材料（Board Materials）

**定义**：为董事会会议准备的报告和演示文稿。

**内容**：
- 财务业绩报告
- 投资组合更新
- 重大事项说明
- 战略建议
- 风险报告

**标准**：
- 专业、简洁
- 数据准确、完整
- 突出关键要点
- 便于决策

### 10. 预警系统（Early Warning System）

**定义**：识别潜在问题的监控机制。

**颜色编码**：
- **绿色**：正常运营
- **黄色**：需要关注
- **红色**：需要立即行动

**触发条件**：
- 财务指标偏离计划
- 市场环境变化
- 管理层变动
- 竞争威胁
