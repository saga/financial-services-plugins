# 回报分析

description: 为私募股权交易评估构建快速IRR/MOIC敏感性表格。模拟不同进入倍数、杠杆、退出倍数、增长和持有期限情景下的回报。在评估交易规模、压力测试假设或准备IC回报展示时使用。触发词："回报分析"、"IRR敏感性"、"MOIC表格"、"回报是多少"、"模拟回报"或"粗略估算"。

## 工作流程

### 步骤1：收集交易输入

询问（或从先前分析提取）：

**进入阶段：**
- 进入EBITDA（LTM或NTM）
- 进入倍数（EV / EBITDA）
- 企业价值
- 交易完成时的净债务
- 股权出资额
- 交易费用和开支

**融资：**
- 高级债务（EBITDA倍数、利率、摊销）
- 次级债务/夹层融资（如有）
- 进入时的总杠杆率（EBITDA倍数）
- 股权出资

**运营假设：**
- 收入增长率（年度）
- EBITDA利润率轨迹
- 资本支出占收入百分比
- 营运资金变化
- 债务偿还计划

**退出阶段：**
- 持有期限（年）
- 退出倍数（EV / EBITDA）
- 退出EBITDA（根据增长假设计算）

## 关键公式

- **MOIC** = 退出股权价值 / 投入股权
- **IRR** = 求解r：投入股权 × (1 + r)^n = 退出股权价值（调整中期现金流）
- **回报归因**：
  - 增长：(退出EBITDA - 进入EBITDA) × 退出倍数 / 股权
  - 倍数：(退出倍数 - 进入倍数) × 进入EBITDA / 股权
  - 杠杆：持有期内债务偿还 / 股权

## 重要提示

- 在适用情况下，始终显示扣除费用/carry前后的回报
- 管理层滚动投资和共同投资会改变股权出资额 — 如相关请询问
- 股息资本重组或中期分配显著影响IRR — 如计划请包含
- 不要忘记交易成本（通常为EV的2-4%）— 它们会降低首日股权价值
- 税务考虑（资产vs股票交易、338(h)(10)选择）会显著影响税后回报

## 📚 金融术语和知识解释

### 1. 回报分析（Returns Analysis）

**定义**：评估投资预期回报的过程，包括IRR、MOIC等关键指标。

**目的**：
- 评估投资吸引力
- 比较不同投资机会
- 支持投资决策
- 向投资者报告

**关键维度**：
- 时间维度（持有期）
- 风险维度（敏感性分析）
- 情景维度（牛市/熊市）

### 2. 敏感性分析（Sensitivity Analysis）

**定义**：分析关键假设变化对投资回报的影响。

**方法**：
- **单向分析**：一次改变一个变量
- **双向分析**：同时改变两个变量（矩阵形式）
- **情景分析**：设定不同情景组合

**关键变量**：
- 进入/退出倍数
- EBITDA增长率
- 杠杆水平
- 持有期限
- 交易费用

**输出**：
- 敏感性矩阵
- 龙卷风图（显示影响程度排序）
- 盈亏平衡分析

### 3. 情景分析（Scenario Analysis）

**定义**：评估不同未来情景下投资表现的方法。

**常见情景**：
| 情景 | 描述 | 关键假设 |
|------|------|----------|
| **基准情景** | 最可能的结果 | 市场平均假设 |
| **牛市情景** | 乐观结果 | 高于预期增长 |
| **熊市情景** | 悲观结果 | 低于预期增长 |

**构建步骤**：
1. 定义情景边界
2. 设置各情景假设
3. 计算每种情景的回报
4. 评估概率和影响

### 4. 回报归因（Return Attribution）

**定义**：分解驱动投资回报的各个因素。

**组成部分**：
- **增长贡献**：EBITDA增长带来的回报
- **倍数贡献**：估值倍数变化带来的回报
- **杠杆贡献**：债务偿还带来的回报

**计算公式**：
```
总回报 = 增长回报 + 倍数回报 + 杠杆回报 - 费用
```

**用途**：
- 理解回报来源
- 识别关键价值驱动因素
- 优化投资策略

### 5. LBO模型（杠杆收购模型）

**定义**：分析杠杆收购交易的财务模型。

**关键组成部分**：
- **资金来源与用途**：融资结构
- **运营预测**：收入、EBITDA
- **债务偿还**：摊销计划
- **退出分析**：回报计算

**假设**：
- 进入/退出倍数
- 增长率和利润率
- 杠杆水平和利率
- 持有期限

**输出**：
- IRR和MOIC
- 敏感性分析
- 情景比较

### 6. 交易费用（Transaction Costs）

**定义**：完成交易产生的各项费用。

**组成部分**：
- 投资银行费用
- 法律费用
- 会计/审计费用
- 尽职调查费用
- 顾问费用

**范围**：通常为交易价值的2-4%

**处理方式**：
- 从首日股权价值中扣除
- 影响MOIC和IRR计算
- 在资金用途表中明确列出

### 7. 股息资本重组（Dividend Recapitalization）

**定义**：通过增加债务向股东派发特别股息的行为。

**目的**：
- 返还部分投资本金
- 提高IRR（提前回收资金）
- 优化资本结构

**影响**：
- 增加杠杆水平
- 减少未来利息覆盖
- 可能触发契约违约

### 8. 338(h)(10)选举

**定义**：美国税法中的一项选举，允许资产收购被视为股票收购。

**好处**：
- 获得资产税基提升
- 增加折旧摊销抵扣
- 降低有效税率

**考虑因素**：
- 税务成本与收益权衡
- 卖方税务影响
- 交易结构复杂性

### 9. Carry（附带权益）

**定义**：私募股权基金经理获得的利润分成。

**结构**：
- 通常为利润的20%
- 有门槛收益率（hurdle rate）
- 可能有追回条款（clawback）

**影响**：
- 激励基金经理创造价值
- 影响投资者净回报
- 需在回报分析中考虑

### 10. 共同投资（Co-Investment）

**定义**：投资者在主基金之外直接投资于特定交易。

**优势**：
- 增加投资敞口
- 降低费用负担
- 增强与GP的关系

**考虑因素**：
- 投资能力
- 尽职调查资源
- 利益冲突管理

## Appendix: 金融背景知识

这份文件是"回报分析（Returns Analysis）"技能的详细说明。在 PE 基金中，每评估一笔潜在交易，都必须做回报分析——模拟在不同情景下这笔投资能赚多少钱。

---

### 1. PE 回报的两大指标

| 指标 | 英文 | 含义 |
|------|------|------|
| IRR | Internal Rate of Return | 内部收益率——考虑时间价值的年化回报率 |
| MOIC | Multiple on Invested Capital | 投资回报倍数——绝对金额倍数（投入 1 元收回 X 元） |

**例子**：
- 投入 100，5 年后收回 200，MOIC = 2.0x
- 投入 100，5 年后收回 200，IRR ≈ 15%（因为是年化的）

---

### 2. 回报分析的"四驾马车"

| 变量 | 含义 | 对回报的影响 |
|------|------|------------|
| 买入倍数 | 买的时候付了多少倍 EV/EBITDA | 越低越好 |
| 杠杆 | 借了多少债 | 适量借钱可以放大回报（LBO 效应） |
| 退出倍数 | 卖的时候能卖多少倍 | 越高越好 |
| 增长 | 公司在持有期间的 EBITDA 增长 | 越快越好 |

---

### 3. 敏感性分析的"五要素"

| 要素 | 取值范围 |
|------|---------|
| 持有期 | 3 年、5 年、7 年 |
| 买入倍数 | 8x、10x、12x |
| 退出倍数 | 比买入高 1x、相同、低 1x |
| EBITDA 增长率 | +5%、+10%、+15% |
| 杠杆 | 4x、5x、6x EBITDA |

---

### 给小白的一句话

> Returns Analysis 就是 PE 交易的"天气预报"——你无法精确知道 5 年后公司值多少钱，但你可以模拟"今天是晴天（乐观）、阴天（基准）、暴雨（悲观）"三种情景下 IRR 分别是多少。别只看"基准情景 IRR 25% 很漂亮"——更关键的是问：如果悲观情景发生（行业崩盘/管理层离职），IRR 会不会变成 -30%？PE 赚的是"控制风险"的钱，不是"赌博"的钱。
