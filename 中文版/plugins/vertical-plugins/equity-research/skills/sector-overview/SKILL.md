# 行业综述

description: 创建涵盖市场动态、竞争格局、核心参与者及专题趋势的全面行业与板块全景报告。适用于客户请求、行业覆盖初始研究、专题研究报告或内部知识库构建。触发词包括“行业综述”、“行业报告”、“市场格局”、“板块分析”、“行业深度调研”或“专题研究”。

## 工作流

### 第一步：界定范围

- **行业 / 细分板块**：涉及哪个行业？定义的边界有多窄？
- **目的**：客户报告、内部研究、路演推介材料、投资策略生成
- **深度**：高层级概览（5-10页）或深度调研（20-30页）
- **角度**：中立的全景图 vs. 专题性论点（例如：“AI 基础设施建设”）
- **研究范围**：仅限上市公司，还是包含非上市公司？

### 第二步：市场概况

**市场规模与增长**
- 可触达市场总量 (TAM) 及其数据来源
- 历史增长率（5年复合年均增长率，CAGR）
- 预测增长率及核心假设
- 市场细分（按产品、地理区域、终端市场、客户类型）

**行业结构**
- 碎片化 vs. 集中化 —— 前五大市场份额占比
- 价值链图谱 —— 价值在何处累积？
- 商业模式类型（订阅制、交易型、授权许可、服务类）
- 准入门槛（资本、监管、技术、网络效应）

**核心趋势与驱动因素**
- 长期利好推力（3-5 个主要趋势）
- 行业逆风与风险
- 技术颠覆路径
- 监管动态
- 并购 (M&A) 活动及行业集中化趋势

### 第三步：竞争格局

**公司概况**（针对前 5-10 名参与者）：

| 公司 | 营收 | 增长率 | EBITDA 利润率 | 市场份额 | 核心差异化优势 |
|---------|---------|--------|--------------|-------------|-------------------|
| | | | | | |

对每家公司进行简要描述：
- 业务描述（2-3 句）
- 战略定位与护城河
- 近期动态（财报、并购、产品发布）
- 估值速览（市盈率 P/E、企业价值/息税折旧摊销前利润 EV/EBITDA、企业价值/营收 EV/Revenue）

**竞争动态**
- 公司间如何竞争？（价格、产品、服务、渠道）
- 谁在获得/失去份额？原因何在？
- 来自新进入者或相邻领域参与者的颠覆风险

### 第四步：估值背景

- 板块交易倍数（当前及历史区间）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 该板块与大盘相比表现几何？

### 第五步：投资启示

- 最具风险收益比的机会在哪里？
- 可以通过该板块表达哪些专题性头寸？
- 板块内的核心争论点（看多与看空观点）
- 可能改变板块逻辑的事态催化剂

### 第六步：产出

- Word 文档或 PowerPoint 简报，包含：
  - 市场概况与规模测算
  - 竞争格局地图
  - 公司对比表
  - 估值摘要
  - 核心图表：市场增长、份额趋势、估值历史曲线
- 包含详细公司数据的 Excel 附录

## 注意事项

- 注明所有市场规模数据的来源 —— 引用研究机构或测算方法
- 区分“TAM 噱头”与实际可触达的真实市场
- 行业综述时效性强 —— 必须标注日期，并对可能过时的数据进行预警
- 图表至关重要 —— 市场增长瀑布图、竞争定位矩阵、估值散点图
- 如针对特定客户，需根据其具体情况定制“结论意义”（如并购标的识别、竞争定位、市场准入）

### 1. TAM / SAM / SOM——"不是所有'大市场'你都能抢到"

在行业分析中，市场规模有三个层级——**TAM、SAM、SOM**，它们代表了"市场有多大"从最宏观到最微观的三个回答。

**三层级详解**：

**① TAM（Total Addressable Market，可触达市场总量）**：
- 定义：如果全世界所有潜在客户都买了你的产品，这个市场有多大
- 用途：展示行业的"天花板"——告诉投资者"这门生意最大能做多大"
- 例子：全球电动车市场的 TAM 约 $500B——假设全世界所有车都换成电动车，就是 $500B
- ⚠️ 陷阱：TAM 经常被"美化"——创业公司喜欢说"我们的 TAM 是 $500B！"但实际能抢到的可能只有 $5B

**② SAM（Serviceable Addressable Market，可服务市场）**：
- 定义：在地理、价格、产品技术上你实际能触达的市场
- 用途：展示"在 TAM 中，有多少我真正能碰到的"
- 例子：一家只在中国卖电动车的公司 → SAM = 中国市场（$150B），而不是全球（$500B）
- 更接近"真实的商业机会"

**③ SOM（Serviceable Obtainable Market，可获取市场）**：
- 定义：在 SAM 中，基于你的竞争力和资源，你实际能拿下的份额
- 用途：现实主义的市场规模——"别人也不傻，你能抢多少？"
- 例子：在中国电动车市场（SAM $150B）中，假设你的市场份额目标是 5% → SOM = $7.5B
- **这才是财务模型中最真实的"可持续收入"基线**

**打比方——TAM/SAM/SOM 就像"你家附近的餐饮市场"**：
- TAM = 整个城市所有人的晚餐消费 = $500M → "天花板"
- SAM = 你家 3 公里范围内的晚餐消费 = $50M → "你的配送范围"
- SOM = 在这个范围内，你每天能做多少外卖 = $1M → "你的产能上限"

**所以这意味着什么**：在行业分析中，看到一家公司声称"我们的 TAM 是 $500B"时要保持警惕——问一句："SAM 是多少？SOM 是多少？"只有 SOM 才是和收入模型直接相关的"真实市场"。

### 2. CAGR（复合年均增长率）——"为什么不是简单平均？"

**CAGR（Compound Annual Growth Rate，复合年均增长率）** 是衡量一个行业/公司在多年期间的"平均年增长率"的标准指标。它和"简单平均"的区别在于它考虑了 **"利滚利"** 的效果。

**计算公式**：
```
CAGR = (终值 / 初始值)^(1 / 年数) - 1
```

**CAGR vs 简单平均——一个具体例子**：

假设一个行业 3 年的规模变化：
- 第 1 年：$100M
- 第 2 年：$180M（增长 80%）
- 第 3 年：$108M（下降 40%）

**简单平均**：（80% + (-40%））/ 2 = **20%**
**CAGR**：（$108M / $100M）^（1/2） - 1 = **3.9%**

差距 20% vs 3.9% ——差了 5 倍！为什么差这么大？
- 因为简单平均忽略了"基数变化"——第二年赚到的钱，第三年同样也可以被亏掉
- CAGR 反映的是"如果每年都以相同的速率增长/衰退，最终结果会是什么"
- CAGR = 3.9% 意味着"实际上，过去 2 年的平均年增长率大约只有 3.9%"

**CAGR 的三个应用场景**：

| 场景 | 公式 | 含义 |
|------|------|------|
| **行业历史增长** | 计算过去 5 年的 CAGR | "过去 5 年行业平均每年扩大了 12%" |
| **行业未来预测** | 基于趋势判断未来 5 年 CAGR | "预计未来 5 年行业以 15% 的 CAGR 扩张" |
| **公司增长对比** | 对比多家公司过去 3 年的 CAGR | "A 公司 CAGR 25%，B 公司 CAGR 8%—谁是增长王一目了然" |

**打比方——CAGR 就像"你的投资收益率"**：
- 你投资 $100 → 第一年赚到 $180（+80%）→ 第二年亏到 $108（-40%）
- 你的"年均收益率"不是 20%——（80%-40%）/2——而是 3.9%
- 因为第二年亏的钱，是第一年赚的钱的一部分

**所以这意味着什么**：当看到"行业过去 5 年 CAGR 12%"时，它不是在说每年都增长 12%——而是说从 5 年前的规模到今天的规模，平均每年增长 12%（考虑了复利）。

### 3. 市场集中度——"寡头的土地，还是群狼的草原？"

**市场集中度**衡量的是行业中前几大公司的市场份额之和——它告诉你的，是"这个行业是寡头垄断还是高度分散"。

**两个核心指标**：

**CR（市场集中比，Concentration Ratio）**：
- CR3 = 前 3 名市场份额之和
- CR5 = 前 5 名市场份额之和
- 解读：
  - CR3 > 70% → 高度集中——寡头垄断（如手机芯片：高通 + 联发科 + 苹果 ≈ 85%）
  - CR5 < 30% → 高度分散——充分竞争（如中餐外卖：成千上万的小商家）

**HHI（赫芬达尔-赫希曼指数，Herfindahl-Hirschman Index）**：
- HHI = Σ（每家公司市场份额的平方）× 10,000
- 比 CR 更精细——CR 只看前几名，HHI 考虑了"每一家"公司的份额
- 解读：
  - HHI < 1,500 → 低集中度（分散竞争）
  - 1,500 - 2,500 → 中等集中度
  - HHI > 2,500 → 高集中度（寡头垄断）

**为什么市场集中度很重要**：
- 高集中度 → 前几大公司有"定价权"→ 利润率高 → 但法院可能介入反垄断
- 低集中度 → 没有一家有定价权 → 利润率低 → 但进入门槛可能也低
- 趋势变化 → 如果 CR5 从 35% 升到 55% → 行业在"集中化"→ 大公司在"吞并"小公司的市场份额

**打比方——CR 和 HHI 就像"篮球赛的得分分布"**：
- 如果一支球队的得分 CR3（前三名得分手）占总分的 70% → 说明高度依赖三位明星球员
- HHI 可以告诉你："整支 12 人球队的得分分布是否均匀"→ 不止看前三名

**所以这意味着什么**：在行业分析中，市场集中度告诉你两件事——"谁有定价权"和"小玩家还有没有机会"。

### 4. Secular Tailwinds vs Cyclical Headwinds——"长期顺风 vs 周期性逆风"

在行业分析中，必须区分两种"风"——一种是长期的结构性趋势（Secular），一种是短期的经济周期波动（Cyclical）。

**Secular Tailwinds（长期顺风 / 结构性利好）**：
- 本质：在未来 5-20 年持续推动行业增长的大趋势
- 特征：不可逆转、跨周期、技术/人口/社会趋势驱动
- 例子：
  - 老龄化社会 → 医疗健康行业（结构性利好）
  - 全球碳中和目标 → 新能源行业（结构性利好）
  - 数字化转型 → 云计算/SaaS 行业（结构性利好）
  - 人工智能 → AI 相关行业（结构性利好）
- **重要性**：Secular Tailwinds 让你在分析行业时不只看"未来 12 个月"——而是"未来 10 年这个行业一定会增长"。

**Cyclical Headwinds（周期性逆风 / 周期性不利因素）**：
- 本质：由于经济周期（升息、衰退、通胀）导致的短期行业压力
- 特征：可逆转、受宏观政策影响、通常持续 1-3 年
- 例子：
  - 美联储升息 → 高负债行业（如房地产、公用事业）承压
  - 经济衰退 → 可选消费行业（如旅游、奢侈品）需求下降
  - 原材料涨价 → 制造业利润率收缩
- **重要性**：Cyclical Headwinds 是"暂时的"——当周期反转时，行业会恢复。

**为什么区分两者至关重要**：
- 如果你把"周期性逆风"误判为"结构性衰退" → 你在行业低谷时错误地清仓 → 错过反弹
- 如果你把"结构性衰退"误判为"周期性逆风" → 你在行业长期衰退时坚持持有 → 亏惨了

**经典案例**：
- 2007 年认为"房地产只是周期性调整"（实际上是结构性崩塌）→ 在次贷危机中血本无归
- 2016 年认为"电商只有周期性红利"（实际上结构性改变零售）→ 错过了亚马逊的黄金十年

**打比方——Secular Tailwinds 和 Cyclical Headwinds 就像"天气 vs 气候"**：
- 天气（Cyclical）= 今天下雨 → 影响今天出门
- 气候（Secular）= 全球变暖 → 影响未来 20 年
- 你不能因为"今天下雨"就断言"未来 10 年都会下雨"——同样，不能因为"行业面临周期性压力"就断言"行业要完蛋了"

**所以这意味着什么**：在行业分析中，必须明确标注"什么是结构性的、什么是周期性的"——因为你的投资时间框架完全不同。

### 5. 行业分析中的三张核心图表——"一张好图胜过一千字"

**① 市场规模瀑布图（Market Growth Waterfall）**：
- 展示：5-10 年的市场规模变化，分解增长驱动因素（人口、价格、技术）
- 为什么重要：一眼看出"增长的关键驱动力是什么"

**② 竞争定位矩阵（Competitive Positioning Matrix / Magic Quadrant）**：
- 二维坐标系：X 轴 = "市场份额"或"市场存在感"，Y 轴 = "产品力"或"创新能力"
- 四个象限：领导者（Leader，右上）、挑战者（Challenger，左上）、远见者（Visionary，右下）、利基玩家（Niche，左下）
- 为什么重要：一眼看出"谁是值得关注的、谁在掉队"

**③ 估值散点图（Valuation Scatter Plot）**：
- X 轴 = 增长率（如 EPS 增长 CAGR），Y 轴 = 估值倍数（如 Forward P/E）
- 回归线 = "公允估值"
- 回归线上方的点 = "相对于其增长率偏高"→ 可能高估
- 回归线下方的点 = "相对于其增长率偏低"→ 可能低估
- 为什么重要：快速发现"行业内谁贵谁便宜"
