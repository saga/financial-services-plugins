# 行业综述

description: 创建涵盖市场动态、竞争格局、核心参与者及专题趋势的全面行业与板块全景报告。适用于客户请求、行业覆盖初始研究、专题研究报告或内部知识库构建。触发词包括“行业综述”、“行业报告”、“市场格局”、“板块分析”、“行业深度调研”或“专题研究”。

## 工作流

### 第一步：界定范围

- **行业 / 细分板块**：涉及哪个行业？定义的边界有多窄？
- **目的**：客户报告、内部研究、路演推介材料、投资策略生成
- **深度**：高层级概览（5-10页）或深度调研（20-30页）
- **角度**：中立的全景图 vs. 专题性论点（例如：“AI 基础设施建设”）
- **研究范围**：仅限上市公司，还是包含非上市公司？

### 第二步：市场概况

**市场规模与增长**
- 可触达市场总量 (TAM) 及其数据来源
- 历史增长率（5年复合年均增长率，CAGR）
- 预测增长率及核心假设
- 市场细分（按产品、地理区域、终端市场、客户类型）

**行业结构**
- 碎片化 vs. 集中化 —— 前五大市场份额占比
- 价值链图谱 —— 价值在何处累积？
- 商业模式类型（订阅制、交易型、授权许可、服务类）
- 准入门槛（资本、监管、技术、网络效应）

**核心趋势与驱动因素**
- 长期利好推力（3-5 个主要趋势）
- 行业逆风与风险
- 技术颠覆路径
- 监管动态
- 并购 (M&A) 活动及行业集中化趋势

### 第三步：竞争格局

**公司概况**（针对前 5-10 名参与者）：

| 公司 | 营收 | 增长率 | EBITDA 利润率 | 市场份额 | 核心差异化优势 |
|---------|---------|--------|--------------|-------------|-------------------|
| | | | | | |

对每家公司进行简要描述：
- 业务描述（2-3 句）
- 战略定位与护城河
- 近期动态（财报、并购、产品发布）
- 估值速览（市盈率 P/E、企业价值/息税折旧摊销前利润 EV/EBITDA、企业价值/营收 EV/Revenue）

**竞争动态**
- 公司间如何竞争？（价格、产品、服务、渠道）
- 谁在获得/失去份额？原因何在？
- 来自新进入者或相邻领域参与者的颠覆风险

### 第四步：估值背景

- 板块交易倍数（当前及历史区间）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 该板块与大盘相比表现几何？

### 第五步：投资启示

- 最具风险收益比的机会在哪里？
- 可以通过该板块表达哪些专题性头寸？
- 板块内的核心争论点（看多与看空观点）
- 可能改变板块逻辑的事态催化剂

### 第六步：产出

- Word 文档或 PowerPoint 简报，包含：
  - 市场概况与规模测算
  - 竞争格局地图
  - 公司对比表
  - 估值摘要
  - 核心图表：市场增长、份额趋势、估值历史曲线
- 包含详细公司数据的 Excel 附录

## 注意事项

- 注明所有市场规模数据的来源 —— 引用研究机构或测算方法
- 区分“TAM 噱头”与实际可触达的真实市场
- 行业综述时效性强 —— 必须标注日期，并对可能过时的数据进行预警
- 图表至关重要 —— 市场增长瀑布图、竞争定位矩阵、估值散点图
- 如针对特定客户，需根据其具体情况定制“结论意义”（如并购标的识别、竞争定位、市场准入）

> **💡 Appendix: 领域知识小贴士**
>
> 想要看懂这份金融报告，你需要掌握几个像“行业翻译官”一样的核心概念：
>
> 1. **TAM (Total Addressable Market) - 可触达市场总量**：你可以把它理解为一门生意的“天花板”。它告诉我们如果一家公司把市场上所有的潜在客户都拿下，这盘生意最大能做多大。
> 2. **CAGR (Compound Annual Growth Rate) - 复合年均增长率**：这是一种计算平均增速的方法。它不是简单地把几年的增速加起来除以几，而是考虑了“利滚利”的效果。它能帮你拉平每年的波动，看清一个行业的长期成长后劲。
> 3. **EBITDA - 息税折旧摊销前利润**：这是一个更纯粹的赚钱指标。它剥离了借钱利息、政府征税以及设备由于老化变旧（折旧摊销）带来的账面影响，让你一眼看出这家公司单靠“核心手艺”能赚多少现金。
> 4. **护城河 (Moat)**：这是巴菲特最爱的词。它代表了一家公司防止竞争对手抢走生意的能力，比如强大的品牌（可口可乐）、专利技术，或者极高的换用成本（比如你很难从微信迁到别的社交应用）。
> 5. **估值倍数 (Multiples)**：这就像是在超市买水果时看的“单价”。P/E（市盈率）是用股价除以利润，告诉你为了这 1 块钱的利润，你现在要花多少倍的钱去买。倍数越高，通常代表大家对它的未来期待越火热。
> 6. **长期利好 (Secular Tailwinds)**：想象一艘帆船顺风行驶。这是指那些在未来几年甚至几十年都会持续推动行业前进的大趋势，比如老龄化社会对医药行业的推动，或者数字化转型对软件行业的推动。
