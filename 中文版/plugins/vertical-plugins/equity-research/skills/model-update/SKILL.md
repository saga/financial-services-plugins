# 模型更新

description: 用新数据更新财务模型 — 季度财报、管理层指引、宏观变化或修订后的假设。调整预测、重新计算估值，并标记重大变化。在财报发布后、指引更新时或需要刷新假设时使用。触发词包括"更新模型"、"填入财报数据"、"刷新预测"、"更新[公司]数据"、"新指引"或"修订预测"。

## 工作流程

### 步骤1：识别变化内容

确定更新触发因素：
- **财报发布**：需要填入的新季度实际数据
- **指引变更**：公司更新了前瞻展望
- **预测修订**：分析师基于新数据改变假设
- **宏观更新**：利率、汇率、大宗商品价格变化
- **事件驱动**：并购、重组、新产品、管理层变更

### 步骤2：填入新数据

| EBITDA | | | | |
| EPS | | | | |

**分部详情**（如适用）：
- 更新每个分部的收入和利润率
- 注意任何分部结构变化

**资产负债表 / 现金流量表更新**：
- 现金和债务余额
- 股份数量（回购、稀释）
- 资本支出实际 vs. 预测
- 营运资本变化

基于新数据，调整未来预测：

- 你正在改变哪些假设？为什么？
- 收入增长率：旧 → 新（原因）
- 利润率假设：旧 → 新（原因）
- 任何新项目（重组费用、一次性收益等）

| 估值方法 | 先前值 | 更新值 | 变化 |
|---------|--------|---------|--------|
| DCF公允价值 | | | |

### 步骤5：汇总与行动

**预测变更摘要：**
- 一段话：变化内容、原因，以及对股票的意义
- 这是改变投资论点的事件还是噪音？

**评级 / 目价：**
- 维持还是改变评级？
- 新目价（如有变更）及方法论
- 相对当前价格的上涨/下跌空间

### 步骤6：输出

- 更新后的Excel模型（如果用户提供了现有模型）
- 预测变更摘要（markdown或Word）
- 更新的目价推导

## 重要说明

- 在向前预测之前，始终将你的预测与公司的报告数据对账
- 注意任何非经常性项目，以及你的预测是GAAP还是调整后数据
- 跟踪你的预测修订历史 — 它展示了你的分析进展
- 如果季度数据嘈杂，在预测变更中分离信号与噪音
- 更新后检查一致预期 — 你修订后的预测与华尔街一致预期相比如何？
- 股份数量很重要 — 股权激励、可转债或回购带来的稀释可能实质性影响EPS

## Appendix: 金融背景知识

这份文件是"模型更新（Model Update）"技能的详细说明。在卖方研究（Equity Research）中，财务模型是分析师的核心工具——它是一个 Excel 文件，包含了公司过去和未来的收入、成本、利润、现金流等所有关键数据。每当公司发布新财报或管理层给出新指引时，分析师就必须更新这个模型。

---

### 1. 什么是财务模型（Financial Model）？

**类比：**
想象你是一个天气预报员。你有一个"天气模型"，输入今天的温度、气压、湿度，模型就能预测未来 7 天的天气。但每天的实际天气会和预测有偏差，所以你必须每天根据实际天气更新模型，让未来的预测更准确。

财务模型就是金融分析师的"天气模型"——输入历史财务数据，预测未来 3-5 年的业绩。

---

### 2. 模型更新的 5 大触发因素

| 触发因素 | 说明 | 更新内容 |
|---------|------|---------|
| 财报发布 | 公司发布季度/年度财报 | 填入实际数据，调整预测 |
| 指引变更 | 管理层更新业绩指引 | 调整未来收入/利润预测 |
| 宏观变化 | 利率、汇率、大宗商品价格变化 | 调整假设参数 |
| 事件驱动 | 并购、重组、新产品 | 增加新业务板块 |
| 预测修订 | 分析师基于新数据改变假设 | 调整增长假设 |

---

### 3. 模型更新的"黄金流程"

#### 步骤1：识别变化
- 对比"原预测"和"实际值"的差异
- 判断差异是"一次性"还是"持续性"的

#### 步骤2：调整假设
- 如果实际收入超预期 10%，未来 3 年收入预测是否需要上调？
- 如果毛利率下降是因为原材料涨价，这是暂时还是长期的？

#### 步骤3：重新计算估值
- 更新 DCF 模型中的现金流预测
- 重新计算目标价

#### 步骤4：撰写报告
- 解释为什么调整预测
- 说明对投资建议的影响

---

### 4. 关键概念：Beat vs Miss

| 术语 | 含义 | 例子 |
|------|------|------|
| Beat | 实际业绩 > 市场预期 | 预期 EPS $1.00，实际 $1.10 |
| Miss | 实际业绩 < 市场预期 | 预期 EPS $1.00，实际 $0.90 |
| Inline | 实际业绩 ≈ 市场预期 | 预期 EPS $1.00，实际 $1.01 |

**重要**：Beat 不一定涨，Miss 不一定跌——关键看"指引"。

---

### 5. 真实案例：特斯拉 2023 Q3 模型更新

| 指标 | 原预测 | 实际值 | 差异 | 分析师调整 |
|------|--------|--------|------|-----------|
| 营收 | 245 亿 | 234 亿 | -4.5% Miss | 下调全年预测 |
| 毛利率 | 18.5% | 17.9% | -0.6pp | 下调未来毛利率假设 |
| EPS | $0.75 | $0.66 | -12% Miss | 下调目标价 $10 |
| 指引 | 全年 180 万辆 | 维持 | Inline | 维持长期看好 |

结果：股价盘后跌 4%，但第二天反弹——因为管理层表示"Cybertruck 即将量产"。

---

### 给小白的一句话

> 模型更新就是金融分析师的"错题本"——你预测公司今年能赚 100 亿，结果实际只赚了 90 亿。你必须搞清楚"为什么差 10 亿"（是一次性亏损还是主营业务不行了？），然后修正你未来的预测。如果每次都"预测错但不改"，那你的模型就是废纸一张，客户也不会再信你。
