# 催化剂日历

description: 构建并维护覆盖范围内的催化剂日历 — 财报日期、会议、产品发布、监管决定和宏观事件。帮助优先关注并在事件前定位仓位。触发词包括"催化剂日历"、"即将到来的事件"、"接下来有什么"、"财报日历"、"事件日历"或"催化剂追踪器"。

## 工作流程#

### 步骤1：定义覆盖范围#

- 要追踪的公司列表（股票代码或名称）
- 行业 / 板块焦点
- 包含宏观事件？（Fed会议、经济数据、监管截止日期）
- 时间范围（未来2周、1个月、1个季度）

### 步骤2：收集催化剂#

对每个公司，识别即将到来的事件：

**财报与财务事件**
- 季度财报日期和时间（盘前/盘后）
- 年度股东大会
- 投资者日 / 分析师日
- 资本市场上日
- 债务到期 / 再融资日期

**公司事件**
- 产品发布或公告
- FDA批准 / 监管决定
- 合同续签或到期
- 并购里程碑（截止日期、监管批准）
- 管理层变动
- 内部人交易窗口（锁定到期）

**行业事件**
- 主要会议（日期、哪些公司出席）
- 贸易展览和博览会
- 监管评论期或裁决
- 行业数据发布（月度销售、流量等）

**宏观事件**
- Fed会议（FOMC日期）
- 就业报告、CPI、GDP发布
- 央行决定（ECB、BOJ等）
- 有市场影响的地缘政治事件#

### 步骤3：日历视图#

| 日期 | 事件 | 公司/板块 | 类型 | 影响（高/中/低） | 我们的仓位 | 备注 |
|------|-------|---------------|------|----------------|----------|-------|
| | | | 财报/公司/行业/宏观 | | 做多/做空/中性 | |

### 步骤4：每周预览#

每周，生成前瞻摘要：

**本周关键事件：**
1. [日期]：[公司] Q[X]财报 — 一致预期[$X EPS]，我们的预期[$X]，关键焦点：[指标]
2. [日期]：[事件] — 为何对[股票]重要
3. [日期]：[宏观发布] — 预期和我们的观点

**下周预览：**
- 重要事件的提前通知

**仓位启示：**
- 可能移动特定仓位的事件
- 任何事件前的预定位建议

### 步骤5：输出#

- 带日历视图和可排序列的Excel工作簿
- 每周预览邮件/笔记（markdown）
- 可选：与Google日历集成

## 重要说明#

- 财报日期会变化 — 在接近日期时向公司IR页面和Bloomberg/FactSet验证
- 预公告风险：追踪有预公告历史的公司（正面或负面）
- 会议出席列表很有价值 — 哪些公司出席，哪些明显缺席？
- 一些催化剂是周期性的（月度行业数据）— 构建模板并自动填充
- 按影响级别颜色编码：红色 = 高影响，黄色 = 中等，绿色 = 常规

## Appendix: 金融背景知识#

### 什么是催化剂日历（Catalyst Calendar）？

催化剂日历是股权分析师用来追踪即将影响股价的关键事件的时间表。

### 关键术语解释

**FOMC（Federal Open Market Committee，联邦公开市场委员会）**：美联储决定利率政策的会议，通常每季度一次，导致市场波动。

**CPI（Consumer Price Index，消费者价格指数）**：衡量通胀的关键指标，每月发布，通常引起市场反应。

**GDP（Gross Domestic Product，国内生产总值）**：衡量经济总量的指标，季度发布。

**FDA（Food and Drug Administration，食品药品监督管理局）**：美国药品批准机构，对生物科技公司至关重要。

**锁定到期（Lockup Expiration）**：IPO后内部人股票可以出售的日期，通常导致股价压力因为供应增加。

**内部人交易窗口（Insider Trading Window）**：公司允许内部人（高管、董事）交易股票的时间段，通常在财报发布后开放。

**投资者日（Investor Day）**：公司向投资者展示战略、新产品、财务目标的专用活动。

**资本市场上日（Capital Markets Day）**：欧洲公司常用的投资者展示活动。

**ECB（European Central Bank，欧洲央行）**：欧元区央行，决定欧元利率政策。

**BOJ（Bank of Japan，日本央行）**：日本央行，以负利率和收益率曲线控制闻名。
