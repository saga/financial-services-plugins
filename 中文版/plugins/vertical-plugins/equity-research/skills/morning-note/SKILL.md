# 晨会笔记

description: 起草简洁的晨会笔记，总结隔夜动态、交易构想和覆盖股票的关键事件。专为早7点晨会设计 — 紧凑、有观点、可操作。触发词包括"晨会笔记"、"晨会"、"隔夜发生了什么"、"交易构想"、"晨会准备"或"每日笔记"。

## 工作流程

### 步骤1：隔夜动态

扫描覆盖范围内的相关事件：

**财报与指引**
- 是否有覆盖公司在隔夜或盘前发布财报？
- 财报意外（收入、EPS、关键指标超预期/不及预期）
- 指引变更（上调、下调、维持）

**新闻与事件**
- 并购公告或传闻
- 管理层变更
- 产品发布或监管决定
- 竞争对手的分析师上调/下调评级
- 影响行业的宏观数据或政策变化

**市场背景**
- 隔夜期货 / 盘前走势
- 行业ETF表现
- 相关大宗商品或货币走势
- 今日关键经济数据发布

### 步骤2：晨会笔记格式

保持紧凑 — 晨会笔记应在2分钟内可读完：

---

**[日期] 晨会笔记 — [分析师姓名]**
**[行业覆盖]**

**头条呼吁：[PM最需要听到的一件事]**
- 2-3句话说明关键动态及其重要性
- 股票影响：目价、评级重申/变更

**隔夜/盘前动态**
- [公司A]：财报/新闻的一句话摘要 + 我们的观点
- [公司B]：一句话摘要 + 我们的观点
- [行业/宏观]：全行业相关动态

**今日关键事件**
- [时间]：[公司]财报电话会议
- [时间]：经济数据发布（预期 vs. 我们的观点）
- [时间]：会议或投资者日

**交易构想**（如有）
- [做多/做空] [公司]：1-2句话投资论点 + 催化剂
- 风险：什么情况会使这个构想错误

---

### 步骤3：财报快速解读

如果覆盖公司发布了财报，提供快速反应：

| 指标 | 一致预期 | 实际值 | 超预期/不及预期 |
|------|-----------|--------|------------------|
| 收入 | | | |
| EPS | | | |
| [关键指标] | | | |
| 指引 | | | |

**我们的观点**：2-3句话 — 这对股票是利好还是利空？是否改变我们的投资论点？

**行动**：维持 / 上调 / 下调评级？调整目价？

### 步骤4：输出

- 用于邮件/Slack分发的Markdown文本
- 如需正式分发，可提供Word文档
- 控制在1页以内 — PM和交易员不会读更多

## 重要说明

- 要有观点 — 仅总结新闻而无观点的晨会笔记毫无用处
- 最重要的事情放在开头 — 不要埋没头条
- "无新闻"也是有效的晨会笔记 — 写明"隔夜无重大事件，维持仓位"
- 区分可操作事件（财报、并购）和噪音（次要分析师笔记、非事件）
- 为你的观点加上时间戳 — 如果你在早上6点撰写，注明盘前情况可能在开盘前变化
- 如果你错了，在次日晨会笔记中承认 — 可信度比每次都对更重要

- **期权与波动率**：隐含波动率 / Greeks → [期权与波动率.md](../../../../金融知识学习/期权与波动率.md)

**本文件专属**：
- **晨会（Morning Meeting）**：投行/基金每天早晨举行的简短会议，讨论隔夜市场、当日事件、交易机会。
- **盘前（Pre-market）**：美股正式开盘前的交易时段（4:00-9:30 AM ET）。
- **隔夜期货（Overnight Futures）**：在亚欧时段交易的美股期货，预示开盘方向。
- **行业ETF**：跟踪特定行业表现的 ETF，如 XLK（科技）、XLF（金融）、XLE（能源）。
- **耳语数字（Whisper Number）**：市场传闻的非正式业绩预期，通常比官方一致预期更激进。
