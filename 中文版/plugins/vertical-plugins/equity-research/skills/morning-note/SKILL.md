# 晨会笔记

description: 起草简洁的晨会笔记，总结隔夜动态、交易构想和覆盖股票的关键事件。专为早7点晨会设计 — 紧凑、有观点、可操作。触发词包括"晨会笔记"、"晨会"、"隔夜发生了什么"、"交易构想"、"晨会准备"或"每日笔记"。

## 工作流程

### 步骤1：隔夜动态

扫描覆盖范围内的相关事件：

**财报与指引**
- 是否有覆盖公司在隔夜或盘前发布财报？
- 财报意外（收入、EPS、关键指标超预期/不及预期）
- 指引变更（上调、下调、维持）

**新闻与事件**
- 并购公告或传闻
- 管理层变更
- 产品发布或监管决定
- 竞争对手的分析师上调/下调评级
- 影响行业的宏观数据或政策变化

**市场背景**
- 隔夜期货 / 盘前走势
- 行业ETF表现
- 相关大宗商品或货币走势
- 今日关键经济数据发布

### 步骤2：晨会笔记格式

保持紧凑 — 晨会笔记应在2分钟内可读完：

---

**[日期] 晨会笔记 — [分析师姓名]**
**[行业覆盖]**

**头条呼吁：[PM最需要听到的一件事]**
- 2-3句话说明关键动态及其重要性
- 股票影响：目价、评级重申/变更

**隔夜/盘前动态**
- [公司A]：财报/新闻的一句话摘要 + 我们的观点
- [公司B]：一句话摘要 + 我们的观点
- [行业/宏观]：全行业相关动态

**今日关键事件**
- [时间]：[公司]财报电话会议
- [时间]：经济数据发布（预期 vs. 我们的观点）
- [时间]：会议或投资者日

**交易构想**（如有）
- [做多/做空] [公司]：1-2句话投资论点 + 催化剂
- 风险：什么情况会使这个构想错误

---

### 步骤3：财报快速解读

如果覆盖公司发布了财报，提供快速反应：

| 指标 | 一致预期 | 实际值 | 超预期/不及预期 |
|------|-----------|--------|------------------|
| 收入 | | | |
| EPS | | | |
| [关键指标] | | | |
| 指引 | | | |

**我们的观点**：2-3句话 — 这对股票是利好还是利空？是否改变我们的投资论点？

**行动**：维持 / 上调 / 下调评级？调整目价？

### 步骤4：输出

- 用于邮件/Slack分发的Markdown文本
- 如需正式分发，可提供Word文档
- 控制在1页以内 — PM和交易员不会读更多

## 重要说明

- 要有观点 — 仅总结新闻而无观点的晨会笔记毫无用处
- 最重要的事情放在开头 — 不要埋没头条
- "无新闻"也是有效的晨会笔记 — 写明"隔夜无重大事件，维持仓位"
- 区分可操作事件（财报、并购）和噪音（次要分析师笔记、非事件）
- 为你的观点加上时间戳 — 如果你在早上6点撰写，注明盘前情况可能在开盘前变化
- 如果你错了，在次日晨会笔记中承认 — 可信度比每次都对更重要

## Appendix: 金融背景知识

这份文件是"晨会笔记（Morning Note）"技能的详细说明。在股权研究行业中，晨会是每一天最重要的例行活动——每天早上 7 点，所有分析师和销售聚集在一起，讨论隔夜发生了什么、今天要关注什么。

---

### 1. 什么是晨会笔记？

**类比：**
想象你是情报局的每日简报员。每天早上 6 点到办公室，汇总过去 12 小时内全球发生的重要事件：中东又打仗了？某国领导人被刺杀？油价暴涨 10%？

晨会笔记就是金融界的"情报日报"——总结隔夜市场动态，告诉基金经理和交易员"今天你要注意什么"。

---

### 2. 晨会笔记的"三大板块"

#### 板块 1：隔夜动态
- 覆盖股票的财报发布
- 重大新闻（并购、管理层变动、产品发布）
- 评级变动（分析师上调/下调）
- 宏观经济数据发布

#### 板块 2：交易构想
- 基于隔夜事件的投资机会
- 做多/做空建议
- 短期交易策略

#### 板块 3：今日关注
- 即将发布的数据
- 即将发生的事件
- 值得关注的股票

---

### 3. 晨会笔记的"铁血规律"

| 规则 | 说明 |
|------|------|
| 6:30 AM 前交稿 | 必须在 7 点晨会前送到基金经理桌上 |
| 不超过 1 页 | 基金经理每天收 20-30 份晨报，超 1 页不会有人看 |
| 每条不超过 3 句话 | 只说"发生了什么、为什么重要、怎么做" |
| 有观点 | 不要说"Apple 发了财报"，要说"Apple 财报超预期 5%，建议今天加仓" |
| 可操作 | 每条都必须回答"then what?" |

---

### 4. "一条好笔记" vs "一条烂笔记"

**烂笔记**：
> "特斯拉今天发布了 Q3 财报。收入 $230 亿。"

**好笔记**：
> "特斯拉 Q3 收入 $230 亿（vs consensus $240 亿，Miss 4%）。毛利率从 18% 跌至 16%，主要受降价影响。股价盘前跌 6%。我们维持卖出评级——如果开盘跌超 7%，建议平空获利了结；如果跌幅收窄至 3%，可加空。"

**差异：** 好笔记有数据（$230 亿）、有对比（vs consensus）、有分析（毛利率下降原因）、有操作建议（跌多就平空、跌少就加空）。

---

### 5. 真实案例：高盛晨会的 90 分钟

| 时间 | 活动 |
|------|------|
| 5:30 AM | 分析师到办公室，刷 Bloomberg、Reuters |
| 6:00 AM | 写完晨会笔记，发送给团队 |
| 6:30 AM | 内部讨论：今天的交易机会 |
| 7:00 AM | 正式晨会：全球市场回顾 + 重点股票讨论 + 交易构想 |
| 7:30 AM | 销售团队开始给客户打电话："今天早上我们有一个想法..." |

---

### 给小白的一句话

> 晨会笔记是股权研究的"日报头版"——它不需要像研究报告那样 30 页详尽，但要像新闻标题一样"一针见血"。基金经理每天早上要看 20-30 份这样的笔记，你的笔记必须在 10 秒内让他看完并做决定："今天要不要交易这只股票？"记住三个关键词：发生了什么、为什么重要、现在该怎么做。如果只能做到"发生了什么"而做不到"怎么做"，那你的晨报只是新闻汇总，不是研究。
