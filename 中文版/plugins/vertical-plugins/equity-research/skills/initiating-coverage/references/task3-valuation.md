3. **资产负债表数据（当期）**

### 第 2 步：构建 DCF（现金流折现）分析

#### A. 计算 WACC（加权平均资本成本）

**1. 确定无风险利率 (Risk-Free Rate)**
   - 使用 10 年期美国国债收益率（核实现行利率）
   - 示例：假设 2024 年底为 4.0-4.5%

**2. 确定股权资本成本 (CAPM 模型)**
   ```
   股权资本成本 = 无风险利率 + Beta × 股权风险溢价

   输入参数：
   - 无风险利率：[当前 10 年期国债，例如 4.2%]
   - Beta (贝塔系数)：[从 Bloomberg/FactSet 获取的公司 Beta，或可比公司平均值]
   - 股权风险溢价：5-6% (历史平均水平)

   示例计算：
   股权资本成本 = 4.2% + 1.3 × 5.5% = 11.35%
   ```

**3. 确定债务资本成本**
   ```
   债务资本成本 = 当前借款利率或债券的隐含收益率

   针对未上市私人公司：
   债务资本成本 = 无风险利率 + 信用利差 (基于信用评级)

   示例：
   税前债务成本 = 6.5%
   税后债务成本 = 6.5% × (1 - 25% 税率) = 4.875%
   ```

**4. 确定资本结构**
   ```
   使用市场价值（而非账面价值）：

   股权市值 (E) = 股价 × 流通股本
   债务市值 (D) = 债务总额 (如果债券未公开交易，可使用账面价值)
   总资本价值 (V) = E + D

   股权权重 = E / V
   债务权重 = D / V

   示例：
   E = $5,000M (90.9%)
   D = $500M (9.1%)
   V = $5,500M (100%)
   ```

**5. 计算 WACC**
   ```
   WACC = (E/V × 股权资本成本) + (D/V × 税后债务成本)

   示例：
   WACC = (90.9% × 11.35%) + (9.1% × 6.5% × (1 - 25%))
   WACC = 10.32% + 0.44% = 10.76%

   基准情形四舍五入保留为：10.8%
   ```

#### B. 计算终端价值 (Terminal Value)

**方法 1：永续增长法（首选）**
```
终端价值 = FCF(2029) × (1 + g) / (WACC - g)

其中：
- FCF(2029) = 模型中最后一年的无杠杆自由现金流
- g = 永续增长率 (通常介于 2.0-3.0% 之间)
  - 该数值不应超过长期的国内生产总值 (GDP) 增速
  - 基准情形下可设为 2.5%

示例：
FCF(2029) = $500M
g = 2.5%
WACC = 10.8%

终端价值 = $500M × (1.025) / (0.108 - 0.025)
终端价值 = $512.5M / 0.083 = $6,175M
```

**方法 2：退出乘数法（备选）**
```
终端价值 = EBITDA(2029) × 退出乘数

其中：
- 退出乘数 = 当前可比公司的估值乘数中位数 (例如 12-15x EBITDA)

示例：
EBITDA(2029) = $800M
退出乘数 = 13x

终端价值 = $800M × 13x = $10,400M
```

**提示：选择其中一种方法，或取两者的平均值。**

#### C. 将现金流折现为现值

```
预测期 FCF 的现值 = Σ [FCFt / (1 + WACC)^t] （t 介于 1 到 5）

示例：
年份    FCF      折现系数             FCF 现值
        ($M)     (Discount Factor)   ($M)
2025    $250     1/(1.108)^1 = 0.9026    $226
2026    $320     1/(1.108)^2 = 0.8147    $261
2027    $390     1/(1.108)^3 = 0.7353    $287
2028    $450     1/(1.108)^4 = 0.6636    $299
2029    $500     1/(1.108)^5 = 0.5988    $299
                              现值总计:  $1,372M

终端价值现值 = 终端价值 / (1 + WACC)^5
终端价值现值 = $6,175M / (1.108)^5 = $6,175M × 0.5988 = $3,697M

企业价值 (EV) = $1,372M + $3,697M = $5,069M
```

#### D. 计算股权价值与每股价格

```
企业价值 (Enterprise Value)      $5,069M
- 净债务 (债务总额 - 现金)       ($450M)
+ 非主营业务资产                 $0M
- 少数股东权益                   $0M
- 优先股                         $0M
= 股权价值 (Equity Value)        $4,619M

稀释后流通股本                   100M

每股价格 = $4,619M / 100M = $46.19

当前股票价格：$42.00
隐含上涨空间：10.0%
```

### 第 3 步：可比公司分析 (Comparable Companies Analysis)

#### A. 筛选可比公司

**筛选标准：**
- 属于相同行业/板块（核心要求）
- 具备类似的商业模式
- 规模具备可比性（市值、营收体量）
- 处于相近的增长阶段/增速剖面
- 目标市场与地域范围相近

**确定 5-10 家同业公司：**
1. [同业 1] - 直接竞争对手
2. [同业 2] - 直接竞争对手
3. [同业 3] - 延伸领域参与者
4. [同业 4] - 业务模式相似
5. [同业 5] - 区域性竞争对手
6. [再补充 3-5 家]

**要求对每家挑选的同业公司进行逻辑说明并存档记录。**

#### B. 收集同业财务数据

**针对每家可比公司，汇总以下信息：**
- 当前股价
- 流通股本（稀释后）
- 公司市值 (Market Capitalization)
- 债务总额与现金（用于计算 EV）
- 企业价值 (Enterprise Value)
- LTM (过去十二个月) 实际财务数据：
  - 营业收入
  - EBITDA
  - EBIT
  - 净利润
- NTM (未来十二个月) 一致预期数据
- 营收增长率预期
- EBITDA 利润率预期

**推荐数据源：**
- FactSet, CapitalIQ, Bloomberg (首选专业终端)
- 公司 10-K/10-Q 财报 (实际数据)
- Yahoo Finance, Seeking Alpha 一致预期数据（如缺少专业终端）

#### C. 计算估值乘数

**针对每家公司，计算如下乘数：**
```
EV/Revenue (LTM) = 企业价值 / 过去十二个月营业收入
EV/Revenue (NTM) = 企业价值 / 未来十二个月预计营业收入
EV/EBITDA (LTM) = 企业价值 / 过去十二个月 EBITDA
EV/EBITDA (NTM) = 企业价值 / 未来十二个月预计 EBITDA
P/E (NTM) = 市值 / 未来十二个月预计净利润
```

#### E. 将乘数应用于目标公司

**选择核心估值乘数（对成熟公司通常采用 EV/EBITDA）：**

```
目标公司 NTM（未来12个月）EBITDA = $550M (数据提取自财务模型)

采用同业中位数乘数：
同行业中位数 EV/EBITDA (NTM) = 13.8x
隐含企业价值 (EV) = $550M × 13.8x = $7,590M

采用25分位数（保守情形）：
25分位数 EV/EBITDA (NTM) = 12.5x
隐含企业价值 (EV) = $550M × 12.5x = $6,875M

采用75分位数（乐观情形）：
75分位数 EV/EBITDA (NTM) = 14.1x
隐含企业价值 (EV) = $550M × 14.1x = $7,755M

估值区间（可比公司法）：$6,875M - $7,755M
区间中值：$7,315M

换算为股权价值：
隐含企业价值（选取中位数） $7,590M
- 净债务                   ($450M)
= 隐含股权价值             $7,140M

流通股本                   100M
隐含每股价格               $71.40
```

**论证溢价/折价的合理性：**
- 目标公司增速 17% 对比 行业中位数 17% → 符合行业平均水平
- 目标公司 EBITDA 利润率 23% 对比 行业中位数 23% → 符合行业平均水平
- 目标公司市场地位 → [阐述溢价/折价的立论基础]
- **结论**：适用该中位数乘数进行估值（无需进行折溢价调整）

#### B. 权重的决策依据

**常见的权重分配方案：**
- DCF 分析法: 40-60% (预测能见度较高时赋予高权重)
- 二级市场可比公司法: 25-40% (反映当前市场交易情绪)
- 先例交易法: 10-25% (除非短期内具备潜在被收购预期，否则占较低权重)

**本财务模型案例分配：**
- DCF 占 50%：核心财务指引与预测数据可靠度较高
- 市场可比 40%：同业对照组数量够且具有极强的对比度
- 先例交易 10%：短期出现资本并购的概率相对较低

## 合理性交叉检验 (Sanity Checks)

**严格执行以下估值边界校验清单：**

1. **历史乘数中枢校准**
   - 当前我们根据模型做出的定价隐藏乘数是否偏离该标的历史均值交易带？
   - 出现大偏离必须给予极为坚实的拆解理由。

2. **核心同业锚对标**
   - 对比起跑线相同的梯队同业给出的乘数溢价（或是打下的折价）能否被公司的基本面逻辑链条说得通？
   - 在这三个维度进行检验：财务成长势能、盈利空间质量缓冲垫、行业护城河深浅。

3. **逆向隐含增速折算倒推**
   - 假设以市场的即期公允牌价逆推回基本面，市场到底对于这台机器运转给出了多少的超额增长隐性定价？
   - 基于公司历史航道评估，这些预期是过激还是保守偏见？

4. **全局市值边界雷达**
   - 公司在这套加持下所膨胀的总规模块头（市值体盘）放置于产业图解版图里是否还能自圆其说？
   - 会不会使得其体量被异化成产业图谱里的绝对扭曲极值项？

5. **尾箱终端价值警戒监控**
   - 终端价值在企业价值中的权重占比是否守住了 < 60-70% 的生命红线？
   - 如果大幅跳飞穿透超过 70%，高度建议你的测算排期年份不够，请再延展你的跨期预测视野池。

6. **WACC 基础测温与定点检查**
   - 算出的加权融资金额硬成本 (WACC) 指标是否落位于常规公司的正常血压地带范围内（正常阈值在 8-14% 之间）？
   - 科技或扩张成长期：正常在 10-14% 左右。
   - 业务成熟/红海低波动期：正常在 7-10% 左右。

7. **收益安全边际 IRR 排期**
   - 通过买入当下价持有 12 个月兑换最终标的价格的周期里面，内部收益率 (IRR) 能带来多少防守缓冲安全垫？
   - 是否同你的强烈建议购买定调相呼应？

> **💡 Appendix: 领域知识小贴士**
>
> 刚接触这些金融术语？别担心，让我们用最通俗易懂的方式来拆解文中的几个核心金融概念：
> 
> - **自由现金流 (FCF - Free Cash Flow)**：想象一家卖柠檬水的公司。一年下来卖柠檬水赚了钱，扣除买柠檬的成本、付给员工的工资、买新榨汁机（资本支出）以及交税后的钱，这笔真金白银才是可以揣进股东口袋或用来还债的“绝对净收益”。这就是自由现金流。
> 
> - **现金流折现法 (DCF - Discounted Cash Flow)**：“今天的100块由于能存银行生息，肯定比明年才能拿到的100块更值钱”。DCF 的核心就是把公司未来每年能赚到的“自由现金流”，考虑到时间成本和风险后，像打折一样一步步折算成今天的价值。把这些“今天的价值”全加起来，就等于这家公司现在到底值多少钱。
> 
> - **加权平均资本成本 (WACC)**：这是DCF模型中用来“打折”的折现率。你可以把它理解为公司找人要钱的“平均成本”。公司借银行的钱要付利息（债务成本），找股东融资要给回报预期（股权成本）。综合起来，为了让公司转起来，它每年收益率的“及格线”就是 WACC。赚不到这个数字，公司就是在毁灭价值。
> 
> - **终端价值 (Terminal Value)**：我们不可能准确预测一家公司未来100年的现金流（太扯了）。所以做 DCF 模型时，通常只详细预测未来 5 年。至于第 5 年以后的所有年份，我们会用一个公式把它们打包估算成一个巨大的“大礼包价”，并在模型里加到最后一年，这就是终端价值。
> 
> - **EV/EBITDA 估值乘数**：这是投资界最爱用的“比价神器”。EV（企业价值）是要是把你整个公司完全买断并帮你还清债务的总代价。EBITDA 是公司最核心的经营利润（剔除折旧、税收这些账面因素）。两者的比率相当于在问：“假设公司不吃不喝把每年经营赚的钱全给我，我需要等多少年才能把当初买下这家公司掏的钱回本？”倍数越低，说明公司越便宜（或者增长潜力被市场看衰）。
> 
> - **估值足球场图 (Valuation Football Field)**：就像盲人摸象，DCF法、各种可比公司乘数法算出来的公司价值都不一样，有的高有的低。分析师会把这些不同方法得出的“估值区间”画在同一张图表上，看起来就像一个足球场上的不同跑道。最后大家会在这些跑带重合最密集的那个“黄金区域”里拍板定下合理目标价。
