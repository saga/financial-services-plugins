## 现金流折现（DCF）分析

### 概述

DCF 分析根据公司预期未来现金流的现值对公司进行估值。这被认为是在理论上最严谨的估值方法，因为它基于基本的价值创造逻辑。

### DCF 操作步骤

#### 1. 历史财务分析
- 收集 3-5 年的历史财务数据。
- 计算历史自由现金流（FCF）= EBIT × (1 - 税率) + 折旧与摊销 - 资本支出 - 营运资本变动。
- 分析历史增长率和利润率。
- 识别趋势和周期性。

#### 2. 建立营收预测（5-10 年）
**预测方法：**
- **自上而下（Top-down）**：从市场规模（TAM）→ 市场份额 → 营业收入。
- **自下而上（Bottom-up）**：销量 × 单价。
- **混合法**：结合多种驱动因素。

**关键考虑因素：**
- 管理层指引与历史增长。
- 行业增长率及市场趋势。
- 竞争格局与市场份额演变。
- 产品管线与新市场机遇。
- 宏观经济因素。

#### 3. 预测经营费用
- **营业成本（COGS）**：占营收的百分比（分析历史毛利率）。
- **销售与管理费用（SG&A）**：通常具有半固定性质；考虑规模效应，按营收百分比建模。
- **研发费用（R&D）**：对科技/医药行业至关重要；按营收百分比建模。
- **折旧与摊销（D&A）**：基于资本支出假设。

**计算息税前利润（EBIT）** = 营收 - 营业成本 - 经营费用

#### 4. 计算无杠杆自由现金流
```
EBIT
× (1 - 税率)
= NOPAT (税后净营业利润)
+ 折旧与摊销
- 资本支出 (CapEx)
- 营运资本增加额 (NWC)
= 无杠杆自由现金流 (UFCF)
```

**资本支出假设：**
- 维护性支出：维持现有业务运营所需（通常占营收的 2-4%）。
- 扩张性支出：业务扩张所需。
- 参考行业基准和公司指引。

**净营运资本：**
- NWC = (应收账款 + 存货) - 应付账款
- 按营收百分比或周转天数（DSO, DIO, DPO）建模。
- 净营运资本的增加意味着现金的流出。

#### 5. 确定终值 (Terminal Value)

**方法 A：永续增长法 (Perpetuity Growth Method)**
```
终值 = 预测末年 FCF × (1 + g) / (WACC - g)
```
- g = 永续增长率（通常为 2-3%，不应超过 GDP 增长率）。
- 适用于公司进入成熟稳定增长的情况。

**方法 B：退出倍数法 (Exit Multiple Method)**
```
终值 = 预测末年 EBITDA × 退出倍数
```
- 退出倍数基于当前可比公司的市场交易乘数。
- 对周期性业务更为适用。

#### 6. 计算加权平均资本成本 (WACC)

```
WACC = (E/V × 权益成本) + (D/V × 债务成本 × (1 - 税率))
```

**权益成本（使用 CAPM 模型）：**
```
权益成本 = 无风险利率 + Beta × 权益风险溢价
```
- 无风险利率：10 年期国债收益率。
- Beta：个股回报率相对市场的回归系数（或参考可比公司 Beta）。
- 权益风险溢价：历史平均水平约为 5-6%。

**债务成本：**
```
债务成本 = 无风险利率 + 信用利差
```
- 使用公司现有的借贷利率或从债券收益率推导。
- 若无存续债券，则根据信用评级进行调整。

**资本结构：**
- E/V = 股权市值 / 总价值
- D/V = 债务市值 / 总价值
- 使用目标资本结构而非当前结构（若二者差异显著）。

#### 7. 将现金流折现至现值

```
PV = Σ [FCFt / (1 + WACC)^t] + [终值 / (1 + WACC)^n]
```

#### 8. 计算企业价值与权益价值

```
企业价值 (Enterprise Value) = 预测期 FCF 现值 + 终值现值
减：净债务 (总债务 - 现金)
加：非经营性资产
减：少数股东权益
减：优先股
= 权益价值 (Equity Value)

每股价格 = 权益价值 / 稀释后发行在外的总股数
```

### DCF 敏感性分析

始终对核心变量进行敏感性分析：

1. **二维敏感性评估表**：WACC 对比 永续增长率。
2. **营收增长情景**：基准 / 乐观 / 悲观情景。
3. **利润率假设**：经营杠杆情景。
4. **终值倍数敏感性**：如果使用退出倍数法。

**敏感性评估表示例：**
```
             永续增长率
WACC      2.0%    2.5%    3.0%
8.0%      $45     $48     $52
9.0%      $40     $43     $46
10.0%     $36     $39     $41
```

### 应当避免的 DCF 常见陷阱

1. **增长重复计算**：在预测高增长时，不要忽视相应的配套投入（资本支出、营运资本）。
2. **不现实的永续增长率**：不应超过长期 GDP 增长率。
3. **忽视周期性**：对于周期性企业，应对收益进行平准化处理。
4. **现金流定义错误**：应使用无杠杆自由现金流，而非净利润。
5. **假设不一致**：折现率与现金流必须匹配（无杠杆 FCF 对应 WACC）。

## 可比公司分析

### 概述

可比公司分析法基于公开市场对同类公司的定价来对目标公司进行估值。这反映了当前的市场情绪和相对估值水平。

### 可比分析操作步骤

#### 1. 选择可比公司

**筛选标准：**
- 同一行业/板块（核心标准）。
- 相似的业务模式和收入来源。
- 可比的规模（市值、营收）。
- 相似的增长特征和利润率。
- 相似的终端市场和地理区域。

**典型样本库：**
- 初选 8-15 家公司。
- 剔除存在特殊情况的公司。
- 确定最终包含 5-10 家公司的样本组。

#### 2. 收集财务信息

**所需数据：**
- 当前股价及发行在外股数。
- 最近一个财年的财务报表。
- 未来一年（NTM）的分析师一致预期数据。
- 历史增长率。

**计算市场指标：**
- 市值 = 股价 × 发行在外股数。
- 企业价值 (EV) = 市值 + 债务 + 少数股东权益 + 优先股 - 现金。
- 净债务 = 总债务 - 现金及现金等价物。

#### 3. 计算估值乘数

**企业价值乘数：**
- **EV/Revenue (市销率)**：适用于早期阶段或高增长公司。
- **EV/EBITDA**：最常用；适用于资本密集型业务。
- **EV/EBIT**：当折旧与摊销差异显著时非常有用。

**权益价值乘数：**
- **P/E (市盈率)**：应用最广泛。
- **P/B (市净率)**：适用于金融机构。
- **P/S (市销率)**：用于尚未盈利的公司。

**计算周期：**
- 最近十二个月 (LTM) - 历史水平。
- 未来十二个月 (NTM) - 预测水平（首选）。

#### 4. 分析并选择乘数

**创建可比公司明细表：**

| 公司 | 市值 | EV/营收 | EV/EBITDA | EV/EBIT | 市盈率 (NTM) | 营收增长 | EBITDA 利润率 |
|---------|-----------|------------|-----------|---------|-----------|----------------|---------------|
| 公司 A  | $100亿    | 3.5x       | 12.0x     | 18.0x   | 22.0x     | 15%            | 28%           |
| 公司 B  | $80亿     | 3.0x       | 10.5x     | 16.0x   | 19.0x     | 12%            | 27%           |
| ...     | ...       | ...        | ...       | ...     | ...       | ...            | ...           |
| 中位数  | -         | **3.2x**   | **11.0x** | **17.0x** | **20.5x** | 13%            | 27.5%         |

**调整：**
- 剔除异常值（通常指偏离均值 2 个标准差以上的数据）。
- 考虑使用中位数而非平均值（受异常值影响较小）。
- 若某些公司更具代表性，可进行加权处理。
- 根据增长、利润率、风险差异进行调增或调减。

#### 5. 将乘数应用于目标公司

**计算示例：**
```
目标公司 NTM EBITDA = $5亿
选定的 EV/EBITDA 乘数 = 11.0x
隐含企业价值 = $5亿 × 11.0x = $55亿

减：净债务 = $10亿
权益价值 = $45亿

发行在外股数 = 1亿股
隐含每股价格 = $45.00
```

#### 6. 选择合适的乘数

**选择依据：**
- **EV/Revenue**：高增长、未盈利公司（科技、盈利前的生物技术）。
- **EV/EBITDA**：最通用；资本密集型行业（制造、电信）。
- **P/E**：资本结构稳定的盈利公司（消费、零售）。
- **行业特定指标**：银行用 P/B，油气用 EV/产量，媒体用 EV/用户数。

### 溢价/折价分析

根据以下因素进行溢价或折价处理：
- **增长溢价**：增长越高 → 乘数越高。
- **盈利能力**：利润率越高 → 乘数越高。
- **规模**：大型公司通常享有更高的溢价（流动性好）。
- **市场地位**：市场领导者 → 溢价。
- **地理因素**：发达市场 vs. 新兴市场。

## 先例交易分析

### 概述

先例交易分析通过参考并购（M&A）交易中为同类公司支付的价格来估算价值。这反映了控制权溢价和战略价值。

### 先例分析操作步骤

#### 1. 识别相关交易

**筛选标准：**
- 相同或相似行业。
- 相似规模（通常在目标规模的 0.5 倍到 2 倍之间）。
- 相似的业务特征。
- 近期交易（首选过去 3-5 年）。
- 已公告并完成的交易（避免撤回的交易）。

**典型样本库：**
- 至少 5-10 笔交易。
- 侧重近期交易（时间越近，权重越高）。

#### 2. 收集交易细节

**所需信息：**
- 交易日期（公告日及交割日）。
- 收购价格及结构（现金、股票、混合）。
- 标的公司交易时的财务数据。
- 战略意图及协同效应。
- 支付的控制权溢价。

**信息来源：**
- 证交会（SEC）披露文件（S-4, 8-K, 委托代理书）。
- 新闻稿及投资者演示材料。
- 并购数据库（CapIQ, FactSet, Bloomberg）。

#### 3. 计算交易乘数

**乘数体系与可比公司法一致，但基于交易价值：**

```
交易价值 = 股权收购价格 + 承继债务 - 收购的现金

EV/Revenue (LTM) = 交易价值 / 标的公司 LTM 营收
EV/EBITDA (LTM) = 交易价值 / 标的公司 LTM EBITDA
EV/EBIT (LTM) = 交易价值 / 标的公司 LTM EBIT
```

**计算控制权溢价：**
```
控制权溢价 = (报价 - 受影响前价格) / 受影响前价格
```
- 受影响前价格 = 公告前 1-2 天标的公司的股价。
- 典型范围：20-40%。

#### 4. 分析先例交易

**创建先例交易表：**

| 日期 | 目标公司 | 收购方 | 交易金额 | EV/营收 | EV/EBITDA | 溢价 | 交易逻辑 |
|------|--------|----------|------------|------------|-----------|---------|-----------|
| Q1'24 | 公司X | 买方A | $50亿 | 4.0x | 14.0x | 35% | 市场整合 |
| Q3'23 | 公司Y | 买方B | $35亿 | 3.5x | 12.5x | 28% | 战略契合 |
| 中位数 | - | - | - | **3.8x** | **13.0x** | **31%** | - |

#### 5. 应用于目标公司

**重要考虑因素：**
- 先例交易乘数通常高于可比公司分析乘数（因为包含控制权溢价）。
- 根据交易逻辑的差异进行调整。
- 考虑交易当时的市场环境与当前环境的差异。
- 对近期交易给予更高权重。

**计算示例：**
```
目标公司 LTM EBITDA = $4.5亿
选定的 EV/EBITDA 乘数 = 13.0x (先例法)
对比可比公司法乘数 = 11.0x

隐含企业价值 (先例法) = $4.5亿 × 13.0x = $58.5亿
隐含企业价值 (可比法) = $4.5亿 × 11.0x = $49.5亿

隐含控制权溢价 = $58.5亿 / $49.5亿 - 1 = 18%
```

### 交易乘数的调整

**考虑针对以下因素进行调整：**
- **市场环境**：牛市 vs. 熊市（并购活跃程度）。
- **交易结构**：战略买家 vs. 财务买家。
- **协同效应**：协同效应高的交易通常会产生更高溢价。
- **竞价动力**：单一投标人 vs. 多个竞标者。
- **时间价值**：历史过旧的交易参考意义较低。

## 估值汇总与勾稽

### 创建估值桥梁图

在一个统一框架内展示所有三种方法：

**估值摘要示例：**

| 估值方法 | 企业价值 | 权益价值 | 每股价格 | 权重 | 隐含价值 |
|--------|------------------|--------------|-------------|--------|---------------|
| DCF 分析 | $52.0亿 | $42.0亿 | $42.00 | 50% | $21.00 |
| 可比公司分析 | $55.0亿 | $45.0亿 | $45.00 | 30% | $13.50 |
| 先例交易法 | $58.5亿 | $48.5亿 | $48.50 | 20% | $9.70 |
| **加权平均值** | - | - | **$44.20** | - | **$44.20** |

### 权重分配

**典型的权重分配：**
- **DCF**：40-60%（基本面价值，但依赖预测的准确性）。
- **可比公司法**：25-40%（反映当前市场情绪）。
- **先例交易法**：15-25%（除非并购可能性较高，否则参考性较低）。

**权重调整依据：**
- **预测置信度**：信心越高 → DCF 权重越高。
- **市场情况**：牛/熊市会影响可比公司法的可靠性。
- **并购可能性**：若公司处于被收购预期或行业正在整合，则调高权重。
- **公司成熟度**：成熟公司 → 调高可比公司权重；增长型公司 → 调高 DCF 权重。

### 估值区间

始终提供估值区间，而非单一的点估值：

**方法：**
- **基准情景 (Base Case)**：概率最高的情景。
- **乐观情景 (Bull Case)**：乐观的假设（营收增长、利润率）。
- **悲观情景 (Bear Case)**：保守的假设。

**示例：**
```
悲观区间：$38 - $40
基准区间：$42 - $46
乐观区间：$48 - $52

投资建议：买入，目标价 $45（基准情景的中值）
```

### 合理性检查 (Sanity Checks)

**对估值进行交叉验证：**
1. **历史乘数**：当前的估值是否符合历史波动范围？
2. **同业对比**：相对于同行的溢价/折价是否有理有据？
3. **隐含增长率**：市场目前的定价反映了什么样的增长预期？
4. **隐含回报率**：从当前股价到目标价的内部收益率（IRR）。
5. **市值分析**：计算出的总市值是否符合常识逻辑？

## 结论

综合使用这三种估值方法可以构建一个判断公允价值的稳健框架：

- **DCF** 基于基本面提供内在价值参考。
- **可比公司分析** 反映了当前市场的实时定价。
- **先例交易分析** 指示了并购价值和控制权溢价。

成功的关键在于深入理解驱动每种方法的假设，并结合多场景和多维度分析，呈现出一个逻辑审慎的估值区间。

> **💡 Appendix: 领域知识小贴士**
> 
> 什么是“估值”？简单来说，估值就是给公司“贴身价”。就像你买二手房，可以看装修和地段（基本面），也可以看隔壁家刚卖了多少钱（可比分析）。
> 
> *   **DCF (现金流折现)**：想象你有一台能吐钱的机器，DCF 就是算算这台机器未来一共能吐多少钱，然后把这些还没见着的钱换算成现在的价值。毕竟，未来的 100 块钱可没有今天的 100 块香（因为有货币时间价值）。
> *   **乘数 (Multiples)**：这是一种“货比三家”的快捷方式。比如大家买奶茶店，通常说“地段好的一年挣 10 万，卖价 30 万”，也就是“利润的 3 倍”。这里的“3 倍”就是乘数。在股市里，常用的 P/E（市盈率）就是看你要花多少倍的利润才能买下这股。
> *   **控制权溢价 (Control Premium)**：如果你在超市买一瓶水是 2 块，但如果你想把整家超市买下来当老板，你肯定得额外多掏点钱。这多出来的部分就是溢价。因为你不仅买到了水，还买到了“说了算”的权力。
> *   **WACC (资本成本)**：这是公司“借钱买身家”的平均利息。如果公司折腾一年的回报还抵不上这个“利息”，那它其实是在亏钱。
