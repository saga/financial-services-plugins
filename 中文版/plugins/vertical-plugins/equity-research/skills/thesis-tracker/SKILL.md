# 投资逻辑跟踪器 (Thesis Tracker)

description: 维护并更新投资组合持仓及观察名单的投资逻辑。跟踪关键数据点、催化剂以及随时间推移的投资逻辑里程碑。适用于根据新信息更新投资逻辑、评估持仓理由或检查投资逻辑是否依然成立的情况。触发词包括“更新 [公司] 的投资逻辑”、“我的投资逻辑是否依然成立”、“逻辑核查”、“为 [公司] 添加数据点”或“审查我的持仓”。

## 工作流程

### 第 1 步：定义或加载投资逻辑

若创建新逻辑：
- **公司**：名称与股票代码
- **头寸**：多头 (Long) 或 空头 (Short)
- **投资逻辑陈述**：1-2 句核心逻辑（例如：“做多 ACME —— 随着产品组合转向软件业务，凭借定价能力和运营杠杆实现利润率扩张”）
- **核心支柱**：3-5 个支持性论据
- **核心风险**：3-5 个可能导致逻辑失效的风险因素
- **催化剂**：可能证实或推翻逻辑的近期事件（财报发布、产品上线、监管决定）
- **目标价 / 估值**：如果逻辑兑现，该资产价值几何
- **止损触发点**：什么情况下你会选择平仓退出

若更新现有逻辑，请询问用户新的数据点或事态进展。

### 第 2 步：更新日志

针对每个新数据点或进展：

- **日期**：事件发生时间
- **数据点**：发生了什么变化（财报超预期、管理层变动、竞争对手动作等）
- **逻辑影响**：该点是加强、削弱还是中和了某个特定支柱？
- **操作**：维持不变 / 加仓 / 减仓 / 清仓
- **最新确定性**：高 / 中 / 低

### 第 3 步：投资逻辑计分卡

维护一份动态计分卡：

| 核心逻辑支柱 | 初始预期 | 当前状态 | 趋势 |
|--------|---------------------|----------------|-------|
| 营收增长 >20% | 符合预期 | 三季度增长 22% | 稳定 |
| 利润率扩张 | 滞后 | 利润率同比持平 | 令人担忧 |
| 新产品发布 | 待定 | 推迟至二季度 | 观察 |

### 第 4 步：催化剂日历

跟踪即将发生的催化剂：

| 日期 | 事件 | 预期影响 | 备注 |
|------|-------|-----------------|-------|
| | | | |

### 第 5 步：输出

适用于以下场景的投资逻辑摘要：
- 晨会讨论
- 投资组合审查
- 风险委员会陈述

格式：包含计分卡、最新更新及当前确定性水平的精炼 Markdown 或 Word 文档。

## 重要说明

- 投资逻辑必须是**可证伪**的 —— 如果没有任何证据能推翻它，那它就不是投资逻辑，而是偏见。
- 跟踪“证实证据”的同时，必须同样严格地记录“证伪证据”。
- 即使没有发生重大事件，也应至少每季度审查一次投资逻辑。
- 如果用户管理多个头寸，建议进行完整的投资组合逻辑审查。
- 以结构化格式存储投资逻辑数据，以便跨会话参考。

## Appendix: 金融背景知识

这份文件是"投资逻辑跟踪器（Thesis Tracker）"的详细说明，教你怎么建立和跟踪一个靠谱的股票投资逻辑。投行的股票分析师、对冲基金的基金经理、私募基金的投研人员每天都在做这件事——给每一只持仓或者观察中的股票写一个"剧本"，然后随时根据新发生的事件来更新它。很多散户亏钱的根源，不是因为选错了股票，而是因为根本没有想清楚"我为什么要买它"以及"什么情况下我应该认赔走人"。本章将从零开始，带你理解什么是投资逻辑、为什么它如此重要、以及一个完整的投资逻辑应该包含哪些要素。

### 1. 什么是投资逻辑？为什么不写逻辑就是赌博？

**用一个生活中的类比：**

想象你和朋友打了一个赌："我赌老王家明年一定能考上清华。" 如果你只是凭感觉说的，这叫"猜"。但如果你有理有据地说："老王上学期年级排名稳定在前5，他的模拟考分数连续三次超过清华录取线，他参加了清华的自主招生还拿了加分资格，而且他家住在海淀区——这就是他的主场"，那这个就叫"投资逻辑"。

在股票投资里，投资逻辑（Thesis）就是你买入一只股票之前必须想清楚的"作战计划"：

- **你为什么觉得它会涨？**（核心论点）
- **有哪些证据支持你的想法？**（支撑支柱）
- **什么时候证明你错了？**（可证伪条件）
- **涨到什么程度你满意了？**（目标价）
- **跌到什么程度你必须跑？**（止损线）

**没有投资逻辑的买入，本质上就是赌博。** 你只是在赌"我感觉它会涨"，而不是在"投资一个我相信会成功的故事"。

### 2. 投资逻辑的核心要素：一步步拆解

一个完整的投资逻辑就像一个侦探破案的推理过程，需要包含以下几个关键部分：

#### 2.1 核心论题陈述

这是你投资逻辑的"一句话总结"。好的论题陈述应该像电梯演讲那样——如果有人和你在电梯里相遇，只有30秒的时间，你能不能讲清楚你为什么买这股票？

**好例子：** "我做多特斯拉，因为我认为它的下一代4680电池技术将把每千瓦时成本降到比行业平均低30%，从而在未来三年内把毛利率从20%拉高到30%，而市场还没有完全定价这个效率提升。"

**烂例子：** "我买特斯拉，因为电动车是大趋势啊，而且我觉得马斯克很牛。"

这两种说法的区别在哪里？前者是可验证的（你可以去查4680电池的成本数据），后者是模糊不可验证的（"大趋势"、"很牛"都是无法量化的感觉）。

#### 2.2 支撑支柱

把核心论题拆成3-5个可以独立验证的小论点。每个支柱必须能回答："我怎么判断这事对不对？"

| 支柱维度 | 可以追踪什么数据？ | 什么时候算"验证成功"？ |
|---------|---------------------|----------------------|
| 营收增长 | 季度收入同比增长率 | 连续2个季度营收增速 > 20% |
| 利润率扩张 | 毛利率、运营利润率 | 毛利率从 45% → 52% |
| 新产品贡献 | 新产品收入占比 | 新产品贡献 > 总收入的 30% |
| 市场份额 | 市占率对比（vs 竞品） | 市占率连续4个季度上升 |
| 管理执行力 | 管理层给出的指引 vs 实际 | 实际业绩 ≥ 管理层预估值 |

#### 2.3 反向证据（证伪条件）

这是大多数人最不爱做但最重要的一步：写清楚什么情况下你就认输。

**为什么要提前想好"错了怎么办"？**

心理学上有个著名的"证实偏见"（Confirmation Bias）——人天然地倾向于只关注支持自己观点的信息，而忽视反对自己观点的信息。如果你不提前把"错在哪"写下来，到时候你的大脑会自动帮你找借口：

- "这个季度财报虽然很差，但肯定只是暂时的..."
- "虽然CEO跑路了，但公司基本面没变..."
- "虽然监管要罚款了，但这说明公司已经大到不能倒的地步了..."

**一个严格的证伪条件应该像这样：**

> "如果在接下来的两个季度内，公司的核心产品收入增速连续跌破10%（vs 我现在假设的25%），而且同行业的竞品却还在高速增长，那就说明不是行业问题，而是这家公司自己的竞争力出了毛病。我在这个时间点会无条件清仓。"

**关键原则：证伪条件必须在买入之前就写好，而不是亏了30%之后才思考"要不要割肉"。**

#### 2.4 催化剂日历

催化剂（Catalyst）就是可能让投资逻辑"兑现"或"崩塌"的事件。这些事件就像考试日期一样，你应该提前标注在日历上，而不是它们发生了你才临时抱佛脚。

| 催化剂类型 | 具体例子 | 可能的影响 |
|-----------|---------|-----------|
| 财报发布 | 每季度财报、年度报告 | 验证营收和利润假设是否正确 |
| 产品发布 | 新手机上市、新药获批 | 验证新产品收入贡献假设 |
| 监管决定 | FDA审批、反垄断裁决 | 可能彻底改变公司前景 |
| 管理层变动 | CEO被换、CFO辞职 | 影响执行力和战略方向 |
| 行业事件 | 重要展会、行业政策出台 | 影响整个行业的估值 |

### 3. 投资逻辑的生命周期：从建立到退出

一个投资逻辑不是写完了就放在那里吃灰的，它应该有一个完整的生命周期：

```
建立逻辑 → 持续跟踪 → 验证/证伪 → 调整仓位 → 最终退出
```

**建立阶段**：在买入之前，强迫自己回答所有上面的问题。如果你发现自己回答不出来（比如"我也说不清它的竞争优势是什么"），那就说明你还没准备好买入。

**跟踪阶段**：每次有新数据点（财报、新闻、行业报告），你都要问自己一个问题："这个新信息是支持还是削弱了我的投资逻辑？"然后更新逻辑计分卡。

**退出阶段**：有三种情况你应该卖出：
1. **证明对了**：股价涨到了你的目标价，逻辑已经充分兑现，再持有下去性价比不高了。
2. **证明错了**：你的证伪条件被触发，逻辑不成立，认赔走人。
3. **变化了**：出现了一个全新的情况（比如行业出现颠覆性技术），让你的原始逻辑不再适用，即使股价没怎么动也应该退出。

### 4. 现实案例：一个真实的投资逻辑该怎么写

> **场景**：2023年初，你关注到一家做企业级AI助手软件的公司（假设叫"智语科技"）。

**投资逻辑陈述**：我做多智语科技，因为我相信企业级AI助手市场在未来三年内将从现在的200亿规模增长到800亿，而智语科技作为唯一一家同时拥有微信和企业微信双入口的产品，将凭借这个独特的分发渠道拿走至少30%的市场份额。

**核心支柱：**

| 支柱 | 当前数据 | 目标数据 | 如何验证 |
|------|---------|---------|---------|
| 市场增长 | AI助手市场年增40% | 持续3年 | 行业报告、IDC数据 |
| 渠道壁垒 | 微信+企业微信独家合作 | 维持独家 | 关注竞品是否也拿到合作 |
| 产品力 | 付费客户5000家 | 季度新增>1000家 | 公司季报 |
| 变现能力 | ARPU ¥8000/年 | 提升至¥12000 | 每季度ARPU趋势 |
| 技术壁垒 | 50项AI专利 | 季度新增3+项 | 专利数据库查询 |

**证伪条件（任一触发即清仓）：**
1. 连续2个季度付费客户增速 < 15%
2. 微信终止了与智语的独家合作协议
3. 行业中出现免费替代品，导致ARPU连续下降
4. CEO或CTO同时离职

**催化剂日历：**
- 3月15日：四季度财报 → 验证ARPU和客户增速
- 5月1日：行业AI大会 → 是否有新产品发布
- 7月：微信开发者大会 → 合作关系是否加深
- 8月15日：二季度财报 → 关键验证节点

### 5. 注意事项和常见错误

| 常见错误 | 为什么危险 | 怎么避免 |
|---------|-----------|---------|
| 只看好消息，不看坏消息 | 证实偏见导致"温水煮青蛙" | 强制每天读一篇做空该公司的报告 |
| 逻辑太模糊不可验证 | "这公司很棒"没法验证对错 | 使用可量化的指标和具体的时间节点 |
| 没有止损计划 | 小亏变大亏，10%变50% | 买入当天就写好证伪条件 |
| 太频繁调整逻辑 | 一有风吹草动就改逻辑，等于没有逻辑 | 区分"噪音"和"信号"，只对实质性变化做反应 |
| 逻辑建立时就是"后见之明" | 在股价已经涨了300%之后再写"我看好它"毫无意义 | 逻辑必须建在买入之前，且记录下当时的股价 |

---

### 给小白的一句话

> 投资逻辑就是你买股票前的"作战计划"——你要提前想清楚"为什么买"、"凭什么涨"、"涨到哪里卖"、"跌到哪里跑"这四个问题，然后像侦探一样持续跟踪证据，支持就留着，不支持就果断走。没作战计划的买入，那不是投资，是期望天上掉馅饼。
