# 投资思路生成

description: 系统性的股票筛选与投资思路发掘。结合量化筛选、主题研究和模式识别，挖掘新的做多与做空投资机会。在寻找新投资思路、运行筛选器或进行主题扫描时使用。触发指令包括 "idea generation"、"stock screen"、"find ideas"、"what looks interesting"、"screen for"、"new ideas" 或 "pitch me something"。

## 工作流程

### 第一步：定义搜索标准

向用户询问以下参数：
- **投资方向 (Direction)**：做多思路、做空思路，或两者兼具
- **市值 (Market cap)**：大盘、中盘、小盘、微盘
- **行业 (Sector)**：特定行业或跨行业
- **风格 (Style)**：价值、成长、质量、特殊情况、事件驱动
- **地域 (Geography)**：美国、国际、全球
- **主题 (Theme)**：任何特定的主题视角（如人工智能、制造业回流、人口老龄化等）

### 第二步：量化筛选

根据投资风格运行筛选程序：

**价值型筛选 (Value Screen)**
- 市盈率（P/E）低于行业中位数
- 企业价值倍数（EV/EBITDA）低于历史平均水平
- 自由现金流收益率（FCF Yield）> 5%
- 市净率（P/B）低于 1.5 倍
- 过去 90 天内有内部人士买入
- 股息率高于市场平均水平

**成长型筛选 (Growth Screen)**
- 营收同比增长（YoY）> 15%
- 盈利同比增长（YoY）> 20%
- 营收加速（增长率不断提升）
- 利润率扩张
- 高投入资本回报率（ROIC > 15%）
- 强劲的净收入留存率（SaaS 行业 > 110%）

**质量型筛选 (Quality Screen)**
- 持续的营收增长（5 年以上）
- 利润率稳定或不断扩张
- 净资产收益率（ROE）> 15%
- 低资产负债率
- 高自由现金流转化率
- 内部人士持股比例 > 5%

**做空筛选 (Short Screen)**
- 营收下降或增长急剧放缓
- 利润率受压（Margin compression）
- 应收账款/存货相对于销售额的比例上升
- 内部人士抛售
- 相比同业存在缺乏基本面支撑的估值溢价
- 基本面恶化且做空比例（Short interest）高企
- 财务红旗预警（更换审计师、财务重述）

**特殊事件筛选 (Special Situation Screen)**
- 近期完成 IPO / SPAC 且面临限售股解禁
- 过去 12 个月内发生过业务分拆（Spin-offs）
- 公司刚完成重组摆脱困境
- 维权投资者（Activist investor）介入
- 业绩表现不佳公司的管理层变更

### 第三步：主题扫描

对于主题类投资思路，深入研究该主题并明确受益标的：

1. 界定投资逻辑（例如：“AI 基础设施支出将在 2026 年之前持续加速”）
2. 梳理价值链——区分直接受益者与间接受益者
3. 识别纯正概念股（Pure-play）与多元化资产敞口公司
4. 评估哪些标的已经“充分定价（Priced in）”，哪些仍被“严重低估（Under-appreciated）”
5. 寻找市场尚未与该主题建立联系的二阶受益者（Second-order beneficiaries）

### 第四步：投资思路展示

针对每个通过筛选的思路，采用以下格式进行展示：

**[公司名称] — [做多/做空] — [一句话核心逻辑]**

| 指标 (Metric) | 数据值 (Value) | 对比同业 (vs. Peers) |
|--------|-------|-----------|
| 市值 (Market cap) | | |
| EV/EBITDA (NTM，未来12个月) | | |
| 市盈率 (P/E，NTM，未来12个月) | | |
| 营收增速 (Revenue growth) | | |
| EBITDA利润率 (EBITDA margin) | | |
| 自由现金流收益率 (FCF yield) | | |

**投资逻辑 (3-5 个要点)：**
- 为什么该标的被错误定价
- 市场忽略了什么
- 推动价值兑现的催化剂

**核心风险：**
- 什么情况发生会导致该观点被证伪

**下一步建议：**
- 构建完整财务模型？开展深度尽调？安排专家访谈？

### 第五步：成果输出

- 包含 5-10 个精选思路的候选名单（各附一页执行摘要）
- 记录筛选标准与方法论
- 所有投资思路的横向对比表格
- 优先级列表：建议接下来应优先研究哪些标的

## 重要注意事项

- 筛选器产生的是“潜在候选者”而非最终结论——每一个筛选结果都需要扎实的基本面研究来验证
- 最好的投资思路往往诞生于交叉领域（例如：因遭遇短期逆风而拥有“价值型定价”的“高质量”公司）
- 避免拥挤的交易——仔细检查所有权结构数据、做空比例以及覆盖该标的的分析师数量
- 逆向投资思路（Contrarian ideas）必须有催化剂——如果判断过早又没有催化剂，跟做错毫无区别
- 持续追踪思路的命中率——总结出哪些筛选条件和方法能够产生最优质的投资策略
- 做空思路需要极高的确信度（Conviction）——做空的择时更为困难，且面临着收益有限但亏损无上限的收益风险不对称性

> **💡 Appendix: 领域知识小贴士**
> 
> 如果你是金融投资的新手，不要被上面的专业词汇吓到！我们可以用非常生活化的大白话来理解这些核心概念：
> 
> *   **做多 (Long)与做空 (Short)：** “做多”就是经典的“低买高卖”，看好一个东西会涨；“做空”则是你觉得某个东西太贵了一定会跌，于是你先借别人的东西按高价卖掉，等价格跌了再低价买回来还给他，中间赚个差价。
> *   **市盈率 (P/E) 与 EV/EBITDA：** 这是公司的“贵贱标签”。就像去菜市场买菜，市盈率告诉你“如果要买这家公司每年赚的1块钱，你现在得付几块钱成本”。所以数字越低，往往说明它越便宜（即“价值型”股票）。
> *   **自由现金流 (Free Cash Flow, FCF)：** 想象你每个月的工资发下来，扣除吃饭租房等必须续命的开销，剩下的可以真正自由支配、拿去买排骨或者存起来的钱。对公司来说，自由现金流越多，它抵御风险和给股东发钱的能力就越强。
> *   **催化剂 (Catalyst)：** 股票世界的“闹钟”。如果你发现一只股票特别便宜，但所有人都没关注它，它可能会一直便宜下去。你需要一个“催化剂”（比如突然发布爆款新产品、换了非常有能力的新CEO）来把市场唤醒，让大家蜂拥而至把价格买上去。
> *   **充分定价 (Priced in)：** 金融圈常说“买预期，卖事实”。如果大家都知道某家公司下个月要发大财，那它的股票今天早就已经涨上去了，这就叫好处已经被“充分定价”了。这时候哪怕好消息真的兑现，股票可能也不会涨，甚至还会跌（因为没有更多惊喜了）。
> *   **纯正概念股 (Pure-play)：** 比如你看好“咖啡”，直接卖咖啡豆或者咖啡机的公司就是纯正概念股（纯度极高）；而一家既卖冰淇淋、卖汉堡又顺带卖点咖啡的餐厅，就叫“多元化敞口”（不够纯粹）。在炒某个主题时，通常 Pure-play 的股价弹跳最猛！
