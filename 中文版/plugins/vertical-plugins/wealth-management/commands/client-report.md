## Appendix: 金融背景知识

这份文件是"客户报告（Client Report）"命令的详细说明。在财富管理（Wealth Management）中，客户经理需要定期向高净值客户汇报资产状况——这份报告不仅要展示"赚了多少钱"，还要解释"为什么赚/为什么亏"，以及"接下来怎么办"。

---

### 1. 什么是财富管理（Wealth Management）？

**类比：**
想象你是一个"家庭财务管家"。一个有钱家庭有 5000 万资产，分散在股票、债券、房产、保险、信托中。家庭成员（客户）不懂投资，也不想花时间去管。你的工作是：
1. 帮他们配置资产
2. 定期汇报收益
3. 根据市场变化调整策略
4. 回答他们的问题

财富管理就是"为有钱人打理钱"的专业服务。

---

### 2. 客户报告的"三大核心内容"

#### 2.1 资产概览（Portfolio Overview）
- 总资产市值
- 各资产类别占比（股票、债券、现金、另类投资）
- 与上期的对比

#### 2.2 业绩归因（Performance Attribution）
- 赚了多少钱？（绝对收益）
- 跑赢/跑输基准多少？（相对收益）
- 哪些资产贡献了收益？哪些拖了后腿？

#### 2.3 市场展望与建议（Outlook & Recommendations）
- 宏观经济分析
- 市场趋势判断
- 调仓建议

---

### 3. 业绩归因的"底层逻辑"

**核心问题：** 客户赚了 10%，是因为"选股能力强"，还是因为"市场整体涨了 10%"？

| 归因维度 | 说明 |
|---------|------|
| 资产配置效应 | 因为多配了股票（涨得多）而赚钱 |
| 选股效应 | 因为选了牛股而赚钱 |
| 择时效应 | 因为低买高卖而赚钱 |
| 交互效应 | 以上三者的叠加 |

**例子：**
- 客户赚了 15%
- 基准（市场平均）涨了 10%
- 超额收益 5% 来自：资产配置贡献 2% + 选股贡献 3%

---

### 4. 客户沟通的"黄金法则"

#### 法则 1：先讲结论
```
错误："上个季度市场波动很大，美联储加息了..."
正确："上个季度您的组合赚了 5%，跑赢基准 2%"
```

#### 法则 2：用客户能懂的语言
```
错误："Beta 调整为 0.8，Sharpe Ratio 提升至 1.2"
正确："我们降低了风险，让您的投资更稳健"
```

#### 法则 3：主动承认亏损
```
错误：避而不谈亏损部分
正确："XX 股票亏了 3%，原因是...我们的应对措施是..."
```

---

### 5. 真实案例：某家族办公室的客户报告

**客户背景：** 资产 2 亿美元，风险偏好中等

| 模块 | 内容 |
|------|------|
| 资产概览 | 股票 40%、债券 30%、另类 20%、现金 10% |
| 业绩 | 本季度 +3.2%，基准 +2.5%，超额 +0.7% |
| 归因 | 股票选股贡献 +1.5%，债券利率风险拖累 -0.8% |
| 展望 | 预计下半年通胀回落，建议增配成长股 |
| 建议 | 减持 5% 债券，增持 5% 科技基金 |

---

### 给小白的一句话

> 客户报告就是财富管理的"成绩单"——想象你是家长，每个学期结束班主任都会给你一份孩子的成绩单：语文 90、数学 85、英语 95，还会告诉你"数学退步了因为最近状态不好"。客户报告就是给客户看的"投资成绩单"：股票赚了 10%、债券亏了 2%、整体赚了 5%。好的客户报告不仅展示数字，还会解释"为什么赚、为什么亏、接下来怎么办"——让客户觉得"这个管家靠谱"。
