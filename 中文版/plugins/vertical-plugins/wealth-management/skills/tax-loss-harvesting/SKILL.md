# 税务亏损收割 (Tax-Loss Harvesting)

description: 识别应税账户中的税务亏损收割机会。找到具有未实现亏损的持仓，建议替代证券，并跟踪洗售窗口期。触发词："税务亏损收割"、"TLH"、"收割亏损"、"税务亏损"、"未实现亏损"或"年终税务规划"。

## 工作流程

### 步骤1：识别候选

扫描应税账户寻找具有未实现亏损的持仓：

| 证券 | 资产类别 | 成本基础 | 当前价值 | 未实现亏损 | 持有期 | 亏损% |
|------|----------|----------|----------|------------|--------|-------|
| | | | | | 短期/长期 | |

**优先顺序：**
1. 绝对亏损最大（最大税务收益）
2. 先处理短期亏损（抵消按普通收入税率征税的短期收益）
3. 亏损百分比最大的持仓（不太可能快速恢复）

### 步骤2：收益/亏损预算

计算客户的税务状况：

| 类别 | 金额 |
|------|------|
| 年初至今已实现短期收益 | |
| 年初至今已实现长期收益 | |
| 年初至今已实现亏损 | |
| 净收益/(亏损)状况 | |
| 往年结转亏损 | |
| **目标收割金额** | |

**税务节省估算：**
- 短期亏损 × 边际普通收入税率
- 长期亏损 × 资本利得税率
- 最多$3,000净亏损可抵扣普通收入
- 超额部分结转至未来年度

### 步骤3：替代证券

对于每个收割候选，建议替代证券：
- 保持类似市场敞口（相同资产类别、行业、地理区域）
- 不是"实质相同"（洗售规则）
- 具有相似风险/回报特征

| 卖出 | 替代 | 理由 | 跟踪误差风险 |
|------|------|------|--------------|
| SPDR S&P 500 (SPY) | iShares Core S&P 500 (IVV) | 相同指数，不同基金家族 | 极小 |
| Vanguard Total Intl (VXUS) | iShares MSCI ACWI ex-US (ACWX) | 类似敞口，不同指数 | 低 |
| 个股ABC | 行业ETF (XLK) | 更广泛敞口，无洗售风险 | 中等 |

### 步骤4：洗售检查

执行前，验证无洗售：

- 检查家庭所有账户（应税、IRA、Roth、配偶账户）
- 30天回顾：过去30天是否购买了实质相同的证券？
- 30天前瞻：锁定30天内不得回购相同证券
- 检查可能触发洗售的股息再投资计划(DRIPs)
- 记录每笔交易的洗售窗口期

| 卖出证券 | 洗售窗口期开始 | 窗口期结束 | DRIP活跃？ | 风险 |
|----------|----------------|------------|-----------|------|
| | | | | |

### 步骤5：执行计划

| 交易# | 账户 | 行动 | 证券 | 股数 | 预计收益 | 预计亏损 | 替代证券 | 备注 |
|-------|------|------|------|------|----------|----------|----------|------|
| | | 卖出 | | | | | | |
| | | 买入 | | | | | | |

**摘要：**
- 预计收割亏损总额：$
- 预计税务节省：$（边际税率为%）
- 投资组合净影响：极小（替代证券维持敞口）
- 洗售窗口管理：[日期]

### 步骤6：收割后跟踪

30+天后，可选：
- 换回原证券（如偏好）
- 维持替代证券（如无换回理由）
- 更新成本基础记录
- 为税务报告记录

### 步骤7：输出

- 收割机会清单（Excel）
- 交易执行表
- 洗售跟踪日历
- 税务节省估算摘要
- 替代证券理由

## 重要提示

- 洗售规则严格——违规会导致亏损无效且调整成本基础
- 实质相同指相同证券，而非相同资产类别——跟踪不同指数的ETF通常没问题
- 始终协调家庭所有账户，包括退休账户
- 考虑长期成本基础下降——收割会重置成本基础，意味着未来更多收益
- 年终是收割旺季，但全年都有机会
- 12月共同基金资本利得分配可能增加收割紧迫性
- 为税务报告和合规记录所有内容
- 并非所有亏损都值得收割——交易成本和跟踪误差有实际成本

## 金融术语和知识解释

### 1. 税务亏损收割 (Tax-Loss Harvesting)

**定义**：通过出售亏损投资来减少或抵消资本利得税的策略。

**核心原理**：
1. 卖出亏损资产实现亏损
2. 用亏损抵消资本利得
3. 购买类似但不相同的资产维持市场敞口
4. 减少当期税负

**时机**：
- 市场下跌后
- 年终税务规划
- 有大量资本利得时

### 2. 洗售规则 (Wash Sale Rule)

**定义**：美国国税局规则，禁止在卖出亏损资产前后30天内购买实质相同的证券。

**规则细节**：
- **窗口期**：卖出前30天至卖出后30天（共61天）
- **适用范围**：所有账户（包括IRA、配偶账户）
- **实质相同**：同一证券、同一公司的不同份额类别

**后果**：
- 亏损不能在当期扣除
- 亏损金额添加到新购买资产的成本基础
- 持有期顺延

### 3. 资本利得税 (Capital Gains Tax)

**定义**：对出售资本资产获得的利润征收的税。

**分类**：
| 类型 | 持有期 | 税率 |
|------|--------|------|
| 短期资本利得 | <1年 | 普通收入税率（最高37%） |
| 长期资本利得 | ≥1年 | 优惠税率（0%/15%/20%） |

**税率层级（2024年）**：
- 单身收入<$47,025：0%
- $47,026-$518,900：15%
- >$518,900：20%

### 4. 亏损结转 (Loss Carryforward)

**定义**：未使用的资本亏损可结转至未来年度。

**规则**：
- 每年最多可抵扣$3,000普通收入
- 超额亏损无限期结转
- 先抵扣资本利得，再抵扣普通收入

**示例**：
- 年度亏损$10,000
- 当年抵扣$3,000
- $7,000结转至下一年

### 5. 成本基础 (Cost Basis)

**定义**：资产的原始购买价格，用于计算资本利得或损失。

**计算**：
```
资本利得/损失 = 卖出价格 - 成本基础
```

**调整项**：
- 佣金和费用
- 股票分割
- 股息再投资
- 洗售亏损结转

### 6. 未实现亏损 (Unrealized Loss)

**定义**：资产当前市值低于成本基础但尚未卖出时的亏损。

**特点**：
- 纸面亏损（未实现）
- 无税务后果
- 随市场价格波动

**示例**：
- 成本基础：$50
- 当前价格：$30
- 未实现亏损：$20

### 7. 实质相同证券 (Substantially Identical Securities)

**定义**：根据洗售规则，指相同或几乎相同的投资工具。

**判定因素**：
- 同一证券的不同份额类别（如A类和C类）
- 跟踪完全相同指数的ETF
- 同一公司的股票

**安全港**：
- 跟踪不同指数的ETF
- 不同资产类别的投资
- 同一行业但不同公司的股票

### 8. 股息再投资计划 (DRIP)

**定义**：自动将股息再投资购买更多股份的计划。

**洗售风险**：
- DRIP自动购买可能触发洗售
- 需要暂停DRIP或选择现金分红
- 适用于卖出前30天和卖出后30天

**管理策略**：
- 提前暂停DRIP
- 选择现金分红
- 使用替代证券

### 9. 跟踪误差 (Tracking Error)

**定义**：投资组合表现与基准之间的差异。

**原因**：
- 替代证券与原证券不完全相同
- 费用差异
- 再平衡频率

**最小化方法**：
- 选择高度相关的替代证券
- 使用跟踪相同指数的不同ETF
- 定期再平衡

### 10. 税务效率 (Tax Efficiency)

**定义**：衡量投资策略最小化税负的程度。

**税务效率高的投资**：
- 指数基金/ETF（低换手率）
- 市政债券（免税利息）
- 成长型股票（低分红）

**税务效率低的投资**：
- 高换手率主动基金
- 债券基金（利息收入按普通收入征税）
- REITs（分配通常按普通收入征税）

## Appendix: 金融背景知识

这份文件是"税务亏损收割（Tax-Loss Harvesting）"技能的详细说明。TLH 是美国个人投资者最常用的合法避税手段——它巧妙地把"投资亏损"变成"节税武器"。

---

### 1. 什么是 Tax-Loss Harvesting？

**类比：**
你今年炒股赚了 100 万（要交 20 万的税）。但你账户里还有一只烂股票，亏了 50 万（一直舍不得卖）。

省税神操作：年底前把烂股票卖掉，确认 50 万亏损。跟税务局说："今年净赚 = 100 万 - 50 万 = 50 万"，按 50 万缴税（只交 10 万）。比你硬扛着不卖省了 10 万的税。

---

### 2. TLH 的"四步操作法"

```
第一步：扫描账户 → 找出浮亏 > 10% 的品种

第二步：评估价值 → 税额节省 > 交易成本吗？

第三步：执行收割 → 年底前卖出确认亏损

第四步：替换资产 → 立即买入类似品种（避免 Wash Sale）
```

---

### 3. Wash Sale（洗售规则）

| 行为 | 违反？ |
|------|--------|
| 卖 Apple，30天内买回 Apple | 违反 |
| 卖 Vanguard S&P 500 ETF，买 iShares S&P 500 ETF | 可能（实质相同） |
| 卖 Apple，买 Microsoft | 不违反 |
| 卖 S&P 500 ETF，买 Total Market ETF | 不违反 |

---

### 4. TLH 的应用场景

| 场景 | 操作 |
|------|------|
| 每年 11-12 月 | 回顾年度盈亏，决定是否收割 |
| 市场大跌后 | 趁机收割亏损，降低税基 |
| 有大额资本利得 | 用亏损对冲当年的暴利 |

---

### 给小白的一句话

> TLH 就是"以毒攻毒"——你手里有烂股票舍不得卖，IRS（美国国税局）却给你一个机会：卖掉烂股票确认亏损可以直接减少你当年的税单。但有一个大坑：卖完 30 天内不能买回同样的股票（Wash Sale 规则），所以要买一个"长得很像但不是同一只"的替代品。这就像你把家里发霉的橘子扔了，然后换成橙子——维生素 C 照样补，但橘子不是橘子，税务大爷不会查你。
