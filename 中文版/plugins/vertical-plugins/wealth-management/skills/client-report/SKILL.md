# 客户报告 (Client Report)

description: 生成专业的面向客户的绩效报告，包含投资组合回报、配置细分和市场评论。适用于季度或年度分发。触发词："客户报告"、"绩效报告"、"[客户]季度报告"、"生成报告"或"客户对账单"。

## 工作流程

### 步骤1：报告参数

- **客户姓名**和家庭
- **报告期间**：季度、年初至今、年度、自定义范围
- **账户**：所有账户或特定账户
- **基准**：标普500、60/40混合、符合IPS的自定义基准
- **公司品牌**：Logo、颜色、免责声明

### 步骤2：绩效摘要

**家庭摘要：**

| | 本季度 | 年初至今 | 1年 | 3年年化 | 5年年化 | 成立至今年化 |
|---|--------|----------|-----|---------|---------|--------------|
| 投资组合 | | | | | | |
| 基准 | | | | | | |
| +/- | | | | | | |

**按账户：**

| 账户 | 类型 | 价值 | 本季度 | 年初至今 | 基准 |
|------|------|------|--------|----------|------|
| 共同应税账户 | 经纪账户 | | | | |
| John IRA | 传统IRA | | | | |
| Jane Roth | Roth IRA | | | | |
| 529计划 | 教育 | | | | |
| **总计** | | | | | |

### 步骤3：配置概述

当前配置及可视化（饼图或柱状图）：

| 资产类别 | 投资组合占比 | 金额 | 基准占比 |
|----------|-------------|------|----------|
| | | | |

### 步骤4：持仓明细

| 证券 | 资产类别 | 股数 | 价格 | 价值 | 投资组合占比 | 本季度回报 |
|------|----------|------|------|------|-------------|-----------|
| | | | | | | |

### 步骤5：市场评论

根据客户成熟度定制的简短市场摘要：
- 本季度市场发生了什么（2-3句话）
- 对投资组合的影响
- 展望和定位理由（2-3句话）
- 零售客户避免使用行话；成熟投资者可以更技术化

### 步骤6：活动摘要

- 本期间执行的交易
- 缴款和取款
- 收到的股息和利息
- 收取的费用
- 再平衡活动

### 步骤7：规划笔记

- 财务目标进展（退休、教育等）
- 任何计划变更或建议
- 即将到来的行动项目
- 下次审查日期

### 步骤8：输出

- PDF报告（8-12页），带有公司品牌
- Word文档用于定制
- Excel数据附录（可选）

**报告结构：**
1. 封面页（客户姓名、期间、公司Logo）
2. 执行摘要（1页）
3. 绩效摘要（1-2页）
4. 配置概述与图表（1页）
5. 持仓明细（1-2页）
6. 市场评论（1页）
7. 活动摘要（1页）
8. 规划笔记（1页）
9. 披露和免责声明（1页）

## 重要提示

- 除非客户/合规要求毛收益，否则绩效必须扣除费用计算
- 始终包含适当的免责声明（过往业绩、风险因素）
- 报告应在客户之间保持一致——使用标准模板
- 根据客户需求匹配详细程度——有些客户想要每个持仓，有些只想要一页摘要
- 基准选择很重要——使用IPS中的基准，而不是看起来最好的
- 在首次分发新模板之前，审查合规批准

## 金融术语和知识解释

### 1. 客户报告 (Client Report)

**定义**：财富管理公司定期向客户提供的投资组合绩效和状态报告。

**目的**：
- 保持透明度
- 展示价值
- 促进客户沟通
- 满足合规要求

**频率**：
- 季度报告（最常见）
- 年度报告
- 按需报告

### 2. 投资政策声明 (Investment Policy Statement, IPS)

**定义**：定义客户投资目标、风险承受能力和投资指导方针的正式文件。

**核心要素**：
- 投资目标（收益、增长、资本保全）
- 风险承受能力
- 资产配置目标
- 基准选择
- 流动性需求
- 约束条件

**重要性**：
- 作为投资决策的指南
- 管理客户预期
- 评估投资绩效的标准

### 3. 基准 (Benchmark)

**定义**：用于衡量投资组合绩效的参照标准。

**常见基准**：
| 基准 | 描述 |
|------|------|
| S&P 500 | 美国大盘股 |
| 60/40混合 | 60%股票/40%债券 |
| 定制基准 | 根据IPS定制 |

**选择原则**：
- 与投资目标匹配
- 可投资性
- 透明度
- 一致性

### 4. 年化回报 (Annualized Return)

**定义**：将一段时间的回报转换为年度等效值。

**计算公式**：
```
年化回报 = (1 + 总回报)^(1/年数) - 1
```

**示例**：
- 3年总回报50%
- 年化回报 = (1.5)^(1/3) - 1 ≈ 14.5%

**用途**：
- 比较不同时间段的回报
- 评估长期表现

### 5. 资产配置 (Asset Allocation)

**定义**：投资组合中不同资产类别的资金分配。

**常见资产类别**：
| 类别 | 描述 | 风险/回报 |
|------|------|-----------|
| 股票 | 公司所有权 | 高/高 |
| 债券 | 债务工具 | 低/低 |
| 现金 | 现金等价物 | 极低/极低 |
| 另类投资 | 对冲基金、私募股权等 | 高/高 |

**配置策略**：
- 战略性配置（长期目标）
- 战术性配置（短期调整）

### 6. 再平衡 (Rebalancing)

**定义**：调整投资组合回到目标配置的过程。

**触发条件**：
- 时间触发（季度/年度）
- 偏离触发（通常3-5%）

**方法**：
- 卖出超配资产
- 买入低配资产

**目的**：
- 控制风险
- 维持目标风险水平
- 低买高卖（自动实现）

### 7. 总回报 vs 净回报 (Gross vs Net Return)

**定义**：
- **总回报**：扣除费用前的回报
- **净回报**：扣除费用后的回报

**重要性**：
- 客户实际获得的是净回报
- 费用对长期回报有显著影响

**示例**：
- 总回报10%，费用1% → 净回报9%

### 8. 绩效归因 (Performance Attribution)

**定义**：分析投资组合回报来源的过程。

**归因维度**：
| 维度 | 分析内容 |
|------|----------|
| 资产配置 | 各类别权重贡献 |
| 证券选择 | 个股选择贡献 |
| 市场时机 | 择时贡献 |

**用途**：
- 理解回报驱动因素
- 评估投资决策质量
- 识别优势和改进领域

### 9. alpha (Alpha)

**定义**：投资组合超出基准的超额回报。

**计算公式**：
```
Alpha = 投资组合回报 - 基准回报
```

**解读**：
- **正alpha**：跑赢基准
- **负alpha**：跑输基准

**重要性**：
- 衡量主动管理能力
- 扣除风险调整后的超额回报

### 10. KYC (Know Your Client)

**定义**：了解客户身份、财务状况和投资目标的流程。

**要求**：
- 身份验证
- 财务状况评估
- 投资经验评估
- 风险承受能力评估

**合规意义**：
- 防止洗钱
- 确保投资建议适合客户
- 满足监管要求

## Appendix: 金融背景知识

这份文件是"客户报告（Client Report）"技能的详细说明。在财富管理行业，客户报告是客户经理与高净值客户沟通的核心工具——它告诉客户"你的钱怎么样了"。

---

### 1. 什么是客户报告？

**类比：**
想象你是私人医生。每季度你要给病人一份"体检报告"：血压、血糖、胆固醇、跟去年比怎么样、需要调整什么药。

客户报告就是财富管的"季度体检报告"——用专业但易懂的方式告诉客户他的资产状况。

---

### 2. 一份好报告的"三段式"

| 段落 | 内容 | 语言原则 |
|------|------|---------|
| 总结 | 2-3句话概括：赚了多少、为什么 | 先报结果，再解释 |
| 归因 | 谁贡献了收益、谁拖了后腿 | 用"结构性变化/暂时性波动" |
| 展望 | 接下来怎么做、关注什么 | 给出明确行动建议 |

---

### 3. 报告写作的"三不要"

| 不要 | 原因 |
|------|------|
| 不要用术语轰炸 | 客户看不懂"因子暴露"说"某类股票偏多" |
| 不要回避亏损 | 越是亏钱越要解释清楚原因 |
| 不要堆数据 | 50页报告没人看，5页精华才有效 |

---

### 4. 业绩归因的"三层分析法"

```
第一层：市场效应 → 是市场涨了所以赚了？还是你选股选得好？
第二层：行业效应 → 你重仓的行业是顺风还是逆风？
第三层：个股效应 → 你选的每一只股票跑赢还是跑输同行？
```

---

### 5. 真实案例：一份让客户"转忧为喜"的报告

**背景**：客户组合下跌 8%，客户准备撤资

**好的报告**：
> "本季度组合下跌 8%，确实不理想。但我们分析了三个层面：
> 1. 市场效应：大盘下跌 12%，你不是最差的
> 2. 行业效应：你重仓的科技板块跌 20%——这是最大的拖累
> 3. 个股效应：我们选的 5 只科技股平均只跌 15%，跑赢行业 5 个百分点——说明我们的选股能力在保护你的资产"

**结果**：客户决定继续持有。

---

### 给小白的一句话

> 客户报告就是财富管的"成绩单"——客户不关心你的模型多复杂，只关心"赚钱了没、亏了为什么、接下来怎么办"。最好的报告不是"数字最多的报告"，而是"看完三分钟就知道该怎么办的报告"。记住三句话：先说结果、再讲原因、最后给建议。
