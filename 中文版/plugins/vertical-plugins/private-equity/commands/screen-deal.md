### 1. 项目初筛的本质——"PE 的 99% 法则"

**PE 基金的投资漏斗**：
- 每年收到 500-1000 个项目推介
- 初筛后留下 50-100 个（90% 被淘汰）
- 深入研究后留下 5-10 个（又淘汰 90%）
- 投资委员会批准 2-5 个（又淘汰 50%）
- 最终投资 1-3 个

**所以 99% 的项目都被 Pass 掉了**。这不是因为项目"不好"，而是因为 PE 基金有非常明确的投资标准，不符合标准的项目再好也不投。

**打比方**：你是一个只吃素的餐厅老板。每天有 100 家食材供应商来推销，其中 99 家卖的是肉类——你不需要试吃就知道不合适。你只需要快速识别出那 1 家卖蔬菜的，然后仔细研究它的菜新不新鲜。项目初筛就是这个"快速识别"的过程。

**所以这意味着什么**：初筛的目标不是"找到好项目"，而是"快速排除不合适的项目"。**说"不"的速度越快，你花在真正有可能的项目上的时间就越多。**

---

### 2. CIM 和 Teaser——"卖方推销的两种武器"

**Teaser（项目简介 / 一页纸）**：
- 通常只有 1-2 页
- 包含：行业、大致收入/利润范围、为什么卖、联系方式
- **不透露公司名称**（匿名）
- 目的：引起买方兴趣，让你主动联系

**CIM（Confidential Information Memorandum，保密信息备忘录）**：
- 通常 30-80 页
- 包含：公司全名、详细财务数据、客户名单、管理层介绍、增长故事、投资亮点
- **签署保密协议（NDA）后才能拿到**
- 目的：让买方做初步判断，决定是否进入下一轮

**打比方**：
- Teaser = 相亲网站上的"个人简介"——只告诉你性别、年龄、爱好，不告诉你名字
- CIM = 签了保密协议后的"详细背景调查"——告诉你真实姓名、收入、家庭情况、为什么想找对象

**真实案例**：2023 年，一家中型 PE 基金一年收到了约 800 份 Teaser，其中约 200 份签了 NDA 拿到了 CIM，最终只投资了 2 个项目。从 Teaser 到投资的转化率只有 0.25%。

**所以这意味着什么**：本命令的核心输入就是 CIM 或 Teaser 文件。你需要从这些文件中快速提取关键信息，与基金的投资标准做比对，给出"Pass / 继续"的判断。

### 3. 投资标准（Investment Criteria）——"基金的择偶条件"

每个 PE 基金都有明确的投资标准，通常包括：

**① 行业偏好**：
- 专注行业的基金（如只看医疗、只看科技、只看消费）
- 通用型基金（什么行业都看，但通常有偏好）
- 为什么重要：PE 需要行业专长来做投后增值——如果你不懂这个行业，你买了公司也帮不上忙

**② 规模要求**：
- 收入门槛：通常 $50M-$500M
- EBITDA 门槛：通常 $10M-$100M
- 企业价值：通常 $100M-$2B
- 为什么重要：太小的公司不值得 PE 花时间管理，太大的公司 PE 买不起

**③ 财务特征**：
- 稳定现金流：EBITDA 利润率 > 15%
- 增长潜力：收入增长率 > 5%
- 可预测性：经常性收入占比高、客户留存率高
- 为什么重要：PE 需要用公司的现金流来还债——如果现金流不稳定，还债就成问题

**④ 交易类型**：
- 控股权交易（Buyout）：PE 要买 51%+ 的股权，获得控制权
- 少数股权投资（Growth Equity）：PE 只买 10-30% 的股权，不谋求控制权
- 为什么重要：不同交易类型的回报模型完全不同

**打比方**：投资标准就像"相亲条件"——你要找的是"30-40 岁、身高 170 以上、年收入 50 万以上、有房有车、性格稳定"的人。如果来的人是 20 岁的学生，不管多优秀，都不符合你的条件。

**所以这意味着什么**：初筛的第一步就是把项目的关键信息与基金的投资标准做"匹配度打分"。如果行业不对、规模太小、现金流不稳定——直接 Pass，不需要深入分析。

### 4. 初筛到底在筛什么——"五个红旗，一个就够"

初筛不是"全面分析"，而是"找红旗（Red Flags）"。只要发现一个红旗，就可以直接 Pass。

**五大红旗**：

**① 行业不匹配**：
- 基金只看 SaaS，来了一个制造业项目 → Pass
- 基金只看北美，来了一个东南亚项目 → Pass

**② 规模不达标**：
- 基金要求 EBITDA > $20M，项目 EBITDA 只有 $5M → Pass
- 基金要求收入 > $100M，项目收入只有 $30M → Pass

**③ 现金流不稳定**：
- 项目收入高度依赖前 3 大客户（占 80%）→ 客户一走就崩
- 项目是项目制收入（如咨询、定制开发）→ 没有经常性收入，不可预测

**④ 估值不现实**：
- 卖方要价 15x EBITDA，基金最多出 10x → 价格差距太大
- 卖方预期"战略性溢价"但实际没有战略买家 → 估值虚高

**⑤ 退出路径不明**：
- 行业在衰退，5 年后可能没有买家 → 退出困难
- 公司太依赖创始人，创始人不愿签竞业协议 → 创始人走了公司就完了

**打比方**：你买房，中介带你看 10 套房。你不需要每套都仔细检查——只要发现"漏水"（行业不匹配）、"产权不清"（估值不现实）、"学区不对"（规模不达标），就可以直接跳过。

**所以这意味着什么**：初筛是"排除法"——不需要证明项目好，只需要发现一个致命问题。这和尽职调查不同——DD 是"确认法"，需要证明项目确实没问题。

### 5. Inbound vs Outbound——"两种项目来源，两种策略"

**Inbound Deal（被动接收 / 案源送上门）**：
- 投行或中介主动发来的项目
- 优点：不需要自己找，省时省力
- 缺点：好项目通常不会"送上门"——真正好的项目，卖方会主动找顶级 PE 竞价，而不是广撒网
- 转化率：通常只有 1-3% 的 Inbound Deal 最终会投资

**Outbound Deal（主动出击 / 自行挖掘）**：
- PE 团队主动联系潜在目标公司
- 优点：可以找到"还没在卖"的公司——这些公司往往估值更低、竞争更少
- 缺点：需要大量人力和时间
- 转化率：通常 5-10% 的 Outbound Deal 最终会投资

**打比方**：
- Inbound = 相亲网站上的"被动匹配"——别人主动联系你，但质量参差不齐
- Outbound = 主动去社交场合认识人——费时费力，但遇到合适人选的概率更高

**真实案例**：顶级 PE 基金（如 KKR、Blackstone）的 Outbound Deal 占投资组合的 60-70%——因为最好的交易通常是"还没在市场上"的。中小型 PE 基金的 Inbound Deal 占比更高，因为缺乏 Outbound 的资源。

**所以这意味着什么**：本命令主要针对 Inbound Deal 的初筛。但 PE 基金不应该只依赖 Inbound——Outbound 才是获取高质量项目的关键。

---

### 6. 初筛输出——"不是写一篇论文，而是一页纸的判断"

初筛的输出通常是一页纸的**项目简报（One-Pager / Deal Summary）**，包含：

1. **公司概况**：名称、行业、规模、关键财务数据
2. **投资亮点**：为什么这个项目值得关注（2-3 条）
3. **红旗/顾虑**：发现了什么问题（1-3 条）
4. **与基金标准的匹配度**：行业匹配、规模匹配、估值匹配
5. **建议**：Pass / 继续研究 / 需要更多信息

**打比方**：初筛输出就像"餐厅点评"——不是写一篇美食论文，而是一句话："这家店环境好、菜品新鲜，但价格偏贵、停车位少。如果你不在乎价格，值得去。"

**所以这意味着什么**：初筛输出的价值不在于"分析有多深"，而在于"判断是否清晰"。一个明确的"Pass，因为行业不匹配"比一个含糊的"需要进一步研究"更有价值——因为它节省了团队的时间。

### 给小白的一句话

> **项目初筛不是"找好项目"，而是"快速排除不合适的项目"——就像你不会对每个相亲对象都深入了解，而是先看年龄、身高、收入这些硬条件，不符合的直接跳过。PE 基金每年看上千个项目，99% 都被 Pass 掉——这不是因为项目不好，而是因为不符合基金的投资标准。说"不"的速度越快，你花在真正有可能的项目上的时间就越多。**
