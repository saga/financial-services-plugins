## 金融术语和知识解释

### 1. 投资委员会备忘录 (Investment Committee Memo, IC Memo)

**定义**：投资委员会备忘录是私募股权公司内部用于向投资委员会汇报拟议投资的正式文件。它是决策过程中的核心文件，包含投资的所有关键信息。

**核心组成部分**：
| 部分 | 内容 | 重要性 |
|------|------|--------|
| 执行摘要 | 公司描述、交易理由、关键条款、建议和预期回报 | 高 |
| 公司概述 | 业务描述、产品/服务、客户基础、竞争定位 | 高 |
| 行业与市场 | 市场规模、竞争格局、长期趋势、监管环境 | 中 |
| 财务分析 | 历史业绩、收益质量调整、营运资本分析 | 高 |
| 投资论点 | 投资吸引力的支柱、价值创造杠杆 | 高 |
| 交易条款 | 企业价值、资金来源与用途、资本结构 | 高 |
| 回报分析 | IRR、MOIC、情景分析、敏感性分析 | 高 |
| 风险因素 | 风险排名、缓解措施、交易终止风险 | 高 |

**写作原则**：
- **事实性和平衡性**：诚实呈现看涨和看跌情景
- **风险透明**：不要低估风险，可信度至关重要
- **数字一致性**：确保财务表格相互匹配

### 2. 投资委员会 (Investment Committee, IC)

**定义**：投资委员会是私募股权基金的最高决策机构，负责批准或拒绝投资建议。

**组成**：
- 基金合伙人/管理人
- 行业专家
- 独立董事（部分基金）

**职责**：
- 审查投资提案
- 评估风险回报概况
- 批准投资决策
- 监督投资组合表现

### 3. 内部收益率 (Internal Rate of Return, IRR)

**定义**：IRR是衡量投资回报率的指标，考虑了资金的时间价值。它是使投资净现值(NPV)等于零的折现率。

**计算公式**：
```
NPV = Σ [CF_t / (1 + IRR)^t] = 0
```
其中 CF_t = 第t期的现金流

**解读**：
- **IRR > 门槛收益率(hurdle rate)**：投资可接受
- **IRR < 门槛收益率**：投资不可接受

**示例**：
如果一项投资投入100万美元，5年后收回300万美元，IRR约为24.57%。

### 4. 投资倍数 (Multiple on Invested Capital, MOIC)

**定义**：MOIC表示投资资本的回报倍数，即总收益与初始投资的比率。

**计算公式**：
```
MOIC = 总收益 / 初始投资
```

**与IRR的区别**：
| 指标 | 特点 | 关注点 |
|------|------|--------|
| IRR | 考虑时间价值 | 效率 |
| MOIC | 不考虑时间价值 | 规模 |

**示例**：投入100万美元，收回300万美元，则MOIC = 3x。

### 5. 资金来源与用途 (Sources & Uses)

**定义**：这是一份表格，详细说明交易的资金来源和用途。

**来源(Sources)**：
- 股权资本（GP和LP出资）
- 债务融资（银行贷款、债券）
- 管理层股权

**用途(Uses)**：
- 收购价格
- 交易费用
- 重组成本
- 流动资金

**重要性**：确保资金来源总额等于用途总额。

### 6. 企业价值 (Enterprise Value, EV)

**定义**：企业价值是公司的总价值，包括股权和债务。

**计算公式**：
```
EV = 股权价值 + 债务 - 现金及等价物
```

**用途**：用于计算估值倍数，如EV/EBITDA。

### 7. 收益质量调整 (Quality of Earnings Adjustments)

**定义**：对报告收益进行的调整，以反映真实的可持续盈利能力。

**常见调整项**：
- 一次性非经常性收入/费用
- 未实现收益/损失
- 管理层激励费用
- 关联方交易调整

**目的**：提供更准确的盈利预测基础。

### 8. 100天计划 (100-Day Plan)

**定义**：投资完成后的前100天行动计划，旨在快速创造价值。

**关键要素**：
- 管理层评估和调整
- 成本削减措施
- 战略计划制定
- 关键流程优化

**重要性**：早期行动往往决定投资的最终成败。

### 9. 敏感性分析 (Sensitivity Analysis)

**定义**：分析关键假设变化对投资回报的影响。

**常见分析维度**：
- 收入增长率变化
- EBITDA利润率变化
- 退出倍数变化
- 持有期变化

**输出**：展示不同情景下的IRR和MOIC范围。

### 10. 杠杆收购 (Leveraged Buyout, LBO)

**定义**：使用大量债务融资进行的收购，目标公司的资产作为贷款抵押。

**结构特点**：
- 高杠杆率（通常债务占60-80%）
- 目标公司现金流用于偿还债务
- 股权投资者追求高回报

**回报来源**：
1. 运营改善（EBITDA增长）
2. 债务偿还（权益增值）
3. 估值倍数扩张

## Appendix: 金融背景知识

这份文件是"投决会备忘录（IC Memo）"命令的详细说明。IC Memo 命令是 PE 项目团队"提交项目上会"时使用的标准化文档生成工具。

---

### 1. IC 的"仪式感"

在 PE 基金内部，IC 不是一个简单的会议——它是一个高度仪式化的决策流程：

1. **提前 2 周通知**——IC 秘书发出会议邀请
2. **材料提前 3 天送达**——每个 IC 成员必须提前看完
3. **现场汇报 30-60 分钟**——项目负责人做口头汇报
4. **问答环节 30-60 分钟**——IC 成员质询
5. **闭门讨论 15-30 分钟**——项目负责人退出
6. **投票表决**——"赞成" / "反对" / "弃权"

---

### 2. IC Memo 的"生死线"

IC Memo 不是普通的"分析报告"，它的使命是：**用 100 页纸说服 5 个聪明人投 5 个亿。**

**三大铁律：**
1. **逻辑必须自洽**——数字不能自相矛盾
2. **风险必须充分揭示**——不能藏着掖着
3. **结论必须明确**——必须给出"投"或"不投"的建议

---

### 3. IC 决策的"潜规则"

1. **没有 IC 成员会公开投反对票**——除非项目团队没有提前沟通
2. **IC 主席的意向通常就是最终结果**——所以要先搞定主席
3. **数据错误是致命伤**——如果 IC 发现数据有问题，项目基本被否
4. **事后追责**——如果项目亏了，写 IC Memo 的人要背锅

---

### 4. 真实案例：软银投资 Uber

2017 年软银以 680 亿美元估值投资 Uber。

| IC Memo 亮点 | IC Memo 风险 |
|-------------|-------------|
| 全球出行市场 TAM 超 5 万亿美元 | 创始人 Travis 被驱逐，公司治理混乱 |
| 中国市场已被滴滴收购（降低风险） | 全球市场面临监管打压 |
| 自动驾驶是未来 | 自动驾驶技术遥遥无期 |
| 70% 投资回报率 | 实际回报远低于预期 |

结果：Uber 2019 年 IPO 估值 820 亿美元，但股价长期低迷。

---

### 给小白的一句话

> IC Memo 命令就是 PE 基金的"考前押题"——项目团队花了 2-3 个月做尽调，最后用一份 100 多页的"标准答案"去说服"阅卷老师"（投决会）。但和高考不同的是，IC Memo 的"阅卷老师"不仅看你的答案，还会当面质问你"为什么选 A 不选 B"。如果你答不上来，或者答案里有明显错误，那这道题就"挂科"了。
