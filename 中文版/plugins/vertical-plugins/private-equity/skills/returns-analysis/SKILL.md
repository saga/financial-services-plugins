# 回报率分析

description: 为私募股权（PE）交易评估构建快速的 IRR/MOIC 敏感性分析表。在不同的入场乘数、杠杆水平、退出乘数、业绩增长率和持有期情境下进行回报建模。适用于初步评估交易规模、对假设条件进行压力测试或准备投资委员会（IC）的回报展示材料。触发指令包括（Triggers on）："returns analysis"、"IRR sensitivity"、"MOIC table"、"what's the return at"、"model the returns" 或 "back of the envelope"。

## 工作流程

### 步骤 1：收集交易输入数据

询问（或从先前的分析中提取）：

**入场假设（Entry）：**
- 入场 EBITDA（过去十二个月 LTM 或 未来十二个月 NTM）
- 入场乘数（企业价值 EV / EBITDA）
- 企业价值（EV）
- 交割时净债务（Net debt）
- 股权出资规模（Equity check size）
- 交易费用与支出

**融资安排（Financing）：**
- 优先债务（EBITDA 倍数、利率、摊销）
- 次级债务 / 夹层债务（如有）
- 入场总杠杆（EBITDA 倍数）
- 股权占比 / 股权贡献

**运营假设（Operating Assumptions）：**
- 营业收入增长率（年度）
- EBITDA 利润率轨迹
- 资本支出（Capex）占比（% 营业收入）
- 营运资金的变动
- 债务偿还计划

**退出假设（Exit）：**
- 持有期（年）
- 退出乘数（企业价值 EV / EBITDA）
- 退出 EBITDA（根据增长假设计算得出）

计算：

|--------|-------|

- 债务偿付（去杠杆）贡献
- 费用/支出拖累

**持有期 vs. 退出乘数**

### 步骤 5：输出内容

  - 假设条件标签页 (Assumptions)
  - 回报率计算标签页
  - 敏感性分析表（应用条件格式着色）
  - 情境总结标签页
- 适合用于投资委员会（IC）PPT 汇报的单页回报率总结

## 核心公式

- **MOIC（投资回报倍数）** = 退出股权价值 / 股权投资额
- **IRR（内部收益率）** = 求解 r：股权投资额 × (1 + r)^n = 退出股权价值（需根据期间现金流进行调整）
- **收益归因（Returns attribution）**：
  - 增长贡献：(退出 EBITDA - 入场 EBITDA) × 退出乘数 / 股权投资额
  - 乘数贡献：(退出乘数 - 入场乘数) × 入场 EBITDA / 股权投资额
  - 杠杆贡献：持有期间偿付的债务总额 / 股权投资额

## 重要注意事项

- 在适用情况下，务必同时展示扣除管理费与附带权益（Carry）前后的毛回报和净回报（Gross and net returns）。
- 管理层结转投资（Management rollover）和跟投（Co-invest）会改变实际的股权出资规模——需询问是否存在这种情况。
- 股息资本重组（Dividend recaps）或期间分配会显著影响 IRR——若有计划请务必纳入计算。
- 不要遗漏交易成本（通常为 EV 的 2-4%）——这会直接扣减第一天的股权价值。
- 税务考量（资产收购 vs 股权收购、338(h)(10) 税务选择）可能会对税后回报产生实质性影响。

### 1. LBO 赚钱的三个引擎——"增长、乘数、杠杆"

一笔 LBO 投资的回报来自三个"引擎"：

**引擎一：EBITDA 增长（增长贡献）**
- 你把公司经营得更好 → EBITDA 从 $10M 涨到 $15M
- 即使卖出的乘数不变（都是 8x），EV 也从 $80M 涨到 $120M
- **这是最"正当"的赚钱方式**——靠真本事

**引擎二：乘数扩张（乘数贡献）**
- 你买入时乘数是 8x，卖出时乘数是 10x
- 即使 EBITDA 不变（都是 $10M），EV 也从 $80M 涨到 $100M
- **这是最"看运气"的赚钱方式**——取决于市场环境

**引擎三：债务偿付（杠杆贡献）**
- 你买入时借了 $40M 的债，持有期间用公司利润还了 $20M
- 即使 EV 不变，你的股权价值也增加了 $20M
- **这是 LBO 最独特的赚钱方式**——用别人的钱帮你赚钱

**打比方**：LBO 就像"贷款买房出租"——
- 增长 = 房子翻修后租金涨了 → 更值钱
- 乘数扩张 = 房价整体涨了 → 同样的房子卖更贵
- 杠杆 = 用租金还贷款 → 你欠银行的钱少了，你的净资产就多了

**真实案例**：假设你以 8x EBITDA 买入一家 EBITDA $10M 的公司，总杠杆 4x（借了 $40M），5 年后以 8x EBITDA 卖出，EBITDA 增长到 $15M：
- 增长贡献：($15M - $10M) × 8x = $40M
- 乘数贡献：0（乘数没变）
- 杠杆贡献：5 年内还了 $20M 债务 = $20M
- 总回报 = $40M + $0 + $20M = $60M（股权从 $40M 变成 $100M，MOIC = 2.5x）

**所以这意味着什么**：收益归因瀑布图（Returns Waterfall）就是把这三个引擎的贡献拆开来看——让你知道这笔钱到底是"靠本事赚的"还是"靠运气赚的"。

### 2. MOIC vs IRR——"两个黄金指标，缺一不可"

**MOIC（Multiple on Invested Capital，投资回报倍数）** 和 **IRR（Internal Rate of Return，内部收益率）** 是衡量 PE 投资回报的两个核心指标。

**MOIC——看绝对金额**：
- MOIC = 退出股权价值 / 股权投资额
- 投了 $10M，拿回 $30M → MOIC = 3.0x
- **优点**：简单直观，不受持有期影响
- **缺点**：不考虑时间——3 年赚 3x 和 10 年赚 3x，MOIC 一样

**IRR——看时间效率**：
- IRR 是让"投入的现值 = 回收的现值"的那个折现率
- 同样是 3.0x MOIC：3 年做到 → IRR ≈ 44%；10 年做到 → IRR ≈ 11%
- **优点**：考虑了资金的时间价值——同样的回报，越快越好
- **缺点**：对中间现金流非常敏感——如果中途有分红（Dividend Recap），IRR 会被大幅拉高

**为什么两个都要看**：
- 只看 MOIC → 可能投了 15 年才赚 3x，资金效率极低
- 只看 IRR → 可能投了 $1M 赚 $0.5M，IRR 很高但绝对金额很小
- **PE 行业标准**：好的投资通常要求 MOIC > 2.5x 且 IRR > 20%

**打比方**：
- MOIC = 你跑了多少公里 → 衡量总量
- IRR = 你的平均配速 → 衡量效率
- 跑 42 公里（MOIC 高）但用了 10 小时（IRR 低）→ 完赛了但很慢
- 跑 5 公里（MOIC 低）但用了 20 分钟（IRR 高）→ 很快但距离短

**真实案例**：
| 情景 | 投入 | 回收 | 持有期 | MOIC | IRR |
|------|------|------|--------|------|-----|
| A | $10M | $30M | 3 年 | 3.0x | 44% |
| B | $10M | $30M | 7 年 | 3.0x | 17% |
| C | $10M | $20M | 2 年 | 2.0x | 41% |

情景 A 和 B 的 MOIC 一样，但 A 的 IRR 远高于 B——因为 A 更快。情景 C 的 IRR 接近 A，但 MOIC 只有 2.0x——绝对金额少。

**所以这意味着什么**：本技能要求"同时展示 IRR 和 MOIC"——因为只看一个指标会误导决策。

### 3. 敏感性分析——"PE 的防身术"

**敏感性分析（Sensitivity Analysis）** 是通过改变一个或多个假设条件，观察回报指标如何变化。它是 PE 投资决策中最重要的"防身术"。

**为什么需要敏感性分析**：
- 所有 LBO 模型的假设都是"猜的"——没人能准确预测未来
- 如果你的模型只在一种假设下好看，那这个投资就很危险
- 敏感性分析告诉你："在最坏的情况下，我还能赚钱吗？"

**四种常见的敏感性矩阵**：

**① 入场乘数 vs 退出乘数**：最基础——"如果我买贵了，还能赚钱吗？"
- 如果入场 10x / 退出 8x（乘数收缩），MOIC 还能 > 1.5x 吗？
- 如果不能 → 这笔交易太依赖"乘数扩张"，风险很大

**② EBITDA 增长 vs 退出乘数**：最实用——"增长和估值哪个更重要？"
- 如果低增长 + 高退出乘数才能赚钱 → 你在赌市场环境
- 如果高增长 + 低退出乘数也能赚钱 → 你在靠真本事

**③ 杠杆水平 vs 退出乘数**：最关键——"杠杆是朋友还是敌人？"
- 高杠杆 + 退出乘数下降 → 可能亏损（杠杆放大损失）
- 低杠杆 + 退出乘数下降 → 亏损较小（杠杆低，风险小）

**④ 持有期 vs 退出乘数**：最现实——"时间站在我这边吗？"
- 持有期越长，IRR 越低（时间拖累回报）
- 但持有期越长，EBITDA 可能增长越多（时间帮助增长）

**打比方**：敏感性分析就像"压力测试"——你买车之前，不仅看"正常路况"下的油耗，还要看"高速"、"堵车"、"开空调"等各种情况下的油耗。如果只有理想路况才省油，那这车就不靠谱。

**真实案例**：一家 PE 在分析某工业公司时，基准情形 MOIC = 3.0x / IRR = 25%。但敏感性分析显示：如果退出乘数从 8x 降到 6x，MOIC 就降到 1.8x / IRR = 12%——低于 PE 的最低要求。最终 PE 决定不投，因为"对退出乘数的依赖太大"。

**所以这意味着什么**：本技能要求"构建双向敏感性矩阵"——这是 PE 投资决策的必备工具，不是可选项。

### 4. 情境分析——"乐观、基准、悲观三重奏"

**情境分析（Scenario Analysis）** 与敏感性分析不同——敏感性分析一次只变一两个变量，情境分析则同时改变多个变量，构建"完整的故事"。

**三种情境**：

**乐观情形（Bull Case）**：
- 一切都按最好的方向发展
- 增长超预期、利润率提升、退出乘数扩张
- **用途**：看看"天花板"在哪里——最好的情况能赚多少

**基准情形（Base Case）**：
- 最可能发生的情况
- 增长符合预期、利润率稳定、退出乘数与入场持平
- **用途**：这是投资决策的核心依据——基准情形必须能赚钱

**悲观情形（Bear Case）**：
- 一切都按最差的方向发展
- 增长不及预期、利润率下滑、退出乘数收缩
- **用途**：看看"地板"在哪里——最差的情况亏多少，能不能承受

**打比方**：情境分析就像"出门看天气预报"——
- 乐观 = 晴天 → 穿短袖就行
- 基准 = 多云 → 带件外套
- 悲观 = 暴雨 → 必须带伞
- 如果悲观情形是"龙卷风" → 你可能根本不该出门（不投）

**关键判断标准**：
- 如果**悲观情形**的 MOIC > 1.5x → 安全边际大，值得投
- 如果**基准情形**的 MOIC < 2.0x → 回报不够，可能不值得
- 如果**乐观情形**的 MOIC < 3.0x → 天花板太低，没有想象空间

**所以这意味着什么**：本技能要求"构建 3 种情境"——这是 PE 投资委员会（IC）做决策的核心依据。

### 5. 毛回报 vs 净回报——"你看到的和你拿到的差很多"

**毛回报（Gross Returns）** 是扣除管理费和业绩分成之前的回报。**净回报（Net Returns）** 是 LP 实际拿到手的回报。两者差距巨大。

**PE 的费用结构**：
- **管理费（Management Fee）**：通常每年 2%（按承诺资本计算）
- **业绩分成（Carried Interest / Carry）**：通常 20%（超过门槛回报的部分）

**费用对回报的影响**：
- 假设一笔投资毛 MOIC = 3.0x，持有 5 年
- 管理费：5 年 × 2% = 10% 的资本被吃掉
- 业绩分成：利润的 20% 被拿走
- **净 MOIC 可能只有 2.2x-2.4x**——比毛回报少了 20-25%

**为什么这很重要**：
- PE 向 LP 报告的是**净回报**——因为 LP 关心的是"我实际赚了多少"
- 但 PE 内部分析时通常看**毛回报**——因为管理费是基金层面的，不是单个项目层面的
- **投资委员会（IC）** 通常要求同时看毛回报和净回报

**打比方**：毛回报就像"税前工资"，净回报就像"税后工资"——你看到税前工资很高，但到手可能少了很多。PE 的"税率"（管理费 + Carry）大约是 25-30%。

**所以这意味着什么**：本技能要求"同时展示毛回报和净回报"——因为 LP 只关心净回报，而 PE 内部需要看毛回报来评估投资团队的能力。

### 6. 股息资本重组（Dividend Recap）——"PE 的'作弊'技巧"

**股息资本重组（Dividend Recapitalization）** 是 PE 在持有期间让目标公司再次借债，用借来的钱给自己发分红。这是 PE 提高 IRR 的"秘密武器"。

**Dividend Recap 怎么运作**：
1. PE 买入公司，借了 $40M
2. 3 年后，公司业绩变好，可以借更多债
3. PE 让公司再借 $15M，用这 $15M 给 PE 发分红
4. PE 提前回收了 $15M 现金 → IRR 大幅提升

**为什么 Dividend Recap 能提高 IRR**：
- IRR 对"提前回收现金"非常敏感
- 如果你在第 3 年就回收了一部分投资，IRR 会显著提升
- 即使最终的 MOIC 不变，IRR 也会更高

**Dividend Recap 的风险**：
- 公司的债务更多了 → 财务风险更大
- 如果经济下行，公司可能还不起债 → 破产风险
- 公司的信用评级可能下降 → 未来融资成本更高

**打比方**：Dividend Recap 就像"把房子的增值部分再抵押出来花"——你买了房子，3 年后房价涨了，你再去银行多借点钱出来花。你的现金流变好了，但房子的贷款也更多了。如果房价下跌，你可能还不起贷款。

**真实案例**：2005-2007 年，很多 PE 基金在买入公司后 1-2 年就做 Dividend Recap，有些甚至回收了全部投资额（MOIC = 1x，但 IRR 可能高达 50%+）。2008 年金融危机时，这些高杠杆公司大量破产——因为债务太多，收入一下降就还不起了。

**所以这意味着什么**：本技能提醒"股息资本重组或期间分配会显著影响 IRR——若有计划请务必纳入计算"——因为忽略 Dividend Recap 会让你的 IRR 预测严重失真。

### 给小白的一句话

> **LBO 赚钱靠三个引擎：增长（靠本事）、乘数（靠运气）、杠杆（用别人的钱）。MOIC 看绝对金额，IRR 看时间效率——两个都要看。敏感性分析是"防身术"——告诉你最坏情况还能不能活。情境分析是"三重奏"——乐观/基准/悲观，基准情形必须能赚钱。毛回报和净回报差 25-30%——LP 只关心净回报。Dividend Recap 能提高 IRR，但也增加风险。记住：好的 PE 投资，基准情形 MOIC > 2.5x 且 IRR > 20%，悲观情形 MOIC > 1.5x。**
