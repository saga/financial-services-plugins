# 并购模型

您是并购建模专家。构建增厚/稀释分析，模拟不同收购结构和假设对收购方每股收益的影响。

## 核心原则

增厚/稀释分析回答一个核心问题：这笔交易对收购方的每股收益（EPS）是增加还是减少？增厚 = EPS 增加，稀释 = EPS 减少。稀释不一定是坏交易——如果战略理由充分且稀释是暂时的。模型应测试多种情景。

## 模型结构

### 1. 输入假设
- 收购方和目标公司财务数据（收入、EBITDA、净利润、EPS、流通股数）
- 交易结构（全现金、全股票或混合）
- 购买价格和溢价
- 融资假设（债务利率、新股发行价格）
- 协同效应（成本协同 + 收入协同）
- 交易费用和整合成本

### 2. 计算
- 交易总价值 = 购买价格 × 目标完全稀释股数 + 承担净债务
- 资金来源：可用现金、新债务融资、新股发行
- 备考合并利润表（调整后净利润）
- 备考每股收益
- 增厚/稀释百分比

### 3. 敏感性分析
| 购买溢价 | 协同效应水平 | 融资方式 | 增厚/稀释 |
|---|---|---|---|
| +20% | 低 | 全现金 | ... |
| +20% | 中 | 全股票 | ... |
| +30% | 中 | 混合 | ... |

## 关键指标

- **增厚/稀释率%**：备考 EPS / 收购方独立 EPS - 1。正 = 增厚
- **净现值（NPV）**：未来增厚现金流的现值
- **内部收益率（IRR）**：收购方的投资回报率
- **投资回收期**：需要多少年收回交易溢价

## 方法论

始终使用现金/股票混合假设建立基准情景，然后测试极端情况。敏感性分析应覆盖购买价格、协同效应和融资成本的变化。

## Appendix: 金融背景知识

这份文件是"并购模型技能（Merger Model Skill）"的详细说明。并购模型（Merger Model）是投行分析师面试中最常被问到的技术问题，也是实际工作中使用频率最高的财务模型之一。它的核心功能是：当一家公司（买方/Acquirer）想要收购另一家公司（目标/Target）时，计算这笔交易对买方每股收益（EPS）的影响——是增厚（Accretive）还是摊薄（Dilutive）。

---

### 1. 并购模型解决的核心问题

假设你是一家上市公司的 CFO，董事会问你："我们要花 50 亿美元收购 XYZ 公司。这笔交易做完之后，我们每股能赚多少钱？比不收购的时候多还是少？"

这就是并购模型要回答的问题。它把两家公司合并后的财务数据"模拟"出来，然后计算合并后的 EPS，和买方原来的 EPS 做对比。

**核心公式：**
```
合并后 EPS = (买方净利润 + 目标净利润 + 协同效应 - 新增利息费用 - 新增折旧摊销) 
              ÷ (买方原有股数 + 新增发行股份数)
```

---

### 2. 增厚（Accretion）vs 摊薄（Dilution）——最核心的概念

这是并购模型中最重要的概念，也是投行面试 100% 会问的问题。

#### 2.1 用最通俗的例子理解

假设你和你朋友各自经营一家奶茶店：

**你的奶茶店（买方）：**
- 每年净利润：100 万元
- 发行在外的股份：100 万股
- 每股收益（EPS）：100万 ÷ 100万 = **1.00 元/股**

**朋友的奶茶店（目标）：**
- 每年净利润：30 万元
- 发行在外的股份：50 万股
- 每股收益（EPS）：30万 ÷ 50万 = **0.60 元/股**

现在你想把朋友的店买下来，有两种付款方式：

**方案 A：全现金收购（借钱买）**
- 你花 500 万元现金买下朋友的店（假设全部从银行借，年利率 5%）
- 每年新增利息支出：500万 × 5% = 25 万元
- 合并后净利润：100万 + 30万 - 25万利息 = **105 万元**
- 你的股份没有增加（没发新股）= **100 万股**
- 合并后 EPS：105万 ÷ 100万 = **1.05 元/股** ✅

**结论**：收购前 EPS 是 1.00，收购后是 1.05 → **增厚了 5%（Accretive）**

**方案 B：全股票收购（发新股买）**
- 你不花现金，而是发行 50 万股新股给朋友作为对价
- 合并后净利润：100万 + 30万 = **130 万元**（没有新增利息）
- 你的股份增加了：100万 + 50万 = **150 万股**
- 合并后 EPS：130万 ÷ 150万 = **0.87 元/股** ❌

**结论**：收购前 EPS 是 1.00，收购后是 0.87 → **摊薄了 13%（Dilutive）**

#### 2.2 为什么全现金通常增厚，全股票通常摊薄？

| 付款方式 | 对 EPS 的影响 | 原因 |
|---------|-------------|------|
| 全现金 | 通常增厚 | 虽然要付利息，但股份不增加。如果目标公司的利润率高于借款利率，就能增厚 |
| 全股票 | 通常摊薄 | 虽然不用付利息，但股份大幅增加。如果目标公司的 EPS 低于买方，就会被摊薄 |
| 混合 | 取决于比例 | 现金比例越高越可能增厚，股票比例越高越可能摊薄 |

**关键洞察**：目标公司的"估值倍数"和买方的"估值倍数"之间的关系决定了增厚还是摊薄。

- 如果目标公司便宜（低 P/E）→ 全股票也可能增厚（因为"便宜货"带来的利润增长超过了股份稀释）
- 如果目标公司贵（高 P/E）→ 全股票几乎一定摊薄（因为"贵货"带来的利润增长不够弥补股份增加）

### 3. 并购模型的三大核心假设

#### 3.1 购买价格（Purchase Price）和溢价（Premium）

```
购买价格 = 目标公司当前市值 × (1 + 溢价率)
```

| 溢价率 | 市场含义 | 典型场景 |
|-------|---------|---------|
| 10-20% | 温和溢价 | 友好协商收购 |
| 25-35% | 标准溢价 | 大多数正常交易 |
| 40-50% | 激进溢价 | 竞争性竞标或恶意收购 |
| 50%+ | 过高溢价 | 通常引发买方股东抗议 |

**为什么要溢价？** 因为上市公司的股票在市场上公开交易，你不可能按昨天的收盘价把整个公司买下来——现任股东不会同意。你必须出一个比市场价高得多的"一口价"，才能让足够多的股东投票同意出售。

#### 3.2 融资结构（Sources & Uses）

这是并购模型中最经典的表格——"资金来源与用途表"。

**Uses（钱花在哪儿）：**
| 用途 | 金额 |
|------|------|
| 购买目标公司股权 | $4.5B |
| 偿还目标公司现有债务 | $0.5B |
| 交易费用（投行费、律师费等） | $0.2B |
| **总用途** | **$5.2B** |

**Sources（钱从哪儿来）：**
| 来源 | 金额 |
|------|------|
| 买方自有现金 | $1.0B |
| 新增银行贷款 | $2.5B |
| 发行债券 | $1.0B |
| 发行新股 | $0.7B |
| **总来源** | **$5.2B** |

**关键原则**：Sources 必须等于 Uses（会计恒等式）。

#### 3.3 协同效应（Synergies）

协同效应是并购案号称"这笔交易值得做"的核心理由。分为两类：

**成本协同（Cost Synergies）：**
- 裁员重复岗位（两个公司合并后不需要两个 CFO）
- 统一采购（更大的采购量带来更低的单价）
- 共享基础设施（办公室、IT 系统、物流网络）
- 历史数据显示：成本协同通常有 50-70% 的实现概率

**收入协同（Revenue Synergies）：**
- 交叉销售（A 公司的客户可以卖 B 公司的产品）
- 新产品开发（两家公司的技术合并后产生新产品）
- 地理扩张（利用对方的渠道进入新市场）
- 历史数据显示：收入协同通常只有 20-30% 的实现概率（因为"多赚钱"比"少花钱"难得多）

**协同效应的实现时间表：**
- 第一年：通常只能实现 20-30% 的协同效应
- 第二年：实现 50-60%
- 第三年及以后：实现 80-100%

---

### 4. 并购模型的输出：敏感性分析

一个好的并购模型不会只给一个数字，而是会给一张"敏感性分析表"，展示在不同假设下结果如何变化。

**示例：EPS 增厚/摊薄敏感性分析**

| 协同效应实现率 \ 溢价率 | 20% | 30% | 40% | 50% |
|------------------------|-----|-----|-----|-----|
| **0%**（无协同） | -2% | -5% | -8% | -12% |
| **50%**（一半协同） | +1% | -2% | -5% | -8% |
| **100%**（全部协同） | +5% | +2% | -1% | -4% |

**解读**：如果溢价 30% 且协同效应实现 100%，EPS 增厚 2%；但如果溢价 50% 且协同效应只实现 50%，EPS 摊薄 8%。

---

### 5. 真实案例：迪士尼收购 21 世纪福克斯

2019 年，迪士尼以 713 亿美元收购了 21 世纪福克斯的大部分资产。让我们用并购模型的视角来分析：

**交易参数：**
- 购买价格：$713 亿
- 付款方式：混合（股票 + 现金）
- 迪士尼当时的 P/E：约 15x
- 福克斯的 P/E：约 12x（被收购前）

**为什么这笔交易是增厚的？**
1. 福克斯的 P/E（12x）低于迪士尼的 P/E（15x）→ 用"贵股票"买"便宜资产"，天然增厚
2. 迪士尼预计每年能产生 $20 亿的协同效应（主要来自内容库整合和成本削减）
3. 迪士尼的信用评级高，能以较低利率融资

**结果**：交易完成后，迪士尼的 EPS 在第一年就增厚了约 7-10%。

---

### 给小白的一句话

> 并购模型就是帮上市公司算"娶媳妇划不划算"的 Excel 计算器。核心问题只有一个：娶了她以后，家里每个人（每股）能分到的钱是变多了（增厚/Accretive）还是变少了（摊薄/Dilutive）。如果变多了，董事会大概率会批准；如果变少了，除非有特别强的战略理由，否则股东们会把你骂得狗血淋头。投行面试 100% 会问这个，因为这是并购分析中最核心、最基础、也最能看出一个人是否懂财务逻辑的问题。
