# 公司摘要（Teaser）生成

您是并购营销专家，负责起草公司摘要（Teaser）。Teaser 是出售流程中发送给潜在买家的第一份文件——它必须引发兴趣，同时隐藏公司身份，直到签署 NDA。

## 核心原则

Teaser 是一份"诱饵"文件。它展示公司的吸引力（规模、增长、行业、利润），但隐藏能识别公司的具体信息。目标是让足够多的买家签署 NDA。

## Teaser 结构

### 1. 交易摘要
- 行业
- 大致收入规模（如 $50-75M）
- 大致 EBITDA 规模（如 $10-15M）
- 地理位置
- 交易类型（股权出售/资产出售/资本重组）

### 2. 公司亮点
- 业务模式描述（不透露公司名称）
- 竞争定位
- 增长驱动因素
- 客户和供应商多样性

### 3. 财务概览
匿名化的关键财务指标：
| 指标 | FY-2 | FY-1 | 最新 LTM |
|---|---|---|---|
| 收入 ($M) | ~XX | ~XX | ~XX |
| EBITDA ($M) | ~XX | ~XX | ~XX |
| EBITDA 利润率 | ~X% | ~X% | ~X% |

### 4. 交易理由
为什么这是个好机会——成长潜力、市场地位、管理团队质量。

## 格式说明

- 不超过 1 页
- 不提及公司名称、品牌、具体客户或具体数字
- 使用"约"（~）修饰财务数据
- 专注于讲述投资故事，而非罗列数据

## Appendix: 金融背景知识

这份文件是"匿名项目简介（Teaser）"技能的详细说明。在投行的并购流程中，Teaser 是卖方投行发给潜在买家的第一份正式文件——它的角色就像追女孩子时发的第一封"暧昧情书"：不能直接告诉对方你是谁，但要足够吸引人，让对方忍不住想进一步了解你。Teaser 写得好坏，直接决定有多少买家会签 NDA（保密协议）进入下一轮。

---

### 1. Teaser 在整个并购流程中的位置

想象你开了一家非常赚钱的甜品店，现在你想把这家店卖掉。你找到了投行帮忙。整个卖店流程大概是这样的：

```
第一步：准备材料
├── Teaser（匿名简介）：1-2页，不说店名
├── CIM（机密备忘录）：50-100页，签了NDA才能看
└── Management Presentation：管理层亲自路演

第二步：接触买家
├── 第一轮：发 Teaser → 感兴趣的买家签 NDA → 发 CIM
├── 第二轮：收到初步报价（IOI） → 筛选 → 尽调（DD）
└── 第三轮：最终报价 → 签合同 → 成交
```

**Teaser 是第一块敲门砖**。如果你的 Teaser 写得像报税单一样无聊，那就没人会签 NDA；如果 Teaser 写得像悬疑小说的预告片，那买家就会蜂拥而至。

---

### 2. 为什么必须匿名？——保密的艺术

这是一个经典的"谈判博弈"问题。

假设你是"老王餐饮集团"的老板，你想卖掉你旗下一家子公司。如果全世界都知道你要卖公司，会发生什么？

- **你的员工会恐慌**："公司要卖了，我是不是要失业了？" → 核心员工开始投简历
- **你的客户会担心**："换了老板，合同还作数吗？" → 大客户可能不再续约
- **你的供应商会收紧**："这家公司马上换老板了，先不给账期了。"
- **你的竞争对手会攻击你**："老王不行了，要卖资产了，快抢他的客户！"
- **你的股价会暴跌**：如果是上市公司，"出售资产"的消息可能被市场解读为"公司不行了"

所以，在第一轮接触买家的时候，必须**匿名**——用代号、用行业描述、用模糊的数字范围，让买家只能看到"这是一家好公司"，但绝对无法推断出"这家公司到底是哪家"。

**真实案例**：2018年，一家全球知名的消费品巨头在出售旗下某个亚洲业务单元时，Teaser 上写的是"A leading premium dairy brand in Southeast Asia with >$500M revenue"。所有线索都可以指向好几家公司——买家无法确定到底是哪一家，只知道"这规模不小，值得签 NDA 看看"。

### 3. Teaser 的黄金法则：钓胃口但不暴露

| 应该说的 | 不应该说的 |
|---------|-----------|
| "北美前三的医疗器械分销商" | ABC Medical Supplies Inc. |
| "年收入 $100-150M" | "2023年收入 $127.3M" |
| "在中西部拥有100+门店" | "在伊利诺伊州芝加哥市有113家门店" |
| "服务于财富500强客户" | "主要客户包括沃尔玛和可口可乐" |
| "EBITDA利润率 > 25%" | "EBITDA $34.2M, 利润率 26.8%" |
| "项目代号：Project Eagle" | "老王餐饮集团子公司出售" |

基本原则：**说出"吸引力"，隐藏"身份"。**

### 4. 一个好的 Teaser 应该具备什么？

#### 4.1 交易摘要（Transaction Summary）
给买家一个"快照"：
- 行业（够具体但不够辨识）
- 收入范围（模糊化处理，比如 "$200-250M" 而不是 "$237M"）
- 地理位置（够大到无法定位，比如"东南亚"而不是"泰国曼谷"）
- 交易类型（股权出售/资产出售/融资）
- 项目代号（Project Eagle / Project Phoenix 等）

#### 4.2 投资亮点（Investment Highlights）
3-5 条让买家心动的理由：
- "拥有超过15年的运营历史和行业中最高客户留存率（>95%）"
- "在新兴市场中占据领导地位，该市场未来5年预计增长30% CAGR"
- "轻资产运营模式，资本支出仅占收入的2%"
- "创始人寻求战略合作伙伴以加速国际扩张"

#### 4.3 财务概览（Financial Overview）
匿名但诱人的数字：
- 展示3年的收入/EBITDA趋势——但数字模糊化处理
- 强调利润率高于行业平均
- 展示增长趋势

```
财务概览（近似值）
┌──────────┬───────┬───────┬──────┐
│          │ FY1   │ FY2   │ LTM  │
├──────────┼───────┼───────┼──────┤
│ 收入     │ ~$60M │ ~$80M │ ~$95M│
│ EBITDA   │ ~$15M │ ~$22M │ ~$28M│
│ 利润率   │ ~25%  │ ~28%  │ ~30% │
└──────────┴───────┴───────┴──────┘
```

#### 4.4 交易逻辑（Transaction Rationale）
为什么要卖？买家为什么应该买？
- 常见卖的理由："创始人年事已高，无子女愿意接班"（退休/退出逻辑）
- "公司发展到了一个阶段，需要大平台的资源才能进一步增长"
- "第一代投资人的基金已经到期，需要退出变现"
- "寻求战略合作伙伴以加速产品创新和市场扩张"

---

### 5. Teaser 的常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 透露公司真实名称 | 消息泄露，员工恐慌，竞争对手攻击 | 使用项目代号 |
| 给出精确数字 | 买家可以通过公开数据反推公司身份 | 使用范围或近似值 |
| 写得太长 | 买家没耐心看完，直接跳过 | 控制在1-2页 |
| 过于谦虚 | 买家觉得"这公司没什么特别的" | 突出亮点，但保持诚实 |
| 过于夸张 | 买家签 NDA 后发现差距太大，失去信任 | 适度包装，不造假 |
| 没有明确的下一步 | 买家不知道该怎么联系你 | 提供清晰的联系方式和下一步流程 |

### 给小白的一句话

> Teaser 就是投行帮老板卖公司时发的"匿名征婚广告"——上面写着"某优质单身男，38岁，年收入500万，有房有车，爱运动爱旅行"，但绝不会写上他的真实名字和工作单位。只有那些真心感兴趣并且签了保密协议的人，才能拿到他的详细资料和联系方式。写得好，能让100个潜在买家里有30个愿意进一步了解；写得烂，可能只剩下3个好奇的。
