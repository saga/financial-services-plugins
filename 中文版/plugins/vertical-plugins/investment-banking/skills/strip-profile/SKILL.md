# 公司一页纸画像 (Strip Profile)

本技能的目标是实现**极高的信息密度**。一份优秀的 Strip Profile 应该能让忙碌的高管在 30 秒内完整理解一家公司的核心故事、财务状况和市场地位。

## 核心要求

### 1. 布局原则：四象限模型
除非用户另有要求，否则默认使用经典的四象限布局：
- **象限 1 (左上)**：**公司概览 (Company Overview)** —— 总部、创立时间、关键运营数据、核心高管、股票代码等。
- **象限 2 (右上)**：**业务与定位 (Business & Positioning)** —— 收入驱动因素、产品组合、市场份额、竞争护城河。
- **象限 3 (左下)**：**核心财务与估值 (Financials & Valuation)** —— 收入、EBITDA、利润率、市值、倍数（EV/EBITDA）等。
- **象限 4 (右下)**：**市场表现或最新进展 (Stock / Developments)** —— 股价走势图（公有公司）或主要股东及并购历史（私有公司）。

### 2. 信息压缩技术
- **合并事实**：例如“总部：德州奥斯汀；创立于 2003 年；员工 14.5 万人”。
- **量化描述**：避免使用“巨大的收入”，而应使用“514 亿美元收入 (+6.0% YoY)”。
- **上下文对比**：例如“EBITDA 利润率：25%（行业平均：18%）”。

### 3. 创建流程 (Workflow)
**关键：必须根据用户反馈逐页创建。**
1. **研究与策划**：通过 SEC 文件（BamSEC）、投资者推介会（Investor Deck）收集数据。
2. **品牌对齐**：必须通过 web 搜索确认该公司的实际品牌标准色（如可口可乐红、IBM 蓝），禁止猜测。
3. **视觉校验**：生成 PPT 后，**必须**将其转换为图片进行视觉审查，检查是否存在文字重叠（Overlap）或被切断（Cutoff）的情况。

## 视觉标准
- **比例**：必须使用 **4:3** 标准（投行投标书通用比例）。
- **字体大小**：标题 24pt，象限标题 14pt（带品牌色侧边装饰条），正文 11pt，注脚 8pt。
- **严禁事项**：严禁在正文中使用大段段落，必须全部使用列表（Bullet points）。

> **💡 Appendix: 领域知识小贴士**
>
> 1. **Strip Profile (公司画像/剥带图)**：在投行里，这也被戏称为“一页纸”。它的存在的意义就是为了“快”。当一个资深银行家在路上，或者在会议间隙想了解一个从未听过的潜在客户时，他没有时间看 50 页的年报，这一页纸就是他全部的信息来源。
> 2. **30 秒 Comprehension Test (三十秒理解测试)**：这是 Strip Profile 的金标准。如果一个懂金融的人看这页 PPT 超过 30 秒还没弄清楚这家公司是卖什么的、赚不赚钱，那这就是一份不及格的投行文档。
> 3. **信息密度 (Information Density)**：在金融圈，空白往往意味着“懒惰”或“搜集数据不力”。我们要尽可能像填鸭一样把高价值的数字塞进去。但难点在于，塞满的同时还要保证整洁对齐，这就是投行分析师的功力所在。
> 4. **EV/EBITDA**：这是一个非常常用的估值倍数。EV 是企业价值（公司身价），EBITDA 是息税折旧摊销前利润。简单说，这个倍数告诉你“如果要买下这家公司，需要花它几年赚到的这种钱”。

## Appendix: 金融背景知识

这份文件是"公司简介生成（Strip Profile）"技能的详细说明。在投行和 PE 中，Strip Profile（也叫 Company Profile 或 Tear Sheet）是一家公司的"一页纸简历"——用最精简的格式呈现公司的核心信息。

---

### 1. 什么是 Strip Profile？

**类比：**
想象你去相亲，对方递给你一张"个人简介"：姓名、年龄、身高、学历、工作、收入、兴趣爱好。1 分钟看完，你对这个人就有了基本了解。

Strip Profile 就是公司的"相亲简历"——1-2 页纸，让投资人在 2 分钟内了解一家公司。

---

### 2. Strip Profile 的"标准内容"

| 模块 | 内容 |
|------|------|
| 基本信息 | 公司名、股票代码、上市地、行业 |
| 业务概述 | 公司做什么、主要产品/服务 |
| 财务摘要 | 收入、利润、利润率、增长率 |
| 估值指标 | P/E、EV/EBITDA、P/B |
| 股价表现 | 52 周最高/最低、年初至今涨幅 |
| 关键数据 | 员工数、市值、企业价值 |
| 投资亮点 | 3-5 个核心投资逻辑 |
| 风险因素 | 3-5 个主要风险 |

---

### 3. Strip Profile 的"使用场景"

1. **初步筛选**——快速判断"值不值得深入看"
2. **内部沟通**——向合伙人/投决会介绍公司
3. **客户汇报**——向 LP 汇报被投公司情况
4. **行业研究**——快速了解竞争对手

---

### 4. 真实案例：苹果 Strip Profile

| 项目 | 数据 |
|------|------|
| 公司名 | Apple Inc. |
| 股票代码 | AAPL（NASDAQ） |
| 行业 | 科技/消费电子 |
| 市值 | ~3 万亿美元 |
| 年收入 | ~3800 亿美元 |
| 净利润 | ~1000 亿美元 |
| P/E | ~30x |
| ROE | ~160% |
| 投资亮点 | 品牌护城河、现金流强劲、服务业务增长 |
| 风险因素 | iPhone 依赖、中国市场风险、监管压力 |

---

### 给小白的一句话

> Strip Profile 就是金融圈的"身份证"——2 页纸告诉你"这家公司是干什么的、赚多少钱、值多少钱、有什么风险"。对于专业投资人来说，Strip Profile 是"快速筛选工具"——1 分钟看完就能判断"值不值得花更多时间研究"。对于金融小白来说，Strip Profile 是"入门地图"——帮你把复杂的公司信息简化成"能看懂的关键数字"。
