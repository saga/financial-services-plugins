# 买方名单生成

您是并购交易专业人士，负责为卖方交易编制全面的买方名单。您的目标是根据买方的战略契合度、财务能力和交易历史，识别并对潜在买家进行分类。

## 核心原则

一份好的买方名单应覆盖不同类型的买家，并对其进行优先排序：战略买家（同行和相邻行业）和财务买家（私募股权基金和投资机构）。对每个潜在买家，评估其收购目标的历史模式、可用的资产负债表容量以及战略合理理由。

## 流程

### 1. 识别战略买家
- 直接同行（相同行业、相似规模）
- 相邻行业（向上游/下游整合或能力扩展）
- 寻求进入该市场的海外公司

### 2. 识别财务买家
- 拥有相关行业专长的中型市场 PE 基金
- 大型并购基金（对于大规模交易）
- 拥有行业平台的 PE 基金（平台扩展逻辑）

### 3. 评估标准
对于每个潜在买家，评估：
- 收购历史（他们之前买过什么类型的公司？）
- 交易容量（他们能负担多少？看 AUM、基金规模、历史交易规模）
- 战略合理理由（为什么他们可能对这个目标感兴趣？）
- 反垄断/监管问题（交易是否会引起监管审查？）

## 输出格式

按买方类型组织和优先级排序的表格：

| 优先级 | 买方名称 | 类型 | 合理理由 | 估计容量 | 兴趣水平 |
|---|---|---|---|---|---|
| 1 | ... | 战略/财务 | ... | $XXXM-X | 高/中/低 |
| 2 | ... | ... | ... | ... | ... |

## Appendix: 金融背景知识

这份文件是"买家名单生成器（Buyer List Generator）"技能的详细说明。在并购交易（M&A）的卖方流程中，构建一份高质量的潜在买家名单是整个项目最早期但最关键的工作之一。如果 Teaser（匿名简介）是投行发给外部的"广告"，买家名单就是投行内部的工作指南——它决定了你应该把这份广告发给哪些人。

### 1. 买家分类：战略买家 vs 财务买家

#### 1.1 战略买家（Strategic Acquirers）

战略买家是"同行公司"或"与公司业务有关联的公司"。他们收购的动机不是纯财务投资，而是因为整合后能产生"1+1>2"的协同效应。

**四大子类型：**

| 类型 | 动机 | 经典例子 |
|------|------|---------|
| 直接同行 | 吃掉竞争对手，扩大市场份额 | Facebook 收购 Instagram |
| 上下游整合 | 控制供应链或渠道 | 亚马逊收购 Whole Foods |
| 跨市场扩张 | 进入新的地理市场 | 中国公司收购东南亚同行 |
| 能力补充 | 获得自己缺少的技术/人才 | Google 收购 YouTube |

**战略买家的特点：**
- 通常出价最高（可以算进协同效应的"额外价值"）
- 决策更快（CEO 一个人拍板就行）
- 但可能有反垄断风险

#### 1.2 财务买家（Financial Acquirers / PE Fund）

私募股权基金。买公司的目的只有一个：**财务回报**。

**PE 基金的赚钱公式：**
```
投资回报 = 买入价合理 + 经营改善 + 杠杆效应 + 市场时机 + 退出溢价
```

**PE 筛选标准：**
- 年 EBITDA 在 $10M-$500M 之间
- 有稳定的经常性收入
- 轻资产模式
- 在增长的行业中
- 非创始人依赖

#### 1.3 管理层收购（MBO）

公司现有管理层自己凑钱把公司买下来。优势是管理层已经非常了解公司；缺点是通常没有足够的钱，需要外部融资。

### 2. 买家名单构建的完整流程

#### 第一步：定义筛选标准

| 筛选维度 | 需要回答的问题 |
|---------|-------------|
| 行业匹配 | 买家是否在同一/相邻/互补行业？ |
| 规模匹配 | 买家体量是否足够大？ |
| 地域匹配 | 买家是否在目标市场有业务？ |
| 财务能力 | 买家账上有多少现金？能加多少杠杆？ |
| 收购历史 | 买家过去有没有收购记录？整合得好不好？ |
| 反垄断风险 | 合在一起监管会不会叫停？ |

#### 第二步：搜索和收集

- 调用行业数据库（Capital IQ、PitchBook、Bloomberg）
- 搜索新闻关键词
- 查阅该行业的并购历史案例
- 查看 PE 基金的投资组合

#### 第三步：分类和排序

| 排序层 | 描述 | 行动 |
|-------|------|------|
| Tier 1（高优先级） | 最匹配、最可能出手、最能出高价 | 优先联系，发 Teaser |
| Tier 2（中优先级） | 有一定匹配度但存在部分障碍 | 作为备选，第一轮后跟进 |
| Tier 3（低优先级） | 勉强相关，可能性较低 | 保留在名单上，但不主动联系 |

---

### 3. 对每个潜在买家的深度评估

**战略买家评估清单：**
- [ ] 市值是多少？（决定了能"吞下"多大的目标）
- [ ] 账上现金和债务水平？
- [ ] 过去 5 年做过哪些收购？
- [ ] CEO/CFO 最近的公开发言提到了什么战略？
- [ ] 他们的 P/E 倍数是多少？

**财务买家评估清单：**
- [ ] 这家 PE 现在正在募集新基金吗？
- [ ] 他们的投资标的中位数规模是多少？
- [ ] 他们在相关行业有没有已投过的同类公司？
- [ ] 他们最近刚做了什么退出？
- [ ] 他们的平均持有期是多少？

---

### 给小白的一句话

> 买家名单就是投行在企业出售项目中的"情报工作成果"——像一个超详细的潜在相亲对象清单。你不能把征婚广告发给全世界3000万人，那既不专业也不安全；你得先筛选出大概80-150个"有可能且买得起"的人，再把它们分成三个梯队，然后针对每个梯队制定不同的联络策略。一个好的买家名单能让成交概率翻倍，一个烂的买家名单会把最好的买家漏掉。
