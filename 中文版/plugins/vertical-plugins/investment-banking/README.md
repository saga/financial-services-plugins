# 投资银行插件

投资银行生产力工具，用于股权研究、估值、演示文稿和交易材料。

## 功能特色

- **交易材料** — CIM、摘要、流程函和买方名单
- **演示文稿** — 公司简介、含品牌模板的推介材料
- **交易支持** — 并购模型、交易追踪和数据包

## 安装

```bash
claude --plugin-dir /path/to/investment-banking
```

或复制到项目的 `.claude-plugin/` 目录。

## 命令

| 命令 | 说明 |
|------|------|
| `/one-pager [公司]` | 用于推介资料的单页公司简介 |
| `/cim [公司]` | 起草保密信息备忘录（CIM） |
| `/teaser [公司]` | 匿名一页公司摘要 |
| `/buyer-list [公司]` | 战略和财务买方名单 |
| `/merger-model [交易]` | 增厚/稀释并购分析 |
| `/process-letter [交易]` | 投标指示和流程函件 |
| `/deal-tracker` | 追踪进行中的交易、里程碑和行动项 |

## 示例工作流

### 单页公司简介
```
/one-pager 目标公司

# 生成：
# - 使用 PPT 模板的单页公司简介
# - 4 象限：概况、业务、财务、股权结构
# - 保留模板边距和品牌风格
```

### CIM 起草
```
/cim 目标公司

# 生成：
# - 完整的 CIM 文档，含执行摘要、业务概述、
#   财务分析和市场定位
```

### 并购模型
```
/merger-model 收购方收购目标公司

# 生成：
# - 增厚/稀释分析
# - 资金来源与用途、备考财务报表
# - 对购买价格和协同效应的敏感性分析
```

## 📖 本篇金融知识小贴士

**CIM（保密信息备忘录）**：出售公司时向潜在买家提供的详细文件，包含业务、财务、市场等全面信息。

**Teaser（摘要）**：CIM 的简化一页版本——不透露公司名称，仅给出大致规模、行业和亮点，用来"钓鱼"。

**流程函（Process Letter）**：卖方投行发给潜在买家的正式文件，说明交易流程、投标截止日期和出价要求。

**买方名单（Buyer List）**：列出可能对收购目标感兴趣的买家清单，分为战略买家（同行公司）和财务买家（私募股权基金）。

**增厚/稀释分析（Accretion/Dilution）**：评估收购对收购方每股收益的影响。增厚 = 收购后 EPS 更高，是好交易；稀释 = EPS 下降，需要理由说明战略价值。

**并购模型（Merger Model）**：模拟收购后的财务报表，测试不同假设（价格、融资方式、协同效应）对财务指标的影响。

**备考报表（Pro Forma）**："假设已经完成"的财务报表。比如"假设收购已完成，公司今年的收入会是多少"。

**数据包（Datapack）**：从 CIM 和公开文件中提取的关键财务数据汇总，供买方建模使用。

## Appendix: 金融背景知识

这份文件是"投资银行插件（Investment Banking Plugin）"的总体说明。投资银行（Investment Banking，简称 IB）是金融行业最核心、最传统的业务之一——帮助公司做并购、上市（IPO）、发债融资。

---

### 1. 什么是投资银行？

**类比：**
想象你是一个"企业媒人"。一家公司想收购另一家公司，但不知道怎么操作——你得帮忙：
1. 找到合适的"对象"（寻找潜在标的）
2. 评估"彩礼"多少（估值分析）
3. 和对方"家长"谈判（交易条款）
4. 准备"婚礼"（交易文件）

投资银行就是大型企业的"媒人+财务顾问"——帮助企业完成最重大的财务交易。

---

### 2. 投资银行的"四大核心业务"

#### 业务 1：并购咨询（M&A Advisory）
- 买方顾问：帮客户找并购标的、谈判、估值
- 卖方顾问：帮客户找买家、包装公司、拍卖流程
- 典型交易：微软收购动视暴雪（$69B）、特斯拉收购 SolarCity

#### 业务 2：股权融资（ECM）
- IPO：首次公开发行，帮公司上市
- 增发（Follow-on）：已上市公司再发新股融资
- 典型交易：阿里巴巴 IPO（$25B）、Facebook IPO

#### 业务 3：债务融资（DCM）
- 发债：帮公司发行债券融资
- 银团贷款：组织多家银行联合放贷
- 信用评级：帮助公司获得评级

#### 业务 4：重组（Restructuring）
- 帮助陷入困境的公司重组债务
- 破产保护、资产出售
- 典型交易：雷曼兄弟破产、通用汽车重组

### 3. 投行交易材料的"四大件"

| 文件类型 | 英文 | 用途 | 类似 |
|---------|------|------|------|
| 推介材料 | Pitch Deck | 说服客户"选我们" | 竞标PPT |
| 保密信息备忘录 | CIM | 介绍标的公司给潜在买方 | 出售说明书 |
| 一页摘要 | One-Pager | 快速了解公司 | 名片 |
| 流程函 | Process Letter | 告诉买方"怎么出价" | 拍卖规则 |

---

### 4. 投行的"等级制度"

| 头衔 | 工作内容 | 典型工作时长 |
|------|---------|------------|
| Analyst（分析师） | Excel 建模、PPT 画图 | 80-100 小时/周 |
| Associate（经理） | 管分析师、写报告 | 70-90 小时/周 |
| VP（副总裁） | 对接客户、管理项目 | 60-70 小时/周 |
| MD（董事总经理） | 拉客户、谈交易 | 不定 |

---

### 5. 真实案例：高盛承销阿里巴巴 IPO

| 环节 | 高盛做了什么 |
|------|------------|
| 准备阶段 | 调研阿里巴巴业务模式，做估值模型 |
| 路演阶段 | 带马云去全球见基金经理，推销 |
| 定价阶段 | 根据订单簿决定 IPO 价格 |
| 上市当天 | 确保股价平稳开盘，不暴涨暴跌 |

**交易规模**：$25B（当时史上最大 IPO）
**高盛佣金**：约 $500M（2%）

---

### 给小白的一句话

> 投资银行就是金融界的"超级中介"——你帮大公司做并购（A 买 B）、做上市（IPO）、做发债（借钱），每笔交易收取 1-3% 的佣金。一个 $10 亿的交易 = $2000 万的顾问费。代价是：分析师每周工作 80-100 小时，凌晨 2 点还在改 PPT 配色。但为什么这么多人挤破头想进投行？因为你在 25 岁就能坐在会议室里和一个 50 岁的上市公司 CEO 讨论他的公司值多少钱——这种经历和能力积累，是其他行业几乎无法提供的。
