## Appendix: 金融背景知识

这份文件是投行"匿名项目简介（Teaser）"生成命令。在投行帮助企业出售的整个流程中，Teaser是最先发出的第一份文件——它的角色就像追女孩子时发的第一封"暧昧情书"：不能直接告诉对方你是谁，但要足够吸引人，让对方忍不住想进一步了解你。Teaser好与坏，直接决定有多少买家会签NDA（保密协议）进入下一轮。

---

### 1. Teaser 在整个并购流程中的位置

想象你开了一家非常赚钱的甜品店，现在你想把这家店卖掉。你找到了投行帮忙。整个卖店流程大概是这样的：

```
第一步：准备材料
├── Teaser（匿名简介）：1-2页，不说店名，只说"某市核心商圈黄金地段、年净利200万的甜品店"
├── CIM（机密备忘录）：50-100页，签了NDA才能看，详细到每天卖多少杯奶茶
└── Management Presentation：管理层亲自出来路演

第二步：接触买家
├── 第一轮：发Teaser → 感兴趣的买家签NDA → 发CIM
├── 第二轮：收到初步报价（IOI） → 筛选 → 尽调（DD）
└── 第三轮：最终报价 → 签合同 → 成交
```

**Teaser是第一块敲门砖**。如果你的Teaser写得像报税单一样无聊，那就没人会签NDA；如果Teaser写得像悬疑小说的预告片，那买家就会蜂拥而至。

---

### 2. 为什么必须匿名？

这是一个经典的"谈判博弈"问题。

假设你是"老王餐饮集团"的老板，你想卖掉你旗下一家子公司。如果全世界都知道你要卖公司，会发生什么？

- **你的员工会恐慌**："公司要卖了，我是不是要失业了？" → 核心员工开始投简历
- **你的客户会担心**："换了老板，合同还作数吗？" → 大客户可能不再续约
- **你的供应商会收紧**："这家公司马上换老板了，先不给账期了，现金交易吧。"
- **你的竞争对手会攻击你**："老王不行了，要卖资产了，快抢他的客户！"

所以，在第一轮接触买家的时候，必须**匿名**——用代号、用行业描述、用模糊的数字范围，让买家只能看到"这是一家好公司"，但绝对无法推断出"这家公司到底是哪家"。

**真实案例**：2018年，一家全球知名的消费品巨头在出售旗下某个亚洲业务单元时，Teaser上写的是"A leading premium dairy brand in Southeast Asia with >$500M revenue"。所有线索都可以指向好几家公司——蒙牛、伊利、味全——买家无法确定到底是哪一家，只知道"这规模不小，值得签NDA看看"。

### 3. Teaser 的黄金法则：钓胃口但不暴露

| 应该说的 | 不应该说的 |
|---------|-----------|
| "北美前三的医疗器械分销商" | ABC Medical Supplies Inc. |
| "年收入 $100-150M" | "2023年收入 $127.3M" |
| "在中西部拥有100+门店" | "在伊利诺伊州芝加哥市有113家门店" |
| "服务于财富500强客户" | "主要客户包括沃尔玛和可口可乐" |
| "EBITDA利润率 > 25%" | "EBITDA $34.2M, 利润率 26.8%" |

基本原则：**说出"吸引力"，隐藏"身份"。**

---

### 4. 一个好的 Teaser 应该具备什么？

**4.1 交易摘要**：给买家一个"快照"
- 行业（够具体但不够辨识）
- 收入范围（模糊化处理，比如 "$200-250M" 而不是 "$237M"）
- 地理位置（够大到无法定位，比如"东南亚"而不是"泰国曼谷"）
- 交易类型（股权出售/资产出售/融资）

**4.2 投资亮点**：3-5条让买家心动的理由
- "拥有超过15年的运营历史和行业中最高客户留存率（>95%）"
- "在新兴市场中占据领导地位，该市场未来5年预计增长30%CAGR"
- "轻资产运营模式，资本支出仅占收入的2%"

**4.3 财务概览**：匿名但诱人的数字
- 展示3年的收入/EBITDA趋势——但数字模糊化处理
- 强调利润率高于行业平均——但不说具体行业平均是多少
- 展示增长趋势——但不说绝对数字

**4.4 交易逻辑**：为什么要卖？买家为什么应该买？
- 常见卖的理由："创始人年事已高，无子女愿意接班"（这叫"退休/退出"逻辑）
- "公司发展到了一个阶段，需要大平台的资源才能进一步增长"
- "第一代投资人的基金已经到期，需要退出变现"

### 5. 真实 Teaser 格式赏析

一个典型的 Teaser（2页PPT的格式）大概长这样：

```
┌─────────────────────────────────────┐
│     CONFIDENTIAL - PROJECT EAGLE     │
│                                      │
│  A Leading B2B SaaS Platform in     │
│  Supply Chain Management            │
│                                      │
│  Investment Highlights:             │
│  • >$80M ARR, growing 30%+ YoY     │
│  • 95%+ gross retention rate       │
│  • #1 market share in APAC region   │
│  • Founder-led, seeking growth      │
│    capital partner                  │
│                                      │
│  Financial Overview (Approximate):  │
│  ┌──────────┬───────┬───────┬──────┐│
│  │          │ FY1   │ FY2   │ LTM  ││
│  ├──────────┼───────┼───────┼──────┤│
│  │ Revenue  │ ~$60M │ ~$80M │ ~$95M││
│  │ EBITDA   │ ~$15M │ ~$22M │ ~$28M││
│  │ Margin   │ ~25%  │ ~28%  │ ~30% ││
│  └──────────┴───────┴───────┴──────┘│
│                                      │
│  Contact: [Banker Name]             │
│  [Bank Name], [Phone]               │
└─────────────────────────────────────┘
```

### 给小白的一句话

> Teaser就是投行帮老板卖公司时发的"匿名征婚广告"——上面写着"某优质单身男，38岁，年收入500万，有房有车，爱运动爱旅行"，但绝不会写上他的真实名字和工作单位。只有那些真心感兴趣并且签了保密协议的人，才能拿到他的详细资料和联系方式。写得好，能让100个潜在买家里有30个愿意进一步了解；写得烂，可能只剩下3个好奇的。
