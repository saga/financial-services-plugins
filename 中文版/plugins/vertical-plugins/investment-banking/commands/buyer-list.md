## Appendix: 金融背景知识

这份文件是"买家名单（Buyer List）"的生成命令。在并购交易（M&A）的卖方流程中，构建一份高质量的潜在买家名单是整个项目最早期但最关键的工作之一。如果Teaser（匿名简介）是投行发给外部的"广告"，买家名单就是投行内部的工作指南——它决定了你应该把这份广告发给哪些人。发多了会泄密、会显得你"到处兜售"不专业；发少了可能漏掉真正有意向且能出高价的金主。

### 1. 买家分类：战略买家 vs 财务买家 vs 管理层收购

在并购世界里，潜在的卖家（买家）通常分为三大类：

#### 1.1 战略买家（Strategic Acquirers）

战略买家是"同行公司"或"与公司业务有关联的公司"。他们收购目标公司的动机不是纯财务投资，而是因为整合后能产生"1+1>2"的协同效应。

**战略买家的四大子类型：**

| 类型 | 动机 | 例子 |
|------|------|------|
| 直接同行 | 吃掉竞争对手，扩大市场份额 | Facebook收购Instagram |
| 上游/下游整合 | 控制供应链或渠道 | 亚马逊收购Whole Foods（进入线下零售） |
| 跨市场扩张 | 进入新的地理市场 | 某中国公司收购东南亚同行以进入东南亚市场 |
| 能力补充 | 获得自己缺少的技术/人才/产品 | Google收购YouTube（Google自己没有视频平台） |

**战略买家的特点：**
- 通常出价最高（因为他们可以算进协同效应的"额外价值"）
- 决策更快（CEO一个人拍板就行，不需要LP投票）
- 但是可能有反垄断风险（如果已经是行业前几名，再收购同行可能被监管否决）

#### 1.2 财务买家（Financial Acquirers / Sponsors）

财务买家就是私募股权基金（PE Fund）。他们买公司的目的只有一个：**财务回报**。他们不搞"1+1>2"的协同效应，而是赌"我今天能在合理价格买入，包装几年后以更高价格卖掉"。

**PE基金的赚钱公式：**
```
投资回报 = 买入价合理 + 经营改善 + 杠杆效应 + 市场时机 + 退出溢价
```

**详细拆解：**
- **买入价合理**：买得便宜就已经赢了一半。PE最怕的就是"在牛市高点追高买入"。
- **经营改善**：买下后换管理层、砍掉不赚钱的业务、优化供应链、开拓新市场，提升EBITDA。
- **杠杆效应**：PE通常只用30-40%的自有资金（LP的钱），剩下的60-70%用银行贷款。如果公司增值20%，PE的实际回报可以放大到40-50%（因为有杠杆）。
- **市场时机**：投资4-6年后找一个好时机卖出（比如行业景气度高、二级市场估值倍数高的时候）。
- **退出溢价**：卖给战略买家或者IPO上市，通常会获得比买入价高2-5倍的回报。

**PE基金筛选标准（买什么样的公司？）：**
- 年EBITDA在 $10M-$500M 之间（太小了不值得花精力，太大了买不起）
- 有稳定的经常性收入（recurring revenue），不要过度依赖一次性大合同
- 轻资产模式（不需要大量资本支出也能增长）
- 在增长的行业中，公司有独特的竞争地位
- 非创始人依赖（不能公司离了创始人就完蛋）

#### 1.3 管理层收购（MBO, Management Buyout）

这是比较少见的模式——公司现有的管理层自己凑钱（通常是"自己的少量存款 + 大量PE基金支持"）把公司从现任股东手里买下来，自己做老板。MBO的优势是管理层已经非常了解公司，几乎没有"交接期"和"学习成本"；缺点是管理层通常没有足够的钱，需要外部融资。

### 2. 买家名单构建的完整流程

#### 第一步：定义筛选标准

在开始找买家之前，先想清楚"什么样的买家最适合这家公司"：

| 筛选维度 | 需要回答的问题 |
|---------|-------------|
| 行业匹配 | 买家是否在同一/相邻/互补行业？ |
| 规模匹配 | 买家体量是否足够大来"消化"这家目标公司？ |
| 地域匹配 | 买家是否在目标公司所处的市场有业务？ |
| 财务能力 | 买家账上有多少现金？能加多少杠杆？ |
| 收购历史 | 买家过去有没有收购的记录？他们整合得好不好？ |
| 反垄断风险 | 这两家合在一起，监管会不会叫停？ |

#### 第二步：搜索和收集

- 调用行业数据库（如Capital IQ、PitchBook、Bloomberg）
- 搜索新闻关键词（"XX公司 收购" "XX公司 战略投资"）
- 查阅该行业的并购历史案例，看看之前是谁在买这类公司
- 查看该行业的PE基金投资组合，看看哪些PE已经投了同类型公司（意味着他们懂这个行业，更可能再次出手）

#### 第三步：分类和排序

把收集到的所有潜在买家按照"优先级"分成三层：

| 排序层 | 描述 | 行动 |
|-------|------|------|
| Tier 1（高优先级） | 最匹配、最可能出手、最能出高价的买家 | 优先联系，发Teaser |
| Tier 2（中优先级） | 有一定匹配度但存在部分障碍 | 作为备选，第一轮后跟进 |
| Tier 3（低优先级） | 勉强相关，可能性较低 | 保留在名单上，但不主动联系 |

### 3. 对每个潜在买家的深度评估

对名单上排名靠前的买家，你还要做更细的功课：

**战略买家评估清单：**
- [ ] 他们家的市值是多少？（决定了他们能"吞下"多大的目标）
- [ ] 账上现金和债务水平？（算一下他们杠杆空间还有多少？）
- [ ] 过去5年他们做过哪些收购？（每次收购的规模、行业、整合效果）
- [ ] CEO/CFO最近的公开发言提到了什么战略？（比如"我们要大力进军XX领域"）
- [ ] 他们的PE倍数是多少？如果远高于目标公司，用自家股票来收购就更"划算"

**财务买家评估清单：**
- [ ] 这家PE现在正在募集新基金吗？（正在募集新基金=有钱花）
- [ ] 他们的投资标的中位数规模是多少？（如果太大或太小，都不合适）
- [ ] 他们在相关行业有没有已投过的同类公司？（行业专长）
- [ ] 他们最近刚做了什么退出？（刚卖了一家公司=口袋里有钱=急着找新项目）
- [ ] 他们的平均持有期是多少？（3年还是7年？影响他们的投资决策风格）

### 给小白的一句话

> 买家名单就是投行在企业出售项目中的"情报工作成果"——像一个超详细的潜在相亲对象清单。你不能把征婚广告（Teaser）发给全世界3000万人，那既不专业也不安全；你得先筛选出大概80-150个"有可能且买得起"的人，再把它们分成三个梯队（特别合适/还行/备胎），然后针对每个梯队的人制定不同的联络策略。一个好的买家名单能让成交概率翻倍，一个烂的买家名单会把最好的买家漏掉、把最差的买家请进来。
