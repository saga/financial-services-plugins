## Appendix: 金融背景知识

这份文件是"并购模型（Merger Model）"的命令指引。如果你看过电影《华尔街之狼》或者《大空头》，你应该对"并购"这个过程有点初步印象——一家公司想买另一家公司，请投行来帮忙算清楚"这笔交易值不值得花这笔钱"。并购模型就是专门用来回答这个核心问题的Excel工具。它是每一个投行Analyst/Associate面试必考的题目，也是实际工作中用得最频繁的模型之一。

---

### 1. 什么是并购模型？它回答什么问题？

假设你是一家上市公司（买方/Acquirer）的CEO，董事会问你想花15亿美元收购一家竞争对手（目标公司/Target）。你必须回答三个问题：

1. **买完以后，我公司的EPS（每股收益）是变多了还是变少了？**（增厚 vs 摊薄）
2. **我应该用什么方式付钱？全现金？全股票？还是混合？**
3. **多付了10%或者少付10%的购买价格，结果会差多少？**（敏感性分析）

并购模型就是为这三个问题量身打造的专用计算器。

---

### 2. Accretion/Dilution（增厚/摊薄分析）——最核心的概念

这个概念是每一家投行面试都必考的最基础也是最核心的知识点，我在这里用大白话给你拆解：

#### 用最通俗的类比：

你和你朋友各有自己的小生意：

- **你的公司**：每年赚100万，发行了100万股在市场上。所以每股收益（EPS）= 100万 ÷ 100万股 = **$1.00/股**
- **朋友的公司**：每年赚30万，发行了50万股。EPS = 30万 ÷ 50万股 = **$0.60/股**

现在你想把朋友的公司买下来。你有两个付款方案：

**方案A：全现金收购**
- 你花$500万现金买下朋友公司（假设用银行贷款，年利率5%，每年多付$25万利息）
- 收购后两家合在一起：赚 100万 + 30万 - 25万利息 = 105万
- 你的股份还是100万股（没增发新股）
- 新EPS = 105万 ÷ 100万 = **$1.05/股** ✅ 比收购前的$1.00高 → **这叫 Accretion（增厚）**

**方案B：全股票收购**
- 不用现金，而是增发新股给朋友当对价。假设你发行50万股新股给朋友
- 收购后两家合在一起：赚 100万 + 30万 = 130万
- 你的股份变成 100万 + 50万 = 150万股
- 新EPS = 130万 ÷ 150万 = **$0.87/股** ❌ 比收购前的$1.00低 → **这叫 Dilution（摊薄）**

**结论一目了然**：全现金方案更划算（因为增厚了EPS）。那么问题来了——为什么市场上还有那么多全股票收购？因为：
- 现金不够（不是每家公司都有$500万现金或者能从银行借到$500万）
- 被收购的公司想要股权而不是现金（他们想继续享受合并后的增长红利）
- 用股票支付更"安全"（如果收购后发现不对劲，风险由新老股东共担）

### 3. 并购模型的三大关键输入假设

#### 3.1 收购价格和溢价

收购价格 = 目标公司的当前市值 ×（1 + 溢价率）

| 溢价率 | 价格 | 市场反应 |
|-------|------|---------|
| +10-15% | 较低 | 买方股东高兴，但目标公司董事可能不同意 |
| +20-30% | 标准 | 行业平均水平 |
| +40-50% | 较高 | 买方股东可能不满（"你花太多钱了！"）但目标公司无法拒绝 |
| +50%+ | 过高 | 通常被认为是"疯狂"，日后可能大额减记 |

为什么必须溢价？上市公司股票在市场上每天都在交易，你不可能按昨天收盘价把整个公司买下来——现任股东不会同意的。你必须出一个比市场价高得多的"一口价"，才能让足够多的股东投票同意卖掉公司。这就像别人住的别墅，你不可能以市场价把人赶出去。

#### 3.2 融资结构

全现金收购的钱从哪里来？

| 资金来源 | 优点 | 缺点 |
|---------|------|------|
| 公司自有现金 | 不需要支付利息 | 现金耗尽后抗风险能力变弱 |
| 银行债务 | 利息可以抵税 | 杠杆（负债率）飙升，信用评级可能被降级 |
| 发行债券 | 融资额度大 | 需要支付债券利息 |
| 发行股票 | 不增加债务负担 | 稀释现有股东权益（摊薄） |

大多数交易都是**混合融资**——一部分现金，一部分债务，一部分股票。

#### 3.3 协同效应（Synergies）

协同效应是任何并购案号称"这笔交易值得做"的核心理由。一般分为两类：

**成本协同效应**：
- 两家公司合并后，可以裁掉重复的部门（比如不需要两个CFO、两个HR部门）
- 统一的采购量更大，可以跟供应商谈更好的价格
- 共享基础设施（办公室、IT系统、物流网络）

**收入协同效应**：
- 交叉销售（A公司的客户可以顺便卖B公司的产品）
- 新产品开发（两家公司的技术团队合并后研发能力更强）
- 地理扩张（B公司在A公司没有覆盖的地区有渠道）

**一个关键的警示**：在研究报告中，成本协同通常有50-70%的实现概率，而收入协同可能只有20-30%。因为"省钱"是看得见摸得着的（裁一个部门就是省钱），但"多赚"经常是画饼——客户不一定愿意买你的交叉销售产品。

### 4. 增厚/摊薄百分比的计算公式

```
增厚/稀释 (%) = (备考合并后 EPS - 买方独立 EPS) ÷ 买方独立 EPS × 100%
```

- 正值 = 增厚（Accretive，好事）
- 负值 = 摊薄（Dilutive，坏事但可接受）
- 零 = 中性（EPS Neutral——不多不少）

注意：摊薄不一定是坏交易。如果战略理由充分（比如这次收购能帮助公司进入一个全新的万亿级市场），暂时被摊薄的EPS是可以接受的——只要董事会有信心在未来几年内通过协同效应"回本"。

### 给小白的一句话

> 并购模型就是帮上市公司算"娶媳妇划不划算"的Excel计算器。你得算清楚：娶了她以后，家里每个人（每股）能分到的钱是更多了（Accretion/增厚）还是更少了（Dilution/摊薄）；用什么方式下聘礼——全现金（借钱背利息）、全股票（稀释自家股份）还是混合支付；如果多给了10%彩礼或少给了10%，结果会有多大差别。投行面试90%会问这个问题，因为它是整个并购分析中最核心的一环。
