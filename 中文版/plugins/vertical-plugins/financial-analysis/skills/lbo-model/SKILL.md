*   **每项计算必须是 Excel 公式** —— 严禁在代码中计算数值并硬编码。必须写入如 `cell.value = "=B5*B6"` 的公式字符串。模型必须是动态的，能随输入变化而更新。
*   **严格遵循蓝/黑/绿颜色代码**：
    *   **蓝色 (0000FF)**：硬编码输入。
    *   **黑色 (000000)**：包含计算逻辑的公式。
    *   **紫色 (800080)**：同表单元格引用。
    *   **绿色 (008000)**：跨表单元格引用。
*   **分部分构建，每步确认** —— 完成一个部分后（如 Sources & Uses），先展示给用户并确认无误后再继续下一步。

- [ ] **资金平衡**：Sources（资金来源）必须等于 Uses（资金用途），通常设置一个“配平项（Plug）”。

### 1. LBO——"金融界最经典的以小博大"

**LBO（Leveraged Buyout，杠杆收购）**，直译就是"用杠杆来收购"。这里的"杠杆"就是借钱。

**最经典的生活比方——买房**：

假设你看中一套 $100 万的房子。
- **全款买**：你出 $100 万。5 年后房子涨到 $150 万，你赚了 $50 万。回报率 = 50%。
- **按揭买**：你只出 $20 万首付（20%），找银行借 $80 万（80%）。5 年后房子还是涨到 $150 万。你卖掉房子，还掉银行的 $80 万，剩 $70 万——但你只投入了 $20 万。回报率 = ($70-$20)/$20 = 250%。

**同样的房子，同样的涨幅，只因为你借了钱，回报率从 50% 变成了 250%。这就是"杠杆"的魔力。**

当然，杠杆是一把双刃剑：
- 房子涨了 → 放大收益
- 房子跌了 → 放大亏损（你投的 $20 万可能全部归零甚至倒欠银行钱）

**LBO 和买房的区别**：

| | 买房按揭 | LBO（杠杆收购） |
|---|---|---|
| **资产** | 房子 | 一家公司 |
| **首付（权益）** | 你的积蓄 | PE 基金从 LP（投资人）募来的钱 |
| **贷款（债务）** | 按揭贷款 | 银行贷款 + 高收益债 |
| **谁来还贷** | 你的工资 | 目标公司自己赚的现金流 |
| **怎么赚钱** | 房子升值 + 租金 | 公司利润增长 + 还债（让权益占比逐步变大）|

LBO 和买房最核心的区别是：**买房是你用工资还贷，LBO 是目标公司用自己的经营现金流还贷。** 这就意味着，PE 基金收购一家公司后，赌的是这家公司能持续产生稳定的现金流来还债——所以不是所有公司都适合做 LBO。

### 2. Sources & Uses——"LBO 模型的心脏"

**Sources & Uses（资金来源与用途）** 是所有 LBO 模型的第一张表，也是一切计算的起点。

它像是一个**天平**：

```
          资金来源 (Sources)                    资金用途 (Uses)
  ┌────────────────────┐         ┌────────────────────┐
  │ 银行高级贷款  $300M    │         │ 买股权       $450M  │
  │ 高收益债     $100M    │         │ 偿还旧债     $50M   │
  │ PE 出资     $150M    │         │ 交易费用     $50M   │
  │                     │         │                     │
  │ 总计:      $550M    │    =    │ 总计:      $550M  │
  └────────────────────┘         └────────────────────┘
```

**Sources（钱从哪来）**：
- **银行高级贷款（Senior Debt）**：银行借的，成本最低（利率约 5-7%），风险也最低（如果公司破产，银行最先拿回钱）。通常占总资金 40-50%。
- **高收益债 / 夹层融资（Subordinated Debt / Mezzanine）**：利率更高（8-12%），风险也更高（银行之后才能拿钱）。占总资金 15-25%。
- **PE 出资（Equity）**：PE 基金自己出的钱，回报最高但也最后拿钱。占总资金 30-40%。

**Uses（钱花在哪）**：
- **买股权（Purchase of Equity）**：最大的开销，付给目标公司的老股东。
- **偿还旧债（Refinancing）**：目标公司之前可能有银行贷款，收购后通常要还掉。
- **交易费用（Transaction Fees）**：支付给投行、律师、会计师的费用，通常是交易金额的 2-5%。

**核心约束**：Sources 必须 ≡ Uses（一分不差）。如果差了，就靠"配平项（Plug）"来调整——通常是 PE 的出资额（因为 PE 是"最后的出资人"，多退少补）。

### 3. LBO 的三种债务——"就像吃饭，有人坐在餐桌旁，有人在厨房等着"

LBO 中的债务不是"一种"，而是一个**多层结构（Capital Structure）**。不同层的债务有不同的风险等级、利率和偿还优先级。

打比方——一个公司的破产就像分蛋糕：
- **高级有担保贷款（Senior Secured）** = "坐在餐桌旁的 VIP"。公司破产了，他们最先拿钱，所以利率最低。
- **高级无担保贷款（Senior Unsecured）** = "在门口排队的人"。没有抵押品，但比次级债优先。
- **次级债 / 高收益债（Subordinated / High-Yield）** = "在走廊里等着的人"。只有前面的人拿完了才轮到你，所以利率很高（8-12%）。
- **权益（Equity）** = "在门外等的人"。最后才拿钱，但也可能拿最多（如果公司没破产，大部分价值属于你）。

| 层级 | 典型利率 | 占比 | 偿还优先级 | 风险等级 |
|------|----------|------|-----------|----------|
| 高级有担保贷款 | SOFR+2-4% (~5-7%) | 35-45% | 第 1 | 低 |
| 高级无担保贷款 | SOFR+4-6% (~7-9%) | 10-15% | 第 2 | 中 |
| 次级/高收益债 | 8-12% | 10-20% | 第 3 | 高 |
| PE 权益 | 20%+目标 | 30-40% | 第 4（最后） | 最高 |

**关键概念：偿债顺序（Cash Sweep / Waterfall）**

每一年，目标公司赚了钱，这笔钱的分配顺序是固定的：
```
1. 付员工工资和供应商款（这不用说）
2. 付利息（给银行和债券持有人）
3. 还本金（先还高级贷款，再还次级债）
4. 剩下的钱归 PE（可以留作未来发展，也可以分红给 LP）
```

这个顺序就像瀑布流水一样，一层一层往下流——所以叫"Cash Waterfall"。

### 4. MOIC vs IRR——"总路程 vs 平均速度"

PE 基金有两个最重要的回报指标：MOIC 和 IRR。

**MOIC（Multiple of Invested Capital）——投资成本倍数**
```
MOIC = 最终拿回的总金额 / 初始投资额
```
- 投了 $1 亿，最后拿回 $3 亿 → MOIC = 3.0x
- **MOIC 只看"翻了几倍"，不管用了多少时间。**

**IRR（Internal Rate of Return）——内部收益率**
```
IRR = 使得所有现金流净现值为零的那个折现率
```
- 投了 $1 亿，第一年收回 $6000 万，第二年收回 $6000 万 → IRR ≈ 13%
- **IRR 考虑了时间——同样的 MOIC，时间越短，IRR 越高。**

**打一个最直观的比方——开车旅行**：

| | 从北京到上海 | 从北京到广州 |
|---|---|---|
| **总路程** | 1,200 公里 | 2,200 公里 |
| **时间** | 8 小时 | 16 小时 |
| **平均速度** | 150 km/h（高铁） | 137.5 km/h（高铁） |

- **MOIC 就像是"总路程"**——你走了多远。1200 公里 vs 2200 公里。
- **IRR 就像是"平均速度"**——你走得多快。150 km/h vs 137.5 km/h。

两个指标必须一起看：
- 一个项目 MOIC = 5.0x，但用了 15 年 → IRR 只有约 11.3%（一般般）
- 一个项目 MOIC = 2.0x，但只用 2 年 → IRR 高达 41.4%（非常好！）

**PE 基金的典型目标**：
| 指标 | 目标值 | 说明 |
|------|--------|------|
| 毛 MOIC | 2.5-3.5x | 3 年翻 3 倍是 PE 的"及格线" |
| 净 IRR | 15-25% | 扣掉管理费和业绩分成后 |
| 持有期 | 3-7 年 | 典型的 PE 投资周期 |

### 5. PE 怎么赚钱？——"2-20 收费模式"

PE 基金的收费模式叫**"2 and 20"**：

- **管理费（Management Fee）= 2%**：每年按管理资产的 2% 收费。比如一个 $10 亿的基金，每年收 $2000 万管理费。这笔钱用来支付 PE 团队的工资、办公、出差等日常开销。
- **业绩分成（Carried Interest / "Carry"）= 20%**：如果投资收益超过了"门槛收益率（Hurdle Rate，通常是 8%）"，超过部分的 20% 归 PE 团队，80% 归投资者（LP）。

**举个例子**：

PE 基金规模 $10 亿，5 年后翻到 $30 亿（MOIC=3.0x）。

```
总收益 = $30 亿 - $10 亿 = $20 亿
先确保 LP 拿回本金 + 8% 年化回报（门槛）
  门槛 = $10 亿 × (1+8%)⁵ = $14.7 亿
  超额收益 = $30 亿 - $14.7 亿 = $15.3 亿
  
PE 团队拿走超额收益的 20% = $15.3 亿 × 20% = $3.06 亿（Carry）
LP 拿走剩下的 = $30 亿 - $3.06 亿 = $26.94 亿
```

**这就是为什么 PE 行业能造富**——一个 $10 亿的基金，如果表现好，PE 团队光 Carry 就能赚 $3 亿。如果是中型以上基金（$50-100 亿），Carry 可以超过 $10 亿。

**J 曲线效应（J-Curve Effect）**：

PE 基金的生命周期通常有一个"J 型"的回报曲线：

```
回报
  ^
  |                              /
  |                             /
  |                            /
  |          ----------------
  |         /
  |        /
  |       /
  |      /
  +----->→→→→→→→→→ 时间
  前 2-3 年      后 3-5 年
  （亏损期）     （收获期）
```

- **前 2-3 年**：IRR 是负的。因为管理费在消耗资金，且被投公司的价值还没有释放。这期间给投资者（LP）的报表很难看。
- **后 3-5 年**：IRR 开始快速上升。被投公司的改善开始见效，陆续通过出售/上市退出。

这就是为什么 LP 需要有"耐心资本（Patient Capital）"——不能在 J 曲线的谷底退出。

### 6. 什么公司最适合做 LBO？——"理想目标的五个特征"

不是所有公司都适合被 LBO。PE 基金寻找目标公司时，会看以下五个特征：

**① 稳定且可预测的现金流（Stable & Predictable Cash Flows）**
- 为什么重要：因为 LBO 的核心是用公司的现金流还债。如果现金流不稳定、忽高忽低，债务可能还不出来。
- 好例子：软件订阅公司（续费率高）、消费品公司、基础设施（水电煤）
- 坏例子：采矿公司（大宗商品价格波动大）、生物科技（没有收入只有研发支出）

**② 低资本支出需求（Low CapEx Requirements）**
- 为什么重要：如果公司每年赚 $100M 但要花 $80M 更新设备，那可以用来还债的钱只剩 $20M。还债效率太低。
- 好例子：软件公司、服务公司（CapEx / 营收 < 3%）
- 坏例子：制造业、航空业（CapEx / 营收可能 > 10%）

**③ 强大的市场地位和护城河（Strong Market Position & Moat）**
- 为什么重要：LBO 后公司背着大量债务，如果市场地位脆弱、客户随时可能流失，公司可能撑不过去。
- 好例子：细分市场的龙头、高客户转换成本的公司

**④ 有改善空间（Opportunities for Operational Improvement）**
- 为什么重要：PE 不只是"借钱买然后等着"，而是通过改善经营来提升公司价值。如果没有改善空间，就不值得 PE 来做。
- 改善方向：削减成本、优化定价、拓展新市场、补强收购

**⑤ 合适的估值（Attractive Valuation）**
- 为什么重要：LBO 的回报来自"买得便宜 + 经营改善 + 杠杆放大"。如果买得太贵，杠杆会放大损失而不是收益。
- 典型的 EV/EBITDA 倍数：8-12x（取决于行业和增长前景）

### 7. 经典 LBO 案例——"史上最赚钱的 PE 交易"

**案例一：Dell（戴尔）私有化——"史上最大的科技 LBO"（2013 年）**

- **背景**：2013 年，PC 市场持续萎缩，戴尔股价低迷（$13/股），市值约 $220 亿。
- **交易**：Michael Dell 本人联合 Silver Lake（银湖资本）——一家专注科技投资的 PE——以 $24.4B 将戴尔私有化退市。Michael Dell 自己出了 $750M（个人），Silver Lake 出了 $1.4B，其余靠银行贷款和高收益债。
- **为什么这么做**：Michael Dell 认为市场太短视——只盯着 PC 销量，看不到戴尔正在转型为企业 IT 解决方案（服务器、存储、软件）。在公开市场上，转型期的短期业绩波动会被"惩罚"（股价大跌）。私有化后，他可以不受季度财报压力的束缚，进行"手术式"改革。
- **结果**：
  - 2016 年，戴尔收购了 EMC（数据存储巨头）——$67B，至今仍是史上最大的科技收购案。
  - 2018 年，戴尔重新上市——此时估值约 $50B。
  - Michael Dell 的 $750M 投资变成了约 $10B（**超过 13 倍的回报**）。
  - Silver Lake 的 $1.4B 变成了约 $10B（**超过 7 倍的回报**）。

**案例二：Hilton（希尔顿酒店）——"PE 史上最赚钱的单笔交易"（2007-2018 年）**

- **背景**：2007 年，黑石集团（Blackstone）以 $26B 收购希尔顿酒店。这是私募股权史上最大（当时）的交易之一。
- **时机**：交易在 2007 年 7 月完成。仅仅几个月后，次贷危机爆发，酒店业遭受毁灭性打击，希尔顿的收入暴跌。黑石的这笔投资一度看起来是"灾难"。
- **黑石做了什么**：
  - 更换管理层，引入酒店管理明星 Chris Nassetta 担任 CEO
  - 大规模削减成本、重新谈判加盟合同
  - 在危机期间继续投资——翻修酒店、扩张品牌线
  - 利用低利率环境反复再融资，降低利息成本
- **结果**：
  - 2013 年，希尔顿重新上市（IPO），估值远超收购价。
  - 2018 年，黑石最终完全退出——总投资回报超过 **$14B**。
  - 这是 PE 历史上单笔净赚最多的一笔交易。

**案例三：RJR Nabisco——"贪婪时代的标志"（1988 年）**
- **背景**：RJR Nabisco 是美国最大的食品和烟草公司之一（旗下有奥利奥饼干、乐事薯片、骆驼香烟）。1988 年，KKR 以 $25B 收购了它。
- **传奇**：这笔交易是 1980 年代 LBO 狂潮的巅峰之作，也是畅销书《门口的野蛮人》(Barbarians at the Gate) 的主角。
- **结果**：交易过于复杂、杠杆过高。虽然 KKR 最终没有亏钱，但回报远低于预期。这笔交易成为了"过度融资"的经典反面教材。

**从这些案例学到了什么**：
- LBO 不是"买了就不管"——PE 必须主动改善经营
- 时机很重要——买得太贵、杠杆太高，神仙也救不了
- 管理团队是核心——一个对的 CEO 可能比"低估值"更重要
- 退出能力决定成败——再好的公司，如果卖不出去或无法上市，PE 也赚不到钱

### 8. LBO 模型的敏感性分析——"如果……会怎样？"

就像 DCF 一样，LBO 模型也非常依赖假设。一个标准的 LBO 敏感性分析会在 **退出倍数（Exit Multiple）vs 持有期（Holding Period）** 这两个维度上测试 IRR 和 MOIC。

**假设场景**（买入 EV/EBITDA = 10x）：

| IRR | 退出 3 年 | 退出 5 年 | 退出 7 年 |
|-----|----------|----------|----------|
| 退出倍数 = 8x | -5% | 2% | 5% |
| **退出倍数 = 10x** | **15%** | **18%** | **14%** |
| 退出倍数 = 12x | 35% | 28% | 22% |
| 退出倍数 = 15x | 70% | 45% | 30% |

**解读**：
- 如果买入时的 EV/EBITDA = 10x，卖出的 EV/EBITDA = 10x（"倍数不变"），IRR = 15-18%——还不错但不是惊艳。这主要靠还债带来的价值（debt paydown）。
- 如果买入 10x、卖出 12x（"倍数扩张"），IRR = 28-35%——非常好。这是 PE 最喜欢的"多重回报来源"。
- 如果买入 10x、卖出 8x（"倍数压缩"），IRR = -5% 到 5%——灾难级。这就是为什么 PE "买得便宜"很重要。

**PE 回报的四个来源（Value Creation Bridge）**：

```
最终权益价值 = 初始权益价值
  + EBITDA 增长（经营改善带来的利润增长）
  + 债务偿还（每年用现金流还债，权益价值自然变大）
  + 估值倍数变化（买入 10x，卖出 12x）
  - 现金流消耗（分红、收购支出等）
```

专业 PE 会做一张"价值创造桥（Value Creation Bridge）"表，把最终的回报拆解到以上四个来源。如果"倍数变化"占了回报的大头，说明你主要靠运气（市场变好了）；如果"EBITDA 增长"和"债务偿还"占了大头，说明你的经营改善是有成效的。

### 给小白的一句话

> **LBO（杠杆收购）是 PE 基金"以小博大"的核心武器——用大量借来的钱收购公司，靠公司的现金流还债，退出时赚取放大后的权益价值回报。Sources & Uses 是 LBO 的"心脏"，两边必须一分不差地对齐。MOIC = "翻了几倍"（总路程），IRR = "赚钱速度"（平均速度）。PE 基金的收费模式叫"2-20"：每年收 2% 管理费，超额收益拿 20% 业绩分成（Carry）。Dell 私有化（Michael Dell 用 $750M 变成 $10B，超过 13 倍的回报）和 Hilton（黑石在金融危机后净赚 $14B，PE 史上最赚钱的单笔交易）是最经典的 LBO 案例。一家公司是否适合做 LBO，要看五个特征：稳定现金流、低资本支出、强大护城河、有经营改善空间、买入价格合理。记住："杠杆是一把双刃剑"——公司经营好时放大收益，经营差时放大亏损，甚至还不上债可能导致破产。**
