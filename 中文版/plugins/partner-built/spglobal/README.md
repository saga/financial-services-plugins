# S&P Global 插件

本插件通过一系列预制技能，将 S&P Global 的金融数据和分析直接引入您的 AI 工作流。专为需要 AI 辅助研究、分析和文档生成的金融专业人士设计，所有数据均来源于权威的 S&P Global。

这些技能基于开放标准（MCP）构建，可在各种 AI 平台和代理框架中使用。虽然本插件遵循 Claude Cowork 标准，但所有技能及其底层数据层均与平台无关。如果您想在其它环境中使用这些技能，欢迎使用。

我们理解每个组织都有独特需求。这些技能是起点，帮助您快速构建最终产品。欢迎您根据公司的具体流程、模板和数据需求调整提示词、输出格式和工作流。

本插件中的技能按"原样"提供。生成的输出和数据不保证正确性。**务必验证 LLM 生成的输出是否准确。**

技能上传后即可在 Claude Desktop 中使用。您可以根据需要安装一个或多个。

### 1. S&P Global 是什么——金融数据的"自来水公司"

**想象一个场景**：你是一个投行分析师，明天要给客户做一份并购估值报告。你需要目标公司的财务数据、行业可比公司的估值倍数、近期的并购交易先例……这些数据从哪来？

**答案是 S&P Global**。它就像金融世界的"自来水公司"——你拧开水龙头（调用 API），数据就流出来了。没有它，投行分析师、基金经理、咨询顾问都得手动翻 SEC 文件，效率差 100 倍。

**S&P Global 的核心业务板块**：

| 业务 | 产品 | 干什么用的 | 谁在用 |
|------|------|-----------|--------|
| **市场情报** | Capital IQ Pro | 公司财务数据、估值、并购 | 投行、PE、资管 |
| **评级** | S&P Global Ratings | 给债券和公司打信用评级 | 投资者、发行人 |
| **指数** | 标普500、标普600 | 被动投资的基准 | ETF 发行人、基金经理 |
| **商品洞察** | Platts（普氏） | 大宗商品价格基准 | 能源企业、交易商 |
| **可持续** | ESG Scores | 环境/社会/治理评分 | 资管、监管 |
| **Kensho** | AI API | 大语言模型+金融数据 | 投行、资管 |

**真实例子**：你打开 Bloomberg 终端查苹果的 P/E 倍数——那个数字背后很可能就是 S&P Global 提供的数据。全球 80% 以上的金融数据终端都依赖 S&P Global 的底层数据。

### 2. Capital IQ——金融从业者的"瑞士军刀"

**Capital IQ（简称 CapIQ）** 是 S&P Global 旗下最核心的金融数据平台。你可以把它理解为一个**超级强大的 Excel + 数据库**——你想要任何公司的任何财务数据，它几乎都有。

**Capital IQ 能查什么**：

| 数据类型 | 举例 | 你什么时候需要它 |
|---------|------|-----------------|
| 公司基本信息 | 总部、行业、员工数、股权结构 | 做公司简介（Tearsheet） |
| 财务报表 | 收入、EBITDA、净利润、自由现金流 | 估值分析、财务建模 |
| 估值倍数 | P/E、EV/EBITDA、EV/收入 | 可比公司分析（Comps） |
| 一致预期 | 分析师对未来的收入/EPS 预测 | 前瞻估值（NTM） |
| 并购交易 | 交易金额、EV/EBITDA 倍数 | 先例交易分析（Precedent） |
| 关系数据 | 客户、供应商、竞争对手 | 战略分析、销售准备 |
| 信用评级 | S&P 评级、违约概率 | 信用分析、债券投资 |

**真实例子**：高盛分析师给微软做估值，需要 5 家可比公司的 EV/EBITDA（NTM）倍数。他在 Capital IQ 里输入这 5 家公司的 ticker，30 秒就拿到了数据。如果手动翻每家公司的 10-K 文件再自己算，可能要 2 小时。

**所以这意味着什么**：Capital IQ 不是"可选的"，它是金融从业者的**基础设施**。就像程序员离不开 GitHub，金融人离不开 Capital IQ。

### 3. 本插件的核心技能——Tearsheet、Earnings Preview、Funding Digest

**a) Tearsheet（公司概览页面）**：

**想象一下**：你明天要见客户，需要一份 1-2 页的公司简介。你不想花 3 小时手动做——Tearsheet 技能帮你自动生成。

它从 Capital IQ 拉取数据，按投行标准格式排版，输出包含：公司标题、业务概述、财务摘要、估值倍数、并购活动、关键关系。根据受众不同，有 4 个版本：
- **IB/M&A 版**：给投行家看，侧重估值和并购
- **Equity Research 版**：给分析师看，侧重前瞻数据和评级
- **Sales/BD 版**：给销售看，侧重战略重点和对话切入点
- **Corp Dev 版**：给企业发展战略部看，侧重收购评估

**b) Earnings Preview（财报预览）**：

公司发财报前，分析师需要一份"预告"——市场预期什么、关键关注点是什么、管理层最近说了什么。这个技能自动从 Capital IQ 拉取一致预期数据，生成预览报告。

**c) Funding Digest（融资摘要）**：

追踪各赛道初创公司的融资动态。从"行业种子"出发，扩展到整个赛道，汇总最近一轮融资的金额、轮次、投资方。

### 4. 数据完整性——为什么"没数据"比"错误数据"好

**本插件最重要的原则**：宁可留空，不可编造。

**想象一下**：你在做一份并购报告，某笔交易的金额查不到。你有两个选择：
1. 写"未披露"——客户可能觉得你不专业
2. 猜一个数字——客户可能基于错误数据做决策

**正确答案是 1**。在金融行业，基于错误数据的决策比没有数据更危险。S&P Global 的数据覆盖很广但不完美——私人公司、小型交易的数据经常缺失。遇到缺失，标注"不可用"或"未披露"，不要编造。
