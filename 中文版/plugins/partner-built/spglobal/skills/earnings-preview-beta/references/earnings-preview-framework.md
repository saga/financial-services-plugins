# 财报预览参考

## 查询框架

本参考提供了用于构建财报预览的查询框架。

### 查询1：基本信息

```
[公司] 股票代码 板块 市值 股价 52周区间
```

**目的：** 确认公司标识和当前估值水平

**预期输出：**
- 股票代码（如 NASDAQ: AAPL）
- 交易所和板块
- 市值（通常以十亿计）
- 当前股价
- 52周高低价和当前价所处的百分位

### 查询2：共识估计

```
[公司] 共识估计 收入 EPS EBITDA 下季度 华尔街估计
```

**目的：** 获取华尔街对即将发布的财报的共识预期

**预期输出：**
- 共识收入估计
- 共识EPS估计
- 共识EBITDA估计（如适用）
- 估计数量（有多少分析师发布估计）
- 最高和最低估计（范围）

### 查询3：近期股价表现

```
[公司] 股价 1个月 3个月 YTD 表现
```

**目的：** 了解近期股价走势和趋势

**预期输出：**
- 1个月回报
- 3个月回报
- YTD回报
- 与板块/指数的比较
- 近期是否有重大波动

### 查询4：历史超预期/低于预期

```
[公司] 财报 超预期 低于预期 意外 过去4个季度
```

**目的：** 识别历史超预期/低于预期模式

**预期输出：**
- 最近4-6个季度的超预期/低于预期历史
- 超预期/低于预期的幅度
- 连续超预期/低于预期的模式
- 股价对超预期/低于预期的反应

### 查询5：近期发展

```
[公司] 最新新闻 近期发展 重大公告 过去60-90天
```

**目的：** 识别可能影响即将发布财报的近期事件

**预期输出：**
- 产品发布公告
- 管理层变动
- 重大合同或合作
- 法律或监管事项
- 并购活动

### 查询6：历史财务数据

```
[公司] 季度损益表 收入 EBITDA 净利润 EPS 过去4个季度
```

**目的：** 获取历史财务数据用于比较

**预期输出：**
- 最近4个季度的收入
- 最近4个季度的EBITDA
- 最近4个季度的净利润
- 最近4个季度的EPS
- 同比变化趋势

### 查询7：管理层指引

```
[公司] 指引 上一季度指引 管理层指引 收入 EPS 下一季度
```

**目的：** 获取管理层发布的指引用于比较

**预期输出：**
- 收入指引范围
- EPS指引范围
- 其他关键指引指标
- 指引与共识的比较

### 查询8：细分数据

```
[公司] 细分收入 业务单元 产品线 过去4个季度
```

**目的：** 获取按细分分类的收入数据

**预期输出：**
- 按业务单元分类的收入
- 按地理区域分类的收入（如有）
- 按产品线分类的收入（如有）
- 各细分的增长趋势

## 数据解读指南

### 共识估计解读

- **窄范围 vs 宽范围：** 窄范围表示分析师高度共识；宽范围表示不确定性
- **趋势：** 估计是否在过去几周上调或下调？
- **vs 指引：** 共识是否高于或低于管理层指引？

### 超预期/低于预期解读

- **持续性：** 公司是否经常超预期或低于预期？
- **幅度：** 超预期/低于预期的幅度是否很大？
- **反应：** 股价通常如何反应超预期/低于预期？

### 估值解读

- **当前估值：** 与历史范围相比如何？
- **增长预期：** 当前的估值是否反映了高增长预期？
- **风险/回报：** 在当前价位，风险/回报比是否有利？

## 财报前检查清单

在发布预览前，验证：

- [ ] 公司标识符（股票代码、名称）正确
- [ ] 财报日期确认
- [ ] 共识估计可用
- [ ] 历史超预期/低于预期数据可用
- [ ] 有可形成交易建议的数据
- [ ] 预览在1-2页以内
- [ ] 交易建议清晰且可操作

### 1. 公司标识符与基础信息

#### 1.1 股票代码体系

**定义**：股票代码（Ticker Symbol）是交易所分配给上市公司的唯一字母缩写，用于在交易系统中识别公司。

**美国主要交易所**：

| 交易所 | 全称 | 代码后缀 | 特点 |
|--------|------|---------|------|
| **NYSE** | New York Stock Exchange | 无 | 传统大公司居多 |
| **NASDAQ** | National Association of Securities Dealers Automated Quotations | 无 | 科技公司为主 |
| **NYSE Arca** | - | 无 | ETF 为主 |
| **CBOE** | Chicago Board Options Exchange | 无 | 衍生品 |

**股票代码示例**：

| 公司 | 代码 | 交易所 |
|------|------|--------|
| Apple | AAPL | NASDAQ |
| Microsoft | MSFT | NASDAQ |
| NVIDIA | NVDA | NASDAQ |
| Tesla | TSLA | NASDAQ |
| JPMorgan | JPM | NYSE |
| Exxon Mobil | XOM | NYSE |
| Berkshire Hathaway | BRK.B | NYSE |
| Amazon | AMZN | NASDAQ |

**其他市场代码**：

| 市场 | 代码示例 | 后缀 |
|------|---------|------|
| **香港** | 00700（腾讯） | .HK |
| **中国 A 股** | 600519（贵州茅台） | .SH（上证）/ .SZ（深证） |
| **伦敦** | BP. | .L |
| **东京** | 7203（丰田） | .T |
| **法兰克福** | BMW | .DE |

#### 1.2 板块分类（GICS）

**定义**：GICS（Global Industry Classification Standard，全球行业分类标准）由 S&P 与 MSCI 共同制定，将所有上市公司分为 11 大板块、24 个行业组、69 个行业和 158 个子行业。

**11 大板块**：

| 代码 | 中文 | 英文 |
|------|------|------|
| 10 | 能源 | Energy |
| 15 | 原材料 | Materials |
| 20 | 工业 | Industrials |
| 25 | 非必需消费品 | Consumer Discretionary |
| 30 | 必需消费品 | Consumer Staples |
| 35 | 医疗保健 | Health Care |
| 40 | 金融 | Financials |
| 45 | 信息技术 | Information Technology |
| 50 | 通信服务 | Communication Services |
| 55 | 公用事业 | Utilities |
| 60 | 房地产 | Real Estate |

**用途**：
- 同业对比（peer comparison）
- 板块 ETF 构建（XLK、XLF 等）
- 投资风格分类
- 风险敞口分析

#### 1.3 市值分级

**定义**：市值（Market Capitalization）= 股价 × 流通股数。

**分级标准**：

| 类型 | 市值范围 | 特点 |
|------|---------|------|
| **Mega Cap** | > $200B | 苹果、微软、沙特阿美等 |
| **Large Cap** | $10B - $200B | 行业龙头 |
| **Mid Cap** | $2B - $10B | 成长型公司 |
| **Small Cap** | $300M - $2B | 早期成熟公司 |
| **Micro Cap** | $50M - $300M | 小型公司 |
| **Nano Cap** | < $50M | 极小盘 |

**Russell 指数分层**：
- **Russell 1000**：美国大盘 1000 大
- **Russell 2000**：美国小盘 2000 大
- **Russell 3000**：Russell 1000 + 2000

#### 1.4 52 周区间与百分位

**定义**：52 周高低价（52-Week Range）是过去 52 周内股票交易的最高价和最低价，反映股票在这一年的波动区间。

**百分位解读**：

| 当前价位置 | 含义 |
|-----------|------|
| **> 90%** | 接近 52 周高点，强势 |
| **50-90%** | 中高位 |
| **10-50%** | 中低位 |
| **< 10%** | 接近 52 周低点，弱势 |

**为什么 52 周区间重要**：
- 心理锚定：投资者常在 52 周高/低做出决策
- 技术分析：突破 52 周高是买入信号
- 风险评估：波动区间大 = 风险高
- 期权定价：影响期权希腊值

### 2. 共识估计（Consensus Estimates）深度解析

#### 2.1 数据来源

**主要数据提供商**：

| 提供商 | 母公司 | 数据特点 |
|--------|--------|---------|
| **Refinitiv I/B/E/S** | LSEG（原 Refinitiv） | 历史悠久，覆盖广 |
| **FactSet** | FactSet | 高端机构客户 |
| **S&P Capital IQ** | S&P Global | 数据全面 |
| **Bloomberg** | Bloomberg L.P. | 实时数据 |
| **Zacks** | Zacks Investment Research | 风格化分类 |
| **Visible Alpha** | Visible Alpha | 独立预测 |
| **Wolfe Research** | Wolfe | 卖方整合 |

**数据更新频率**：
- 实时（财报前后高频更新）
- 每日（一般情况）
- 财报季前 2-4 周：调整最频繁
- 财报季期间：每天多次

#### 2.2 共识统计方法

**中位数 vs 平均数**：

| 方法 | 计算 | 优点 | 缺点 |
|------|------|------|------|
| **中位数** | 排序取中间 | 不受极端值影响 | 忽略数据分布 |
| **算术平均** | 求和 ÷ 个数 | 简单直观 | 受极端值影响 |
| **加权平均** | 按历史准确度加权 | 最准确 | 需历史数据 |
| **几何平均** | 乘积开 n 次方 | 适合比率 | 不常用 |

**行业标准**：主流数据商默认用中位数，但会提供其他统计量。

#### 2.3 时间维度

| 缩写 | 含义 | 用途 |
|------|------|------|
| **NTM** | Next Twelve Months，未来 12 个月 | 主流估值基准 |
| **NTQ** | Next Three Quarters，未来 3 个季度 | 短期估值 |
| **FY+0** | 当年 | 当前年度预测 |
| **FY+1** | 明年 | 主流远期 |
| **FY+2** | 后年 | 长期预测 |
| **LTG** | Long-Term Growth，长期增长率 | DCF 永续增长 |

#### 2.4 估计"健康度"指标

**估计分散度（Dispersion）**：
```
分散度 = (最高 - 最低) / 中位数
```

| 分散度 | 含义 |
|--------|------|
| < 5% | 高度共识 |
| 5-15% | 一般共识 |
| 15-30% | 较大分歧 |
| > 30% | 高度不确定 |

**估计修订比（Revision Ratio）**：
- 过去 3 个月，上调数 ÷ 下调数
- > 1.5：分析师乐观
- 0.8-1.2：中性
- < 0.8：分析师悲观

**时间衰减**：
- 距离财报日越近，估计越准
- 财报前 1 周的共识最可靠
- 财报前 1 月的估计可能大幅调整

### 3. 股价表现分析

#### 3.1 回报率时间框架

| 周期 | 英文 | 适用场景 |
|------|------|---------|
| **日内** | Intraday | 短线交易 |
| **5 日** | 5-Day | 短期动能 |
| **1 月** | 1-Month | 短期趋势 |
| **3 月** | 3-Month | 中期趋势 |
| **6 月** | 6-Month | 中期 |
| **YTD** | Year-to-Date | 年度表现 |
| **1 年** | 1-Year | 长期 |
| **3 年** | 3-Year | 周期对比 |
| **5 年** | 5-Year | 长期投资 |
| **10 年** | 10-Year | 超长期 |

#### 3.2 基准对比（Beta）

**定义**：Beta（β）衡量股票相对于市场的波动性。

**计算**：
```
β = Cov(股票收益, 市场收益) / Var(市场收益)
```

| Beta 值 | 含义 |
|---------|------|
| β = 0 | 与市场无关（如公用事业） |
| 0 < β < 1 | 防御性（波动小于市场） |
| β = 1 | 与市场同步 |
| β > 1 | 激进型（波动大于市场） |
| β < 0 | 反向（如黄金股） |

**典型 Beta**：

| 公司 | Beta | 类型 |
|------|------|------|
| Apple (AAPL) | 1.25 | 偏激进 |
| Microsoft (MSFT) | 0.90 | 接近市场 |
| Tesla (TSLA) | 2.10 | 高度激进 |
| Coca-Cola (KO) | 0.60 | 防御性 |
| Utilities ETF (XLU) | 0.45 | 防御性 |

#### 3.3 波动性指标

**标准差（Standard Deviation）**：
- 衡量日/周收益的波动幅度
- 股票年化波动率：日波动率 × √252
- 10-20%：低波动
- 20-30%：中等
- > 30%：高波动

**最大回撤（Max Drawdown）**：
- 历史最高点到最低点的跌幅
- 衡量极端风险
- Apple 2022 回撤：-27%
- Tesla 2022 回撤：-65%

**夏普比率（Sharpe Ratio）**：
```
夏普 = (收益 - 无风险利率) / 标准差
```

- > 1：良好
- > 2：优秀
- > 3：卓越

### 4. 财报超预期/不及预期模式

#### 4.1 Beat 比率（Beat Rate）

**定义**：在过去 N 个季度中超预期的比例。

| 行业典型 | Beat 比率 | 说明 |
|---------|---------|------|
| **软件 SaaS** | 75-85% | 经常超预期 |
| **消费必需品** | 70-80% | 较稳定 |
| **金融** | 65-75% | 受利率影响 |
| **半导体** | 60-70% | 周期性 |
| **生物科技** | 50-60% | 难预测 |
| **零售** | 55-65% | 消费波动 |
| **能源** | 50-60% | 油价主导 |

#### 4.2 连续超预期（Streak）

**统计意义**：
- 连续 4 季度超预期：稳健
- 连续 8 季度超预期：优秀
- 连续 12 季度超预期：行业领先
- Apple 长期保持 8 季以上连续 Beat

**"Streak 风险"**：
- 投资者开始"预期超预期"
- 共识预期不断上调
- 难度越来越高
- 一旦中断，市场反应剧烈

#### 4.3 Beat/Miss 幅度（Surprise %）

**计算**：
```
Surprise % = (实际 - 共识) / 共识
```

| Surprise | 影响 |
|----------|------|
| > +10% | 重大超预期 |
| +2% to +10% | 稳健超预期 |
| -2% to +2% | 符合预期 |
| -10% to -2% | 不及预期 |
| < -10% | 严重不及预期 |

**收入 vs EPS 超预期**：
- 收入超预期 + EPS 符合预期：质量一般
- 收入符合 + EPS 超预期：靠成本控制
- 双双超预期：最佳

#### 4.4 股价对 Beat/Miss 的反应

**反应曲线**：

| 事件 | 1 天反应 | 1 月反应 |
|------|---------|---------|
| 重大 Beat（>+5%） | +5-15% | +10-25% |
| 小幅 Beat | +1-3% | +2-5% |
| In-line | ±1% | ±2% |
| 小幅 Miss | -3-7% | -5-10% |
| 重大 Miss | -10-20% | -15-30% |

**长期表现研究**：
- 财报后 1 周内股价常"回归基本面"
- 财报后 3-6 个月表现与基本面更相关
- Beat + 指引上调：长期最佳
- Miss + 指引下调：长期最差

### 5. 管理层指引分析

#### 5.1 指引更新模式

**正向指引（Guidance Raise）**：
- 指引上调 = 重大利好
- 通常伴随股价跳涨
- 表明管理层信心强
- 引发分析师上调预测

**负向指引（Guidance Cut）**：
- 指引下调 = 重大利空
- 股价通常下跌
- 可能预示行业拐点
- 引发分析师下调

**指引撤回**：
- 不给具体数字
- 投资者失去锚定
- 短期可能涨（管理灵活性）
- 长期可能跌（不透明）

#### 5.2 指引 vs 共识的差距

**Beat Consensus, Miss Guidance**：
- 实际超预期但指引低于预期
- 短期股价可能下跌
- 长期投资者需关注指引

**Miss Consensus, Beat Guidance**：
- 实际不及预期但指引超预期
- 短期股价可能下跌
- 长期看指引更有意义

**Both Beat**：
- 实际 + 指引都超预期
- 最佳组合
- 长期股价表现最强

#### 5.3 指引可信度

**"Beat and Raise"**（超预期+上调指引）：
- 最佳结果
- Apple、Nvidia、Microsoft 经常如此

**"Beat and Cut"**（超预期+下调指引）：
- 不利信号
- 暗示未来增长放缓
- 常见于周期股

**"In-line and Raise"**（符合+上调）：
- 管理层信心强
- 长期正面

**"Miss and Cut"**（不及+下调）：
- 重大利空
- 业务恶化的双重确认

### 6. 财务数据查询语义

#### 6.1 利润表查询

**核心字段**：

| 字段 | 英文 | 数据源 |
|------|------|--------|
| **收入** | Revenue/Sales/Turnover | 10-Q/10-K |
| **销售成本** | Cost of Goods Sold (COGS) | 10-Q/10-K |
| **毛利** | Gross Profit | 10-Q/10-K |
| **营业费用** | Operating Expenses | 10-Q/10-K |
| **销售费用** | SG&A | 10-Q/10-K |
| **研发费用** | R&D Expense | 10-Q/10-K |
| **营业利润** | Operating Income | 10-Q/10-K |
| **利息费用** | Interest Expense | 10-Q/10-K |
| **税前利润** | Pre-tax Income | 10-Q/10-K |
| **所得税** | Income Tax | 10-Q/10-K |
| **净利润** | Net Income | 10-Q/10-K |
| **EPS** | Earnings Per Share | 10-Q/10-K |
| **摊薄 EPS** | Diluted EPS | 10-Q/10-K |

**注意事项**：
- Non-GAAP 调整项
- 一次性损益
- 外汇影响
- 报告期与日历期差异

#### 6.2 关键比率

| 比率 | 公式 | 含义 |
|------|------|------|
| **毛利率** | 毛利 / 收入 | 定价能力 |
| **营业利润率** | 营业利润 / 收入 | 经营效率 |
| **净利润率** | 净利润 / 收入 | 最终盈利能力 |
| **EBITDA 率** | EBITDA / 收入 | 现金盈利能力 |
| **ROE** | 净利润 / 股东权益 | 股东回报率 |
| **ROA** | 净利润 / 总资产 | 资产效率 |
| **ROIC** | NOPAT / 投入资本 | 投入资本回报 |

### 8. 估值倍数与基准

#### 8.1 估值倍数分类

**盈利能力倍数**：

| 倍数 | 公式 | 适用 |
|------|------|------|
| **P/E** | 股价 / EPS | 成熟盈利公司 |
| **Forward P/E** | 股价 / NTM EPS | 远期估值 |
| **PEG** | P/E / 增长率 | 成长股 |
| **EV/EBIT** | 企业价值 / EBIT | 通用 |
| **EV/EBITDA** | 企业价值 / EBITDA | 通用（剔除折旧影响） |

**收入倍数**：

| 倍数 | 公式 | 适用 |
|------|------|------|
| **P/S** | 股价 / 每股收入 | 亏损公司 |
| **EV/Revenue** | 企业价值 / 收入 | SaaS、亏损公司 |
| **EV/ARR** | 企业价值 / 年化订阅 | SaaS |
| **EV/Gross Profit** | 企业价值 / 毛利 | 高毛利业务 |

**资产倍数**：

| 倍数 | 公式 | 适用 |
|------|------|------|
| **P/B** | 股价 / 每股净资产 | 银行、保险 |
| **P/TBV** | 股价 / 每股有形净资产 | 银行 |
| **EV/IC** | 企业价值 / 投入资本 | 重资产 |

| **必需消费** | 18-25x | 12-18x | 2-4x |

#### 8.3 估值差距分析

**历史估值带**：

| 状态 | 当前估值 | 历史 P/E |
|------|---------|---------|
| **深度低估** | < 历史 25% 分位 | < 15x |
| **低估** | 25-40% 分位 | 15-20x |
| **合理** | 40-60% 分位 | 20-25x |
| **高估** | 60-75% 分位 | 25-30x |
| **深度高估** | > 75% 分位 | > 30x |

**估值理由**：
- 高估时：需找出增长溢价理由
- 低估时：需识别市场担忧

### 9. 数据解读方法论

#### 9.1 共识范围解读

**窄范围（< 5% 分散度）**：
- 高度共识
- 任何偏离都构成意外
- 股价反应可能放大
- 适合趋势交易

**宽范围（> 20% 分散度）**：
- 高度不确定
- 实际值对股价影响较小
- 适合"惊喜交易"
- 需深入研究隐藏假设

#### 9.2 趋势解读

**估计上调趋势**：
- 过去 30 天：上调数 > 下调数 = 利好
- 上调幅度：> 2% = 显著
- 推动者：哪家投行上调？

**估计下调趋势**：
- 过去 30 天：下调数 > 上调数 = 利空
- 下调幅度：> 2% = 显著
- 警示信号

**拐点识别**：
- 连续上调后转下调 = 拐点
- 反之亦然
- 通常领先于实际数据

#### 9.3 估值与增长的匹配

**PEG 比率**：
```
PEG = (P/E) / 盈利增长率%
```

| PEG | 含义 |
|-----|------|
| < 1 | 低估或增长可持续性差 |
| 1-2 | 合理 |
| > 2 | 高估或增长确定性高 |

**Yardeni 模型**：
```
Fair P/E = (盈利增长率 + 股息率) / 长期国债收益率
```

- 当前 < Fair P/E：低估
- 当前 > Fair P/E：高估

#### 9.4 风险/回报分析

**不对称机会**：
- 上涨空间 > 下跌空间 = 买入机会
- 通常认为 > 2:1 比例较好

**情景分析**：

| 情景 | 概率 | 目标价 | 期望值 |
|------|------|--------|--------|
| **乐观** | 30% | $200 | $60 |
| **基准** | 50% | $150 | $75 |
| **悲观** | 20% | $100 | $20 |
| **总期望值** | 100% | - | $155 |

## Appendix: 金融背景知识

这份文件是"财报预览框架（Earnings Preview Framework）"参考文档的详细说明。在股权研究中，财报预览（Earnings Preview）是分析师在公司发布季度财报前撰写的预测报告——它告诉客户"我们预计这家公司会交出什么样的成绩单，以及你应该在财报前做什么准备"。

---

### 1. 什么是 Earnings Preview？

**类比：**
想象你对即将到来的高考预测：你根据某学生的模拟考试成绩、最近的学习状态、对手的实力，预测他最终能考多少分。

Earnings Preview 就是金融分析师对公司即将发布财报的"考前预测"——预计收入多少、利润多少、有什么需要注意的。

---

### 2. Earnings Preview 的"四大预测支柱"

| 支柱 | 内容 | 来源 |
|------|------|------|
| 收入预测 | 公司这个季度卖了多少钱 | 行业数据、管理层指引、渠道调研 |
| 利润预测 | 扣完成本后赚了多少 | 模型推算、费率假设 |
| EPS 预测 | 每股利润 | 利润 ÷ 股数 |
| 对比分析 | 预计 vs 市场共识（Consensus） | 高于/低于华尔街预期的含义 |

---

### 3. "高于/低于共识"的重要性

| 情况 | 英文 | 市场含义 | 原因 |
|------|------|---------|------|
| 高于共识 | Beat | 股价通常涨 | 公司比预想的更好 |
| 符合共识 | In Line | 股价波动不大 | 没惊喜也没惊吓 |
| 低于共识 | Miss | 股价通常跌 | 公司比预想的更差 |

**关键洞察**：股价的反应取决于"实际 vs 预期"，而不是"实际 vs 去年"。

**例子**：公司收入增长 30%（vs 去年），但华尔街预期增长 35% → 股价可能跌。因为虽然增长很快，但"没有达到华尔街的期待"。

---

### 4. Earnings Preview 的"标准结构"

```
标题：[公司名] | 2024 Q3 Earnings Preview
日期：2024年10月15日
评级：Buy | 目标价：$50

一、核心预测
  - 收入：$1.2B（vs Consensus $1.15B，Beat）
  - EPS：$0.85（vs Consensus $0.80，Beat）
  - 关键假设：新 iPhone 销量 5000 万台

二、重点关注
  - 毛利率指引：43% vs 42% consensus
  - 中国市场表现：预计恢复增长
  - 新业务：Vision Pro 数据

三、风险提示
  - 供应链中断风险
  - 汇率波动影响
  - 竞争加剧

四、估值与建议
  - 当前价 $42，目标价 $50（上涨 19%）
  - 建议：在财报前买入
```

---

### 5. "财报季"的精髓：预期管理

**三种典型情景：**

| 情景 | 实际 vs 共识 | 股价反应 | 例子 |
|------|------------|---------|------|
| 超预期 | Beat 5% | 大涨 10% | NVIDIA Q2 2023 |
| 符合预期 | In Line | 微涨/微跌 | Apple Q4 2023 |
| 低于预期 | Miss 5% | 大跌 15% | Tesla Q4 2022 |

**教训**：在"财报季"，市场不看绝对值，看的是"预期差"——你比"别人以为的"好还是差？

---

### 6. 真实案例：NVIDIA 2023 Q2 Earnings Preview

| 预测项 | 分析师预测 | 市场共识 | 差距 |
|--------|-----------|---------|------|
| 收入 | $12B | $11B | Beat 9% |
| 数据中心收入 | $8.5B | $7.5B | Beat 13% |
| 毛利率 | 71% | 70% | 微 Beat |
| EPS | $2.50 | $2.07 | Beat 21% |

**结果**：财报发布当天股价跳涨 24%。NVIDIA 市值一天增加 $300B——比 Nike 的总市值还大。

---

### 给小白的一句话

> Earnings Preview 就是金融分析师的"高考预测卷"——你根据公司过去的表现、管理层的"风声"、行业的变化，预测公司即将发布的季度报会是什么成绩。最关键的三个数字是：收入、EPS、收入增长率。它们和"华尔街共识"的比较结果决定了股价是涨还是跌。重要洞察：公司增长 50% 不一定让股价涨（如果华尔街预期增长 60%），公司增长 5% 不一定让股价跌（如果华尔街预期只增长 2%）。股价反应永远是"实际 vs 预期"——这也是为什么 Earnings Preview 如此重要，因为它帮你提前"猜透"市场在想什么。
