# 股权研究概览页面

## 用途
为买方或卖方分析师评估投资而设计的信息密集型快照。侧重于估值、前瞻预期、财务轨迹和分析师一致预期。一切内容都应支持或挑战投资论点。

**默认页数：** 1 页。股权研究概览页传统上是单页——信息密度是卖点。如用户要求更多篇幅，可扩展到 2 页。

## 查询计划

从以下查询开始。如结果不完整，拆分为更窄的后续查询。

**查询 1 — 简介 + 市场数据：**
"[公司] 公司概况 行业 市值 企业价值 股价 52周高低 流通股数 Beta"
→ 标题、业务描述
→ **立即写入** `/tmp/tear-sheet/company-profile.txt`

**查询 2 — 历史财务数据：**
"[公司] 年度利润表 收入 毛利润 EBITDA 净利润 EPS 自由现金流 总债务 现金及等价物 最近4个财年"
→ 财务摘要（拉取4年，展示3年；最早一年仅用于同比增速计算）
→ **立即写入** `/tmp/tear-sheet/financials.csv`

**查询 2b — 分部数据（如可用）：**
"[公司] 收入按业务分部 最近2个财年"
→ 收入与分部结构（需要2年数据计算同比增速）。如无分部数据，跳过此节——不要留空白表格。
→ **立即写入** `/tmp/tear-sheet/segments.csv`（无分部数据则跳过）

**查询 3 — 估值 + 一致预期：**
"[公司] P/E EV/EBITDA EV/收入 估值倍数 一致预期收入 EPS 分析师评级 目标价"
→ 估值快照、一致预期
→ **立即写入** `/tmp/tear-sheet/valuation.csv`
→ **立即写入** `/tmp/tear-sheet/consensus.csv`

**查询 4 — 财报：**
"[公司] 最近一次财报电话会 关键要点 指引"
→ 财报亮点
→ **立即写入** `/tmp/tear-sheet/earnings.txt`

**查询 5 — 股价表现：**
"[公司] 股价回报 1个月 3个月 6个月 1年 年初至今"
→ 股价表现（无中间文件——数据直接用于文档）

**查询 6（如用户提供了可比公司）：**
"[同一家公司] 市值 EV/收入 EV/EBITDA 收入增长" — 每家公司重复
→ 估值快照的同行背景
→ **立即写入** `/tmp/tear-sheet/peer-comps.csv`

## 章节

按优先级排序。如需限制在一页，从底部开始裁剪。

### 1. 公司标题
紧凑的关键值块，按全局样式配置渲染为两列无边框表格。

左列：股票代码（交易所）、行业/子行业、总部
右列：股价、52周范围、市值、EV、流通股数、Beta

### 2. 业务描述
2-3 句紧凑描述。**以分析师受众的角度用自己的话重写**——不要直接粘贴 CIQ 公司摘要。CIQ 摘要是输入，输出应简洁且以论点为导向。

对于知名度高、广泛覆盖的公司，假设分析师已经了解基本业务。以**正在变化**的内容开头——战略转型、投资组合重塑、新的增长方向——而不是维基百科式的概述。例如，对一家财富500强公司，不要花一句话写"为机构提供金融数据"。改为："正在将其投资组合转向更高利润的数据和分析业务，有正在进行的 Mobility 分拆计划，并通过其 Kensho 子公司加大 AI 投资。"

对于不太知名的公司，在转向论点相关的动态之前，可以写一句简短的"他们做什么"。

### 3. 估值快照
股权概览页的核心。

| 指标 | 回溯（Trailing） | 前瞻（NTM） |
|---|---|---|
| P/E | | |
| EV/EBITDA | | |
| EV/收入 | | |
| P/FCF | | |
| 股息率 | | |

**有前瞻数据时，前瞻倍数必须显示。** 只有回溯倍数而前瞻数据存在是重大缺失——分析师用前瞻盈利来估值。如前瞻倍数不可用，仅显示回溯倍数并注明"远期预测不可用"。

如用户提供了可比公司，为每家公司加一列（如有3家以上可比公司则加"同行中位数"列）。如用户未提供可比公司，而工具返回同行数据，包含同行中位数列。两者都不可用时，仅显示公司自身倍数。

### 4. 一致预期
市场在预期什么——这是此类概览页的关键差异点。

**数据可用性说明：** 一致预期数据的深度因公司和覆盖范围而异。包含工具返回的任何数据——分析师数量、目标价、买入/持有/卖出分布——但不要编造或估算任何一致预期数据。对于覆盖较少的公司，此节可能仅包含收入和 EPS 预测，仍然有价值。

| 指标 | FY[年]预测 | FY[年+1]预测 |
|---|---|---|
| 收入 | | |
| EPS（调整后） | | |
| EBITDA | | |

尽可能使用调整后 EPS——GAAP EPS 可能被一次性项目扭曲，前瞻预测使用价值较低。

在主要预测表下方，包含一个紧凑的分析师一致意见块（如工具提供数据）：

| 分析师一致意见 | |
|---|---|
| 平均目标价 | XXX美元 |
| 预测数量 | XX |
| 买入 / 持有 / 卖出 | XX / XX / XX |

如目标价或评级数据不可用，省略分析师一致意见块而非显示空行。不要编造这些数字。

如一致预期数据完全不可用，跳过整个小节并注明"一致预期不可用"。不要估算。

### 6. 收入与分部结构
如分部数据可用，包含紧凑的分部收入表。这对股权研究分析师至关重要——他们需要看哪些业务线在推动增长。

| 分部 | 收入（百万美元） | 占总收入% | 同比增速 |
|---|---|---|---|
| [分部A] | | | |
| [分部B] | | | |

为一页概览保持紧凑：仅表格，无定性段落。如无分部数据，完全跳过此节——不要包含空白或占位表。

### 7. 近期财报亮点
包含季度和日期（如"Q3 FY2025 — 2025年10月"）。以**投资视角**的3-4条要点——这个受众想要分部层面的具体数据，而非战略主题：

**如果有指引，首先放在前面。** 如管理层发布了前瞻指引，将其作为第一条要点，用加粗的**指引：**前缀（例如"**指引：** FY2026 EPS $19.40-$19.65，隐含 9-10%增长；有机收入增长 6-8%。"）。指引是分析师可获得的最可操作的远期数据点——不要将其埋没在经营亮点中。

**超越/不及背景（数据许可）：** 如报告期的一致预期数据可用，将标题结果要点表述为超越/不及："收入 X 超越/不及一致预期 Y，差距 Z%。"如无报告期的一致预期数据，仅用绝对值和增长率描述结果——不要编造一致预期数字。

然后：
- 分部表现：哪些业务线加速或减速
- 利润率轨迹：关于成本结构或盈利趋势的评论

归属于管理层："管理层强调…"或"CFO 指出…"，而非陈述性表述。这部分应像财报回顾，而非新闻稿摘要。

### 8. 关键经营指标
2-3 个行业相关的 KPI（如数据可用）：订户数、同店销售、NIM 等。如无行业特定指标，从已检索的财务数据中计算 ROE 和 ROIC。可选——空间紧张时删除。

## 格式说明
- **这是最密集的概览页类型。** 应明显比其他模板更紧凑。
- 财务摘要表：8pt 字体（size: 16），行高 240 DXA。最紧凑的间距。
- 估值快照和一致预期：8.5pt 字体，占据视觉中心——分析师首先扫读这些。
- 正文：8.5pt（size: 17）— 小于全局默认的 9pt。每一半磅对一页纸都很重要。
- 章节间距：最小化。节标题前 6pt（非全局的 12pt），分割线后 2pt。
- 任何显示显著变化的指标（增长转正、利润率扩张/压缩）用加粗。
- 处处优先信息而非空白。

## 📖 本篇金融知识小贴士

**回溯（Trailing）**：基于过去12个月（LTM）实际数据的指标。反映"已经发生了什么"。

**前瞻（NTM，Forward）**：基于未来12个月的预期数据的指标。反映"大家预期会发生什么"。投资决策本质上是前瞻性的。

**P/FCF（市现率，Price to Free Cash Flow）**：股价 ÷ 每股自由现金流。类似P/E但用自由现金流替代净利润——更能反映公司"真正创造现金"的能力。

**分红收益率（Dividend Yield）**：每股分红 ÷ 股价。类似"股票给你多少利息"。

**GAAP vs 调整后（Adjusted）**：GAAP（通用会计准则）是法定标准。调整后通常是公司自称的"非GAAP"版本——排除了一次性项目。"调整后"数字通常比GAAP更好看。

**同店销售（Same-Store Sales, SSS）**：零售业关键指标——已经开业超过一年的店铺的销售变化。排除了新开门店的影响，更真实反映业务有机增长。

**NIM（净息差，Net Interest Margin）**：银行赚取的贷款利率 - 支付的存款利率。银行的"毛利率"。

**OHLCV**：开盘价（Open）、最高价（High）、最低价（Low）、收盘价（Close）、成交量（Volume）。K线图（蜡烛图）的基础数据。

## Appendix: 金融背景知识

读这份文件的时候，你会被一堆 P/E、EV/EBITDA、共识预期、目标价、NTM、LTM 这样的词包围。第一次看像天书，但讲穿了就是一件事：**怎么判断一家公司现在卖得贵不贵、未来还能不能赚钱**。这份附录就是来帮你把这件事讲明白的。

### 1. 先搞清楚"股权研究"是个什么工作

**想象一个场景**：你的朋友想花 100 万买一家奶茶店 20% 的股份，你得帮他看清楚这家店到底值不值 500 万。这时候你要做的事就是"股权研究"——给一家公司做体检，判断它值多少钱、以后能不能持续赚钱。

现实里的股权研究有两类人：

- **卖方研究（Sell-side Research）**：就是投行里那些写报告的分析师，比如高盛、摩根士丹利、摩根大通、中金。他们的客户是基金和上市公司，报告是公开的。这类机构覆盖 200-1000 只股票，每天就是"看公司、写报告、跟客户沟通"。他们既是裁判员（出投资评级），又是半运动员（因为他们家投行部门可能在帮公司做 IPO 或并购）。所以 2010 年以后，监管要求他们把研究和投行业务隔开。
- **买方研究（Buy-side Research）**：贝莱德（BlackRock，AUM 10 万亿美元）、桥水、先锋领航这些大型基金内部的研究员。他们的报告不公开，只给自己公司的基金经理用。所以他们通常只深入看 20-50 只自己真正会买的股票，更深、更专注。

两者的区别一句话总结：**卖方研究是"我写给你看"，买方研究是"我自己拍板用"**。

#### 1.1 投资评级是怎么定的

最常见的 3 档评级：

- **买入（Buy/Outperform）**：未来 12 个月预期回报 > 15%
- **持有（Hold/Neutral）**：未来 12 个月预期回报在 -10% 到 +15% 之间
- **卖出（Sell/Underperform）**：未来 12 个月预期回报 < -10%

也有投行用 5 档：强烈买入/买入/持有/卖出/强烈卖出。MiFID II（欧盟的金融法规）还要求投行公布自己评级的分布——典型是 40-50% 买入 / 40-50% 持有 / 5-15% 卖出。如果哪家投行 80% 都是"买入"，那你得想想了：是真的全都好，还是分析师怕得罪上市公司不敢写卖出？

#### 1.2 分析师的一天长什么样

财报季尤其累：

- 6:00-7:00 看新闻、刷邮件
- 7:00-8:00 准备当天材料
- 8:00 美股开盘
- 8:00-9:00 客户电话（"你们怎么看特斯拉？"）
- 9:00-12:00 写报告
- 13:00-17:00 客户会议
- 17:00-18:00 复盘、写报告
- 18:00 后还要继续

### 2. 估值——"这家公司到底值多少钱"

#### 2.1 估值的核心思路

**想象一下**：你要花 100 万开一家奶茶店。店开起来后预计一年能赚 20 万。问题来了——你愿意花多少钱把它盘下来？

如果你觉得 100 万买下来合理，那等于你用 5 年的盈利回本。投资圈把这种"几年回本"叫**估值倍数**。对股票来说，最常用的就是**市盈率（P/E）**，也就是"花多少钱买 1 块钱的盈利"。

举个真实的例子：

- 苹果公司 2024 财年（截至 2024 年 9 月）净赚约 940 亿美元。如果当时市值约 3.5 万亿美元，那 P/E ≈ 37 倍。意思是市场愿意付 37 块钱"买"苹果未来 1 年的利润。
- 同期可口可乐的 P/E 大概在 25 倍——市场认为可口可乐赚 1 块只值 25 块。
- 而特斯拉因为增长快、波动大，P/E 一度冲到 100 多倍（按当时盈利算），甚至在某些年份盈利为负，根本算不出 P/E。

**P/E 高不代表贵，低不代表便宜**。高 P/E 通常意味着市场认为公司未来赚得更多；低 P/E 可能是真便宜，也可能是"大家都觉得这公司不行了"。

#### 2.2 回溯 vs 前瞻——两个完全不同的视角

- **回溯（Trailing，LTM = Last Twelve Months）**：过去 12 个月实际赚了多少。反映"已经发生了什么"。
- **前瞻（Forward，NTM = Next Twelve Months）**：未来 12 个月预计赚多少。反映"市场预期会发生什么"。

举个具体例子：英伟达 2024 年中市值一度突破 3 万亿美元，静态市盈率（LTM P/E）大概 60-70 倍，但分析师对它未来一年的盈利预期非常高，所以远期市盈率（NTM P/E）可能只有 30-40 倍。这个差距本身就告诉了你——市场在为它的未来买单。

为什么前瞻更重要？因为**投资本质上是赌未来**。你已经赚到的钱是历史，关键是接下来能不能继续赚。

#### 2.3 估值倍数家族

| 倍数 | 怎么算 | 适合谁 |
|------|--------|--------|
| **P/E（市盈率）** | 股价 ÷ 每股盈利，或市值 ÷ 净利润 | 大多数盈利稳定的公司 |
| **EV/EBITDA** | 企业价值 ÷ 息税折旧摊销前利润 | 适合资本结构差异大的公司对比（比如重资产 vs 轻资产） |
| **EV/收入** | 企业价值 ÷ 收入 | 适合还没盈利的 SaaS、生物科技 |
| **P/FCF** | 股价 ÷ 每股自由现金流 | 适合真实现金流好的公司，比 P/E 难造假 |
| **Dividend Yield（股息率）** | 每年股息 ÷ 股价 | 适合稳定派息的公司 |

**EBITDA** 听起来高大上，其实就是"在付利息、所得税、机器设备折旧之前，公司的进账"。比如你开奶茶店，扣掉牛奶、茶叶、糖、包装这些"直接成本"，剩下来的就是毛利；再扣掉房租、人工、水电但不算你贷款利息和店里装修的折旧，就接近 EBITDA。它衡量的是"公司主营业务本身能不能赚钱"。

**企业价值（EV, Enterprise Value）** = 市值 + 净债务（总借款 - 现金）。比单纯看市值更全面。比如 A 公司市值 100 亿、净债务 50 亿，EV 就是 150 亿；B 公司市值 100 亿、净现金 30 亿，EV 只有 70 亿。

#### 2.4 行业基准

不同行业的合理估值差很多（2024-2025 年大致水平）：

- **大型科技股**（苹果、微软、谷歌）：P/E 25-35 倍
- **SaaS 公司**：P/E 40-80 倍（因为增长快）
- **半导体**（英伟达、台积电）：P/E 20-30 倍
- **银行**：P/E 8-12 倍（盈利受经济周期影响大）
- **必需消费**（可口可乐、宝洁）：P/E 18-25 倍
- **生物科技**（还没盈利的）：根本不算 P/E，看 EV/收入

**一个重要的判读方法**：拿苹果当前 P/E 跟它过去 10 年的 P/E 范围比。如果苹果过去 P/E 在 10-35 倍之间波动，现在 30 倍就是历史高位，要小心。

### 3. 共识预期——"市场到底在赌什么"

#### 3.1 这是什么

分析师对一家公司未来的盈利做预测，市场把这些预测加总取中位数，就形成了**共识预期（Consensus）**。主要的"记账本"公司是 FactSet、Refinitiv（I/B/E/S）、S&P Capital IQ、Bloomberg、Visible Alpha。

你可以把它理解成"专家们集体投票"——不是哪一个人说了算，是几十个分析师一起预测。

#### 3.2 共识预期为什么重要

因为它直接决定股价。**股价不是由真实业绩决定的，而是由真实业绩 vs 共识的差距决定的**。

举个例子：英伟达在 2024 年某季度收入 350 亿美元，市场预期是 320 亿美元。实际比预期高 9%，股价盘后大涨 6%。但如果市场预期是 360 亿，那么即使英伟达真的赚了 350 亿——超出真实水平——股价也会跌 3%。

这就是"**Beat or Miss**"（超越或不及预期）文化的核心。

#### 3.3 估计修订

分析师会不断修改自己的预测。**修订的方向比绝对数字更重要**：

- **过去 30 天内，5 个分析师上调，1 个下调** → 修订动能向上 → 股价大概率涨
- **过去 30 天内，1 个上调，5 个下调** → 修订动能向下 → 股价大概率跌

**"估计分散度"** 也很关键——分析师之间分歧越大，惊喜的机会越大（因为任何一边都可能错得离谱）。

### 4. 目标价——分析师心里的"应该多少钱"

#### 4.1 这是什么

分析师除了给"买入/持有/卖出"评级，还会给一个具体的**12 个月目标价（Target Price）**。这是他基于模型算出来的"合理价位"。

#### 4.2 怎么算出来的

**方法一：DCF（现金流折现）**
想象你朋友那家奶茶店，未来 5 年预计每年赚 20 万、25 万、30 万、35 万、40 万，第 6 年开始按 3% 永久增长。那你把这 5 年的钱 + 之后的"永续价值"打个折现到今天，就是公司现在的"内在价值"。

**方法二：相对估值（Comps）**
看同行——比如苹果对标微软、谷歌，假设你认为苹果的合理 P/E 应该是 30 倍，那目标价 = 30 × 苹果未来 1 年的 EPS。

**方法三：先例交易（Precedent Transactions）**
看历史上类似公司被收购时卖了多少钱，反推这家值多少。

**方法四：历史估值带**
看这家公司过去 5 年 P/E 在什么范围，假设市场会回到中间位置，那目标价就 = 中位数 P/E × 未来 EPS。

实际分析师会把这 4 种方法加权平均（DCF 30-40% + Comps 30-40% + 先例 20-30% + 历史 5-10%）。

#### 4.3 目标价 vs 当前价 = Upside（上行空间）

- 当前价 100 块，目标价 130 块 → Upside 30% → 强烈买入
- 当前价 100 块，目标价 95 块 → Upside -5% → 持有
- 当前价 100 块，目标价 70 块 → Upside -30% → 卖出

**关键技巧**：光看 Upside 不够，还要看**风险回报比**。比如分析师给的目标价范围是 80-150 块（最坏 80，最好 150），那潜在上涨 50、下跌 20，回报比 2.5:1，挺划算。但如果范围是 95-110，下跌空间 5、上涨空间 10，回报比 2:1，但绝对收益太小，不值得冒险。

#### 4.4 目标价分布

真实世界里同一家公司会有 30+ 分析师覆盖。比如特斯拉，2024 年覆盖它的分析师有 40 多个，目标价从最低 85 美元到最高 400 美元都有，**共识中位数目标价大概 200 美元**——这种巨大的分歧本身就告诉你：分歧大 = 机会大（也可能是错的）。

### 5. 财报——"公司这季度到底怎么样"

#### 5.1 三大财务报表

任何公司都要披露三张表：

1. **利润表（Income Statement）**：收入多少、成本多少、最后剩多少。最直观的"赚不赚钱"。
2. **资产负债表（Balance Sheet）**：公司有多少资产、多少负债、股东权益多少。反映"家底厚不厚"。
3. **现金流量表（Cash Flow Statement）**：真金白银的进出。利润表可以"会计调整"出好看的数字，但现金难造假。

举个真实场景：2024 年某 AI 公司财报显示营收 50 亿、净利润 5 亿，看起来不错。但现金流量表显示经营现金流是 -3 亿——意思是账面上赚了 5 亿，实际上客户还没付钱。**这是危险信号**，可能客户在压价、或者应收账款收不回来。

#### 5.2 利润表关键数字

- **收入（Revenue）**：卖东西总共收了多少
- **COGS（销售成本）**：直接成本（奶茶店的牛奶、茶叶、包装）
- **毛利（Gross Profit）**：收入 - COGS
- **毛利率（Gross Margin）**：毛利 ÷ 收入。**软件公司一般 70-90%**（复制成本低），**零售业一般 20-40%**（要进货）
- **营业费用（OpEx）**：SG&A（销售管理）+ R&D（研发）
- **营业利润（Operating Income）**：毛利 - 营业费用
- **净利润（Net Income）**：最终落到股东口袋的钱
- **EPS（每股收益）**：净利润 ÷ 流通股数。苹果 2024 财年 EPS 约 6.11 美元

**毛利率为什么对销售很重要**：高毛利率 = 公司有定价权 = 客户离不开 = 销售更容易。例如 SaaS 毛利率 80%，意味着签下一个 100 万的合同，扣掉服务器和客服成本，公司能拿到 80 万；传统零售 30% 毛利率，卖 100 万的货，扣掉成本只剩 30 万。

#### 5.3 GAAP vs 调整后（Adjusted / Non-GAAP）

公司财报会同时给两个数字：

- **GAAP（通用会计准则）**：法律规定的、必须用的。最严格。
- **调整后（Non-GAAP）**：公司自己剔除了"一次性项目"（比如并购费、重组费、股票薪酬）。

为什么有调整后？因为管理层觉得"一次性项目"会误导投资者对"持续经营能力"的判断。

**举个例子**：苹果 2024 财年 GAAP 净利润 930 亿，但调整后净利润 1010 亿——差的那 80 亿主要是诉讼和解和欧洲税务调整。**分析师共识预期基于调整后**，所以看财报时要先看调整后数字。

**警惕信号**：如果 GAAP 和调整后差距越来越大（比如差 30% 以上），说明公司可能在"过度调整"，用来粉饰真实表现。SBC（股票薪酬）虽然是公司常剔除的，但它是真实成本——员工拿股票不是免费的。

#### 5.4 现金流转换率

```
FCF Conversion = 自由现金流 ÷ 净利润
```

- **> 100%**：优秀（典型如 SaaS，递延收入先行）
- **80-100%**：良好
- **50-80%**：一般
- **< 50%**：危险

**自由现金流（FCF）** = 经营活动现金流 - 资本支出（CapEx）。FCF 是公司真金白银能拿去分红、回购、扩张的钱。

### 6. 估值方法实操

#### 6.1 相对估值（Comps / Trading Multiples）

最常用的方法：**找 3-5 家可比公司，算它们的中位数倍数值，再套到目标公司上**。

举例：你要给 Anthropic 估值。找同行 OpenAI（未上市）、Cohere、AI21 Labs。发现同行 EV/收入中位数是 25 倍，Anthropic 预计明年收入 10 亿美元，那 EV 估值 ≈ 250 亿美元。

#### 6.2 先例交易（Precedent Transactions）

**看过去类似公司被收购时卖了多少钱**。比如你要给 Cohere 估值，找过去 3 年 AI 公司被收购的案例：Inflection 被微软"收编"（2024 年，65 亿美元）、Adept 被亚马逊收购（2024 年，~9.7 亿美元）。算这些交易的 EV/收入中位数，反推 Cohere 估值。

#### 6.3 DCF 实战

假设要给一家年赚 1 亿美元的 SaaS 公司估值：

1. 预测未来 5 年现金流：1.1 亿、1.3 亿、1.5 亿、1.8 亿、2.1 亿
2. 算 WACC（加权平均资金成本），比如 10%
3. 算第 5 年后的"永续价值"：第 6 年 2.16 亿 ÷ (10% - 3%) = 31 亿
4. 把所有未来现金流贴现到今天，加起来减净债务，除以流通股数

**DCF 的核心缺陷**：对未来 5 年增长和折现率的微小假设变化，会让结果差好几倍。所以分析师一般用 DCF 给出范围（比如 200-300 美元），不是精确数字。

#### 6.4 LBO 分析

PE 基金常用。**假设一家 PE 用 70% 借债 + 30% 自己出的钱买下一家公司，目标 5 年后退出时 IRR > 20%**。倒推今天最多能出多少钱买。

LBO 适用于：稳定现金流、可预测收入、有资产可抵押。**典型案例**：2023 年 Adobe 宣布以 200 亿美元收购 Figma（虽然后来告吹），是 LBO 思维的体现。

### 7. 行业关键指标（KPI）——不同行业看不同数字

#### 7.1 SaaS 公司看什么

- **ARR（Annual Recurring Revenue）**：年化经常性收入。**最重要的指标**。Snowflake 2024 财年 ARR 约 33 亿美元，同比增长 38%。
- **NRR（Net Retention Rate）**：净收入留存率。> 120% 表示老客户不仅不流失，还越买越多。Datadog、Salesforce 的 NRR 长期 > 120%。
- **Rule of 40**：增长率 + 利润率 > 40%。比如增长 30%、营业利润率 15% → 45%，达标。
- **CAC Payback**：获客成本回收期。最好 < 18 个月。

#### 7.2 电商公司看什么

- **GMV（成交总额）**：阿里 2024 财年 GMV 超过 7 万亿人民币。
- **AOV（客单价）**：平均每个订单多少钱。
- **转化率**：浏览的人中多少买了。

#### 7.3 银行看什么

- **NIM（净息差）**：贷款利率 - 存款利率。**银行的"毛利率"**。摩根大通 2024 年 NIM 约 2.7%。
- **NPL（不良贷款率）**：借出去的钱收不回来的比例。越低越好。
- **CET1（资本充足率）**：监管要求的核心资本比例。越高越安全。

#### 7.4 消费零售看什么

- **同店销售增长（SSS）**：已经开业 1 年以上的店销售变化。**排除新开店的影响**。
- **库存周转**：一年卖了几轮库存。Zara 一年周转 6-8 次，传统百货 2-3 次。

#### 7.5 半导体看什么

- **ASP（平均售价）**：每颗芯片卖多少钱。英伟达 H100 单价 2.5-3 万美元，H200 更贵。
- **利用率**：产能利用率。台积电 2024 年长期 90%+。
- **Bit Shipment / Wafer Shipment**：出货量。

### 9. 一些容易混淆的概念

#### 9.1 EBITDA ≠ 现金 ≠ 利润

- **利润**：账面上的数字，可以被会计方法调整
- **EBITDA**：在利息、税金、折旧摊销前的盈利，反映"经营本身赚不赚"
- **现金**：真金白银，最难造假

**为啥要看 EBITDA**：比如特斯拉 2020 年净利 7 亿、2024 年净利 70 多亿，但 2020 年 EBITDA 已经有 58 亿了——它本来就赚钱，只是折旧大。EBITDA 剔除了"非现金"项目（折旧），更接近"经营本身的赚钱能力"。

#### 9.2 市值 ≠ 企业价值

- **市值（Market Cap）**：股价 × 流通股数
- **企业价值（EV）**：市值 + 净债务 + 少数股东权益 - 投资

买公司 = 买市值 + 承担债务 - 拿回现金。所以 EV 更能反映"买下整家公司要花多少钱"。

#### 9.3 EPS 增长 ≠ 利润增长

公司可以回购股票减少流通股数，让 EPS 看起来涨了，但实际净利润没涨那么多。**苹果 2013-2023 年累计回购 6000+ 亿美元股票，相当于把流通股数减少了 40%**——这是 EPS 增长的重要原因之一。

#### 9.4 共识预期 vs 实际

如果分析师整体太乐观，公司很容易"低于预期"——因为基数太高。所以专业投资者会**关注"哪些公司共识预期可能太低了"**——这才是真正的 Alpha 机会。

#### 9.5 Beta vs Alpha

- **Beta（β）**：股票相对于大盘的波动。苹果 β ≈ 1.2，比大盘波动大一点；英伟达 β ≈ 1.7，波动大很多；公用事业 β ≈ 0.5，几乎不动。
- **Alpha（α）**：跑赢大盘的超额收益。

**巴菲特的目标**：低 Beta、高 Alpha——稳赚。

### 10. 给小白的一份实操清单

如果你第一次看一份股权研究报告，按这个顺序读：

1. **公司是干什么的？** —— 业务描述 1-2 句话能不能讲清楚
2. **规模多大？** —— 收入（市值）数字感觉大不大
3. **赚不赚钱？** —— 净利润、利润率多少
4. **贵不贵？** —— P/E、EV/EBITDA 多少，跟同行比、跟自己历史比
5. **市场预期是什么？** —— 共识预期目标价多少
6. **最近一次财报超预期还是低于预期？** —— Beat or Miss
7. **管理层在电话会怎么说？** —— 战略重点、指引

**重要提醒**：
- **没有人能精准预测股价**。分析师给的目标价只是"基于当下信息的合理估计"，不是承诺。
- **研究是"概率思维"，不是"是非题"**。一个分析师写"买入"，意思是他认为上涨概率 > 50%，不是说一定会涨。
- **没有共识的独立研究更有价值**。如果所有人都看好一个股票，那它大概率已经 price in 了。真正赚钱的机会，是你在大家都悲观时发现"其实没那么糟"，或者在大家都乐观时发现"其实有问题"。

---

> **给小白的一句话**：金融研究本质上就是回答三个问题——**这家公司是干啥的？现在卖多少钱合理？未来会变好还是变差？** 所有的术语、模型、报告，都是为了回答这三个问题而服务的工具。掌握了这个主线，再看到 P/E、EV/EBITDA、共识预期这些词，你都不会怕——因为你知道它们只是工具的零件。
