# 审查固定收益组合

> 本命令使用 LSEG 债券定价、YieldBook 分析和收益率曲线工具。可用工具参见 [CONNECTORS.md](../CONNECTORS.md)。

通过对所有持仓定价、丰富参考数据、预测现金流以及在利率情景下进行压力测试，生成综合固定收益组合风险和回报报告。

领域知识参见 **fixed-income-portfolio** 技能。

## 工作流程

### 1. 收集组合持仓

询问用户：
- 债券标识符（必填）— 逗号分隔的 ISIN、CUSIP 或 RIC
- 持仓规模/权重（可选 — 如未提供，假设等权重）
- 特定测试情景（可选 — 如 "+100bp"，默认为标准网格）
- 估值日期（可选，默认今日）

### 2. 对所有债券定价

使用所有标识符调用 `bond_price`。

每只债券提取：净价/全价、收益率、久期、凸性、DV01、货币。

汇总组合级别：加权收益率、加权久期、总 DV01、总市值。

### 3. 丰富参考数据

为每只债券调用 `yieldbook_bond_reference`。

提取：证券类型、行业、评级、票息类型、赎回条款、发行人、国家。

构建组成分解：按行业、评级、到期区间、货币。

### 4. 预测现金流

为每只债券调用 `yieldbook_cashflow`。

汇总为季度现金流瀑布图。标记集中到期的时段。

### 5. 运行情景分析

调用 `yieldbook_scenario`，设置利率变动：-200bp、-100bp、-50bp、0bp、+50bp、+100bp、+200bp。

识别哪些债券对上行和下行的风险贡献最大。

### 6. 曲线背景

为组合主要货币调用 `interest_rate_curve`。

计算每只债券相对曲线的利差。评估曲线环境。

### 7. 整合报告

呈现：组合汇总指标、组成分解、现金流瀑布、含风险贡献的情景损益表、曲线敞口。

## 输出格式

以组合汇总指标开头，然后分部分详述组成、现金流和风险分析。

### 1. 债券组合审查——"不只是看收益率，还要看风险"

**想象一个场景**：你管理一个 10 亿美元的债券组合，持有 50 只不同的债券。今天早上，美联储宣布可能加息 100 个基点。你的第一个问题不是"哪只债券收益率最高"，而是"我的组合会亏多少钱"。

**单只债券 vs 组合的区别**：

| 维度 | 单只债券 | 债券组合 |
|------|---------|---------|
| **收益率** | 看这只债券的 YTM | 看加权平均收益率 |
| **风险** | 看这只债券的久期 | 看组合的总 DV01 和有效久期 |
| **分散化** | 无 | 按行业、评级、到期日分散 |
| **现金流** | 一串票息和本金 | 多只债券的现金流汇总成"瀑布" |
| **情景** | 利率变动时这只债券亏多少 | 利率变动时整个组合亏多少 |

**所以"审查"什么**：
1. **定价**：每只债券现在值多少钱？
2. **组成**：组合里有什么——按行业、评级、到期日分解
3. **现金流**：未来什么时候有钱进来？
4. **风险**：利率变动时组合亏多少？

### 2. 组合级别的久期与 DV01——"利率动一下，组合亏多少"

**久期（Duration）** 衡量债券价格对利率变动的敏感度。**DV01（Dollar Value of a Basis Point）** 是久期的"金额版"——利率变动 1 个基点（0.01%），你的组合亏赚多少钱。

**打比方**：假设你有一个装了 50 个鸡蛋的篮子。每个鸡蛋的"脆弱度"不同——有的蛋壳厚（短久期），有的蛋壳薄（长久期）。组合久期就是"整个篮子的平均脆弱度"，DV01 就是"如果你轻轻晃一下篮子，有多少鸡蛋会碎"。

**组合级别的计算**：

```
组合加权久期 = Σ (每只债券权重 × 该债券久期)
组合总 DV01  = Σ (每只债券持仓面值 × 该债券 DV01)
```

**真实例子**：
- 假设你持有 3 只债券：
  - 债券 A：面值 $5M，久期 2 年，DV01 = $1,000/bp
  - 债券 B：面值 $3M，久期 7 年，DV01 = $2,100/bp
  - 债券 C：面值 $2M，久期 12 年，DV01 = $2,400/bp
- 组合加权久期 = (5/10)×2 + (3/10)×7 + (2/10)×12 = 1.0 + 2.1 + 2.4 = **5.5 年**
- 组合总 DV01 = 1,000 + 2,100 + 2,400 = **$5,500/bp**
- 这意味着：利率上升 1bp，组合亏损 $5,500；利率上升 100bp（1%），组合亏损约 $550,000

**为什么 DV01 比久期更实用**：久期是百分比概念（"利率升 1%，价格跌 5.5%"），DV01 是金额概念（"利率升 1bp，亏 $5,500"）。基金经理看的是"亏多少钱"，不是"跌百分之几"。

### 3. 现金流瀑布图——"未来什么时候有钱进来"

**现金流瀑布图（Cashflow Waterfall）** 是把组合中所有债券的未来现金流按时间排列，像瀑布一样从上往下流。

**想象一下**：你有 50 个租客，每个租客每月交房租，但交的日期不同、金额不同、租约到期日也不同。现金流瀑布图就是一张"未来 10 年每个月能收多少租金"的时间表。

**为什么现金流瀑布重要**：

**① 再投资风险**：
- 如果某个月有大量本金到期（比如 $50M），你需要再投资——如果当时利率很低，你就亏了
- 真实例子：2020 年大量债券到期，但利率接近 0%，再投资回报极差

**② 流动性管理**：
- 如果某几个月几乎没有现金流，你可能需要卖债券来支付赎回
- 现金流"集中到期"的时段需要特别关注

**③ 免疫策略（Immunization）**：
- 如果你管理一个养老金组合，负债在 2035 年到期，你需要确保现金流也在那个时段集中
- 现金流瀑布帮你检查"资产端和负债端是否匹配"

**瀑布图怎么看**：
- 横轴：时间（季度或月度）
- 纵轴：现金流金额
- 每根柱子分为"票息"（矮的、频繁的）和"本金"（高的、到期时一次性的）
- 标记"集中到期"的时段——这些是再投资风险最高的时间点

### 4. 情景分析——"加息时亏多少，降息时赚多少"

**情景分析（Scenario Analysis）** 是在假设利率变动的条件下，计算组合的损益。它回答的问题是："如果美联储加息 100 个基点，我的组合亏多少？"

**打比方**：你要开车走一条山路。出发前，你不会只看"平均海拔"，你会问"如果下雨路滑怎么办？如果遇到落石怎么办？"。情景分析就是给组合做"压力测试"——在各种利率环境下看看组合会怎样。

**标准情景网格**：

| 情景 | 利率变动 | 含义 | 典型触发 |
|------|---------|------|---------|
| -200bp | 利率下降 2% | 极端宽松 | 经济危机、紧急降息 |
| -100bp | 利率下降 1% | 温和宽松 | 衰退预期 |
| -50bp | 利率下降 0.5% | 小幅宽松 | 经济放缓 |
| 0bp | 不变 | 基准 | — |
| +50bp | 利率上升 0.5% | 小幅收紧 | 通胀略高 |
| +100bp | 利率上升 1% | 温和收紧 | 央行加息 |
| +200bp | 利率上升 2% | 极端收紧 | 恶性通胀 |

**怎么读情景损益表**：

假设一个久期 5.5 年、DV01 $5,500/bp 的组合：

| 情景 | 利率变动 | 估算损益 |
|------|---------|---------|
| -200bp | -2% | +$1,100,000（赚！降息利好债券） |
| -100bp | -1% | +$550,000 |
| -50bp | -0.5% | +$275,000 |
| 0bp | 不变 | $0 |
| +50bp | +0.5% | -$275,000 |
| +100bp | +1% | -$550,000 |
| +200bp | +2% | -$1,100,000（亏！加息利空债券） |

**风险贡献（Risk Contribution）**：情景分析不只告诉你"组合亏多少"，还告诉你"哪只债券亏最多"。比如加息 100bp 时，组合亏 $550,000，其中债券 C（长久期）贡献了 $240,000——它就是"风险贡献最大"的持仓。

**真实例子**：
- **2022 年**：美联储加息 425bp，全球债券组合普遍亏损 10-20%。久期长的组合亏得更多
- **2023 年 3 月**：硅谷银行（SVB）倒闭，原因就是持有大量长久期债券，加息导致浮亏巨大，被迫卖出实现亏损

### 5. YTW——"最保守的收益率"

**YTW（Yield to Worst，收益率至最差）** 是所有可能赎回情景中最低的收益率。它比 YTM（Yield to Maturity，到期收益率）更保守。

**想象一下**：你借钱给朋友，约定 5 年后还，年利率 5%。但合同里有一条：朋友可以在第 3 年提前还钱，提前还时利率降到 3%。YTM 是假设他借满 5 年（5%），YTW 是假设他第 3 年就还了（3%）。作为出借人，你应该按 YTW 来算——因为最差的情况就是他提前还。

**为什么 YTW 重要**：
- 很多公司债和市政债都有"赎回条款（Call Provision）"——发行人可以在到期前提前赎回
- 当利率下降时，发行人倾向于提前赎回旧债、发行低利率新债——这对你不利
- YTW 告诉你"最坏情况下你能赚多少"

**YTM vs YTW 对比**：

| 指标 | 假设 | 适用 | 保守程度 |
|------|------|------|---------|
| **YTM** | 持有到期 | 不可赎回债券 | 一般 |
| **YTW** | 最差赎回情景 | 可赎回债券 | 最保守 |
| **YTC** | 首次赎回 | 可赎回债券 | 中等 |

**经验法则**：对于可赎回债券，永远看 YTW，不看 YTM。YTM 是"理想情况"，YTW 是"现实最差"。

### 6. 组成分解——"你的组合里有什么"

**组成分解（Composition Breakdown）** 是把组合按不同维度切片，看"你的风险集中在哪里"。

**四个关键维度**：

**① 按行业（Sector）**：
- 如果你 80% 的持仓是金融债，一旦金融行业出问题，整个组合都受影响
- 2023 年区域银行危机时，重仓金融债的组合亏损严重

**② 按评级（Rating）**：
- 投资级（IG：AAA 到 BBB-）vs 高收益（HY：BB+ 以下）
- 评级越低，违约风险越高，但收益率也越高
- 健康的组合通常有评级分散

**③ 按到期区间（Maturity Bucket）**：
- 短期（0-3 年）、中期（3-7 年）、长期（7 年+）
- 短期债券对利率不敏感，长期债券对利率很敏感
- 如果你预期加息，应该缩短组合久期（多持短期）

**④ 按货币（Currency）**：
- 如果组合有美元债和欧元债，汇率波动也会影响回报
- 2022 年欧元对美元贬值 15%，持有欧元债的美国投资者额外亏了 15%

> **给小白的一句话**：审查债券组合就像体检——定价是"量体重"，组成分解是"查各指标是否正常"，现金流瀑布是"看未来资金流是否健康"，情景分析是"压力测试——如果利率大幅变动，你能承受吗？"。四个维度缺一不可。
