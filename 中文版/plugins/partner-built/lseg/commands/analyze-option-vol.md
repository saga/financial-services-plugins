# 分析期权波动率

> 本命令使用 LSEG 波动率曲面、期权定价和历史数据工具。可用工具参见 [CONNECTORS.md](../CONNECTORS.md)。

通过生成波动率曲面、定价含完整希腊字母的期权、以及比较隐含波动率与已实现波动率，分析标的资产的波动率环境。

领域知识参见 **option-vol-analysis** 技能。

## 工作流程

### 1. 收集输入

询问用户：
- 标的资产（必填）：
  - 股票/指数：RIC 格式（如 "VOD.L@RIC"、".SPX@RIC"）
  - 期货：RICROOT 格式（如 "ES@RICROOT"、"CL@RICROOT"）
  - 外汇：ISO 货币对（如 "EURUSD"、"USDJPY"）
- 行权价（可选，默认平值）
- 到期日或期限（可选，默认 3 个月）
- 看涨或看跌（可选，默认两者）

判断是股票/指数还是外汇，以选择正确的波动率曲面工具。

### 2. 生成波动率曲面

**股票/指数/期货：** 调用 `equity_vol_surface`。

**外汇：** 调用 `fx_vol_surface`。

提取：各期限的 ATM 波动率、25-delta 风险反转、25-delta 蝶式。

### 3. 发现期权模板

为标的资产调用 `option_template_list`。识别可用的类型、到期日和行权价。

### 4. 定价期权

使用标的资产、行权价和到期日调用 `option_value`。

提取：权利金、delta、gamma、vega、theta、隐含波动率。

### 5. 计算已实现波动率

使用 `interval: "P1D"`、`tenor: "1Y"` 调用 `tscc_historical_pricing_summaries`。

计算 20 天、60 天、90 天窗口的收盘价-收盘价已实现波动率。与对应期限的隐含波动率比较。

### 6. 整合报告

呈现：波动率曲面汇总表、希腊字母表、隐含 vs 已实现对比、波动率制度评估、策略建议。

## 输出格式

以关键波动率发现（隐含相对已实现贵/便宜）开头，后跟曲面汇总、期权定价和详细比较。
