# 分析债券相对价值

> 本命令使用 LSEG 债券定价、收益率曲线、信用曲线和情景分析工具。可用工具参见 [CONNECTORS.md](../CONNECTORS.md)。

结合定价分析、收益率曲线背景、信用利差分解和利率冲击情景，对一只或多只债券进行相对价值分析。

领域知识参见 **bond-relative-value** 技能。

## 工作流程

### 1. 收集债券标识符

询问用户：
- 债券标识符 — ISIN、RIC 或 CUSIP（必填）
- 可选的基准债券用于比较
- 估值日期（可选，默认今日）

### 2. 对债券定价

使用标识符调用 `bond_price`。

提取：净价/全价、收益率、久期、凸性、DV01、货币。

如果提供了基准，也对其定价。

### 3. 获取无风险收益率曲线

调用 `interest_rate_curve`（先列出再计算）获取债券对应货币的曲线。

在债券到期日插值，计算 G-spread。

### 4. 获取信用利差曲线

调用 `credit_curve`（按国家/发行人类型搜索，然后计算）。

计算剩余利差 = 债券 G-spread 减去相同期限的信用曲线利差。剩余利差为正 = 便宜，为负 = 贵。

### 5. 运行情景分析

调用 `yieldbook_scenario`，设置平行利率变动：-100bp、-50bp、0bp、+50bp、+100bp。

提取每种情景下的价格变动和损益。

### 6. 整合报告

呈现：债券汇总表、利差分解（G-spread、信用利差、剩余利差）、情景损益表和贵贱评估。

如果提供了基准，包含并排比较。

## 输出格式

以贵/贱评估及支持证据开头，后跟利差分解和情景表格。

### 1. 相对价值——"不是看绝对收益率，而是看跟同类比贵不贵"

**相对价值（Relative Value，简称 RV）** 分析的核心思想是：一只债券的收益率高不高，不能只看绝对数字，要看它跟"同等级的债券"比是不是更便宜。

**打比方**：你想买一套两居室。A 小区报价 500 万，B 小区报价 480 万。你不能直接说 B 更便宜——如果 A 小区在市中心、B 小区在郊区，B 反而可能更贵。你需要找"同地段、同品质"的房子来比较。债券的 RV 分析就是找"同评级、同期限"的债券来比较。

**RV 分析的三步走**：
1. **算总利差**：这只债券的收益率比无风险利率高多少？（G-spread）
2. **拆信用利差**：同等级债券"应该"给多少利差？（信用曲线）
3. **看剩余利差**：总利差 - 信用利差 = 剩余利差。正 = 便宜，负 = 贵

---

### 2. G-spread——"比国债高多少"

**G-spread（Government Spread，政府债利差）** 是债券收益率与同期限国债收益率的差。它回答最基本的问题："这只债券比无风险的国债多给了多少收益？"

**想象一下**：你把钱存国有银行（相当于国债），年利率 3%。你朋友开了一家公司，找你借钱，年利率 5%。G-spread = 5% - 3% = 2%（200bp）。这 200bp 就是你承担"朋友可能还不上钱"这个风险所要求的额外补偿。

**G-spread 怎么算**：
```
G-spread = 债券收益率 - 同期限国债收益率
```

**关键细节——"同期限"**：
- 如果债券还有 7.3 年到期，你不能直接用 7 年或 10 年国债收益率
- 需要在国债收益率曲线上"插值"——找到 7.3 年那个点对应的收益率
- 本命令用 `interest_rate_curve` 工具获取国债曲线，然后在债券到期日插值

**G-spread 的局限**：
- 它只告诉你"比国债高多少"，没告诉你"这个差距合不合理"
- 同样是 200bp 的 G-spread，A 级债券可能太贵（应该只给 100bp），BB 级债券可能太便宜（应该给 300bp）
- 所以需要信用利差曲线来做"标准化比较"

---

### 3. 信用利差曲线——"同等级债券应该给多少利差"

**信用利差曲线（Credit Curve）** 是同一评级/同一发行人类型下，不同期限的信用利差连成的曲线。它告诉你："一个 A 级工业公司、7 年期的债券，市场认为应该给多少利差"。

**打比方**：你去买车险。保险公司不是随便给你一个价格——他们有一个"定价表"：25 岁男性、驾龄 3 年、无事故记录 = 年费 $1,200。信用利差曲线就是债券市场的"定价表"：A 级工业公司、7 年期 = 利差 120bp。

**信用曲线怎么用**：

假设你分析一只债券：
- 评级：BBB+
- 行业：工业
- 剩余期限：7 年
- G-spread：180bp

你查信用曲线：BBB+ 工业公司、7 年期 = 信用利差 150bp

**利差分解**：
```
总利差（G-spread）= 信用利差 + 剩余利差
180bp = 150bp + 30bp
```

**剩余利差 = 30bp**：这只债券比"同等级债券应该给的利差"多了 30bp——它更便宜！

**为什么剩余利差为正 = 便宜**：
- 利差越大 = 收益率越高 = 价格越低 = 越便宜
- 如果一只债券的利差比同类多 30bp，说明市场在"打折卖"——可能是流动性差、可能有负面新闻、可能只是被忽视
- 你的工作就是判断：这 30bp 是"真有问题"还是"被错杀"——如果是后者，就是买入机会

---

### 4. 剩余利差的三种解读

**剩余利差（Residual Spread）** 是 G-spread 减去信用曲线利差后的"残差"。它反映的是"无法被评级和期限解释的利差"。

**剩余利差为正（便宜）** 的可能原因：
1. **流动性折价**：这只债券交易不活跃，买家要求额外补偿
2. **被忽视**：小发行人、冷门行业，分析师覆盖少
3. **负面新闻**：市场知道一些信用曲线没反映的风险
4. **技术面因素**：大量卖压（比如某基金被迫清仓）

**剩余利差为负（贵）** 的可能原因：
1. **流动性溢价**：交易活跃、容易买卖，投资者愿意接受更低利差
2. **稀缺价值**：某类债券供给少，需求推高价格
3. **指数效应**：被纳入主要指数后，被动基金必须买入
4. **正面新闻**：市场预期评级上调

**真实例子**：
- **2023 年 3 月**：硅谷银行倒闭后，所有区域银行债的 G-spread 飙升。但信用曲线还没来得及调整，所以区域银行债的"剩余利差"突然变成很大的正值——有些是"真有问题"（第一共和银行），有些是"被错杀"（资本充足的区域银行），后者就是 RV 交易员的目标
- **2020 年 3 月**：COVID 恐慌时，投资级公司债利差从 100bp 飙到 400bp。信用曲线严重滞后，大量债券出现"剩余利差为正"——美联储介入后，这些债券价格大幅反弹

---

### 5. 基准比较——"跟同类比一比"

**基准比较（Benchmark Comparison）** 是 RV 分析的重要环节——找一只"可比债券"做并排对比。

**怎么选基准债券**：
- **同发行人**：同一公司的不同期限债券（比如苹果 5 年期 vs 苹果 10 年期）
- **同评级同行业**：不同公司但评级和行业相似（比如摩根大通 7 年期 vs 花旗 7 年期）
- **同期限**：到期日接近的债券

**并排比较看什么**：

| 指标 | 债券 A | 基准 B | 解读 |
|------|--------|--------|------|
| G-spread | 180bp | 150bp | A 比 B 宽 30bp |
| 信用曲线利差 | 150bp | 150bp | 同等级，应该一样 |
| 剩余利差 | +30bp | 0bp | A 更便宜 |
| 久期 | 6.5 年 | 6.8 年 | 类似 |
| 评级 | BBB+ | BBB+ | 相同 |

**如果 A 的剩余利差 +30bp 而 B 为 0**：要么 A 被低估（买入机会），要么市场知道 A 有额外风险你需要搞清楚。

---

### 6. 情景分析在 RV 中的应用——"利率变了，利差怎么变"

**情景分析** 在 RV 分析中的核心问题是：如果利率变动，这只债券的价格会怎么变？跟基准比，谁更抗跌？

**平行利率冲击**：

| 情景 | 利率变动 | 债券 A 价格变动 | 基准 B 价格变动 | 差异 |
|------|---------|----------------|----------------|------|
| -100bp | -1% | +6.5% | +6.8% | B 涨更多（久期更长） |
| -50bp | -0.5% | +3.2% | +3.4% | 类似 |
| 0bp | 不变 | 0% | 0% | — |
| +50bp | +0.5% | -3.1% | -3.3% | A 跌更少（久期更短） |
| +100bp | +1% | -6.2% | -6.5% | A 更抗跌 |

**RV 交易员怎么用情景分析**：
- 如果你觉得加息，选久期短的债券（A）——跌得更少
- 如果你觉得降息，选久期长的债券（B）——涨得更多
- 但别忘了利差！如果 A 的剩余利差收窄 20bp（从 180bp → 160bp），即使加息，A 也可能不亏——因为利差收窄会抵消利率上升的负面影响

**利差情景 vs 利率情景**：
- 利率情景：假设利率平行移动，利差不变
- 利差情景：假设利差收窄或扩大，利率不变
- 真实世界中两者同时变动——最好的 RV 交易是"利率和利差都朝有利方向走"

> **给小白的一句话**：债券相对价值分析就是"比价"——G-spread 告诉你"比国债贵多少"，信用曲线告诉你"同等级应该贵多少"，剩余利差告诉你"还有没有便宜可占"。剩余利差为正 = 便宜（可能是机会），为负 = 贵（可能要回避）。但别忘了搞清楚"便宜"的原因——是真有问题还是被错杀。
