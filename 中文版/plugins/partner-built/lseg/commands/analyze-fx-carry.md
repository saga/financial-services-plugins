# 分析外汇利差交易

> 本命令使用 LSEG 外汇定价、远期曲线、波动率曲面和历史数据工具。可用工具参见 [CONNECTORS.md](../CONNECTORS.md)。

结合即期汇率、远期点数、利差期限结构、波动率风险和价格历史背景，评估货币对的利差交易机会。

领域知识参见 **fx-carry-trade** 技能。

## 工作流程

### 1. 收集输入

询问用户：
- 货币对（必填）— 例如 USDJPY、EURUSD、AUDUSD
- 目标期限（可选，默认 3 个月）
- 估值日期（可选，默认今日）

### 2. 获取即期汇率

使用货币对调用 `fx_spot_price`。

提取：中间价/买价/卖价、买卖价差。

### 3. 定价目标期限远期

使用货币对和目标期限调用 `fx_forward_price`。

提取：远期汇率、远期点数。计算年化利差。

### 4. 绘制完整利差曲线

为货币对调用 `fx_forward_curve`（先列出再计算）。

呈现各期限（隔夜到 1 年）的利差概况：远期点数、年化利差、累计利差。识别最佳期限。

### 5. 评估波动率风险

为货币对调用 `fx_vol_surface`。

提取：目标期限的 ATM 波动率、25-delta 风险反转、25-delta 蝶式。

计算利差波动比 = 年化利差 / ATM 隐含波动率。

### 6. 历史即期背景

使用 `interval: "P1D"`、`tenor: "1Y"` 调用 `tscc_historical_pricing_summaries`，获取货币对的 RIC 数据。

评估：52 周范围、当前位置、趋势方向。

### 7. 整合报告

呈现：利差波动比和总体评估、即期与远期定价、利差期限结构表、波动率曲面快照、历史背景。

## 输出格式

以利差波动比和总体评估（有吸引力 / 中等 / 不具吸引力）开头，后跟详细数据表格。

### 1. 利差交易——"借便宜的钱，买贵的存款"

**外汇利差交易（FX Carry Trade）** 是最古老、最简单的交易策略之一：借低利率货币，换成高利率货币存起来，赚利率差。

**想象一个场景**：日本利率 0.1%，澳大利亚利率 4.5%。你借 1 亿日元（年利息 10 万日元），换成澳元存银行（年利息 450 万日元等值）。什么都不做，一年赚 440 万日元的利差——这就是利差交易。

**但有一个风险**：如果日元兑澳元升值超过 4.4%，你的利差就被汇率亏损吃掉了。比如你借日元时 1 AUD = 90 JPY，还钱时变成 1 AUD = 85 JPY——澳元贬值了 5.6%，你不仅没赚钱，还亏了。

**利差交易的数学**：
```
总收益 = 利差收益 + 汇率变动
       = (高利率 - 低利率) + (期末汇率 - 期初汇率) / 期初汇率
```

**真实例子**：2000-2007 年，日元利差交易是华尔街最火的策略。交易员借日元（利率接近 0%），买澳元（利率 6%+）、新西兰元（利率 8%+）、英镑（利率 5%+），每年稳赚 5-8% 利差。但 2008 年金融危机时，日元在几周内升值 20%+，所有利差交易巨亏——这叫"利差交易大平仓（Carry Unwind）"。

---

### 2. 远期点数——利差交易的"自动定价器"

**远期点数（Forward Points）** 是远期汇率和即期汇率的差值。它不是"手续费"，而是两国利率差的直接反映。

**利率平价公式**：
```
远期汇率 = 即期汇率 × (1 + 报价货币利率 × 天数/360) / (1 + 基础货币利率 × 天数/360)
远期点数 = 远期汇率 - 即期汇率
```

**想象一下**：USDJPY 即期汇率 150，美元利率 5%，日元利率 0.1%。3 个月远期点数 ≈ 150 × (5% - 0.1%) × 90/360 ≈ +1.84。所以 3 个月远期汇率 ≈ 151.84。

**远期点数的含义**：
- **正数（远期 > 即期）**：报价货币利率 > 基础货币利率 → 高息货币远期贴水
- **负数（远期 < 即期）**：报价货币利率 < 基础货币利率 → 低息货币远期升水

**关键洞察**：远期点数已经"定价了"利差。如果你做利差交易，你的实际利差收益不是"即期利率差"，而是"即期利率差 - 远期点数年化"。因为远期市场已经把利差反映在价格里了。

**真实例子**：AUDJPY 即期 90，3 个月远期 91.5（远期点数 +1.5）。你借日元买澳元，3 个月利差收益约 1.1%（4.4% 年化 × 3/12），但远期点数已经"吃掉"了 1.67%（1.5/90 × 4）。所以你的"超额利差"其实是负的——市场已经把利差定价进去了！

**所以这意味着什么**：利差交易不是"免费午餐"。远期点数会抵消大部分利差收益。真正赚钱的利差交易，需要**汇率朝着有利于你的方向移动**——也就是高息货币升值，低息货币贬值。

---

### 3. 利差波动比——"每承担一单位风险，赚多少利差"

**利差波动比（Carry-to-Vol Ratio）** = 年化利差 / ATM 隐含波动率。它衡量利差交易的"性价比"——每承担 1% 的波动率风险，你能赚多少利差。

**想象一下**：两个利差交易：
- 交易 A：年化利差 5%，波动率 10% → Carry-to-Vol = 0.5
- 交易 B：年化利差 3%，波动率 5% → Carry-to-Vol = 0.6

交易 A 利差更高，但风险也更大。交易 B "性价比"更好——每单位风险赚的利差更多。

**Carry-to-Vol 的判断标准**：

| Carry-to-Vol | 评估 | 含义 |
|-------------|------|------|
| **> 1.0** | 非常有吸引力 | 利差远超风险，罕见机会 |
| **0.5 - 1.0** | 有吸引力 | 值得做，但要注意风险管理 |
| **0.3 - 0.5** | 中等 | 利差勉强补偿风险 |
| **< 0.3** | 不具吸引力 | 风险太大，利差不够补偿 |

**真实例子**：2024 年初，USDJPY 的 Carry-to-Vol 约为 0.8（利差约 5%，波动率约 6%），被认为"有吸引力"。但 AUDJPY 的 Carry-to-Vol 只有 0.4（利差约 4%，波动率约 10%），因为澳元波动太大——虽然利差看起来高，但风险调整后不如 USDJPY。

---

### 4. 波动率曲面——"市场对不同方向的风险定价"

**波动率曲面** 展示了不同行权价和到期日的隐含波动率。它告诉你市场对"汇率会怎么动"的预期。

**三个关键指标**：

| 指标 | 含义 | 怎么读 |
|------|------|--------|
| **ATM 波动率** | 平值期权的隐含波动率 | "市场认为未来波动有多大" |
| **25-delta 风险反转** | 看涨期权波动率 - 看跌期权波动率 | "市场更怕涨还是更怕跌" |
| **25-delta 蝶式** | (看涨 + 看跌)/2 - ATM | "市场对极端事件的恐惧程度" |

**风险反转（Risk Reversal）** 详解：

**想象一下**：USDJPY 的 25-delta 风险反转是 -2%。这意味着看跌 USDJPY 的期权（日元升值）比看涨的贵 2%——市场更担心日元升值（利差交易大平仓），而不是日元贬值。

**蝶式（Butterfly）** 详解：

**想象一下**：USDJPY 的 25-delta 蝶式是 1.5%。这意味着市场为"极端波动"（不管是涨还是跌）额外定价了 1.5%——市场认为出现黑天鹅事件的概率比正态分布预测的高。

**真实例子**：2022 年 9 月，英国迷你预算危机时，GBPUSD 的风险反转飙升至 -5%（极度看跌英镑），蝶式飙升至 3%（极度恐惧极端波动）。如果你在做 GBP 利差交易，这两个数字告诉你"市场非常恐慌，赶紧跑"。

---

### 5. 利差期限结构——"做多 1 个月还是 1 年？"

**利差期限结构** 展示了不同期限的远期点数和年化利差。它帮你找到"最佳期限"——做多长时间最划算。

**想象一下**：你做 AUDJPY 利差交易。1 个月远期点数年化利差 4.2%，3 个月 4.5%，6 个月 4.3%，1 年 4.0%。3 个月利差最高——这就是"最佳期限"。

**为什么不同期限利差不一样？** 因为远期曲线反映了市场对未来利率变化的预期：
- 如果市场预期澳联储会降息，长期利差会下降
- 如果市场预期日银会加息，长期利差也会下降
- 短期利差更接近"当前现实"，长期利差包含更多"预期"

**真实例子**：2024 年，USDJPY 3 个月利差约 5%，但 1 年利差只有 4%——因为市场预期美联储会降息，美元利率会下降。所以短期利差交易更划算，但你需要更频繁地滚动（roll）头寸。

---

### 6. 历史背景——"现在的汇率在历史上算什么位置"

**52 周范围** 告诉你当前汇率在过去一年的位置——是接近高点还是低点。

**想象一下**：USDJPY 现在 155，52 周范围 140-162。155 在这个范围的 75% 位置——偏高了。如果你做利差交易（做多美元/做空日元），汇率偏高意味着"如果回归均值，你可能亏汇率"。

**但利差交易不是均值回归交易**。很多利差交易员不在乎汇率在什么位置——他们只在乎利差能不能覆盖波动。如果 Carry-to-Vol > 0.5，即使汇率在历史高位，他们也可能做。

**真实例子**：2024 年 USDJPY 突破 160（34 年新高），很多交易员觉得"太高了不敢做"。但 Carry-to-Vol 仍然 > 0.7，利差交易仍然有吸引力——直到日本央行突然加息，日元在两周内从 160 暴涨到 142，利差交易巨亏。

> **给小白的一句话**：外汇利差交易就像"借便宜的钱存高息银行"——听起来简单，但远期点数已经把利差定价进去了，波动率随时可能吃掉你的收益。记住三个关键数字：**利差（能赚多少）、波动率（可能亏多少）、Carry-to-Vol（性价比）**。风险反转告诉你"市场更怕什么方向"，蝶式告诉你"市场多怕黑天鹅"。利差交易最危险的时刻不是利率变化，而是**利差交易大平仓**——所有人同时跑向出口，低息货币瞬间暴涨，你的利差收益一年赚的钱一天亏完。
