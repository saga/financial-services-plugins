# 分析外汇利差交易

> 本命令使用 LSEG 外汇定价、远期曲线、波动率曲面和历史数据工具。可用工具参见 [CONNECTORS.md](../CONNECTORS.md)。

结合即期汇率、远期点数、利差期限结构、波动率风险和价格历史背景，评估货币对的利差交易机会。

领域知识参见 **fx-carry-trade** 技能。

## 工作流程

### 1. 收集输入

询问用户：
- 货币对（必填）— 例如 USDJPY、EURUSD、AUDUSD
- 目标期限（可选，默认 3 个月）
- 估值日期（可选，默认今日）

### 2. 获取即期汇率

使用货币对调用 `fx_spot_price`。

提取：中间价/买价/卖价、买卖价差。

### 3. 定价目标期限远期

使用货币对和目标期限调用 `fx_forward_price`。

提取：远期汇率、远期点数。计算年化利差。

### 4. 绘制完整利差曲线

为货币对调用 `fx_forward_curve`（先列出再计算）。

呈现各期限（隔夜到 1 年）的利差概况：远期点数、年化利差、累计利差。识别最佳期限。

### 5. 评估波动率风险

为货币对调用 `fx_vol_surface`。

提取：目标期限的 ATM 波动率、25-delta 风险反转、25-delta 蝶式。

计算利差波动比 = 年化利差 / ATM 隐含波动率。

### 6. 历史即期背景

使用 `interval: "P1D"`、`tenor: "1Y"` 调用 `tscc_historical_pricing_summaries`，获取货币对的 RIC 数据。

评估：52 周范围、当前位置、趋势方向。

### 7. 整合报告

呈现：利差波动比和总体评估、即期与远期定价、利差期限结构表、波动率曲面快照、历史背景。

## 输出格式

以利差波动比和总体评估（有吸引力 / 中等 / 不具吸引力）开头，后跟详细数据表格。

## 📖 本篇金融知识小贴士

**即期汇率（Spot Rate）**：现在兑换货币的价格。比如现在用1美元换多少日元。

**远期汇率（Forward Rate）**：约定在未来某个时间兑换货币的价格。

**远期点数（Forward Points）**：远期汇率 - 即期汇率。正数表示远期更贵（高息货币），负数表示远期更便宜（低息货币）。

**利差交易（Carry Trade）**：借入低利率货币，买入高利率货币，赚取利率差。比如借日元（利率近0%）买澳元（利率5%）。本质是赚息差，但要承担汇率波动的风险。

**利差波动比（Carry-to-Vol Ratio）**：年化利差 ÷ ATM 隐含波动率。比值越高说明"赚得多、风险小"。>0.5算好，>1.0算很好。

**ATM（At-The-Money，平值）**：期权的行权价等于当前标的价格。ATM 期权流动性最好，隐含波动率也是最常参考的。

**风险反转（Risk Reversal）**：看涨期权隐含波动率 - 看跌期权隐含波动率。正值说明市场情绪偏乐观（看涨期权更贵），负值说明偏悲观。

**蝶式（Butterfly）**：（25d 看涨 + 25d 看跌）- 2×ATM 波动率。衡量市场对尾部风险的定价。值越高，市场越担心大幅波动。
