# 宏观利率仪表盘

> 本命令使用 LSEG 宏观经济数据、收益率曲线、通胀曲线、互换定价和历史数据工具。可用工具参见 [CONNECTORS.md](../CONNECTORS.md)。

构建全面的宏观经济和利率仪表盘，展示关键经济指标、含斜率分析的收益率曲线、实际利率分解和互换利差背景。

领域知识参见 **macro-rates-monitor** 技能。

## 工作流程

### 1. 收集输入

询问用户：
- 国家（必填）— 例如 US、DE、GB、JP、CH
- 历史数据时间范围（可选，默认 3 年）
- 感兴趣的特定指标（可选）

将国家映射为货币：US→USD、DE→EUR、GB→GBP、JP→JPY。

### 2. 拉取宏观指标

为关键指标调用 `qa_macroeconomic`：
- GDP 增长（季度序列）
- CPI/通胀（月度序列）
- 失业率（月度序列）
- 政策利率/央行利率

使用通配符助记符模式发现可用序列（如 "US\*GDP\*"、"US\*CPI\*"）。

### 3. 获取收益率曲线

为国家的国债曲线调用 `interest_rate_curve`（先列出再计算）。

提取标准期限的收益率。计算：2s10s 斜率、3M-10Y 斜率、5s30s 斜率。对曲线形态分类。

### 4. 分解实际利率

为货币调用 `inflation_curve`（先搜索再计算）。

计算实际利率 = 名义利率减去关键期限的盈亏平衡率。评估实际利率是宽松还是紧缩。

### 5. 互换利差分析

在 2Y、5Y、10Y 调用 `ir_swap`（先列出再定价）。

计算互换利差 = 各期限的互换利率减去国债收益率。评估金融状况。

### 6. 历史收益率背景

在用户指定的时间范围内，为基准收益率 RIC 调用 `tscc_historical_pricing_summaries`。

评估：当前收益率在历史范围内的位置、趋势方向。

### 7. 整合仪表盘

呈现：宏观汇总表、含斜率指标的收益率曲线、实际利率分解、互换利差表、历史背景和总体宏观利率评估（2-3 句话）。

## 输出格式

以仪表盘形式呈现，各部分清晰标注。先给出总体宏观评估，然后逐项详细说明。

### 1. 宏观利率仪表盘——"利率市场的全身体检"

**想象一个场景**：你是一个全球宏观基金经理，今天要决定"美国利率是会涨还是跌"。你不能只看一个指标——你需要同时看经济增长、通胀、央行态度、市场预期、金融状况……所有这些拼在一起才能形成判断。宏观利率仪表盘就是帮你做这件事的工具。

**仪表盘的五个面板**：

| 面板 | 看什么 | 回答什么问题 |
|------|--------|------------|
| **宏观指标** | GDP、CPI、失业率、政策利率 | 经济好不好？通胀高不高？ |
| **收益率曲线** | 各期限利率 + 斜率 | 市场预期利率往哪走？ |
| **实际利率** | 名义利率 - 盈亏平衡通胀率 | 央行是在刺激还是在压制？ |
| **互换利差** | 互换利率 - 国债收益率 | 金融系统紧张吗？ |
| **历史背景** | 当前利率在历史范围的位置 | 现在的利率是高还是低？ |

### 2. 收益率曲线斜率——"最准的衰退信号灯"

**收益率曲线斜率（Yield Curve Slope）** 是长端利率和短端利率的差。它是金融市场最著名的"衰退预测器"——过去 60 年，每次美国经济衰退前，收益率曲线都出现了倒挂。

**三种关键斜率**：

| 斜率 | 计算 | 含义 | 典型阈值 |
|------|------|------|---------|
| **2s10s** | 10 年利率 - 2 年利率 | 最常用的衰退指标 | < 0 = 倒挂（衰退信号） |
| **3M-10Y** | 10 年利率 - 3 个月利率 | 纽约联储模型用的 | < 0 = 倒挂 |
| **5s30s** | 30 年利率 - 5 年利率 | 长端斜率 | < 0 = 极端信号 |

**为什么倒挂 = 衰退信号**：

**想象一下**：银行赚钱的方式是"借短贷长"——吸收短期存款（利率低），发放长期贷款（利率高），赚利差。如果短期利率比长期利率还高（倒挂），银行就赚不到钱了——于是银行减少贷款，企业借不到钱，经济就放缓了。

**真实例子**：
- **2019 年 8 月**：美国 2s10s 倒挂（-5bp），市场一片恐慌。2020 年 2 月美国经济确实陷入衰退（虽然 COVID 加速了）
- **2006 年**：2s10s 倒挂持续数月，2007-2008 年爆发金融危机
- **2000 年**：2s10s 倒挂，2001 年互联网泡沫破裂引发衰退

**但有一个重要细节——时滞**：倒挂不是"明天就衰退"。从倒挂到衰退，通常有 12-18 个月的时滞。而且倒挂期间股市可能还在涨——因为短期利率高说明经济还很强。

**曲线形态分类**：

| 形态 | 长短端关系 | 含义 | 类比 |
|------|-----------|------|------|
| **正常（陡峭）** | 长端 >> 短端 | 经济强劲，央行可能加息 | 上坡路 |
| **平坦** | 长端 ≈ 短端 | 加息接近尾声 | 平路 |
| **倒挂** | 长端 < 短端 | 衰退风险高 | 下坡路 |

### 3. 实际利率——"央行是在踩油门还是踩刹车"

**实际利率（Real Interest Rate）** = 名义利率 - 通胀预期。它告诉你"扣除通胀后，钱到底是贵还是便宜"。

**打比方**：银行贷款利率 5%，但通胀 3%。你借 100 元，一年后还 105 元——但物价涨了 3%，所以你"实际"只多还了 2 元。实际利率 = 5% - 3% = 2%。

**为什么实际利率比名义利率更重要**：
- 名义利率 5% + 通胀 1% = 实际利率 4% → 央行在"踩刹车"（钱很贵）
- 名义利率 5% + 通胀 7% = 实际利率 -2% → 央行在"踩油门"（钱其实很便宜）

**怎么算实际利率**：
```
实际利率 = 名义国债收益率 - 盈亏平衡通胀率（Breakeven）
```

**盈亏平衡通胀率** 是 TIPS（通胀保值债券）市场隐含的通胀预期。比如 10 年国债收益率 4.5%，10 年 TIPS 收益率 2.0%，盈亏平衡通胀率 = 4.5% - 2.0% = 2.5%。实际利率 = 4.5% - 2.5% = 2.0%。

**实际利率的"宽松 vs 紧缩"判断**：

| 实际利率水平 | 含义 | 央行态度 | 对经济的影响 |
|------------|------|---------|------------|
| **< -1%** | 极度宽松 | 踩死油门 | 刺激增长，但可能引发通胀 |
| **-1% 到 0%** | 宽松 | 轻踩油门 | 温和刺激 |
| **0% 到 1%** | 中性 | 不踩 | 经济自己走 |
| **1% 到 2%** | 紧缩 | 轻踩刹车 | 抑制增长 |
| **> 2%** | 极度紧缩 | 踩死刹车 | 可能引发衰退 |

**真实例子**：
- **2021 年**：名义利率 ~1.5%，通胀 7%，实际利率 -5.5% → 极度宽松 → 经济过热
- **2023 年**：名义利率 ~5%，通胀降到 3%，实际利率 +2% → 紧缩 → 经济开始放缓
- **2008 年**：名义利率降到 0%，通胀 -1%（通缩），实际利率 +1% → 看似零利率，实际还在紧缩

### 4. 互换利差——"金融系统的体温计"

**互换利差（Swap Spread）** = 互换利率 - 同期限国债收益率。它反映的是"银行之间借钱的成本"和"政府借钱成本"的差距——差距越大，金融系统越紧张。

**想象一下**：正常情况下，你借钱给美国政府（买国债）的利率是 4%，借钱给大银行（做互换）的利率是 4.15%——银行比政府多付 15bp，因为银行有微小违约风险。但如果金融危机来了，银行之间互不信任，互换利率飙升到 5%，而国债利率还是 4%——互换利差从 15bp 飙到 100bp。这就是"金融系统发烧了"。

**互换利差的解读**：

| 利差水平 | 含义 | 金融状况 |
|---------|------|---------|
| **0-15bp** | 正常 | 银行系统健康 |
| **15-30bp** | 略紧 | 需要关注 |
| **30-50bp** | 紧张 | 信用压力上升 |
| **> 50bp** | 危机模式 | 流动性严重紧张 |

**为什么互换利差会变**：
- **变宽（利差增大）**：银行信用风险上升、流动性紧张、监管要求银行持有更多资本
- **变窄（利差缩小）**：银行系统健康、流动性充裕、央行提供大量支持

**真实例子**：
- **2008 年 10 月**：30 年互换利差变成负数（互换利率 < 国债利率），这是极端异常——说明市场在疯狂买入互换（对冲利率风险），同时抛售一切非政府资产
- **2020 年 3 月**：COVID 恐慌时互换利差飙升，美联储紧急推出互换额度（swap lines）来稳定市场
- **2023 年 3 月**：硅谷银行倒闭后，互换利差短暂变宽，但很快恢复——因为美联储迅速提供流动性

**不同期限的互换利差含义不同**：
- **短期（2Y）**：反映银行短期融资压力
- **中期（5Y）**：反映信用和利率预期
- **长期（10Y）**：反映长期金融结构和监管影响

### 5. 历史收益率背景——"现在的利率在历史上算高还是算低"

**历史收益率背景** 帮你判断"当前利率水平在历史上处于什么位置"。

**打比方**：你去看医生，医生不会只看你的体温——他会看你的体温在"你的正常范围"里算不算异常。如果你平时体温 36.5°C，今天 37.5°C，就是发烧了。但如果另一个人平时 37.2°C，今天 37.5°C 就不算什么。历史背景就是利率的"正常体温范围"。

**怎么看历史背景**：
- **百分位法**：当前利率在过去 10 年中排第几？如果排 90%+，说明利率很高（历史上很少这么高）
- **趋势方向**：利率是在上升通道还是下降通道？
- **均值回归**：利率偏离历史均值太远，通常会回归

**真实例子**：
- **2023 年**：美国 10 年国债收益率 5.0%，在过去 15 年中排 95%+ 分位——非常高。市场预期会回落
- **2020 年**：美国 10 年国债收益率 0.5%，在过去 50 年中排最低 1%——极低。市场预期会回升
- 两个极端都提供了交易机会：高利率时买债券（赌利率回落），低利率时卖债券（赌利率回升）

### 6. 宏观利率整合判断——"五个面板怎么合在一起"

**仪表盘的最终输出是"总体宏观利率评估"——2-3 句话的综合判断。** 它不是简单地把五个面板的结论加在一起，而是要看它们之间是否一致。

**三种典型场景**：

**① 一致看多利率（利率会涨）**：
- GDP 强劲 + 失业率低 → 经济过热
- 通胀高于目标 → 央行需要加息
- 曲线陡峭 → 市场预期利率上升
- 实际利率偏低 → 央行还不够紧
- 互换利差窄 → 金融系统健康，能承受加息
- **结论**：利率大概率上行，考虑做空债券

**② 一致看空利率（利率会跌）**：
- GDP 放缓 + 失业率上升 → 衰退风险
- 通胀回落 → 央行可以降息
- 曲线倒挂 → 衰退信号
- 实际利率偏高 → 央行过于紧缩
- 互换利差宽 → 金融系统紧张
- **结论**：利率大概率下行，考虑做多债券

**③ 信号矛盾（最难判断）**：
- GDP 还行但通胀在降 → 央行可能"按兵不动"
- 曲线平坦但没倒挂 → 不确定
- 实际利率接近中性 → 不松不紧
- **结论**：利率方向不明，保持中性仓位

> **给小白的一句话**：宏观利率仪表盘就像体检报告——宏观指标是"血压和心率"，收益率曲线斜率是"心电图"，实际利率是"血液含氧量"，互换利差是"体温"，历史背景是"跟去年体检结果对比"。五个指标都正常 = 健康，多个指标亮红灯 = 该警惕了。
