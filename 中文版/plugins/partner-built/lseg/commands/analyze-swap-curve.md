# 分析互换利率曲线

> 本命令使用 LSEG 互换定价、利率曲线和通胀曲线工具。可用工具参见 [CONNECTORS.md](../CONNECTORS.md)。

构建和分析利率互换曲线，叠加国债收益率和通胀盈亏平衡率，并识别曲线交易机会。

领域知识参见 **swap-curve-strategy** 技能。

## 工作流程

### 1. 收集输入

询问用户：
- 货币（必填）— 例如 EUR、USD、GBP、CHF、JPY
- 参考利率指数（可选）— 例如 ESTR、SOFR、SONIA、TONA
- 估值日期（可选，默认今日）

### 2. 发现互换模板

使用货币和可选指数，以列表模式调用 `ir_swap`。

提取：可用模板参考、指数详情、惯例。

### 3. 构建互换曲线

以定价模式为标准期限（2Y、5Y、7Y、10Y、20Y、30Y）调用 `ir_swap`。

提取：各期限的平价互换利率和 DV01。

### 4. 叠加国债曲线

为相同货币调用 `interest_rate_curve`（先列出再计算）。

计算互换利差 = 各期限的互换利率减去国债收益率。

### 5. 分解实际利率

为货币调用 `inflation_curve`（先搜索再计算）。

计算实际互换利率 = 名义互换利率减去各期限的通胀盈亏平衡率。

### 6. 整合曲线策略观点

计算曲线指标：2s10s 斜率、5s30s 斜率、2s5s10s 蝶式。

识别机会：基于当前水平与历史常态的比较，判断陡化、平坦化、蝶式或互换利差交易。

呈现：含国债叠加的互换曲线表、曲线指标、实际利率分解、以及带 DV01 中性比例的交易建议。

## 输出格式

以曲线形态汇总和关键指标（2s10s、蝶式）开头，后跟详细表格和交易思路部分。

## 📖 本篇金融知识小贴士

**互换利率（Swap Rate）**：利率互换中的固定利率。A 同意支付固定利率换 B 的浮动利率，这个固定利率就是互换利率。

**互换利差（Swap Spread）**：互换利率 - 同期限国债收益率。反映银行间信用风险和流动性溢价。利差扩大通常意味着金融系统压力上升。

**利率互换（IRS）**：双方约定交换利息支付，一方付固定利率、一方付浮动利率。最常用的利率风险管理工具。

**平价互换利率（Par Swap Rate）**：使互换初始净现值为零的固定利率。也就是说，一开始谁也不欠谁。

**通胀盈亏平衡率（Inflation Breakeven）**：普通国债收益率 - 通胀保护债券（TIPS）收益率。市场预期的未来平均通胀率。

**实际利率（Real Rate）**：名义利率 - 通胀率。你赚到的钱"真正"能买多少东西。实际利率为负意味着存钱跑不赢通胀。

**2s10s 斜率**：10 年期收益率 - 2 年期收益率。最受关注的收益率曲线指标。为正且扩大表示经济向好预期；为负（倒挂）通常预示衰退。

**陡化/平坦化/蝶式**：交易收益率曲线形状变化的三种策略。陡化 = 赌长端涨更多，平坦化 = 赌短端涨更多，蝶式 = 赌中间相对两端变化。
