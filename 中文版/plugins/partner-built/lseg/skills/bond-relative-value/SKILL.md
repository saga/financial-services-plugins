# 债券相对价值分析

您是专业的固定收益分析师，专注于相对价值分析。结合 MCP 工具的债券定价、收益率曲线、信用曲线和情景分析，评估债券是贵、便宜还是合理。专注于将工具输出组织为利差分解和情景表格——让工具计算，您负责综合和建议。

## 核心原则

相对价值关注的是债券的利差是否充分补偿了其相对于可比工具的风险。始终将总利差分解为无风险 + 信用 + 剩余组成部分。剩余部分（扣除利率和信用后的剩余）揭示了真实的贵贱。通过情景压力测试验证观点在不同利率环境下是否成立。

## 可用 MCP 工具

- **`bond_price`** — 定价债券。返回净价/全价、收益率、久期、凸性、DV01、Z-spread。接受 ISIN、RIC 或 CUSIP。
- **`interest_rate_curve`** — 国债和互换收益率曲线。两阶段：先列出再计算。用于计算 G-spread。
- **`credit_curve`** — 按发行人类型的信用利差曲线。两阶段：按国家/发行人类型搜索，然后计算。用于分离信用组成部分。
- **`yieldbook_scenario`** — 平行利率变动的情景分析。返回每种情景下的价格变动和损益。
- **`tscc_historical_pricing_summaries`** — 历史定价数据。用于历史利差背景和 Z-score 分析。
- **`fixed_income_risk_analytics`** — OAS、有效久期、关键利率久期。用于可赎回债券和更深层的风险分解。

## 工具链工作流

1. **债券定价：** 为目标债券和任何比较债券调用 `bond_price`。提取收益率、Z-spread、久期、凸性、DV01。
2. **获取无风险曲线：** 调用 `interest_rate_curve`（先列出再计算）获取债券货币曲线。在债券到期日插值计算 G-spread。
3. **获取信用曲线：** 为发行人的国家和类型调用 `credit_curve`。提取债券到期日的信用利差。计算剩余利差 = G-spread 减去信用曲线利差。
4. **运行情景：** 以平行变动（-100bp、-50bp、0、+50bp、+100bp）调用 `yieldbook_scenario`。提取每种情景下的价格变动和损益。
5. **历史背景（可选）：** 为债券调用 `tscc_historical_pricing_summaries`，评估当前利差相对历史的位置。
6. **综合：** 将利差分解、情景结果和历史背景合并为贵/贱评估。

### 1. 相对价值分析的本质——"不是绝对判断，而是找参照物"

**相对价值分析（Relative Value / RV）** 不是问"这只债券值多少钱"，而是问"跟同类债券比，这只贵了还是便宜了"。

**打比方**：你去看房子，不是判断"这房子值 300 万"，而是看"同小区同户型的房子都卖 300 万，这套卖 350 万，贵了 50 万"。这就是相对价值——你的结论依赖"参照物"。

**在债券世界里**，参照物可能是：
- **同期限国债**：看 G-spread
- **同信用等级的债券**：看信用曲线利差
- **同一发行人的其他债券**：看历史利差区间
- **同一行业的债券**：看行业平均利差

**所以这意味着什么**：相对价值分析天然依赖于"选对参照物"。如果参照物选错了，你的结论就全错了。比如拿腾讯的债跟国债比，利差当然大，但这不代表腾讯债"便宜"——这只是信用风险补偿。

### 2. Z-spread vs G-spread vs OAS——三种利差，三种"参照物"

这是债券相对价值分析中最核心的三个概念，初学者最容易搞混。它们的区别在于**参照物不同**。

**G-spread（名义利差 / Government Spread）**：
- 参照物：**同期限国债收益率**
- 计算：G-spread = 债券收益率 - 同期限国债收益率
- 最简单、最直观，但**最粗糙**
- 问题：债券期限不同，国债曲线上的"插值"会有误差

**Z-spread（零波动利差 / Zero-Volatility Spread）**：
- 参照物：**整条国债收益率曲线**（不只是同期限一个点）
- 计算：把债券的每笔现金流都用国债即期利率折现，找到一个"统一加在国债利率上的溢价"，使折现值 = 债券市价
- 比 G-spread 精确，因为考虑了**整条曲线**的形状，不只是到期日那一个点
- 但**仍假设债券不会被提前赎回**——如果债券有嵌入期权（如可赎回债），Z-spread 就不准了

**打比方**：你去餐厅吃饭，G-spread 等于"只看最后一道菜的价格"判断贵不贵，Z-spread 等于"把每道菜都拆开看单价"。

**OAS（期权调整利差 / Option-Adjusted Spread）**：
- 参照物：整条国债曲线 + **期权模型**
- OAS 是 Z-spread 的升级版——它先把债券中嵌入的期权价值"剥离"出来，剩下的才是真正的信用/流动性溢价
- 适用于**可赎回债券（Callable Bond）**、**可回售债券（Putable Bond）**、**MBS/ABS** 等有嵌入期权的债券

**打比方**：你买了一个"带退货权的商品"，表面价格 1000 元。但"退货权"本身值 100 元。所以商品本身只值 900 元。Z-spread 告诉你的是"1000 元相对于标价贵不贵"，OAS 扣掉退货权后告诉你"900 元值不值"。

**所以这意味着什么**：
- 普通国债/公司债（无期权）→ 用 Z-spread 就够了
- 可赎回债 → 必须用 OAS，否则 Z-spread 会"虚高"（因为可赎回对发行方有利，对投资者不利）
- 本文件中的 `fixed_income_risk_analytics` 工具就是用来计算 OAS 的

| 利差类型 | 参照物 | 适用场景 | 精确度 |
|---------|--------|---------|--------|
| G-spread | 同期限国债 | 快速估算 | 低 |
| Z-spread | 整条国债曲线 | 普通债券 | 中 |
| OAS | 整条曲线 + 期权模型 | 可赎回/结构化债券 | 高 |

### 3. 利差分解——"剩下的部分才是真正的信号"

相对价值分析的核心操作是**利差分解（Spread Decomposition）**：

```
总利差（G-spread）= 无风险利差 + 信用利差 + 剩余利差
```

**无风险利差**：同期限国债收益率（这是"基准"）
**信用利差**：该发行人这个信用等级应该有的利差（从信用曲线获取）
**剩余利差（Residual Spread）**：总利差扣掉无风险和信用之后剩下的部分

**剩余利差的三层含义**：
- **流动性溢价**：这只债券交易活跃不活跃？交易量小的债券往往需要额外补偿（流动性折价）
- **技术面因素**：供需失衡、大资金进出、指数调整、季末调仓等短期因素
- **真正的错误定价**：市场对这只债券的定价"错了"——这才是相对价值分析要捕捉的

**打比方**：你买机票，北京到上海的经济舱，国航卖 1500 元，东航卖 1200 元。同样的航线、同样的舱位，差 300 元。这 300 元就是"剩余利差"——它可能因为国航起飞时间更好（品质溢价）、东航有促销（技术面）、或者你单纯觉得东航被低估了（错误定价）。

**真实案例**：2023 年硅谷银行（SVB）倒闭后，地区性银行债券的利差大幅飙升。但如果你仔细做利差分解，会发现有些银行的剩余利差远大于其基本面恶化程度——这些债券就是"被错杀"的便宜货。

**所以这意味着什么**：正剩余利差（债券便宜）≠ 自动买入。你要区分"便宜是因为市场恐慌（买入机会）"还是"便宜是因为基本面真的恶化（价值陷阱）"。

### 4. Z-score 与历史百分位——"便宜"是相对于历史而言的

**Z-score（Z 值）**：
- 衡量当前利差偏离历史均值的程度，单位是**标准差**
- 公式：Z-score = (当前利差 - 历史平均利差) ÷ 历史利差标准差
- |Z| > 2 = 显著偏离（约 95% 置信区间）
- |Z| > 3 = 极端偏离（约 99.7% 置信区间）

**打比方**：你身高 180cm，中国男性平均身高 170cm，标准差 6cm。你的 Z-score = (180-170)/6 = 1.67——你比平均高，但不算"异常高"。

**百分位（Percentile）**：
- 当前利差在历史数据中排第几
- 例如：百分位 90% = 当前利差比 90% 的历史时期都高（债券可能被低估）
- 百分位 10% = 当前利差比 90% 的历史时期都低（债券可能被高估）

**Z-score 和百分位的关系**：
| Z-score | 百分位（近似） | 解读 |
|--------|------------|------|
| -3 | 0.1% | 极端便宜 |
| -2 | 2.3% | 显著便宜 |
| -1 | 15.9% | 偏便宜 |
| 0 | 50% | 均值 |
| +1 | 84.1% | 偏贵 |
| +2 | 97.7% | 显著贵 |
| +3 | 99.9% | 极端贵 |

**“所以这意味着什么”**：
- Z-score = -2.5 → 当前利差远高于历史均值 → 债券可能被低估（便宜）→ 考虑买入
- Z-score = +2.5 → 当前利差远低于历史均值 → 债券可能被高估（贵）→ 考虑卖出
- **但注意**：历史不代表未来。"均值回归"是一种信仰，不是物理定律。如果发行人基本面发生了永久性变化，利差"回不到"历史均值。

### 5. 情景压力测试——"不是跑个数字，而是找隐藏风险"

本文件要求对每个债券运行利率平行移动（-100bp、-50bp、0、+50bp、+100bp）的情景分析。这不是为了"看看数字"，而是为了回答三个问题：

**① 非对称风险**：你的债券在利率上升和下降时的损失对称吗？
- 如果利率上升 100bp 亏 5%，下降 100bp 只赚 1% → 非对称，持有风险大
- 原因可能是：债券有嵌入期权（可赎回）、久期随利率变化（凸性）、或者债券本身结构特殊

**② 最大损失**：在最坏情景下你会亏多少？你能承受吗？
- 如果你管理一个基金，持仓 1000 万，+100bp 下的损失是 50 万。如果基金只能承受 30 万回撤，这只债券就不能买

**③ 压力下的流动性**：在极端情景下，这只债券的定价模型还准吗？
- 很多债券在正常市场下定价准确，但在极端市场下流动性枯竭，实际交易价格会远差于模型价格

**打比方**：你买保险的"压力测试"——你开车 30 年没出过事故，但保险还是要买。因为"正常情况"下的经验不能告诉你"极端情况"下会发生什么。情景分析就是你的压力测试。

**真实案例**：2020 年 3 月 COVID 恐慌期间，美国国债市场出现了历史上罕见的流动性危机——连最安全的国债都出现了异常的买卖价差。这意味着即使是"无风险"的国债，在极端情景下也可能出现定价扭曲。

### 6. 流动性溢价——"为什么同样的债券，交易量小的要便宜"*

**流动性溢价** 是市场为"交易不活跃"付出的代价补偿。

**打比方**：你在北京二环有一套房子，在三线城市也有一套，面积一样。但二环的房子随时能卖，三线城市的可能挂半年都卖不掉。所以即使两套房子"同等质量"，三线城市的也要便宜一些——这就是流动性溢价。

**在债券世界里**：
- **国债（On-the-run）**：最新发行的国债，每天交易量巨大，流动性极好，买卖价差可能只有 0.01%
- **老国债（Off-the-run）**：发行超过 6 个月的国债，交易量下降，买卖价差可能到 0.1%
- **小众公司债**：某些发行人只发了一只债券，交易量极小，买卖价差可能到 1-2%

**流动性溢价的典型表现**：
- 同样的信用等级，发行规模 50 亿的债券（流动性好）比发行规模 2 亿的债券（流动性差）利差低 20-50bp
- 剩余利差中可能有一大块是"流动性溢价"，不是"错误定价"

**所以这意味着什么**：剩余利差为正 ≠ 立刻买入。如果正剩余利差主要是"流动性溢价"，而你恰好是一个需要频繁交易的投资者，那这个"便宜"对你来说可能不值得。

### 7. 贵/贱分析的完整框架——"不是一句话，而是一个判断体系"

本文件最终要求给出"贵/贱/合理"的判断。这不仅仅是看一个数字，而是综合多个维度：

| 维度 | 信号 | 权重 |
|------|------|------|
| **Z-spread 绝对值** | 相对同类债券利差是高是低 | 30% |
| **Z-score / 历史百分位** | 当前利差在历史中的位置 | 25% |
| **剩余利差** | 扣掉信用后的超额利差 | 20% |
| **情景分析** | 在极端情景下是否脆弱 | 15% |
| **流动性** | 交易是否活跃、买卖价差 | 10% |

**判断标准**：
- **便宜（Cheap）**：Z-spread 高于历史均值 2σ、剩余利差显著为正、情景压力下不脆弱、流动性可接受 → 买入/超配
- **贵（Rich）**：Z-spread 低于历史均值 2σ、剩余利差显著为负 → 卖出/低配
- **合理（Fair）**：各项指标在历史均值附近，无明显偏离 → 中性

**打比方**：你买二手车，不只是看价格（Z-spread），还要看这辆车在二手车市场的行情（历史百分位）、车况扣掉里程后的"剩余价值"（剩余利差）、如果出事故能承受多大损失（情景分析）、以及这辆车在二手市场好不好脱手（流动性）。

**真实案例**：2024 年高收益债（垃圾债）利差一度压缩到 300bp 以下，远低于历史均值 450bp。从 Z-score 角度看，这是"极端贵"的信号。但如果你判断美联储降息周期刚刚开始、经济软着陆概率高，那么"贵"可能只是反映了"市场对信用风险的定价更乐观了"——这个"贵"不一定意味着应该卖出。

---

### 给小白的一句话

> **债券相对价值分析不是算一个数字说"贵了"或"便宜了"，而是把利差一层层剥开——先看相对于国债（G-spread）、再看相对于信用曲线（信用利差）、最后看剩下的（剩余利差）——然后结合历史位置和极端情景，判断这个价格是"市场恐慌给的便宜"还是"大家都不傻的合理定价"。**
