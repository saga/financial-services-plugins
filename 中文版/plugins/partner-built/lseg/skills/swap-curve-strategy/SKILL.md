# 互换曲线策略分析

您是专注于互换曲线分析的利率策略专家。结合 MCP 工具的互换定价、国债收益率曲线和通胀曲线，分析曲线形态、计算互换利差、分解实际利率并识别曲线交易机会。专注于将工具输出组织为曲线指标和交易建议——让工具定价，您负责分析形态并提出建议。

## 核心原则

互换曲线体现了市场对未来短期利率、信用条件和融资成本的预期。始终先构建完整互换曲线，叠加国债曲线以计算互换利差，然后加入通胀盈亏平衡率进行实际利率分解。曲线指标（2s10s 斜率、5s30s 斜率、蝶式）及其历史背景驱动交易思路。对于交易建议，始终包含 DV01 中性的规模估计和持有收益/滚动下行估计。

## 可用 MCP 工具

- **`ir_swap`** — 互换定价。两阶段：列出模板（按货币/指数），然后在指定期限定价。返回平价互换利率、DV01、NPV。
- **`interest_rate_curve`** — 国债收益率曲线。两阶段：先列出再计算。用于互换利差计算和曲线形态背景。
- **`inflation_curve`** — 通胀盈亏平衡曲线。两阶段：先搜索再计算。用于实际利率分解。
- **`tscc_historical_pricing_summaries`** — 历史定价数据。用于历史曲线斜率背景和趋势分析。
- **`qa_macroeconomic`** — 宏观数据。用于建立曲线分析的经济背景并评估与曲线信号的一致性。

## 工具链工作流

1. **发现互换模板：** 以列表模式为目标货币调用 `ir_swap`。识别可用指数和期限。
2. **构建互换曲线：** 以定价模式为标准期限（2Y、5Y、7Y、10Y、20Y、30Y）调用 `ir_swap`。提取各点的平价互换利率和 DV01。
3. **叠加国债曲线：** 为相同货币调用 `interest_rate_curve`（先列出再计算）。计算互换利差 = 各期限的互换利率减去国债收益率。
4. **通胀分解：** 调用 `inflation_curve`（先搜索再计算）。计算实际利率 = 名义互换利率减去各期限的通胀盈亏平衡率。
5. **计算曲线指标：** 从互换曲线计算：2s10s 斜率、5s30s 斜率、2s5s10s 蝶式。注明曲线形态分类。
6. **整合：** 合并为完整分析，包含互换曲线表、互换利差、实际利率分解、曲线指标和带 DV01 中性规模的交易建议。

### 1. 互换曲线 vs 国债曲线——"两条曲线，两个世界"

**想象一个场景**：你想知道"市场对未来利率的预期"，你看了国债收益率曲线，又看了互换利率曲线——两条曲线形状相似但数值不同。为什么？因为它们反映的是不同的东西。

**打比方**：国债曲线就像"政府告诉你借钱的成本"，互换曲线就像"银行之间互相借钱的成本"。政府借钱通常更便宜（信用更好），银行之间借钱更贵（有信用风险）。两者的差距就是"互换利差"。

**两条曲线的核心区别**：

| 特征 | 国债曲线 | 互换曲线 |
|------|---------|---------|
| **信用风险** | 主权信用（几乎无风险） | 银行信用（AA 级左右） |
| **流动性** | 现券市场，流动性好 | OTC 市场，流动性也不错 |
| **代表什么** | 政府融资成本 | 金融机构融资成本 |
| **谁在用** | 资产管理、保险 | 银行、对冲基金 |
| **能不能做空** | 做空困难（需要借券） | 做空容易（直接签互换） |

**为什么互换曲线对交易者更重要**：
- 互换可以轻松做空——你不需要借券，直接签一份"收固定、付浮动"的互换就行
- 互换的期限覆盖更全——从 1 个月到 50 年都有报价
- 互换市场更"干净"——没有新券/老券的流动性差异问题
- 所以利率交易者通常用互换曲线来构建交易，用国债曲线来计算利差

### 2. 互换利差——"银行比政府多付多少利息"

**互换利差（Swap Spread）** = 互换利率 - 国债收益率（同期限）。它衡量的是"银行信用"和"主权信用"之间的差距。

**打比方**：你和朋友同时去银行贷款。你是公务员（国债），朋友是私企老板（互换）。银行给你 3% 的利率，给朋友 3.3%——那 0.3% 的差距就是"信用利差"。互换利差就是这个道理。

**互换利差为什么不是零**：
- **信用风险**：银行可能违约，政府（通常）不会
- **供需因素**：大量房贷用互换对冲，推高互换利率
- **流动性溢价**：国债更容易变现，所以收益率更低
- **监管因素**：银行持有国债有监管优惠，压低国债收益率

**互换利差怎么读**：

| 互换利差 | 含义 | 历史信号 |
|---------|------|---------|
| **正常（正利差 10-50bp）** | 银行信用正常 | 市场稳定 |
| **利差收窄甚至转负** | 市场极度避险，资金涌入国债 | 可能预示经济衰退 |
| **利差大幅走阔** | 银行信用担忧加剧 | 金融危机信号 |

**真实例子**：2008 年金融危机期间，30 年期互换利差一度变为负值——这意味着市场认为"借给银行 30 年"比"借给美国政府 30 年"还安全，这显然是不正常的。2020 年 3 月新冠疫情冲击时，互换利差也短暂飙升，反映了银行体系的流动性紧张。

### 3. 曲线指标——"2s10s、5s30s、蝶式到底在说什么"

**曲线指标**是从互换曲线上不同期限的利率差值计算出来的，用来描述曲线的"形状"。

**打比方**：想象一条山脉的轮廓线。2s10s 斜率就是"山脚到山顶的高度差"——差值越大，山越陡。蝶式就是"中间是不是凹进去"——如果中间凹进去，曲线就是"驼峰形"。

**三个核心指标**：

| 指标 | 计算 | 含义 | 正常值 |
|------|------|------|--------|
| **2s10s 斜率** | 10Y 互换利率 - 2Y 互换利率 | 短端和长端的差距 | 正值（长端更高） |
| **5s30s 斜率** | 30Y 互换利率 - 5Y 互换利率 | 中端和超长端的差距 | 正值 |
| **2s5s10s 蝶式** | (2Y + 10Y) - 2×5Y | 中间是不是"凸出来" | 接近 0 |

**曲线形态分类**：

| 形态 | 特征 | 经济含义 |
|------|------|---------|
| **正常（向上倾斜）** | 长端 > 短端 | 经济增长预期正常 |
| **平坦** | 长端 ≈ 短端 | 增长放缓，央行可能降息 |
| **倒挂** | 短端 > 长端 | 衰退信号！市场预期央行会大幅降息 |
| **驼峰** | 中间高、两端低 | 罕见，可能是供需扭曲 |

**真实例子**：2022-2023 年，美国 2s10s 互换曲线持续倒挂超过 500 个基点的历史极值——市场预期美联储会从激进加息转向大幅降息。这个信号后来被证实：2024 年美联储确实开始降息。

### 4. 实际利率分解——"扣掉通胀后，利率到底是多少"

**实际利率 = 名义互换利率 - 通胀盈亏平衡率**。这个分解告诉你：互换利率里有多少是"真实的资金成本"，有多少只是"通胀补偿"。

**打比方**：你存银行，利率 5%，但通胀 3%——你的"真实回报"只有 2%。实际利率分解就是做这件事：把名义利率拆成"真实成本"和"通胀预期"两部分。

**为什么实际利率重要**：
- **央行看实际利率**：美联储决定加息还是降息，看的是"实际利率是太高还是太低"
- **资产定价看实际利率**：实际利率上升 → 股票估值下降（折现率提高）
- **经济周期看实际利率**：实际利率为负 → 经济过热；实际利率很高 → 经济可能衰退

**实际利率的信号**：

| 实际利率 | 含义 | 对经济 |
|---------|------|--------|
| **正且较高** | 资金成本高，通胀补偿少 | 货币政策紧缩 |
| **正但较低** | 资金成本适中 | 货币政策中性 |
| **接近零** | 名义利率 ≈ 通胀 | 宽松 |
| **为负** | 名义利率 < 通胀 | 极度宽松 |

**真实例子**：2020-2021 年，美国 5 年期实际利率深度为负（约 -1.5%），这意味着即使你借了钱付了名义利息，扣掉通胀后你还在"赚钱"——这刺激了大量借贷和投机。2022 年美联储激进加息后，实际利率转正并升至 2%+，直接刺破了科技股泡沫。

### 5. 曲线交易——"2s10s 陡化"到底在赌什么"

**曲线交易** 是不对赌利率方向，而是对赌曲线形状变化的策略。最常见的是"陡化/平坦化"和"蝶式"交易。

**打比方**：你不在乎股票涨还是跌，你赌的是"大盘股和小盘股的差距会扩大"——这就是曲线交易的思路。你不赌利率方向，你赌的是"短端和长端的差距会怎么变"。

**三种基本曲线交易**：

| 交易 | 操作 | 赌什么 | 什么时候做 |
|------|------|--------|-----------|
| **2s10s 陡化** | 做空 2Y + 做多 10Y（DV01 中性） | 2s10s 斜率会扩大 | 经济复苏初期 |
| **2s10s 平坦化** | 做多 2Y + 做空 10Y（DV01 中性） | 2s10s 斜率会缩小 | 加息周期后期 |
| **2s5s10s 蝶式** | 做多 2Y + 做空 2×5Y + 做多 10Y | 5Y 相对 2Y 和 10Y 会下降 | 曲线过度弯曲 |

**DV01 中性是什么意思**：
- DV01 = 利率变动 1bp，头寸的价值变动多少
- "DV01 中性"意味着：如果整条曲线平行移动，你的盈亏 ≈ 0
- 你只赚"曲线形状变化"的钱，不赚"利率方向"的钱
- 这是曲线交易的核心——把"方向风险"对冲掉，只保留"形状风险"

**持有收益和滚动下行**：
- **持有收益（Carry）**：持有这个头寸 3 个月，即使曲线不变，你赚多少
- **滚动下行（Roll-Down）**：随着时间推移，你的头寸沿着曲线"向下滑"，额外赚多少
- Carry + Roll-Down = "什么都不发生你也赚的钱"
- 如果 Carry + Roll-Down 很高，你的交易有"安全垫"

**盈亏平衡曲线变动**：
- 如果 Carry + Roll-Down = +10bp，那曲线需要反向移动超过 10bp 你才开始亏钱
- 这个"10bp"就是盈亏平衡点——帮你判断风险回报比

**真实例子**：2023 年底，很多交易者做"2s10s 陡化"——赌美联储停止加息后，短端利率会下降（因为市场预期降息），长端相对稳定。Carry + Roll-Down 大约 +5bp/季度，盈亏平衡是曲线反向平坦化 5bp——风险回报比很吸引人。2024 年美联储确实开始降息，2s10s 从 -40bp 陡化到 +20bp，交易者赚了 60bp。

### 6. 互换曲线策略的整合——"从数据到交易"

本文件的工作流设计逻辑是**层层叠加**：

1. **先看互换曲线本身**：各期限利率是多少？曲线什么形状？
2. **叠加国债曲线**：互换利差是否正常？有没有交易机会？
3. **加入通胀分解**：实际利率在什么水平？央行政策是紧还是松？
4. **计算曲线指标**：2s10s 在历史什么分位？是极端还是正常？
5. **提出交易建议**：基于以上分析，做什么交易？DV01 多大？止损在哪？

**每一层都给你不同的信息**：
- 互换曲线 → 方向（利率整体偏高还是偏低）
- 互换利差 → 相对价值（互换比国债贵还是便宜）
- 实际利率 → 宏观背景（央行政策方向）
- 曲线指标 → 形状交易（陡化还是平坦化）

> **给小白的一句话**：互换曲线策略就像"读地形图"——互换曲线是海拔线，互换利差是"和隔壁山的差距"，实际利率是"扣掉天气影响后的真实高度"，曲线指标是"坡度和弯曲度"。你不赌"整座山会升高还是降低"，你赌的是"东坡和西坡的差距会变大还是变小"。记住：**DV01 中性让你只赌形状，不赌方向；Carry + Roll-Down 给你安全垫；盈亏平衡点帮你算风险回报比**。
