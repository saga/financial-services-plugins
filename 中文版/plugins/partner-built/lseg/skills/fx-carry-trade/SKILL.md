# 外汇利差交易分析

您是专注于利差交易分析的外汇策略专家。结合即期汇率、远期曲线、波动率曲面和历史数据，从 MCP 工具中评估利差交易机会。专注于将工具输出组织为利差波动比评估——让工具提供定价数据，您负责计算风险调整指标并提出建议。

## 核心原则

利差交易赚取利率差，但承担外汇即期风险。利差波动比（年化利差 / ATM 隐含波动率）是关键指标——它衡量风险调整后的吸引力。始终绘制完整远期曲线以找到最佳期限，叠加波动率曲面以评估风险，并检查历史即期趋势以了解方向性背景。利差交易本质上是做空波动率——波动率上升是主要风险信号。

## 可用 MCP 工具

- **`fx_spot_price`** — 货币对的当前即期汇率。返回中间价/买价/卖价。所有利差分析的起点。
- **`fx_forward_price`** — 特定期限的远期汇率。返回远期点和远期全价。用于在目标期限计算利差。
- **`fx_forward_curve`** — 所有标准期限的完整远期曲线。两阶段：先列出再计算。用于绘制利差期限结构。
- **`fx_vol_surface`** — 按 delta 和到期的隐含波动率曲面。返回 ATM 波动率、风险反转、蝶式。用于利差波动比和偏斜评估。
- **`tscc_historical_pricing_summaries`** — 历史即期价格数据。用于计算已实现波动率和评估即期趋势方向。
- **`interest_rate_curve`** — 按货币的收益率曲线。用于理解驱动利差的利率差异。

## 工具链工作流

1. **获取即期汇率：** 为货币对调用 `fx_spot_price`。记录买卖价差作为流动性指标。
2. **定价远期：** 在目标期限调用 `fx_forward_price`。从远期点计算年化利差。
3. **绘制利差曲线：** 调用 `fx_forward_curve`（先列出再计算）。计算各期限的年化利差。识别风险调整后利差最佳期限。
4. **评估波动率风险：** 调用 `fx_vol_surface`。提取目标期限的 ATM 波动率、25-delta 风险反转（偏斜）和蝶式（尾部风险）。计算利差波动比。
5. **历史背景：** 调用 `tscc_historical_pricing_summaries` 获取 1 年日频数据。评估 52 周范围、趋势方向以及当前即期在范围内的位置。
6. **整合：** 合并为利差概况，包含利差波动比、波动率曲面信号和历史背景。推荐入场点并提供头寸规模指导。

### 1. Pips——"外汇市场最小的价格刻度"

**Pip（Percentage in Point，点）** 是外汇市场最小的价格变动单位。对于大多数货币对，1 pip = 0.0001（万分之一）。

**打比方**：你去超市买苹果，价格牌上写的是"每斤 ¥5.03"——最小刻度是 1 分钱。外汇市场的"1 分钱"就是 1 pip。EUR/USD 从 1.0850 涨到 1.0851，就是涨了 1 pip。

**不同货币对的 pip 值**：

| 货币对 | 1 pip | 例子 |
|--------|-------|------|
| EUR/USD | 0.0001 | 1.0850 → 1.0851 |
| USD/JPY | 0.01 | 149.50 → 149.51 |
| USD/CNY | 0.0001 | 7.2450 → 7.2451 |

**为什么日元（JPY）不一样**：日元面值太大（1 美元 = 149 日元），所以 pip 是 0.01 而不是 0.0001。这是历史惯例。

**Pips 和钱的关系**：假设你做多 100 万 EUR/USD，涨 1 pip 你赚多少？
```
盈利 = 1,000,000 × 0.0001 = $100
```
所以 1 pip = $100（对 100 万名义金额）。如果你做多 1000 万，1 pip = $1,000。

**真实例子**：2023 年 1 月，EUR/USD 从 1.0500 涨到 1.0900，涨了 400 pips。如果你做多 1000 万欧元，赚了 $400,000。

---

### 2. 远期点数→年化利差——"怎么把 pips 变成百分比"

**远期点数（Forward Points）** 是远期汇率和即期汇率的差，以 pips 表示。但利差交易关心的是"年化利差"——百分比形式的利率差。怎么转换？

**转换公式**：
```
年化利差 = (远期点数 / 即期汇率 × 10000) × (12 / 期限月数)
```

**实际例子**：
- AUD/JPY 即期 = 95.00
- 3 个月远期点数 = -120 pips（远期比即期低 1.20 日元）
- 远期汇率 = 95.00 - 1.20 = 93.80

年化利差 = (120 / 9500) × (12 / 3) = 0.01263 × 4 = **5.05%**

这意味着：借日元买澳元，3 个月能赚约 1.26% 的利差，年化约 5.05%。

**为什么远期点数是负的**：根据利率平价（Covered Interest Parity），高利率货币的远期汇率低于即期——这叫"远期贴水"（Forward Discount）。远期点数为负 = 高利率货币在远期贬值 = 利差被汇率抵消了一部分。

**关键洞察**：远期点数不是"免费的午餐"。它反映的是两国利率差——利差交易赚的利息，一部分会被远期贴水吃掉。所以你要看的是"净利差"而不是"名义利差"。

### 3. 利差期限结构——"哪个期限最划算"

**利差期限结构（Carry Term Structure）** 是不同期限（1M、3M、6M、1Y）的年化利差连成的曲线。它告诉你"做多哪个期限最划算"。

**想象一下**：你去银行存钱，1 个月利率 3%、3 个月 4%、6 个月 3.5%、1 年 3%。你不会自动选 1 年期——因为 3 个月利率最高。利差期限结构就是帮你找"最佳存款期限"。

**三种典型形态**：

| 形态 | 含义 | 最佳期限 | 原因 |
|------|------|---------|------|
| **向上倾斜** | 长期利差更高 | 1Y | 锁定长期利差 |
| **倒挂** | 短期利差更高 | 1M-3M | 短期赚完就走 |
| **驼峰形** | 中期最高 | 3M-6M | 中期甜蜜点 |

**为什么不只选利差最高的期限**：因为期限越长，汇率风险越大。3 个月利差 5% 但汇率可能波动 10%，1 年利差 4% 但汇率波动可能 15%。所以需要利差波动比来"风险调整"。

**本技能的工作流**：先调用 `fx_forward_curve` 获取所有期限的远期点数，然后计算每个期限的年化利差，画出利差期限结构，找到"风险调整后最佳期限"。

### 4. 25-delta 期权——"为什么比 ATM 更重要"

**Delta** 是期权价格对标的资产价格变动的敏感度。ATM 期权的 delta 大约是 0.50，25-delta 期权的 delta 是 0.25——更"虚值"（Out-of-the-Money）。

**打比方**：你买汽车保险。ATM 保险是"车稍微刮一下就赔"（保费贵），25-delta 保险是"车撞报废了才赔"（保费便宜）。利差交易员更关心 25-delta——因为利差交易的灾难是"汇率暴跌"，而 25-delta Put 的价格反映了"市场认为暴跌的概率有多大"。

**25-delta 在利差交易中的两个关键指标**：

**① 25-delta 风险反转（25Δ Risk Reversal）**：
```
风险反转 = 25Δ Call 隐含波动率 - 25Δ Put 隐含波动率
```
- **正值**：Call 比 Put 贵 → 市场更担心涨 → 对做多本币利差交易有利
- **负值**：Put 比 Call 贵 → 市场更担心跌 → 对做多本币利差交易不利

**② 25-delta 蝶式（25Δ Butterfly）**：
```
蝶式 = (25Δ Call IV + 25Δ Put IV) / 2 - ATM IV
```
- 蝶式越大 → 尾部风险越大 → 市场认为"极端行情"概率高
- 蝶式很小 → 市场认为价格会在正常范围内波动

**真实例子**：
- **2022 年 USD/JPY**：风险反转深度为负（Put 远贵于 Call），因为市场担心日本央行干预日元贬值。做多 USD/JPY 利差交易的人面临"日本央行突然拉升日元"的风险
- **2023 年 AUD/JPY**：风险反转略负但不太深，蝶式中等——利差交易可以接受的风险水平

### 5. 偏斜信号——"市场在怕什么"

**偏斜信号（Skew Signal）** 是风险反转的方向和大小，它告诉你"市场对这只货币对的情绪偏向哪边"。

**三种偏斜状态**：

| 偏斜 | 风险反转 | 含义 | 对利差交易的影响 |
|------|---------|------|----------------|
| **看涨偏斜** | 正值，较大 | 市场更担心涨 | 做多利差交易有利（市场跟你站一边） |
| **中性偏斜** | 接近零 | 市场没有方向偏好 | 利差交易主要看利差/波动比 |
| **看跌偏斜** | 负值，较大 | 市场更担心跌 | 做多利差交易危险（市场在跟你对赌） |

**怎么用偏斜信号**：

**想象一下**：你想在一条河上开船赚钱（利差交易）。偏斜信号就是"天气预报"——如果天气预报说"今天可能有大风从西边来"（看跌偏斜），你就知道往东开（做多利差）有风险。

**偏斜信号 + 利差波动比的组合判断**：

| 利差波动比 | 偏斜 | 结论 |
|-----------|------|------|
| 高（>1.0） | 看涨 | 强烈推荐做多 |
| 高（>1.0） | 中性 | 推荐，但设好止损 |
| 高（>1.0） | 看跌 | 谨慎——利差诱人但市场在担心什么 |
| 低（<0.5） | 看涨 | 利差不够，不值得冒险 |
| 低（<0.5） | 看跌 | 远离！利差低 + 市场看跌 = 火坑 |

**真实例子**：
- **2008 年金融危机前**：AUD/JPY 利差波动比很高（>1.5），但偏斜已经开始转负——聪明的交易员在危机前减仓了
- **2013 年"缩减恐慌"（Taper Tantrum）**：新兴市场货币利差波动比骤降，偏斜深度转负——大量利差交易被迫平仓

### 6. 利差交易的本质——"做空波动率"

**利差交易本质上是做空波动率**——你赌的是"汇率不会大幅波动"。只要汇率稳，你就赚利差；汇率一旦剧烈波动，你就亏钱。

**打比方**：你开了一家保险公司，卖"汇率稳定险"。平时每年收保费（利差），但如果出大事（汇率暴跌），你要赔很多钱。2008 年、2020 年 3 月、2022 年——每次市场恐慌，利差交易都遭受重创。

**所以本技能的 6 步工作流不是随便排的**：
1. **即期** → 确认"现在汇率是多少"
2. **远期** → 确认"利差有多少"
3. **利差曲线** → 确认"哪个期限最划算"
4. **波动率** → 确认"风险有多大"——**这是最关键的一步**
5. **历史** → 确认"过去有没有出过大事"
6. **整合** → 把利差和风险放在一起，得出"值不值得做"

> **给小白的一句话**：利差交易就像"收租金"——平时稳稳收钱，但遇到台风（汇率暴跌）房子可能被吹倒。Pips 是租金的最小单位，远期点数告诉你"租金是多少"，利差曲线告诉你"签多久的租约最划算"，波动率和偏斜告诉你"台风来了怎么办"。记住：利差是收入，波动率是风险，两者必须一起看。
