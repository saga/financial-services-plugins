# LSEG 金融分析插件

使用 LSEG 金融数据与分析工具，对债券定价、分析收益率曲线、评估外汇利差交易、为期权估值并构建宏观仪表盘。

## 功能概述

本插件将 LSEG 的金融分析 MCP 工具封装为 8 个高级工作流，将多个工具调用组合为完整的金融分析任务。您无需逐一调用单个工具，每个命令会自动编排 4-5 个工具完成连贯的分析。

## 命令

| 命令 | 说明 |
|------|------|
| `/analyze-bond-rv` | 分析债券相对价值，含利差分解和情景压力测试 |
| `/analyze-fx-carry` | 评估外汇利差交易机会，结合即期、远期、波动率曲面和历史背景 |
| `/research-equity` | 生成股权研究快照，含一致预期、基本面数据和价格表现 |
| `/analyze-swap-curve` | 分析互换利率曲线，叠加国债和通胀曲线寻找曲线交易机会 |
| `/analyze-option-vol` | 分析期权波动率，展示波动率曲面、希腊字母和隐含 vs 已实现波动率对比 |
| `/review-fi-portfolio` | 审查固定收益组合，含定价、现金流和情景分析 |
| `/macro-rates` | 构建宏观利率仪表盘，包含经济指标、收益率曲线和互换利差 |
| `/analyze-bond-basis` | 分析国债期货基差，识别最便宜可交割债券并计算隐含回购利率 |

## 技能

每个命令有对应的技能，提供深度领域知识：

| 技能 | 领域知识 |
|------|----------|
| `bond-relative-value` | 利差框架、G-spread/Z-spread/OAS、贵贱分析 |
| `fx-carry-trade` | 利差机制、利差波动比、G10 和新兴市场利差动态 |
| `equity-research` | IBES 一致预期解读、基本面分析、估值指标 |
| `swap-curve-strategy` | 互换曲线构建、曲线交易、实际利率分析 |
| `option-vol-analysis` | 波动率曲面解读、SABR 模型、希腊字母、隐含 vs 已实现波动率 |
| `fixed-income-portfolio` | 组合分析、关键利率久期、现金流分析、情景测试 |
| `macro-rates-monitor` | 宏观指标、收益率曲线形态、实际利率、金融状况 |
| `bond-futures-basis` | CTD 机制、基差计算、隐含回购、交割期权 |

## 集成

本插件连接到 **LFA MCP Server**，可获取 LSEG 金融数据和以下分析功能：

- **债券定价** — 债券及期货估值
- **外汇定价** — 即期和远期汇率
- **曲线** — 利率、信用、通胀和外汇远期曲线
- **互换** — 利率互换定价
- **期权** — 含完整希腊字母的期权估值
- **波动率** — 外汇和股票的隐含波动率曲面
- **量化分析** — 分析师预测、公司基本面、股价、宏观数据
- **时间序列** — 历史价格汇总
- **YieldBook** — 固收参考数据、现金流、情景和风险分析

完整工具参考见 [CONNECTORS.md](CONNECTORS.md)。

### 1. LSEG 是什么——金融数据的"瑞士军刀"

**想象一个场景**：你是一家对冲基金的交易员，今天早上要同时做三件事：①检查日本国债期货的基差交易机会、②评估做多澳元兑日元的利差交易、③看看标普 500 期权的波动率是不是偏贵了。你需要债券数据、外汇数据、期权数据——从哪来？

**答案是 LSEG**。LSEG（London Stock Exchange Group，伦敦证券交易所集团）不只是"伦敦的证券交易所"，它更像金融数据的"瑞士军刀"——一把工具集齐了债券、外汇、利率、期权、股票、宏观……几乎所有金融市场的数据。

**LSEG 的核心产品线**：

| 产品 | 干什么 | 类比 |
|------|--------|------|
| **LSEG Data & Analytics** | 金融数据终端（类似 Bloomberg） | 金融世界的"搜索引擎" |
| **Refinitiv** | 数据和交易平台（2021 年收购） | 金融数据的"自来水管" |
| **LCH** | 全球最大清算所之一 | 交易的"担保人" |
| **FTSE Russell** | 指数提供商（如富时 100 指数） | 股市的"评分机构" |
| **YieldBook** | 固收分析平台 | 债券的"计算器" |

**真实例子**：全球 40,000+ 家机构使用 LSEG 数据，包括几乎所有大型银行和对冲基金。当摩根大通的债券交易员查看某只德国国债的收益率时，背后很可能就是 LSEG 提供的数据。

### 3. MCP——让 AI "动手"的协议

**MCP（Model Context Protocol，模型上下文协议）** 是让 AI 模型安全调用外部工具和数据的开放协议。

**想象一下**：你有一个很聪明的助手（AI），但它只能"说话"不能"动手"——它不能帮你查实时汇率、不能帮你计算债券价格、不能帮你画波动率曲面。MCP 就是给这个助手装上"手"——它现在可以去 LSEG 的数据库里取数据、调用定价模型、把结果整理好给你。

**MCP 的三层架构**：

| 层级 | 角色 | 类比 |
|------|------|------|
| **AI 模型** | "大脑"——理解你的需求，决定调用什么工具 | 餐厅经理 |
| **MCP 协议** | "手"——安全地连接 AI 和外部工具 | 服务员 |
| **LSEG MCP Server** | "工具箱"——提供金融数据和计算能力 | 厨房 |

**真实例子**：你说"分析 USDJPY 的利差交易机会"。AI 通过 MCP 调用 LSEG 的 `fx_spot_price`（拿即期汇率）、`fx_forward_price`（拿远期汇率）、`fx_vol_surface`（拿波动率）、`tscc_historical_pricing_summaries`（拿历史数据），4 个工具调用组合成一份完整报告——你不需要知道这些工具的名字，AI 自动编排。

### 4. IBES——"华尔街共识"的数据库

**IBES（Institutional Brokers Estimate System，机构经纪人估计系统）** 是全球最大的分析师预测数据库，中文常称"一致预期数据"。

**想象一下**：你想知道苹果下一季度能赚多少钱。你可以自己猜，但更聪明的做法是看看覆盖苹果的 40 个分析师的平均预测——这就是"一致预期"。IBES 就是收集和汇总这些预测的数据库。

**IBES 里有什么**：
- **EPS 预测**：每个分析师对每股收益的预测
- **收入预测**：每个分析师对总收入的预测
- **评级分布**：多少分析师给"买入"、多少给"持有"、多少给"卖出"
- **目标价**：每个分析师给出的 12 个月目标价

**真实例子**：2024 年 Q1，英伟达的一致预期 EPS 是 $5.58，实际报了 $6.12——超出预期 9.7%。这个"Beat"信息就来自 IBES 数据。如果你是 `research-equity` 命令的用户，你会看到一致预期 vs 实际结果的对比。

**为什么 IBES 重要**：在股权研究中，一致预期是"市场定价的基准"。股票涨跌往往不是看"公司赚了多少"，而是看"公司赚的比大家预期的多还是少"——这就是 IBES 的价值。

### 5. YieldBook——债券世界的"超级计算器"

**YieldBook** 是 LSEG 旗下的固定收益分析平台，提供债券定价、现金流分析、情景测试和风险指标。

**想象一下**：你管理一个 100 只债券的组合。利率上升 50 个基点，你的组合亏多少？哪只债券亏最多？如果信用利差扩大 100 个基点呢？这些计算手动做要一整天，用 YieldBook 只要几秒。

**YieldBook 能做什么**：
- **债券定价**：给定收益率，计算债券价格（反之亦然）
- **风险指标**：久期、凸性、DV01、关键利率久期
- **现金流分析**：每只债券未来每年的现金流
- **情景测试**：利率变、利差变、曲线变，组合怎么变
- **OAS 分析**：含权债券的期权调整利差

**真实例子**：太平洋投资管理公司（PIMCO）管理着 2 万亿美元的债券组合。他们的组合经理每天用 YieldBook 计算组合的久期和 DV01，确保利率风险在可控范围内。当美联储加息时，他们能立刻知道"利率上升 25bp，我们亏多少"。

### 6. 8 个命令的"数据流"——每个命令调用了什么

**每个命令不是只调一个工具，而是编排 4-5 个工具调用形成完整分析**。这就像做一道菜需要多种食材——光有盐不够，还需要油、酱油、糖……

| 命令 | 调用的核心工具 | 组合逻辑 |
|------|--------------|---------|
| `analyze-bond-basis` | 期货定价 + 债券定价 + 利率曲线 + 历史数据 + 信用曲线 | 5 步：定价 → 找CTD → 算基差 → 算隐含回购 → 历史对比 |
| `analyze-swap-curve` | 互换定价 + 利率曲线 + 通胀曲线 | 3 层叠加：互换 → 国债 → 通胀 |
| `analyze-fx-carry` | 即期 + 远期 + 远期曲线 + 波动率曲面 + 历史数据 | 5 维：即期 → 远期 → 利差曲线 → 波动率 → 历史 |
| `analyze-option-vol` | 波动率曲面 + 期权定价 + 历史数据 | 3 维：曲面 → 希腊字母 → IV vs RV |
| `research-equity` | IBES + 基本面 + 价格 + 时间序列 | 4 维：预期 → 基本面 → 价格 → 历史 |

**所以这意味着什么**：LSEG 插件的强大之处不在于单个工具，而在于**工具组合**。就像乐高积木——单块积木没什么用，但组合起来可以搭出任何东西。

> **给小白的一句话**：LSEG 插件就像一个"金融分析厨房"——LSEG 提供食材（数据）和厨具（工具），8 个命令是预制菜谱，MCP 是服务员帮你传菜。你不需要知道每个工具怎么用，只需要告诉 AI "我要什么分析"，它就会自动编排工具、取数据、出报告。记住：**金融分析的本质不是算数，而是把不同维度的信息组合成有意义的判断**——LSEG 插件做的就是帮你把散落的数据拼成完整的画面。
