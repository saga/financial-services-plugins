# LSEG 金融分析插件

使用 LSEG 金融数据与分析工具，对债券定价、分析收益率曲线、评估外汇利差交易、为期权估值并构建宏观仪表盘。

## 功能概述

本插件将 LSEG 的金融分析 MCP 工具封装为 8 个高级工作流，将多个工具调用组合为完整的金融分析任务。您无需逐一调用单个工具，每个命令会自动编排 4-5 个工具完成连贯的分析。

## 命令

| 命令 | 说明 |
|------|------|
| `/analyze-bond-rv` | 分析债券相对价值，含利差分解和情景压力测试 |
| `/analyze-fx-carry` | 评估外汇利差交易机会，结合即期、远期、波动率曲面和历史背景 |
| `/research-equity` | 生成股权研究快照，含一致预期、基本面数据和价格表现 |
| `/analyze-swap-curve` | 分析互换利率曲线，叠加国债和通胀曲线寻找曲线交易机会 |
| `/analyze-option-vol` | 分析期权波动率，展示波动率曲面、希腊字母和隐含 vs 已实现波动率对比 |
| `/review-fi-portfolio` | 审查固定收益组合，含定价、现金流和情景分析 |
| `/macro-rates` | 构建宏观利率仪表盘，包含经济指标、收益率曲线和互换利差 |
| `/analyze-bond-basis` | 分析国债期货基差，识别最便宜可交割债券并计算隐含回购利率 |

## 技能

每个命令有对应的技能，提供深度领域知识：

| 技能 | 领域知识 |
|------|----------|
| `bond-relative-value` | 利差框架、G-spread/Z-spread/OAS、贵贱分析 |
| `fx-carry-trade` | 利差机制、利差波动比、G10 和新兴市场利差动态 |
| `equity-research` | IBES 一致预期解读、基本面分析、估值指标 |
| `swap-curve-strategy` | 互换曲线构建、曲线交易、实际利率分析 |
| `option-vol-analysis` | 波动率曲面解读、SABR 模型、希腊字母、隐含 vs 已实现波动率 |
| `fixed-income-portfolio` | 组合分析、关键利率久期、现金流分析、情景测试 |
| `macro-rates-monitor` | 宏观指标、收益率曲线形态、实际利率、金融状况 |
| `bond-futures-basis` | CTD 机制、基差计算、隐含回购、交割期权 |

## 集成

本插件连接到 **LFA MCP Server**，可获取 LSEG 金融数据和以下分析功能：

- **债券定价** — 债券及期货估值
- **外汇定价** — 即期和远期汇率
- **曲线** — 利率、信用、通胀和外汇远期曲线
- **互换** — 利率互换定价
- **期权** — 含完整希腊字母的期权估值
- **波动率** — 外汇和股票的隐含波动率曲面
- **量化分析** — 分析师预测、公司基本面、股价、宏观数据
- **时间序列** — 历史价格汇总
- **YieldBook** — 固收参考数据、现金流、情景和风险分析

完整工具参考见 [CONNECTORS.md](CONNECTORS.md)。

```
claude plugins add LSEG
```

## 📖 本篇金融知识小贴士

**LSEG**：伦敦证券交易所集团（London Stock Exchange Group），全球领先的金融基础设施和数据提供商。

**MCP（Model Context Protocol）**：一种让 AI 模型安全调用外部工具和数据的开放协议。本插件通过 MCP 连接 LSEG 的金融分析服务器。

**收益率曲线（Yield Curve）**：同一信用质量、不同期限的利率连成的曲线。正常曲线向上倾斜（借越久利率越高），倒挂（短端高于长端）常预示经济衰退。

**利差（Spread）**：两种利率之差。比如公司债收益率减国债收益率，差额就是"信用利差"，代表市场对该公司违约风险的定价。

**久期（Duration）**：债券价格对利率变动的敏感度，以年为单位。久期=5 意味着利率每涨1%，债券价格约跌5%。

**DV01（Dollar Value of 1bp）**：利率变动1个基点（0.01%），债券价格变动多少钱。衡量债券的利率风险敞口。

**OAS（期权调整利差，Option-Adjusted Spread）**：剔除嵌入式期权影响后的债券利差，用于公平比较含权债券和不含权债券。

**隐含波动率（Implied Volatility）**：从期权市场价格反推出来的"市场预期波动幅度"。越高说明市场认为未来价格波动越大。

**希腊字母（Greeks）**：衡量期权价格对不同因素敏感度的指标。Delta（对股价）、Gamma（Delta变化速度）、Vega（对波动率）、Theta（时间流逝）、Rho（对利率）。

**IBES（Institutional Brokers' Estimate System）**：全球最大的分析师预测数据库，收集所有券商分析师对公司盈利的预测。中文常称"一致预期"。

**EV（企业价值，Enterprise Value）**：市值 + 总负债 - 现金及等价物。收购一家公司的实际成本，比市值更全面地反映公司价值。
