# 会计与审计（Accounting & Audit）

> 适用文件：variance-commentary/SKILL.md, roll-forward/SKILL.md, accrual-schedule/SKILL.md, 任何提到"应计/试算表/总账/审计"的文件

---

## 1. 会计是什么

**想象一个场景**：你开了家小公司。一年下来，你要回答这些问题：
- 今年赚了多少钱？
- 现在手头有多少现金？
- 别人欠我多少钱？我欠别人多少钱？
- 今年的支出合理吗？

**会计就是回答这些问题的"记账系统"**。

**会计的两大分支**：
- **管理会计（Management Accounting）**：给内部管理者用
- **财务会计（Financial Accounting）**：给外部投资者、监管用

**核心原则**（GAAP / IFRS）：
- **权责发生制（Accrual）**：费用/收入在"发生时"记录，不是"收/付现金时"
- **配比原则（Matching）**：费用和产生收入的同一期确认
- **历史成本（Historical Cost）**：资产按买入价记账（市价变动不调整）
- **谨慎性（Conservatism）**：不确定时，宁可低估资产/收入、高估负债/费用

## 2. 三大财务报表

### 2.1 利润表（Income Statement）

**翻译**：看你赚不赚钱。

**结构**：
```
收入（Revenue）
- 销售成本（COGS）
= 毛利（Gross Profit）
- 营业费用（SG&A + R&D）
= 营业利润（Operating Income）
+ 利息收入 / - 利息支出
- 所得税
= 净利润（Net Income）
```

**关键概念**：

| 概念 | 公式 | 含义 |
|------|------|------|
| **毛利** | 收入 - COGS | 直接赚的钱 |
| **毛利率** | 毛利 ÷ 收入 | 定价权（软件 70-90%，零售 20-40%） |
| **营业利润率** | 营业利润 ÷ 收入 | 主营赚钱能力 |
| **净利润率** | 净利润 ÷ 收入 | 最终落到口袋的比例 |
| **EPS** | 净利润 ÷ 流通股数 | 每股赚多少 |

### 2.2 资产负债表（Balance Sheet）

**翻译**：看家底厚不厚。

**核心恒等式**：

```
资产（Assets）= 负债（Liabilities）+ 股东权益（Equity）
```

**结构**：
```
流动资产（< 1 年）
  - 现金
  - 应收账款（AR）
  - 存货
  - 预付费用
+ 长期资产
  - PP&E（不动产、厂房、设备）
  - 无形资产（专利、品牌、商誉）
= 总资产

流动负债（< 1 年）
  - 应付账款（AP）
  - 短期借款
  - 应计费用
+ 长期负债
  - 长期借款
  - 债券
= 总负债

+ 股东权益
  - 实收资本
  - 留存收益
  - 其他综合收益
= 总权益
```

**关键比率**：

| 比率 | 公式 | 含义 |
|------|------|------|
| **流动比率** | 流动资产 ÷ 流动负债 | 短期偿债（>1 健康） |
| **速动比率** | (流动资产 - 存货) ÷ 流动负债 | 严格流动性 |
| **债务/权益** | 总负债 ÷ 股东权益 | 财务杠杆 |
| **ROE** | 净利润 ÷ 股东权益 | 股东回报率 |
| **ROA** | 净利润 ÷ 总资产 | 资产效率 |

### 2.3 现金流量表（Cash Flow Statement）

**翻译**：看真金白银的进出。

**三段**：
```
经营活动现金流（CFO）
  + 投资活动现金流（CFI）
  + 融资活动现金流（CFF）
= 现金净变动
```

**为什么重要**：利润可以"会计调整"出好看的数字，**现金难造假**。

**FCF（自由现金流）**：

```
FCF = CFO - CapEx（资本支出）
```

**含义**：公司"真金白银"能拿来回购、分红、扩张的钱。

**FCF 转换率**：

```
FCF Conversion = FCF ÷ 净利润
```

- **> 100%**：优秀（SaaS，递延收入）
- **80-100%**：良好
- **< 50%**：危险（赚的是"应收"不是现金）

## 3. GAAP vs 调整后

### 3.1 GAAP（通用会计准则）

**美国**：GAAP（由 FASB 制定）
**国际**：IFRS（由 IASB 制定）

**特点**：
- 法定、严格
- 上市公司必须用
- 比较公司之间可比

### 3.2 调整后（Non-GAAP / Adjusted）

**公司自己定义**——剔除"一次性项目"：

- 重组费用
- 收购相关成本
- 股票薪酬（SBC）
- 减值
- 汇兑损益
- 诉讼和解

**举例**（苹果 FY2024）：
- GAAP 净利润：930 亿
- 调整后：1010 亿（差 80 亿，主要 SBC + 一次性项目）

**警惕信号**：
- GAAP vs 调整后差距 > 30% → 可能过度调整
- SBC 被剔除但其实是真实成本
- 多年趋势差距扩大

## 4. 关键会计操作

### 4.1 应计（Accrual）

**核心**：钱进出跟记录的时间可能不同。

**举例**：
- 12 月 31 日，水电公司还没寄账单
- 但 12 月的 1 万美元水电费"已经发生"
- 会计应计：
  - 借：水电费 1 万
  - 贷：应付水电费 1 万
- 1 月实际付时：回转应计，正常过账

### 4.2 滚调表（Roll-forward Schedule）

**核心**：账户余额从期初到期末的变动明细。

```
期初余额
+ 新增/新活动
+ 应计
- 回转
- 付款/结算
+ 重分类
+ 外汇折算
= 期末余额
```

**举例**（应付账款滚调）：
- 期初：100 万
- 本月新增采购：500 万
- 本月付款：450 万
- 期末：150 万

**作用**：
- 验证期末余额
- 识别异常变动
- 提供审计轨迹

### 4.3 重分类（Reclass）

**核心**：把钱从一个账户搬到另一个账户。

**典型场景**：
- 客户预付从"递延收入" → "收入"（确认时）
- PP&E 出售从"PP&E" → "现金 + 处置损益"
- 短期借款重分类为长期借款（接近到期）

### 4.4 应计项目类型

| 类型 | 会计 |
|------|------|
| **应付工资** | 12 月工资 1 月发 → 12 月应计 |
| **应付利息** | 季付利息，月度应计 |
| **预提费用** | 已知金额但未开发票 |
| **递延收入** | 客户预付，分期确认 |
| **坏账准备** | 预计收不回的应收账款 |

## 5. 月末结账（Month-End Close）

**翻译**：每月结束后，把账做完，生成财务报表。

**典型流程**（1-10 天）：

| 日 | 工作 |
|---|------|
| **D+1-D+3** | 录入所有交易、过账日记账 |
| **D+4-D+5** | 银行对账、应计项目计提 |
| **D+6-D+7** | 子公司报表合并、内部对账 |
| **D+8-D+9** | 差异说明、管理层审阅 |
| **D+10** | 财务报表定稿、签字 |

**关键文档**：
- **试算表（Trial Balance）**：所有总账账户余额
- **结账包（Close Package）**：财务报表 + 支持文件
- **波动分析（Variance Analysis）**：实际 vs 预算/前期

**签字流程**：
- 财务分析师 → 财务经理 → 主管（Controller）→ CFO

---

## 6. 波动分析（Variance Analysis）

**翻译**：实际 vs 预算/前期的差异。

**关键概念**：
- **绝对差异**：金额差（如收入 +50 万美元）
- **相对差异**：百分比差（如收入 +10%）
- **有利（Favorable）**：收入高 / 费用低
- **不利（Unfavorable）**：收入低 / 费用高

**材料性阈值（Materiality Threshold）**：差异超过多少要解释。
- 百分比：5%
- 绝对值：$10,000
- 取较大者

**波动分解**：
- **价格影响**：单价变化
- **销量影响**：销售数量变化
- **混合影响**：产品组合变化
- **汇率影响**：外币折算

**举例**（餐饮连锁）：
- 本月销售比预算高 10%
- 拆解：销量 +8%（新店开业）+ 涨价 +2%

## 7. 审计（Audit）

**类型**：
- **年度审计（Statutory Audit）**：法定年度审计
- **季度复核（Quarterly Review）**：事务所复核
- **内部审计（Internal Audit）**：公司内部审计部门
- **SOX 审计**：上市公司内控审计

**关键概念**：
- **重要性（Materiality）**：会影响财务报表使用者决策的金额
- **审计风险（Audit Risk）**：审计师发表不适当意见的风险
- **控制测试（Control Testing）**：测试公司内控有效性
- **实质性程序（Substantive Procedures）**：直接测试金额

**审计意见类型**：
- **无保留（Unqualified）**：一切正常
- **保留（Qualified）**：有特定问题
- **否定（Adverse）**：财务报表不可信
- **无法表示意见（Disclaimer）**：证据不足

---

## 8. 给小白的一句话

> **会计是商业的"语言"**——不会会计就像不会说话，看不懂财报就看不懂公司。**权责发生制是核心**——记录的是"经济实质"不是"现金流时点"。**三大报表是窗口**：**利润表**看赚钱能力，**资产负债表**看家底，**现金流量表**看真金白银。**GAAP 是法律底线，调整后可能粉饰**——警惕"调整后 vs GAAP 差距过大"。**月末结账是月度大考**——考验的是流程、系统和人的协作。**审计是外部独立检查**——给投资者信心，但也耗时耗钱。
