# LSEG 金融分析插件

利用 LSEG 金融数据与分析工具，进行债券定价、收益率曲线分析、外汇套息交易评估、期权估值，并构建宏观数据看板。

## 插件功能概述

本插件将 LSEG 的金融分析 MCP 工具封装为 8 个高级工作流。这些工作流将针对常见金融分析任务的多个工具调用整合在一起。用户无需逐个调用原始工具，每个命令都会统筹 4 到 5 个工具，形成严密的分析逻辑。

## 命令列表

| 命令 | 描述 |
|---------|-------------|
| `/analyze-bond-rv` | 债券相对价值分析：通过利差拆解和场景压力测试，深入剖析债券价值 |
| `/analyze-fx-carry` | 外汇套息评估：综合现货、远期、波动率曲面及历史背景，评估套息交易机会 |
| `/research-equity` | 权益研究概览：生成包含共识预期、基本面信息和价格表现的分析简报 |
| `/analyze-swap-curve` | 掉期曲线分析：叠加国债与通胀数据，挖掘曲线交易策略 |
| `/analyze-option-vol` | 期权波动率分析：结合波动率曲面、希腊字母，并对比隐含与实际波动率 |
| `/review-fi-portfolio` | 固收投资组合审查：通过定价、现金流和场景分析，全面审查投资组合 |
| `/macro-rates` | 宏观利率看板：集成经济指标、收益率曲线和掉期利差的综合数据面板 |
| `/analyze-bond-basis` | 债券基差分析：进行国债期货基差分析，包括 CTD 识别和隐含回购利率计算 |

## 专业技能

每个命令都由相应的技能模块支持，提供深度的行业领域知识：

| 技能名称 | 领域知识库 |
|-------|-----------------|
| `bond-relative-value` | 利差分析框架、G-利差/Z-利差/OAS、高估-低估分析 |
| `fx-carry-trade` | 套息机制、收益波动率比、G10 及新兴市场套息动态 |
| `equity-research` | IBES 共识解读、基本面分析、估值指标 |
| `swap-curve-strategy` | 掉期曲线构建、曲线交易策略、实际利率分析 |
| `option-vol-analysis` | 波动率曲面解读、SABR 模型、希腊字母风险指标、隐含与实际波动率对比 |
| `fixed-income-portfolio` | 投资组合分析、关键期限久期、现金流分析、场景测试 |
| `macro-rates-monitor` | 宏观指标、收益率曲线形态、实际利率、金融状况指数 |
| `bond-futures-basis` | CTD 机制、基差计算、隐含回购利率、交割期权 |

## 系统集成

本插件连接至 **LFA MCP 服务器**，可跨以下领域获取 LSEG 的金融数据与分析能力：

- **债券定价** — 债券及国债期货估值
- **外汇定价** — 现货与远期汇率
- **曲线分析** — 利率、信贷、通胀及外汇远期曲线
- **掉期工具** — 利率掉期定价
- **期权工具** — 期权估值及全套希腊字母风险指标
- **波动率分析** — 外汇与权益隐含波动率曲面
- **量化分析** — 分析师预测、公司基本面、股价、宏观经济数据
- **时间序列** — 历史定价摘要
- **YieldBook** — 固定收益参考数据、现金流、情景模拟及风险分析

完整的工具参考请参阅 [CONNECTORS.md](CONNECTORS.md)。

```
claude plugins add LSEG
```

- 需拥有相关产品线的 LSEG 数据使用授权

> **💡 Appendix: 领域知识小贴士**
>
> - **收益率曲线 (Yield Curve)**：简单来说，就是把借钱期限（比如1个月、10年、30年）和对应的利息连成一条线。通常借钱时间越长，利息越高（曲线向上）。如果短期的利息反而比长期的还高，那就叫“曲线倒挂”，往往是经济不景气的预警信号。
> - **外汇套息交易 (FX Carry Trade)**：金融界的“借低买高”。由于不同国家的利率不同，交易员会借入低利率国家的钱（如日元），换成高利率国家的货币（如澳元）存起来赚利差。但风险在于，如果汇率波动剧烈，赚的利息可能还不够赔汇率的损失。
> - **希腊字母 (Greeks)**：这是期权交易员的“仪表盘”。Delta 告诉你价格变动的影响，Gamma 告诉你加速的情况，Vega 关注波动率，Theta 则是时间的流逝。通过这些指标，交易员能精准调控风险。
> - **相对价值分析 (Relative Value, RV)**：这是一种“货比三家”的策略。与其争论一个债券本身好不好，不如看它和同类债券相比是“贵了”还是“便宜了”。通过捕捉这种定价不平衡，交易员可以进行套利。
> - **CTD (最便宜可交割券)**：在国债期货交割时，卖方可以从一系列符合条件的国债中挑选一种进行实物交割。理性的卖方肯定会选那个对自己最划算、成本最低的，这个债券就叫 CTD。它是国债期货定价的核心锚点。
