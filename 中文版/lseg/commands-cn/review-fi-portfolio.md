# 审查固定收益组合

> 本命令使用 LSEG 债券定价、YieldBook 分析以及收益率曲线工具。可用工具请参阅 [CONNECTORS.md](../CONNECTORS.md)。

通过对所有持仓进行定价、填充参考数据、预测现金流以及在不同利率情景下进行压力测试，生成一份综合的固定收益组合风险与回报报告。

关于投资组合分析和情景分析的领域知识，请参阅 **fixed-income-portfolio** 技能。

## 工作流

### 1. 收集组合持仓

向用户询问：
- 债券识别码（必填）— 以逗号分隔的 ISIN、CUSIP 或 RIC 代码
- 头寸规模/权重（可选 — 若未提供，则假设为等权重）
- 待测试的具体情景（可选 — 例如“+100bp”，默认为标准网格）
- 估值日期（可选，默认为今日）

### 2. 为所有债券定价

调用 `bond_price` 获取所有识别码的定价信息。

提取每支债券的：净价/全价、收益率、久期、凸性、DV01、币种。

汇总组合层面指标：加权收益率、加权久期、总 DV01、总市值。

### 3. 填充参考数据

为每支债券调用 `yieldbook_bond_reference`。

提取：证券类型、行业板块、评级、票息类型、赎回条款、发行人、国家。

构建构成拆解：按行业、评级、到期期限分布、币种。

### 4. 预测现金流

为每支债券调用 `yieldbook_cashflow`。

汇总为季度现金流瀑布图。标记到期日高度集中的时段。

### 5. 运行情景分析

调用 `yieldbook_scenario` 进行利率偏移测试：-200bp, -100bp, -50bp, 0bp, +50bp, +100bp, +200bp。

识别哪些债券对上行和下行风险的贡献最大。

### 6. 曲线环境分析

调用 `interest_rate_curve` 获取投资组合主币种的利率曲线。

计算每支债券相对于曲线的利差（Spread to Curve）。评估当前的收益率曲线环境。

### 7. 综合报告

展示：投资组合摘要指标、构成拆解、现金流瀑布、带有风险贡献者的情景损益表，以及曲线风险敞口。

## 输出格式

首先展示投资组合摘要指标，随后分章节详细阐述构成分布、现金流和风险分析。

> **💡 Appendix: 领域知识小贴士**
>
> 1. **固定收益 (Fixed Income)**：简单来说就是“借钱收息”。债券就是最典型的固收产品，你把钱借给政府或公司，他们承诺定期给你利息并在到期时还本。
> 2. **久期 (Duration)**：这是衡量债券对利率变化“敏感度”的标尺。如果久期是 5，当利率上升 1% 时，债券价格大约会下跌 5%。久期越长，波动风险越大。
> 3. **净价 (Clean Price) 与 全价 (Dirty Price)**：在两次付息之间买债，你需要把这段时间的利息补偿给前任持有人。全价 = 净价 + 应计利息。金融市场上报价通常用净价，但实际交易付钱用全价。
> 4. **基点 (Basis Point / bp)**：金融圈的最小计量单位。1 个基点等于 0.01%（万分之一）。所以 "+100bp" 意思就是利率上升了 1%。
> 5. **利差 (Spread)**：代表了“风险溢价”。如果一支企业债的收益率比国债高 2%，这 2% 就是利差，反映了市场对这家公司违约风险的补偿要求。
