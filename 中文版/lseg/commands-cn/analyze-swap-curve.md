# 分析互换曲线

> 本命令使用 LSEG 互换定价、利率曲线及通胀曲线工具。可用工具请参阅 [CONNECTORS.md](../CONNECTORS.md)。

构建并分析利率互换曲线，叠加政府债券收益率及通胀盈亏平衡率，从而识别曲线交易机会。

有关曲线分析和交易构建的领域知识，请参阅 **swap-curve-strategy** 技能。

## 工作流

### 1. 采集输入

询问用户：
- 货币（必填）—— 示例：EUR, USD, GBP, CHF, JPY
- 参考利率指数（选填）—— 示例：ESTR, SOFR, SONIA, TONA
- 估值日期（选填，默认为今日）

### 2. 探索互换模板

以列表（list）模式调用 `ir_swap` 工具，输入货币和可选指数。

提取：可用模板参考、指数详情、操作惯例。

### 3. 构建互换曲线

针对标准期限（2Y, 5Y, 7Y, 10Y, 20Y, 30Y），以定价（price）模式调用 `ir_swap`。

提取：每个期限的平价互换利率（Par Swap Rate）和 DV01。

### 4. 叠加政府债券曲线

对同一货币调用 `interest_rate_curve`（先列出列表，再进行计算）。

计算互换利差（Swap Spread） = 每个期限的互换利率减去政府债券收益率。

### 5. 分解真实利率

对该货币调用 `inflation_curve`（先搜索，再进行计算）。

计算真实互换利率 = 每个期限的名义互换利率减去通胀盈亏平衡率。

### 6. 综合曲线策略观点

计算曲线指标：2s10s 斜率、5s30s 斜率、2s5s10s 蝶式组合（Butterfly）。

识别机会：根据当前水平与历史常态的对比，识别陡峭化交易（Steepener）、平坦化交易（Flattener）、蝶式交易或互换利差交易。

展示：包含政府债券叠加的互换曲线表、曲线指标、真实利率分解，以及基于 DV01 中性比例的交易推荐。

## 输出格式

首段通过曲线形状摘要和关键指标（2s10s, 蝶式组合）开篇。随后提供详细数据表和交易策略建议章节。

> **💡 Appendix: 领域知识小贴士**
>
> *   **利率互换 (Interest Rate Swap)**：就像是两个人交换“付利息的方式”。一方按固定利率付息，另一方按波动的浮动利率付息。这能帮助企业锁住筹资成本或规避利率波动风险。
> *   **互换曲线 (Swap Curve)**：这是一张显示不同借款期限（如 2 年到 30 年）下，互换利率高低趋势的“地图”。它是金融市场定价的重要基准。
> *   **互换利差 (Swap Spread)**：它是互换利率与国债收益率之间的差额。因为国债被认为最安全，这个差额反映了银行间市场的信用风险和流动性溢价。
> *   **通胀盈亏平衡率 (Inflation Breakeven)**：这是市场“预测”的未来通胀水平。它通过名义债券和抗通胀债券的利差计算得出。
> *   **DV01 (Dollar Value of a Basis Point)**：指的是利率每变动 0.01%（1 个基点）时，头寸价值变动的金额。它是衡量利率风险最基本的指标。
> *   **陡峭化 (Steepener) 与平坦化 (Flattener)**：这是交易员对曲线形状变化的打赌。如果你认为长短期利差会变大，就做“陡峭化”；如果你认为长短期利率会趋于一致，就做“平坦化”。
> *   **蝶式组合 (Butterfly)**：一种进阶策略，同时操作曲线的短、中、长三个点（如 2年、5年、10年），用来赌曲线中间部分相对于两头的“弯曲”程度。
