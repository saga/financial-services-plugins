### 步骤 2：构建 DCF 分析

#### A. 计算 WACC

**1. 确定无风险利率 (Risk-Free Rate)**
   - 使用 10 年期国债收益率（查看当前利率）
   - 示例：截至 2024 年底约为 4.0-4.5%

**2. 确定权益资本成本 (CAPM)**
   ```
   权益资本成本 = 无风险利率 + Beta × 权益风险溢价

   输入变量：
   - 无风险利率：[当前 10 年期国债，例如 4.2%]
   - Beta：[来自 Bloomberg/FactSet 的公司 Beta 或同行均值]
   - 权益风险溢价：5-6% (历史均值)

   示例：
   权益资本成本 = 4.2% + 1.3 × 5.5% = 11.35%
   ```

**3. 确定债务资本成本**
   ```
   债务资本成本 = 当前借贷利率或债券隐含收益率

   对于私有公司：
   债务资本成本 = 无风险利率 + 信用利差 (基于信用评级)

   示例：
   税前债务成本 = 6.5%
   税后债务成本 = 6.5% × (1 - 25% 税率) = 4.875%
   ```

**4. 确定资本结构**
   ```
   使用市场价值（而非账面价值）：

   权益市值 (E) = 股价 × 发行在外股份数
   债务市值 (D) = 总债务 (若债券未上市交易，则使用账面价值)
   总价值 (V) = E + D

   权益权重 = E / V
   债务权重 = D / V

   示例：
   E = $5,000M (90.9%)
   D = $500M (9.1%)
   V = $5,500M (100%)
   ```

**5. 计算 WACC**
   ```
   WACC = (E/V × 权益成本) + (D/V × 债务成本 × (1 - 税率))

   示例：
   WACC = (90.9% × 11.35%) + (9.1% × 6.5% × (1 - 25%))
   WACC = 10.32% + 0.44% = 10.76%

   基准情景取整为：10.8%
   ```

#### B. 计算终值 (Terminal Value)

**方法 1：永续增长法（首选）**
```
终值 = FCF(2029) × (1 + g) / (WACC - g)

其中：
- FCF(2029) = 模型中最后一年的无杠杆 FCF
- g = 永续增长率 (通常为 2.0-3.0%)
  - 不应超过长期 GDP 增长率
  - 基准情景通常使用 2.5%

示例：
FCF(2029) = $500M
g = 2.5%
WACC = 10.8%

终值 = $500M × (1.025) / (0.108 - 0.025)
终值 = $512.5M / 0.083 = $6,175M
```

**方法 2：退出倍数法（备选）**
```
终值 = EBITDA(2029) × 退出倍数

其中：
- 退出倍数 = 当前同行交易倍数中位数 (例如 12-15x EBITDA)

示例：
EBITDA(2029) = $800M
退出倍数 = 13x

终值 = $800M × 13x = $10,400M
```

**选择其中一种方法，或取两者的平均值。**

#### C. 将现金流折现为现值 (PV)

```
预测期 FCF 现值 = Σ [FCFt / (1 + WACC)^t] (t = 1 到 5)

示例：
年份    FCF      折现因子    FCF 现值
        ($M)     (Factor)    ($M)
2025    $250     1/(1.108)^1 = 0.9026    $226
2026    $320     1/(1.108)^2 = 0.8147    $261
2027    $390     1/(1.108)^3 = 0.7353    $287
2028    $450     1/(1.108)^4 = 0.6636    $299
2029    $500     1/(1.108)^5 = 0.5988    $299
                              总现值： $1,372M

终值现值 = 终值 / (1 + WACC)^5
终值现值 = $6,175M / (1.108)^5 = $6,175M × 0.5988 = $3,697M

企业价值 (Enterprise Value) = $1,372M + $3,697M = $5,069M
```

#### D. 计算权益价值和每股股价

```
企业价值 (Enterprise Value)       $5,069M
- 净债务 (债务 - 现金)             ($450M)
+ 非经营性资产                    $0M
- 少数股东权益                    $0M
- 优先股                          $0M
= 权益价值 (Equity Value)         $4,619M

稀释后发行在外股份数               100M

每股股价 = $4,619M / 100M = $46.19

当前股价：$42.00
隐含上涨空间：10.0%
```

### 步骤 3：可比公司分析 (Comps)

#### A. 筛选可比公司

**筛选标准：**
- 同一行业/领域（首要要求）
- 类似的业务模式
- 可比的规模（市值、营收）
- 类似的增长概况
- 类似的地域分布

**识别 5-10 家同行公司：**
1. [同行 1] - 直接竞对
2. [同行 2] - 直接竞对
3. [同行 3] - 相关领域玩家
4. [同行 4] - 类似业务模式
5. [同行 5] - 区域性竞争对手
6. [再添加 3-5 家]

**说明每家入选同行的理由。**

#### B. 搜集同行财务数据

**针对每家可比公司，搜集：**
- 当前股价
- 发行在外股份数（稀释后）
- 总市值
- 总债务与现金（用于计算企业价值 EV）
- 企业价值 (EV)
- LTM（过去十二个月）财务指标：
  - 营收
  - EBITDA
  - EBIT
  - 净利润
- NTM（未来十二个月）市场共识预期
- 营收增长率
- EBITDA 利润率

**数据来源：**
- FactSet, CapitalIQ, Bloomberg（首选）
- 公司 10-K/10-Q 报告（实际值）
- Yahoo Finance, Seeking Alpha 的市场共识预期（若无专业工具）

#### C. 计算估值倍数

**针对每家同行，计算：**
```
EV/Revenue (LTM) = 企业价值 / LTM 营收
EV/Revenue (NTM) = 企业价值 / NTM 预期营收
EV/EBITDA (LTM) = 企业价值 / LTM EBITDA
EV/EBITDA (NTM) = 企业价值 / NTM 预期 EBITDA
P/E (NTM) = 市值 / NTM 预期净利润
```

#### E. 将倍数应用于目标公司

**选择主要倍数（对于成熟公司通常使用 EV/EBITDA）：**

```
目标公司 NTM 预期 EBITDA = $550M（来自财务模型）

应用同行中位数倍数：
同行中位数 EV/EBITDA (NTM) = 13.8x
隐含企业价值 (EV) = $550M × 13.8x = $7,590M

应用 25 分位数（保守）：
25 分位数 EV/EBITDA (NTM) = 12.5x
隐含企业价值 (EV) = $550M × 12.5x = $6,875M

应用 75 分位数（乐观）：
75 分位数 EV/EBITDA (NTM) = 14.1x
隐含企业价值 (EV) = $550M × 14.1x = $7,755M

估值区间 (Comps)：$6,875M - $7,755M
中价：$7,315M

转换为权益价值 (Equity Value)：
隐含企业价值 (中位数)      $7,590M
- 净债务                 ($450M)
= 隐含权益价值           $7,140M

发行在外股份数           100M
隐含每股股价             $71.40
```

**溢价/折价理由：**
- 目标公司增长率 17% vs. 同行中位数 17% → 持平
- 目标公司 EBITDA 利润率 23% vs. 同行中位数 23% → 持平
- 目标公司市场地位 → [解释溢价/折价理由]
- **结论**：应用中位数倍数（无需调整）

#### B. 确定权重逻辑

**典型权重分配：**
- DCF：40-60%（当预测数据可靠时权重较高）
- 公开交易倍数：25-40%（反映当前市场情绪）
- 历史先例交易：10-25%（除非极有可能发生并购，否则权重较低）

**本示例：**
- DCF 50%：对预测数据信心较高
- Comps 40%：同行数据集完整且稳健
- Precedent 10%：近期发生并购的可能性较低

## 合理性检查 (Sanity Checks)

**务必执行以下验证检查：**

1. **历史倍数检查**
   - 隐含倍数是否符合该公司的历史交易区间？
   - 若不符合，请解释原因。

2. **同行对比**
   - 相对于同行的溢价/折价是否由基本面支撑？
   - 检查：增长率、利润率、市场地位。

3. **隐含增长检查**
   - 在当前股价下，市场定价中隐含的增长预期是多少？
   - 基于公司轨迹，该预期是否合理？

4. **市值合理性**
   - 考虑到公司规模和同行情况，总市值是否逻辑通顺？
   - 相对于整个行业，该公司是否会显得过大或过小？

5. **终值检查**
   - 终值是否 < 企业价值总额的 60-70%？
   - 若 > 70%，说明预测期可能不够长。

6. **WACC 合理性**
   - WACC 是否在典型公司的 8-14% 区间内？
   - 科技/高增长类：10-14%
   - 成熟/稳定类：7-10%

7. **隐含收益检查**
   - 按照当前价格到 12 个月后的目标价，内部收益率 (IRR) 是多少？
   - 这与投资建议评级是否一致？

> **💡 Appendix: 领域知识小贴士**
>
> 1. **现金流折现 (DCF) 模型**：这是估值的“黄金标准”。它的原理很简单：如果你知道一家公司未来每年能赚多少钱，把这些未来的钱按照一定的“折扣”换算成现在的价值，加在一起就是这家公司的身价。这就像是在算：为了在未来 5 年每年拿 100 块，你现在愿意掏多少钱买这张彩票。
>
> 2. **WACC (加权平均资本成本)**：这是理解“折扣”的关键。公司筹钱通常有两个途径：找银行借（债权）和找股东投（股权）。借钱要还利息，股东投钱想要分红，这两者的平均成本就是 WACC。在 DCF 里，WACC 越高，未来的钱就越不值钱（折现得越狠）。
>
> 3. **可比公司法 (Comps)**：这就像买房子参考邻居的价格。如果这一地段（行业）类似的房子（公司）都是按面积（营收或利润）的 10 倍卖的，那你的房子大概也能卖到这个倍数。
>
> 4. **足球场图 (Football Field Chart)**：别被名字吓到，它不是在聊球，而是因为这种图长得像橄榄球场。它把 DCF、Comps 等不同方法算出来的估值区间画在一起。如果所有方法的条状区间都在某一处重合，那这个价格就比较靠谱。
>
> 5. **EBITDA**：这是一个非常常用的利润指标。它排除了利息、税收、折旧和摊销这些“杂音”，能更纯粹地反映公司主营业务的赚钱能力。金融圈喜欢用它来比较不同负债水平和不同地税政策的公司。
