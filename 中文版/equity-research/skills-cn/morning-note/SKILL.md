# 晨间简报

description: 撰写简明扼要的晨会纪要，汇总隔夜市场动态、交易策略及覆盖股票的关键事件。专为早间 7 点晨会量身打造——内容精炼、观点明确、极具行动导向。触发词：“晨报”、“晨会”、“隔夜市场动态”、“交易策略”、“晨会准备”或“每日简报”。

## 工作流程

### 步骤 1：隔夜动态扫描

针对所覆盖的研究领域，扫描相关关键事件：

**业绩表现与指引 (Earnings & Guidance)**
- 隔夜或盘前是否有覆盖公司发布财报？
- 业绩超预期/不及预期情况（营收、每股收益 EPS、关键业务指标）
- 业绩指引变动（上调、下调、维持）

**新闻与重大事件 (News & Events)**
- 并购 (M&A) 公告或传闻
- 管理层变动
- 产品发布或监管准入决定
- 竞争对手分析师的评级上调/下调
- 影响该行业的宏观数据或政策变动

**市场背景 (Market Context)**
- 隔夜期货 / 盘前走势
- 行业指数 ETF 表现
- 相关大宗商品或汇率波动
- 今日待发布的关键经济数据

### 步骤 2：晨间简报格式

保持内容精炼——一份晨间简报应能在 2 分钟内阅读完毕：

---

**[日期] 晨间简报 — [分析师姓名]**
**[覆盖行业]**

**核心观点：[标题 — 基金经理最需要听到的关键信息]**
- 用 2-3 句话阐述关键进展及其重要性
- 股票影响：目标价变动、维持/调整评级

**隔夜/盘前动态**
- [公司 A]：业绩/新闻的一句话总结 + 我们的评论
- [公司 B]：一句话总结 + 我们的评论
- [行业/宏观]：相关的全行业重大进展

**今日关键事件**
- [时间]：[公司] 业绩电话会
- [时间]：经济数据发布（市场预期 vs. 我们的观点）
- [时间]：行业会议或投资者日

**交易策略**（如有）
- [多头/空头] [公司]：1-2 句投资逻辑 + 催化剂
- 风险点：什么情况下该逻辑会失效

---

### 步骤 3：业绩快评

若覆盖公司发布了财报，请提供快速反馈：

| 指标 | 市场一致预期 | 实际值 | 超预期/不及预期 |
|--------|-----------|--------|-----------|
| 营收 (Revenue) | | | |
| 每股收益 (EPS) | | | |
| [关键指标] | | | |
| 业绩指引 | | | |

**我们的观点**：用 2-3 句话评价——这对股价有利还是不利？是否改变了我们的核心投资逻辑？

**操作建议**：维持 / 上调 / 下调评级？是否调整目标价？

### 步骤 4：输出形式

- 用于邮件/Slack 分发的 Markdown 文本
- 若需正式分发，则提供 Word 文档
- 篇幅严格控制在 1 页以内 — 投资组合经理和交易员不会看更长的内容

## 重要注意事项

- 观点要鲜明：只汇总新闻而不发表见解的晨报是毫无价值的
- 结论先行：最重要的信息放在最前面，不要埋没标题
- “无事发生”也是有效的汇报：可以写“隔夜无实质性变动，维持现有头寸”
- 区分具有可操作性的事件（业绩、并购）与市场噪音（琐碎的研报、无影响事件）
- 注明观点的时间戳：如果你在清晨 6 点撰写，请注明盘前走势在开盘时可能发生变化
- 如果判断错误，在下一份晨报中大方承认：公信力比保持全胜记录更重要

> **💡 Appendix: 领域知识小贴士**
> 1. **晨间简报 (Morning Note)**：金融圈的“早间情报站”。在股市开盘前，分析师会将隔夜全世界发生的各种大事、各家公司的财报，浓缩成几百个字，告诉老板今天该买什么，就像一份精准的“作战地图”。
> 2. **业绩指引 (Guidance)**：这就是公司的“期中/期末预告”。公司不光会说明过去赚了多少钱，还会预告未来几个月想赚多少。在股市里，未来往往比过去更值钱，所以指引的变化经常会导致股价大涨大跌。
> 3. **市场一致预期 (Consensus)**：这是华尔街各路大神的“平均分”。如果大家都觉得某公司能考 90 分，哪怕它考了 80（其实也不错），但在资本市场眼里它就是“不及预期”，股价可能会跌。
> 4. **催化剂 (Catalyst)**：引爆股价的“导火索”。比如一项新政策、一个超预期的业绩、或者一个收购传闻。没有催化剂的股票就像没有风的帆船，涨跌都很难。
> 5. **多头/空头 (Long/Short)**：多头就是看好并买入，期待价格上涨；空头就是看衰并借入再卖出，期待价格下跌。这是金融交易里的两个最基本姿势。
