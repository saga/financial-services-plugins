# 权益研究业绩更新报告

创建专业的**业绩更新报告 (EARNINGS UPDATE REPORTS)**，分析已覆盖公司的季度业绩，遵循国际投行标准（如摩根大通、高盛、摩根士丹利格式）。

**核心特征：**
- **长度**：8-12页
- **字数**：3,000-5,000字
- **表格**：1-3个摘要表格（非全面性）
- **图表**：8-12个图表
- **周转时间**：1-2天（财报发布后24-48小时内）
- **受众**：已熟悉该公司的客户
- **侧重点**：新进展——超预期/不及预期情况、预测更新、对投资逻辑的影响
- **字体**：全文使用 Times New Roman（除非用户另有说明）

## 适用场景

当用户请求以下内容时使用：
- “为 [公司] 2024年第三季度创建业绩更新”
- “分析 [公司] 的季度业绩”
- “为 [公司] 撰写财报后报告”
- “[公司] Q1/Q2/Q3/Q4 更新”

**以下情况请勿使用：**
- 用户请求“首次覆盖报告 (initiation report)” → 使用其他技能
- 用户请求“简评 (flash note)”或“快速观点 (quick take)” → 使用不同格式
- 公司尚未被覆盖 → 需先进行首次覆盖

## 关键要求

### 1. 速度与及时性
- 在财报发布后 24-48 小时内发布
- 仅关注**新**信息
- 不要冗长地重复公司背景

### 2. 超预期/不及预期分析 (Beat/Miss Analysis)
- 首要说明公司业绩是超出还是低于市场预期
- 量化差异（例如：“营收超出预期 1.2 亿美元，即 3%”）
- 解释结果与预期产生差异的**原因**

### 3. 摘要格式
- 表格控制在 1-3 个（仅限摘要，非全面性）
- 无需提供完整的损益表/现金流量表/资产负债表（仅列出关键指标）
- 假设读者已经看过首次覆盖报告

### 4. 引用与来源归属 ⭐⭐⭐ 强制性

**至关重要**：使用**具体来源**和**可点击超链接**正确引用所有数据。

**在每个图表和表格中包含具体的引用及可点击链接：**

```
来源：2024年11月8日提交的 2024年三季度 10-Q 报表；公司新闻稿
     [将“10-Q”链接至：https://www.sec.gov/cgi-bin/viewer?accession=...]
     [将“新闻稿”链接至：https://investor.company.com/news/q3-2024]
```

**超链接在 Word 中的呈现方式：**
- 文件名称显示为蓝色、带下划线的可点击链接
- 读者可以 Ctrl+Click 直接打开来源
- 非纯文本 URL，而是带显示文本的格式化超链接

**必须提供的来源清单：**

在每份业绩更新中引用：
- ✅ 财报新闻稿（带日期和 URL）
- ✅ 10-Q 申报文件（带申报日期和 EDGAR 链接）
- ✅ 业绩电话会议纪要（带日期）
- ✅ 投资者简报/补充材料（如有）
- ✅ 市场一致预期来源（Bloomberg/FactSet 等，带日期）
- ✅ 先前指引（源自上季度材料）
