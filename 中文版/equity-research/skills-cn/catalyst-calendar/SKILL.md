# 催化剂日历 (Catalyst Calendar)

description: 构建并维护覆盖股票池的催化剂日历——包括盈余日期、行业会议、产品发布、监管决策及宏观事项。协助确定优先级并提前进行头寸布局。触发词：“催化剂日历”、“近期事项”、“接下来有什么”、“财报日历”、“活动日历”或“催化剂跟踪”。

## 工作流程

### 第一步：定义覆盖范围

- 需跟踪的公司列表（股票代码或名称）
- 板块 / 行业聚焦
- 是否包含宏观事件？（美联储会议、经济数据、监管截止日期）
- 时间跨度（未来 2 周、一个月、一个季度）

### 第二步：收集催化剂

针对每家公司，识别即将发生的关键事项：

**业绩与财务事项**
- 季度盈余发布日期与时间（盘前/盘后）
- 年度股东大会
- 投资者日 / 分析师日
- 资本市场日
- 债务到期 / 再融资日期

**公司特定事项**
- 产品发布或公告
- FDA 审批 / 监管决策
- 合同续约或到期
- 并购（M&A）里程碑（交易完成日、监管审批）
- 管理层变动
- 内部人交易窗口（禁售期届满）

**行业事项**
- 重要行业会议（日期、参会公司名单）
- 贸易展会与博览会
- 监管征求意见期或裁决
- 行业数据发布（月度销售、流量数据等）

**宏观事项**
- 美联储会议（FOMC 日期）
- 非农就业报告、CPI、GDP 发布
- 央行决议（欧洲央行 ECB、日本央行 BOJ 等）
- 具有市场影响力的地缘政治事件

### 第三步：日历视图

| 日期 | 事项 | 公司/板块 | 类型 | 影响力 (高/中/低) | 仓位布局 | 备注 |
|------|-------|---------------|------|-----------------|----------------|-------|
| | | | 业绩/公司/行业/宏观 | | 多头/空头/中性 | |

### 第四步：每周前瞻

每周生成一份前瞻性总结：

**本周关键事项：**
1. [日期]：[公司] 第 [X] 季度财报 —— 市场共识 [每股收益 $X]，我们预期 [$X]，核心关注指标：[财务指标]
2. [日期]：[事项] —— 对 [相关股票] 的战略意义
3. [日期]：[宏观数据发布] —— 市场预期与仓位应对建议

**下周预告：**
- 提前关注下周的重要事项

**对持仓的影响：**
- 可能推动特定头寸变动的事件
- 建议的预布局操作
- 针对二元事件（Binary Events）的风险管理方案

### 第五步：输出

- 带有日历视图和可排序列表的 Excel 工作簿
- 每周前瞻邮件/简报（Markdown 格式）
- 可选：与 Google 日历同步

## 重要提示

- 财报日期可能发生变动 —— 临近日期时请根据公司投资者关系（IR）页面及 Bloomberg/FactSet 进行核实
- 预披露风险：重点跟踪有“提前发布业绩指引”（正面或负面）历史的公司
- 会议参会名单深具价值 —— 关注哪些公司出席了，哪些本该出现却显眼地缺席了？
- 部分催化剂具有周期性（如月度行业数据） —— 建议建立模板并自动填充
- 使用颜色标注意义明确：红色 = 高影响力，黄色 = 中等影响力，绿色 = 常规事项
- 归档历史催化剂及其实际结果 —— 这有助于随着时间的推移建立模式识别能力

>
> 1. **什么是“催化剂” (Catalyst)？**
>    在金融界，这可不是化学试剂。它是指那些能让股价在短时间内产生剧烈波动的事项。比如药企能不能拿到新药批文？科技公司能不能发布重磅产品？如果你能提前预测并布局，就像是在发令枪响前就预判了起
