# 投资假设追踪器 (Thesis Tracker)

description: 维护并更新投资组合持仓及观察名单的投资假设。长期跟踪关键数据点、催化剂和投资逻辑里程碑。适用于根据新信息更新投资假设、回顾持仓理由或核查投资逻辑是否依然成立。触发词包括：“更新 [公司名] 的投资假设”、“我的投资逻辑还成立吗”、“投资假设核查”、“为 [公司名] 添加数据点”或“审视我的持仓”。

## 工作流

### 第一步：定义或加载投资假设

如果创建新的投资假设：
- **公司**：名称及股票代码
- **头寸**：做多 (Long) 或 做空 (Short)
- **投资假设陈述**：1-2 句核心逻辑（例如：“做多 ACME —— 定价权带来的利润率扩张 + 随着业务向软件转型产生的经营杠杆”）
- **核心支柱**：3-5 个支撑论点
- **关键风险**：3-5 个可能导致投资假设失效的风险
- **催化剂**：即将发生的可能证实或证伪假设的事件（财报发布、产品上线、监管决策）
- **目标价 / 估值**：如果逻辑兑现，该资产的价值是多少
- **止损触发点**：什么情况会导致你离场

如果是更新现有假设，请询问用户最新的数据点或进展。

### 第二步：更新日志

针对每一个新的数据点或进展：

- **日期**：事件发生时间
- **数据点**：发生了什么变化（业绩超预期、管理层变动、竞对动作等）
- **假设影响**：这能加强、削弱还是中性化某个特定支柱？
- **操作建议**：维持现状 / 增持 / 减持 / 清仓
- **更新后的确定性**：高 / 中 / 低

### 第三步：投资假设评分卡

维护一个持续更新的评分卡：

| 逻辑支柱 | 初始预期 | 当前状态 | 趋势 |
|----------|----------|----------|------|
| 营收增长 >20% | 符合预期 | 第三季度为 22% | 稳定 |
| 利润率扩张 | 进度落后 | 利润率同比持平 | 令人担忧 |
| 新产品发布 | 待定 | 推迟至第二季度 | 观察中 |

### 第四步：催化剂日历

追踪即将发生的催化剂事件：

| 日期 | 事件 | 预期影响 | 备注 |
|------|------|----------|------|
|      |      |          |      |

### 第五步：输出

生成适用于以下场景的投资假设摘要：
- 晨会讨论
- 投资组合回顾
- 风险委员会汇报

格式：简洁的 Markdown 或 Word 文档，包含评分卡、近期更新和当前的投资信心水平。

## 重要说明

- 投资假设必须具备“可证伪性”——如果没有任何证据能推翻它，那它就不是一个合格的假设
- 追踪“证伪证据”的严谨程度应与追踪“证实证据”一致
- 至少每季度回顾一次投资逻辑，即使没有发生剧烈波动
- 如果用户管理多个头寸，建议进行全组合的投资假设审查
- 以结构化格式存储投资假设数据，以便跨会话引用

> **💡 Appendix: 领域知识小贴士**
> 
> **什么是“投资假设” (Investment Thesis)？**
> 简单来说，就是你买入或卖出一只股票的“理由”。它不只是“我觉得它会涨”，而是基于事实和逻辑的推演。类似警察破案时的“推论”，你需要列出证据来支撑它（核心支柱），并实时关注新线索。
> 
> **这两个杠杆是什么意思？**
> 1. **定价权 (Pricing Power)**：公司涨价了客户还愿意买。这是利润翻倍的利器。
> 2. **经营杠杆 (Operating Leverage)**：公司规模变大时，成本增长比收入增长慢。就像你开餐厅，多卖 100 碗面不需要再雇一个厨师，多出来的钱全是利润。
> 
> **为什么要关注“催化剂” (Catalysts)？**
> 催化剂是点燃股价的“火苗”。没有催化剂，好的投资逻辑可能要等很久才会被市场发现。比如一次亮眼的财报、一个重磅新手机的发布，都是让大家意识到公司价值的转折点。
> 
> **为什么强调“可证伪性”？**
> 如果你认为“只要这公司在，股价就会涨”，这就是不可证伪的，因为它更像信仰而非投资。一个好的投资者会问：“如果发生什么，我就承认我错了？” 这种“认错标准”就是止损触发点，它能保护你不在错误的方向上越陷越深。
