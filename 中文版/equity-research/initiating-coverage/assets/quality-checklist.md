# 初始覆盖报告质量控制清单

在交付初始报告之前，验证以下所有项目已完成。

## 关键最低要求 - 报告必须满足这些

**关键 - 如有以下情况请勿交付：**
- ❌ DOCX报告少于30页 → 不完整
- ❌ 少于25张嵌入式图表 → 不完整
- ❌ 少于12张综合表格 → 不完整
- ❌ 少于10,000字 → 不完整
- ❌ 没有XLS财务模型 → 缺失可交付成果
- ❌ 图表是文字描述而非实际PNG/JPG文件 → 重大失败

## 可交付成果清单

- [ ] DOCX报告文件已创建
- [ ] XLS财务模型文件已创建
- [ ] 两个文件命名正确：`[公司]_[日期]_初始报告.docx` 和 `[公司]_[日期]_财务模型.xlsx`

## DOCX报告 - 长度和内容

**长度验证：**
- [ ] 报告为30-50页（最终文档中的页数）
- [ ] 字数为10,000-15,000字
- [ ] 如果少于30页：停止并添加更多内容

**视觉元素：**
- [ ] 嵌入了25-35张图表（计数：_____张图表）
- [ ] 所有图表都是实际的PNG/JPG图片文件（不是文字描述）
- [ ] 包含12-20张综合表格（计数：_____张表格）
- [ ] 图表和表格穿插全文，而非集中在末尾

**图表要求：**
- [ ] 按产品分类收入图表：堆叠面积图格式 ✓
- [ ] 按地区分类收入图表：堆叠柱状图格式 ✓
- [ ] DCF敏感性分析：带颜色编码的双向热图 ✓
- [ ] 估值足球场图：水平条形图 ✓
- [ ] 所有其他图表都是实际图片文件 ✓

**表格要求：**
- [ ] 完整利润表（40-50行），含5年历史+5年预测
- [ ] 完整现金流量表（30-40行）
- [ ] 完整资产负债表（35-45行）
- [ ] 按产品分类收入表（20-30行）
- [ ] 按地区分类收入表（15-20行）
- [ ] 按渠道分类收入表（10-15行）
- [ ] 可比公司表，含统计摘要（最大/75分位/中位数/25分位/最小）
- [ ] DCF计算表（30-40行）
- [ ] WACC计算表（8-10行）
- [ ] 两个敏感性分析表
- [ ] 2-3张其他财务/竞争表格

## DOCX报告 - 结构

**第1页要求：**
- [ ] 包含"INITIATING COVERAGE"标题（不是"公司更新"）
- [ ] 以主题为中心的标题（不是事件驱动的如"强劲的第四季度业绩"）
- [ ] 包含评级框：评级、价格、目标价格、52周区间、市值、企业价值
- [ ] 3-4段要点，使用■字符和粗体标题
- [ ] 财务和估值指标表，含2-3年历史和2年预测
- [ ] 表格使用"A"后缀表示实际，"E"后缀表示估计
- [ ] 所有图表都有来源行

**内容章节：**
- [ ] 目录（第2页）
- [ ] 投资论点与风险（3-5页）
- [ ] 公司概况（6-12页）包括：
  - [ ] 公司描述
  - [ ] 历史和里程碑
  - [ ] 管理层简历（3-4名高管每人300-400字）
  - [ ] 产品/服务详情
  - [ ] 竞争格局
- [ ] 财务分析与预测（10-15页）
- [ ] 估值分析（8-12页）
- [ ] 假设部分（2,000-3,000字记录所有预测假设）
- [ ] 情景分析（1,500-2,000字，含牛市/基准/熊市参数）
- [ ] 附录，包括数据来源和参考页面

## DOCX报告 - 格式

**图表和表格格式：**
- [ ] 每个图表上方有标题："图X - [公司] [描述性标题]"
- [ ] 每个图表下方有来源行："来源：[具体来源和日期]"
- [ ] 顺序图表编号（图1、图2、图3...无间断）
- [ ] 每个表格有带阴影的标题行
- [ ] 每个表格底部有来源行
- [ ] 所有年份使用"A"表示实际，"E"表示估计

**专业格式：**
- [ ] 全文使用一致字体（Calibri、Arial或类似）
- [ ] 页眉和页脚含页码
- [ ] 密集布局：60-80%页面覆盖率，最小留白
- [ ] 每一页同时有文字和视觉元素（图表或表格）
- [ ] 使用专业商业报告模板

## 引用和来源 ⭐⭐⭐ 关键

**来源归属：**
- [ ] 每个图表都有具体来源（文档名称和日期）
- [ ] 每个表格都有具体来源参考
- [ ] 正文中的关键统计数据有脚注来源
- [ ] 不仅限于"公司数据" - 必须具体

**超链接：** ⭐⭐⭐ 必需
- [ ] 所有URL都是可点击的超链接（不是纯文本）
- [ ] SEC文件链接到EDGAR查看器
- [ ] 财报电话会议记录链接（Seeking Alpha或公司投资者关系）
- [ ] 新闻稿链接到公司投资者关系页面
- [ ] 演示文稿链接到PDF URL
- [ ] 行业报告链接（如有公开）
- [ ] 订阅数据（Bloomberg、FactSet）注明"（需订阅）"
- [ ] 任何地方都没有原始URL显示 - 全部格式化为超链接
- [ ] 测试3-5个样本超链接以确保它们有效（Ctrl+点击）

**参考页面：**
- [ ] 报告末尾有"数据来源和参考"页面
- [ ] 列出报告中使用的所有来源
- [ ] 来源按类别组织（SEC文件、财报电话会议等）
- [ ] 每个来源都有日期
- [ ] 每个来源都有可点击超链接（如适用）

## XLS财务模型 - 结构

**文件结构：**
- [ ] Excel工作簿有15个以上标签页
- [ ] 标签页包括：执行摘要、假设、历史财务、收入模型、运营费用、利润表、资产负债表、现金流量、支持性明细表、DCF估值、可比公司分析、优先交易、情景、敏感性分析、图表

**格式：**
- [ ] 蓝色文本表示硬编码输入
- [ ] 黑色文本表示公式
- [ ] 绿色文本表示与其他工作表的链接
- [ ] 专业的边框和阴影格式
- [ ] 清晰的部分标题和标签

**模型功能：**
- [ ] 所有数字都能流动（更改假设 → 整个模型更新）
- [ ] DCF链接到假设和预测
- [ ] 无循环引用或错误
- [ ] 所有重要单元格/范围都已命名
- [ ] 敏感性分析表动态工作

## XLS财务模型 - 内容

**预测：**
- [ ] 3-5年历史数据
- [ ] 5年未来预测（当年+1至当年+5）
- [ ] 按产品、地区、渠道分类的收入
- [ ] 完整的损益表（40-50行）
- [ ] 完整的现金流量表（30-40行）
- [ ] 完整的资产负债表（35-45行）

**估值：**
- [ ] 完整的DCF模型，显示所有计算
- [ ] 所有组成部分的WACC计算
- [ ] 终值计算
- [ ] 可比公司分析（5-10家公司）
- [ ] 优先交易分析（5-10笔交易）
- [ ] 情景分析（牛市/基准/熊市）
- [ ] 两个敏感性分析表

## 跨文件一致性

**关键**：DOCX和XLS之间的数字必须完全匹配

- [ ] 两个文件之间的收入数字匹配
- [ ] 两个文件之间的EPS数字匹配
- [ ] 两个文件之间的利润率百分比匹配
- [ ] 两个文件之间的估值数字匹配
- [ ] 两个文件之间的目标价格匹配
- [ ] 两个文件之间的所有预测年份匹配

**验证方法**：在DOCX报告和XLS模型之间抽查10-15个关键数字。

## 内容质量

**投资论点：**
- [ ] 3-5个清晰的论点支柱
- [ ] 每个支柱都有具体数据和量化支持
- [ ] 每个支柱的财务影响都已量化
- [ ] 已识别催化剂并有明确时间线

**分析深度：**
- [ ] 全面的商业模式分析
- [ ] 详细的竞争评估
- [ ] 分析了3-5年的财务趋势
- [ ] 已识别并量化了8-12个风险
- [ ] 分析了管理团队（每名高管300-400字）

**假设：**
- [ ] 2,000-3,000字记录所有假设
- [ ] 按类别/地区分类的收入增长假设
- [ ] 带驱动因素桥接的利润率假设
- [ ] 营运资本假设
- [ ] 资本支出假设
- [ ] 每个假设都有具体量化

**情景：**
- [ ] 1,500-2,000字的情景分析
- [ ] 牛市情况含具体参数和催化剂
