# 任务2：财务建模 - 详细工作流程

本文件提供了执行初始覆盖技能任务2（财务建模）的分步说明。

## 任务概述

**目的**：提取历史财务数据并构建综合Excel财务模型，包含预测和情景。

**前置条件**：⚠️ 开始前验证
- **必需**：公司财务数据访问权限
  - 上市公司：从SEC EDGAR获取最新10-K和近期10-Q
  - 私营公司：财务报表或可用来源的估计
  - 或者：用户提供预先提取的历史财务数据
- **可选**：公司研究（任务1）了解业务背景

**输出**：Excel财务模型（.xlsx），包含6个基本标签页：
1. 收入模型
2. 利润表
3. 现金流量表
4. 资产负债表
5. 情景
6. DCF输入

## 模型结构和格式

### 颜色编码（行业标准）
- **蓝色文本**：硬编码输入（用户可以更改）
- **黑色文本**：公式和计算
- **绿色文本**：与其他工作表的链接
- **红色文本**：错误或标记（应解决）

### 格式标准
- 专业的边框和阴影
- 清晰的部分标题
- 可折叠的分组行
- 关键输入/输出的命名范围
- 公式中没有硬编码数字（常数如12个月除外）
- 清晰的单位（千美元、百万美元等）

### 公式最佳实践
- 所有数字都应从假设流动
- 更改假设 → 整个模型更新
- 无循环引用
- 对关键单元格使用命名范围
- 保持公式简单和可审计
- 为复杂计算添加注释

## 分步建模工作流程

### 第1步：提取历史财务数据

**如果历史财务数据已提取，跳至第2步。**

**对于上市公司：**

1. **下载10-K文件**
   - 进入SEC EDGAR（https://www.sec.gov/edgar/searchedgar/companysearch.html）
   - 搜索公司名称或股票代码
   - 下载最新10-K（年度报告）
   - 导航至第8项：财务报表和补充数据

2. **创建历史财务Excel文件**
   - 文件名：`[公司]_[日期]_历史财务报表.xlsx`
   - 此文件将成为模型的基础

3. **提取损益表（3-5年）**
   - 创建工作表1："历史损益表"
   - 提取所有明细项目3-5年：
     - 收入（总额和细分，如有披露）
     - 营业成本/销货成本
     - 毛利润
     - 运营费用（研发、销售与市场营销、G&A细分）
     - EBITDA（如未披露则计算：EBIT +折旧摊销）
     - EBIT/营业收入
     - 利息费用/收入
     - 其他收入/费用
     - 税前收入
     - 所得税和税率
     - 净利润
     - EPS（基本和摊薄）
     - 流通股数（基本和摊薄）

4. **提取现金流量表（3-5年）**
   - 创建工作表2："历史现金流量"
   - 提取所有明细项目：
     - 运营活动（从净利润开始）
     - 折旧和摊销
     - 股票薪酬
     - 营运资本变动（应收账款、存货、应付账款）
     - 运营现金
     - 投资活动（资本支出、收购）
     - 融资活动（债务发行/偿还、股权、股息）
     - 现金净变动
     - 期初和期末现金

5. **提取资产负债表（3-5年）**
   - 创建工作表3："历史资产负债表"
   - 提取所有明细项目：
     - 流动资产（现金、应收账款、存货、其他）
     - 非流动资产（财产厂房设备、无形资产、商誉）
     - 总资产
     - 流动负债（应付账款、应计费用、流动债务）
     - 非流动负债（长期债务、递延税项）
     - 总负债
     - 股东权益（普通股、留存收益）
     - 负债和权益总额

6. **计算历史指标**
   - 创建工作表4："历史指标"
   - 从报表计算：
     - 收入增长率%（同比）
     - 毛利率%
     - EBITDA利润率%
     - 营业利润率%
     - 净利润率%
     - 自由现金流（CFO - 资本支出）
     - FCF利润率%
     - ROIC（近似：税后净营业利润/投入资本）
     - 债务/权益比率
     - 流动比率（流动资产/流动负债）

7. **记录来源和注释**
   - 创建工作表5："注释"
   - 记录：
     - 10-K文件日期和财年结束
     - 注意到的一次性项目或调整
     - 非GAAP与GAAP差异
     - 细分分解（如按产品/地区分类的收入）
     - 数据质量注释和限制

**对于私营公司：**

1. **收集可用数据**
   - 财务报表（如有）
   - 带收入数据的新闻稿
   - 融资公告
   - 行业估计或可比公司数据

2. **创建简化的历史文件**
   - 估计收入（如有）
   - 估计利润率（如需要可从可比公司）
   - 关键比率和指标
   - 记录所有假设和来源

**验证：**
- [ ] 已提取所有3份财务报表（3-5年）
- [ ] 数字在报表之间核对一致（净利润衔接）
- [ ] 关键指标计算正确
- [ ] Excel文件已保存且可打开
- [ ] 数据来源已记录（10-K日期、页码）

**预测模型的基础现已完成。继续第2步。**
