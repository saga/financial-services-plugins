## 折现现金流（DCF）分析

### 概述

DCF分析基于公司预测未来现金流的现值进行估值。这被认为是最具理论基础的估值方法，因为它基于基本价值创造。

### 分步DCF流程

#### 1. 历史财务分析
- 收集3-5年的历史财务报表
- 计算历史FCF = EBIT(1-税率) + 折旧摊销 - 资本支出 - 营运资本变动
- 分析历史增长率和利润率
- 识别趋势和周期性

#### 2. 构建收入预测（5-10年）
**方法：**
- **自上而下**：从市场规模（TAM）→市场份额→收入
- **自下而上**：销售数量×单价
- **混合**：结合多个驱动因素

**关键考虑因素：**
- 管理层指引和历史增长
- 行业增长率和市场趋势
- 竞争动态和市场份额演变
- 产品管道和新市场机会
- 宏观经济因素

#### 3. 预测运营费用
- **销货成本**：占收入的百分比（分析历史利润率）
- **销售及管理费用**：通常是半固定的；按收入百分比建模，考虑规模效应
- **研发**：对于科技/制药至关重要；按收入百分比建模
- **折旧摊销**：基于资本支出假设

**计算EBIT** = 收入 - 销货成本 - 运营费用

#### 4. 计算无杠杆自由现金流
```
EBIT
× (1 - 税率)
= 税后净营业利润（NOPAT）
+ 折旧和摊销
- 资本支出
- 营运资本增加
= 无杠杆自由现金流（UFCF）
```

**资本支出假设：**
- 维护资本支出：维持当前运营所需的（通常为收入的2-4%）
- 增长资本支出：扩张所需的
- 考虑行业基准和公司指引

**营运资本：**
- NWC =（应收账款 + 存货）- 应付账款
- 按收入百分比或天数（DSO、DIO、DPO）建模
- NWC增加是现金的使用

#### 5. 确定终值

**方法A：永续增长法**
```
终值 = FCF（终年）× (1 + g) / (WACC - g)
```
- g = 永续增长率（通常为2-3%，不超过GDP增长）
- 当公司已达到稳定、成熟增长时使用

**方法B：退出倍数法**
```
终值 = EBITDA（终年）× 退出倍数
```
- 退出倍数基于当前交易可比公司
- 更适用于周期性业务

#### 6. 计算加权平均资本成本（WACC）

```
WACC = (E/V × 股权成本) + (D/V × 债务成本 × (1 - 税率))
```

**股权成本（使用CAPM）：**
```
股权成本 = 无风险利率 + Beta × 股权风险溢价
```
- 无风险利率：10年期国债收益率
- Beta：股票收益与市场的回归（或使用可比Beta）
- 股权风险溢价：历史平均值约5-6%

**债务成本：**
```
债务成本 = 无风险利率 + 信用利差
```
- 使用公司当前借款利率或债券隐含收益率
- 如果没有未偿债券，根据信用评级调整

**资本结构：**
- E/V = 股权市场价值/总价值
- D/V = 债务市场价值/总价值
- 使用目标资本结构，而非当前（如果显著不同）

#### 7. 将现金流折现到现值

```
PV = Σ [FCFt / (1 + WACC)^t] + [终值 / (1 + WACC)^n]
```

#### 8. 计算企业价值和股权价值

```
企业价值 = 预测FCF的PV + 终值的PV
减：净债务（总债务 - 现金）
加：非运营资产
减：少数股东权益
减：优先股
= 股权价值

每股价格 = 股权价值 / 摊薄流通股数
```

### DCF敏感性分析

始终对关键变量进行敏感性分析：

1. **双因素敏感性表**：WACC vs 终值增长率
2. **收入增长情景**：基准/牛市/熊市情况
3. **利润率假设**：运营杠杆情景
4. **终值倍数敏感性**：如果使用退出倍数法

**敏感性表示例：**
```
           终值增长率
WACC      2.0%    2.5%    3.0%
8.0%      $45     $48     $52
9.0%      $40     $43     $46
10.0%     $36     $39     $41
```

### 应避免的常见DCF陷阱

1. **重复计算增长**：不要在没有相应投资（资本支出、NWC）的情况下预测高增长
2. **不切实际的终值增长**：不应超过长期GDP增长
3. **忽视周期性**：对周期性业务进行盈利正常化
4. **错误的现金流定义**：使用无杠杆FCF，而非净利润
5. **不一致的假设**：匹配折现率与现金流（无杠杆FCF → WACC）

## 交易可比公司分析

### 概述

交易可比公司基于类似公司在公开市场的估值方式进行估值。这反映了当前市场情绪和相对估值。

### 分步可比公司流程

#### 1. 选择可比公司

**选择标准：**
- 相同行业/板块（主要标准）
- 相似的商业模式和收入流
- 可比的规模（市值、收入）
- 相似的增长概况和利润率
- 相同的终端市场和地区

**典型范围：**
- 从8-15家公司开始
- 剔除情况独特的公司
- 最终选定5-10家公司

#### 2. 收集财务信息

**所需数据：**
- 当前股价和流通股数
- 最近财年财务报表
- 下一年度（NTM）共识估计
- 历史增长率

**计算市场指标：**
- 市值 = 股价 × 流通股数
- 企业价值 = 市值 + 债务 + 少数股东权益 + 优先股 - 现金
- 净债务 = 总债务 - 现金及等价物

#### 3. 计算估值倍数

**企业价值倍数：**
- **EV/收入**：适合早期/高增长公司
- **EV/EBITDA**：最常见；适合资本密集型业务
- **EV/EBIT**：当折旧摊销差异较大时有用

**股权价值倍数：**
- **P/E（市盈率）**：最广泛使用
- **P/B（市净率）**：适合金融机构
- **P/S（市销率）**：适合亏损公司

**计算：**
- 最近十二个月（LTM）- 历史
- 未来十二个月（NTM）- 前瞻（首选）

#### 4. 分析和选择倍数

**创建可比公司表：**

| 公司 | 市值 | EV/收入 | EV/EBITDA | EV/EBIT | P/E（NTM） | 收入增长 | EBITDA利润率 |
|---------|-----------|----------|------------|----------|-------------|----------|--------------|
| 可比公司A | $10B | 3.5x | 12.0x | 18.0x | 22.0x | 15% | 28% |
| 可比公司B | $8B | 3.0x | 10.5x | 16.0x | 19.0x | 12% | 27% |
| ... | ... | ... | ... | ... | ... | ... | ... |
| 中位数 | - | **3.2x** | **11.0x** | **17.0x** | **20.5x** | 13% | 27.5% |

**调整：**
- 剔除异常值（通常大于2个标准差）
- 考虑使用中位数而非平均值（受异常值影响较小）
- 如果某些可比公司更可比，则加权倍数
- 根据增长、利润率、风险差异进行调整

#### 5. 将倍数应用于目标公司

**示例计算：**
```
目标公司NTM EBITDA = $500M
选定EV/EBITDA倍数 = 11.0x
隐含企业价值 = $500M × 11.0x = $5,500M

减：净债务 = $1,000M
股权价值 = $4,500M

流通股数 = 100M
隐含每股价格 = $45.00
```

#### 6. 选择适当的倍数

**选择基于：**
- **EV/收入**：高增长、亏损公司（科技、获批前生物技术）
- **EV/EBITDA**：最常见；资本密集型行业（制造、电信）
- **P/E**：盈利稳定的企业（消费、零售）
- **行业特定**：银行用P/B、油气用EV/产量、媒体用EV/订阅者

### 溢价/折价分析

根据以下因素应用溢价或折价：
- **增长溢价**：更高增长 → 更高倍数
- **盈利能力**：更高利润率 → 更高倍数
- **规模**：较大公司通常有溢价（流动性）
- **市场地位**：市场领导者 → 溢价
- **地区**：发达市场 vs 新兴市场

## 优先交易分析

### 概述

优先交易基于M&A交易中类似公司的成交价格进行估值。这反映了控制权溢价和战略价值。

### 分步流程

#### 1. 识别相关交易

**选择标准：**
- 相同或类似行业
- 类似规模（在目标规模的0.5x至2x范围内）
- 相似的业务特征
- 近期交易（首选过去3-5年）
- 已公布和已完成的交易（避免撤回的交易）

**典型范围：**
- 最少5-10笔交易
- 更重视近期交易

#### 2. 收集交易详情

**所需信息：**
- 交易日期（公告和完成）
- 收购价格和结构（现金、股票、混合）
- 交易时目标公司的财务数据
- 战略理由和协同效应
- 支付的控制权溢价

**来源：**
- SEC文件（S-4、8-K、委托声明）
- 新闻稿和投资者演示
- M&A数据库（CapIQ、FactSet、Bloomberg）

#### 3. 计算交易倍数

**与交易可比公司相同的倍数，但基于交易价值：**

```
交易EV = 交易价格 + 债务 + 少数股东权益 + 优先股 - 现金及等价物
```
