# 📖 金融术语速查附录

> 为金融小白准备的简明术语表。本文档解释本插件集中出现的高频金融概念。

---

## 一、核心财务指标

### 营收与利润类

| 术语 | 英文全称 | 大白话解释 |
|------|----------|-----------|
| **Revenue** | 收入/营收 | 公司卖东西收到的钱。比如苹果卖iPhone收到的钱 |
| **Gross Profit（毛利润）** | Revenue − 直接成本 | 卖iPhone收入减去造iPhone的成本（芯片、屏幕、组装费） |
| **Gross Margin（毛利率）** | 毛利润 ÷ 收入 | 每赚1块钱，毛利润占多少。毛利率高说明产品"含金量"高 |
| **EBITDA** | 息税折旧摊销前利润 | 利润 + 利息 + 税 + 折旧 + 摊销。用来衡量公司**实际经营**赚了多少钱，把融资和会计处理的影响去掉 |
| **Net Income（净利润）** | 税后利润 | 公司最终到手的钱。EBITDA - 利息 - 税 - 折旧 = 净利润 |
| **Operating Income（营业利润）** | EBIT（息税前利润） | 公司主业赚的钱，还没扣利息和税 |
| **Free Cash Flow（FCF）** | 自由现金流 | 经营赚的钱 - 必需的投资。公司"真正能花的钱" |

### 估值类

| 术语 | 公式 | 大白话解释 |
|------|------|-----------|
| **P/E（市盈率）** | 股价 ÷ 每股收益 | 回本需要多少年。P/E = 20 意味着按当前利润，20年回本 |
| **EV/EBITDA** | 企业价值 ÷ EBITDA | 买下整个公司（含负债）要花多少钱，相当于利润的多少倍 |
| **EV/Revenue（市销率）** | 企业价值 ÷ 收入 | 每赚1块钱收入，市场愿意出多少钱买 |
| **Market Cap（市值）** | 股价 × 流通股数 | 公司在股市上的"标价" |
| **Enterprise Value（EV）** | 市值 + 负债 − 现金 | 收购一家公司实际要花多少钱（因为你要承担它的债务） |

### 增长率与期间

| 术语 | 解释 |
|------|------|
| **YoY（同比）** | 今年Q1 vs 去年Q1，去掉季节性影响 |
| **LTM（过去12个月）** | Last Twelve Months，最近4个季度的合计数据 |
| **NTM（未来12个月）** | Next Twelve Months，预测的未来4个季度 |
| **CAGR** | 复合年增长率，多年平均增长速度 |

## 二、债券市场基础

### 债券定价

- **Clean Price（净价）**：债券的"裸价"，不含应计利息
- **Dirty Price（全价）**：净价 + 应计利息 = 你实际要付的钱
- **YTM（到期收益率）**：如果你持有到期，每年能赚多少（年化）
- **Duration（久期）**：债券价格对利率变动的敏感度。久期=5 意味着利率涨1%，价格跌约5%
- **Convexity（凸性）**：修正久期误差的"补丁"，用来更精确预测价格变化
- **DV01**：利率变动1个基点（0.01%），债券价格变动多少钱

### 信用利差

- **G-spread**：债券收益率 − 同期限国债收益率。衡量这只债券比国债"多赚多少"
- **Z-spread**：更精确的利差，考虑了整条利率曲线
- **OAS（期权调整利差）**：含权债券的利差，把期权的影响剥离掉
- **Rich / Cheap**：Rich = 价格偏贵（收益率偏低），Cheap = 价格偏便宜（收益率偏高）

### 债券期货

- **Basis（基差）**：债券期货价格 − 最便宜可交割债券的价格。基差越小，期货越"便宜"
- **CTD（Cheapest to Deliver）**：最便宜可交割债券——期货空头会选最便宜的交割
- **Implied Repo Rate（隐含回购利率）**：从期货价格反推出来的"融资利率"，用来判断期货是否定价合理

## 三、利率与曲线

### 收益率曲线

- **Normal Curve（正常曲线）**：长期利率 > 短期利率（向上倾斜），经济正常
- **Flat Curve（平坦曲线）**：长短利率差不多，市场不确定
- **Inverted Curve（倒挂曲线）**：短期利率 > 长期利率（向下倾斜），通常预示衰退
- **Steepener（陡化交易）**：赌长端利率比短端涨得快（曲线变陡）
- **Flattener（平坦化交易）**：赌短端比长端涨得快（曲线变平）
- **Butterfly（蝶式交易）**：同时做多/做空三个期限，赌曲线的"驼峰"形状变化

### 利率互换

- **Swap Rate（互换利率）**：用固定利率换浮动利率的"价格"
- **Swap Spread（互换利差）**：互换利率 − 同期限国债收益率。反映了银行间融资成本
- **利率互换（IRS）**：A给B固定利率，B给A浮动利率。用来管理利率风险

### 通胀

- **Breakeven Inflation（盈亏平衡通胀率）**：普通国债收益率 − 通胀保护债券收益率。市场预期的未来通胀率
- **Real Rate（实际利率）**：名义利率 − 通胀率。真正反映"购买力增长"

## 四、外汇与利差交易

### 基础概念

- **Spot Rate（即期汇率）**：现在兑换的价格
- **Forward Points（远期点）**：未来某个时间兑换，比现在贵/便宜多少点
- **Forward Rate（远期汇率）**：即期 + 远期点 = 未来的实际汇率
- **Carry（利差）**：借低息货币买高息货币，赚利息差

### Carry Trade（利差交易）

- 借入低利率货币（如日元），买入高利率货币（如澳元），赚利息差
- **Carry-to-Vol Ratio（利差波动比）**：年化利差 ÷ 隐含波动率。越高越好，说明"赚得多、风险小"
- **Risk Reversal（风险反转）**：看涨/看跌期权的隐含波动率差。正值 = 市场看涨
- **Butterfly（蝶式）**：衡量极端行情（"黑天鹅"）的概率

## 五、期权与波动率

### Greeks（希腊字母）

| 希腊字母 | 含义 | 大白话 |
|----------|------|--------|
| **Delta** | 标的价格涨$1，期权价格涨多少 | 期权变"股票"的程度 |
| **Gamma** | Delta的变化速度 | Delta自己的加速度 |
| **Vega** | 波动率涨1%，期权价格涨多少 | 对"市场恐慌"的敏感度 |
| **Theta** | 每天期权贬值多少 | "时间就是金钱"——每过一天亏多少 |
| **Rho** | 利率涨1%，期权价格涨多少 | 对利率的敏感度（通常影响小） |

### 波动率

- **Implied Vol（隐含波动率，IV）**：从期权价格反推出来的"市场预期波动"。高IV = 市场觉得"会大动"
- **Realized Vol（已实现波动率，RV）**：过去实际涨跌了多少
- **Vol Premium（波动率溢价）**：IV − RV。溢价高说明期权贵
- **ATM（平值）**：行权价 ≈ 当前价格
- **Vol Surface（波动率曲面）**：不同行权价和到期日的隐含波动率三维图
- **Skew（偏斜）**：不同行权价的波动率差异。反映市场对涨/跌的恐惧程度

### 波动率曲面

- **Vol Surface Snapshot（波动率曲面快照）**：一个表格/图表，展示不同到期日（横轴）和不同行权价（纵轴）的隐含波动率。用于快速评估期权的贵贱和市场的预期走向。
- **Skew Direction（偏斜方向）**：通常股票期权有"负偏斜"（put贵、call便宜），因为大家更怕跌；外汇期权两边都可能。
- **Term Structure（期限结构）**：同一行权价（如ATM）在不同到期日的波动率走势——近低远高通常意味着中期不确定性更大。

## 六、固定收益组合

- **Key Rate Duration（关键利率久期）**：不同期限的利率变动对债券价格的影响。比久期更精细——告诉你2年期利率涨了怎么办、10年期涨了怎么办
- **Scenario Analysis（情景分析）**：假设利率涨/跌一定幅度，组合赚/亏多少
- **Cashflow Waterfall（现金流瀑布图）**：未来每个季度能收到多少利息和本金。可以看到什么时候"钱到账"最多
- **OAS（Option-Adjusted Spread）**：含权债券的真实利差，去掉期权价值干扰
- **Spread to Curve（曲线利差）**：债券收益率 − 同期限国债利率 = 信用风险溢价

## 七、并购与投行知识

| 术语 | 大白话解释 |
|------|-----------|
| **M&A（并购）** | A公司买B公司 |
| **CIM（保密信息备忘录）** | 卖公司时给买家看的详细资料，像"卖房说明书" |
| **Teaser（摘要）** | CIM的简化版——先吊胃口用的"一页纸摘要" |
| **Process Letter（流程函）** | 告诉买家"怎么出价、什么时间截止"的正式文件 |
| **Buyer List（买方名单）** | 列出"哪些公司可能会买"的清单 |
| **Accretion（增厚）** | 并购后新公司每股利润 > 收购前。好的并购 |
| **Dilution（稀释）** | 并购后每股利润 < 收购前。不太好的并购 |
| **Sources and Uses（资金来源与用途）** | 收购的钱从哪来（借款+自有资金），花到哪去（给股东+还债） |
| **Pro Forma（备考报表）** | 假设收购已经完成后的"模拟财务报表" |
| **LBO（杠杆收购）** | 借钱买公司，用被买公司的利润还债 |
| **DCF（现金流折现）** | 预测公司未来赚多少钱，折算成今天的价值。用来估值的"黄金标准" |
| **Comps（可比公司分析）** | 看看同类公司卖多少钱，用来给目标公司定价 |

## 八、基金行政管理

- **NAV（资产净值）**：基金的总资产 − 总负债。你持有基金份额对应的价值
- **Accrual（应计费用）**：已经发生但还没付的钱。如管理费每天都在累积
- **Break（差异分析）**：账面数和实际数的差异。找出"哪里对不上"
- **Roll Forward（滚动报表）**：上期余额 + 本期变化 = 本期余额。类似"期初 + 变动 = 期末"
- **Variance Commentary（差异说明）**：解释为什么预算和实际不一样

---

## 九、KYC与合规

- **KYC（了解你的客户）**：金融机构必须核实客户身份，防止洗钱
- **Sanctions Screening（制裁筛查）**：检查客户是否在制裁名单上
- **PEP（政治公众人物）**：担任重要公职的人及其亲属——他们洗钱风险更高
- **AML（反洗钱）**：防止通过金融系统洗钱的一整套制度

## 十、私募股权与风险投资

### 基金结构

| 术语 | 英文全称 | 大白话解释 |
|------|----------|-----------|
| **GP（普通合伙人）** | General Partner | 基金的管理者，负责投资决策和日常运营。承担无限责任，收取管理费和业绩报酬 |
| **LP（有限合伙人）** | Limited Partner | 基金的出资人，如养老金、大学捐赠基金、家族办公室。承担有限责任，不参与日常管理 |
| **Fund Size（基金规模）** | - | 基金募集的总资本额。如"10亿美元基金" |
| **Commitment（承诺资本）** | - | LP 承诺要出资的总额。通常分多年缴付 |
| **Capital Call（缴款通知）** | - | GP 需要投资时，通知 LP 按比例出资 |
| **Distribution（分配）** | - | 基金退出投资后，将收益返还给 LP |
| **Carried Interest（业绩报酬）** | - | GP 从基金超额收益中分成的部分，通常为 20% |
| **Management Fee（管理费）** | - | GP 每年收取的费用，通常为承诺资本的 1.5-2%，用于覆盖运营成本 |
| **Hurdle Rate（门槛收益率）** | Preferred Return | LP 先获得的最低回报率，通常为 8%。超过后才分配 carried interest |
| **Waterfall（瀑布分配）** | - | 收益分配的顺序和规则。通常：返还本金 → 门槛收益 → GP Catch-up → 80/20 分配 |
| **Catch-up（追赶条款）** | - | 超过门槛收益后，GP 快速获得部分 carried interest，直到达到约定比例 |
| **Clawback（回拨条款）** | - | 如果早期分配过多，GP 需要将多拿的钱退还给 LP |

### 投资回报指标

| 术语 | 英文全称 | 计算公式 | 大白话解释 |
|------|----------|---------|-----------|
| **IRR（内部收益率）** | Internal Rate of Return | 使 NPV=0 的折现率 | 考虑时间价值的年化回报率。IRR=20% 意味着每年赚 20% |
| **MOIC（投资倍数）** | Multiple on Invested Capital | (已实现 + 未实现价值) / 投资成本 | 简单直接的回报倍数。MOIC=3x 意味着赚了 3 倍 |
| **DPI（实付倍数）** | Distributed to Paid-In | 累计分配 / 累计缴款 | 已经拿到手的钱的倍数。DPI>1 意味着已收回本金 |
| **RVPI（未实现倍数）** | Residual Value to Paid-In | 未实现价值 / 累计缴款 | 还没变现的投资价值倍数 |
| **TVPI（总倍数）** | Total Value to Paid-In | DPI + RVPI | 总回报的倍数 |
| **J-Curve（J曲线）** | - | - | 基金早期 IRR 为负（因为交了管理费还没退出），后期转正并上升，形状像字母 J |

### 投资阶段

| 阶段 | 英文 | 投资对象 | 典型金额 | 风险/回报 |
|------|------|---------|---------|----------|
| **天使轮** | Angel | 概念/原型阶段 | $10万-$100万 | 极高风险，极高回报 |
| **种子轮** | Seed | 产品初步验证 | $100万-$300万 | 很高风险 |
| **A轮** | Series A | 产品市场匹配 | $300万-$1500万 | 高风险 |
| **B轮** | Series B | 规模化扩张 | $1500万-$5000万 | 中等风险 |
| **C轮+** | Series C+ | 市场主导/上市准备 | $5000万+ | 较低风险 |
| **成长期** | Growth | 已盈利，快速扩张 | $5000万-$2亿 | 中等风险 |
| **并购** | Buyout | 成熟公司控制权 | $1亿-$100亿+ | 较低风险，稳定回报 |
| **困境投资** | Distressed | 财务困难公司 | 视情况而定 | 高风险，高回报 |

### 尽职调查（Due Diligence）

**定义**：投资前对目标公司进行全面调查，评估投资价值和风险。

**调查维度**：

| 维度 | 内容 | 关键问题 |
|------|------|---------|
| **财务 DD** | 财务报表、税务、内控 | 收入是否真实？利润能否持续？ |
| **法律 DD** | 合同、诉讼、知识产权 | 有无重大法律风险？IP 是否清晰？ |
| **商业 DD** | 市场、竞争、客户 | 市场多大？护城河在哪？ |
| **运营 DD** | 供应链、生产、团队 | 运营效率如何？关键人是谁？ |
| **技术 DD** | 技术架构、研发能力 | 技术是否领先？能否规模化？ |
| **环境 DD** | ESG、环保合规 | 有无环境 liability？ |

### 退出方式

| 方式 | 英文 | 说明 | 特点 |
|------|------|------|------|
| **IPO（首次公开募股）** | Initial Public Offering | 公司在证券交易所上市 | 回报最高，但周期长、成本高 |
| **并购退出** | Trade Sale / M&A | 卖给战略买家或财务买家 | 最常见，速度快 |
| **二次转让** | Secondary Sale | 把老股卖给其他基金 | 流动性好，但可能折价 |
| **回购** | Buyback | 公司或创始人回购股份 | 保底退出方式 |
| **清算** | Write-off / Liquidation | 投资失败，认赔退出 | 最坏情况 |

## 十一、财务报表分析深度解析

### 三张报表的关系

**利润表（Income Statement）**：
- 回答：公司赚了多少钱？
- 核心公式：收入 - 成本 = 利润
- 时间维度：一段时期（如 2024 年全年）

**资产负债表（Balance Sheet）**：
- 回答：公司有多少家底？
- 核心公式：资产 = 负债 + 股东权益
- 时间维度：某个时点（如 2024 年 12 月 31 日）

**现金流量表（Cash Flow Statement）**：
- 回答：钱从哪来，到哪去？
- 核心公式：期初现金 + 现金流入 - 现金流出 = 期末现金
- 时间维度：一段时期

**三张报表的勾稽关系**：
1. 利润表的净利润 → 资产负债表的留存收益
2. 现金流量表的期末现金 → 资产负债表的现金科目
3. 资产负债表的变动 → 现金流量表的分类解释

### 关键财务比率详解

**盈利能力比率**：

| 比率 | 公式 | 意义 | 优秀标准 |
|------|------|------|---------|
| **毛利率** | 毛利润 / 收入 | 产品本身的盈利能力 | > 40%（软件）, > 20%（制造） |
| **营业利润率** | 营业利润 / 收入 | 主营业务的盈利能力 | > 15% |
| **净利润率** | 净利润 / 收入 | 最终盈利能力 | > 10% |
| **ROE（净资产收益率）** | 净利润 / 股东权益 | 股东投入的回报率 | > 15% |
| **ROA（总资产收益率）** | 净利润 / 总资产 | 资产的综合回报 | > 8% |
| **ROIC（投入资本回报率）** | 税后营业利润 / 投入资本 | 核心业务的真正回报 | > 12% |

**营运能力比率**：

| 比率 | 公式 | 意义 | 优秀标准 |
|------|------|------|---------|
| **存货周转天数** | 平均存货 / 日均销售成本 | 存货变现速度 | 越短越好 |
| **应收账款周转天数** | 平均应收账款 / 日均收入 | 回款速度 | < 60 天 |
| **应付账款周转天数** | 平均应付账款 / 日均采购成本 | 占用供应商资金能力 | 适度延长有利 |
| **现金转换周期** | 存货天数 + 应收天数 - 应付天数 | 从付现到收现的周期 | 越短越好 |

**偿债能力比率**：

| 比率 | 公式 | 意义 | 安全标准 |
|------|------|------|---------|
| **流动比率** | 流动资产 / 流动负债 | 短期偿债能力 | > 1.5 |
| **速动比率** | (流动资产 - 存货) / 流动负债 | 更严格的短期偿债能力 | > 1.0 |
| **资产负债率** | 总负债 / 总资产 | 杠杆水平 | < 60% |
| **利息保障倍数** | EBIT / 利息费用 | 偿付利息的能力 | > 3x |
| **债务/EBITDA** | 净债务 / EBITDA | 杠杆倍数 | < 3x |

## 十二、估值方法详解

### 绝对估值法

**DCF（现金流折现模型）**：

**核心思想**：公司价值 = 未来所有现金流的现值之和

**计算步骤**：
1. **预测自由现金流（FCF）**：未来 5-10 年的 FCF 预测
2. **确定折现率（WACC）**：加权平均资本成本
3. **计算终值（Terminal Value）**：永续增长假设下的价值
4. **折现求和**：将所有现金流折现到今天

**WACC 公式**：
```
WACC = (E/V) × Re + (D/V) × Rd × (1 - Tc)
```
- E：股权价值
- D：债务价值
- V = E + D：企业总价值
- Re：股权成本（通常用 CAPM 计算）
- Rd：债务成本
- Tc：企业所得税率

**CAPM（资本资产定价模型）**：
```
Re = Rf + β × (Rm - Rf)
```
- Rf：无风险利率（如 10 年期国债收益率）
- β：股票相对于市场的波动程度
- Rm：市场预期回报率
- (Rm - Rf)：市场风险溢价

**终值计算**：
```
终值 = FCF_n × (1 + g) / (WACC - g)
```
- g：永续增长率（通常 2-3%，不超过 GDP 增速）

**DCF 的优缺点**：
- **优点**：理论上最严谨，考虑未来增长
- **缺点**：对假设极度敏感，预测误差会被放大

### 相对估值法

**可比公司分析（Trading Comps）**：

**步骤**：
1. 选择同行业、同规模的可比上市公司
2. 计算各公司的估值倍数（P/E、EV/EBITDA 等）
3. 取中位数或平均值
4. 应用到目标公司

**常用倍数**：

| 倍数 | 适用场景 | 不适用场景 |
|------|---------|-----------|
| **P/E** | 盈利稳定的公司 | 亏损公司、强周期公司 |
| **EV/EBITDA** | 资本密集型、跨国比较 | 高杠杆公司 |
| **EV/Revenue** | 高增长、未盈利公司 | 成熟低增长公司 |
| **P/B** | 银行、保险公司 | 轻资产科技公司 |

**先例交易分析（Transaction Comps）**：

**与 Trading Comps 的区别**：
- 使用的是并购交易价格，而非市场交易价格
- 通常包含控制权溢价（Control Premium，20-40%）
- 反映的是"买下整个公司"的价格

### LBO（杠杆收购）模型

**核心逻辑**：
1. 用少量自有资金 + 大量借款收购公司
2. 用被收购公司的现金流偿还债务
3. 几年后卖出公司，赚取股权增值

**LBO 回报驱动因素**：

| 因素 | 说明 | 影响 |
|------|------|------|
| **购买价格** | 买得便宜还是贵 | 越低越好 |
| **杠杆倍数** | 债务/EBITDA | 适度杠杆放大回报 |
| **债务成本** | 借款利率 | 越低越好 |
| **EBITDA 增长** | 经营改善 | 增长越快越好 |
| **退出倍数** | 卖出时的估值 | 越高越好 |
| **持有期** | 持有时间 | 通常 4-7 年 |

**LBO 典型资本结构**：

| 层级 | 比例 | 成本 | 特点 |
|------|------|------|------|
| 银行债务 | 40-60% | 最低 | 有抵押，先偿还 |
| 高收益债券 | 20-30% | 中等 | 无抵押，次级 |
| 夹层融资 | 10-20% | 较高 | 股债混合 |
| 自有资金 | 20-40% | 股权回报 | 最后分配，风险最高 |

## 十四、风险管理基础

### 市场风险

**定义**：因市场价格变动导致损失的风险。

**度量方法**：

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| **VaR（风险价值）** | 在 X% 置信水平下，未来 N 天最大损失 | 组合整体风险 |
| **Expected Shortfall** | 超过 VaR 时的平均损失 | 尾部风险 |
| **Stress Testing** | 假设极端情景，评估损失 | 监管要求 |
| **Scenario Analysis** | 模拟特定情景（如 2008 年危机） | 战略风险 |

**VaR 计算方法**：

| 方法 | 原理 | 优缺点 |
|------|------|--------|
| **历史模拟法** | 用历史数据直接计算 | 简单，但假设历史会重复 |
| **方差-协方差法** | 假设正态分布 | 计算快，但低估尾部风险 |
| **蒙特卡洛模拟** | 随机模拟成千上万条路径 | 最灵活，但计算量大 |

### 信用风险

**定义**：交易对手违约导致损失的风险。

**度量指标**：

| 指标 | 说明 |
|------|------|
| **PD（违约概率）** | 未来一段时间内违约的可能性 |
| **LGD（违约损失率）** | 违约后损失的比例 |
| **EAD（违约风险暴露）** | 违约时的风险敞口 |
| **Expected Loss** | PD × LGD × EAD |

### 操作风险

**定义**：因内部流程、人员、系统不完善或外部事件导致损失的风险。

**例子**：
- 交易员输入错误指令（"胖手指"）
- 系统故障导致交易无法执行
- 内部欺诈（如 rogue trader）
- 自然灾害导致业务中断

## 十六、ESG（环境、社会、治理）

### ESG 投资

**定义**：在投资决策中考虑环境、社会和公司治理因素。

**E（环境）因素**：
- 碳排放和气候变化
- 资源使用和效率
- 污染和废弃物管理
- 生物多样性保护

**S（社会）因素**：
- 员工权益和多样性
- 供应链劳工标准
- 产品安全和质量
- 社区关系

**G（治理）因素**：
- 董事会独立性和多样性
- 高管薪酬合理性
- 股东权利保护
- 反腐败和合规

### ESG 评级

| 评级机构 | 特点 |
|----------|------|
| **MSCI ESG** | 最广泛使用的评级之一 |
| **Sustainalytics** | 晨星旗下，风险导向 |
| **CDP** | 专注气候变化披露 |
| **SASB** | 行业特定标准 |
| **TCFD** | 气候相关财务披露建议 |

### ESG 投资策略

| 策略 | 说明 |
|------|------|
| **负面筛选** | 排除烟草、武器等行业 |
| **正面筛选** | 选择 ESG 表现最好的公司 |
| **ESG 整合** | 将 ESG 因素纳入传统分析 |
| **影响力投资** | 主动追求社会/环境效益 |
| **股东参与** | 通过投票和对话推动改进 |
