# 客户复核准备

description: 为客户复核会议提供投资组合业绩摘要、资产配置分析、谈话要点及行动项准备。将分散的账户数据汇总为简练、专业的会议格式。适用于季度复核、年度检查或临时客户会议。触发词包括：“客户复核 (client review)”、“为[客户名]准备会议 (meeting prep for [client])”、“季度复核 (quarterly review)”、“为[客户名]做准备 (prep for [client name])”或“客户会议 (client meeting)”。

## 工作流

### 第 1 步：客户背景

收集或查询以下信息：
- **客户姓名**及家庭成员
- **账户类型**：应税账户、IRA、Roth、401(k)、信托等
- **管理资产总额 (AUM)**：跨账户汇总
- **投资政策说明书 (IPS)**：目标配置、风险承受能力、限制条件
- **生命周期阶段**：资产积累期、准退休期、退休期、财富传承期
- **上次会议日期**及任何待办行动项

### 第 2 步：投资组合业绩

针对每个账户及家庭整体资产：

| 指标 | 季初至今 (QTD) | 年初至今 (YTD) | 1 年期 | 3 年期 | 自成立以来 |
|--------|-----|-----|--------|--------|----------------|
| 投资组合回报率 | | | | | |
| 基准回报率 | | | | | |
| 阿尔法 (Alpha) | | | | | |

**业绩归因分析：**
- 哪些资产类别/头寸驱动了回报？
- 前三大贡献项与前三大减分项
- 是否存在单一头寸对整体产生的过度影响？

### 第 3 步：配置复核

当前配置与目标配置对比：

| 资产类别 | 目标 | 当前 | 偏离度 | 行动建议 |
|------------|--------|---------|-------|--------|
| 美国大盘股 | | | | |
| 美国中/小盘股 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 固定收益 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记任何超过 IPS 再平衡阈值（通常为 3-5%）的偏离。

### 第 4 步：谈话要点

生成会议议程：

1. **市场概览** (2-3 分钟)：简要宏观背景及展望
2. **投资组合业绩** (5 分钟)：我们的表现如何？原因是什么？
3. **配置复核** (5 分钟)：是否需要进行再平衡？
4. **规划更新** (5-10 分钟)：
   - 生活变化？（职业、健康、家庭、房产、教育）
   - 收入需求是否有变？
   - 税务状况更新
   - 遗产规划更新
5. **行动项** (5 分钟)：下次会议前我们需要完成什么？

### 第 5 步：主动建议

根据复核情况提出建议：
- 再平衡交易（若偏离度超过阈值）
- 税收亏损收割 (Tax-loss harvesting) 机会
- 现金部署或提款计划
- Roth 账户转换机会（如适用）
- 受益人信息更新或遗产规划需求
- 保险复核（寿险、伤残险、长期护理险）

### 第 6 步：输出

- 一页纸客户复核摘要（Word 或 PDF）
- 包含基准对比的业绩表格
- 资产配置饼图（当前 vs. 目标）
- 建议行动清单
- 会议议程

## 重要注意事项

- 会前深谙客户情况 —— 复习上次会议记录
- 从客户关注点出发，而非单纯陈述你想讲的内容
- 如果业绩不佳，请直接面对 —— 不要回避或粉饰
- 结尾务必提供清晰的行动项、下一步计划及完成日期
- 记录会议纪要以及 IPS 的任何变更
- 合规性：确保所有材料符合公司政策及监管要求

> **💡 Appendix: 领域知识小贴士**
>
> 欢迎来到财富管理的奇妙世界！为了让你更轻松地理解这些“高大上”的词汇，我们准备了一份简单易懂的说明：
>
> 1. **AUM (管理资产规模)**：简单来说，就是你交给专业机构打理的所有钱的总额。它是衡量一个理财师“管了多少钱”的关键指标。
> 2. **IPS (投资政策说明书)**：这相当于你和理财师之间的“投资合同”或“航海图”。它规定了你能冒多大险、想赚多少钱以及哪些东西绝对不能碰，确保投资不偏离初衷。
> 3. **Alpha (阿尔法)**：这是一个代表“超额能力”的加分项。如果市场整体涨了 10%，而你的组合涨了 12%，那多出来的 2% 就是 Alpha，代表了你或你的理财师跑赢大盘的本事。
> 4. **再平衡 (Rebalancing)**：想象你做菜讲究营养均衡。如果一段时间后“肉”变多了“菜”变少了，你得调整一下，把多出来的肉卖掉换成菜，回到最初健康的配比。这在投资中是控制风险的绝佳手段。
> 5. **税收亏损收割 (Tax-loss harvesting)**：这是一种“反败为胜”的小技巧。把亏损的股票卖掉来抵消盈利产生的税，就像是用“坏运气”来省钱，从而提高你的实际到手收益。
