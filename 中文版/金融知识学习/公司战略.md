## 1. 公司战略是什么

**翻译**：公司决定"做什么、不做什么、怎么做"的顶层规划。

**想象一个场景**：你是星巴克 CEO。公司已经在美国开了 15000 家店。接下来怎么办？
- 继续在美国开店？（市场渗透）
- 去中国开店？（市场开发）
- 卖茶饮？（产品开发）
- 做外卖平台？（多元化）

**这就是战略选择**。

## 2. 战略框架

### 2.1 Ansoff Matrix（安索夫矩阵）

|  | 现有产品 | 新产品 |
|------|---------|--------|
| **现有市场** | 市场渗透 | 产品开发 |
| **新市场** | 市场开发 | **多元化**（最危险） |

**举例**：
- 星巴克在中国开店 = 市场开发
- 星巴克推出冷萃 = 产品开发
- 星巴克收购 Teavana 茶饮 = 多元化

### 2.2 BCG Matrix（波士顿矩阵）

```
       高增长
        |
明星    |   问题
(Stars) |  (Question Marks)
--------+--------
金牛    |   瘦狗
(Cash Cows) |  (Dogs)
        |
       低增长
```

- **金牛（Cash Cows）**：低增长高份额 → 收钱养其他业务（可口可乐）
- **明星（Stars）**：高增长高份额 → 投资维持（iPhone）
- **问题（Question Marks）**：高增长低份额 → 决定投资还是放弃
- **瘦狗（Dogs）**：低增长低份额 → 出售或关闭

### 2.3 Porter's Five Forces（波特五力）

判断一个行业**值不值得进**：
1. **供应商议价能力**
2. **买家议价能力**
3. **替代品威胁**
4. **新进入者威胁**
5. **现有竞争强度**

## 3. 竞争优势（Competitive Advantage）

### 3.1 护城河（Moat）

**Warren Buffett 的概念**：一家公司能长期保持高利润的"壁垒"。

**五种护城河**：
- **网络效应**：用户越多越值钱（微信、Facebook）
- **品牌**：消费者愿意多付钱（可口可乐、苹果）
- **转换成本**：换供应商太麻烦（SAP、Salesforce）
- **成本优势**：比对手便宜（沃尔玛、台积电）
- **规模效应**：越大越便宜（亚马逊物流）

### 3.2 VRIO 框架

判断一个资源是否带来可持续竞争优势：
- **V**（Valuable）：有价值？
- **R**（Rare）：稀缺？
- **I**（Inimitable）：难以模仿？
- **O**（Organized）：组织能利用？

**四个都满足 = 可持续竞争优势**。

## 4. 战略合作 vs 并购

| | 并购 | 战略合作 |
|------|------|---------|
| **控制权** | 100% 拿下 | 共享 |
| **投入** | 大（数十亿） | 小（数千万） |
| **风险** | 高 | 低 |
| **退出** | 难 | 易 |

**举例**：
- 苹果和 Google 合作：Google 是 Safari 默认搜索 → 苹果每年从 Google 收 200 亿美元
- 苹果没有"收购" Google，因为 Google 太大

---

## 5. 真实战略合作案例

- **Salesforce + Google**：2017 年达成战略合作，Google Cloud 是 Salesforce 首选云
- **Microsoft + OpenAI**：2019 年投资 10 亿，2023 年再投 100 亿 → 微软云独家运行 OpenAI 模型
- **NVIDIA + Mercedes-Benz**：合作自动驾驶
- **苹果 + TSMC**：苹果是 TSMC 最大客户

---

## 6. 给小白的一句话

> **战略就是"选择做什么、不做什么"**。**安索夫矩阵看增长方向，BCG 矩阵看资源分配，波特五力看行业吸引力**。**护城河是长期赚钱的根基**——没有护城河的公司，利润迟早被竞争吃掉。**战略合作是"轻量版并购"**——风险低、退出容易，但控制权也有限。
