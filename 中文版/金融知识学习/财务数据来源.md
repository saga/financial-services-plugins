## 1. 为什么需要专业数据源

在金融世界，"从哪里拿数据"决定了分析的起点。

**类比**：做饭需要食材。如果食材是臭的（错误数据），再好的厨师也做不出好菜。金融分析师 80% 的时间在"找数据"——不是因为他们不行，而是数据来源太分散。

## 2. 核心数据库：你应该认识的七大金刚

### 2.1 Bloomberg Terminal（彭博终端）

**一句话**：金融界的"瑞士军刀"——什么都能干，但什么都得会操作。

| 维度 | 内容 |
|------|------|
| **价格** | ~$24,000/年/终端（最贵，但行业标配） |
| **核心功能** | 实时行情、公司/行业/市场数据、新闻、分析工具 |
| **独特优势** | 固收数据（债券、CDS、IRS）无人能及 + IB 聊天（Bloomberg Chat）|
| **学习成本** | 极高（有独立的操作语言，像 DOS 命令行） |
| **主力用户** | 卖方交易员、固收分析师、投行 M&A |

**彭博终端的黑话**：
- `AAPL US Equity DES` → 苹果公司描述页
- `IBM US Equity FA` → IBM 财务分析页
- `SPX Index MEMB` → 标普 500 成分股列表

### 2.2 Capital IQ（S&P Global）

**一句话**：公司数据最强，Excel 集成最好。

| 维度 | 内容 |
|------|------|
| **价格** | ~$15,000-30,000/年 |
| **核心功能** | 公司财务、Comps、行业数据、交易数据 |
| **独特优势** | Excel 插件（CapIQ Plugin）直接拉数据进模型 |
| **学习成本** | 中 |
| **主力用户** | 买方（PE/VC）、投行分析师 |

**CapIQ 的 Excel 公式**：
```
=CIQ("AAPL", "IQ_TOTAL_REV", 2025)     ← 拉 2025 年总收入
=CIQ("AAPL", "IQ_MARKET_CAP")           ← 拉当前市值
```

### 2.3 FactSet

**一句话**：分析师工具集 + 组合管理。

| 维度 | 内容 |
|------|------|
| **价格** | ~$12,000-20,000/年 |
| **核心功能** | 公司数据、行业统计、组合分析、图表工具 |
| **独特优势** | 界面好、Excel 集成强、Portfolio Analysis 出色 |
| **学习成本** | 中 |
| **主力用户** | 买方分析师、组合经理、投行 |

### 2.4 Refinitiv Eikon / Workspace（LSEG / 原汤森路透）

**一句话**：路透社的数据终端。

| 维度 | 内容 |
|------|------|
| **核心功能** | 实时行情、公司基本面、IBES 分析师一致预期、新闻 |
| **独特优势** | 新闻源极强（路透社自己就是新闻社）+ IBES 预期数据 |
| **主力用户** | 全球投行、财富管理 |

**IBES**：最权威的分析师一致预期数据库。想看"华尔街共识对苹果 Q3 的预测是多少"→ 看 IBES。

### 2.5 PitchBook

**一句话**：PE/VC/并购领域的 Bloomberg。

| 维度 | 内容 |
|------|------|
| **价格** | ~$20,000+/年 |
| **核心功能** | PE/VC 交易数据、LP/GP 关系图、退出数据、融资轮 |
| **独特优势** | 私有公司的信息（没上市的也收录） |
| **主力用户** | PE/VC、投行 M&A |

### 2.6 Preqin

**一句话**：PE/VC/对冲基金的"选基金"数据。

| 维度 | 内容 |
|------|------|
| **核心功能** | PE/VC/对冲基金业绩（IRR/MOIC）、LP 配置数据、基金募集信息 |
| **独特优势** | LP 可以用来看哪个 GP 表现好，GP 可以看 LP 的配置偏好 |
| **主力用户** | LP（养老基金、捐赠基金）、基金 of funds |

### 2.7 SEC EDGAR（免费）

**一句话**：所有美国上市公司必须向 SEC 申报的文件库。

| 维度 | 内容 |
|------|------|
| **价格** | 免费 |
| **核心文件** | 10-K（年报）、10-Q（季报）、8-K（重大事件）、S-1（IPO 招股书）、Proxy Statement（股东会材料） |
| **独特优势** | 原始数据——没有被终端"处理"过 |

**EDGAR 搜索**：
```
google: "AAPL 10-K site:sec.gov"   ← 找苹果年报
```

## 3. 其他重要数据源

### 3.1 经济数据

| 来源 | 数据 | 用途 |
|------|------|------|
| **美联储 FRED** | 利率、就业、GDP、M2 | 宏观分析 |
| **BLS** | CPI、非农就业 | 通胀、就业 |
| **BEA** | GDP、国际收支 | 宏观基础 |
| **OECD** | 跨国比较数据 | 国际宏观 |
| **IMF** | 各国经济指标 | 国际宏观 |

### 3.2 行业/专业数据

| 领域 | 数据源 | 
|------|--------|
| 地产 | CoStar, REIS |
| 医疗 | IQVIA, CMS |
| 油气 | Rystad, Wood Mackenzie |
| 科技 | IDC, Gartner |
| 汽车 | IHS Markit, J.D. Power |
| 电商 | Similarweb, Sensor Tower |

### 3.3 另类数据（Alternative Data）

| 类型 | 例子 | 用途 |
|------|------|------|
| 信用卡交易 | 由 Yodlee, Earnest 等聚合 | 预测零售公司季度收入 |
| 卫星图 | 停车场车辆数、油罐阴影 | 预测沃尔玛人流量、石油库存 |
| APP 下载 | Sensor Tower, App Annie | 预测科技公司用户增长 |
| 招聘信息 | Revelio, LinkUp | 公司是否在快速扩张？ |
| 船运数据 | Marine Traffic, Panjiva | 国际贸易流量 |

## 4. 数据质量等级

| 等级 | 类型 | 可靠度 | 例子 |
|------|------|--------|------|
| **L1** | 审计过的数据 | 最高 | 10-K 中的审计后财务报表 |
| **L2** | 公司发布但未经审计 | 高 | 公司新闻稿中的季度数据 |
| **L3** | 第三方经过验证 | 中-高 | Bloomberg/CapIQ 拉取的数据 |
| **L4** | 分析师估计 | 中 | 卖方研报中的盈利预测 |
| **L5** | 自建模型估算 | 低-中 | 基于行业趋势的猜测 |
| **L6** | 二手/聚合源 | 低 | Wikipedia、免费网站 |

**黄金法则**：优先用 L1-L3，引用时要注明数据来源和日期。

## 5. 数据源选择建议

| 你用在哪 | 推荐 |
|----------|------|
| 公司财务分析 | Capital IQ（方便） 或 Bloomberg（最强） |
| Comps 可比公司 | Capital IQ + FactSet |
| PE/VC 交易 | PitchBook |
| 固收/债券 | Bloomberg |
| 分析师预期 | IBES（LSEG Eikon） |
| 宏观经济 | FRED + Bloomberg |
| 行业报告 | 行业专门数据源 |
| 美股招股书 | SEC EDGAR（免费） |

---

## 给小白的一句话

> 金融数据源就像不同等级的"食材市场"——SEC EDGAR 是免费的菜市场（原始但真材实料），Bloomberg 是进口超市（什么都有但贵到心疼），Capital IQ 是你专属的切菜工（帮你处理好了直接放进 Excel 里炒）。厉害的不是"有这些数据"——谁都买得起——而是在几千行数据中发现别人没看到的规律。
