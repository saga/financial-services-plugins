## 1. 什么是竞业分析（Competitive Analysis）

**竞业分析** 是系统性地研究目标公司所处竞争环境的过程——谁是主要对手、护城河是什么、公司在行业里排第几。

**类比**：你想开一家奶茶店。在做生意之前，你会站在要租的店面门口数人流量，看看方圆 500 米内有多少家奶茶店（直接的竞争），有多少家咖啡馆（间接的竞争），看看哪家排队最长。竞业分析就是把这种做法系统化、专业化。

## 2. 波特五力（Porter's Five Forces）

由哈佛教授迈克尔·波特（Michael Porter）1979 年提出，是竞业分析的**圣经级框架**。

### 2.1 五力解析

```
                    ┌──────────────────┐
                    │   新进入者的威胁    │  ← 门槛低 = 谁都能进 ≠ 不赚钱
                    └────────┬─────────┘
                             │
    ┌──────────────────┐     │     ┌──────────────────┐
    │   供应商的议价力    │─────┼─────│   买家的议价力     │
    │ (你原材料是谁说了算) │     │     │ (客户能不能压你价) │
    └──────────────────┘     │     └──────────────────┘
                             │
                    ┌────────┴─────────┐
                    │   替代品的威胁    │  ← 不只竞争公司，整个产品形式
                    └──────────────────┘
                    ┌──────────────────┐
                    │ 现有竞争者间的竞争  │  ← 行业中心：越激烈利润越低
                    └──────────────────┘
```

### 2.2 每项的力量怎么判断

| 力量 | 强弱信号（强 = 行业不好做） |
|------|----------------------|
| **新进入者** | 不需要许可证（餐馆）/ 技术门槛低 / 资本需求小 |
| **供应商** | 只有 1-2 个供应商 / 他们的产品你无可替代 / 他们可以自己卖成品 |
| **买家** | 一个大客户占 40% 收入 / 替代品多 / 转换成本低 |
| **替代品** | Netflix 替代 DVD、Zoom 替代出差——完全不同品类但解决同一个需求 |
| **内部竞争** | 十几家同等规模公司、增长缓慢、固定成本高（航空公司/钢铁厂）|

### 2.3 五力分析例子：航空业

| 力量 | 状态 | 原因 |
|------|------|------|
| 新进入者 | 中等 | 资本壁垒高（飞机贵），但新航线可以租赁飞机 |
| 供应商 | 强 | 只有波音和空客两大制造商，飞机订单排到 5 年后 |
| 买家 | 强 | 乘客只比较价格，一分钟就可以换航司 |
| 替代品 | 中 | 高铁（短途）/ Zoom（商务出差减少）|
| 内部竞争 | 极强 | 价格战、运力过剩 |

**结论**：航空业利润极薄——五力分析完美验证了这一点。

## 3. SWOT 分析

| | 内部因素 | 外部因素 |
|---|---------|---------|
| **正面** | **S**trengths（优势）| **O**pportunities（机会）|
| **负面** | **W**eaknesses（劣势）| **T**hreats（威胁）|

### 3.1 怎么做一个好 SWOT

**反面教材**（太肤浅）：
- S："团队能力强"
- W："品牌知名度不高"
- O："市场在增长"
- T："竞争激烈"

**正确做法**——每项都要具体、可量化：

- S："研发团队 15 人，5 个 PhD，行业平均经验 12 年；拥有 8 项核心专利"
- W："销售渠道 80% 依赖一位大代理，他走了 = 收入断崖"
- O："RCEP 生效后，对日本出口关税从 15% 降到 0%"
- T："三家公司同时在做竞品，其中一家融资 $1 亿"

## 4. 竞争定位矩阵

### 4.1 二维定位图

画一个 X-Y 图，用两个维度来定位所有竞争对手。

**举例（奶茶行业）**：

```
价格高 ↑
       │ 喜茶           奈雪の茶
       │   (高端 + 新鲜)    (高端 + 空间)
       │
       │         CoCo
       │          (中端)
       │   蜜雪冰城
价格低 │     (低价 + 规模)
       └──────────────────→
            少            多
              品种/门店数
```

### 4.2 市场份额 "龙卷风图"（Tornado Chart）

```
市场 100% ├─────────────────────────────────────────┤
           │█████████████ 公司A (35%)               │
           │███████████ 公司B (28%)                 │
           │████ 公司C (10%)                       │
           │███ 公司D (8%)                         │
           │█ 其他 (19%)                           │
```

**前 3 名** 通常占整个市场的 60-80%。

## 5. BCG 矩阵（增长-份额矩阵）

由波士顿咨询集团（BCG）开发，用两个维度将业务分类：

|  | 市场份额低 | 市场份额高 |
|---|----------|----------|
| **增长率高** | **问号**（Question Mark）| **明星**（Star）|
| **增长率低** | **瘦狗**（Dog）| **现金牛**（Cash Cow）|

- **明星**：投钱养大 → 未来的现金牛（如 Apple Watch）
- **现金牛**：少投入、多收钱、养明星和问号（如 iPhone）
- **问号**：可能变明星也可能变瘦狗 → 挑着投
- **瘦狗**：考虑卖掉或关掉（如 iPod）

## 7. 市场规模计算：TAM SAM SOM

| 缩写 | 全称 | 含义 | 例子（奶茶） |
|------|------|------|-----------|
| **TAM** | Total Addressable Market | 全球/全国所有可能的需求 | 全中国所有人喝的茶饮 $3000 亿 |
| **SAM** | Serviceable Addressable Market | 你的产品能触及的 | 你所在城市及周边城市的茶饮 $300 亿 |
| **SOM** | Serviceable Obtainable Market | 你实际能拿到的 | 你开第1年能拿到的 $3000 万 |

**常识**：初创公司写 TAM $1000 亿但 SOM 可能只有 $100 万——TAM 大不等于你能拿到。投资者最关心的是 **TAM → SOM 的路径**。

## 给小白的一句话

> 竞业分析就是"搞清楚这场仗你打的是谁、他有多强、你凭什么能赢"。波特五力给你一个"战争地图"（行业的利润被谁偷走了？），SWOT 是你自己的"装备清单"（强在哪、弱在哪？），竞争矩阵是你的"敌军分布图"（谁在哪里？）。没有竞业分析就做投资 = 连对手是谁都不知道就上战场 = 大概率输掉。
