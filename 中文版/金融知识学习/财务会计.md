# 财务会计（Financial Accounting）

> 适用文件：variance-commentary/SKILL.md, roll-forward/SKILL.md, accrual-schedule/SKILL.md, 3-statement-model/SKILL.md, 任何提到"应计/试算表/总账/三大报表"的文件

---

## 1. 会计是什么

**翻译**：记录、分类、汇总一个组织的财务交易的系统。

**核心原则**（GAAP / IFRS）：
- **权责发生制（Accrual）**：费用/收入在"发生时"记录，不是"收/付现金时"
- **配比原则（Matching）**：费用和产生收入的同一期确认
- **历史成本（Historical Cost）**：资产按买入价记账
- **谨慎性（Conservatism）**：不确定时，宁可低估资产/收入、高估负债/费用

## 2. 三大财务报表

### 2.1 利润表（Income Statement）

**翻译**：看你赚不赚钱。

```
收入（Revenue）
- 销售成本（COGS）
= 毛利（Gross Profit）
- 营业费用（SG&A + R&D）
= 营业利润（Operating Income）
+/- 利息/其他
- 所得税
= 净利润（Net Income）
```

### 2.2 资产负债表（Balance Sheet）

**翻译**：看家底厚不厚。

**核心恒等式**：**资产 = 负债 + 股东权益**

### 2.3 现金流量表（Cash Flow Statement）

**翻译**：看真金白银的进出。

**三段**：经营活动（CFO）+ 投资活动（CFI）+ 融资活动（CFF）

**FCF（自由现金流）** = CFO - CapEx

---

## 3. GAAP vs 调整后

### 3.1 GAAP

法定、严格、上市公司必须用。

### 3.2 调整后（Non-GAAP / Adjusted）

公司自己剔除"一次性项目"：重组费用、收购成本、SBC、减值、汇兑损益。

**警惕信号**：GAAP vs 调整后差距 > 30% → 可能过度调整。

---

## 4. 应计项目（Accruals）

**核心**：钱进出跟记录的时间可能不同。

**典型应计**：

| 类型 | 举例 |
|------|------|
| **应付工资** | 12 月工资 1 月发 → 12 月应计 |
| **应付利息** | 季付利息，月度应计 |
| **预提费用** | 已知金额但未开发票 |
| **递延收入** | 客户预付，分期确认 |
| **坏账准备** | 预计收不回的应收账款 |

**应计流程**：
1. 期末计提（借：费用，贷：应计负债）
2. 下期初回转（借：应计负债，贷：费用）
3. 实际收到发票时正常过账

---

## 5. 滚调表（Roll-forward Schedule）

**翻译**：账户余额从期初到期末的变动明细。

```
期初余额
+ 新增/新活动
+ 应计
- 回转
- 付款/结算
+ 重分类
+ 外汇折算
= 期末余额
```

**作用**：验证期末余额、识别异常变动、提供审计轨迹。

---

## 6. 月末结账（Month-End Close）

**典型流程**（1-10 天）：

| 日 | 工作 |
|---|------|
| **D+1-D+3** | 录入所有交易、过账日记账 |
| **D+4-D+5** | 银行对账、应计项目计提 |
| **D+6-D+7** | 子公司报表合并、内部对账 |
| **D+8-D+9** | 差异说明、管理层审阅 |
| **D+10** | 财务报表定稿、签字 |

---

## 7. 波动分析（Variance Analysis）

**翻译**：实际 vs 预算/前期的差异。

**关键概念**：
- **有利（Favorable）**：收入高 / 费用低
- **不利（Unfavorable）**：收入低 / 费用高
- **材料性阈值**：通常 5% 或 $10,000（取较大者）

**波动分解**：价格影响 + 销量影响 + 混合影响 + 汇率影响

---

## 8. 给小白的一句话

> **会计是商业的"语言"**——不会会计就像不会说话。**权责发生制是核心**——记录的是"经济实质"不是"现金流时点"。**三大报表是窗口**：利润表看赚钱能力，资产负债表看家底，现金流量表看真金白银。**GAAP 是法律底线，调整后可能粉饰**——警惕差距过大。
