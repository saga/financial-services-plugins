# 投委会与投资备忘录（Investment Committee & IC Memo）

> 适用文件：ic-memo/SKILL.md, ic-memo.md (command), deal-screening/SKILL.md, 任何提到"投委会/IC Memo/投资决策"的文件

---

## 1. 什么是投委会（Investment Committee, IC）

**投委会** 是 PE/VC 基金内部的最高决策机构——所有投资决策必须经过投委会批准才能执行。

**类比**：一家公司的董事会。CEO 可以推荐收购，但最终要董事会投票。同理——投资团队可以推荐投某公司，但必须投委会批准才能签支票。

---

## 2. 投委会的构成

| 角色 | 谁 | 权限 |
|------|-----|------|
| **主席（Chair）** | 基金创始合伙人 / CIO | 一票否决权 |
| **常务委员** | 资深合伙人、行业组长 | 投票权 |
| **临时委员** | 外部专家（特定项目） | 咨询权（通常不投票）|
| **列席人员** | 项目组成员、Associate | 无投票权 |

### 2.1 典型规模

- **小型基金**（< $5 亿）：3-5 人投委会
- **中型基金**（$5-30 亿）：5-8 人投委会
- **大型基金**（>$30 亿）：8-12 人，常分设"行业分委会"

---

## 3. 投委会怎么开会

### 3.1 节奏

- **周度 IC**：每周一次，审核筛选后的项目（deal screening）
- **专题 IC**：特定项目需要投资决策时特别召开

### 3.2 流程

```
第1步：项目组提交 IC Memo（投资备忘录） 
   ↓  提前 3-7 天发
第2步：投资团队做口头汇报（Pitch）
   ↓  30-45 分钟讲述 + 15-30 分钟 Q&A
第3步：投委会委员"拷问"
   ↓  风险的各个方面
第4步：投票表决
   ↓  多数通过（有些基金要求全票 + 创始合伙人批准）
第5步：批准（有条件或无条件的）
   ↓  "可以投，但必须满足以下条件..."
第6步：执行交易（Deal Execution）
```

### 3.3 表决机制

| 机制 | 需要 | 适用于 |
|------|------|--------|
| 简单多数 | >50% 赞成 | 小项目 |
| 绝对多数 | >66% 或 >75% 赞成 | 大中型项目 |
| 创始合伙人批准 | 简单多数 + 创始合伙人赞成 | 多数基金 |
| 一票否决 | 主席/CIO 一人反对即否决 | 多数基金 |

## 4. IC Memo — 投资备忘录

### 4.1 定义

IC Memo 是投资团队写给投委会的"投资申请书"——说清为什么要投、多少钱、风险是什么。

**类比**：大学毕业论文答辩。导师组坐在对面，你要用 篇论文 + 口头陈述 + 回答问题来证明你的结论是对的。IC Memo 就是那篇"论文"。

### 4.2 IC Memo 的典型结构

| 章节 | 内容 | 页数 |
|------|------|------|
| **1. 执行摘要** | 一页讲清：投什么公司、投多少、为什么 | 1-2 |
| **2. 投资机会** | 行业的"风口"：为什么这个赛道好 | 3-5 |
| **3. 公司概况** | 目标是谁、谁在管、怎么赚钱 | 3-5 |
| **4. 投资论点** | 我们买它的核心理由是什么 | 2-4 |
| **5. 财务分析** | 历史财务、增长动力、盈利质量 | 5-8 |
| **6. 估值与回报** | Comps / DCF / LBO 估值 + IRR/MOIC 预测 | 5-8 |
| **7. 风险管理** | 最大的 5 个风险 + 缓解措施 | 3-5 |
| **8. 退出策略** | 卖给谁？上市？几年内？ | 2-3 |
| **9. 交易结构** | 股权/债务/管理层留用的分配 | 2-3 |
| **10. 下一步行动** | 签字日程、DD 进度、条件满足时间表 | 1-2 |

**总页数**：40-60 页（正文）+ 50-200 页附件

### 4.3 IC Memo 中的关键部分

#### 投资论点（Investment Thesis）

**三个核心问题**：
1. 为什么要在这个行业买？（"风口"）
2. 为什么要买这个公司？（"独特优势"）
3. 为什么是我们？（"我们的优势"）

**例子**（买一家物流公司）：
> 由于电商渗透加速，中国冷链物流市场规模预计到 2030 年翻三倍。XX 物流是中国冷链物流的第三名，运营利润率比行业平均高 5 个百分点。我们的运营合伙人团队有丰富的物流行业经验，可以在并购后优化车队调度系统，进一步提升利润率。

#### 风险管理

每个风险必须有对应的"缓解措施"：

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|---------|
| 客户集中度高（Top 3 = 40% 收入） | 中 | 高 | 签长协 5 年期合同保护 |
| 管理层流失 | 低-中 | 极高 | 管理层留用机制 + 股权激励锁 4 年 |
| 监管风险 | 低 | 中 | 聘律所做监管审查 |

## 5. IC 会议上委员的经典问题

| 问题 | 在问什么 |
|------|---------|
| "这个公司的护城河是什么？" | 竞争对手为什么不能轻易复制？ |
| "这个倍数是不是买贵了？" | 估值相对同行和历史如何？ |
| "如果明年宏观衰退，这公司会怎样？" | 抗周期能力？ |
| "为什么他们愿意卖？" （卖方动机） | 卖方是否有隐藏问题？ |
| "我们的增值在哪里？" | PE 基金能做什么公司自己做不了的事？ |
| "退出路径是什么？有 3-5 个潜在买家吗？" | 退出风险——不能只靠 IPO |
| "DD 还缺什么？" | 正式签字前还有哪些未解答的问题 |

### 5.1 最可怕的回答

- "我们觉得..."——而不是"数据表明..."
- "风险很小"——但没有量化或缓解措施
- "管理层说..."——但没有独立验证
- "行业会一直增长"——但没有结构性论据

**投委会找的不是"完美的投资"（不存在），而是"风险已充分识别且有应对方案的投资"。**

## 6. 投委会的文化差异

| | 美国 | 欧洲 | 亚洲 |
|---|------|------|------|
| **风格** | 直来直去 | 较含蓄 | 层级分明 |
| **讨论** | 踊跃、平等讨论 | 综合委员意见 | 资历优先 |
| **速度** | 快（一周内） | 慢（2-3 周） | 中-快（依基金） |
| **文件** | 精简 | 详细 | 较详细 |

## 给小白的一句话

> 投委会就是 PE/VC 的"董事会投票"——投资团队拿到一个项目不能直接买，必须写"投委会备忘录"向决策委员推销。IC Memo 本质上是"投资申请书"：说清买什么、多少钱、为什么、可能出什么问题。如果论文没写好、答辩时回答不了问题——对不起，这个项目出不了门。
