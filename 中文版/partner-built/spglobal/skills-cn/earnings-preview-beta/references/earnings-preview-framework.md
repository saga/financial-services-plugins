# 财报预览参考

## 查询框架

本参考提供了用于构建财报预览的查询框架。

### 查询1：基本信息

```
[公司] 股票代码 板块 市值 股价 52周区间
```

**目的：** 确认公司标识和当前估值水平

**预期输出：**
- 股票代码（如 NASDAQ: AAPL）
- 交易所和板块
- 市值（通常以十亿计）
- 当前股价
- 52周高低价和当前价所处的百分位

### 查询2：共识估计

```
[公司] 共识估计 收入 EPS EBITDA 下季度 华尔街估计
```

**目的：** 获取华尔街对即将发布的财报的共识预期

**预期输出：**
- 共识收入估计
- 共识EPS估计
- 共识EBITDA估计（如适用）
- 估计数量（有多少分析师发布估计）
- 最高和最低估计（范围）

### 查询3：近期股价表现

```
[公司] 股价 1个月 3个月 YTD 表现
```

**目的：** 了解近期股价走势和趋势

**预期输出：**
- 1个月回报
- 3个月回报
- YTD回报
- 与板块/指数的比较
- 近期是否有重大波动

### 查询4：历史超预期/低于预期

```
[公司] 财报 超预期 低于预期 意外 过去4个季度
```

**目的：** 识别历史超预期/低于预期模式

**预期输出：**
- 最近4-6个季度的超预期/低于预期历史
- 超预期/低于预期的幅度
- 连续超预期/低于预期的模式
- 股价对超预期/低于预期的反应

### 查询5：近期发展

```
[公司] 最新新闻 近期发展 重大公告 过去60-90天
```

**目的：** 识别可能影响即将发布财报的近期事件

**预期输出：**
- 产品发布公告
- 管理层变动
- 重大合同或合作
- 法律或监管事项
- 并购活动

### 查询6：历史财务数据

```
[公司] 季度损益表 收入 EBITDA 净利润 EPS 过去4个季度
```

**目的：** 获取历史财务数据用于比较

**预期输出：**
- 最近4个季度的收入
- 最近4个季度的EBITDA
- 最近4个季度的净利润
- 最近4个季度的EPS
- 同比变化趋势

### 查询7：管理层指引

```
[公司] 指引 上一季度指引 管理层指引 收入 EPS 下一季度
```

**目的：** 获取管理层发布的指引用于比较

**预期输出：**
- 收入指引范围
- EPS指引范围
- 其他关键指引指标
- 指引与共识的比较

### 查询8：细分数据

```
[公司] 细分收入 业务单元 产品线 过去4个季度
```

**目的：** 获取按细分分类的收入数据

**预期输出：**
- 按业务单元分类的收入
- 按地理区域分类的收入（如有）
- 按产品线分类的收入（如有）
- 各细分的增长趋势

## 数据解读指南

### 共识估计解读

- **窄范围 vs 宽范围：** 窄范围表示分析师高度共识；宽范围表示不确定性
- **趋势：** 估计是否在过去几周上调或下调？
- **vs 指引：** 共识是否高于或低于管理层指引？

### 超预期/低于预期解读

- **持续性：** 公司是否经常超预期或低于预期？
- **幅度：** 超预期/低于预期的幅度是否很大？
- **反应：** 股价通常如何反应超预期/低于预期？

### 估值解读

- **当前估值：** 与历史范围相比如何？
- **增长预期：** 当前的估值是否反映了高增长预期？
- **风险/回报：** 在当前价位，风险/回报比是否有利？

## 财报前检查清单

在发布预览前，验证：

- [ ] 公司标识符（股票代码、名称）正确
- [ ] 财报日期确认
- [ ] 共识估计可用
- [ ] 历史超预期/低于预期数据可用
- [ ] 有可形成交易建议的数据
- [ ] 预览在1-2页以内
- [ ] 交易建议清晰且可操作
